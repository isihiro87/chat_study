## 主な改善パターン

1. **同教材内ペアの混線** — 絵踏/踏絵、障壁画/濃絵、銅鏡/銅剣、ひらがな/カタカナなど、同 JSON 内に似た概念がある場合に back の弁別ヒントが不足
2. **back の質問が広すぎる** — 「○○の人物は？」「○○の時代は？」のように front 以外の中学レベル正答も成立してしまう
3. **front が数値/年代/範囲のみ** — その数値に該当する事象が複数あり一意に決まらない
4. **front と back のミスマッチ** — front が物・back が人物を尋ねるなど対応のずれ
5. **抽象的な定義系** — 産業革命・賠償金など、答えが一般概念にとどまり狭めるヒントがない

## 構造

各 md ファイルは元 JSON と 1:1 対応:

```
data/content/history/{era-folder}/{topic}.json
  → data/content/history/_fc-review/{era-folder}/{topic}.md
```

要修正FC が 0 件のファイルでも `特記事項なし` として md を作成済み（網羅性確認用）。
