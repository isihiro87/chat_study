# 酸素がかかわる化学変化 発展問題

## 大問1　金属の酸化と質量変化

ある生徒が、銅粉0.80 gをステンレス皿にのせ、空気中で十分に加熱する実験を行った。加熱後の物質の質量は1.00 gであった。

Q1-1: この反応の化学反応式を書け。
A1-1: 2Cu + O₂ → 2CuO

Q1-2: 加熱後にできた黒い物質の名称を答えよ。
A1-2: 酸化銅

Q1-3: 銅0.80 gと結びついた酸素の質量を求めよ。
A1-3: 1.00 − 0.80 = 0.20 g

Q1-4: この実験から、銅と結びつく酸素の質量比（銅：酸素）を最も簡単な整数比で求めよ。
A1-4: 0.80 : 0.20 = 4 : 1

## 大問2　マグネシウムの燃焼と計算

マグネシウムの粉末を空気中で完全に燃焼させる実験を何回か行い、マグネシウムの質量と燃焼後にできた酸化マグネシウムの質量の関係を調べた。マグネシウム0.60 gを燃焼させたところ、酸化マグネシウムは1.00 gできた。

Q2-1: マグネシウムの燃焼の化学反応式を書け。
A2-1: 2Mg + O₂ → 2MgO

Q2-2: マグネシウム0.60 gと結びついた酸素の質量を求めよ。
A2-2: 1.00 − 0.60 = 0.40 g

Q2-3: マグネシウムと結びつく酸素の質量比（マグネシウム：酸素）を最も簡単な整数比で求めよ。
A2-3: 0.60 : 0.40 = 3 : 2

Q2-4: マグネシウム1.20 gを完全に燃焼させると、酸化マグネシウムは何g できるか。
A2-4: マグネシウム：酸化マグネシウム = 0.60 : 1.00 = 3 : 5 より、1.20 : x = 3 : 5、x = 2.00 g

## 大問3　酸化銅の還元実験の考察

酸化銅と炭素の粉末を混合して試験管で加熱する実験を行った。以下の問いに答えよ。

Q3-1: この実験でガラス管の先を石灰水に入れておく目的を説明せよ。
A3-1: 反応で二酸化炭素が発生するかどうかを確認するため。

Q3-2: 十分に加熱した後、火を消す前にまずガラス管を石灰水から抜く。その理由を説明せよ。
A3-2: 試験管が冷えて内部の気圧が下がり、石灰水が逆流して高温の試験管が割れる危険があるため。

Q3-3: この反応で炭素が果たしている役割を、「酸素」という語を使って説明せよ。
A3-3: 炭素は酸化銅から酸素を奪い取る役割を果たしている。

Q3-4: この反応において、酸化と還元の両面から化学変化を説明せよ。
A3-4: 酸化銅は酸素を失って銅に還元され、同時に炭素は酸素と結びついて二酸化炭素に酸化されている。酸化と還元は同時に起こっている。

Q3-5: 実験後、試験管内の物質が完全に銅に変化したかどうかを確かめる方法を1つ答えよ。
A3-5: 物質を薬さじでこすり、赤い金属光沢が見られるか確認する。（または、電気を通すか調べる。）

## 大問4　酸素との結びつきやすさと還元

次の表は、金属と炭素について酸素との結びつきやすさを比較したものである。

| 物質 | 酸素との結びつきやすさ |
|------|---------------------|
| マグネシウム | 大きい |
| 炭素 | ↑ |
| 鉄 | ｜ |
| 銅 | ↓ |
| 金 | 小さい |

Q4-1: 酸化銅を炭素で還元できる理由を、表を用いて説明せよ。
A4-1: 炭素は銅よりも酸素との結びつきやすさが大きいため、酸化銅から酸素を奪い取ることができる。

Q4-2: 酸化鉄を炭素で還元することは可能か。表をもとに理由とあわせて答えよ。
A4-2: 可能である。炭素は鉄よりも酸素との結びつきやすさが大きいため、酸化鉄から酸素を奪うことができる。

Q4-3: 酸化マグネシウムを炭素で還元することは可能か。表をもとに理由とあわせて答えよ。
A4-3: できない。マグネシウムは炭素よりも酸素との結びつきやすさが大きいため、炭素では酸化マグネシウムから酸素を奪うことができない。

Q4-4: 製鉄所では鉄鉱石（主成分は酸化鉄）をコークス（炭素）で還元して鉄を得ている。この反応が可能である理由を、酸素との結びつきやすさに触れて説明せよ。
A4-4: 炭素は鉄よりも酸素との結びつきやすさが大きいので、酸化鉄中の酸素を炭素が奪い取り、鉄を還元することができる。

## 大問5　総合・思考問題

Q5-1: 木炭（炭素）を空気中で燃やすと二酸化炭素が発生し、質量が減少したように見える。しかし、密閉容器の中で木炭を燃焼させると全体の質量は変化しない。この理由を「酸化」と「気体」という語を使って説明せよ。
A5-1: 木炭の酸化によって生じた二酸化炭素は気体であり、開放状態では空気中に逃げるため質量が減少したように見える。密閉容器では気体も含めた全体の質量が保存されるため変化しない。

Q5-2: 鉄製の道具にさびが生じるのを防ぐ方法を1つあげ、その方法がさびを防げる理由を酸化の観点から説明せよ。
A5-2: 表面に塗装（ペンキ）をする。塗装によって鉄と空気中の酸素が直接触れなくなり、酸化が進まなくなるため。

Q5-3: 水素と炭素では、どちらが酸素と結びつきやすいか判断する実験を考えたい。酸化銅を水素で加熱する実験で銅が得られた場合、水素と銅の酸素との結びつきやすさについて何がいえるか。
A5-3: 水素は銅よりも酸素との結びつきやすさが大きいといえる。水素が酸化銅から酸素を奪って銅を還元できたことから判断できる。

Q5-4: ある生徒が「酸化銅を加熱するだけで銅を取り出せるのでは」と考えた。この考えが正しくない理由を説明せよ。
A5-4: 酸化銅を加熱するだけでは、酸素を奪い取る物質がないため還元は起こらない。酸素と結びつきやすい物質（炭素など）を混ぜる必要がある。
