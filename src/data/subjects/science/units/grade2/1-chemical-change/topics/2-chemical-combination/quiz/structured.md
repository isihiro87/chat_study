# 物質どうしの化学変化 構造化問題

## 大問1: 化合の基本概念

Q1-1: 2種類以上の物質が結びついて、新しい1種類の物質ができる化学変化のことを何というか。
A1-1: 化合

Q1-2: 化合によってできた新しい物質を何というか。
A1-2: 化合物

Q1-3: 水素と酸素が化合すると水ができる。水はもとの水素や酸素と同じ性質をもつか、答えなさい。
A1-3: もたない。化合でできた物質は、もとの物質とはまったく異なる性質をもつ。

Q1-4: 次の物質のうち、化合物をすべて選べ。（ア）水素 （イ）水 （ウ）酸素 （エ）硫化鉄 （オ）鉄
A1-4: （イ）水、（エ）硫化鉄

Q1-5: 次の物質のうち、単体をすべて選べ。（ア）銅 （イ）酸化銅 （ウ）硫黄 （エ）二酸化炭素 （オ）炭素
A1-5: （ア）銅、（ウ）硫黄、（オ）炭素

## 大問2: 鉄と硫黄の化合実験

次の実験について、あとの問いに答えなさい。

【実験】鉄粉7gと硫黄の粉末4gをよく混ぜ合わせ、試験管に入れた。試験管の上部をガスバーナーで加熱したところ、混合物が赤く光り始めたので加熱をやめた。反応後、試験管の中の物質を調べた。

Q2-1: 混合物を加熱するとき、試験管の上部を加熱するのはなぜか。理由を答えなさい。
A2-1: 反応で発生する気体が上から抜けやすくし、試験管が割れるのを防ぐため。

Q2-2: 赤く光り始めたあと加熱をやめても反応が続いた。その理由を簡潔に説明せよ。
A2-2: 鉄と硫黄の化合は熱を発生する反応であり、その熱で隣の部分の反応が次々と進むから。

Q2-3: 反応後にできた物質の名前と化学式を答えよ。
A2-3: 硫化鉄（FeS）

Q2-4: 反応後の物質に磁石を近づけたらどうなるか。また、加熱前の混合物に磁石を近づけたらどうなるか。それぞれ答えよ。
A2-4: 反応後の物質（硫化鉄）は磁石に引きつけられない。加熱前の混合物は鉄粉が含まれるため、磁石に引きつけられる。

Q2-5: 反応後の物質にうすい塩酸を加えたとき、発生する気体の名前とにおいの特徴を答えよ。
A2-5: 硫化水素。腐卵臭（腐った卵のようなにおい）がする。

Q2-6: 鉄にうすい塩酸を加えたときに発生する気体は何か。
A2-6: 水素

Q2-7: この実験の化学反応式を書け。
A2-7: Fe + S → FeS

## 大問3: 水素の燃焼と化学反応式

水素と酸素の混合気体に点火する実験を行った。次の問いに答えなさい。

Q3-1: 水素が燃えたとき、何色の炎が見えるか。
A3-1: 淡い青色

Q3-2: 容器の内側に見られた変化は何か。
A3-2: 水滴がついた。

Q3-3: この反応でできた物質は何か。化学式も書け。
A3-3: 水（H₂O）

Q3-4: この反応の化学反応式を書け。
A3-4: 2H₂ + O₂ → 2H₂O

Q3-5: 化学反応式 2H₂ + O₂ → 2H₂O において、左辺と右辺の水素原子の数がそれぞれ何個か確認し、等しいことを示せ。
A3-5: 左辺：2H₂なので水素原子は2×2＝4個。右辺：2H₂Oなので水素原子は2×2＝4個。左辺と右辺で等しい。

Q3-6: 化学反応式で化学式の前につける数字を何というか。
A3-6: 係数

## 大問4: さまざまな化合の化学反応式

次の化学変化を化学反応式で表しなさい。

Q4-1: 銅と硫黄が化合して硫化銅ができる反応。
A4-1: Cu + S → CuS

Q4-2: 炭素が酸素中で燃焼して二酸化炭素ができる反応。
A4-2: C + O₂ → CO₂

Q4-3: 銅を空気中で加熱して酸化銅ができる反応。
A4-3: 2Cu + O₂ → 2CuO

Q4-4: 酸化銀を加熱すると分解して銀と酸素ができる反応。
A4-4: 2Ag₂O → 4Ag + O₂

Q4-5: 炭酸水素ナトリウムを加熱すると炭酸ナトリウムと二酸化炭素と水ができる反応。
A4-5: 2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O

Q4-6: Q4-3の式で、銅の係数が2である理由を、原子の数に注目して説明せよ。
A4-6: 酸素分子O₂には酸素原子が2個ある。CuOの酸素原子は1個なので、右辺でも酸素原子を2個にするには2CuOが必要。銅原子も左辺と右辺で合わせるため、左辺も2Cuとなる。

## 大問5: 物質の分類と反応の整理

Q5-1: 次の（ア）～（オ）の変化のうち、化合にあたるものをすべて選べ。
（ア）炭酸水素ナトリウムを加熱して分解する
（イ）鉄と硫黄を加熱して硫化鉄をつくる
（ウ）酸化銀を加熱して銀と酸素に分ける
（エ）水素を燃やして水をつくる
（オ）炭素を酸素中で燃やして二酸化炭素をつくる
A5-1: （イ）（エ）（オ）

Q5-2: 次の表の空欄をうめよ。

| 物質 | うすい塩酸を加えたとき |
|------|----------------------|
| 鉄 | （ア）が発生する |
| 硫化鉄 | （イ）が発生する |

A5-2: （ア）水素、（イ）硫化水素

Q5-3: 硫化鉄と鉄を見た目で区別することは容易か。それぞれの色を答えたうえで、見た目以外の区別方法を1つ挙げよ。
A5-3: どちらも黒色（鉄は灰色がかった黒色）なので見た目では区別しにくい。区別方法の例：磁石を近づける（鉄は引きつけられるが、硫化鉄は引きつけられない）。

Q5-4: 化合と分解の違いを、「物質の数」という語句を使って説明せよ。
A5-4: 化合は2種類以上の物質が結びついて1種類の新しい物質ができる変化であるのに対し、分解は1種類の物質が2種類以上の別の物質に分かれる変化である。

Q5-5: 化学反応式を書くとき、もっとも注意すべきルールは何か。
A5-5: 左辺と右辺で、各原子の種類と数が等しくなるように係数をつけること。
