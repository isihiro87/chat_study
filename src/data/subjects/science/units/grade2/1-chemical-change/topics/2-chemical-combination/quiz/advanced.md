# 物質どうしの化学変化 発展問題

## 問1: 鉄と硫黄の化合実験の分析

ある生徒が、鉄粉7gと硫黄の粉末4gをよく混ぜ合わせて試験管に入れ、混合物の上部をガスバーナーで加熱した。混合物の一部が赤くなったところで加熱をやめたが、反応はそのまま続き、全体が黒色の物質Pに変わった。

(1) 物質Pの名前と化学式を答えなさい。
→ 硫化鉄（FeS）

(2) 加熱をやめても反応が続いたのはなぜか。理由を説明しなさい。
→ 鉄と硫黄の化合は発熱反応であり、反応自体が熱を出すため、その熱で次々と反応が広がっていくから。

(3) 物質Pにうすい塩酸を加えたとき発生する気体の名前と、その気体の特徴を答えなさい。
→ 硫化水素。卵の腐ったような臭い（腐卵臭）がする。

(4) 反応前の混合物にうすい塩酸を加えた場合と、物質Pにうすい塩酸を加えた場合の違いを2つ説明しなさい。
→ 混合物に塩酸：鉄が反応して水素（無臭）が発生する。硫黄は反応しない。物質Pに塩酸：硫化鉄が反応して硫化水素（腐卵臭）が発生する。また、混合物中の鉄は磁石につくが、物質P（硫化鉄）は磁石につかない。

## 問2: 化学反応式の係数合わせ

次の化学変化を化学反応式で表しなさい。

(1) マグネシウムが燃焼して酸化マグネシウムができる反応
→ 2Mg + O₂ → 2MgO（左辺：Mg=2, O=2、右辺：Mg=2, O=2）

(2) 酸化銀を加熱して分解する反応
→ 2Ag₂O → 4Ag + O₂（左辺：Ag=4, O=2、右辺：Ag=4, O=2）

(3) 炭酸水素ナトリウムを加熱して分解する反応
→ 2NaHCO₃ → Na₂CO₃ + CO₂ + H₂O（左辺：Na=2, H=2, C=2, O=6、右辺：Na=2, H=2, C=2, O=6）

(4) 次の化学反応式の（ ）に入る数字を答えなさい。
（ ）Cu + O₂ → （ ）CuO
→ 2Cu + O₂ → 2CuO

## 問3: 複数の化学反応の比較

次のA〜Eの化学変化について、あとの問いに答えなさい。

A: 鉄 + 硫黄 → 硫化鉄
B: 水素 + 酸素 → 水
C: 炭酸水素ナトリウム → 炭酸ナトリウム + 二酸化炭素 + 水
D: 水 → 水素 + 酸素
E: 酸化銀 → 銀 + 酸素

(1) A〜Eのうち、化合にあたるものをすべて選びなさい。
→ A, B（2種類以上の物質が結びついて別の物質ができる化学変化）

(2) A〜Eのうち、分解にあたるものをすべて選びなさい。
→ C, D, E（1種類の物質が2種類以上の物質に分かれる化学変化）

(3) BとDの関係を説明しなさい。
→ BはDの逆反応（逆の化学変化）である。Bは水素と酸素が化合して水ができる反応、Dは水が分解して水素と酸素になる反応。

## 問4: 化学反応式の誤り訂正

次の化学反応式にはそれぞれ誤りがある。正しく書き直しなさい。

(1) H₂ + O₂ → H₂O
→ 誤り：左辺O=2、右辺O=1で原子の数が合わない。正しくは 2H₂ + O₂ → 2H₂O

(2) Mg + O₂ → MgO
→ 誤り：左辺O=2、右辺O=1で合わない。正しくは 2Mg + O₂ → 2MgO

(3) Ag₂O → Ag + O₂
→ 誤り：左辺Ag=2, O=1、右辺Ag=1, O=2で合わない。正しくは 2Ag₂O → 4Ag + O₂

(4) 化学反応式で原子の数を合わせるとき、化学式の中の小さい数字（右下の数字）を変えてはいけない理由を説明しなさい。
→ 化学式の右下の数字は物質そのものの構成を表しているため、数字を変えると別の物質になってしまう。係数（化学式の前の大きい数字）だけを変えて原子の数を合わせる。

## 問5: 未知の物質の推定

ある金属Xの粉末と硫黄の粉末を混ぜて加熱したところ、黒色の化合物Yが生成した。化合物Yについて次のことがわかった。

・磁石に引きつけられない
・うすい塩酸を加えると、腐卵臭のある気体が発生した
・金属Xは銀白色で、磁石に引きつけられた

(1) 金属Xは何か。
→ 鉄（Fe）

(2) 化合物Yの名前と化学式を答えなさい。
→ 硫化鉄（FeS）

(3) この反応の化学反応式を書きなさい。
→ Fe + S → FeS

(4) 金属Xの代わりに銅の粉末と硫黄の粉末を加熱した場合、生成する化合物の名前と化学式を答えなさい。
→ 硫化銅（CuS）
