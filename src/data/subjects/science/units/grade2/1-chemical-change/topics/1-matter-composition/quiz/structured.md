# 物質のなり立ち 大問小問

## 大問1: 炭酸水素ナトリウムの熱分解（配点 20点）

ある白い粉末Aを試験管に入れて加熱したところ、気体Bが発生し、試験管の口付近に液体Cがたまり、試験管の底には白い固体Dが残った。以下の問いに答えなさい。

(1) 粉末Aの物質名を答えなさい。
→ 炭酸水素ナトリウム

(2) 発生した気体Bの名称を答えなさい。
→ 二酸化炭素

(3) 気体Bを石灰水に通すと、石灰水はどのように変化するか。
→ 白くにごる

(4) 液体Cの名称を答えなさい。また、それを確認するために用いる試験紙の名前と色の変化を答えなさい。
→ 水。塩化コバルト紙を使い、青色から赤色（桃色）に変わる。

(5) 固体Dの物質名を答えなさい。
→ 炭酸ナトリウム

(6) 粉末Aと固体Dの水溶液にそれぞれフェノールフタレイン溶液を加えたとき、より濃い赤色を示すのはどちらか。理由もあわせて答えなさい。
→ 固体D（炭酸ナトリウム）。炭酸ナトリウムのほうが炭酸水素ナトリウムよりもアルカリ性が強いため。

(7) この実験で試験管の口をやや下向きにする理由を説明しなさい。
→ 加熱で生じた水が加熱部分に流れて、試験管が急に冷やされて割れるのを防ぐため。

## 大問2: 酸化銀の分解（配点 16点）

黒い粉末Xを試験管に入れて十分に加熱したところ、気体Yが発生し、白い固体Zが残った。以下の問いに答えなさい。

(1) 粉末Xの物質名を答えなさい。
→ 酸化銀

(2) 気体Yの名称を答えなさい。
→ 酸素

(3) 気体Yが発生していることを確認する方法を1つ答えなさい。
→ 火のついた線香を近づけると、激しく燃える。

(4) 固体Zの物質名を答えなさい。
→ 銀

(5) 固体Zが金属であることを確認する方法を2つ答えなさい。
→ みがくと金属光沢が出る。電気を通す。（他に「たたくと薄くのびる」でも可）

(6) この化学変化のように、1種類の物質が2種類以上の物質に分かれる変化を何というか。
→ 分解

## 大問3: 水の電気分解（配点 24点）

水に少量の物質Pを溶かし、電圧をかけて電気分解を行ったところ、両方の電極から気体が発生した。以下の問いに答えなさい。

(1) 物質Pとして用いられる代表的な物質の名称を答えなさい。
→ 水酸化ナトリウム

(2) 物質Pを水に溶かす理由を答えなさい。
→ 純粋な水は電流をほとんど通さないため、電流を流れやすくするため。

(3) 陰極（−極）側で発生した気体の名称を答えなさい。
→ 水素

(4) 陽極（＋極）側で発生した気体の名称を答えなさい。
→ 酸素

(5) 陰極側で発生した気体にマッチの火を近づけるとどうなるか。
→ 音を立てて燃える（ポンと音がする）。

(6) 陰極側と陽極側で発生した気体の体積の比を、最も簡単な整数比で答えなさい。
→ 水素：酸素＝2：1

(7) この実験から、水は何という元素からできていることがわかるか。すべて答えなさい。
→ 水素と酸素

## 大問4: 原子と元素記号（配点 20点）

(1) すべての物質をつくっている最も小さな粒子を何というか。
→ 原子

(2) 原子について正しいものをすべて選びなさい。
ア：化学変化によって、別の種類の原子に変わることがある。
イ：化学変化によって、原子はそれ以上分割できない。
ウ：化学変化によって、新しく原子がつくられることがある。
エ：化学変化の前後で、原子の組み合わせは変わることがある。
→ イ、エ

(3) 原子の種類のことを何というか。
→ 元素

(4) 現在確認されている元素はおよそ何種類あるか。
→ 約118種類

(5) 次の元素記号が表す元素名をそれぞれ答えなさい。
① H　② O　③ C　④ Na　⑤ Ag
→ ①水素　②酸素　③炭素　④ナトリウム　⑤銀

(6) 元素記号を書くとき、2文字目のアルファベットはどのように書くか。
→ 小文字で書く

## 大問5: 分子・化学式・物質の分類（配点 20点）

(1) いくつかの原子が結びついてできた粒子を何というか。
→ 分子

(2) 次の物質の化学式を書きなさい。
① 水素　② 酸素　③ 水　④ 二酸化炭素
→ ①H₂　②O₂　③H₂O　④CO₂

(3) 1種類の元素だけからできている物質を何というか。
→ 単体

(4) 2種類以上の元素からできている物質を何というか。
→ 化合物

(5) 次の物質を「単体」と「化合物」に分類しなさい。
水素、水、酸素、二酸化炭素、銀、酸化銀
→ 単体：水素、酸素、銀　化合物：水、二酸化炭素、酸化銀

(6) 1種類の物質だけでできているものを純粋な物質という。では、2種類以上の物質が混じったものを何というか。
→ 混合物

(7) 次のうち混合物であるものをすべて選びなさい。
ア：蒸留水　イ：空気　ウ：食塩水　エ：酸素　オ：海水
→ イ、ウ、オ
