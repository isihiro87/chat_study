# 大気の動き 発展問題

## 発展1: 海陸風の実験モデル（小問4）

バットに砂と水を入れ、その間に火のついた線香を立て、砂と水を電灯の光で同じように温めた。線香のけむりの動きを観察した。

(1) この実験で、砂と水はそれぞれ何のモデルか。
→ 砂は陸（大陸）、水は海（海洋）のモデル。

(2) 電灯で温めると、線香のけむりはどちら向きに動いたか。理由もあわせて答えなさい。
→ けむりは水（海）の方から砂（陸）の方へ動いた。砂は水より温まりやすいため、砂の上の空気が暖められて上昇気流が生じ、気圧が低くなる。そのため水の方から砂の方へ空気が流れる。

(3) この実験結果は、昼に吹く海風と陸風のどちらを再現しているか。
→ 海風を再現している。

(4) この実験で、電灯を消して砂と水を冷やすと、けむりの動く向きはどうなるか。理由もあわせて説明しなさい。
→ けむりは砂（陸）の方から水（海）の方へ動くようになる。砂は水より冷めやすいため、砂の上の空気が冷えて下降気流が生じ、水の上の空気のほうが相対的に温かく上昇気流が生じるため、砂から水へ空気が流れる（陸風の再現）。

## 発展2: 季節風と日本の気候（小問4）

(1) 冬のシベリア高気圧から吹き出す季節風は、最初は乾燥している。この風が日本海を渡るあいだにどのような変化が起こるか。「暖流」「水蒸気」という語を使って説明しなさい。
→ 日本海には暖流（対馬海流）が流れており、冷たく乾いた季節風が暖流の上を通過するとき、暖流から大量の水蒸気を吸収して湿った空気になる。

(2) (1)で湿った空気が山脈にぶつかった後、太平洋側ではどのような天気になるか。その理由とあわせて説明しなさい。
→ 太平洋側は乾燥した晴天になる。湿った空気が山脈にぶつかって日本海側で雪や雨を降らせた後、山脈を越えた空気は水分を失っているため、太平洋側では乾燥した晴れの天気になる。

(3) 冬の天気図で等圧線が縦じまに密に並ぶことと、風の強さの関係を説明しなさい。
→ 等圧線が密に並んでいるということは、短い距離で気圧差が大きいことを意味し、気圧差が大きいほど風が強くなる。そのため、冬の西高東低の気圧配置では北西の強い季節風が吹く。

(4) 大陸と海洋の温まりやすさの違いが、季節風と海陸風の両方に共通する原因であることを説明しなさい。
→ 大陸（陸）は温まりやすく冷めやすい、海洋（海）は温まりにくく冷めにくいという性質がある。この温度差が気圧差を生み、高気圧から低気圧へ風が吹く。季節風は季節単位、海陸風は1日単位で同じ原理が働いている。

## 発展3: 気団と天気の関係（小問3）

(1) シベリア気団、小笠原気団、オホーツク海気団のそれぞれの性質を答えなさい。
→ シベリア気団：寒冷・乾燥。小笠原気団：温暖・湿潤。オホーツク海気団：寒冷・湿潤。

(2) 小笠原気団とオホーツク海気団がぶつかるとどのような現象が起こるか。
→ 両気団の境界に停滞前線（梅雨前線）ができ、日本付近に長雨をもたらす。

(3) 気団の性質が「乾燥」か「湿潤」かは何によって決まるか説明しなさい。
→ 気団が発達する場所による。大陸上で発達する気団は乾燥し、海洋上で発達する気団は海から水蒸気を吸収するため湿潤になる。
