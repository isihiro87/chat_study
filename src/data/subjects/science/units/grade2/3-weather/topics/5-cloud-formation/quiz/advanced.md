# 雲のでき方と前線 発展問題

## 発展1: 雲のでき方の実験考察（小問4）

フラスコに水と線香のけむりを入れ、ピストンを操作する実験を行った。

(1) ピストンを急に押したとき、フラスコ内の変化を「気圧」「温度」という語を使って説明しなさい。
→ フラスコ内の気圧が上がり、空気が圧縮されて温度が上がる。フラスコ内が透明になる（白い曇りが消える）。

(2) ピストンを引いたときにフラスコ内が白くなり、押したときに透明になる理由を「露点」という語を使って説明しなさい。
→ ピストンを引くと空気が膨張して温度が下がり、露点を下回ると水蒸気が凝結して白くなる。押すと空気が圧縮されて温度が上がり、露点より高くなると水滴が蒸発して透明になる。

(3) 線香のけむりを入れなかった場合、ピストンを引いても雲はできにくい。その理由を答えなさい。
→ 水蒸気が凝結するには凝結核（微小な粒子）が必要であり、線香のけむりがないと凝結核が不足して水蒸気が凝結しにくいから。

(4) この実験は、自然界の雲のでき方のどの過程に対応しているか。対応関係を答えなさい。
→ ピストンを引く＝空気の上昇による気圧低下と膨張。フラスコ内の白い曇り＝水蒸気の凝結による雲の形成。線香のけむり＝自然界のちりやほこり（凝結核）。

## 発展2: 総合記述問題（小問3）

(1) 「雲」「露点」「凝結」「膨張」の4語をすべて使って、雲ができるしくみを80字以内で説明しなさい。
→ 水蒸気を含む空気が上昇すると気圧の低下により膨張し、温度が下がる。温度が露点に達すると凝結が起こり、水滴や氷の粒が集まって雲ができる。

(2) 上昇気流が生じる3つのパターンを、それぞれ簡潔に説明しなさい。
→ ①地面が太陽に暖められて、暖かい空気が上昇する。②山の斜面に沿って空気が押し上げられる。③寒気と暖気がぶつかって、密度の小さい暖気が押し上げられる（前線）。

(3) 水の循環について、「太陽エネルギー」「蒸発」「降水」の語を使って説明しなさい。
→ 太陽エネルギーを原動力として海や湖から水が蒸発し、上空で雲になり、降水として地上に戻る。地上の水は河川を通じて海に戻り、再び蒸発する。この繰り返しが水の循環である。
