# 雲のでき方 一問一答

## 雲の正体とでき方

Q1. 上空に浮かぶ小さな水滴や氷の結晶の集まりを何というか。
→ 雲

Q2. 空気が上昇すると体積はどうなるか。
→ 膨張する

Q3. 空気が膨張すると温度はどうなるか。
→ 下がる

Q4. 空気が上昇して温度が下がるのは、上空ほど何が低いからか。
→ 気圧

Q5. 水蒸気が水滴に変わり始める温度を何というか。
→ 露点

Q6. 水蒸気が水滴や氷の粒に変わる現象を何というか。
→ 凝結

Q7. 雲ができるまでの流れを順番に並べよ：膨張→温度低下→（　）→凝結→雲
→ 露点に達する

Q8. 雲のでき方の実験でフラスコに線香のけむりを入れるのはなぜか。
→ けむりの粒子が水蒸気の凝結の核（芯）になるため

Q9. 雲のでき方の実験でピストンを引くとフラスコ内の空気はどうなるか。
→ 膨張して温度が下がる

Q10. 雲のでき方の実験でピストンを引いたとき、フラスコ内はどうなるか。
→ 白くくもる（雲ができる）

## 上昇気流のでき方

Q11. 上昇気流が生じる原因を3つ答えよ。
→ ①地面が暖められて空気が上昇②山の斜面に沿って空気が上昇③寒気と暖気がぶつかって暖気が上昇

Q12. 地面が太陽に暖められて空気が上昇する場合にできやすい雲は何か。
→ 積雲（わた雲）や積乱雲

Q13. 山の斜面に沿って空気が上昇する場合を何というか。
→ 地形性上昇

## 雨や雪と水の循環

Q14. 雲の水滴や氷の粒が大きくなって落下したものを何というか。
→ 雨（または雪）

Q15. 雲の水滴が凍って落下するものを2つ答えよ。
→ 雪、ひょう（あられ）

Q16. 地球上の水が蒸発と降水を繰り返して循環することを何というか。
→ 水の循環

Q17. 水の循環の原動力となるエネルギーは何か。
→ 太陽のエネルギー

Q18. 水の循環を「海→（　）→雲→（　）→河川→海」の順に答えよ。
→ 海→蒸発→雲→降水→河川→海
