# 日本の四季の天気 発展問題

## 発展1: 天気図からの季節判定（小問4）

次のA〜Dの天気図の特徴を読み取り、それぞれどの季節のものか判定し、理由を述べよ。

A: 等圧線が南北に縦じま状に密に並び、西に1040hPaの高気圧、東に低気圧がある。
→ 季節：冬。理由：西高東低の気圧配置であり、等圧線が縦じまで密に並んでいることから北西の季節風が強く吹く冬の典型的な気圧配置である。

B: 太平洋上に大きな高気圧があり、その縁に沿って1つの低気圧（前線なし）が北上している。
→ 季節：夏。理由：太平洋高気圧が広く張り出しており、前線のない低気圧（台風と考えられる）が高気圧の縁に沿って北上している夏の典型的な気圧配置である。

C: 高気圧と低気圧が東西に交互に並び、偏西風に乗って東へ移動している。
→ 季節：春（または秋）。理由：移動性高気圧と低気圧が交互に並んでおり、天気が周期的に変化する春や秋の気圧配置である。

D: 日本列島に沿って東西に停滞前線がのびており、南北に2つの高気圧がある。
→ 季節：梅雨（6月頃）。理由：南の小笠原気団と北のオホーツク海気団がぶつかって停滞前線（梅雨前線）ができている梅雨の気圧配置である。

## 発展2: 台風の進路と気圧配置（小問3）

(1) 台風が発生する海域の特徴を、「海水温」「水蒸気」という語を使って説明しなさい。
→ 台風は海水温が高い（約27℃以上の）熱帯の海上で発生する。暖かい海面から大量の水蒸気が蒸発し、その水蒸気が凝結するときに放出する熱がエネルギーとなって台風が発達する。

(2) 台風の進路が夏と秋で異なる理由を、「太平洋高気圧」「偏西風」という語を使って説明しなさい。
→ 夏は太平洋高気圧の勢力が強く、台風はその縁に沿って西寄りに進むため日本に近づきにくい。秋は太平洋高気圧が弱まって台風が北上しやすくなり、偏西風に乗って東寄りに向きを変え、日本付近を通過することが多くなる。

(3) 台風が日本付近を北上した後に勢力が弱まる理由を2つ答えなさい。
→ ①北上すると海水温が下がり、水蒸気の供給が減少してエネルギー源が失われるため。②陸上に上陸すると地表との摩擦が大きくなり、風速が弱まるため。

## 発展3: 総合問題（小問3）

あるテレビの天気予報で、気象予報士が以下のように解説した。

「現在、日本の西に高気圧、東に低気圧があります。明日の朝にかけて日本海側を中心に雪が降るでしょう。太平洋側は晴れますが、空気が乾燥するため火の取り扱いにご注意ください。」

(1) この天気予報が放送された季節はいつか。理由とあわせて答えなさい。
→ 冬。西に高気圧、東に低気圧という西高東低の気圧配置であり、日本海側に雪、太平洋側に乾燥した晴れという冬の典型的な天気パターンを述べているため。

(2) 太平洋側が晴れて乾燥する理由を、「季節風」「山脈」という語を使って説明しなさい。
→ 北西の季節風が日本海で水蒸気を吸収して日本海側で雪を降らせた後、山脈を越える。山脈を越えた空気は水分を失っているため、太平洋側では乾燥した晴天になる。

(3) この気象予報士の解説から、日本海側に住む人が注意すべきことを1つ、太平洋側に住む人が注意すべきことを1つ、それぞれ答えなさい。
→ 日本海側：大雪による交通障害や雪崩に注意する。太平洋側：空気の乾燥による火災の発生に注意する。
