# 大気の動き 一問一答

## 偏西風と大気循環

Q1. 中緯度帯の上空を西から東に吹く風を何というか。
→ 偏西風

Q2. 地球規模の大気が循環する原因となるエネルギーは何か。
→ 太陽のエネルギー

Q3. 日本付近の天気が西から東へ移り変わるのは、何という風の影響か。
→ 偏西風

Q4. 赤道付近では、上昇気流と下降気流のどちらが生じやすいか。
→ 上昇気流

Q5. 極付近では、上昇気流と下降気流のどちらが生じやすいか。
→ 下降気流

Q6. 地球の大気の厚さ（気象現象が起こる範囲）はおよそ何kmか。
→ 約10km

Q7. 地球規模の大気循環が生じる根本的な原因は何か。
→ 赤道付近と極付近の温度差

## 季節風

Q8. 季節によって異なる方向から吹く風を何というか。
→ 季節風（モンスーン）

Q9. 大陸と海洋では、どちらが温まりやすく冷めやすいか。
→ 大陸

Q10. 冬に日本付近で吹く季節風の向きは何か。
→ 北西

Q11. 夏に日本付近で吹く季節風の向きは何か。
→ 南東

Q12. 冬に大陸に高気圧が発達する理由を簡潔に答えよ。
→ 大陸が冷えて空気が冷却され、下降気流が生じるから

Q13. 夏に大陸に低気圧が発達する理由を簡潔に答えよ。
→ 大陸が温まって上昇気流が生じるから

## 海陸風

Q14. 昼間に海から陸に向かって吹く風を何というか。
→ 海風

Q15. 夜間に陸から海に向かって吹く風を何というか。
→ 陸風

Q16. 海風が吹く理由を気圧の違いで説明せよ。
→ 昼は陸が温まり陸上の気圧が低くなり、海上の気圧が高くなるため、海から陸へ風が吹く

Q17. 陸風が吹く理由を気圧の違いで説明せよ。
→ 夜は陸が冷えて陸上の気圧が高くなり、海上の気圧が低くなるため、陸から海へ風が吹く

Q18. 海陸風が生じる原理は、季節風と同じか異なるか。
→ 同じ原理（大陸と海洋の温まりやすさの違い）

## 日本付近の気団

Q19. 冬に大陸上で発達する、寒冷で乾燥した気団の名前は何か。
→ シベリア気団

Q20. 夏に太平洋上で発達する、温暖で湿潤な気団の名前は何か。
→ 小笠原気団

Q21. 梅雨の時期に北からの冷たく湿った空気をもたらす気団は何か。
→ オホーツク海気団

Q22. シベリア気団の性質を2つ答えよ。
→ 寒冷・乾燥

Q23. 小笠原気団の性質を2つ答えよ。
→ 温暖・湿潤

Q24. オホーツク海気団の性質を2つ答えよ。
→ 寒冷・湿潤
