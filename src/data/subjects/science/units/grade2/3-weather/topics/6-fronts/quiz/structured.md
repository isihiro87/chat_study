# 気団と前線 大問小問

## 大問1: 前線の断面図（配点 22点）

右の図は、ある前線の断面を模式的に表したものである。冷たい空気Aが暖かい空気Bの下にもぐり込んでいる。

(1) 暖かい空気と冷たい空気の境界面を何というか。（2点）
→ 前線面

(2) この前線の名称を答えなさい。（2点）
→ 寒冷前線

(3) この前線付近で発達する雲の種類を答えなさい。（3点）
→ 積乱雲

(4) この前線付近の雨の降り方の特徴を答えなさい。（3点）
→ せまい範囲で短時間に激しい雨が降る。

(5) この前線が通過した後の気温と風向きの変化を答えなさい。（4点）
→ 気温が急激に低下し、風向きが北寄りに変わる。

(6) 暖気の密度が寒気より小さい理由を「温度」という語を使って簡単に説明しなさい。（4点）
→ 温度が高い空気は膨張して体積が大きくなるため、単位体積あたりの質量（密度）が小さくなる。

(7) この前線を天気図で表す記号の形を簡単に述べなさい。（4点）
→ 前線の進む方向に三角形がついた線で表す。

## 大問2: 温暖前線との比較（配点 20点）

寒冷前線と温暖前線について、次の表を完成させなさい。

| | 寒冷前線 | 温暖前線 |
|---|---|---|
| (1) 空気の動き | 寒気が暖気の下にもぐり込む | （　） |
| (2) 雲の種類 | （　） | 乱層雲（高層雲） |
| (3) 雨の降り方 | 短時間の激しい雨 | （　） |
| (4) 通過後の気温 | （　） | 気温が上昇 |
| (5) 通過後の風向 | 北寄りに変わる | （　） |

→ (1) 暖気が寒気の上をゆるやかに這い上がる（各4点）
→ (2) 積乱雲
→ (3) 広い範囲で弱い長雨
→ (4) 急激に低下
→ (5) 南寄りに変わる

## 大問3: 温帯低気圧の天気図（配点 20点）

右の図は日本付近の天気図で、低気圧にPとQの2つの前線が伴っている。Pは低気圧の南東側、Qは南西側にのびている。

(1) 低気圧Pは何前線か。（2点）
→ 温暖前線

(2) 低気圧Qは何前線か。（2点）
→ 寒冷前線

(3) このような低気圧を何というか。（2点）
→ 温帯低気圧

(4) この後、Qの前線がPの前線に追いつくと何前線ができるか。（3点）
→ 閉そく前線

(5) 閉そく前線ができた後、この低気圧の勢力はどうなるか。（3点）
→ 衰退する（弱まる）。

(6) 前線Pの前方（東側）の地点では、今後どのような天気の変化が予想されるか。「雲」「雨」「気温」という語を使って説明しなさい。（8点）
→ まず温暖前線の接近に伴い高層雲や乱層雲が広がり、弱い雨が長時間降る。温暖前線が通過すると気温が上昇し雨がやむ。その後、寒冷前線が近づくと積乱雲が発達し短時間の強い雨が降る。寒冷前線通過後は気温が急低下し北寄りの風に変わる。

## 大問4: 停滞前線（配点 18点）

(1) 停滞前線とはどのような前線か。「暖気」「寒気」「勢力」という語を使って説明しなさい。（6点）
→ 暖気と寒気がほぼ同じ勢力でぶつかり合い、前線がほとんど動かずに同じ位置に長くとどまる前線。梅雨前線や秋雨前線がその例。

(2) 停滞前線の天気図での記号の特徴を述べなさい。（4点）
→ 三角形と半円が交互に前線の両側についた線で表す。

(3) 閉そく前線ができるしくみを説明しなさい。（4点）
→ 寒冷前線の移動速度が温暖前線より速いため、寒冷前線が温暖前線に追いつくことで閉そく前線ができる。

(4) 閉そく前線ができると温帯低気圧が衰退する理由を説明しなさい。（4点）
→ 暖気が地表から切り離されて上空に押し上げられ、暖気の上昇によるエネルギー供給が減少するため。
