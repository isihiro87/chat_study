# 気団と前線 一問一答

## 気団

Q1. 気温や湿度がほぼ一様な大きな空気の塊を何というか。
→ 気団

Q2. 暖かい空気と冷たい空気では、密度が小さいのはどちらか。
→ 暖かい空気（暖気）

Q3. 異なる気団がぶつかったとき、暖気はどうなるか。
→ 密度が小さいため寒気の上に乗り上げる

## 前線面と前線

Q4. 異なる性質をもつ気団の境界面を何というか。
→ 前線面

Q5. 前線面と地表面が交わる線を何というか。
→ 前線

Q6. 前線付近で天気が変わりやすい理由を簡潔に答えよ。
→ 暖気が上昇して雲ができやすいから

## 寒冷前線

Q7. 寒気が暖気の下にもぐり込む前線を何というか。
→ 寒冷前線

Q8. 寒冷前線付近で発達する雲の種類は何か。
→ 積乱雲

Q9. 寒冷前線付近の雨の降り方を答えよ。
→ 短時間に激しい雨（にわか雨）

Q10. 寒冷前線が通過した後の風向きの変化を答えよ。
→ 北寄りの風に変わる

Q11. 寒冷前線が通過した後の気温の変化を答えよ。
→ 急激に低下する

Q12. 寒冷前線の天気図での記号はどのような形か。
→ 三角形が前線の進む方向についた線

## 温暖前線

Q13. 暖気が寒気の上をゆるやかに這い上がる前線を何というか。
→ 温暖前線

Q14. 温暖前線付近で発達する雲の種類は何か。
→ 乱層雲（高層雲）

Q15. 温暖前線付近の雨の降り方を答えよ。
→ 広い範囲で弱い長雨

Q16. 温暖前線が通過した後の風向きの変化を答えよ。
→ 南寄りの風に変わる

Q17. 温暖前線が通過した後の気温の変化を答えよ。
→ 気温が上昇する

Q18. 温暖前線の天気図での記号はどのような形か。
→ 半円が前線の進む方向についた線

## 停滞前線と閉そく前線

Q19. 暖気と寒気がほぼ同じ勢力で同じ位置に停滞する前線を何というか。
→ 停滞前線

Q20. 停滞前線の天気図での記号はどのような形か。
→ 三角形と半円が交互に並んだ線

Q21. 寒冷前線が温暖前線に追いつくとできる前線を何というか。
→ 閉そく前線

Q22. 閉そく前線ができると温帯低気圧はどうなるか。
→ 衰退する（弱まる）

## 温帯低気圧

Q23. 中緯度帯で発生し、前線を伴う低気圧を何というか。
→ 温帯低気圧

Q24. 温帯低気圧の南東側にできる前線は何か。
→ 温暖前線

Q25. 温帯低気圧の南西側にできる前線は何か。
→ 寒冷前線

Q26. 温帯低気圧はおおよそどの方向に移動するか。
→ 西から東

Q27. 温帯低気圧の寒冷前線が温暖前線に追いつくと何前線ができるか。
→ 閉そく前線
