# 大気圧と天気図 大問小問

## 大問1: 大気圧の実験（配点 20点）

次の3つの実験について答えなさい。

【実験A】なめらかなガラス板の上にゴム製の吸盤を強く押しつけたところ、吸盤を引きはがすのに大きな力が必要だった。
【実験B】コップに水を満たしてボール紙でふたをし、逆さにしたところ、水はこぼれなかった。
【実験C】密封した袋入りの菓子を山の頂上に持っていったところ、袋がパンパンにふくらんだ。

(1) 実験Aで吸盤が引きはがしにくい理由を、「大気圧」という語を使って説明しなさい。（5点）
→ 吸盤とガラス板の間の空気を押し出すと、吸盤の上側から大気圧がはたらいて吸盤を押しつけるため、引きはがしにくくなる。

(2) 実験Bで水がこぼれない理由を説明しなさい。（5点）
→ ボール紙の下側から大気圧が上向きに押すため、水の重さを支えてこぼれない。

(3) 実験Cで袋がふくらむ理由を、「気圧」「標高」という語を使って説明しなさい。（5点）
→ 標高が高い場所では外の気圧が低くなるため、袋の内側の空気の圧力が外の気圧を上回り、袋がふくらむ。

(4) 実験A〜Cに共通して関係している力は何か。漢字3字で答えなさい。（2点）
→ 大気圧。

(5) 大気圧がはたらく向きの特徴を答えなさい。（3点）
→ あらゆる向き（上下左右すべての方向）にはたらく。

## 大問2: 天気図の読み取り（配点 20点）

ある天気図にA地点とB地点が記されている。A地点付近では等圧線が閉じた線で囲まれ中心の気圧が1024hPaで周囲より高く、B地点付近では中心の気圧が992hPaで周囲より低い。

(1) A地点付近のような場所を何というか。（3点）
→ 高気圧。

(2) B地点付近のような場所を何というか。（3点）
→ 低気圧。

(3) A地点付近の地表では、風はどのように吹くか。「時計回り」「反時計回り」のどちらかを使って答えなさい（北半球の場合）。（4点）
→ 中心から外に向かって時計回りに吹き出す。

(4) B地点付近の上空では上昇気流と下降気流のどちらが発生するか。それにより天気はどうなりやすいか答えなさい。（5点）
→ 上昇気流が発生する。上昇気流によって雲ができやすく、くもりや雨になりやすい。

(5) 天気図で等圧線は何hPaごとに引かれるか。また基準となる気圧は何hPaか。（5点）
→ 4hPaごとに引かれる。基準は1000hPa。

## 大問3: 風向・風力と天気図記号（配点 18点）

天気図に記された観測地点の記号について答えなさい。ある地点では、棒が北東方向を指しており、矢羽根が3本ついていた。天気記号は◎であった。

(1) 風向とは何か。簡潔に答えなさい。（3点）
→ 風が吹いてくる方向。

(2) 風向は何方位で表すか。（2点）
→ 16方位。

(3) この地点の風向は何か。（3点）
→ 北東。棒が伸びている方向が風の吹いてくる方向を表す。

(4) この地点の風力はいくつか。（3点）
→ 風力3。矢羽根の数が風力を表す。

(5) 天気記号◎が表す天気は何か。（3点）
→ 晴れ。

(6) 等圧線の間隔がせまい地域では、風の強さはどうなるか。理由もあわせて答えなさい。（4点）
→ 風が強くなる。等圧線の間隔がせまいほど短い距離で気圧の差が大きく、気圧差による力が大きくなるため。
