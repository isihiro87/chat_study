# 日本の四季の天気 一問一答

## 冬の天気

Q1. 冬の日本の典型的な気圧配置を何というか。
→ 西高東低

Q2. 冬に西に位置する高気圧の名前は何か。
→ シベリア高気圧

Q3. 冬の日本海側の天気の特徴を答えよ。
→ 雪や雨の日が多い（大雪）

Q4. 冬の太平洋側の天気の特徴を答えよ。
→ 乾燥した晴天

Q5. 冬の日本海側に大雪が降るしくみを簡潔に答えよ。
→ 北西の季節風が日本海の暖流から水蒸気を吸収し、山脈にぶつかって上昇→雪雲が発達→大雪

## 夏の天気

Q6. 夏に日本を覆う高気圧の名前は何か。
→ 太平洋高気圧（小笠原高気圧）

Q7. 夏の日本の天気の特徴を2つ答えよ。
→ 高温・多湿

Q8. 夏に吹く南東の季節風が湿っている理由は何か。
→ 太平洋上を通過するとき、海から水蒸気を吸収するから

## 春と秋の天気

Q9. 春と秋に日本付近を交互に通過するものは何か。
→ 移動性高気圧と低気圧

Q10. 春と秋の天気が周期的に変化する理由を答えよ。
→ 偏西風に乗って移動性高気圧と低気圧が交互に通過するから

Q11. 移動性高気圧が通過するとき天気はどうなるか。
→ 晴れ

## つゆ（梅雨）と秋雨

Q12. 梅雨前線はどのような前線か。
→ 停滞前線

Q13. 梅雨前線ができる原因となる2つの気団を答えよ。
→ 小笠原気団（南）とオホーツク海気団（北）

Q14. 夏の終わりに日本付近に停滞する前線を何というか。
→ 秋雨前線

Q15. 秋雨前線ができるしくみは梅雨前線と同じか。
→ 同じ（停滞前線）

## 台風

Q16. 台風とはどのような気象現象か、定義を答えよ。
→ 熱帯低気圧が発達し、最大風速が17m/s以上になったもの

Q17. 台風が発達するエネルギー源は何か。
→ 暖かい海上から蒸発する大量の水蒸気

Q18. 台風が日本付近で東へ向きを変えるのは何の影響か。
→ 偏西風

Q19. 台風の進路は、春から夏と夏から秋でどう異なるか。
→ 春から夏は太平洋高気圧の縁を西へ進み、夏から秋は太平洋高気圧が弱まり日本方面へ向かう

## 天気の予測と気象災害

Q20. コンピュータで大気の状態を計算して天気を予報する方法を何というか。
→ 数値予報

Q21. 災害の危険がある地域を示した地図を何というか。
→ ハザードマップ

Q22. 重大な災害が予想されるときに発表される警報を何というか。
→ 特別警報

Q23. 気象現象がもたらすめぐみを2つ答えよ。
→ 豊富な降水量による農業用水・生活用水の確保、水力発電など

Q24. 気象災害の例を3つ答えよ。
→ 大雨による河川の氾濫、台風による暴風被害、土砂崩れ（がけ崩れ）
