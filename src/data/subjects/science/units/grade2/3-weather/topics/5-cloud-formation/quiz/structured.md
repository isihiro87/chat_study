# 雲のでき方 大問小問

## 大問1: 雲のでき方の実験（配点 20点）

丸底フラスコに少量の水と線香のけむりを入れ、ゴム栓をした。注射器のピストンを急に引いたところ、フラスコ内が白くくもった。

(1) フラスコに線香のけむりを入れた理由を答えなさい。（4点）
→ けむりの微粒子が水蒸気の凝結の核になるため。自然界ではちりやほこりがこの役割をしている。

(2) ピストンを急に引くと、フラスコ内の空気にどのような変化が起こるか。「膨張」「温度」という語を使って説明しなさい。（4点）
→ フラスコ内の気圧が下がり、空気が膨張して温度が下がる。

(3) フラスコ内が白くくもったのは、空気中の何が変化したからか。（3点）
→ 水蒸気が凝結して水滴（小さな水の粒）になったから。

(4) 自然界で雲ができるとき、空気が上昇する原因を2つ答えなさい。（4点）
→ ①地面が太陽に暖められて空気が上昇する。②山の斜面に沿って空気が押し上げられる。

(5) 雲ができるまでの流れを、次の語句を使って並べかえなさい。（5点）
　〔露点、膨張、凝結、気圧低下、温度低下〕
→ 空気の上昇→気圧低下→膨張→温度低下→露点に到達→凝結→雲ができる

## 大問2: 水の循環（配点 18点）

地球上の水は、海や湖の水面から蒸発し、雲を作り、降水として地上に戻ることを繰り返している。

(1) この水の循環の原動力となるエネルギーは何か。（2点）
→ 太陽のエネルギー

(2) 海面から蒸発した水蒸気が上空で雲になるまでの過程を、順を追って説明しなさい。（6点）
→ 海面から蒸発した水蒸気を含む空気が上昇する。上空ほど気圧が低いため空気が膨張し、温度が下がる。露点に達すると水蒸気が凝結し、小さな水滴や氷の粒になって雲ができる。

(3) 雲から雨が降るのはどのようなときか。（4点）
→ 雲の中の水滴や氷の粒が成長して大きくなり、上昇気流で支えきれなくなったとき。

(4) 雲のでき方の実験で、ピストンを押し込んだ場合にフラスコ内はどうなるか。理由とあわせて答えなさい。（6点）
→ フラスコ内が透明になる。気圧が上がって空気が圧縮され、温度が上がることで水滴が蒸発するため。
