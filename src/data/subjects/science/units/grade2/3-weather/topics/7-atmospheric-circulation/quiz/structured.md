# 大気の動き 大問小問

## 大問1: 偏西風と大気循環（配点 18点）

地球の大気は太陽エネルギーを受けて循環している。これについて、次の問いに答えなさい。

(1) 赤道付近の大気が上昇する理由を「太陽エネルギー」という語を使って説明しなさい。（4点）
→ 赤道付近は太陽エネルギーを多く受けて気温が高くなるため、暖められた空気が上昇するから。

(2) 中緯度帯の上空を西から東に向かって吹く風の名称を答えなさい。（2点）
→ 偏西風

(3) 日本付近の天気が西から東へ移り変わる理由を、(2)の語を使って説明しなさい。（4点）
→ 偏西風の影響で、低気圧や高気圧が西から東へ移動するため、天気も西から東へ移り変わる。

(4) 地球の大気が循環する原動力となるエネルギーは何か。（2点）
→ 太陽のエネルギー

(5) 地球規模の大気の動きが表面から約何kmの範囲で起きているか。（2点）
→ 約10km

(6) 地球規模の大気循環が生じるのは、赤道と極のどのような違いによるか。（4点）
→ 赤道付近と極付近の温度差による。

## 大問2: 季節風と海陸風（配点 22点）

右の図は、日本付近で夏と冬にふく季節風の向きを模式的に表したものである。これについて、次の問いに答えなさい。

(1) 大陸と海洋では、温まりやすく冷めやすいのはどちらか。（2点）
→ 大陸

(2) 冬に大陸上の気圧は高くなるか低くなるか。また、その理由を答えなさい。（4点）
→ 高くなる。大陸が冷えて空気が冷却され、下降気流が生じて高気圧が発達するため。

(3) 冬の日本付近の季節風の風向きを8方位で答えなさい。（2点）
→ 北西

(4) 夏の日本付近の季節風の風向きを8方位で答えなさい。（2点）
→ 南東

(5) 季節によって風向きが変わる風を何というか。カタカナでも答えなさい。（3点）
→ 季節風（モンスーン）

(6) 昼に海から陸に向かって吹く風と、夜に陸から海に向かって吹く風をそれぞれ何というか。（4点）
→ 昼：海風、夜：陸風

(7) 海陸風と季節風は同じ原理で生じる。その共通する原理を「温まりやすさ」という語を使って簡単に書きなさい。（5点）
→ 大陸（陸）と海洋（海）の温まりやすさの違いにより、温度差が生じて気圧差ができ、高気圧から低気圧に向かって風が吹くという原理。

## 大問3: 日本付近の気団（配点 18点）

(1) 冬に大陸上で発達する気団の名前と性質を答えなさい。（3点）
→ シベリア気団。寒冷・乾燥。

(2) 夏に太平洋上で発達する気団の名前と性質を答えなさい。（3点）
→ 小笠原気団。温暖・湿潤。

(3) 梅雨の時期に影響する北の気団の名前と性質を答えなさい。（3点）
→ オホーツク海気団。寒冷・湿潤。

(4) 気団が「乾燥」するか「湿潤」になるかは、発達する場所のどのような違いによるか。（4点）
→ 大陸上で発達すると乾燥し、海洋上で発達すると海から水蒸気を吸収して湿潤になる。

(5) 梅雨前線ができるしくみを、2つの気団の名前を使って説明しなさい。（5点）
→ 南の小笠原気団（温暖湿潤）と北のオホーツク海気団（寒冷湿潤）がぶつかり、その境界に停滞前線（梅雨前線）ができる。
