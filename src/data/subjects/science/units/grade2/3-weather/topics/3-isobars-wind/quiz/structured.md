# 等圧線と風 大問小問

## 大問1: 高気圧と低気圧のちがい（配点 18点）

次の図は、高気圧と低気圧の断面を模式的に表したものである。

(1) 高気圧の中心付近で生じている気流を何というか。（3点）
→ 下降気流

(2) 低気圧の中心付近で生じている気流を何というか。（3点）
→ 上昇気流

(3) 日本付近で高気圧の地表付近の風は、どのような向きに吹き出すか。（3点）
→ 時計回りにまわりへ吹き出す。

(4) 日本付近で低気圧の地表付近の風は、どのような向きに吹き込むか。（3点）
→ 反時計回りに中心へ吹き込む。

(5) 低気圧の接近で天気がくずれやすい理由を、「上昇気流」「凝結」という語を使って説明しなさい。（6点）
→ 低気圧の中心では上昇気流が発生し、空気が上空に運ばれて冷やされる。気温が下がると飽和水蒸気量が小さくなり、含みきれなくなった水蒸気が凝結して雲ができるため、天気がくずれやすくなる。

## 大問2: 乾湿計と湿度（配点 20点）

乾湿計を使って教室の湿度を測定した。乾球の示す温度は22℃、湿球の示す温度は18℃であった。

(1) 乾湿計の乾球と湿球のちがいを簡潔に説明しなさい。（4点）
→ 乾球はそのまま気温を測るための温度計。湿球はガーゼで球部を包み、水で湿らせた温度計。

(2) 湿球の温度が乾球より低くなる理由を説明しなさい。（4点）
→ 湿球のガーゼに含まれた水が蒸発するとき、まわりから熱（気化熱）を奪うため、温度が下がる。

(3) 乾球と湿球の温度差が大きいとき、空気の湿度は高いか低いか。理由もあわせて答えなさい。（4点）
→ 湿度が低い。空気が乾燥しているほど蒸発がさかんになり、湿球から多くの熱が奪われて温度差が大きくなるため。

(4) もし乾球と湿球の温度がまったく同じになった場合、湿度は何%と考えられるか。理由も答えなさい。（4点）
→ 湿度100%。空気中の水蒸気が飽和しているため蒸発が起こらず、湿球の温度が下がらないから。

(5) 乾湿計の湿度表から、乾球22℃・湿球18℃のときの湿度が67%であったとする。22℃の飽和水蒸気量が19.4g/m³のとき、教室の空気1m³中の水蒸気量を求めなさい。（4点）
→ 19.4×0.67＝12.998≒約13.0g/m³
