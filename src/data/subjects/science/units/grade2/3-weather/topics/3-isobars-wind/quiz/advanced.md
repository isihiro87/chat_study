# 等圧線と風 発展問題

## 問1: 天気図から気圧と風の関係を読み取る

ある日の天気図では、日本海にX（中心気圧1028hPa）、太平洋上にY（中心気圧988hPa）がある。XとYの間には等圧線が引かれている。

(1) X、Yはそれぞれ高気圧・低気圧のどちらか。
→ X：高気圧（中心の気圧が周囲より高い）。Y：低気圧（中心の気圧が周囲より低い）。

(2) XとYの気圧差は何hPaか。
→ 1028 − 988 ＝ 40hPa。

(3) 天気図では等圧線は4hPaごとに引かれる。XとYの間には等圧線が最大で何本引かれると考えられるか。（X、Y自身の等圧線は含めない）
→ X（1028hPa）とY（988hPa）の間の等圧線は、1024, 1020, 1016, 1012, 1008, 1004, 1000, 996, 992の9本。

(4) X付近の天気とY付近の天気を、気流の向きと関連づけてそれぞれ説明しなさい。
→ X（高気圧）付近：上空から地表に向かう下降気流が生じ、空気が乾燥して雲ができにくいため、晴れになりやすい。Y（低気圧）付近：地表から上空に向かう上昇気流が生じ、空気が冷やされて雲ができやすいため、くもりや雨になりやすい。

(5) 地表付近でXからYに向かう風は、北半球ではまっすぐ吹かず曲がる。Xの地表付近で風がどのように吹き出すか、またYの地表付近で風がどのように吹き込むかを、それぞれ答えなさい。
→ X（高気圧）の地表付近：中心から外に向かって時計回りに吹き出す。Y（低気圧）の地表付近：外から中心に向かって反時計回りに吹き込む。
