# 日本の四季の天気 大問小問

## 大問1: 日本の四季の天気（配点 24点）

次のA〜Dは、日本付近のある季節の天気の特徴を述べたものである。それぞれどの季節か答え、問いに答えなさい。

A: 大陸に高気圧が発達し、等圧線が南北に縦じま状に並ぶ。
B: 太平洋上の高気圧が日本を覆い、南東の風が吹く。
C: 高気圧と低気圧が交互に通過し、天気が数日ごとに変わる。
D: 2つの気団がぶつかり、停滞前線が東西にのびる。

(1) A〜Dはそれぞれどの季節（または時期）か答えなさい。（各2点、計8点）
→ A：冬、B：夏、C：春（または秋）、D：梅雨（つゆ）

(2) Aの気圧配置を漢字4文字で何というか。（2点）
→ 西高東低

(3) Aのとき、日本海側と太平洋側の天気はそれぞれどうなるか。（4点）
→ 日本海側：雪や雨が多い。太平洋側：乾燥した晴天。

(4) Aのとき日本海側に大雪が降るしくみを、「季節風」「日本海」「山脈」という語を使って説明しなさい。（5点）
→ 北西の季節風が日本海を渡るとき暖流から水蒸気を吸収し、湿った空気が山脈にぶつかって上昇することで雪雲が発達し、大雪を降らせる。

(5) Bのとき日本を覆う高気圧を何というか。（2点）
→ 太平洋高気圧（小笠原高気圧）

(6) Dのとき停滞前線ができる原因となる2つの気団の名前を答えなさい。（3点）
→ 小笠原気団とオホーツク海気団

## 大問2: 台風（配点 18点）

台風について、次の問いに答えなさい。

(1) 台風の定義を「熱帯低気圧」「最大風速」という語を使って述べなさい。（3点）
→ 熱帯低気圧が発達し、最大風速が17m/s以上になったもの。

(2) 台風が発達するためのエネルギー源は何か。（2点）
→ 暖かい海上から蒸発する大量の水蒸気

(3) 台風が北上したあと東へ向きを変えるのは、何という風の影響か。（2点）
→ 偏西風

(4) 夏から秋にかけて台風が日本に接近しやすくなるのはなぜか。「太平洋高気圧」という語を使って説明しなさい。（5点）
→ 夏から秋にかけて太平洋高気圧の勢力が弱まり、台風が太平洋高気圧の縁を回って日本列島の方へ北上しやすくなるため。

(5) 台風がもたらす気象災害を2つ答えなさい。（3点）
→ 暴風による被害、大雨による河川の氾濫や土砂崩れ

(6) 台風が日本付近を通過すると温帯低気圧に変わることがある。その理由を簡単に答えなさい。（3点）
→ 北上すると海水温が下がり、水蒸気の供給が減るため、熱帯低気圧の性質を失い前線を伴う温帯低気圧に変わる。

## 大問3: 天気の予測と気象災害への備え（配点 18点）

(1) コンピュータで大気の状態を数値的にシミュレーションして天気を予報する方法を何というか。（2点）
→ 数値予報

(2) 災害が予想される地域を地図上に示したものを何というか。（2点）
→ ハザードマップ

(3) 重大な災害が予想されるときに気象庁が発表する警報の名称を答えなさい。（2点）
→ 特別警報

(4) 豊富な降水量が日本にもたらすめぐみを2つ答えなさい。（4点）
→ 農業用水の確保、水力発電のエネルギー源

(5) 大雨や台風の接近が予想されるとき、自分の身を守るために事前にすべきことを2つ答えなさい。（4点）
→ ハザードマップで自宅周辺の危険な場所や避難経路を確認する。気象情報を注意して確認し、早めに避難行動をとる。

(6) 日本付近の天気の移り変わりを利用して、翌日の天気を予想する方法を説明しなさい。（4点）
→ 偏西風の影響で天気は西から東へ移るため、西の地域の天気図や気象データを確認すれば、翌日の自分の地域の天気をおおよそ予想できる。
