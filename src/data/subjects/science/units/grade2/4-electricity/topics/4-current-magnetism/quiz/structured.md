# 電流と磁界 大問小問

## 大問1: 棒磁石のまわりの磁界（配点 18点）

棒磁石のまわりの磁界について、次の問いに答えなさい。

(1) 磁力がはたらいている空間を何というか。（2点）
→ 磁界。

(2) 磁界の向きはどのように定められているか。（2点）
→ 磁針のN極が指す向き。

(3) 磁界のようすを線で表したものを何というか。（2点）
→ 磁力線。

(4) 磁力線はどちらの極から出てどちらの極に入るか。（2点）
→ N極から出てS極に入る。

(5) 棒磁石の両極付近では磁力線の間隔がせまくなっている。このことから、両極付近の磁界についてどのようなことがわかるか。（3点）
→ 磁力線の間隔がせまいほど磁界が強いので、両極付近は磁界が強い。

(6) 棒磁石のN極の前に磁針を置いたとき、磁針のN極はどちらを向くか。理由も含めて答えよ。（4点）
→ 棒磁石のN極から遠ざかる方向（S極の方向）を向く。磁界の向きは磁針のN極が指す向きであり、磁力線はN極から出てS極に入るため、棒磁石のN極の前では磁界の向きはN極から離れる方向になる。

(7) 棒磁石のまわりに鉄粉を撒いたときに現れる模様は何を表しているか。（3点）
→ 磁力線の形を表している。鉄粉が磁界の向きに沿って並ぶため、磁力線に沿った曲線の模様が現れる。

## 大問2: 導線・コイルがつくる磁界（配点 20点）

導線やコイルに電流を流したときの磁界について、次の問いに答えなさい。

(1) 1本の直線の導線に電流を流したとき、導線のまわりにできる磁界はどのような形か。（2点）
→ 導線を中心とした同心円状の磁界ができる。

(2) 導線のまわりにできる磁界の向きを求める方法を何というか。（2点）
→ 右手の法則（右ねじの法則）。

(3) 右手の法則について、親指と残りの4本の指はそれぞれ何の向きを表すか。（4点）
→ 親指は電流の向き、残りの4本の指を曲げた向きが磁界の向きを表す。

(4) 導線のまわりに鉄粉を撒くと、どのような模様が見えるか。（2点）
→ 導線を中心とした同心円状の模様が見える。

(5) 導線のそばに置いた磁針のN極が東を指した。電流の向きを逆にすると磁針のN極はどちらを指すか。（3点）
→ 西を指す。電流の向きを逆にすると、磁界の向きも逆になるため。

(6) コイルに電流を流すと、どのような磁界ができるか。（2点）
→ 棒磁石と同様の磁界ができる。コイルの一方がN極、もう一方がS極になる。

(7) コイルの磁界で、N極がどちら側にできるかを右手の法則で求める方法を説明せよ。（5点）
→ 右手の4本の指を電流の流れる向きに曲げたとき、親指が指す方向がN極の側になる。電流の向きを逆にするとN極とS極が入れかわる。

## 大問3: 磁界中のコイルが受ける力とモーター（配点 22点）

磁石の間にコイルを置いて電流を流す実験を行った。次の問いに答えなさい。

(1) 磁界の中に置いた導線に電流を流すと、導線はどうなるか。（2点）
→ 力を受けて動く。

(2) この力の向きは、電流の向きと磁界の向きに対してどのような関係にあるか。（3点）
→ 電流の向きと磁界の向きの両方に垂直な方向にはたらく。

(3) 電流の向きを逆にすると、力の向きはどうなるか。（2点）
→ 力の向きも逆になる。

(4) 磁石のN極とS極の位置を入れかえると、力の向きはどうなるか。（2点）
→ 力の向きが逆になる。磁界の向きが逆になるため。

(5) 電流の向きと磁界の向きの両方を逆にすると、力の向きはどうなるか。理由も答えよ。（3点）
→ 力の向きは変わらない。電流の向きを逆にすることで力が逆になり、さらに磁界の向きを逆にすることでもう一度逆になるため、結果として元の向きに戻る。

(6) モーターが回転し続けるために、半回転ごとに電流の向きを切りかえる部品の名前を答えよ。（2点）
→ 整流子。

(7) 整流子に接触して外部から電流を送る部品の名前を答えよ。（2点）
→ ブラシ。

(8) 整流子がなかったとした場合、コイルはどのような動きをするか。その理由も含めて説明せよ。（6点）
→ 半回転したところで力の向きが逆になるため、コイルは同じ方向に回転し続けることができず、往復運動（振動）をしてしまう。整流子があることで半回転ごとに電流の向きが切りかわり、常に同じ方向の力が加わるので連続回転できる。

## 大問4: 電磁誘導の実験（配点 22点）

コイルに磁石を出し入れして検流計のふれを調べた。次の問いに答えなさい。

(1) コイルの中の磁界が変化したとき、コイルに電圧が生じて電流が流れる現象を何というか。（2点）
→ 電磁誘導。

(2) (1)で流れる電流を何というか。（2点）
→ 誘導電流。

(3) コイルにN極を入れたときと出したときで、検流計の針のふれる向きはどうなるか。（3点）
→ 逆になる。入れるときと出すときで磁界の変化の向きが逆になるため、誘導電流の向きも逆になる。

(4) コイルの中で磁石を静止させると、検流計の針はどうなるか。理由も答えよ。（4点）
→ 検流計の針はふれない（0を示す）。磁石が静止しているとコイル内の磁界が変化しないため、電磁誘導は起こらず誘導電流が流れない。

(5) 誘導電流を大きくするための方法を3つ答えよ。（6点）
→ ①磁石を速く動かす。②コイルの巻き数を増やす。③より強い磁石を使う。

(6) 磁石のN極を入れるときとS極を入れるときで、誘導電流の向きはどうなるか。理由も答えよ。（5点）
→ 逆になる。N極とS極では磁界の向きが逆なので、コイル内の磁界の変化の向きも逆になり、誘導電流の向きが逆になる。

## 大問5: 直流と交流（配点 18点）

直流と交流について、次の問いに答えなさい。

(1) 常に一定の向きに流れる電流を何というか。また、その具体例を1つ答えよ。（3点）
→ 直流（DC）。乾電池から得られる電流。

(2) 電流の向きと大きさが周期的に変化する電流を何というか。また、その具体例を1つ答えよ。（3点）
→ 交流（AC）。家庭のコンセントから得られる電流。

(3) 交流が1秒間に繰り返す振動の回数を何というか。単位も答えよ。（2点）
→ 周波数。単位はHz（ヘルツ）。

(4) 日本の東日本と西日本では交流の周波数が異なる。それぞれ何Hzか答えよ。（2点）
→ 東日本は50Hz、西日本は60Hz。

(5) 直流と交流をオシロスコープで表示すると、それぞれどのような波形になるか。（4点）
→ 直流は時間軸に平行な直線。交流は波（正弦波）の形になり、電圧がプラスとマイナスを周期的に繰り返す。

(6) 発光ダイオードに直流をつないだときと交流をつないだときの光り方のちがいを説明せよ。（4点）
→ 直流では一定の明るさで連続して光る（逆向きにつなぐと光らない）。交流では点滅して光る。発光ダイオードは一方向にしか電流を流さないため、交流では電流が流れる向きのときだけ光り、逆向きのときは消えるため点滅する。
