# 静電気と電流の正体 発展問題

## 問1: 帯電列と電子移動の関係

(1) 「帯電列」とは何か説明しなさい。また、帯電列の中で離れた位置にある2つの物質をこすり合わせると、帯電のしやすさはどう変わるか。
→ 帯電列とは、物質をこすり合わせたときに＋に帯電しやすいものから−に帯電しやすいものまで順番に並べたものである。帯電列で離れた位置にある2つの物質ほど、こすり合わせたときに多くの電子が移動し、強く帯電する。

(2) ガラス棒を絹でこするとガラス棒は＋に帯電し、エボナイト棒を毛皮でこするとエボナイト棒は−に帯電する。このことから、各物質の帯電列における位置関係を推定しなさい。
→ ＋に帯電しやすい順に、毛皮 → ガラス → 絹 → エボナイト と推定できる。ガラスは絹に対して電子を失いやすく（＋に帯電）、エボナイトは毛皮から電子を受け取りやすい（−に帯電）。

(3) 帯電列を知っていると、2つの物質をこすり合わせたとき、どちらが＋に帯電しどちらが−に帯電するか予測できる。この考え方を用いて、ストローとティッシュペーパーでは電子がどちら向きに移動するか予測する方法を説明しなさい。
→ 帯電列で＋側にある物質は電子を失いやすく、−側にある物質は電子を受け取りやすい。ストローが−に帯電することが知られているので、ストローはティッシュより帯電列の−側に位置する。したがって電子はティッシュからストローへ移動すると予測できる。

## 問2: クルックス管での十字形の影の実験

(1) クルックス管の中に十字形の金属板を置いて高い電圧をかけたところ、蛍光面に十字形の影が観察された。この結果から陰極線について何がわかるか、「直進」という語を用いて説明しなさい。
→ 陰極線は直進する性質があるとわかる。十字形の金属板が陰極線をさえぎった部分だけ影になるということは、陰極線が光のようにまっすぐ進んでいることを示している。

(2) 十字形の影ができる実験で、金属板を取り除くと蛍光面はどのように変化するか予測しなさい。
→ 蛍光面全体が均一に光る。金属板がなくなったため陰極線をさえぎるものがなくなり、−極から出た電子が蛍光面全体にまんべんなく当たるため。

(3) この実験で電極の＋極と−極を入れ替えると、十字形の影はどうなるか理由とともに答えなさい。
→ 影の向きが反対（裏返し）になる。陰極線は常に−極（陰極）から＋極（陽極）に向かって飛ぶため、電極を入れ替えると電子の飛ぶ方向が逆になり、十字形の金属板の反対側が照らされる。

(4) クルックス管の実験で、陰極線に上から磁石のN極を近づけると陰極線が曲がった。もしN極の代わりにS極を近づけたら、陰極線はどのように変化するか。
→ 反対方向に曲がる。磁界の向きが逆になるため、電子が受ける力の向きも逆になる。

## 問3: 導線内の電子の動きと電流の向きの関係

(1) 「電流は＋極から−極へ流れる」と定義されているが、実際に導線の中で移動している粒子（電子）の向きは逆である。この矛盾が生じた歴史的背景を説明しなさい。
→ 18世紀に電気の研究が始まったとき、電流は＋の電気が＋極から−極へ流れるものと考えられ、そのように定義された。しかし19世紀末に電子が発見され、実際に移動しているのは−の電気をもった電子であり、−極から＋極へ動いていることがわかった。すでに＋→−の定義が広く使われていたため、変更されずにそのまま残った。

(2) 導線に電流が流れているとき、導線の断面を1秒間に通過する電子の個数が多いほど電流はどう変化するか。電流の単位にもふれて説明しなさい。
→ 電流は大きくなる。電流の大きさ（単位：アンペア〔A〕）は、導線の断面を一定時間に通過する電子の量（電気の量）に比例する。1秒間に通過する電子の個数が多いほど、流れる電気の量が多くなるため、電流が大きくなる。

(3) 乾電池の＋極から−極に向かって電流が流れている回路がある。このとき、導線の中の電子は電池のどちら側から出発し、回路を通ってどちら側に戻るか。電流の向きと比較して説明しなさい。
→ 電子は乾電池の−極から出発し、導線を通って＋極に戻る。電流の向きは＋極→導線→−極と定義されているため、電子の移動方向と電流の向きはちょうど逆である。

## 問4: 放射線の透過性と利用

(1) 放射線の種類（α線・β線・γ線）を透過性が低い順に並べ、それぞれを遮るのに必要なものを答えなさい。
→ 透過性が低い順に、α線（紙で遮れる）→ β線（アルミニウムなどの金属板で遮れる）→ γ線（鉛やコンクリートなどの厚い壁が必要）。

(2) 医療で用いられるX線検査（レントゲン）で骨の形が写る理由を、放射線の透過性の違いから説明しなさい。
→ X線は皮膚や筋肉などの軟らかい組織は透過するが、骨のような密度の高い組織は透過しにくい。そのため、X線を体に当てると骨の部分だけX線が遮られ、フィルムやセンサーに影として骨の形が映し出される。

(3) 放射線は人体に有害な影響を与えることがあるにもかかわらず、医療や工業で広く利用されている。放射線の利用例を医療以外に1つ挙げ、その原理を説明しなさい。
→ 工業製品の非破壊検査。製品を壊さずにγ線やX線を当てて内部の構造や欠陥を調べることができる。放射線の透過性を利用し、製品内部にひびや空洞があればX線の透過具合が変わるため、外見からは見えない欠陥を発見できる。

## 問5: 総合問題

(1) 「静電気」と「電流」はどちらも電子の移動に関係しているが、その違いを電子の移動のしかたに注目して説明しなさい。
→ 静電気は摩擦によって電子が物質間を一度に移動して帯電する現象であり、電子の移動は一時的である。一方、電流は導線の中を電子が継続的に流れ続ける現象である。静電気が放電するときは一気に電子が移動するが、電流は回路が閉じている間ずっと電子が流れ続ける。

(2) クルックス管の実験で観察できる3つの性質（直進性・蛍光板の発光・磁石による曲がり）をすべてふまえて、陰極線の正体が電子であることを論証しなさい。
→ まず、クルックス管の中に十字形の金属板を置くと蛍光面に十字形の影ができることから、陰極線は直進する性質をもつとわかる。次に、蛍光板に当たると発光することから、陰極線はエネルギーをもった粒子であることが示される。さらに、磁石を近づけると進路が曲がることから、この粒子は電気をもっていることがわかる（光は磁石で曲がらない）。以上から、陰極線の正体は−の電気をもった粒子、すなわち電子であると結論づけられる。
