# 電流と磁界 発展問題

## 問1: コイルの中で磁石を止めたとき

コイルの中にN極を下にした棒磁石をすばやく入れたあと、コイルの中で磁石を静止させた。

(1) 磁石をすばやく入れているとき、検流計の針はどうなるか。
→ 一方向にふれる。コイル内の磁界が変化しているため、電磁誘導が起こり誘導電流が流れる。

(2) 磁石をコイルの中で静止させると、検流計の針はどうなるか。
→ 0に戻る（ふれなくなる）。磁石が静止しているとコイル内の磁界は変化しないため、誘導電流は流れない。

(3) 電磁誘導が起こるために必要な条件を「磁界の変化」という語を使って説明せよ。
→ 電磁誘導はコイル内の磁界が変化しているときにだけ起こる。磁石が動くとコイルを貫く磁界が変化するが、磁石が静止すると磁界の変化がなくなり、誘導電流は流れなくなる。

## 問2: モーターの整流子の役割

モーターの構造と整流子のはたらきについて考える。

(1) モーターのコイルが半回転すると、コイルの両辺の位置が入れかわる。このとき整流子がなければ何が起こるか。
→ コイルが半回転すると、磁界の中でコイルの両辺の位置が入れかわるため、各辺に流れる電流と磁界の関係が変わり、力の向きが逆になる。その結果、コイルは逆方向に回転しようとし、連続回転せずに振動する。

(2) 整流子はこの問題をどのように解決しているか、具体的に説明せよ。
→ 整流子はコイルが半回転するごとに、コイルに流れる電流の向きを自動的に切りかえる。これにより、コイルの各辺が常に同じ方向に力を受けるようになり、コイルは一定方向に連続して回転し続けることができる。

(3) ブラシの役割を説明せよ。
→ ブラシは整流子に接触して外部の電源からコイルに電流を送る部品である。整流子が回転しても、ブラシが常に整流子に触れていることで電流が途切れずに流れ続ける。

## 問3: 送電と変圧

発電所で発生した電気が家庭に届くまでの過程について考える。

(1) 発電所から家庭への送電に交流が使われる最大の理由を説明せよ。
→ 交流は変圧器（トランス）で電圧を簡単に変えられるため。直流は変圧器で電圧を変えることが難しい。

(2) 送電するとき、電圧を高くして送ることで送電中のエネルギー損失を減らせる理由を、「電力」と「電流」と「発熱」という語を使って説明せよ。
→ 同じ電力（電圧×電流）を送るとき、電圧を高くすると電流を小さくできる。送電線での発熱は電流の2乗に比例するため、電流が小さいほど送電線での発熱によるエネルギー損失が少なくなる。

(3) 発電所から家庭に届くまでに電圧はどのように変化するか。
→ 発電所で数十万Vの高電圧に上げて送電し、途中の変電所で段階的に電圧を下げ、最終的に家庭では100Vまたは200Vに下げて届けられる。

## 問4: 直流と交流の発光ダイオードの光り方

2つの発光ダイオードを逆向きに並列につないだ回路をつくり、直流電源と交流電源にそれぞれつないで光り方を調べた。

(1) 直流電源につないだとき、2つの発光ダイオードの光り方はどうなるか。
→ 一方の発光ダイオードだけが連続して光り、もう一方は光らない。直流は一定方向に電流が流れるため、電流の向きに合った発光ダイオードだけが光る。

(2) 交流電源につないだとき、2つの発光ダイオードの光り方はどうなるか。
→ 2つの発光ダイオードが交互に点滅する。交流は電流の向きが周期的に変わるため、電流が一方向のときは片方が光り、逆向きのときはもう片方が光る。

(3) (2)で周波数が50Hzの交流を使った場合、1秒間にそれぞれの発光ダイオードは何回点灯するか。
→ それぞれ50回点灯する。50Hzの交流は1秒間に50回周期を繰り返し、各周期で電流が一方向に1回ずつ流れるため、各発光ダイオードは50回ずつ点灯する。

(4) この実験から、直流と交流のどのようなちがいを確認できるか。
→ 直流は常に一定方向に電流が流れるのに対し、交流は電流の向きが周期的に変化することを確認できる。発光ダイオードが一方向にしか電流を流さない性質を利用して、このちがいを視覚的に判別できる。

## 問5: 電磁誘導と発電機の原理

コイルの中で磁石を回転させる発電機について考える。

(1) 発電機が電磁誘導を利用して電気をつくるしくみを説明せよ。
→ コイルの中で磁石を回転させると、コイルを貫く磁界が連続的に変化する。この磁界の変化によって電磁誘導が起こり、コイルに誘導電流が連続的に流れる。

(2) 発電機でつくられる電流は直流か交流か。その理由も答えよ。
→ 交流。磁石が回転するとコイルを貫く磁界の向きが周期的に変化するため、誘導電流の向きも周期的に変化する。

(3) 発電機の回転を速くすると、発生する交流の周波数はどうなるか。
→ 周波数が大きくなる。磁石の回転が速くなると、磁界の変化が速くなるため、1秒あたりの振動回数（周波数）が増える。
