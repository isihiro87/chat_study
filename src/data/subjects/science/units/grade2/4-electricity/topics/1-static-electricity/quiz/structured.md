# 静電気と電流の正体 大問小問

## 大問1: ストローの摩擦実験（配点 20点）

次の文を読んで、あとの問いに答えなさい。

ストローAをティッシュペーパーでこすると、ストローAは（ ア ）に帯電した。同じようにストローBもティッシュペーパーでこすった。次に、ストローAに帯電したストローBを近づけると（ イ ）し合った。さらに、ストローAにこすった後のティッシュペーパーを近づけると（ ウ ）合った。

(1) ア〜ウにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：−（マイナス）、イ：反発（しりぞけ）、ウ：引き

(2) ストローAが−に帯電した理由を、「電子」という語を用いて説明しなさい。（3点）
→ ティッシュペーパーからストローAへ電子が移動したため。電子は−の電気をもつ粒子なので、電子を受け取ったストローAは−に帯電した。

(3) こすった後のティッシュペーパーは＋と−のどちらに帯電しているか。理由とともに答えなさい。（3点）
→ ＋に帯電している。電子を失ったため、＋の電気が残るから。

(4) ストローAとストローBが反発し合った理由を説明しなさい。（3点）
→ どちらも同じティッシュペーパーでこすったため、同じ−に帯電している。同じ種類の電気どうしは反発し合う性質があるから。

(5) ストローAとティッシュペーパーが引き合った理由を説明しなさい。（3点）
→ ストローAは−に帯電し、ティッシュペーパーは＋に帯電している。異なる種類の電気どうしは引き合う性質があるから。

## 大問2: クルックス管の実験（配点 20点）

次の文を読んで、あとの問いに答えなさい。

クルックス管（ア）に高い電圧をかけると、（ イ ）極から（ ウ ）極に向かって粒子が飛び出す。この粒子の流れを（ エ ）という。管の中に蛍光板を置くと（ オ ）が見られた。また、この流れに磁石を近づけると（ カ ）ことがわかった。

(1) アにあてはまるクルックス管の別名を答えなさい。（2点）
→ 真空放電管。

(2) イ〜エにあてはまる語句をそれぞれ答えなさい。（各2点）
→ イ：−（陰）、ウ：＋（陽）、エ：陰極線（電子線）

(3) オにあてはまる現象を答えなさい。（2点）
→ 蛍光板が光る（発光する）。

(4) カにあてはまる現象を答えなさい。（2点）
→ 陰極線の進む向きが曲がる。

(5) 陰極線の正体は何か。名前とその性質を答えなさい。（4点）
→ 電子。−の電気をもつ非常に小さな粒子である。

(6) 陰極線が磁石で曲がることから何がわかるか説明しなさい。（4点）
→ 陰極線の正体は電気をもった粒子であること。光は磁石で曲がらないため、陰極線は光ではなく、電気をもった粒子（電子）の流れであるとわかる。

## 大問3: 電流の正体（配点 20点）

次の問いに答えなさい。

(1) 電流の正体は何か。（2点）
→ 導線の中を電子が移動すること。

(2) 導線の中で、電子はどちらの極からどちらの極に移動するか。（2点）
→ −極から＋極へ移動する。

(3) 電流の向きはどちらの極からどちらの極に流れると定義されているか。（2点）
→ ＋極から−極へ流れると定義されている。

(4) 電流の向きと電子の移動の向きが逆であるのはなぜか説明しなさい。（4点）
→ 電子が発見される前に、歴史的に電流は＋極から−極へ流れるものと決められたため。のちに電子の移動方向が判明し、逆向きであることがわかったが、電流の向きの定義はそのまま残った。

(5) 電流の単位を答えなさい。（2点）
→ アンペア（記号：A）。

(6) 次の図で、乾電池のA端子が＋極、B端子が−極のとき、電流の向きと電子の移動の向きをそれぞれ答えなさい。（各4点）
→ 電流の向き：A（＋極）→導線→B（−極）。電子の移動の向き：B（−極）→導線→A（＋極）。

## 大問4: 放射線の性質と利用（配点 20点）

次の問いに答えなさい。

(1) X線やα線、β線、γ線などをまとめて何というか。（2点）
→ 放射線。

(2) 放射線を出す物質を何というか。（2点）
→ 放射性物質。

(3) α線を遮ることができるものは何か。（3点）
→ 紙。α線は透過性が最も低い。

(4) β線を遮ることができるものは何か。（3点）
→ アルミニウムなどの金属板。

(5) γ線を遮るには何が必要か。（3点）
→ 鉛やコンクリートなどの厚い壁。

(6) X線が医療で利用されている例を1つ答えなさい。（3点）
→ レントゲン検査（X線検査）。X線の透過性を利用して骨や体内の様子を撮影する。

(7) 自然放射線とは何か説明しなさい。（4点）
→ 自然界に存在する放射線のこと。宇宙から降り注ぐ宇宙線や、大地や食物に含まれる放射性物質から出る微量の放射線がある。

## 大問5: 放電現象（配点 20点）

次の問いに答えなさい。

(1) たまった電気が空間を移動する現象を何というか。（2点）
→ 放電。

(2) 雷が起こるしくみを「静電気」「放電」という語を用いて説明しなさい。（4点）
→ 雲の中で氷の粒がぶつかり合い、摩擦によって大量の静電気がたまる。たまった静電気が空気中を一気に流れる放電が雷である。

(3) 真空放電とはどのような現象か説明しなさい。（4点）
→ 気圧を低くした空間（真空に近い状態）で電圧をかけたとき、電流が流れる現象。

(4) 真空放電が利用されている製品を2つ答えなさい。（各2点）
→ 蛍光灯、ネオンサイン。

(5) ふだん空気中で放電が起こりにくい理由を説明しなさい。（3点）
→ 空気は電気を通しにくい性質をもっているため。非常に大きな電気がたまったときだけ放電が起こる。

(6) 蛍光灯が光るしくみを「真空放電」という語を用いて説明しなさい。（3点）
→ 蛍光灯は管内の気圧を低くした状態で真空放電を起こし、管内の物質に電子がぶつかって発光するしくみである。
