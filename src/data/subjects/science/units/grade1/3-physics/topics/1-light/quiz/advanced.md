# 光の世界 発展問題

## 問1: レンズの公式を使った計算

焦点距離12cmの凸レンズを使い、物体の位置を変えてスクリーンに実像ができる距離を調べた。次の表は実験結果である。

| 物体の距離（cm） | 像の距離（cm） | 像の大きさ |
|------|------|------|
| 60 | 15 | 物体より小さい |
| 36 | 18 | 物体より小さい |
| 24 | 24 | 物体と同じ |
| 18 | 36 | 物体より大きい |
| 15 | 60 | 物体より大きい |

(1) 表の結果から、物体の距離と像の距離はどのような関係にあるといえるか。
→ 物体の距離が小さくなる（レンズに近づく）と像の距離は大きくなり、物体の距離が大きくなる（レンズから遠ざかる）と像の距離は小さくなる。

(2) 物体と像が同じ大きさになるのは、物体がレンズからどの位置にあるときか。焦点距離との関係で答えよ。
→ 焦点距離の2倍の位置（12cm×2＝24cm）。

(3) レンズの公式 1/a + 1/b = 1/f を用いて、物体を16cmの距離に置いたときの像の距離を求めよ。
→ 1/16 + 1/b = 1/12。1/b = 1/12 - 1/16 = 4/48 - 3/48 = 1/48。b = 48cm。

(4) (3)のとき、像の大きさは物体の何倍か。
→ b/a = 48/16 = 3倍。物体の3倍の大きさの倒立の実像ができる。

(5) 物体を12cm（焦点の位置）に近づけていくと、像の距離はどうなるか。グラフの概形を言葉で説明せよ。
→ 像の距離は急激に大きくなり、物体が焦点に達すると像の距離は無限大になる（像ができなくなる）。物体が焦点より内側に入ると実像はできず、虚像が見える。

## 問2: 鏡の全身像

壁に高さ40cmの鏡を目の高さを中心にして垂直にかけた。鏡から2m離れた位置に立って自分の姿をうつす。目の高さは床から150cm、身長は170cmとする。

(1) この鏡で見える範囲（上端と下端）を、反射の法則を使って求めよ。
→ 鏡の上端は目の高さ＋20cm＝170cmの位置。反射の法則から、鏡の上端で反射する光は、目の上方20cmの2倍＝40cm上まで見える。よって上端は150＋40＝190cmまで。同様に下端は150−40＝110cm。見える範囲は床から110cm〜190cmの80cm分（鏡の高さの2倍）。

(2) 鏡からの距離を1mに近づけた場合、見える範囲はどう変わるか。
→ 見える範囲は変わらない。鏡の高さの2倍（80cm）の範囲が見える。鏡からの距離に関係なく、鏡で見える範囲は一定である。

(3) 全身をうつすためには、鏡の高さを何cm以上にする必要があるか。
→ 身長170cmの半分＝85cm以上。ただし鏡の位置は、上端を頭と目の中点（＝160cm）、下端を目と足先の中点（＝75cm）の高さに合わせる必要がある。

(4) 目の高さが150cmで身長が170cmの場合、鏡の下端を床から何cmの高さに設置すれば全身が映るか。
→ 目と足先の中点＝150÷2＝75cm。鏡の下端を床から75cmの高さに設置する。

## 問3: 凸レンズの像の変化

焦点距離10cmの凸レンズを使い、物体の位置を変えて像のようすを調べた。

(1) 物体を焦点距離の2倍の位置（レンズから20cm）に置いたとき、像はどこにできるか。像の大きさと向きも答えよ。
→ レンズから20cmの位置に、物体と同じ大きさの倒立の実像ができる。

(2) 物体をレンズから40cmの位置に置いたとき、像はどこにできるか。像の大きさも答えよ。
→ 1/40 + 1/b = 1/10 より 1/b = 1/10 − 1/40 = 3/40 よって b ≒ 13.3cm。物体の1/3倍の小さい倒立の実像ができる。

(3) 物体を焦点（レンズから10cm）に近づけていくと、像はどのように変化するか。
→ 実像は大きくなり、像の位置はレンズから遠ざかる。焦点上に置くと像はできなくなる。

(4) 物体を焦点より内側（レンズから5cm）に置くと、レンズを通して見える像はどのようなものか。スクリーンに映せるかどうかも答えよ。
→ 正立で物体より大きい虚像が見える。スクリーンには映らない。光が実際に集まっているわけではなく、レンズを通して見たときに光の延長線上に像が見えるだけだから。

## 問4: 水中の見え方

水深20cmの位置にコインが沈んでいる。水面の真上から見ると、コインは浅い位置にあるように見える。

(1) コインが浅く見える理由を「屈折」という語を使って説明しなさい。
→ コインからの光が水面で屈折して空気中に出るとき、法線から遠ざかるように曲がる。観察者の目に届いた光をまっすぐ延長した位置にコインがあるように見えるため、実際より浅い位置に見える。

(2) 空気中から水中を見るとき、屈折角と入射角の大小関係から、光がどのように曲がるか説明しなさい。
→ 水中から空気中に出る光は屈折角＞入射角となり、法線から遠ざかるように曲がる。そのため、光の延長線上（見かけの位置）は実際のコインの位置より上（浅い位置）になる。

(3) プールの水深が実際より浅く見える現象は、(1)と同じ原理か。簡潔に説明しなさい。
→ 同じ原理である。プールの底からの光が水面で屈折するため、底が実際より浅い位置にあるように見える。

(4) 水面の斜め方向から水中の物体を見ると、物体の見かけの位置はどのようにずれるか。
→ 実際の位置よりも浅く、かつ遠い位置にあるように見える。水面での屈折により、光の延長線上の見かけの位置が実際より上方向にずれるため。

## 問5: 光ファイバーと全反射の応用

光ファイバーは全反射を利用した通信技術である。

(1) 全反射が起こる条件を2つ答えなさい。
→ ①光が密度の大きい物質（水・ガラスなど）から密度の小さい物質（空気）に向かって進むこと。②入射角がある角度（臨界角）以上であること。

(2) 光ファイバーの内部で全反射が繰り返し起こるしくみを説明しなさい。
→ 光ファイバーは屈折率の大きいガラスの芯を屈折率の小さいガラスで覆った構造をしている。芯の中を進む光が外側のガラスとの境界面に当たるとき、入射角が臨界角以上になるため全反射が起こり、光はファイバー内部で反射を繰り返しながら進む。

(3) 全反射は空気中から水に光が入るときにも起こるか。理由を答えなさい。
→ 起こらない。全反射は密度の大きい物質から小さい物質に光が進むときにだけ起こる。空気中から水に入るときは密度の小さい物質から大きい物質に進むため、全反射は起こらない。

(4) 光ファイバーが通信に利用される利点を2つ答えなさい。
→ ①電気信号より大量のデータを高速に伝えられる。②電磁波の影響を受けにくく、ノイズが少ない。

(5) 光ファイバーが細く曲がっていても光を伝えられる理由を、全反射の性質を用いて説明せよ。
→ 光ファイバーの内部と外部の物質では屈折率が異なり、内部を進む光が側面に当たるときの入射角が常に臨界角を超えるように設計されている。そのため、ファイバーが曲がっていても内壁で全反射を繰り返しながら光が進むことができる。
