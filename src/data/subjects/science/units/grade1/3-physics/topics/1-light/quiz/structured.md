# 光の世界 大問小問

## 大問1: 光の直進と反射の法則（配点 18点）

光の性質について、次の問いに答えなさい。

(1) 自ら光を出す物体を何というか。（2点）
→ 光源。

(2) 次のア〜オのうち、光源であるものをすべて選び記号で答えよ。（3点）
ア 太陽　イ 月　ウ LED電球　エ ダイヤモンド　オ スマートフォンの画面
→ ア・ウ・オ。月は太陽の光を反射しており、ダイヤモンドも光を反射しているだけなので光源ではない。

(3) 光が鏡に当たるとき、鏡の面に垂直な直線と入射光のなす角を何というか。（2点）
→ 入射角。

(4) 入射角が35°のとき、反射角は何度か。（2点）
→ 35°。反射の法則により入射角＝反射角。

(5) ざらざらした面で光がさまざまな方向に反射する現象を何というか。（2点）
→ 乱反射。

(6) 紙に書かれた文字がどの方向からでも読めるのは、(5)の現象のおかげである。その理由を説明せよ。（4点）
→ 紙の表面はざらざらしているため乱反射が起こり、光がさまざまな方向に反射する。そのため、どの方向から見ても反射した光が目に届くから。

(7) りんごが赤く見える理由を、「吸収」「反射」という語を使って説明せよ。（3点）
→ 太陽の光のうち赤以外の色の光は吸収され、赤色の光だけが反射して目に届くため、赤く見える。

## 大問2: 鏡にうつる像（配点 18点）

平面鏡に物体をうつしたときの像について、次の問いに答えなさい。

(1) 鏡にうつった像は、鏡の面に対してどのような位置にできるか。（2点）
→ 鏡の面に対して物体と対称（線対称）の位置にできる。

(2) 鏡にうつった像の大きさは物体と比べてどうなるか。（2点）
→ 物体と同じ大きさ。

(3) 鏡にうつった像は実像か虚像か。また、その理由を答えよ。（4点）
→ 虚像。鏡のうしろに実際に光が集まっているわけではなく、目に届く光を延長した先に像があるように見えるだけだから。

(4) 身長150cmの生徒が、壁にかけた鏡で全身をうつしたい。鏡の高さは最低何cm必要か。（3点）
→ 75cm（身長の半分）。

(5) (4)で鏡の高さが身長の半分でよい理由を、光の道すじにふれて説明せよ。（5点）
→ 頭のてっぺんから出た光が鏡で反射して目に届くには、鏡の上端が頭と目の中点の高さにあればよい。同様に、つま先から出た光については鏡の下端が目とつま先の中点の高さにあればよい。したがって必要な鏡の高さは「頭から目の中点」と「目からつま先の中点」の差で、身長のちょうど半分になる。

(6) 鏡から1m離れて立っていた人が、鏡から2mの距離に後退した。全身を映すために必要な鏡の高さは変わるか。理由もあわせて答えよ。（2点）
→ 変わらない。反射の法則から、必要な鏡の高さは距離に関係なく身長の半分で一定である。

## 大問3: 光の屈折と全反射（配点 20点）

水そうに水を入れ、光の進み方を調べた。次の問いに答えなさい。

(1) 光が空気中から水中に進むとき、光の進む方向が変わる現象を何というか。（2点）
→ 光の屈折。

(2) 空気中から水中に光が入るとき、入射角と屈折角の大小関係はどうなるか。（2点）
→ 屈折角＜入射角。

(3) 水中から空気中に光が出るとき、入射角と屈折角の大小関係はどうなるか。（2点）
→ 屈折角＞入射角。

(4) 水中に沈めた硬貨を水面の上から見ると、実際の位置より浅いところにあるように見えた。この理由を「屈折」という語を使って説明せよ。（4点）
→ 硬貨から出た光が水面で屈折し、空気中では境界面から遠ざかる方向に曲がる。目に届いた光をまっすぐ延長した先に硬貨があるように見えるため、実際の位置より浅く見える。

(5) 水中から空気中に光が出ようとするとき、入射角がある角度以上になると光が水面をすべて反射して空気中に出なくなる。この現象を何というか。（2点）
→ 全反射。

(6) 全反射を利用した身近な製品を1つ答え、そのしくみを簡潔に説明せよ。（4点）
→ 光ファイバー。ファイバーの内部で光が全反射を繰り返すことで、光が外に漏れずに遠くまで伝わる。

(7) 全反射は空気中からガラスに光が入るときにも起こるか。理由を含めて答えよ。（2点）
→ 起こらない。全反射はガラスや水のように密度の大きい物質から空気のように密度の小さい物質に光が出ようとするときにのみ起こる。

(8) コップの水にストローを入れると曲がって見える理由を、「屈折」を使って説明せよ。（2点）
→ 水中から空気中に出る光が水面で屈折するため、ストローの位置がずれて見え、曲がっているように見える。

## 大問4: 凸レンズの像（配点 24点）

焦点距離が15cmの凸レンズを使い、物体の位置を変えてスクリーンにできる像を調べた。次の問いに答えなさい。

(1) 凸レンズの中心を通り、レンズの面に垂直な直線を何というか。（2点）
→ 光軸。

(2) 光軸に平行な光が凸レンズを通ったあとに集まる点を何というか。（2点）
→ 焦点。

(3) 物体をレンズから45cm（焦点距離の3倍）の位置に置いた。スクリーンにできる像について、次の①②に答えよ。（4点）
① 像の向きは物体と比べてどうなるか。
→ 上下左右が逆（倒立）。
② 像の大きさは物体と比べてどうなるか。
→ 物体より小さい。

(4) 物体をレンズから30cm（焦点距離の2倍）の位置に置いた。像の大きさは物体と比べてどうなるか。また像ができる位置はレンズから何cmか。（4点）
→ 物体と同じ大きさの実像ができる。像の位置はレンズから30cm。

(5) 物体をレンズから20cm（焦点と焦点距離の2倍の間）の位置に置いた。像の大きさは物体と比べてどうなるか。（2点）
→ 物体より大きい実像ができる。

(6) 物体をレンズから10cm（焦点より近い位置）に置いた。このとき見える像について、次の①〜③に答えよ。（6点）
① この像を何というか。
→ 虚像。
② 像の向きは物体と比べてどうなるか。
→ 物体と同じ向き（正立）。
③ この像はスクリーンに映すことができるか。理由もあわせて答えよ。
→ できない。虚像は実際に光が集まっているわけではなく、レンズを通して目で直接見たときだけ見えるから。

(7) 物体をレンズに近づけていくと、実像の大きさと位置はどのように変化するか。（4点）
→ 実像は大きくなり、レンズからの像の距離は遠くなる。

## 大問5: 光の作図と総合問題（配点 20点）

光の反射・屈折に関する次の問いに答えなさい。

(1) 凸レンズの作図で使う3本の光の道筋をそれぞれ説明せよ。（6点）
→ ①光軸に平行に入った光→レンズを通ったあと焦点を通る。②レンズの中心を通った光→そのまま直進する。③焦点を通ってからレンズに入った光→レンズを通ったあと光軸に平行に進む。

(2) 文字「F」を凸レンズの焦点より遠くに置いてスクリーンにうつした。スクリーンに見える文字の向きを答えよ。（3点）
→ 上下左右が逆になるため、上下が逆でかつ左右が逆の向きに見える。

(3) 焦点距離10cmの凸レンズで、物体を25cmの距離に置いたとき、像ができる位置をレンズの公式を使って求めよ。（5点）
→ 1/a + 1/b = 1/f より、1/25 + 1/b = 1/10。1/b = 1/10 - 1/25 = 5/50 - 2/50 = 3/50。b = 50/3 ≒ 16.7cm。レンズから約16.7cmの位置に実像ができる。

(4) 物体を焦点の位置ちょうどに置くとどうなるか。理由もあわせて答えよ。（4点）
→ 像はできない。焦点の位置にある物体からの光は凸レンズを通ったあと平行に進むため、どこにも集まらない。

(5) 遠くの景色を凸レンズでスクリーンに映す実験をした。はっきりした像がうつるのはレンズからどのくらいの距離か。「焦点距離」という語を使って答えよ。（2点）
→ 焦点距離にほぼ等しい距離。非常に遠くの物体からの光はほぼ平行に凸レンズに入るため、焦点に集まる。
