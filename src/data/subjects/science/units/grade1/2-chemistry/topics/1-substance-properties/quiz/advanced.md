# 身のまわりの物質とその性質 発展問題

## 問1: 密度の計算とグラフ

ある生徒が5種類の金属A〜Eの質量と体積を測定し、次の表にまとめた。

| 金属 | 質量（g） | 体積（cm³） |
|------|---------|-----------|
| A | 27.0 | 10.0 |
| B | 79.0 | 10.0 |
| C | 89.0 | 10.0 |
| D | 13.5 | 5.0 |
| E | 39.5 | 5.0 |

（参考：アルミニウム2.7g/cm³、鉄7.9g/cm³、銅8.9g/cm³）

(1) 金属A〜Eの密度をそれぞれ求めなさい。
→ A：27.0÷10.0＝2.7g/cm³、B：79.0÷10.0＝7.9g/cm³、C：89.0÷10.0＝8.9g/cm³、D：13.5÷5.0＝2.7g/cm³、E：39.5÷5.0＝7.9g/cm³。

(2) 同じ金属であると考えられるものの組み合わせをすべて答えよ。
→ AとD（どちらも密度2.7g/cm³でアルミニウム）、BとE（どちらも密度7.9g/cm³で鉄）。

(3) 金属AとDは質量も体積も異なるが、同じ物質と判断できる理由を「密度」という語を使って説明せよ。
→ 密度は物質ごとに固有の値をもつため、質量や体積が異なっても密度が同じであれば同じ物質と判断できる。

(4) 5種類の金属をすべて水（密度1.0g/cm³）に入れた場合、浮くものはあるか。理由を含めて答えよ。
→ ない。すべての金属の密度が水の密度（1.0g/cm³）より大きいため、すべて沈む。

## 問2: 実験計画 ─ 未知の金属の特定

理科室で見つかった金属のかたまりが何の金属か調べたい。電子てんびん、メスシリンダー、水が使える。

(1) この金属を特定するための実験手順を、順を追って説明しなさい。
→ ①電子てんびんで金属の質量をはかる。②メスシリンダーに適量の水を入れ、体積を読む。③金属を静かに水中に沈め、増えた分の体積を読む。④密度＝質量÷体積を計算する。⑤求めた密度を金属の密度表と比較して物質を特定する。

(2) メスシリンダーで金属の体積を測定するとき、水の量はどの程度にすべきか。理由もあわせて答えよ。
→ 金属を沈めたとき水面がメスシリンダーの目盛りの範囲内におさまる程度にする。少なすぎると金属が水に完全に沈まない可能性があり、多すぎると水があふれて正確に測定できないから。

(3) 測定した質量が23.7g、メスシリンダーの水位が沈める前45.0mL、沈めた後47.7mLであった。この金属の密度を求め、何の金属か推定せよ。（参考：銀10.5g/cm³、鉛11.3g/cm³、銅8.9g/cm³）
→ 体積：47.7−45.0＝2.7cm³（＝2.7mL）。密度：23.7÷2.7≒8.78g/cm³。密度約8.8g/cm³は銅の密度8.9g/cm³に最も近いので、この金属は銅と推定される。

(4) 測定誤差が生じる原因を2つ答えよ。
→ ①メスシリンダーの目盛りの読み取り誤差（目分量による誤差）。②金属表面に気泡が付着して体積が実際より大きく測定される。

## 問3: 白い粉末の実験設計

5種類の白い粉末（砂糖・食塩・デンプン・重曹・小麦粉）がラベルのないビンに入っている。これらを区別する実験を計画しなさい。

(1) まず有機物と無機物に分けるには、どのような実験をすればよいか。
→ ガスバーナーで加熱する。黒くこげて二酸化炭素が発生するものが有機物、変化しないものが無機物。

(2) 有機物に分類されるのはどれか。すべて答えよ。
→ 砂糖・デンプン・小麦粉（炭素をふくむ有機物）。

(3) 無機物の食塩と重曹を区別する方法を1つ考えて説明せよ。
→ 水に溶かして加熱する。重曹（炭酸水素ナトリウム）の水溶液を加熱すると二酸化炭素が発生するが、食塩水を加熱しても気体は発生しない。または、酢を加えると重曹は泡を出して溶けるが食塩は泡が出ない。

(4) 有機物の砂糖・デンプン・小麦粉を区別する実験を2つ提案し、それぞれの結果の違いを説明せよ。
→ 実験1：水に入れてかき混ぜる。砂糖は完全に溶けるが、デンプンと小麦粉はほとんど溶けない。実験2：ヨウ素液を加える。デンプンと小麦粉（デンプンを含む）は青紫色に変化するが、砂糖は変化しない。小麦粉はデンプンのほかにタンパク質も含むため、水を加えてこねると粘りが出る点でデンプンと区別できる。

## 問4: 浮き沈みの総合問題

次の表は、いくつかの物質の密度を示している。

| 物質 | 密度（g/cm³） |
|------|-------------|
| 氷 | 0.92 |
| 木材（スギ） | 0.40 |
| アルミニウム | 2.7 |
| 鉄 | 7.9 |
| 水銀（液体の金属） | 13.5 |
| 水 | 1.0 |
| エタノール | 0.79 |

(1) 水に浮く物質をすべて選びなさい。
→ 氷（0.92g/cm³）と木材（0.40g/cm³）。どちらも密度が水（1.0g/cm³）より小さいから。

(2) 水銀に浮く物質をすべて選びなさい。
→ 氷・木材・アルミニウム・鉄・水。すべて水銀（13.5g/cm³）より密度が小さい。

(3) エタノールに氷を入れると、氷は浮くか沈むか。理由もあわせて答えよ。
→ 沈む。氷の密度（0.92g/cm³）はエタノールの密度（0.79g/cm³）より大きいから。

(4) 水に氷が浮くのに、エタノールでは氷が沈む理由を「密度」という語を使って説明しなさい。
→ 水の密度（1.0g/cm³）は氷の密度（0.92g/cm³）より大きいので氷は水に浮くが、エタノールの密度（0.79g/cm³）は氷の密度より小さいので氷はエタノールに沈む。浮き沈みは物質と液体の密度の大小関係で決まる。

## 問5: 金属と非金属の総合判定

ある生徒が6種類の固体ア〜カについて次の実験を行い、金属か非金属かを調べた。

| 固体 | みがいたとき | 電気を通すか | たたいたとき |
|------|-----------|-----------|-----------|
| ア | 光った | 通した | うすく広がった |
| イ | 光らない | 通さない | くだけた |
| ウ | 光った | 通した | うすく広がった |
| エ | 光らない | 通した | くだけた |
| オ | 光った | 通した | うすく広がった |
| カ | 光らない | 通さない | 曲がった |

(1) 金属と判断できるものをすべて選びなさい。判断の根拠も答えよ。
→ ア・ウ・オ。金属光沢があり、電気を通し、展性（たたくと広がる）をもつという金属の性質をすべて満たしているから。

(2) 固体エは電気を通すが金属ではないと考えられる。このような物質の例を1つあげよ。
→ 炭素（黒鉛）。黒鉛は非金属だが電気を通す性質がある。

(3) 金属であるア・ウ・オをさらに区別するために密度を調べた。それぞれ2.7g/cm³、7.9g/cm³、8.9g/cm³であった。ア・ウ・オはそれぞれ何の金属と考えられるか。
→ ア（2.7g/cm³）＝アルミニウム、ウ（7.9g/cm³）＝鉄、オ（8.9g/cm³）＝銅。

(4) 金属か非金属かを判断するとき、「電気を通すかどうか」だけでは不十分である理由を、この実験の結果をもとに説明しなさい。
→ 固体エのように電気を通すが金属ではない物質（黒鉛など）が存在するため。金属かどうかの判断には、金属光沢・電気伝導性・展性・延性の4つの性質を総合的に確認する必要がある。
