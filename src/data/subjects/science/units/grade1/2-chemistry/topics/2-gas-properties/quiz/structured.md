# 気体の性質 大問小問

## 大問1: 酸素と二酸化炭素の発生実験（配点 20点）

酸素と二酸化炭素の発生実験について、次の問いに答えなさい。

(1) 酸素を発生させるために必要な薬品の組み合わせを答えよ。（3点）
→ 二酸化マンガンとうすい過酸化水素水（オキシドール）。

(2) (1)の実験で、二酸化マンガンはどのようなはたらきをしているか。「触媒」という語を使って説明せよ。（4点）
→ 二酸化マンガンは触媒としてはたらき、自分自身は変化せずに過酸化水素水の分解を速めている。

(3) 酸素が発生したことを確認する方法を答えよ。（3点）
→ 火のついた線香を入れると、激しく燃える。

(4) 二酸化炭素を発生させるために必要な薬品の組み合わせを答えよ。（3点）
→ 石灰石とうすい塩酸。

(5) 二酸化炭素が発生したことを確認する方法を答えよ。（3点）
→ 石灰水に通すと白くにごる。

(6) 二酸化炭素を水に溶かした水溶液にBTB溶液を加えると何色になるか。また、それは何性を示しているか。（4点）
→ 黄色になる。酸性を示している。

## 大問2: 水素の性質と確認（配点 16点）

水素の発生実験を行い、その性質を調べた。

(1) 水素を発生させるために必要な薬品の組み合わせを答えよ。（3点）
→ 亜鉛とうすい塩酸。

(2) 水素が発生したことを確認する方法を答えよ。（3点）
→ 火を近づけるとポンと音をたてて燃える。

(3) 水素が燃えるとできる物質の名前を答えよ。（3点）
→ 水。

(4) 水素はすべての気体の中でどのような特徴があるか、密度に着目して答えよ。（3点）
→ すべての気体の中で最も密度が小さい（最も軽い）。

(5) 水素を集めるのに最も適した方法を答え、その理由を説明しなさい。（4点）
→ 水上置換法。水素は水にとけにくいため使うことができ、空気が混じらず純粋な気体を集めることができるから。

## 大問3: 気体の集め方（配点 20点）

気体の集め方には3つの方法がある。次の問いに答えなさい。

(1) 気体の3つの集め方の名前をすべて答えよ。（3点）
→ 水上置換法・上方置換法・下方置換法。

(2) 水上置換法はどのような性質の気体に使えるか。（3点）
→ 水にとけにくい気体。

(3) 上方置換法はどのような性質の気体に使うか。（3点）
→ 空気より密度が小さい（軽い）気体。

(4) 下方置換法はどのような性質の気体に使うか。（3点）
→ 空気より密度が大きい（重い）気体。

(5) 水上置換法で気体を集めるとき、最初に出てくる気体を集めずに捨てるのはなぜか。（4点）
→ 最初に出てくる気体にはもともと装置の中にあった空気が混じっているため、純粋な気体を集めるには最初の分を捨てる必要がある。

(6) 次の気体を集めるのに最も適した方法をそれぞれ答えよ。①酸素 ②アンモニア ③二酸化炭素（4点）
→ ①水上置換法 ②上方置換法 ③下方置換法（水上置換法も可）。

## 大問4: アンモニアの噴水実験（配点 22点）

アンモニアを満たした丸底フラスコにフェノールフタレイン溶液を加えた水を入れたビーカーをつなぎ、スポイトで少量の水をフラスコ内に入れたところ、噴水のように水がフラスコ内に吹き上がり、水溶液が赤色になった。

(1) アンモニアを発生させるために必要な薬品の組み合わせと操作を答えよ。（4点）
→ 塩化アンモニウムと水酸化カルシウムを混ぜて加熱する。

(2) アンモニアを集めるのに上方置換法を使う理由を2つ答えよ。（4点）
→ ①空気より密度が小さい（軽い）ため。②水に非常にとけやすいため水上置換法が使えないため。

(3) フラスコ内に噴水が起きた理由を「気圧」という語を使って説明しなさい。（5点）
→ スポイトの水にアンモニアが一気にとけ、フラスコ内の気体が減って気圧が下がった。外の気圧との差で水がフラスコ内に押し上げられ、噴水になった。

(4) 噴水の水がフラスコ内で赤色になった理由を説明しなさい。（5点）
→ アンモニアが水にとけてアンモニア水（アルカリ性）になり、フェノールフタレイン溶液がアルカリ性で赤色に変色したため。

(5) フェノールフタレイン溶液は何性のとき何色に変色するか。（4点）
→ アルカリ性のとき赤色に変色する（中性・酸性では無色のまま）。
