# 物質の姿と状態変化 大問小問

## 大問1: ロウの状態変化実験（配点 18点）

ロウを加熱して液体にし、容器に流し込んで冷やす実験を行った。次の問いに答えなさい。

(1) ロウが液体から固体になったとき、表面はどのようになったか。（3点）
→ 表面が中央にへこんだ。

(2) (1)のようになった理由を「体積」という語を使って説明しなさい。（4点）
→ 液体から固体に変化すると粒子の間隔が縮まり、体積が小さくなるため、表面がへこんだ。

(3) ロウが固体になる前後で質量を測定した。質量はどうなるか。理由もあわせて答えなさい。（5点）
→ 質量は変化しない。状態変化では粒子の数が変わらないため、質量は一定に保たれる。

(4) 同じ実験を水で行った場合、凍ったときの体積はどうなるか。ロウとの違いを説明しなさい。（6点）
→ 水は凍ると体積が大きくなる。ロウは一般の物質と同じく固体になると体積が小さくなるが、水は氷の結晶構造がすき間の多い構造をとるため例外的に体積が増加する。

## 大問2: 加熱曲線の読み取り（配点 22点）

純粋な物質Xを加熱したところ、次のような温度変化が見られた。
- 0〜2分：温度が上昇（-10℃→0℃）
- 2〜5分：温度が0℃で一定
- 5〜8分：温度が上昇（0℃→100℃）
- 8〜12分：温度が100℃で一定
- 12分以降：温度が上昇

(1) 2〜5分の間に起こっている状態変化の名称を答えよ。（3点）
→ 融解。

(2) 8〜12分の間に起こっている状態変化の名称を答えよ。（3点）
→ 沸騰（蒸発）。

(3) 2〜5分の間に温度が一定になる理由を説明しなさい。（4点）
→ 加えた熱エネルギーが固体を液体に変える状態変化に使われ、温度の上昇に使われないため。

(4) 物質Xの融点と沸点をそれぞれ答えよ。（4点）
→ 融点：0℃、沸点：100℃。

(5) 物質Xは何か。根拠もあわせて答えなさい。（4点）
→ 水。融点が0℃、沸点が100℃であることから判断できる。

(6) 同じ実験を混合物で行った場合、加熱曲線はどのように異なるか説明しなさい。（4点）
→ 混合物では温度が一定になる部分（水平な部分）がなく、なだらかに温度が上昇し続ける。

## 大問3: 蒸留実験（配点 24点）

水とエタノールの混合物を蒸留で分離する実験を行った。次の問いに答えなさい。

(1) この実験で使うフラスコの名前を答えよ。（3点）
→ 枝つきフラスコ。

(2) フラスコに沸騰石を入れる理由を答えよ。（3点）
→ 突沸（急に激しく沸騰すること）を防ぐため。

(3) 温度計の球部はどの位置に置くか。理由もあわせて答えなさい。（5点）
→ 枝つきフラスコの枝の付け根の高さに置く。蒸気の温度を正確に測るため。

(4) 出てきた蒸気を液体に戻すためにはどうするか。（3点）
→ ガラス管を水（冷却水）で冷やす。

(5) 加熱を始めて最初に出てくる液体は何を多くふくむか。理由もあわせて答えなさい。（5点）
→ エタノールを多くふくむ。エタノールの沸点（約78℃）は水の沸点（100℃）より低いため、先に蒸発するから。

(6) 集めた液体がエタノールを多くふくむかどうかを確認する方法を1つ答えなさい。（5点）
→ 脱脂綿にしみこませて火をつけ、燃えるかどうかを確認する。エタノールを多くふくむ液体は燃える。

## 大問4: 粒子モデルと状態変化（配点 18点）

物質の状態変化を粒子のモデルで考える。次の問いに答えなさい。

(1) 固体・液体・気体のうち、粒子の間隔がもっとも広いのはどれか。（3点）
→ 気体。

(2) 固体から液体、液体から気体へと変化すると、粒子の運動はどのように変わるか。（4点）
→ 粒子の運動が激しくなり、粒子どうしの間隔が広がる。

(3) 状態変化で体積が変化する理由を粒子のモデルを使って説明しなさい。（5点）
→ 状態変化によって粒子の間隔が変わるため、物質全体の体積が変化する。固体→液体→気体と変化するにつれて粒子の間隔が広がり、体積が大きくなる。

(4) 状態変化で質量が変わらない理由を粒子のモデルを使って説明しなさい。（3点）
→ 状態変化では粒子そのものの種類や数は変わらないため、質量は一定に保たれる。

(5) 状態変化は化学変化か。理由もあわせて答えなさい。（3点）
→ 化学変化ではない。状態変化では物質の種類は変わらず、粒子の運動のようすが変わるだけだから。

## 大問5: 物質の融点・沸点の比較（配点 18点）

次の表は、いくつかの物質の融点と沸点を示している。

| 物質 | 融点(℃) | 沸点(℃) |
|------|---------|---------|
| 水 | 0 | 100 |
| エタノール | -115 | 78 |
| 鉄 | 1535 | 2750 |
| 酸素 | -218 | -183 |

(1) 20℃で液体の物質をすべて答えよ。（4点）
→ 水、エタノール。（融点が20℃以下かつ沸点が20℃以上の物質）

(2) 20℃で気体の物質を答えよ。（3点）
→ 酸素。（沸点が20℃より低いため）

(3) -120℃のとき、エタノールはどの状態か。（3点）
→ 固体。（エタノールの融点-115℃より低いため）

(4) 1600℃のとき、鉄はどの状態か。（3点）
→ 液体。（鉄の融点1535℃より高く、沸点2750℃より低いため）

(5) 酸素を液体にするにはどうすればよいか。温度を使って説明しなさい。（5点）
→ 酸素を-183℃以下に冷却すればよい。酸素の沸点は-183℃なので、それより温度を下げれば気体から液体に凝縮する。
