# 身のまわりの物質とその性質 大問小問

## 大問1: 物体と物質・金属の性質（配点 20点）

次の表は、身のまわりにあるものをまとめたものである。

| もの | 材料 |
|------|------|
| スプーン | ステンレス |
| 消しゴム | ゴム |
| 窓 | ガラス |
| 鍋 | アルミニウム |
| 定規 | プラスチック |

(1) 表の「もの」の欄の呼び方は、物体・物質のどちらか。（2点）
→ 物体。形や用途に注目した呼び方だから。

(2) 表の中から金属でできているものをすべて選び、その材料名を答えよ。（4点）
→ ステンレス（スプーン）とアルミニウム（鍋）。

(3) 金属に共通する性質を4つ答えよ。（4点）
→ ①みがくと光る（金属光沢）②電気や熱をよく通す③たたくとうすく広がる（展性）④引っ張ると細く伸びる（延性）。

(4) ゴム・ガラス・プラスチックのように金属以外の物質をまとめて何というか。（2点）
→ 非金属。

(5) アルミニウムは軽くてさびにくい性質がある。この性質を利用した製品を、鍋以外で2つ答えなさい。（4点）
→ 飲料缶・航空機の部品（1円硬貨・アルミホイルなども可）。

(6) 「みがくと光る」という性質は金属だけがもつ性質か。理由を含めて答えなさい。（4点）
→ 金属光沢は金属に共通する性質である。非金属のプラスチックなども表面を加工すれば光るが、これは金属光沢とは異なる。

## 大問2: 密度の計算と物質の特定（配点 22点）

次の表は、いくつかの金属の密度を示したものである。

| 金属 | 密度（g/cm³） |
|------|-------------|
| アルミニウム | 2.7 |
| 鉄 | 7.9 |
| 銅 | 8.9 |
| 金 | 19.3 |

ある金属のかたまりの質量を電子てんびんではかると71.2gであった。次にメスシリンダーに水を入れて体積を測定すると8.0cm³であった。

(1) 密度を求める公式を書きなさい。（2点）
→ 密度（g/cm³）＝質量（g）÷体積（cm³）。

(2) この金属の密度を求めなさい。（3点）
→ 71.2÷8.0＝8.9g/cm³。

(3) この金属は表の中のどの金属か。（2点）
→ 銅。

(4) メスシリンダーで水の体積を読むとき、目の高さと読む位置について正しい方法を説明せよ。（4点）
→ 目の高さを液面と同じ水平の位置に合わせ、液面の最も低い部分（へこんだ部分）の目盛りを読む。

(5) メスシリンダーの最小目盛りが1mLのとき、何mLの位まで読み取るか。（3点）
→ 0.1mLの位まで（最小目盛りの10分の1まで目分量で読む）。

(6) 密度2.7g/cm³の金属と密度0.9g/cm³の液体がある。この金属を液体に入れるとどうなるか。理由もあわせて答えよ。（4点）
→ 金属は沈む。金属の密度（2.7g/cm³）が液体の密度（0.9g/cm³）より大きいから。

(7) 体積が同じ10cm³のアルミニウムと鉄では、どちらが重いか。それぞれの質量を求めて答えよ。（4点）
→ アルミニウム：2.7×10＝27g、鉄：7.9×10＝79g。鉄の方が重い。

## 大問3: 白い粉末の見分け方（配点 20点）

4種類の白い粉末A〜Dについて、次の実験を行った。A〜Dは砂糖・食塩・デンプン・グラニュー糖のいずれかである。

【実験1】粒のようすを虫めがねで観察した。
【実験2】水に入れてかき混ぜた。
【実験3】ガスバーナーで加熱した。

| | 粒のようす | 水への溶け方 | 加熱したとき |
|---|----------|-----------|-----------|
| A | 細かい白い粉 | 溶けない | 黒くこげた |
| B | 角ばった透明な粒 | よく溶けた | 黒くこげた |
| C | 立方体の白い粒 | よく溶けた | 変化なし |
| D | 不規則な白い粒 | よく溶けた | 黒くこげた |

(1) 加熱したとき黒くこげた物質は有機物・無機物のどちらか。（2点）
→ 有機物。

(2) 有機物を加熱したとき黒くこげるのはなぜか。「炭素」という語を使って説明せよ。（4点）
→ 有機物にふくまれる炭素が残って炭になるから。

(3) 物質Cは何か。判断した理由を2つ答えよ。（6点）
→ 食塩。理由：①水によく溶ける②加熱しても変化がない（無機物である）。

(4) 物質Aは何か。判断した理由を答えよ。（4点）
→ デンプン。水に溶けず、加熱すると黒くこげる（有機物で水に溶けない）から。

(5) 物質BとDはどちらが砂糖でどちらがグラニュー糖か。見分けた根拠を答えよ。（4点）
→ Bがグラニュー糖、Dが砂糖。グラニュー糖は角ばった透明な結晶だが、砂糖（上白糖）は不規則な白い粒であることから判断できる。

## 大問4: ガスバーナーの操作（配点 18点）

次の図はガスバーナーの各部の名称を示したものである。

(1) ガスバーナーに火をつける正しい手順を、次のア〜オを正しい順番に並べなさい。（5点）
ア 空気調節ねじで炎を調節する
イ ガス調節ねじをゆるめて点火する
ウ 元栓を開ける
エ マッチに火をつけて近づける
オ コックを開ける
→ ウ→オ→エ→イ→ア。

(2) マッチとガスではどちらを先に用意するか。その理由を答えよ。（4点）
→ マッチに先に火をつける。ガスを先に出すとガスがたまり、引火する危険があるから。

(3) 炎がオレンジ色のとき、何が不足しているか。また、どのねじを操作して調節するか。（4点）
→ 空気（酸素）が不足している。空気調節ねじを少しずつ開いて調節する。

(4) ガスバーナーの火を消す手順を答えなさい。（5点）
→ ①空気調節ねじを閉める②ガス調節ねじを閉める③コックを閉める④元栓を閉める。（つけるときの逆順）

## 大問5: 有機物と無機物の分類（配点 20点）

次の物質を有機物と無機物に分類しなさい。
砂糖、鉄、エタノール、ガラス、ろう、食塩、デンプン、水、プラスチック、アルミニウム

(1) 有機物をすべて選びなさい。（4点）
→ 砂糖・エタノール・ろう・デンプン・プラスチック。

(2) 無機物をすべて選びなさい。（4点）
→ 鉄・ガラス・食塩・水・アルミニウム。

(3) 有機物に共通する特徴を「炭素」「二酸化炭素」という語を使って説明せよ。（4点）
→ 有機物は炭素をふくんでおり、燃やすと二酸化炭素が発生する。

(4) 有機物を燃やしたとき、二酸化炭素のほかに発生する物質は何か。確認方法もあわせて答えよ。（4点）
→ 水（水蒸気）。冷えた金属板やガラスを炎に近づけるとくもる（水滴がつく）ことで確認できる。

(5) 二酸化炭素が発生したことを確認する方法を答えなさい。（4点）
→ 発生した気体を石灰水に通すと白くにごる。
