# 顕微鏡の詳細 大問小問

## 大問1: 顕微鏡の操作（配点 22点）

次の図は顕微鏡の各部分を示したものである。A〜Fは各部分の名称を表している。

(1) A〜Fの名称を答えよ。（6点）
→ A：接眼レンズ　B：鏡筒　C：レボルバー　D：対物レンズ　E：ステージ　F：反射鏡

(2) レンズを取り付ける正しい順番を答え、その理由を説明しなさい。（4点）
→ 接眼レンズを先に取り付け、次に対物レンズを取り付ける。鏡筒の中にほこりが入るのを防ぐため。

(3) 観察を始めるとき、倍率はどのようにするか。理由もあわせて答えなさい。（4点）
→ 低倍率から始める。視野が広く、観察するものを見つけやすいため。

(4) ピントを合わせる手順を説明しなさい。（4点）
→ 横から見ながら調節ねじを回してステージ（または鏡筒）と対物レンズを近づけ、その後接眼レンズをのぞきながら少しずつ離してピントを合わせる。

(5) 視野が暗いとき、明るくする方法を2つ答えなさい。（4点）
→ ①しぼりを開いて光の量を増やす。②反射鏡を凹面鏡に変えて多くの光を集める。

## 大問2: プレパラートと倍率（配点 22点）

顕微鏡で水中の微生物を観察するため、プレパラートを作成した。この顕微鏡には接眼レンズ（10倍・15倍）と対物レンズ（4倍・10倍・40倍）がある。

(1) プレパラートは何と何を使って作るか。（3点）
→ スライドガラスとカバーガラス。

(2) カバーガラスをかぶせるとき、気泡が入らないようにするにはどうするか。（4点）
→ カバーガラスの端をスライドガラスにつけて、ゆっくりと静かに下ろす。

(3) 接眼レンズ10倍、対物レンズ4倍のとき、顕微鏡の倍率は何倍か。（3点）
→ 40倍。倍率＝接眼レンズの倍率×対物レンズの倍率＝10×4＝40。

(4) この顕微鏡で最も高い倍率は何倍か。そのときのレンズの組み合わせを答えよ。（4点）
→ 600倍。接眼レンズ15倍×対物レンズ40倍＝600倍。

(5) 倍率を40倍から400倍に上げたとき、視野の広さと明るさはそれぞれどう変化するか。（4点）
→ 視野は狭くなり、明るさは暗くなる。

(6) 倍率を上げて暗くなったとき、視野を明るくするためにはどうすればよいか。（4点）
→ しぼりを開く、または反射鏡を凹面鏡に変えて光を多く集める。
