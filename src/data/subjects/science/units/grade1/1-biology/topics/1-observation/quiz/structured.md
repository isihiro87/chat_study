# 観察のしかた 大問小問

## 大問1: 観察記録とスケッチ（配点 18点）

中学1年生の理科の授業で、校庭に出て身近な生物を観察することになった。生徒は観察記録用紙にスケッチと記録をまとめた。

(1) 観察記録に書くべき環境の情報を4つ答えよ。（4点）
→ 日時・場所・天気・気温。

(2) 理科のスケッチのルールとして正しいものを次からすべて選べ。（4点）
ア 太い線で力強く描く
イ 細い1本の線で輪郭を描く
ウ 影をつけて立体感を出す
エ ぬりつぶさない
オ 背景も描いて場所がわかるようにする
→ イ・エ。理科のスケッチは細い線で輪郭を描き、影をつけず、ぬりつぶさず、背景や器具の視野の線は描かない。

(3) スケッチで影をつけてはいけない理由を答えなさい。（4点）
→ 影は観察者によって異なり、生物の形を正確に記録できなくなるから。

(4) 美術のデッサンと理科のスケッチの違いを、「影」「線」という語を使って説明しなさい。（6点）
→ 美術のデッサンは影をつけて立体感を出し、太さの異なる線を使って表現するが、理科のスケッチは影をつけず、細い1本の線で輪郭を正確に描く。

## 大問2: ルーペの使い方（配点 16点）

次の図は、生徒がルーペを使って花を観察している様子である。

(1) ルーペで観察するとき、ルーペをどこに持つのが正しいか。（3点）
→ 目に近づけて持つ。

(2) ピントを合わせるにはどうするか。（3点）
→ 観察するものを前後に動かす。

(3) 花壇に咲いている花のように動かせないものを観察するとき、どのようにしてピントを合わせるか。（4点）
→ ルーペを目に近づけたまま、自分の顔（体）を前後に動かしてピントを合わせる。

(4) ルーペで太陽を見てはいけない理由を説明しなさい。（3点）
→ レンズで光が集められて目に入り、目を傷めるおそれがあるから。

(5) ルーペを目から離して使うと不便な理由を答えなさい。（3点）
→ 視野が狭くなり、ピントも合わせにくくなるため。

## 大問3: 生物の分類（配点 22点）

校庭で見つけた次の10種類の生物を分類しなさい。
タンポポ、チョウ、コケ、サクラ、カエル、シダ、ダンゴムシ、スギナ、アリ、ツツジ

(1) 「移動するか、しないか」で2つのグループに分けなさい。（4点）
→ 移動する：チョウ・カエル・ダンゴムシ・アリ。移動しない：タンポポ・コケ・サクラ・シダ・スギナ・ツツジ。

(2) 移動しないグループをさらに「花が咲くか、咲かないか」で分けなさい。（4点）
→ 花が咲く：タンポポ・サクラ・ツツジ。花が咲かない：コケ・シダ・スギナ。

(3) 分類するときに基準にするのは、どのような特徴がよいか。理由もあわせて答えなさい。（6点）
→ 体のつくりなど変化しにくい特徴がよい。花の色や大きさなどは環境や個体によって変わりやすいため、安定した基準にならないから。

(4) 「花の色」を分類の基準にするのは適切か。その理由を答えなさい。（4点）
→ 適切ではない。花の色は環境や品種によって変化しやすく、安定した基準にならないから。

(5) 分類の基準を変えると、同じ生物でも違うグループに入ることがある。このことから、分類について言えることを説明しなさい。（4点）
→ 分類の基準は1つではなく、目的に応じてさまざまな基準で分類できる。どの基準を選ぶかによってグループ分けの結果が変わる。
