## 画像生成プロンプト一覧

以下は、SVGでは表現しきれない写実的・複雑な図のための画像生成プロンプトです。

### プロンプト1: `electrolysis-photo.png`（トピック1用）
**対応問題**: 大問1、大問2（参考画像）

```
中学理科の教科書に使われるような、塩化銅水溶液の電気分解実験装置のイラスト。
白い背景。ビーカーに青色の水溶液（塩化銅水溶液）が入っている。
2本の炭素棒の電極が水溶液に差し込まれ、導線で電源装置につながっている。
一方の電極（陰極）には赤褐色の銅が付着しており、もう一方の電極（陽極）からは小さな気泡（塩素）が発生している。
電源装置には「＋」「−」の表示がある。シンプルで見やすい教育用イラスト。
日本の中学校理科の雰囲気。注釈やテキストは不要。
```

### プロンプト5: `microplate-experiment.png`（トピック3用）
**対応問題**: 大問3（参考画像）

```
中学理科の金属のイオン化傾向を調べるマイクロプレート実験のイラスト。白い背景。
白い6穴マイクロプレートに、3種類の水溶液（硫酸銅水溶液=青色、硫酸マグネシウム水溶液=無色、硫酸亜鉛水溶液=無色）が入っている。
それぞれの穴に小さな金属片（銅、マグネシウム、亜鉛）が入れられている。
一部の穴では金属片の表面に色の変化（赤色の銅が付着）が見られ、
一部の穴では変化がない状態。
俯瞰で見たイラスト。シンプルで見やすい教育用イラスト。テキスト不要。
```

### プロンプト6: `daniel-cell-photo.png`（トピック3用）

中学理科のダニエル電池の実験装置イラスト。白い背景。

### プロンプト7: `fuel-cell-diagram.png`（トピック3用）
**対応問題**: 発展5 全問

```
燃料電池のしくみを示す教育用イラスト。白い背景。
左側から水素ガス（H₂）のボンベ、右側から酸素ガス（O₂）のボンベがそれぞれ電極に供給されている。
中央に電解質膜があり、その両側に電極がある。
水素側が負極、酸素側が正極。
導線で電球やモーターにつながっている。
排出口からは水（H₂O）の水滴が出ている。
矢印で水素→電子→酸素→水の流れを示す。
シンプルで見やすい教育用イラスト。テキスト不要。
```
