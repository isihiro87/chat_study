# 一問一答：いろいろな電池

## ダニエル電池の構造

Q1. ダニエル電池の−極に使われる金属板は何か。
A1. 亜鉛板（Zn板）

Q2. ダニエル電池の＋極に使われる金属板は何か。
A2. 銅板（Cu板）

Q3. ダニエル電池で−極側の水溶液は何か。
A3. 硫酸亜鉛水溶液（ZnSO₄水溶液）

Q4. ダニエル電池で＋極側の水溶液は何か。
A4. 硫酸銅水溶液（CuSO₄水溶液）

Q5. ダニエル電池で2つの水溶液を仕切るものは何か。
A5. セロハン膜

Q6. ダニエル電池のセロハン膜の役割を答えなさい。
A6. 2つの水溶液の混合を防ぎながら、イオンを通す

Q7. セロハン膜を通るものは何か。
A7. 水の分子や一部のイオン

Q8. ダニエル電池がうすい塩酸の電池より優れている点は何か。
A8. ＋極に水素の気泡が付着する問題がなく、電圧が安定する

## ダニエル電池の反応

Q9. ダニエル電池の−極で起こる反応をイオン式で書きなさい。
A9. Zn → Zn²⁺ + 2e⁻

Q10. ダニエル電池の＋極で起こる反応をイオン式で書きなさい。
A10. Cu²⁺ + 2e⁻ → Cu

Q11. ダニエル電池で電流を流し続けると、Zn板はどうなるか。理由も答えなさい。
A11. 小さくなる。亜鉛原子がZn²⁺として溶け出すため

Q12. ダニエル電池で電流を流し続けると、Cu板はどうなるか。理由も答えなさい。
A12. 大きくなる。Cu²⁺が電子を受け取って銅としてCu板の表面に付着するため

Q13. ダニエル電池で電流を流し続けると、硫酸銅水溶液の青色はどうなるか。
A13. うすくなる（Cu²⁺が減少するため）

Q14. ダニエル電池で電子が流れる向きを答えなさい。
A14. −極（Zn板）から導線を通って＋極（Cu板）へ

Q15. ダニエル電池で電流が流れる向きを答えなさい。
A15. ＋極（Cu板）から−極（Zn板）へ（電子の流れと逆）

## 一次電池と二次電池

Q16. 使い切りで充電できない電池を何というか。
A16. 一次電池

Q17. 充電して繰り返し使える電池を何というか。
A17. 二次電池

Q18. マンガン乾電池は一次電池か二次電池か。
A18. 一次電池（使い切り）

Q19. アルカリ乾電池は一次電池か二次電池か。
A19. 一次電池（使い切り）

Q20. リチウム電池は一次電池か二次電池か。
A20. 一次電池（使い切り）

Q21. リチウムイオン電池は一次電池か二次電池か。
A21. 二次電池（充電可能）

Q22. 鉛蓄電池は一次電池か二次電池か。
A22. 二次電池（充電可能）

Q23. ニッケル水素電池は一次電池か二次電池か。
A23. 二次電池（充電可能）

Q24. 「充電」とは何をすることか説明しなさい。
A24. 外部から電流を流し、放電時と逆向きの化学反応を起こしてもとの物質に戻す操作

Q25. 電池を使い続けると電圧が低下する現象を何というか。
A25. 放電

Q26. リチウムイオン電池が主に使われている身近な製品を1つ挙げなさい。
A26. スマートフォン（ノートパソコン、タブレットなども可）

Q27. 鉛蓄電池が主に使われている身近なものを1つ挙げなさい。
A27. 自動車のバッテリー

Q28. 一次電池の例を3つ挙げなさい。
A28. マンガン乾電池、アルカリ乾電池、リチウム電池

Q29. 二次電池の例を3つ挙げなさい。
A29. リチウムイオン電池、鉛蓄電池、ニッケル水素電池

## 燃料電池

Q30. 水素と酸素を反応させて電気エネルギーを取り出す電池を何というか。
A30. 燃料電池

Q31. 燃料電池の化学反応式を書きなさい。
A31. 2H₂ + O₂ → 2H₂O

Q32. 燃料電池の反応で生成される物質は何か。
A32. 水（H₂O）

Q33. 燃料電池が環境にやさしいと言われる理由を答えなさい。
A33. 反応で生成されるのが水のみで、CO₂などの温室効果ガスを排出しないから

Q34. 燃料電池は、ある化学反応の逆の反応を利用している。その化学反応は何か。
A34. 水の電気分解（2H₂O → 2H₂ + O₂）

Q35. 水の電気分解の化学反応式を書きなさい。
A35. 2H₂O → 2H₂ + O₂

Q36. 水の電気分解はどのようなエネルギー変換か。
A36. 電気エネルギーを化学エネルギーに変換する

Q37. 燃料電池はどのようなエネルギー変換か。
A37. 化学エネルギーを電気エネルギーに変換する

Q38. 水の電気分解と燃料電池は「逆の反応」であると言える理由を説明しなさい。
A38. 電気分解は水を水素と酸素に分解し（2H₂O → 2H₂ + O₂）、燃料電池は水素と酸素から水を生成する（2H₂ + O₂ → 2H₂O）ため、反応が逆向きだから

Q39. 燃料電池を利用した自動車を何というか。
A39. 燃料電池自動車（FCV）

Q40. 二次電池を充電するとき、電池の内部ではどのような変化が起きているか。
A40. 放電時とは逆向きの化学反応が起こり、消費された物質がもとに戻る
