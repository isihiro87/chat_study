# 生物の成長と生殖 大問小問

## 大問1: 細胞分裂と生物の成長（配点 20点）

タマネギの種子を水に浸して発根させ、伸びてきた根の成長のようすを調べた。根の先端から2mmごとに印をつけ、2日後に印の間隔を測定したところ、先端に近い部分の間隔は大きく広がっていたが、根もとに近い部分の間隔はほとんど変化していなかった。

(1) この結果から、根のどの部分で成長がさかんに行われていると考えられるか。（3点）
→ 先端に近い部分。

(2) 先端に近い部分の細胞と根もとに近い部分の細胞を顕微鏡で比較した。細胞の大きさと数について、それぞれどのような違いが見られるか。（4点）
→ 先端に近い部分は小さい細胞が多く見られ、根もとに近い部分は大きい細胞が見られる。

(3) 先端に近い部分で小さい細胞が多く見られる理由を、「細胞分裂」という語を使って説明しなさい。（4点）
→ 先端に近い部分ではさかんに細胞分裂が行われているため、分裂したばかりの小さな細胞が多く見られる。

(4) 生物の成長は、2つの過程によって起こる。その2つの過程を答えなさい。（4点）
→ 細胞分裂によって細胞の数がふえることと、ふえた細胞がそれぞれ大きくなること。

(5) からだをつくる細胞が分裂する細胞分裂を特に何というか。（2点）
→ 体細胞分裂。

(6) 体細胞分裂のとき、核の中に現れるひものようなものを何というか。（3点）
→ 染色体。

## 大問2: 体細胞分裂の観察（配点 20点）

タマネギの根の先端を使って、体細胞分裂のようすを観察する実験を行った。次はその手順である。

【手順】
① 塩酸と染色液の混合液にタマネギの根をしばらくひたす。
② 根を水で洗い、スライドガラスにのせる。
③ 先端の部分だけを切りとる。
④ 染色液をたらしてしばらく置く。
⑤ カバーガラスをかけ、その上からおしつぶす。
⑥ 顕微鏡で観察する。

(1) 手順①で塩酸を使う目的を答えなさい。（4点）
→ 細胞の1つ1つを離れやすくするため。

(2) 手順③で先端の部分だけを切りとるのはなぜか。（4点）
→ 細胞分裂がさかんに行われているのは先端に近い部分であり、分裂中の細胞を観察しやすいため。

(3) 手順④で使う染色液として適切なものを2つ答え、それぞれ何色に染まるか答えなさい。（4点）
→ 酢酸カーミン（赤色に染まる）、酢酸ダーリアバイオレット（紫色に染まる）。

(4) 手順⑤でカバーガラスの上からおしつぶす目的を答えなさい。（4点）
→ 根の細胞を薄く広げて、細胞が重なり合わないようにし、1つ1つの細胞を観察しやすくするため。

(5) 観察の結果、分裂中の細胞では染色体がさまざまな状態で見えた。体細胞分裂では、染色体はどのように変化するか、「複製」「両端」の語を使って説明しなさい。（4点）
→ 染色体が複製されて数が2倍になり、それぞれが細胞の両端に分かれて移動し、2つの核ができて細胞が2つに分かれる。

## 大問3: 無性生殖（配点 20点）

次の生物の生殖について、あとの問いに答えなさい。

ア：ゾウリムシ　イ：ジャガイモ　ウ：アメーバ　エ：サクラ（種子）　オ：オランダイチゴ　カ：コダカラベンケイ

(1) ア〜カのうち、無性生殖でふえるものをすべて選び、記号で答えなさい。（4点）
→ ア、イ、ウ、オ、カ

(2) アやウのように、からだが2つに分かれてふえる無性生殖の方法を何というか。（3点）
→ 分裂。

(3) イ、オ、カのように、植物のからだの一部から新しい個体ができる無性生殖を何というか。（3点）
→ 栄養生殖。

(4) ジャガイモはからだのどの部分から新しい個体をつくるか。（3点）
→ いも（地下の茎）。

(5) 無性生殖でできた子の形質は親と比べてどうなるか。その理由を「体細胞分裂」の語を使って答えなさい。（4点）
→ 親と同じ形質になる。無性生殖では体細胞分裂によって子がつくられるため、親の染色体がそのまま受けつがれるから。

(6) 農業で接ぎ木が利用されることがある。接ぎ木が無性生殖に分類される理由を簡潔に答えなさい。（3点）
→ 受精を行わずに、植物のからだの一部から新しい個体をつくる方法だから。

## 大問4: 有性生殖と発生（配点 20点）

次の図は、動物（カエル）と被子植物の有性生殖の過程を模式的に示したものである。

【動物（カエル）】
雌の（ A ）→ 卵 ＋ 雄の（ B ）→ 精子 → （ C ） → 受精卵 → （ D ） → 個体

【被子植物】
花粉 →（ E ）が伸びる → 精細胞と（ F ）が受精 → 受精卵 → （ D ） → 種子

(1) A〜Fにあてはまる語をそれぞれ答えなさい。（12点）
→ A：卵巣　B：精巣　C：受精　D：胚　E：花粉管　F：卵細胞

(2) 受精卵が細胞分裂をくり返して、からだのつくりが完成していく過程を何というか。（3点）
→ 発生。

(3) 被子植物で、卵細胞がふくまれている部分を何というか。（2点）
→ 胚珠。

(4) 有性生殖でできた子の形質が、親とまったく同じにならない理由を「染色体」の語を使って答えなさい。（3点）
→ 有性生殖では両方の親から染色体を半分ずつ受けつぐため、両親とは異なる染色体の組み合わせになり、親とまったく同じ形質にはならない。

## 大問5: 染色体と減数分裂（配点 20点）

ある動物の体細胞の染色体の数は8本である。この動物の生殖について考える。

(1) この動物が無性生殖を行った場合、子の体細胞の染色体の数は何本になるか。（3点）
→ 8本。

(2) この動物が有性生殖を行う場合、生殖細胞（卵や精子）がつくられるときに行われる特別な細胞分裂を何というか。（3点）
→ 減数分裂。

(3) この動物の生殖細胞の染色体の数は何本か。（3点）
→ 4本。

(4) 受精卵の染色体の数は何本か。その理由もあわせて答えなさい。（5点）
→ 8本。卵の4本と精子の4本が受精によって合わさるため、もとの体細胞と同じ8本にもどる。

(5) 品質のよいイチゴを大量に生産したいとき、有性生殖と無性生殖のどちらを利用するのが適切か。理由もあわせて答えなさい。（3点）
→ 無性生殖。親と同じ形質の子ができるため、品質のよいイチゴをそのまま安定してふやすことができる。

(6) 病気に強い新しい品種の作物をつくりたいとき、どちらの生殖を利用するか。理由もあわせて答えなさい。（3点）
→ 有性生殖。異なる形質をもつ親どうしをかけ合わせることで、両親の形質を組み合わせた新しい品種をつくることができるため。
