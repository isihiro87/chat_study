# 遺伝の規則性と遺伝子 発展問題

## 問1: 多世代の交配と遺伝子の組み合わせ

エンドウの種子の形について、丸形の遺伝子をA、しわ形の遺伝子をaとする。丸形の純系（AA）としわ形の純系（aa）をかけ合わせて子を得た。その後、子のエンドウ（Aa）としわ形の純系（aa）をかけ合わせた。

(1) 子（Aa）から作られる生殖細胞がもつ遺伝子をすべて答えなさい。
→ Aをもつ生殖細胞とaをもつ生殖細胞の2種類。

(2) しわ形の純系（aa）から作られる生殖細胞がもつ遺伝子を答えなさい。
→ aのみ。

(3) 子（Aa）としわ形の純系（aa）をかけ合わせたときの、次の表を完成させなさい。

|  | A | a |
|:-:|:-:|:-:|
| a | ア | イ |

→ ア：Aa　イ：aa

(4) このかけ合わせで得られる個体の表現型の比を答えなさい。
→ 丸形:しわ形 = 1:1

## 問2: 未知の遺伝子型の推定

ある植物の花の色には赤色と白色があり、赤色が顕性形質、白色が潜性形質である。赤色の遺伝子をR、白色の遺伝子をrとする。赤色の花をつける個体Xの遺伝子の組み合わせがRRかRrかわからない。

(1) 個体Xの遺伝子の組み合わせを調べるために、どのような個体とかけ合わせればよいか。その個体の遺伝子の組み合わせを答えなさい。
→ 白色の花をつける個体（rr）とかけ合わせる。

(2) 個体XがRRの場合、白色（rr）とかけ合わせると子の表現型はどうなるか。
→ すべて赤色（Rr）。

(3) 個体XがRrの場合、白色（rr）とかけ合わせると子の表現型の比はどうなるか。
→ 赤色:白色 = 1:1

(4) 個体Xと白色（rr）をかけ合わせたところ、子に白色の花が現れた。個体Xの遺伝子の組み合わせは何か。理由もあわせて答えなさい。
→ Rr。子に白色（rr）が現れるためには、個体Xからrの遺伝子が渡される必要があり、個体XはRrでなければならない。

## 問3: 2つの形質の遺伝

エンドウの種子の形（丸形A・しわ形a）と子葉の色（黄色B・緑色b）の2つの形質に注目する。丸形・黄色の純系（AABB）としわ形・緑色の純系（aabb）をかけ合わせた。

(1) 子の遺伝子の組み合わせと表現型を答えなさい。
→ 遺伝子の組み合わせ：AaBb。表現型：丸形・黄色。

(2) 子（AaBb）を自家受粉させたとき、孫に現れる表現型の組み合わせをすべて答えなさい。
→ 丸形・黄色、丸形・緑色、しわ形・黄色、しわ形・緑色の4種類。

(3) 孫における丸形・黄色:丸形・緑色:しわ形・黄色:しわ形・緑色の比を答えなさい。
→ 9:3:3:1

(4) 孫の中で、しわ形・緑色の個体の遺伝子の組み合わせを答えなさい。
→ aabb

## 問4: 実験結果の考察

ある生徒がマルバアサガオの花の色について実験を行った。赤い花の純系と白い花の純系をかけ合わせたところ、子はすべてピンクの花になった。さらに、子のピンクの花同士をかけ合わせたところ、孫には赤い花:ピンクの花:白い花 = 1:2:1の比で現れた。

(1) この結果は、メンデルの実験（顕性形質:潜性形質 = 3:1）とどのように異なるか説明しなさい。
→ メンデルの実験では顕性形質が潜性形質に対して完全に優性を示すが、この実験では赤と白の中間であるピンクが現れており、どちらの遺伝子も完全には優性を示していない。

(2) 赤い花の遺伝子をR、白い花の遺伝子をrとしたとき、ピンクの花の遺伝子の組み合わせを答えなさい。
→ Rr

(3) このような遺伝の仕方を、メンデルの実験の場合と区別して何というか。
→ 不完全優性（不完全顕性）

(4) ピンクの花（Rr）と白い花（rr）をかけ合わせると、子の表現型の比はどうなるか。理由もあわせて答えなさい。
→ ピンクの花:白い花 = 1:1。Rrからの生殖細胞はRとaが半分ずつ、rrからはすべてrなので、子はRr（ピンク）とrr（白）が同数生じる。

## 問5: 遺伝子技術と社会への影響

遺伝子の本体であるDNAに関する技術は、農業や医療の分野で広く活用されている。次の問いに答えなさい。

(1) 遺伝子組換え技術を使って、害虫に強い作物が作られている。この技術では、具体的にどのようなことを行っているか。「遺伝子」という語を使って説明しなさい。
→ 害虫に対する抵抗力をもつ別の生物の遺伝子を、作物の細胞に導入する。

(2) 遺伝子組換え技術によってインスリンが生産されている。インスリンはどのような病気の治療に使われるか。
→ 糖尿病。

(3) 自然界には存在しない青色のバラが遺伝子組換え技術で作られた。このことが可能になった理由を、「遺伝子」という語を使って説明しなさい。
→ バラがもともと持っていない青色の色素を作る遺伝子を、他の植物から取り出してバラに導入したため。

(4) 遺伝子組換え食品に対しては賛否がある。遺伝子組換え食品の利点と懸念される問題点をそれぞれ1つずつ答えなさい。
→ 利点：病害虫に強く農薬の使用量を減らせるなど、安定した食料生産に貢献できる。問題点：生態系への予期しない影響や、長期的な安全性が十分に確認されていないという懸念がある。

(5) クローン技術について、「同一の遺伝子」という語を使って説明し、この技術の農業分野での活用例を1つ答えなさい。
→ クローンとは、起源が同じで同一の遺伝子をもつ個体の集団のことである。農業分野では、品質の良い果物の木を挿し木で増やすことなどに活用されている。
