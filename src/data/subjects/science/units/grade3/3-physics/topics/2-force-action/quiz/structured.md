# 力のはたらき方 大問小問

## 大問1: 力の合成と分解（配点 20点）

ばねばかりA、Bを使って、1つの点Oにそれぞれ異なる方向から力を加え、点Oを一定の位置に引く実験を行った。次に、ばねばかり1本だけで同じ位置に点Oを引くことができる力Fを求めた。

(1) 力Fのように、2つの力と同じはたらきをする1つの力を何というか。（2点）
→ 合力。

(2) 2つの力から力Fを求めることを何というか。（2点）
→ 力の合成。

(3) ばねばかりA、Bの2力が角度をもってはたらくとき、合力を作図で求めるにはどのような法則を使うか。（3点）
→ 平行四辺形の法則。

(4) 一直線上で同じ向きに4Nと6Nの力がはたらくとき、合力の大きさを求めなさい。（3点）
→ 10N。

(5) 一直線上で逆向きに3Nと8Nの力がはたらくとき、合力の大きさと向きを求めなさい。（4点）
→ 大きさ5Nで、8Nの力の向き。

(6) 斜面上に置いた物体にはたらく重力を2つの方向に分解するとき、どの方向とどの方向に分解するか。（3点）
→ 斜面に平行な方向と斜面に垂直な方向。

(7) 斜面の傾きを大きくしたとき、斜面に平行な分力はどう変化するか。また、そのことから物体の運動はどうなるか。（3点）
→ 斜面に平行な分力が大きくなり、物体の加速が大きくなる（物体がより速く滑り落ちる）。

## 大問2: 慣性の法則（配点 20点）

次の文を読んで、あとの問いに答えなさい。

電車に乗っている人が立っているとき、電車が急にブレーキをかけると体が（ ア ）に倒れそうになる。また、電車が急に発車すると体が（ イ ）に倒れそうになる。これは、物体がそれまでの（ ウ ）を続けようとする性質によるものである。

(1) ア、イにあてはまる語をそれぞれ答えなさい。（4点）
→ ア：前　イ：後ろ

(2) ウにあてはまる語を答えなさい。（2点）
→ 運動状態

(3) 下線部の性質を何というか。（3点）
→ 慣性。

(4) この性質に関する法則の名前を答えなさい。（3点）
→ 慣性の法則。

(5) 慣性の法則について、静止している物体と運動している物体に分けて説明しなさい。（4点）
→ 静止している物体は力がはたらかない限り静止し続け、運動している物体は力がはたらかない限り等速直線運動を続ける。

(6) だるま落としで、たたいた段だけが飛び出し、上のだるまがその場にとどまるのはなぜか。「慣性」の語を使って説明しなさい。（4点）
→ 上のだるまには慣性があり、静止している状態を続けようとするため、下の段がたたかれて飛び出しても上のだるまはそのまま静止し続けようとして、その場にとどまるから。

## 大問3: 作用・反作用の法則（配点 20点）

AさんとBさんがスケートリンクの上で向かい合い、AさんがBさんの手を押した。すると、Aさん自身も後ろに動いた。

(1) AさんがBさんを押す力に対して、BさんからAさんにはたらく力を何というか。（3点）
→ 反作用（の力）。

(2) AさんがBさんを押す力と、BさんからAさんにはたらく力の関係について、大きさと向きを答えなさい。（4点）
→ 大きさは同じで、向きは逆（反対）である。

(3) この法則の名前を答えなさい。（3点）
→ 作用・反作用の法則。

(4) 机の上にある本には重力と垂直抗力がはたらいてつり合っている。この2力と作用・反作用の2力の違いを、「物体」の語を使って説明しなさい。（5点）
→ つり合いの2力は同じ物体（本）にはたらくが、作用・反作用の2力は別々の物体にはたらく。

(5) 壁を手で押しても壁が動かないことがある。このとき、作用・反作用の法則は成り立っているか。理由をあわせて答えなさい。（5点）
→ 成り立っている。壁を押す力（作用）に対して、壁から手に同じ大きさの力（反作用）がはたらいている。壁が動かないのは、壁にはたらく力がつり合っているためであり、作用・反作用の法則とは別の問題である。

## 大問4: 浮力の実験と計算（配点 20点）

物体Xをばねばかりにつるして測定する実験を行った。空気中でのばねばかりの値は3.0Nであった。次に、物体Xを完全に水中に沈めたところ、ばねばかりの値は2.2Nになった。

(1) 物体Xにはたらく浮力の大きさを求めなさい。（3点）
→ 0.8N。

(2) 浮力の大きさの求め方を式で表しなさい。（3点）
→ 浮力 = 空気中でのばねばかりの値 − 水中でのばねばかりの値。

(3) 物体Xをさらに深く沈めた場合、浮力の大きさはどうなるか。理由もあわせて答えなさい。（4点）
→ 変わらない。浮力の大きさは水中にある物体の体積で決まり、完全に水中に沈んでいれば深さには関係しないため。

(4) 物体Xと同じ体積で質量が2倍の物体Yを水中に完全に沈めた。物体Yにはたらく浮力の大きさはいくらか。理由もあわせて答えなさい。（4点）
→ 0.8N。浮力の大きさは物体の体積で決まり、質量（重さ）には関係しないため、体積が同じであれば浮力も同じである。

(5) 物体Xの2倍の体積をもつ物体Zを水中に完全に沈めた。物体Zにはたらく浮力の大きさはいくらか。（3点）
→ 1.6N。

(6) 水に浮いている木片にはたらく力を考える。浮力と重力の大きさの関係を答え、その理由を説明しなさい。（3点）
→ 浮力と重力は等しい（つり合っている）。物体が水面に浮いて静止しているということは、上向きの浮力と下向きの重力がつり合っているためである。

## 大問5: 水圧の性質（配点 20点）

ゴム膜を張った円筒形の容器を水中に沈め、さまざまな深さや向きでゴム膜のへこみ方を調べる実験を行った。

(1) ゴム膜を下向きにして深さ10cmと深さ20cmに沈めたとき、ゴム膜のへこみが大きいのはどちらか。理由もあわせて答えなさい。（4点）
→ 深さ20cmの方。水圧は深いほど大きくなるため、深い位置の方がゴム膜を押す力が大きくへこみも大きくなる。

(2) 同じ深さ（20cm）でゴム膜を上向き、横向き、下向きにしたとき、へこみの大きさに違いはあるか。理由もあわせて答えなさい。（4点）
→ ほぼ同じ大きさになる。水圧はあらゆる方向から等しくはたらくため、同じ深さであればどの向きでも水圧は同じである。

(3) 水圧が深いほど大きくなる理由を「水の重力」の語を使って説明しなさい。（4点）
→ 水圧は上にある水の重力によって生じるため、深いところほど上にある水の量が多くなり、水の重力が大きくなるので水圧も大きくなる。

(4) 水中に沈めた直方体の物体の上面と下面にはたらく水圧を比べたとき、どちらが大きいか。理由もあわせて答えなさい。（4点）
→ 下面にはたらく水圧の方が大きい。下面の方が深い位置にあるため、より大きな水圧がはたらく。

(5) 物体にはたらく浮力が、上面と下面の水圧差から生じることを、「上向き」「下向き」の語を使って説明しなさい。（4点）
→ 下面には上向きの水圧がはたらき、上面には下向きの水圧がはたらく。下面の方が深い位置にあるため上向きの水圧の方が大きく、この差が上向きの力（浮力）となる。
