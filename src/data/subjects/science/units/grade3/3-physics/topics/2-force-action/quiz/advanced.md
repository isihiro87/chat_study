# 力のはたらき方 発展問題

## 問1: 力の分解と斜面上の運動

質量500gの物体を斜面に置き、斜面に平行な分力を作図で調べた。100gの物体にはたらく重力を1Nとする。

(1) この物体にはたらく重力の大きさを求めなさい。
→ 5N。

(2) 斜面の傾きがゆるやかなとき、重力を斜面に平行な方向と斜面に垂直な方向に作図で分解すると、斜面に平行な分力と垂直な分力ではどちらが大きくなるか。
→ 斜面に垂直な分力の方が大きい。傾きがゆるやかなとき、重力の大部分は斜面に垂直な方向にはたらく。

(3) 斜面の傾きを大きくしていくと、斜面に平行な分力と垂直な分力はそれぞれどう変化するか。
→ 斜面に平行な分力は大きくなり、斜面に垂直な分力は小さくなる。

(4) 斜面の角度が90°（垂直）になったとき、斜面に平行な分力は重力の何倍になるか。理由も説明しなさい。
→ 重力と同じ大きさ（1倍）になる。斜面が垂直のとき、斜面に平行な方向は真下の方向と一致するため、重力のすべてが斜面に平行な分力となる。これは自由落下の状態に等しい。

## 問2: 浮力の計算問題

ある物体をばねばかりにつるして次の実験を行った。

実験1：空気中でばねばかりの値は4.0Nであった。
実験2：物体の半分だけ水に沈めたところ、ばねばかりの値は3.4Nであった。
実験3：物体を完全に水中に沈めたところ、ばねばかりの値は2.8Nであった。

(1) 実験2で物体にはたらく浮力の大きさを求めなさい。
→ 0.6N。（4.0 − 3.4 = 0.6N）

(2) 実験3で物体にはたらく浮力の大きさを求めなさい。
→ 1.2N。（4.0 − 2.8 = 1.2N）

(3) 実験2と実験3の結果を比較し、浮力と水中にある体積の関係について述べなさい。
→ 物体の半分を沈めたときの浮力（0.6N）は、全体を沈めたときの浮力（1.2N）のちょうど半分である。このことから、浮力は水中にある物体の体積に比例することがわかる。

(4) この物体を完全に水中に沈めた状態で、さらに深く沈めた場合、ばねばかりの値はどうなるか。理由をあわせて答えなさい。
→ 2.8Nのまま変わらない。完全に水中に沈んでいれば水中にある体積は変わらないため、浮力の大きさも変わらず、ばねばかりの値も一定である。

## 問3: 作用・反作用の法則の応用

次の場面について、作用・反作用の法則の観点から分析しなさい。

(1) 地球上で人が地面に立っているとき、人が地面を押す力と地面が人を押す力（垂直抗力）は作用・反作用の関係にある。では、人にはたらく重力（地球が人を引く力）の反作用は何か。
→ 人が地球を引く力。人にはたらく重力の反作用は、人が地球を万有引力で引く力であり、その大きさは重力と同じである。

(2) 重さ50Nの物体が机の上で静止している。このとき物体にはたらく力をすべて挙げ、どれがつり合いの関係でどれが作用・反作用の関係かを整理しなさい。
→ 物体にはたらく力は、下向きの重力（50N）と机からの上向きの垂直抗力（50N）。この2力は同じ物体（物体）にはたらくのでつり合いの関係。一方、物体が机を押す力（50N、下向き）と机が物体を押す力（50N、上向き）は別々の物体にはたらくので作用・反作用の関係。

(3) 綱引きで両チームが同じ力で引き合って動かないとき、作用・反作用の法則が成り立っているにもかかわらず勝敗がつくことがあるのはなぜか。「摩擦力」の語を使って説明しなさい。
→ 作用・反作用の法則により、綱を引く力と引き返される力は常に同じ大きさである。勝敗を決めるのは地面との摩擦力であり、地面をより強く踏みしめて大きな摩擦力を得られるチームの方が、体が動かされにくいため勝つことができる。

## 問4: 水圧差と浮力の関係

水中に一辺10cmの立方体の物体を、上面が水面から20cmの深さになるように沈めた。水の密度を1.0g/cm³、100gの物体にはたらく重力を1Nとする。

(1) 物体の上面にはたらく水圧と下面にはたらく水圧では、どちらが大きいか。理由を答えなさい。
→ 下面にはたらく水圧の方が大きい。下面は水面から30cm（20cm + 10cm）の深さにあり、上面の20cmよりも深い位置にあるため、より大きな水圧がはたらく。

(2) 物体の左右の面にはたらく水圧は浮力に影響するか。理由を答えなさい。
→ 影響しない。左右の面は同じ高さにあるため、同じ大きさの水圧が互いに反対方向からはたらき、打ち消し合うから。

(3) この立方体の体積を求め、水中にある物体の体積から浮力の大きさを求めなさい。
→ 体積は10cm × 10cm × 10cm = 1000cm³ = 1000mL。水の密度が1.0g/cm³なので、1000cm³の水の重さは1000g = 10N。よって浮力は10Nとなる（水中にある物体と同じ体積の水の重さに等しい）。

(4) この物体の重さが8Nのとき、ばねばかりにつるさずに水中に放すとどうなるか。理由を答えなさい。
→ 浮き上がる。浮力（10N）が重力（8N）より大きいため、上向きの力が勝って物体は水面に向かって浮き上がる。

## 問5: 慣性と力の総合問題

なめらかな水平面上に質量200gの台車が静止している。この台車の上に質量100gの物体を乗せた。

(1) 台車を急に右向きに動かしたとき、台車の上の物体はどうなるか。「慣性」の語を使って説明しなさい。
→ 物体は慣性によりその場にとどまろうとするため、台車に対して左に取り残されるように見える（台車から落ちる可能性がある）。

(2) 台車と物体が一緒に右向きに等速直線運動をしているとき、台車を急に止めると物体はどうなるか。理由を答えなさい。
→ 物体は慣性により右向きに動き続けようとするため、台車の上を右向きに滑り、台車の前方に飛び出す。

(3) (2)の現象と同じ原理で起こる日常の例を1つ挙げ、説明しなさい。
→ 走っている自動車が急ブレーキをかけたとき、乗っている人が前に倒れそうになる現象。車は止まろうとするが、人の体は慣性により走っていたときの速度で前に進み続けようとするため。

(4) 台車と物体が一緒に右向きに運動しているとき、物体が台車から受けている摩擦力の向きと、台車が物体から受けている摩擦力の向きを、作用・反作用の法則を使ってそれぞれ答えなさい。
→ 物体が台車から受ける摩擦力は右向き（物体を台車と同じ方向に動かす力）。作用・反作用の法則により、台車が物体から受ける摩擦力はその反対の左向きである。2つの力は同じ大きさで逆向きにはたらく。
