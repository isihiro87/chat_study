# 方程式とグラフ 大問小問

## 大問1: 方程式のグラフをかく（配点 20点）

次の二元一次方程式を $y =$ の形に変形し、傾きと切片を答えなさい。

(1) $2x + y = 6$（3点）
→ $y = -2x + 6$。傾き $-2$、切片 $6$。

(2) $x + 3y = 9$（3点）
→ $y = -\frac{1}{3}x + 3$。傾き $-\frac{1}{3}$、切片 $3$。

(3) $4x - 2y = 8$（4点）
→ $y = 2x - 4$。傾き $2$、切片 $-4$。

(4) $3x + 2y = 12$ のグラフが通る2点を、$x = 0$ と $y = 0$ を代入して求めなさい。（4点）
→ $x = 0$ のとき $y = 6$ → $(0, 6)$。$y = 0$ のとき $x = 4$ → $(4, 0)$。

(5) $2x - y = 4$ のグラフが通る2点を、$x = 0$ と $y = 0$ を代入して求めなさい。（3点）
→ $x = 0$ のとき $y = -4$ → $(0, -4)$。$y = 0$ のとき $x = 2$ → $(2, 0)$。

(6) $-x + 4y = 8$ を $y =$ の形に変形し、傾きと切片を答えなさい。（3点）
→ $y = \frac{1}{4}x + 2$。傾き $\frac{1}{4}$、切片 $2$。

## 大問2: $y = k$, $x = h$ のグラフ（配点 20点）

(1) $y = 5$ のグラフはどんな直線か答えなさい。（3点）
→ $x$ 軸に平行な横の直線で、すべての点の $y$ 座標が $5$。

(2) $x = -3$ のグラフはどんな直線か答えなさい。（3点）
→ $y$ 軸に平行な縦の直線で、すべての点の $x$ 座標が $-3$。

(3) $2y - 6 = 0$ のグラフをかくために、まず式を変形しなさい。（4点）
→ $2y = 6$、$y = 3$。$x$ 軸に平行な横の直線。

(4) $4x + 12 = 0$ のグラフをかくために、まず式を変形しなさい。（4点）
→ $4x = -12$、$x = -3$。$y$ 軸に平行な縦の直線。

(5) $x$ 軸に平行で点 $(2, -4)$ を通る直線の式を求めなさい。（3点）
→ $y = -4$。$x$ 軸に平行な直線は $y = k$ の形で、$y$ 座標が $-4$ だから。

(6) $y$ 軸に平行で点 $(7, 3)$ を通る直線の式を求めなさい。（3点）
→ $x = 7$。$y$ 軸に平行な直線は $x = h$ の形で、$x$ 座標が $7$ だから。

## 大問3: 連立方程式をグラフで解く（配点 20点）

(1) $y = x + 2$ と $y = -x + 6$ の交点の座標を求めなさい。（3点）
→ $x + 2 = -x + 6$、$2x = 4$、$x = 2$、$y = 4$。交点 $(2, 4)$。

(2) $y = 2x - 1$ と $y = -x + 8$ の交点の座標を求めなさい。（3点）
→ $2x - 1 = -x + 8$、$3x = 9$、$x = 3$、$y = 5$。交点 $(3, 5)$。

(3) $y = 3x$ と $y = x + 4$ の交点の座標を求めなさい。（4点）
→ $3x = x + 4$、$2x = 4$、$x = 2$、$y = 6$。交点 $(2, 6)$。

(4) $x + y = 5$ と $2x - y = 4$ の交点の座標を求めなさい。（4点）
→ 2式を足すと $3x = 9$、$x = 3$。①に代入し $y = 2$。交点 $(3, 2)$。

(5) $3x + y = 11$ と $x - y = 1$ の交点の座標を求めなさい。（3点）
→ 2式を足すと $4x = 12$、$x = 3$。②に代入し $y = 2$。交点 $(3, 2)$。

(6) 2直線 $y = 2x + 1$ と $y = 2x - 3$ の交点はあるか。理由を答えなさい。（3点）
→ 交点はない。傾きがどちらも $2$ で平行だから。連立方程式にすると $2x + 1 = 2x - 3$ → $1 = -3$ となり解なし。

## 大問4: 2直線の交点の座標（配点 20点）

(1) $y = \frac{1}{2}x + 3$ と $y = -x + 6$ の交点の座標を求めなさい。（4点）
→ $\frac{1}{2}x + 3 = -x + 6$、$\frac{3}{2}x = 3$、$x = 2$、$y = 4$。交点 $(2, 4)$。

(2) $2x + 3y = 12$ と $x - y = 1$ の交点の座標を求めなさい。（4点）
→ ②より $x = y + 1$。①に代入: $2(y + 1) + 3y = 12$、$5y = 10$、$y = 2$、$x = 3$。交点 $(3, 2)$。

(3) $x + 2y = 8$ と $3x - y = 10$ の交点の座標を求めなさい。（4点）
→ ①より $x = 8 - 2y$。②に代入: $3(8 - 2y) - y = 10$、$24 - 7y = 10$、$y = 2$、$x = 4$。交点 $(4, 2)$。

(4) $y = -2x + 9$ と $y = x$ の交点の座標を求めなさい。（4点）
→ $x = -2x + 9$、$3x = 9$、$x = 3$、$y = 3$。交点 $(3, 3)$。

(5) $y = \frac{3}{2}x - 1$ と $y = -\frac{1}{2}x + 3$ の交点の座標を求めなさい。（4点）
→ $\frac{3}{2}x - 1 = -\frac{1}{2}x + 3$、$2x = 4$、$x = 2$、$y = 2$。交点 $(2, 2)$。

## 大問5: 直線と図形（配点 20点）

(1) $y = x$ と $y = -2x + 6$ と $x$ 軸で囲まれた三角形の3つの頂点を求めなさい。（4点）
→ 2直線の交点: $x = -2x + 6$、$3x = 6$、$x = 2$、$y = 2$ → $(2, 2)$。$y = x$ の $x$ 切片 $(0, 0)$。$y = -2x + 6$ の $x$ 切片: $0 = -2x + 6$、$x = 3$ → $(3, 0)$。頂点は $(0, 0)$, $(3, 0)$, $(2, 2)$。

(2) (1)の三角形の面積を求めなさい。（3点）
→ 底辺 $= 3 - 0 = 3$（$x$ 軸上）、高さ $= 2$。面積 $= \frac{1}{2} \times 3 \times 2 = 3$。

(3) $y = 2x$ と $y = -x + 6$ と $x$ 軸で囲まれた三角形の面積を求めなさい。（4点）
→ 交点: $2x = -x + 6$、$x = 2$、$y = 4$ → $(2, 4)$。$x$ 切片: $(0, 0)$ と $(6, 0)$。底辺 $6$、高さ $4$。面積 $= \frac{1}{2} \times 6 \times 4 = 12$。

(4) $y = x + 2$ と $y = -x + 6$ と $y$ 軸で囲まれた三角形の面積を求めなさい。（4点）
→ 交点: $x + 2 = -x + 6$、$x = 2$、$y = 4$ → $(2, 4)$。$y$ 切片: $(0, 2)$ と $(0, 6)$。底辺（$y$ 軸上）$= 6 - 2 = 4$、高さ $= 2$。面積 $= \frac{1}{2} \times 4 \times 2 = 4$。

(5) $y = 2x$ と $y = -x + 6$ と $x$ 軸で囲まれた三角形で、原点を通り面積を2等分する直線の式を求めなさい。（5点）
→ 三角形の頂点は $(0, 0)$, $(6, 0)$, $(2, 4)$。原点の対辺は $(6, 0)$ と $(2, 4)$ を結ぶ辺。対辺の中点 $= \left(\frac{6 + 2}{2}, \frac{0 + 4}{2}\right) = (4, 2)$。原点と $(4, 2)$ を通る直線: $y = \frac{1}{2}x$。
