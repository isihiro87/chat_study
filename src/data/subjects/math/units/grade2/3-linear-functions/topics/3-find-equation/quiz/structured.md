# 一次関数の式を求める 大問小問

## 大問1: グラフから式を求める（配点 20点）

グラフの傾きと切片を読み取って、一次関数の式を求めなさい。

(1) 傾き $3$、切片 $-1$ のグラフの式を求めよ。（3点）
→ $y = 3x - 1$。$y = ax + b$ に $a = 3$、$b = -1$ を代入する。

(2) 傾き $-2$、切片 $4$ のグラフの式を求めよ。（3点）
→ $y = -2x + 4$。$y = ax + b$ に $a = -2$、$b = 4$ を代入する。

(3) 傾き $\dfrac{1}{2}$、切片 $-3$ のグラフの式を求めよ。（4点）
→ $y = \dfrac{1}{2}x - 3$。$y = ax + b$ に $a = \dfrac{1}{2}$、$b = -3$ を代入する。

(4) グラフが点 $(0, 5)$ を通り、$x$ が $1$ 増えると $y$ が $4$ 減るとき、式を求めよ。（5点）
→ $y = -4x + 5$。$(0, 5)$ を通るので切片 $5$。$y$ が $4$ 減るので傾き $-4$。

(5) グラフが点 $(0, -2)$ と点 $(3, 4)$ を通るとき、式を求めよ。（5点）
→ $y = 2x - 2$。切片 $-2$。傾き $= (4 - (-2)) \div (3 - 0) = 6 \div 3 = 2$。

## 大問2: 傾きと1点から式を求める（配点 20点）

次の条件を満たす一次関数の式を求めなさい。

(1) 傾き $2$ で、点 $(1, 5)$ を通る。（3点）
→ $y = 2x + 3$。$y = 2x + b$ に $(1, 5)$ を代入: $5 = 2 + b$、$b = 3$。

(2) 傾き $-3$ で、点 $(2, -1)$ を通る。（3点）
→ $y = -3x + 5$。$y = -3x + b$ に $(2, -1)$ を代入: $-1 = -6 + b$、$b = 5$。

(3) 傾き $\dfrac{1}{3}$ で、点 $(6, 4)$ を通る。（4点）
→ $y = \dfrac{1}{3}x + 2$。$y = \dfrac{1}{3}x + b$ に $(6, 4)$ を代入: $4 = 2 + b$、$b = 2$。

(4) 傾き $-4$ で、点 $(-1, 7)$ を通る。（4点）
→ $y = -4x + 3$。$y = -4x + b$ に $(-1, 7)$ を代入: $7 = 4 + b$、$b = 3$。

(5) 変化の割合が $5$ で、点 $(2, 13)$ を通る。（3点）
→ $y = 5x + 3$。変化の割合 $=$ 傾きなので $a = 5$。$y = 5x + b$ に $(2, 13)$ を代入: $13 = 10 + b$、$b = 3$。

(6) 傾き $-\dfrac{2}{3}$ で、点 $(3, 0)$ を通る。（3点）
→ $y = -\dfrac{2}{3}x + 2$。$y = -\dfrac{2}{3}x + b$ に $(3, 0)$ を代入: $0 = -2 + b$、$b = 2$。

## 大問3: 2点から式を求める ー 傾きから（配点 20点）

2点の座標から傾きを求め、一次関数の式を求めなさい。

(1) $(1, 2)$ と $(3, 8)$ を通る。（3点）
→ $y = 3x - 1$。傾き $= (8 - 2) \div (3 - 1) = 3$。$y = 3x + b$ に $(1, 2)$ を代入: $2 = 3 + b$、$b = -1$。

(2) $(2, 7)$ と $(5, 16)$ を通る。（3点）
→ $y = 3x + 1$。傾き $= (16 - 7) \div (5 - 2) = 3$。$y = 3x + b$ に $(2, 7)$ を代入: $7 = 6 + b$、$b = 1$。

(3) $(-1, 5)$ と $(3, -3)$ を通る。（4点）
→ $y = -2x + 3$。傾き $= (-3 - 5) \div (3 - (-1)) = -8 \div 4 = -2$。$y = -2x + b$ に $(3, -3)$ を代入: $-3 = -6 + b$、$b = 3$。

(4) $(4, 1)$ と $(6, 4)$ を通る。（4点）
→ $y = \dfrac{3}{2}x - 5$。傾き $= (4 - 1) \div (6 - 4) = \dfrac{3}{2}$。$y = \dfrac{3}{2}x + b$ に $(4, 1)$ を代入: $1 = 6 + b$、$b = -5$。

(5) $(-2, -7)$ と $(4, 5)$ を通る。（3点）
→ $y = 2x - 3$。傾き $= (5 - (-7)) \div (4 - (-2)) = 12 \div 6 = 2$。$y = 2x + b$ に $(4, 5)$ を代入: $5 = 8 + b$、$b = -3$。

(6) $(-3, 8)$ と $(1, 0)$ を通る。（3点）
→ $y = -2x + 2$。傾き $= (0 - 8) \div (1 - (-3)) = -8 \div 4 = -2$。$y = -2x + b$ に $(1, 0)$ を代入: $0 = -2 + b$、$b = 2$。

## 大問4: 2点から式を求める ー 連立方程式（配点 20点）

2点の座標を $y = ax + b$ に代入して連立方程式をつくり、一次関数の式を求めなさい。

(1) $(1, 3)$ と $(4, 12)$ を通る。（4点）
→ $y = 3x$。$(1, 3)$: $a + b = 3$ …①、$(4, 12)$: $4a + b = 12$ …②。②−①より $3a = 9$、$a = 3$。①に代入し $b = 0$。

(2) $(2, 1)$ と $(5, 7)$ を通る。（4点）
→ $y = 2x - 3$。$(2, 1)$: $2a + b = 1$ …①、$(5, 7)$: $5a + b = 7$ …②。②−①より $3a = 6$、$a = 2$。①に代入し $b = -3$。

(3) $(-1, 6)$ と $(3, -2)$ を通る。（4点）
→ $y = -2x + 4$。$(-1, 6)$: $-a + b = 6$ …①、$(3, -2)$: $3a + b = -2$ …②。②−①より $4a = -8$、$a = -2$。①に代入し $b = 4$。

(4) $(-2, -5)$ と $(3, 10)$ を通る。（4点）
→ $y = 3x + 1$。$(-2, -5)$: $-2a + b = -5$ …①、$(3, 10)$: $3a + b = 10$ …②。②−①より $5a = 15$、$a = 3$。①に代入し $b = 1$。

(5) $(1, -1)$ と $(4, 5)$ を通る。（4点）
→ $y = 2x - 3$。$(1, -1)$: $a + b = -1$ …①、$(4, 5)$: $4a + b = 5$ …②。②−①より $3a = 6$、$a = 2$。①に代入し $b = -3$。

## 大問5: 条件から式を求める（配点 20点）

次の条件を満たす一次関数の式を求めなさい。

(1) $y = 2x + 5$ に平行で、点 $(3, 4)$ を通る。（4点）
→ $y = 2x - 2$。平行なので傾き $2$。$y = 2x + b$ に $(3, 4)$ を代入: $4 = 6 + b$、$b = -2$。

(2) $y = -3x + 1$ に平行で、点 $(1, 0)$ を通る。（4点）
→ $y = -3x + 3$。平行なので傾き $-3$。$y = -3x + b$ に $(1, 0)$ を代入: $0 = -3 + b$、$b = 3$。

(3) $x = 2$ のとき $y = 5$、$x = 5$ のとき $y = 14$ である。（4点）
→ $y = 3x - 1$。傾き $= (14 - 5) \div (5 - 2) = 3$。$y = 3x + b$ に $(2, 5)$ を代入: $5 = 6 + b$、$b = -1$。

(4) $x$ が $2$ 増加すると $y$ が $6$ 増加し、点 $(3, 11)$ を通る。（4点）
→ $y = 3x + 2$。傾き $= 6 \div 2 = 3$。$y = 3x + b$ に $(3, 11)$ を代入: $11 = 9 + b$、$b = 2$。

(5) $x$ が $5$ 増加すると $y$ が $10$ 減少し、$x = 3$ のとき $y = -1$ である。（4点）
→ $y = -2x + 5$。傾き $= -10 \div 5 = -2$。$y = -2x + b$ に $(3, -1)$ を代入: $-1 = -6 + b$、$b = 5$。
