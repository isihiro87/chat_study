# 一次関数の意味と変化の割合 大問小問

## 大問1: 一次関数の判別（配点 15点）

次の式について、$y$ が $x$ の一次関数であるものには○、そうでないものには×をつけ、理由を答えなさい。

(1) $y = 4x - 3$（2点）
→ ○。$y = ax + b$ の形で $a = 4$, $b = -3$ の一次関数。

(2) $y = x^2 + 2$（2点）
→ ×。$x^2$ の項があり一次式ではない。

(3) $y = -7x$（2点）
→ ○。$y = ax + b$ の形で $a = -7$, $b = 0$ の一次関数（比例でもある）。

(4) $xy = 10$（2点）
→ ×。$y = \dfrac{10}{x}$ となり反比例の式。$y = ax + b$ の形ではない。

(5) $y = 8$（2点）
→ ×。$a = 0$ なので一次関数ではない。$x$ が変化しても $y$ は変わらない。

(6) $y = \dfrac{2x - 6}{3}$（3点）
→ ○。$y = \dfrac{2}{3}x - 2$ と変形でき、$a = \dfrac{2}{3}$, $b = -2$ の一次関数。

(7) $y = \dfrac{5}{x} + 3$（2点）
→ ×。$\dfrac{5}{x}$ の項があり $y = ax + b$ の形にならない。

## 大問2: 一次関数の式に値を代入（配点 20点）

(1) $y = 3x + 2$ で $x = 4$ のとき $y$ の値を求めなさい。（3点）
→ $y = 3 \times 4 + 2 = 14$。

(2) $y = -2x + 9$ で $x = 3$ のとき $y$ の値を求めなさい。（3点）
→ $y = -2 \times 3 + 9 = 3$。

(3) $y = \dfrac{1}{2}x - 5$ で $x = 6$ のとき $y$ の値を求めなさい。（3点）
→ $y = \dfrac{1}{2} \times 6 - 5 = -2$。

(4) $y = -4x + 20$ で $y = 0$ のとき $x$ の値を求めなさい。（4点）
→ $0 = -4x + 20$ より $4x = 20$、$x = 5$。

(5) $y = 3x - 7$ で $y = 11$ のとき $x$ の値を求めなさい。（4点）
→ $11 = 3x - 7$ より $3x = 18$、$x = 6$。

(6) $y = 2x + b$ が $x = 3$ のとき $y = 11$ を満たすとき、$b$ の値を求めなさい。（3点）
→ $11 = 2 \times 3 + b$ より $6 + b = 11$、$b = 5$。

## 大問3: 変化の割合（配点 20点）

(1) $y = 5x - 1$ の変化の割合を答えなさい。（2点）
→ $5$。一次関数 $y = ax + b$ の変化の割合は $a$ なので $5$。

(2) $y = -\dfrac{3}{4}x + 2$ の変化の割合を答えなさい。（3点）
→ $-\dfrac{3}{4}$。

(3) $y = 2x + 3$ で $x$ が $1$ から $5$ まで増加するとき、変化の割合を計算で求めなさい。（4点）
→ $x = 1$ のとき $y = 5$、$x = 5$ のとき $y = 13$。変化の割合 $= \dfrac{13 - 5}{5 - 1} = \dfrac{8}{4} = 2$。

(4) $y = -3x + 10$ で $x$ が $-2$ から $4$ まで増加するとき、変化の割合を計算で求めなさい。（4点）
→ $x = -2$ のとき $y = 16$、$x = 4$ のとき $y = -2$。変化の割合 $= \dfrac{-2 - 16}{4 - (-2)} = \dfrac{-18}{6} = -3$。

(5) ある一次関数で、$x$ が $2$ から $5$ に変化したとき $y$ が $4$ から $13$ に変化した。変化の割合を求めなさい。（3点）
→ 変化の割合 $= \dfrac{13 - 4}{5 - 2} = \dfrac{9}{3} = 3$。

(6) ある一次関数で、変化の割合が $-2$ で、$x$ が $3$ 増加したとき $y$ はいくつ変化するか。（4点）
→ $y$ の増加量 $= -2 \times 3 = -6$。$y$ は $6$ 減少する。

## 大問4: 増加量の計算（配点 20点）

(1) $y = 4x + 1$ で $x$ の増加量が $3$ のとき、$y$ の増加量を求めなさい。（3点）
→ $y$ の増加量 $= 4 \times 3 = 12$。

(2) $y = -2x + 7$ で $x$ の増加量が $5$ のとき、$y$ の増加量を求めなさい。（3点）
→ $y$ の増加量 $= -2 \times 5 = -10$。

(3) $y = \dfrac{1}{3}x + 2$ で $x$ が $3$ から $9$ に増加するとき、$y$ の増加量を求めなさい。（4点）
→ $x$ の増加量 $= 9 - 3 = 6$。$y$ の増加量 $= \dfrac{1}{3} \times 6 = 2$。

(4) $y = -5x + 20$ で $x$ が $-1$ から $3$ に増加するとき、$y$ の値の変化を求めなさい。（4点）
→ $x = -1$ のとき $y = 25$、$x = 3$ のとき $y = 5$。$y$ は $25$ から $5$ に変化し、$y$ の増加量は $-20$。

(5) ある一次関数で $x$ が $2$ 増加すると $y$ が $8$ 増加する。$x$ が $5$ 増加すると $y$ はいくつ増加するか。（3点）
→ 変化の割合 $= 8 \div 2 = 4$。$x$ が $5$ 増加すると $y$ の増加量 $= 4 \times 5 = 20$。

(6) ある一次関数で $x$ が $4$ 増加すると $y$ が $-12$ 変化する。$x$ が $1$ 増加すると $y$ はいくつ変化するか。（3点）
→ 変化の割合 $= -12 \div 4 = -3$。$x$ が $1$ 増加すると $y$ の増加量 $= -3 \times 1 = -3$。

## 大問5: 文章題（一次関数の立式）（配点 25点）

(1) 長さ20cmのろうそくに火をつけると、1分間に0.4cmずつ短くなる。$x$ 分後のろうそくの長さ $y$ cmを $x$ の式で表しなさい。（4点）
→ $y = -0.4x + 20$。毎分0.4cm短くなるので $a = -0.4$、最初の長さ $b = 20$。

(2) (1) で、ろうそくに火をつけてから10分後の長さを求めなさい。（3点）
→ $y = -0.4 \times 10 + 20 = 16$（cm）。

(3) 水そうに50Lの水が入っていて、毎分3Lずつ水を入れる。$x$ 分後の水の量 $y$ Lを式で表しなさい。（4点）
→ $y = 3x + 50$。毎分3L増えるので $a = 3$、最初の量 $b = 50$。

(4) (3) で、水そうの容量が110Lのとき、満水になるのは何分後か。（4点）
→ $110 = 3x + 50$ より $3x = 60$、$x = 20$。20分後。

(5) A地点から8km離れたB地点に向かって、時速4kmで歩く。出発してから $x$ 時間後のA地点からの距離 $y$ kmを式で表しなさい。また、変化の割合は何を意味するか答えなさい。（5点）
→ $y = 4x$。変化の割合 $4$ は「1時間あたり4km進む」つまり時速4kmを意味する。

(6) ある携帯プランは基本料金800円に、通話1分あたり20円の通話料がかかる。$x$ 分通話したときの月額料金 $y$ 円を式で表し、30分通話したときの月額料金を求めなさい。（5点）
→ $y = 20x + 800$。$x = 30$ のとき $y = 20 \times 30 + 800 = 1400$（円）。
