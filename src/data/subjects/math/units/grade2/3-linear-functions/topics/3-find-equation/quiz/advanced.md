# 一次関数の式を求める 発展問題

## 問1: グラフの読み取りからの式の決定

(1) 一次関数のグラフが $x$ 軸と点 $(4, 0)$ で交わり、$y$ 軸と点 $(0, -8)$ で交わるとき、式を求めよ。
→ $y = 2x - 8$。切片 $-8$。傾き $= (0 - (-8)) \div (4 - 0) = 8 \div 4 = 2$。よって $y = 2x - 8$。

(2) 一次関数のグラフが $x$ 軸と点 $(-3, 0)$ で交わり、$y$ 軸と点 $(0, 6)$ で交わるとき、式を求めよ。
→ $y = 2x + 6$。切片 $6$。傾き $= (6 - 0) \div (0 - (-3)) = 6 \div 3 = 2$。よって $y = 2x + 6$。

(3) 一次関数 $y = ax + b$ のグラフが $x$ 軸と点 $(6, 0)$ で交わり、傾きが $-\dfrac{1}{3}$ のとき、切片 $b$ を求めよ。
→ $b = 2$。$y = -\dfrac{1}{3}x + b$ に $(6, 0)$ を代入: $0 = -2 + b$、$b = 2$。式は $y = -\dfrac{1}{3}x + 2$。

(4) 一次関数のグラフが点 $(0, 3)$ を通り、$x$ 軸上の正の部分と交わる点の $x$ 座標が $6$ である。式を求めよ。
→ $y = -\dfrac{1}{2}x + 3$。切片 $3$、$x$ 切片 $6$ なので 2点 $(0, 3)$、$(6, 0)$ を通る。傾き $= (0 - 3) \div (6 - 0) = -\dfrac{1}{2}$。

## 問2: 複数の条件からの式の決定

(1) 一次関数 $y = ax + b$ で、$x = 3$ のとき $y = 7$、$x = -1$ のとき $y = -5$ である。$a$ と $b$ を求めよ。
→ $a = 3, b = -2$。$(3, 7)$: $3a + b = 7$ …①、$(-1, -5)$: $-a + b = -5$ …②。①−②より $4a = 12$、$a = 3$。②に代入し $b = -2$。式は $y = 3x - 2$。

(2) 一次関数 $y = ax + b$ で、$x = -2$ のとき $y = 9$、$x = 4$ のとき $y = -3$ である。式を求めよ。
→ $y = -2x + 5$。$(-2, 9)$: $-2a + b = 9$ …①、$(4, -3)$: $4a + b = -3$ …②。②−①より $6a = -12$、$a = -2$。①に代入し $b = 5$。

(3) 一次関数 $y = ax + b$ のグラフが点 $(2, 1)$ を通り、$x$ の値が $-1$ から $5$ まで変化すると $y$ の値は $12$ 減少する。式を求めよ。
→ $y = -2x + 5$。変化の割合 $= -12 \div (5 - (-1)) = -12 \div 6 = -2$ なので $a = -2$。$y = -2x + b$ に $(2, 1)$ を代入: $1 = -4 + b$、$b = 5$。

(4) 一次関数で、$x$ が $1$ 増加するごとに $y$ は $4$ ずつ増加し、$x = 0$ から $x = 3$ までの $y$ の増加量が $12$ である。点 $(2, 11)$ を通るとき、式を求めよ。
→ $y = 4x + 3$。$y$ の増加量 $12 \div 3 = 4$ で傾き $4$ と確認。$y = 4x + b$ に $(2, 11)$ を代入: $11 = 8 + b$、$b = 3$。

## 問3: 平行な直線の応用

(1) $y = -3x + 7$ に平行で、$x$ 軸と点 $(2, 0)$ で交わる直線の式を求めよ。
→ $y = -3x + 6$。平行なので傾き $-3$。$y = -3x + b$ に $(2, 0)$ を代入: $0 = -6 + b$、$b = 6$。

(2) $y = \dfrac{1}{2}x - 3$ に平行で、点 $(4, 5)$ を通る直線の式を求めよ。
→ $y = \dfrac{1}{2}x + 3$。平行なので傾き $\dfrac{1}{2}$。$y = \dfrac{1}{2}x + b$ に $(4, 5)$ を代入: $5 = 2 + b$、$b = 3$。

(3) $y = 2x + 1$ に平行で、点 $(-3, -8)$ を通る直線の式を求めよ。また、この直線の切片を答えよ。
→ $y = 2x - 2$、切片 $-2$。平行なので傾き $2$。$y = 2x + b$ に $(-3, -8)$ を代入: $-8 = -6 + b$、$b = -2$。

(4) 2つの直線 $y = 4x - 1$ と $y = 4x + 5$ の間を通り、これらに平行な直線で、点 $(1, 6)$ を通るものの式を求めよ。
→ $y = 4x + 2$。平行なので傾き $4$。$y = 4x + b$ に $(1, 6)$ を代入: $6 = 4 + b$、$b = 2$。切片 $2$ は $-1$ と $5$ の間にある。

## 問4: 式の決定と検算

(1) 2点 $(-2, 11)$ と $(4, -1)$ を通る一次関数の式を連立方程式で求めよ。また、$x = 1$ のときの $y$ の値を求めよ。
→ $y = -2x + 7$。$(-2, 11)$: $-2a + b = 11$ …①、$(4, -1)$: $4a + b = -1$ …②。②−①より $6a = -12$、$a = -2$。①に代入し $b = 7$。$x = 1$ のとき $y = -2 + 7 = 5$。

(2) 2点 $(3, -2)$ と $(-1, 6)$ を通る一次関数の式を求めよ。また、$y = 0$ となる $x$ の値を求めよ。
→ $y = -2x + 4$。傾き $= (6 - (-2)) \div (-1 - 3) = 8 \div (-4) = -2$。$y = -2x + b$ に $(3, -2)$ を代入: $-2 = -6 + b$、$b = 4$。$0 = -2x + 4$ より $x = 2$。

(3) 一次関数 $y = ax + b$ が点 $(5, 3)$ と点 $(-5, -7)$ を通るとき、$a$ と $b$ の値、および $a + b$ の値を求めよ。
→ $a = 1, b = -2, a + b = -1$。$(5, 3)$: $5a + b = 3$ …①、$(-5, -7)$: $-5a + b = -7$ …②。①−②より $10a = 10$、$a = 1$。①に代入し $b = -2$。$a + b = -1$。

## 問5: 総合問題

(1) 一次関数 $y = ax + b$ のグラフが点 $(2, 5)$ を通り、$y = -x + 3$ に平行である。このグラフの $x$ 切片を求めよ。
→ $x$ 切片は $7$。平行なので $a = -1$。$y = -x + b$ に $(2, 5)$ を代入: $5 = -2 + b$、$b = 7$。$y = -x + 7$ で $y = 0$ とすると $x = 7$。

(2) 一次関数のグラフが 2点 $(1, 4)$ と $(5, -4)$ を通る。このグラフが $x$ 軸、$y$ 軸と交わる点の座標をそれぞれ求めよ。
→ $x$ 軸との交点 $(3, 0)$、$y$ 軸との交点 $(0, 6)$。傾き $= (-4 - 4) \div (5 - 1) = -2$。$y = -2x + b$ に $(1, 4)$ を代入: $b = 6$。$y = -2x + 6$ で $y = 0$ とすると $x = 3$、$x = 0$ とすると $y = 6$。

(3) $x = -1$ のとき $y = 8$、$x = 5$ のとき $y = -4$ である一次関数について、$y$ が正となる $x$ の範囲を求めよ。
→ $x < 4$。傾き $= (-4 - 8) \div (5 - (-1)) = -2$。$y = -2x + b$ に $(-1, 8)$ を代入: $b = 6$。$y = -2x + 6 > 0$ より $x < 3$ …ではなく、$-2x + 6 > 0$ を解くと $x < 3$。
→ 訂正: $-2x + 6 > 0$ より $-2x > -6$、$x < 3$。答え: $x < 3$。

(4) 一次関数 $y = ax + b$ が 3点 $(1, 3)$、$(3, 9)$、$(k, 15)$ を通るとき、$a$、$b$、$k$ の値を求めよ。
→ $a = 3, b = 0, k = 5$。$(1, 3)$: $a + b = 3$ …①、$(3, 9)$: $3a + b = 9$ …②。②−①より $2a = 6$、$a = 3$、$b = 0$。$y = 3x$ に $y = 15$ を代入: $15 = 3k$、$k = 5$。
