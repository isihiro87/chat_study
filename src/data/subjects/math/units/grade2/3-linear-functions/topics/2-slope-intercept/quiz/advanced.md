# グラフの傾きと切片 発展問題

## 問1: 変域の応用

(1) $y = 2x + 3$ で $x$ の変域が $a \leqq x \leqq a + 2$ のとき、$y$ の変域を $a$ を使って表しなさい。
→ 傾き正なので、$x = a$ のとき $y = 2a + 3$、$x = a + 2$ のとき $y = 2(a + 2) + 3 = 2a + 7$。よって $2a + 3 \leqq y \leqq 2a + 7$。

(2) $y = -3x + 1$ で $x$ の変域が $-2 \leqq x \leqq 4$ のとき、$y$ の変域を求めなさい。
→ 傾き負なので、$x = -2$ のとき $y = 7$、$x = 4$ のとき $y = -11$。よって $-11 \leqq y \leqq 7$。

(3) $y = ax + 2$ で $x$ の変域が $0 \leqq x \leqq 3$ のとき、$y$ の変域が $2 \leqq y \leqq 8$ であった。$a$ の値を求めなさい。
→ $x = 0$ のとき $y = 2$（一致 ✓）。$x = 3$ のとき $y = 3a + 2 = 8$ より $a = 2$。

(4) $y = ax + 5$ で $x$ の変域が $0 \leqq x \leqq 4$ のとき、$y$ の変域が $-3 \leqq y \leqq 5$ であった。$a$ の値を求めなさい。
→ $x = 0$ のとき $y = 5$（最大値と一致、$a < 0$）。$x = 4$ のとき $y = 4a + 5 = -3$ より $a = -2$。

## 問2: グラフの平行移動

(1) $y = 2x$ のグラフを $y$ 軸方向に $+5$ だけ平行移動すると、どんな式になるか。
→ $y = 2x + 5$。傾きは変わらず、切片が $0$ から $5$ になる。

(2) $y = -3x + 2$ のグラフを $y$ 軸方向に $-7$ だけ平行移動すると、どんな式になるか。
→ $y = -3x + 2 + (-7) = -3x - 5$。傾きは変わらず、切片が $2 - 7 = -5$ になる。

(3) $y = \dfrac{1}{2}x - 3$ のグラフを $y$ 軸方向に平行移動して点 $(0, 4)$ を通るようにするには、何だけ移動すればよいか。移動後の式も答えなさい。
→ もとの切片は $-3$、移動後の切片は $4$ なので、$4 - (-3) = 7$ だけ平行移動する。移動後の式は $y = \dfrac{1}{2}x + 4$。

(4) $y = ax + b$ のグラフが $y = -2x + 1$ のグラフと平行で、点 $(0, -3)$ を通る。$a, b$ の値を求めなさい。
→ 平行なので傾き $a = -2$。点 $(0, -3)$ を通るので切片 $b = -3$。式は $y = -2x - 3$。

(5) $y = 3x + 1$ のグラフと平行で、$y = x - 5$ のグラフと切片が等しい直線の式を求めなさい。
→ 平行なので傾き $a = 3$。$y = x - 5$ の切片は $-5$ なので $b = -5$。式は $y = 3x - 5$。

## 問3: 傾きと切片の条件からグラフの特徴を判断

(1) $y = ax + b$ で $a > 0, b > 0$ のとき、グラフはどの象限を通るか。
→ 第1、第2、第3象限を通る。右上がりで、$y$ 軸との交点が正なので、第4象限は通らない場合もあるが、直線は第1・第2・第3象限を必ず通る。（正確には $b > 0$ かつ右上がりの直線は第1・2・3象限を通る。）

(2) $y = ax + b$ で $a < 0, b > 0$ のとき、グラフは $x$ 軸と正の部分で交わるか負の部分で交わるか。
→ 正の部分で交わる。$y = 0$ のとき $ax + b = 0$ より $x = -\dfrac{b}{a}$。$a < 0, b > 0$ なので $-\dfrac{b}{a} > 0$。

(3) $y = ax + b$ で $a > 0, b < 0$ のとき、グラフは $x$ 軸と正の部分で交わるか負の部分で交わるか。
→ 正の部分で交わる。$x = -\dfrac{b}{a}$。$a > 0, b < 0$ なので $-\dfrac{b}{a} > 0$。

(4) 直線 $y = 2x + b$ が点 $(3, 5)$ を通るとき、$b$ の値を求めなさい。
→ $5 = 2 \times 3 + b$ より $b = -1$。式は $y = 2x - 1$。

(5) 直線 $y = ax + 4$ が点 $(2, 0)$ を通るとき、$a$ の値を求めなさい。
→ $0 = 2a + 4$ より $a = -2$。式は $y = -2x + 4$。

(6) 直線 $y = ax + b$ が点 $(2, 7)$ と点 $(5, 1)$ を通るとき、$a, b$ の値を求めなさい。
→ 傾き $a = \dfrac{1 - 7}{5 - 2} = \dfrac{-6}{3} = -2$。$7 = -2 \times 2 + b$ より $b = 11$。式は $y = -2x + 11$。

## 問4: 傾きの大小とグラフの比較

(1) 次の3つの直線を、グラフの急さの順に並べなさい（急な方から）。
$y = 5x + 1$、$y = -3x + 2$、$y = x - 4$
→ $|5| > |-3| > |1|$ なので、$y = 5x + 1$、$y = -3x + 2$、$y = x - 4$ の順。

(2) $y = 2x + 3$ と $y = 2x - 1$ のグラフの位置関係を説明しなさい。
→ 傾きが等しいので平行。$y = 2x - 1$ を $y$ 軸方向に $4$ だけ平行移動すると $y = 2x + 3$ になる。

(3) $y = 2x + 3$ と $y = -2x + 3$ のグラフの関係を説明しなさい。
→ 切片は同じ $(0, 3)$ で、傾きの絶対値が等しく符号が逆。2直線は点 $(0, 3)$ で交わり、$y$ 軸について対称。

## 問5: 総合問題

(1) 一次関数 $y = -\dfrac{2}{3}x + 4$ について、次の問いに答えなさい。
(a) 傾きと切片を答えなさい。
→ 傾き $-\dfrac{2}{3}$、切片 $4$。

(b) グラフは右上がりか右下がりか。
→ 右下がり。傾きが負だから。

(c) 切片の座標と、そこから傾きを使ってとる次の点の座標を答えなさい。
→ 切片 $(0, 4)$、次の点 $(3, 2)$（右に3、下に2）。

(d) $x$ の変域が $-3 \leqq x \leqq 6$ のとき、$y$ の変域を求めなさい。
→ $x = -3$ のとき $y = -\dfrac{2}{3} \times (-3) + 4 = 6$、$x = 6$ のとき $y = -\dfrac{2}{3} \times 6 + 4 = 0$。傾き負なので $0 \leqq y \leqq 6$。
