# 一次関数の利用 発展問題

## 問1: グラフの読みとり応用

次の問いに答えなさい。

(1) あおいさんは家から 2400m 離れた図書館に向かって分速 80m で歩き始めた。出発して 15 分後に忘れ物に気づき、すぐに分速 120m で家に引き返した。家に着いてから 3 分で準備し、再び分速 80m で図書館に向かった。あおいさんが図書館に着くのは最初に出発してから何分後か。
→ 15 分後の位置: $80 \times 15 = 1200$ m。引き返し時間: $1200 \div 120 = 10$ 分（25 分後に家に到着）。準備 3 分（28 分後に再出発）。図書館まで: $2400 \div 80 = 30$ 分。$28 + 30 = 58$ 分後。

(2) 上の問題で、出発してから $x$ 分後の家からの距離 $y$ m を表すグラフを、4 つの区間に分けてそれぞれ式を求めなさい。
→ $0 \leqq x \leqq 15$: $y = 80x$。$15 \leqq x \leqq 25$: $y = -120(x - 15) + 1200 = -120x + 3000$。$25 \leqq x \leqq 28$: $y = 0$。$28 \leqq x \leqq 58$: $y = 80(x - 28) = 80x - 2240$。

(3) 上の問題で、あおいさんが家から 960m の地点を通過するのは全部で何回か。またそれぞれ出発してから何分後か。
→ $80x = 960$ より $x = 12$（1 回目）。$-120x + 3000 = 960$ より $x = 17$（2 回目）。$80x - 2240 = 960$ より $x = 40$（3 回目）。合計 3 回。12 分後、17 分後、40 分後。

(4) みかさんは、あおいさんが最初に出発してから 20 分後に図書館を出発して、分速 60m であおいさんの家に向かった。みかさんとあおいさんが出会うのは何分後か。
→ みかさん: $x \geqq 20$ で $y = -60(x - 20) + 2400 = -60x + 3600$。あおいさんが引き返し中（$15 \leqq x \leqq 25$）: $-120x + 3000 = -60x + 3600$、$-60x = 600$、$x = -10$（不適）。あおいさんの再出発後（$28 \leqq x$）: $80x - 2240 = -60x + 3600$、$140x = 5840$、$x = \dfrac{5840}{140} \fallingdotseq 41.7$。$x \fallingdotseq 41\dfrac{5}{7}$ 分後（約 41 分 43 秒後）。正確には $\dfrac{292}{7}$ 分後。

## 問2: 水量の応用

次の問いに答えなさい。

(1) 容量 80L の空の水そうに、蛇口 A から毎分 5L、蛇口 B から毎分 3L の水を同時に入れる。水そうが満水になるのは何分後か。
→ 合計毎分 $5 + 3 = 8$ L。$8x = 80$、$x = 10$。10 分後。

(2) 上の問題で、満水になった後、蛇口 A を閉じて蛇口 B だけで水を入れ続けながら、栓を抜いて毎分 7L ずつ水を抜く。水そうが空になるのは満水になってから何分後か。
→ 毎分の水量変化: $3 - 7 = -4$ L/分。$-4t + 80 = 0$、$t = 20$。20 分後。

(3) 容量 100L の水そうに 40L の水が入っている。毎分 $a$ L ずつ水を入れて 15 分で満水にしたい。$a$ の値を求めなさい。
→ $a \times 15 + 40 = 100$、$15a = 60$、$a = 4$。毎分 4L。

(4) 2 つの水そう A（容量 90L、初期水量 0L）と B（容量 90L、初期水量 90L）がある。同時に A に毎分 6L ずつ水を入れ、B から毎分 4L ずつ水を抜く。2 つの水そうの水量が等しくなるのは何分後か。
→ A: $y = 6x$、B: $y = -4x + 90$。$6x = -4x + 90$、$10x = 90$、$x = 9$。9 分後（水量はともに 54L）。

## 問3: 動く点の応用

長方形 ABCD で AB $= 10$ cm、BC $= 6$ cm とする。点 P は A を出発して辺 AB、BC、CD 上を B, C を経由して D まで毎秒 2cm で動く。出発してから $x$ 秒後の $\triangle APD$ の面積を $y$ cm² とする。

(1) 点 P が辺 AB 上にあるとき（$0 \leqq x \leqq 5$）、$y$ を $x$ の式で表しなさい。
→ AP $= 2x$、高さ AD $= 6$。$y = \dfrac{1}{2} \times 2x \times 6 = 6x$。

(2) 点 P が辺 BC 上にあるとき（$5 \leqq x \leqq 8$）、$y$ を $x$ の式で表しなさい。
→ 底辺 AD $= 6$、高さ AB $= 10$。$y = \dfrac{1}{2} \times 6 \times 10 = 30$（一定）。

(3) 点 P が辺 CD 上にあるとき（$8 \leqq x \leqq 13$）、$y$ を $x$ の式で表しなさい。
→ P は C から D に向かう。DP $= $ CD $-$ CP $= 10 - 2(x - 8) = 10 - 2x + 16 = 26 - 2x$。高さ AD $= 6$。$y = \dfrac{1}{2} \times (26 - 2x) \times 6 = 78 - 6x$。

(4) $y = 18$ となる $x$ の値をすべて求めなさい。
→ $6x = 18$ より $x = 3$（AB 上）。$y = 30$ なので BC 上では不適。$78 - 6x = 18$ より $x = 10$（CD 上）。$x = 3, 10$。

(5) グラフの概形を描き、面積が最大になるのは何秒後から何秒後の間か、また最大値を求めなさい。
→ $0 \leqq x \leqq 5$: $y = 6x$（右上がり）。$5 \leqq x \leqq 8$: $y = 30$（水平）。$8 \leqq x \leqq 13$: $y = 78 - 6x$（右下がり）。面積最大は $5 \leqq x \leqq 8$ の間で、最大値 30 cm²。

## 問4: 2人の移動と交差の応用

A 町と B 町は 12km 離れている。太郎さんは A 町から B 町に向かって時速 4km で歩き、花子さんは太郎さんと同時に B 町から A 町に向かって時速 3km で自転車で走った。しかし花子さんは途中で自転車がパンクし、10 分間修理した後、時速 5km で歩いて A 町に向かった。2 人は太郎さんの出発から $1\dfrac{5}{7}$ 時間後に出会った。

(1) 太郎さんの出発から $x$ 時間後の A 町からの距離を $y$ km とするとき、$y$ を $x$ の式で表しなさい。
→ $y = 4x$。

(2) 花子さんが自転車で走っていた時間を $t$ 時間とする。花子さんが自転車で走った距離を $t$ を使って表しなさい。
→ $3t$ km。

(3) 花子さんの自転車がパンクしたのは出発してから何時間後か求めなさい。ただし、修理中と歩き中の花子さんの式を用いて連立方程式を立てて解くこと。
→ パンク時刻を $t$ 時間後とする。パンク位置: B 町から $3t$ km → A 町から $12 - 3t$ km。修理時間 $\dfrac{1}{6}$ 時間。歩き区間の式: $y = -(5)(x - t - \dfrac{1}{6}) + 12 - 3t = -5x + 5t + \dfrac{5}{6} + 12 - 3t = -5x + 2t + \dfrac{77}{6}$。出会い: $4 \times \dfrac{12}{7} = -5 \times \dfrac{12}{7} + 2t + \dfrac{77}{6}$。$\dfrac{48}{7} = -\dfrac{60}{7} + 2t + \dfrac{77}{6}$。$\dfrac{48}{7} + \dfrac{60}{7} = 2t + \dfrac{77}{6}$。$\dfrac{108}{7} = 2t + \dfrac{77}{6}$。$2t = \dfrac{108}{7} - \dfrac{77}{6} = \dfrac{648 - 539}{42} = \dfrac{109}{42}$。$t = \dfrac{109}{84}$ 時間。出発してから約 1 時間 18 分後。

## 問5: グラフと面積の総合

座標平面上に 2 直線 $y = 2x$、$y = -x + 6$ がある。

(1) 2 直線の交点の座標を求めなさい。
→ $2x = -x + 6$、$3x = 6$、$x = 2$、$y = 4$。交点 $(2, 4)$。

(2) 2 直線と $x$ 軸で囲まれた三角形の 3 つの頂点の座標を求めなさい。
→ $y = 2x$ と $x$ 軸の交点: $(0, 0)$。$y = -x + 6$ と $x$ 軸の交点: $y = 0$ より $x = 6$、$(6, 0)$。交点: $(2, 4)$。3 頂点は $(0, 0)$, $(6, 0)$, $(2, 4)$。

(3) (2) の三角形の面積を求めなさい。
→ 底辺 $= 6$（$x$ 軸上の $(0,0)$ から $(6,0)$）、高さ $= 4$（交点の $y$ 座標）。面積 $= \dfrac{1}{2} \times 6 \times 4 = 12$。

(4) $y = k$（$0 < k < 4$）がこの三角形を面積の等しい 2 つの部分に分けるとき、$k$ の値を求めなさい。
→ $y = k$ と $y = 2x$ の交点: $x = \dfrac{k}{2}$。$y = k$ と $y = -x + 6$ の交点: $x = 6 - k$。上の三角形の底辺 $= (6 - k) - \dfrac{k}{2} = 6 - \dfrac{3k}{2}$。上の三角形の高さ $= 4 - k$。上の三角形の面積 $= \dfrac{1}{2}(6 - \dfrac{3k}{2})(4 - k)$。全体の半分 $= 6$。$\dfrac{1}{2}(6 - \dfrac{3k}{2})(4 - k) = 6$。$(6 - \dfrac{3k}{2})(4 - k) = 12$。$24 - 6k - 6k + \dfrac{3k^2}{2} = 12$。$\dfrac{3k^2}{2} - 12k + 12 = 0$。$3k^2 - 24k + 24 = 0$。$k^2 - 8k + 8 = 0$。$k = \dfrac{8 \pm \sqrt{64 - 32}}{2} = \dfrac{8 \pm 4\sqrt{2}}{2} = 4 \pm 2\sqrt{2}$。$0 < k < 4$ より $k = 4 - 2\sqrt{2}$。
