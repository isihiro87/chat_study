# 方程式とグラフ 発展問題

## 問1: 交点と定数の決定

(1) 2直線 $y = ax + 1$ と $y = -2x + 7$ の交点の $x$ 座標が $2$ であるとき、$a$ の値を求めよ。
→ $x = 2$ を $y = -2x + 7$ に代入: $y = -4 + 7 = 3$。交点は $(2, 3)$。$y = ax + 1$ に代入: $3 = 2a + 1$、$2a = 2$、$a = 1$。

(2) 直線 $y = 3x + b$ が直線 $y = -x + 8$ と点 $(1, c)$ で交わるとき、$b$ と $c$ の値を求めよ。
→ $y = -x + 8$ に $x = 1$ を代入: $c = -1 + 8 = 7$。$y = 3x + b$ に $(1, 7)$ を代入: $7 = 3 + b$、$b = 4$。$b = 4, c = 7$。

(3) 2直線 $y = 2x + a$ と $y = -x + b$ の交点が $(3, 5)$ であるとき、$a, b$ の値を求めよ。
→ $5 = 2 \times 3 + a$ より $a = -1$。$5 = -3 + b$ より $b = 8$。

(4) 直線 $ax + 2y = 10$ が点 $(2, 3)$ を通るとき、$a$ の値を求めよ。また、この直線を $y =$ の形に変形せよ。
→ $2a + 6 = 10$、$2a = 4$、$a = 2$。$2x + 2y = 10$ → $y = -x + 5$。

## 問2: 三角形の面積

(1) $y = x + 1$ と $y = -\frac{1}{2}x + 4$ と $x$ 軸で囲まれた三角形の面積を求めよ。
→ 2直線の交点: $x + 1 = -\frac{1}{2}x + 4$、$\frac{3}{2}x = 3$、$x = 2$、$y = 3$ → $(2, 3)$。$y = x + 1$ の $x$ 切片: $0 = x + 1$、$x = -1$ → $(-1, 0)$。$y = -\frac{1}{2}x + 4$ の $x$ 切片: $0 = -\frac{1}{2}x + 4$、$x = 8$ → $(8, 0)$。底辺 $= 8 - (-1) = 9$、高さ $= 3$。面積 $= \frac{1}{2} \times 9 \times 3 = \frac{27}{2}$。

(2) $y = 3x$ と $y = -x + 8$ と $y$ 軸で囲まれた三角形の面積を求めよ。
→ 2直線の交点: $3x = -x + 8$、$4x = 8$、$x = 2$、$y = 6$ → $(2, 6)$。$y$ 切片: $y = 3x$ は $(0, 0)$、$y = -x + 8$ は $(0, 8)$。底辺（$y$ 軸上）$= 8 - 0 = 8$、高さ $= 2$。面積 $= \frac{1}{2} \times 8 \times 2 = 8$。

(3) $y = \frac{1}{2}x + 1$ と $y = -x + 7$ と $x$ 軸で囲まれた三角形の面積を求めよ。
→ 交点: $\frac{1}{2}x + 1 = -x + 7$、$\frac{3}{2}x = 6$、$x = 4$、$y = 3$ → $(4, 3)$。$x$ 切片: $y = \frac{1}{2}x + 1$ は $x = -2$ → $(-2, 0)$。$y = -x + 7$ は $x = 7$ → $(7, 0)$。底辺 $= 7 - (-2) = 9$、高さ $= 3$。面積 $= \frac{1}{2} \times 9 \times 3 = \frac{27}{2}$。

## 問3: 面積を2等分する直線

(1) $y = x + 2$、$y = -2x + 8$、$x$ 軸で囲まれた三角形の面積を求めよ。
→ 交点: $x + 2 = -2x + 8$、$3x = 6$、$x = 2$、$y = 4$ → $(2, 4)$。$x$ 切片: $(-2, 0)$ と $(4, 0)$。底辺 $= 4 - (-2) = 6$、高さ $= 4$。面積 $= \frac{1}{2} \times 6 \times 4 = 12$。

(2) (1)の三角形で、頂点 $(2, 4)$ を通り面積を2等分する直線の式を求めよ。
→ 対辺は $(-2, 0)$ と $(4, 0)$ を結ぶ $x$ 軸上の線分。中点 $= (1, 0)$。$(2, 4)$ と $(1, 0)$ を通る直線: 傾き $= \frac{4 - 0}{2 - 1} = 4$、$y = 4x - 4$。

(3) (1)の三角形で、頂点 $(-2, 0)$ を通り面積を2等分する直線の式を求めよ。
→ 対辺は $(2, 4)$ と $(4, 0)$ を結ぶ線分。中点 $= (3, 2)$。$(-2, 0)$ と $(3, 2)$ を通る直線: 傾き $= \frac{2 - 0}{3 - (-2)} = \frac{2}{5}$、$y = \frac{2}{5}(x + 2) = \frac{2}{5}x + \frac{4}{5}$。

## 問4: 複数の直線で囲まれた図形

(1) 3直線 $y = x$, $y = -x + 6$, $y = 0$（$x$ 軸）で囲まれた三角形の3頂点を求めよ。
→ $y = x$ と $y = -x + 6$ の交点: $(3, 3)$。$y = x$ と $x$ 軸の交点: $(0, 0)$。$y = -x + 6$ と $x$ 軸の交点: $(6, 0)$。

(2) (1)の三角形の面積を求めよ。
→ 底辺 $= 6$、高さ $= 3$。面積 $= \frac{1}{2} \times 6 \times 3 = 9$。

(3) 3直線 $y = 2x$, $y = -x + 9$, $x = 0$（$y$ 軸）で囲まれた三角形の面積を求めよ。
→ $y = 2x$ と $y = -x + 9$ の交点: $2x = -x + 9$、$x = 3$、$y = 6$ → $(3, 6)$。$y$ 切片: $(0, 0)$ と $(0, 9)$。底辺（$y$ 軸上）$= 9$、高さ $= 3$。面積 $= \frac{1}{2} \times 9 \times 3 = \frac{27}{2}$。

(4) 2直線 $y = x + 2$ と $y = -\frac{1}{2}x + 5$ と $x$ 軸で囲まれた三角形の面積を求め、さらにこの三角形の面積を $x$ 軸上の頂点 $(-2, 0)$ を通る直線で2等分するとき、その直線の式を求めよ。
→ 交点: $x + 2 = -\frac{1}{2}x + 5$、$\frac{3}{2}x = 3$、$x = 2$、$y = 4$ → $(2, 4)$。$x$ 切片: $(-2, 0)$ と $(10, 0)$。底辺 $= 12$、高さ $= 4$。面積 $= \frac{1}{2} \times 12 \times 4 = 24$。頂点 $(-2, 0)$ の対辺は $(2, 4)$ と $(10, 0)$ を結ぶ線分。中点 $= (6, 2)$。$(-2, 0)$ と $(6, 2)$ を通る直線: 傾き $= \frac{2}{8} = \frac{1}{4}$、$y = \frac{1}{4}(x + 2) = \frac{1}{4}x + \frac{1}{2}$。

## 問5: グラフの読み取りと総合問題

(1) 2直線 $y = ax + 3$ と $y = -2x + b$ が点 $(2, 1)$ で交わるとき、$a, b$ の値を求めよ。さらに、この2直線と $x$ 軸で囲まれた三角形の面積を求めよ。
→ $1 = 2a + 3$ より $a = -1$。$1 = -4 + b$ より $b = 5$。2直線は $y = -x + 3$ と $y = -2x + 5$。$x$ 切片: $(3, 0)$ と $(\frac{5}{2}, 0)$。底辺 $= 3 - \frac{5}{2} = \frac{1}{2}$、高さ $= 1$。面積 $= \frac{1}{2} \times \frac{1}{2} \times 1 = \frac{1}{4}$。

(2) 直線 $y = 2x - 4$ と $x$ 軸、$y$ 軸で囲まれた三角形の面積を求めよ。
→ $x$ 切片: $0 = 2x - 4$、$x = 2$ → $(2, 0)$。$y$ 切片: $(0, -4)$。底辺 $= 2$（$x$ 軸上、原点から $(2, 0)$）、高さ $= 4$（$y$ 軸上、原点から $(0, -4)$の距離）。面積 $= \frac{1}{2} \times 2 \times 4 = 4$。

(3) 直線 $y = \frac{3}{4}x - 3$ と $x$ 軸、$y$ 軸で囲まれた三角形の面積を求めよ。さらに、原点を通りこの三角形の面積を2等分する直線の式を求めよ。
→ $x$ 切片: $0 = \frac{3}{4}x - 3$、$x = 4$ → $(4, 0)$。$y$ 切片: $(0, -3)$。面積 $= \frac{1}{2} \times 4 \times 3 = 6$。三角形の頂点は $(0, 0)$, $(4, 0)$, $(0, -3)$。原点は頂点なので対辺は $(4, 0)$ と $(0, -3)$ の線分。中点 $= (2, -\frac{3}{2})$。原点と中点を通る直線: $y = -\frac{3}{4}x$。
