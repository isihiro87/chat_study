# 一次関数の意味と変化の割合 発展問題

## 問1: 反比例との変化の割合の比較

(1) 一次関数 $y = 2x + 1$ について、$x$ が $1$ から $3$、$x$ が $3$ から $5$、$x$ が $5$ から $7$ のそれぞれの区間で変化の割合を求めなさい。
→ すべての区間で変化の割合は $2$。$x$ が $1$ から $3$: $\dfrac{7 - 3}{3 - 1} = 2$。$x$ が $3$ から $5$: $\dfrac{11 - 7}{5 - 3} = 2$。$x$ が $5$ から $7$: $\dfrac{15 - 11}{7 - 5} = 2$。一次関数なので変化の割合は常に一定。

(2) 反比例 $y = \dfrac{12}{x}$ について、$x$ が $1$ から $2$、$x$ が $2$ から $4$、$x$ が $4$ から $6$ のそれぞれの区間で変化の割合を求めなさい。
→ $x$ が $1$ から $2$: $\dfrac{6 - 12}{2 - 1} = -6$。$x$ が $2$ から $4$: $\dfrac{3 - 6}{4 - 2} = -\dfrac{3}{2}$。$x$ が $4$ から $6$: $\dfrac{2 - 3}{6 - 4} = -\dfrac{1}{2}$。区間によって変化の割合が異なる。

(3) (1)(2)の結果から、一次関数と反比例の変化の割合のちがいを説明しなさい。
→ 一次関数の変化の割合はどの区間でも一定（$= a$）だが、反比例の変化の割合は区間によって異なり一定ではない。これが一次関数の最大の特徴である。

(4) $y = x^2$ について、$x$ が $1$ から $2$、$x$ が $2$ から $3$、$x$ が $3$ から $4$ のそれぞれの区間で変化の割合を求め、一定かどうか判断しなさい。
→ $x$ が $1$ から $2$: $\dfrac{4 - 1}{1} = 3$。$x$ が $2$ から $3$: $\dfrac{9 - 4}{1} = 5$。$x$ が $3$ から $4$: $\dfrac{16 - 9}{1} = 7$。変化の割合は一定ではないので、$y = x^2$ は一次関数ではない。

## 問2: 複合的な増加量の計算

(1) 一次関数 $y = ax + 3$ で、$x$ が $2$ から $6$ まで増加すると $y$ が $-12$ 変化する。$a$ の値を求めなさい。
→ $x$ の増加量 $= 4$。変化の割合 $= \dfrac{-12}{4} = -3 = a$。よって $a = -3$。

(2) 一次関数 $y = ax + b$ で、$x = 1$ のとき $y = 5$、$x = 4$ のとき $y = 14$ である。$a$ と $b$ の値を求めなさい。
→ 変化の割合 $a = \dfrac{14 - 5}{4 - 1} = \dfrac{9}{3} = 3$。$x = 1$ を代入: $5 = 3 \times 1 + b$ より $b = 2$。よって $a = 3$, $b = 2$。

(3) 一次関数 $y = ax + b$ で、$x = -2$ のとき $y = 11$、$x = 3$ のとき $y = 1$ である。$a$ と $b$ の値を求めなさい。
→ $a = \dfrac{1 - 11}{3 - (-2)} = \dfrac{-10}{5} = -2$。$x = 3$ を代入: $1 = -2 \times 3 + b$ より $b = 7$。よって $a = -2$, $b = 7$。

(4) 一次関数 $y = ax + b$ で、$x$ が $3$ 増加するごとに $y$ が $6$ 増加し、$x = 0$ のとき $y = -4$ である。この一次関数の式を求めなさい。
→ $a = \dfrac{6}{3} = 2$。$x = 0$ のとき $y = b = -4$。よって $y = 2x - 4$。

## 問3: 変域と一次関数

(1) $y = 3x - 2$ で $x$ の変域が $1 \leqq x \leqq 5$ のとき、$y$ の変域を求めなさい。
→ $x = 1$ のとき $y = 1$、$x = 5$ のとき $y = 13$。$a = 3 > 0$ なので $y$ は増加する。$y$ の変域は $1 \leqq y \leqq 13$。

(2) $y = -2x + 10$ で $x$ の変域が $0 \leqq x \leqq 4$ のとき、$y$ の変域を求めなさい。
→ $x = 0$ のとき $y = 10$、$x = 4$ のとき $y = 2$。$a = -2 < 0$ なので $y$ は減少する。$y$ の変域は $2 \leqq y \leqq 10$。

(3) $y = 4x + 1$ で $y$ の変域が $5 \leqq y \leqq 21$ のとき、$x$ の変域を求めなさい。
→ $y = 5$ のとき $5 = 4x + 1$ より $x = 1$。$y = 21$ のとき $21 = 4x + 1$ より $x = 5$。$x$ の変域は $1 \leqq x \leqq 5$。

(4) $y = -3x + b$ で $x$ の変域が $-1 \leqq x \leqq 2$ のとき、$y$ の変域が $a \leqq y \leqq 9$ であった。$a$ と $b$ の値を求めなさい。
→ $a = -3 < 0$ より、$x = -1$ で $y$ が最大、$x = 2$ で $y$ が最小。$x = -1$ のとき $y = 3 + b = 9$ より $b = 6$。$x = 2$ のとき $y = -6 + 6 = 0 = a$。よって $a = 0$, $b = 6$。

## 問4: 一次関数の応用文章題

(1) 2つのバネA, Bがある。バネAは自然長が10cmで、1gのおもりをつるすごとに0.5cm伸びる。バネBは自然長が6cmで、1gのおもりをつるすごとに0.8cm伸びる。おもりを $x$ gつるしたときのバネの長さ $y$ cmをそれぞれ式で表しなさい。
→ バネA: $y = 0.5x + 10$。バネB: $y = 0.8x + 6$。

(2) (1) で、バネAとバネBの長さが等しくなるのは、おもりが何gのときか。
→ $0.5x + 10 = 0.8x + 6$ より $4 = 0.3x$、$x = \dfrac{40}{3} \fallingdotseq 13.3$（g）。

(3) 兄は家から3km離れた図書館に向かって分速80mで歩いている。弟は兄が出発してから10分後に、家から分速200mの自転車で同じ道を追いかけた。弟が出発してからの時間を $x$ 分として、兄と弟それぞれの家からの距離 $y$ mを式で表しなさい。
→ 兄は弟が出発した時点で $80 \times 10 = 800$m 進んでいる。兄: $y = 80x + 800$。弟: $y = 200x$。

(4) (3) で、弟が兄に追いつくのは、弟が出発してから何分後か。そのとき家から何mの地点か。
→ $200x = 80x + 800$ より $120x = 800$、$x = \dfrac{20}{3} \fallingdotseq 6.7$（分後）。$y = 200 \times \dfrac{20}{3} = \dfrac{4000}{3} \fallingdotseq 1333$（m）。

(5) 水そうに80Lの水が入っている。ポンプAは毎分4Lの水を入れ、同時にポンプBは毎分2Lの水を排出する。$x$ 分後の水の量 $y$ Lを式で表し、水そうの容量が200Lのとき何分後に満水になるか求めなさい。
→ 毎分の水の増減は $4 - 2 = 2$（L）。$y = 2x + 80$。$200 = 2x + 80$ より $2x = 120$、$x = 60$。60分後に満水になる。
