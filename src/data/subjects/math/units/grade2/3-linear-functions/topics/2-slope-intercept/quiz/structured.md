# グラフの傾きと切片 大問小問

## 大問1: 傾きと切片を読み取る（配点 20点）

次の一次関数について、傾きと切片をそれぞれ答えなさい。

(1) $y = 3x + 5$（3点）
→ 傾き $3$、切片 $5$。$y = ax + b$ の $a$ が傾き、$b$ が切片。

(2) $y = -2x + 4$（3点）
→ 傾き $-2$、切片 $4$。

(3) $y = x - 7$（3点）
→ 傾き $1$、切片 $-7$。$x$ の係数が省略されているときは $1$。

(4) $y = -x$（3点）
→ 傾き $-1$、切片 $0$。$-x = -1 \times x$ で、$b = 0$ は省略されている。

(5) $y = \dfrac{2}{3}x + 1$（4点）
→ 傾き $\dfrac{2}{3}$、切片 $1$。

(6) $y = -\dfrac{3}{4}x - 2$（4点）
→ 傾き $-\dfrac{3}{4}$、切片 $-2$。

## 大問2: グラフをかく（整数の傾き）（配点 20点）

次の一次関数のグラフについて、切片の座標と、切片の次にとる点の座標を答え、グラフが右上がりか右下がりかを答えなさい。

(1) $y = 2x + 1$（4点）
→ 切片 $(0, 1)$、次の点 $(1, 3)$（右に1、上に2）。右上がり。

(2) $y = -3x + 6$（4点）
→ 切片 $(0, 6)$、次の点 $(1, 3)$（右に1、下に3）。右下がり。

(3) $y = x - 2$（4点）
→ 切片 $(0, -2)$、次の点 $(1, -1)$（右に1、上に1）。右上がり。

(4) $y = -x + 4$（4点）
→ 切片 $(0, 4)$、次の点 $(1, 3)$（右に1、下に1）。右下がり。

(5) $y = 4x - 3$（4点）
→ 切片 $(0, -3)$、次の点 $(1, 1)$（右に1、上に4）。右上がり。

## 大問3: グラフをかく（分数の傾き）（配点 20点）

次の一次関数のグラフについて、切片の座標と、切片の次にとる点の座標を答えなさい。

(1) $y = \dfrac{1}{2}x + 3$（4点）
→ 切片 $(0, 3)$、次の点 $(2, 4)$（右に2、上に1）。

(2) $y = \dfrac{2}{3}x + 1$（4点）
→ 切片 $(0, 1)$、次の点 $(3, 3)$（右に3、上に2）。

(3) $y = -\dfrac{1}{3}x + 2$（4点）
→ 切片 $(0, 2)$、次の点 $(3, 1)$（右に3、下に1）。

(4) $y = -\dfrac{3}{2}x + 6$（4点）
→ 切片 $(0, 6)$、次の点 $(2, 3)$（右に2、下に3）。

(5) $y = \dfrac{4}{5}x - 2$（4点）
→ 切片 $(0, -2)$、次の点 $(5, 2)$（右に5、上に4）。

## 大問4: $x$ の変域から $y$ の変域を求める（配点 20点）

次の一次関数で、与えられた $x$ の変域に対する $y$ の変域を求めなさい。

(1) $y = 2x + 1$、$x$ の変域: $1 \leqq x \leqq 4$（3点）
→ $x = 1$ のとき $y = 3$、$x = 4$ のとき $y = 9$。傾き正なので $3 \leqq y \leqq 9$。

(2) $y = -3x + 10$、$x$ の変域: $0 \leqq x \leqq 3$（3点）
→ $x = 0$ のとき $y = 10$、$x = 3$ のとき $y = 1$。傾き負なので $1 \leqq y \leqq 10$。

(3) $y = \dfrac{1}{2}x + 2$、$x$ の変域: $-2 \leqq x \leqq 6$（4点）
→ $x = -2$ のとき $y = 1$、$x = 6$ のとき $y = 5$。$1 \leqq y \leqq 5$。

(4) $y = -2x + 5$、$x$ の変域: $-1 \leqq x \leqq 3$（4点）
→ $x = -1$ のとき $y = 7$、$x = 3$ のとき $y = -1$。傾き負なので $-1 \leqq y \leqq 7$。

(5) $y = \dfrac{2}{3}x - 1$、$x$ の変域: $-3 \leqq x \leqq 6$（3点）
→ $x = -3$ のとき $y = -3$、$x = 6$ のとき $y = 3$。$-3 \leqq y \leqq 3$。

(6) $y = -\dfrac{3}{4}x + 3$、$x$ の変域: $0 \leqq x \leqq 8$（3点）
→ $x = 0$ のとき $y = 3$、$x = 8$ のとき $y = -3$。傾き負なので $-3 \leqq y \leqq 3$。

## 大問5: グラフから式を読み取る（配点 20点）

次の条件をもつ直線の式を $y = ax + b$ の形で求めなさい。

(1) $y$ 軸上の点 $(0, 3)$ を通り、$x$ が1増えると $y$ が2増える。（3点）
→ 切片 $b = 3$、傾き $a = 2$。$y = 2x + 3$。

(2) $y$ 軸上の点 $(0, -1)$ を通り、$x$ が1増えると $y$ が4減る。（3点）
→ 切片 $b = -1$、傾き $a = -4$。$y = -4x - 1$。

(3) 点 $(0, 2)$ と点 $(3, 4)$ を通る直線。（4点）
→ 切片 $b = 2$。傾き $a = \dfrac{4 - 2}{3 - 0} = \dfrac{2}{3}$。$y = \dfrac{2}{3}x + 2$。

(4) 点 $(0, 5)$ と点 $(2, 1)$ を通る直線。（4点）
→ 切片 $b = 5$。傾き $a = \dfrac{1 - 5}{2 - 0} = \dfrac{-4}{2} = -2$。$y = -2x + 5$。

(5) 点 $(0, -2)$ と点 $(4, 1)$ を通る直線。（3点）
→ 切片 $b = -2$。傾き $a = \dfrac{1 - (-2)}{4 - 0} = \dfrac{3}{4}$。$y = \dfrac{3}{4}x - 2$。

(6) 比例 $y = 3x$ のグラフを $y$ 軸方向に $-4$ だけ平行移動した直線。（3点）
→ 傾きは $3$ のまま、切片が $-4$ になる。$y = 3x - 4$。
