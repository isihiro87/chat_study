# 一次関数の式を求める 一問一答

## 傾きと切片がわかるとき

Q: 傾き $2$、切片 $3$ の一次関数の式を求めよ。
A: $y = 2x + 3$。$y = ax + b$ に $a = 2$、$b = 3$ を代入する。

Q: 傾き $-3$、切片 $5$ の一次関数の式を求めよ。
A: $y = -3x + 5$。$y = ax + b$ に $a = -3$、$b = 5$ を代入する。

Q: 傾き $\dfrac{1}{2}$、切片 $-4$ の一次関数の式を求めよ。
A: $y = \dfrac{1}{2}x - 4$。$y = ax + b$ に $a = \dfrac{1}{2}$、$b = -4$ を代入する。

Q: 傾き $-1$、切片 $0$ の一次関数の式を求めよ。
A: $y = -x$。切片が $0$ なので $y = -x + 0 = -x$ となる。

Q: 傾き $4$、切片 $-7$ の一次関数の式を求めよ。
A: $y = 4x - 7$。$y = ax + b$ に $a = 4$、$b = -7$ を代入する。

Q: 傾き $\dfrac{2}{3}$、切片 $1$ の一次関数の式を求めよ。
A: $y = \dfrac{2}{3}x + 1$。$y = ax + b$ に $a = \dfrac{2}{3}$、$b = 1$ を代入する。

## グラフから式を読み取る

Q: グラフが右上がりで、$x$ が $1$ 増えると $y$ が $3$ 増え、$y$ 軸との交点が $(0, -2)$ のとき、式を求めよ。
A: $y = 3x - 2$。傾き $3$、切片 $-2$ だから $y = 3x - 2$。

Q: グラフが右下がりで、$x$ が $1$ 増えると $y$ が $2$ 減り、$y$ 軸との交点が $(0, 6)$ のとき、式を求めよ。
A: $y = -2x + 6$。傾き $-2$、切片 $6$ だから $y = -2x + 6$。

Q: グラフが点 $(0, 4)$ と点 $(1, 7)$ を通るとき、傾きと切片はいくらか。
A: 傾き $3$、切片 $4$。$y$ 軸上の点 $(0, 4)$ から切片は $4$。$x$ が $0$ から $1$ に増えると $y$ は $4$ から $7$ に増えるので傾き $3$。

Q: グラフが点 $(0, -1)$ と点 $(2, 5)$ を通るとき、式を求めよ。
A: $y = 3x - 1$。切片 $-1$。傾き $= (5 - (-1)) \div (2 - 0) = 6 \div 2 = 3$。

Q: グラフが点 $(0, 3)$ と点 $(4, -1)$ を通るとき、式を求めよ。
A: $y = -x + 3$。切片 $3$。傾き $= (-1 - 3) \div (4 - 0) = -4 \div 4 = -1$。

## 傾きと1点の座標がわかるとき

Q: 傾き $2$ で、点 $(3, 8)$ を通る一次関数の式を求めよ。
A: $y = 2x + 2$。$y = 2x + b$ に $(3, 8)$ を代入: $8 = 6 + b$、$b = 2$。

Q: 傾き $-1$ で、点 $(4, 1)$ を通る一次関数の式を求めよ。
A: $y = -x + 5$。$y = -x + b$ に $(4, 1)$ を代入: $1 = -4 + b$、$b = 5$。

Q: 傾き $3$ で、点 $(-1, 2)$ を通る一次関数の式を求めよ。
A: $y = 3x + 5$。$y = 3x + b$ に $(-1, 2)$ を代入: $2 = -3 + b$、$b = 5$。

Q: 傾き $\dfrac{1}{2}$ で、点 $(4, 5)$ を通る一次関数の式を求めよ。
A: $y = \dfrac{1}{2}x + 3$。$y = \dfrac{1}{2}x + b$ に $(4, 5)$ を代入: $5 = 2 + b$、$b = 3$。

Q: 傾き $-2$ で、点 $(2, -1)$ を通る一次関数の式を求めよ。
A: $y = -2x + 3$。$y = -2x + b$ に $(2, -1)$ を代入: $-1 = -4 + b$、$b = 3$。

Q: 傾き $5$ で、点 $(1, 3)$ を通る一次関数の式を求めよ。
A: $y = 5x - 2$。$y = 5x + b$ に $(1, 3)$ を代入: $3 = 5 + b$、$b = -2$。

Q: 傾き $-3$ で、点 $(-2, 10)$ を通る一次関数の式を求めよ。
A: $y = -3x + 4$。$y = -3x + b$ に $(-2, 10)$ を代入: $10 = 6 + b$、$b = 4$。

## 変化の割合・平行な直線から傾きがわかるとき

Q: 変化の割合が $4$ で、点 $(2, 11)$ を通る一次関数の式を求めよ。
A: $y = 4x + 3$。一次関数の変化の割合 $=$ 傾きだから $a = 4$。$y = 4x + b$ に $(2, 11)$ を代入: $11 = 8 + b$、$b = 3$。

Q: $y = 3x + 1$ に平行で、点 $(2, 9)$ を通る一次関数の式を求めよ。
A: $y = 3x + 3$。平行なので傾きは同じ $3$。$y = 3x + b$ に $(2, 9)$ を代入: $9 = 6 + b$、$b = 3$。

Q: $y = -2x + 5$ に平行で、点 $(1, 1)$ を通る一次関数の式を求めよ。
A: $y = -2x + 3$。平行なので傾き $-2$。$y = -2x + b$ に $(1, 1)$ を代入: $1 = -2 + b$、$b = 3$。

Q: $y = \dfrac{1}{3}x - 2$ に平行で、点 $(6, 5)$ を通る一次関数の式を求めよ。
A: $y = \dfrac{1}{3}x + 3$。平行なので傾き $\dfrac{1}{3}$。$y = \dfrac{1}{3}x + b$ に $(6, 5)$ を代入: $5 = 2 + b$、$b = 3$。

Q: 変化の割合が $-5$ で、点 $(1, -2)$ を通る一次関数の式を求めよ。
A: $y = -5x + 3$。変化の割合 $=$ 傾き $= -5$。$y = -5x + b$ に $(1, -2)$ を代入: $-2 = -5 + b$、$b = 3$。

Q: $y = 4x - 7$ に平行で、点 $(-1, -3)$ を通る一次関数の式を求めよ。
A: $y = 4x + 1$。平行なので傾き $4$。$y = 4x + b$ に $(-1, -3)$ を代入: $-3 = -4 + b$、$b = 1$。

## 2点から傾きを求める

Q: 2点 $(1, 3)$ と $(3, 7)$ を通る直線の傾きを求めよ。
A: $a = (7 - 3) \div (3 - 1) = 4 \div 2 = 2$。

Q: 2点 $(2, 1)$ と $(5, 10)$ を通る直線の傾きを求めよ。
A: $a = (10 - 1) \div (5 - 2) = 9 \div 3 = 3$。

Q: 2点 $(-1, 4)$ と $(3, -4)$ を通る直線の傾きを求めよ。
A: $a = (-4 - 4) \div (3 - (-1)) = -8 \div 4 = -2$。

Q: 2点 $(0, 5)$ と $(4, 3)$ を通る直線の傾きを求めよ。
A: $a = (3 - 5) \div (4 - 0) = -2 \div 4 = -\dfrac{1}{2}$。

Q: 2点 $(-2, -3)$ と $(4, 9)$ を通る直線の傾きを求めよ。
A: $a = (9 - (-3)) \div (4 - (-2)) = 12 \div 6 = 2$。

## 2点から式を求める（代入法）

Q: 2点 $(1, 5)$ と $(3, 11)$ を通る一次関数の式を求めよ。
A: $y = 3x + 2$。傾き $= (11 - 5) \div (3 - 1) = 3$。$y = 3x + b$ に $(1, 5)$ を代入: $5 = 3 + b$、$b = 2$。

Q: 2点 $(2, 3)$ と $(4, 9)$ を通る一次関数の式を求めよ。
A: $y = 3x - 3$。傾き $= (9 - 3) \div (4 - 2) = 3$。$y = 3x + b$ に $(2, 3)$ を代入: $3 = 6 + b$、$b = -3$。

Q: 2点 $(-1, 7)$ と $(2, 1)$ を通る一次関数の式を求めよ。
A: $y = -2x + 5$。傾き $= (1 - 7) \div (2 - (-1)) = -6 \div 3 = -2$。$y = -2x + b$ に $(2, 1)$ を代入: $1 = -4 + b$、$b = 5$。

Q: 2点 $(3, 4)$ と $(6, 10)$ を通る一次関数の式を求めよ。
A: $y = 2x - 2$。傾き $= (10 - 4) \div (6 - 3) = 2$。$y = 2x + b$ に $(3, 4)$ を代入: $4 = 6 + b$、$b = -2$。

Q: 2点 $(-2, 5)$ と $(3, -5)$ を通る一次関数の式を求めよ。
A: $y = -2x + 1$。傾き $= (-5 - 5) \div (3 - (-2)) = -10 \div 5 = -2$。$y = -2x + b$ に $(3, -5)$ を代入: $-5 = -6 + b$、$b = 1$。

Q: 2点 $(1, 2)$ と $(4, 8)$ を通る一次関数の式を求めよ。
A: $y = 2x$。傾き $= (8 - 2) \div (4 - 1) = 2$。$y = 2x + b$ に $(1, 2)$ を代入: $2 = 2 + b$、$b = 0$。

## 2点から式を求める（連立方程式法）

Q: 2点 $(1, 4)$ と $(3, 10)$ を通る一次関数の式を連立方程式で求めよ。
A: $y = 3x + 1$。$(1, 4)$ より $a + b = 4$、$(3, 10)$ より $3a + b = 10$。引くと $2a = 6$、$a = 3$、$b = 1$。

Q: 2点 $(2, 5)$ と $(5, 14)$ を通る一次関数の式を連立方程式で求めよ。
A: $y = 3x - 1$。$(2, 5)$ より $2a + b = 5$、$(5, 14)$ より $5a + b = 14$。引くと $3a = 9$、$a = 3$、$b = -1$。

Q: 2点 $(-1, 5)$ と $(2, -1)$ を通る一次関数の式を連立方程式で求めよ。
A: $y = -2x + 3$。$(-1, 5)$ より $-a + b = 5$、$(2, -1)$ より $2a + b = -1$。引くと $3a = -6$、$a = -2$、$b = 3$。

Q: 2点 $(3, 1)$ と $(6, 7)$ を通る一次関数の式を連立方程式で求めよ。
A: $y = 2x - 5$。$(3, 1)$ より $3a + b = 1$、$(6, 7)$ より $6a + b = 7$。引くと $3a = 6$、$a = 2$、$b = -5$。

Q: 2点 $(-3, 4)$ と $(1, -4)$ を通る一次関数の式を連立方程式で求めよ。
A: $y = -2x - 2$。$(-3, 4)$ より $-3a + b = 4$、$(1, -4)$ より $a + b = -4$。引くと $4a = -8$、$a = -2$、$b = -2$。

Q: 2点 $(2, 3)$ と $(6, 11)$ を通る一次関数の式を連立方程式で求めよ。
A: $y = 2x - 1$。$(2, 3)$ より $2a + b = 3$、$(6, 11)$ より $6a + b = 11$。引くと $4a = 8$、$a = 2$、$b = -1$。

## 条件から式を求める

Q: $x = 2$ のとき $y = 7$、$x = 4$ のとき $y = 13$ である一次関数の式を求めよ。
A: $y = 3x + 1$。傾き $= (13 - 7) \div (4 - 2) = 3$。$y = 3x + b$ に $(2, 7)$ を代入: $7 = 6 + b$、$b = 1$。

Q: $x = 1$ のとき $y = -1$、$x = 3$ のとき $y = 5$ である一次関数の式を求めよ。
A: $y = 3x - 4$。傾き $= (5 - (-1)) \div (3 - 1) = 3$。$y = 3x + b$ に $(1, -1)$ を代入: $-1 = 3 + b$、$b = -4$。

Q: $x = -1$ のとき $y = 6$、$x = 3$ のとき $y = -2$ である一次関数の式を求めよ。
A: $y = -2x + 4$。傾き $= (-2 - 6) \div (3 - (-1)) = -2$。$y = -2x + b$ に $(3, -2)$ を代入: $-2 = -6 + b$、$b = 4$。

Q: $x = 0$ のとき $y = -3$、$x = 5$ のとき $y = 7$ である一次関数の式を求めよ。
A: $y = 2x - 3$。$x = 0$ のとき $y = -3$ なので切片 $= -3$。傾き $= (7 - (-3)) \div (5 - 0) = 2$。

Q: $x$ が $3$ 増加すると $y$ が $6$ 増加し、点 $(2, 7)$ を通る一次関数の式を求めよ。
A: $y = 2x + 3$。傾き $= 6 \div 3 = 2$。$y = 2x + b$ に $(2, 7)$ を代入: $7 = 4 + b$、$b = 3$。

Q: $x$ が $4$ 増加すると $y$ が $8$ 減少し、点 $(1, 3)$ を通る一次関数の式を求めよ。
A: $y = -2x + 5$。傾き $= -8 \div 4 = -2$。$y = -2x + b$ に $(1, 3)$ を代入: $3 = -2 + b$、$b = 5$。
