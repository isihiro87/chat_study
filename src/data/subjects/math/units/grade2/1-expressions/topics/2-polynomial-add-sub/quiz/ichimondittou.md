# 一問一答：多項式の加法・減法

## 多項式の加法（かっこを外す）

Q1. $(3x - y) + (4x + 2y)$ を計算しなさい。
A1. $7x + y$

Q2. $(5a + 3b) + (2a - b)$ を計算しなさい。
A2. $7a + 2b$

Q3. $(2x^2 + 3x) + (x^2 - 5x)$ を計算しなさい。
A3. $3x^2 - 2x$

Q4. $(-4a + 6b) + (7a - 3b)$ を計算しなさい。
A4. $3a + 3b$

Q5. $(x + 2y - 3) + (3x - y + 5)$ を計算しなさい。
A5. $4x + y + 2$

## 多項式の減法（符号反転に注意）

Q6. $(3x - y) - (4x + 2y)$ を計算しなさい。
A6. $-x - 3y$

Q7. $(5a + 3b) - (2a - b)$ を計算しなさい。
A7. $3a + 4b$

Q8. $(7x - 4y) - (3x - 6y)$ を計算しなさい。
A8. $4x + 2y$

Q9. $(2x^2 + 5x) - (4x^2 - 3x)$ を計算しなさい。
A9. $-2x^2 + 8x$

Q10. $(a + 2b - 5) - (3a - b + 2)$ を計算しなさい。
A10. $-2a + 3b - 7$

## かっこの外し方の確認

Q11. $+(3x - 2y)$ のかっこを外しなさい。
A11. $3x - 2y$（「+」のかっこはそのまま外す）

Q12. $-(3x - 2y)$ のかっこを外しなさい。
A12. $-3x + 2y$（「−」のかっこは全部の符号を反転）

Q13. $-(5a + 4b - 1)$ のかっこを外しなさい。
A13. $-5a - 4b + 1$

Q14. $+(-2x^2 + 3x - 7)$ のかっこを外しなさい。
A14. $-2x^2 + 3x - 7$

Q15. $-(-x + 4y)$ のかっこを外しなさい。
A15. $x - 4y$

## 縦書き（筆算）の加法・減法

Q16. 縦書きで $(3x + 5y)$ と $(2x - 3y)$ を加法しなさい。
A16. $5x + 2y$（$x$ の列: $3 + 2 = 5$、$y$ の列: $5 + (-3) = 2$）

Q17. 縦書きで $(5a - 2b)$ から $(3a + 4b)$ を引きなさい。
A17. $2a - 6b$（$a$ の列: $5 - 3 = 2$、$b$ の列: $-2 - 4 = -6$）

Q18. 縦書きで $(4x^2 + 3x - 1)$ と $(-2x^2 + 5x + 3)$ を加法しなさい。
A18. $2x^2 + 8x + 2$

Q19. 縦書きで $(6a - 3b + 2)$ から $(2a - 5b - 1)$ を引きなさい。
A19. $4a + 2b + 3$

## 数×多項式（分配法則）

Q20. $2(3x + 5y)$ を計算しなさい。
A20. $6x + 10y$

Q21. $-3(2a - 4b)$ を計算しなさい。
A21. $-6a + 12b$

Q22. $5(-x + 3y)$ を計算しなさい。
A22. $-5x + 15y$

Q23. $-2(3x^2 - x + 4)$ を計算しなさい。
A23. $-6x^2 + 2x - 8$

Q24. $\frac{1}{2}(4a + 6b)$ を計算しなさい。
A24. $2a + 3b$

Q25. $-5(-a + 4b)$ を計算しなさい。
A25. $5a - 20b$

## 多項式÷数

Q26. $(10x + 6y) \div 2$ を計算しなさい。
A26. $5x + 3y$

Q27. $(12a - 8b) \div 4$ を計算しなさい。
A27. $3a - 2b$

Q28. $(24a - 15b) \div (-3)$ を計算しなさい。
A28. $-8a + 5b$

Q29. $(9x^2 + 6x) \div 3$ を計算しなさい。
A29. $3x^2 + 2x$

Q30. $(-x + 2y) \div \frac{1}{5}$ を計算しなさい。
A30. $-5x + 10y$（$\div \frac{1}{5}$ は $\times 5$ と同じ）

Q31. $(6a - 9b) \div (-3)$ を計算しなさい。
A31. $-2a + 3b$

## かっこがある式の計算（混合）

Q32. $2(x + y) + 3(x - 2y)$ を計算しなさい。
A32. $5x - 4y$

Q33. $3(2a - b) - 2(a + 3b)$ を計算しなさい。
A33. $4a - 9b$

Q34. $3(7a - 2b) - 5(4a - b)$ を計算しなさい。
A34. $a - b$

Q35. $4(x - 3y) + 2(2x + y)$ を計算しなさい。
A35. $8x - 10y$

Q36. $-2(3x + y) + 5(x - 2y)$ を計算しなさい。
A36. $-x - 12y$

## 分数の形の式の計算

Q37. $\frac{x + 3y}{2} + \frac{x - y}{2}$ を計算しなさい。
A37. $x + y$

Q38. $\frac{3x - y}{2} - \frac{x + 3y}{2}$ を計算しなさい。
A38. $x - 2y$

Q39. $\frac{4a + b}{3} + \frac{a - 7b}{5}$ を計算しなさい。
A39. $\frac{23a - 16b}{15}$（通分して $\frac{20a + 5b + 3a - 21b}{15}$）

Q40. $\frac{3x - y}{2} - \frac{3x - 2y}{3}$ を計算しなさい。
A40. $\frac{3x + y}{6}$（通分して $\frac{9x - 3y - 6x + 4y}{6}$）

Q41. $\frac{1}{2}a + \frac{1}{3}(a + 2b)$ を計算しなさい。
A41. $\frac{5}{6}a + \frac{2}{3}b$

Q42. $\frac{2x + y}{4} - \frac{x - 3y}{6}$ を計算しなさい。
A42. $\frac{4x + 9y}{12}$（通分して $\frac{3(2x+y) - 2(x-3y)}{12}$）

## 式の値

Q43. $x = 4, y = -3$ のとき、$(3x - 4y) - (2x - 5y)$ の値を求めなさい。
A43. $1$（まず $x + y$ に簡単にしてから代入: $4 + (-3) = 1$）

Q44. $a = -2, b = 3$ のとき、$3(2a + b) - 2(a - 3b)$ の値を求めなさい。
A44. $19$（まず $4a + 9b$ に簡単にしてから代入: $-8 + 27 = 19$）

Q45. $x = \frac{1}{2}, y = -\frac{1}{3}$ のとき、$6(x + 2y) - 3(2x - y)$ の値を求めなさい。
A45. $-5$（まず $15y$ に簡単にしてから代入: $15 \times (-\frac{1}{3}) = -5$）
