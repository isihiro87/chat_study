# 大問形式：単項式と多項式

## 大問1：用語と定義（配点20点）

次の問いに答えなさい。

(1) 数や文字のかけ算だけでできている式を何というか。（3点）

(2) 単項式の和の形になっている式を何というか。（3点）

(3) 多項式の中の一つひとつの単項式を何というか。（3点）

(4) 単項式において、かけ合わされている文字の個数を何というか。（3点）

(5) 単項式において、文字にかかっている数を何というか。（4点）

(6) 文字の部分が同じ項どうしを何というか。（4点）

## 大問2：項と次数の識別（配点20点）

次の式について、各問いに答えなさい。

(1) $3x^2 - 5x + 2$ の項をすべて答えなさい。（3点）

(2) $-4ab + 7a - 3$ の項をすべて答えなさい。（3点）

(3) $7a$ の次数と係数をそれぞれ答えなさい。（3点）

(4) $-a^2$ の次数と係数をそれぞれ答えなさい。（3点）

(5) $4x^2y$ の次数と係数をそれぞれ答えなさい。（4点）

(6) $2x^2 - 3xy + y^2 + 5$ の次数を答えなさい。理由もあわせて説明しなさい。（4点）

## 大問3：同類項の識別（配点20点）

(1) 次のア〜カの中から、同類項の組をすべて選びなさい。（4点）

ア $3x$　イ $-2x^2$　ウ $5x$　エ $4x^2$　オ $-x$　カ $7$

(2) 次の式から同類項の組をすべて見つけなさい。（4点）

$2a - 3b + 5a + b - 4$

(3) $4ab$ と $-7ba$ は同類項か。理由もあわせて答えなさい。（4点）

(4) $3xy^2$ と $3x^2y$ は同類項か。理由もあわせて答えなさい。（4点）

(5) 次の式の中で、同類項をまとめられる組を見つけ、式を整理しなさい。（4点）

$x^2 + 3x - 2x^2 + 5 - x + 1$

## 大問4：同類項のまとめ（基本）（配点20点）

次の式の同類項をまとめて簡単にしなさい。

(1) $5a + 3a$（3点）

(2) $7x - 4x + 2$（3点）

(3) $3x + 2y - x + 5y$（3点）

(4) $4a^2 - 3a + 2a^2 + 5a$（4点）

(5) $-2x + 3y + 5x - 7y + 1$（4点）

(6) $x^2 + 9x + 3x^2 - 4x$（3点）

## 大問5：同類項のまとめ（分数と応用）（配点20点）

次の式の同類項をまとめて簡単にしなさい。

(1) $\frac{1}{2}a + \frac{1}{3}a$（3点）

(2) $\frac{2}{3}x - \frac{1}{4}x + 1$（4点）

(3) $\frac{1}{2}a + b - \frac{1}{3}a + 2b$（4点）

(4) $\frac{3}{4}x^2 + 2x - \frac{1}{4}x^2 - x$（4点）

(5) $\frac{1}{3}a^2 - \frac{1}{2}a + \frac{2}{3}a^2 + \frac{3}{2}a - 5$（5点）
