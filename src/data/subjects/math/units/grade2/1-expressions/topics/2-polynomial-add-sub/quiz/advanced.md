# 発展問題：多項式の加法・減法

## 発展1：分数の形の複合計算（配点20点）

次の計算をしなさい。

(1) $\frac{2a + 3b}{4} + \frac{3a - b}{6}$（4点）

(2) $\frac{5x - 2y}{3} - \frac{2x + y}{4} + \frac{x - y}{6}$（5点）

(3) $\frac{1}{2}(3a - 2b) + \frac{1}{3}(a + 6b)$（5点）

(4) $\frac{x + 2y}{3} - \frac{2x - y}{4} + \frac{5x + y}{6}$（6点）

## 発展2：縦書き計算と工夫（配点20点）

(1) 次の3つの多項式の和を縦書き（筆算）で求めなさい。（5点）

$2x^2 + 3x - 1$、$-x^2 + 5x + 4$、$3x^2 - 2x - 7$

(2) 次の3つの多項式について、1番目から2番目と3番目の和を引きなさい。（5点）

$5a - 3b + 7$、$2a + b - 4$、$-a + 5b - 1$

(3) ある多項式に $(3x - 2y)$ を加えると $(7x + y)$ になった。ある多項式を求めなさい。（5点）

(4) $(8a - 5b)$ からある多項式を引くと $(2a + b)$ になった。ある多項式を求めなさい。（5点）

## 発展3：式の代入と計算（配点20点）

$A = 3x - 2y$、$B = x + 4y$ とするとき、次の式を計算しなさい。

(1) $A + B$（3点）

(2) $A - B$（3点）

(3) $2A + 3B$（4点）

(4) $3A - 2B$（4点）

(5) $A - B$ を計算し、$x = 5, y = -1$ のときの値を求めなさい。（6点）

## 発展4：間違い探しと正しい計算（配点20点）

次の計算にはそれぞれ間違いがある。間違いを指摘し、正しく計算しなさい。

(1) $(5x - 3y) - (2x - 4y) = 5x - 3y - 2x - 4y = 3x - 7y$（4点）

(2) $3(2a - b) - 2(a + 3b) = 6a - b - 2a + 6b = 4a + 5b$（4点）

(3) $\frac{x + y}{2} + \frac{x - y}{3} = \frac{x + y + x - y}{6} = \frac{2x}{6} = \frac{x}{3}$（5点）

(4) $x = -2$ のとき、$3(2x - 1) - 2(x + 3) = 6x - 1 - 2x + 3 = 4x + 2 = -6$（7点）

## 発展5：総合問題（配点20点）

(1) $a = \frac{1}{3}, b = -\frac{1}{2}$ のとき、$\frac{3a - 2b}{2} - \frac{a + 4b}{3}$ の値を求めなさい。式を簡単にしてから代入すること。（6点）

(2) $P = 2x - 3y$、$Q = -x + 5y$ のとき、$3P - 2Q + (P + Q)$ を $x, y$ の式で表しなさい。（6点）

(3) ある2つの多項式の和が $6a + 2b$ で、差が $2a - 4b$ であるとき、その2つの多項式をそれぞれ求めなさい。（8点）
