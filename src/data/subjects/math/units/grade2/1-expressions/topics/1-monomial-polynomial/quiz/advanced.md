# 発展問題：単項式と多項式

## 発展1：分数係数の同類項まとめ（配点20点）

次の式の同類項をまとめて簡単にしなさい。

(1) $\frac{2}{3}x + \frac{1}{6}x - \frac{5}{6}x$（4点）

(2) $\frac{3}{4}a - \frac{2}{3}b + \frac{1}{4}a + \frac{1}{3}b$（5点）

(3) $\frac{1}{2}x^2 - \frac{2}{3}x + \frac{3}{4}x^2 + \frac{1}{6}x - 1$（5点）

(4) $\frac{5}{6}a^2b - \frac{1}{3}ab^2 + \frac{1}{6}a^2b + \frac{2}{3}ab^2$（6点）

## 発展2：多変数の同類項の識別と整理（配点20点）

(1) 次のア〜キの中から同類項の組をすべて選びなさい。（5点）

ア $3xy$　イ $-2x^2y$　ウ $5yx$　エ $4xy^2$　オ $-xy$　カ $x^2y$　キ $2yx^2$

(2) 次の式の同類項をまとめて簡単にしなさい。（5点）

$2x^2y - 3xy^2 + x^2y + 5xy^2 - 4x^2y$

(3) 次の式の次数を求め、各項の次数と係数をすべて答えなさい。（5点）

$3a^2b - 2ab + 5b^2 - 1$

(4) 多項式 $ax^2 + bx + c$ が2次式であるための条件を、$a$, $b$, $c$ を使って説明しなさい。（5点）

## 発展3：間違い訂正問題（配点20点）

次の計算にはそれぞれ間違いがある。間違いを指摘し、正しい答えを求めなさい。

(1) 「$3x + 2y = 5xy$」（4点）

(2) 「$x^2 + x = x^3$」（4点）

(3) 「$4a^2 - a^2 = 4$」（4点）

(4) 「$\frac{1}{2}x + \frac{1}{3}x = \frac{1}{5}x$」（4点）

(5) 「$2xy + 3yx = 5x^2y^2$」（4点）

## 発展4：逆算・応用問題（配点20点）

(1) $3x^2 + 5x - 2$ にどんな式を加えると $5x^2 + 2x + 1$ になるか。加える式を求めなさい。（5点）

(2) $4a - 3b + 7$ からどんな式をひくと $a + 2b - 1$ になるか。ひく式を求めなさい。（5点）

(3) ある多項式に $2x^2 - 3x + 1$ を加えたら $x^2 + 4x - 5$ になった。もとの多項式を求めなさい。（5点）

(4) 2つの多項式 $A = 3x^2 - 2x + 5$、$B = x^2 + 4x - 3$ があるとき、$2A - B$ を計算しなさい。（5点）

## 発展5：総合問題（配点20点）

(1) $n$ が自然数のとき、$2x^n + 3x^n$ を簡単にしなさい。また、なぜ同類項としてまとめられるか説明しなさい。（5点）

(2) 長方形の縦の長さが $(2a + 3)$ cm、横の長さが $(a - 1)$ cm のとき、周の長さを同類項をまとめて簡単にしなさい。（5点）

(3) ある式を $A$ とする。$A + (3x - 2y) = 5x + y$ のとき、$A - (3x - 2y)$ を求めなさい。（5点）

(4) 3つの多項式の和 $(2a + b) + (-a + 3b) + (a - 2b)$ を計算し、何次式になるか答えなさい。（5点）
