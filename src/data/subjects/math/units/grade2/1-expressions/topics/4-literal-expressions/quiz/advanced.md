# 発展問題：文字式の利用

## 発展1：カレンダーの数の性質（配点20点）

カレンダーで、縦に並ぶ3つの数（例: 3, 10, 17）について考える。

(1) 縦に並ぶ3つの数のうち、真ん中の数を $n$ とするとき、上の数と下の数をそれぞれ $n$ を使って表しなさい。（4点）

(2) 縦に並ぶ3つの数の和が、真ん中の数の3倍になることを証明しなさい。（5点）

(3) カレンダーで横に並ぶ3つの数の和は、真ん中の数の3倍になるか。証明しなさい。（5点）

(4) カレンダーで、十字型に5つの数（上・下・左・右・中央）を選ぶ。その5つの数の和が中央の数の5倍になることを証明しなさい。（6点）

## 発展2：複雑な整数の性質の証明（配点20点）

(1) 連続する5つの偶数の和が10の倍数になることを証明しなさい。（5点）

(2) 連続する5つの整数の和が5の倍数になることを証明しなさい。（4点）

(3) 2つの偶数の積は4の倍数になることを証明しなさい。（5点）

(4) 奇数の2乗は必ず奇数になることを証明しなさい。（6点）

## 発展3：3けたの自然数の発展（配点20点）

(1) 3けたの自然数 $100x + 10y + z$ と、数字の並びを逆にした数 $100z + 10y + x$ について、$x > z$ のとき差が99の倍数になることを証明しなさい。（4点）

(2) 3けたの自然数 $100a + 10b + c$ の各位の数の和 $a + b + c$ が9の倍数のとき、もとの3けたの自然数も9の倍数であることを説明しなさい。（ヒント: $100a + 10b + c = 99a + 9b + (a + b + c)$）（6点）

(3) 3けたの自然数の百の位と十の位を入れかえた数との差を考える。$100a + 10b + c$ と $100b + 10a + c$ の差は何の倍数になるか、証明しなさい（$a > b$ のとき）。（5点）

(4) ある3けたの自然数の百の位を $a$、十の位を $b$、一の位を $c$ とする。この数から各位の数の和を引いた数は9の倍数になることを証明しなさい。（5点）

## 発展4：等式の変形と図形（配点20点）

(1) 円すいの体積 $V = \frac{1}{3}\pi r^2 h$ を $r^2$ について解きなさい。（4点）

(2) 台形の面積 $S = \frac{(a + b)h}{2}$ を $a$ について解きなさい。（4点）

(3) 球の表面積 $S = 4\pi r^2$ を $r^2$ について解きなさい。（4点）

(4) オームの法則 $V = RI$ を $R$ について解きなさい。また、$V = 12$（V）、$I = 0.5$（A）のとき $R$ の値を求めなさい。（4点）

(5) 底面の半径 $r$、高さ $h$ の円柱の表面積は $S = 2\pi r^2 + 2\pi rh$ である。これを $h$ について解きなさい。（4点）

## 発展5：証明の誤り発見と修正（配点20点）

(1) 次の証明には誤りがある。誤りを指摘し、正しく修正しなさい。（5点）

「連続する2つの整数の和は偶数になることを証明する。
連続する2つの整数を $2n, 2n + 1$ とする。
和は $2n + (2n + 1) = 4n + 1$。
$4n + 1 = 2(2n) + 1$ なので奇数になる。」

(2) 次の等式の変形には誤りがある。誤りを指摘し、正しく修正しなさい。（5点）

「$3x + 2y = 12$ を $y$ について解く。
$2y = 12 + 3x$
$y = 6 + \frac{3}{2}x$」

(3) 「偶数と偶数の差は必ず偶数になる」ことを証明しなさい。（4点）

(4) 「3の倍数と3の倍数の和は3の倍数になる」ことを証明しなさい。さらに、「3の倍数と3の倍数でない数の和は3の倍数でない」ことは正しいか、理由を説明しなさい。（6点）
