# 一問一答：文字式の利用

## 整数の文字表現

Q1. 整数 $n$ を使って偶数を表しなさい。
A1. $2n$

Q2. 整数 $n$ を使って奇数を表しなさい。
A2. $2n + 1$

Q3. 連続する2つの整数を $n$ を使って表しなさい。
A3. $n, n + 1$

Q4. 連続する3つの整数を $n$ を使って表しなさい。
A4. $n, n + 1, n + 2$

Q5. 連続する3つの偶数を $n$ を使って表しなさい。
A5. $2n, 2n + 2, 2n + 4$

Q6. 連続する3つの奇数を $n$ を使って表しなさい。
A6. $2n + 1, 2n + 3, 2n + 5$

Q7. 3の倍数を整数 $n$ を使って表しなさい。
A7. $3n$

Q8. 5の倍数を整数 $n$ を使って表しなさい。
A8. $5n$

Q9. 偶数と偶数の和はどんな数になるか。
A9. 偶数（$2m + 2n = 2(m + n)$ で $2 \times$ 整数の形）

Q10. 奇数と奇数の和はどんな数になるか。
A10. 偶数（$(2m + 1) + (2n + 1) = 2(m + n + 1)$ で $2 \times$ 整数の形）

## 証明の基本

Q11. 文字式を使った証明の3ステップを答えなさい。
A11. ① 文字で表す → ② 式を計算する → ③ 結論を書く

Q12. 「$3$ の倍数である」ことを式で示すには、どんな形にすればよいか。
A12. $3 \times$（整数）の形にする

Q13. 「偶数である」ことを式で示すには、どんな形にすればよいか。
A13. $2 \times$（整数）の形にする

Q14. 連続する3つの整数の和が3の倍数になることを式で示しなさい。
A14. $n + (n + 1) + (n + 2) = 3n + 3 = 3(n + 1)$。$3 \times$ 整数の形なので3の倍数

Q15. 連続する3つの整数の和は、真ん中の数の何倍になるか。
A15. 3倍（和 $= 3(n + 1)$ で $n + 1$ は真ん中の数）

Q16. 奇数から偶数を引くとどんな数になるか。
A16. 奇数（$(2m + 1) - 2n = 2(m - n) + 1$ で奇数の形）

Q17. 偶数から奇数を引くとどんな数になるか。
A17. 奇数（$2m - (2n + 1) = 2(m - n) - 1 = 2(m - n - 1) + 1$ で奇数の形）

Q18. 連続する3つの偶数の和は何の倍数になるか。
A18. 6の倍数（$2n + (2n + 2) + (2n + 4) = 6n + 6 = 6(n + 1)$）

Q19. 連続する3つの偶数の和は、真ん中の偶数の何倍になるか。
A19. 3倍（和 $= 6(n + 1) = 3 \times 2(n + 1)$ で $2(n + 1)$ は真ん中の偶数）

Q20. 偶数と奇数の積は偶数か奇数か。
A20. 偶数（$2m \times (2n + 1) = 2m(2n + 1)$ で $2 \times$ 整数の形）

## 2けたの自然数

Q21. 十の位が $a$、一の位が $b$ の2けたの自然数を式で表しなさい。
A21. $10a + b$

Q22. 十の位が $a$、一の位が $b$ の2けたの自然数で、十の位と一の位を入れかえた数を式で表しなさい。
A22. $10b + a$

Q23. 2けたの自然数とその十の位と一の位を入れかえた数の和を計算しなさい。
A23. $(10a + b) + (10b + a) = 11a + 11b = 11(a + b)$

Q24. 2けたの自然数とその十の位と一の位を入れかえた数の和は何の倍数になるか。
A24. 11の倍数

Q25. 2けたの自然数から十の位と一の位を入れかえた数を引いた差を計算しなさい（$a > b$ のとき）。
A25. $(10a + b) - (10b + a) = 9a - 9b = 9(a - b)$

Q26. 2けたの自然数とその十の位と一の位を入れかえた数の差は何の倍数になるか。
A26. 9の倍数

Q27. 2けたの自然数の各位の数の和が9の倍数のとき、その2けたの自然数はどんな数か。
A27. 9の倍数（$10a + b$ で $a + b$ が9の倍数なら、$10a + b = 9a + (a + b)$ で9の倍数）

## 3けたの自然数

Q28. 百の位が $x$、十の位が $y$、一の位が $z$ の3けたの自然数を式で表しなさい。
A28. $100x + 10y + z$

Q29. 3けたの自然数の百の位と一の位を入れかえた数を式で表しなさい。
A29. $100z + 10y + x$

Q30. 3けたの自然数とその百の位と一の位を入れかえた数の差を計算しなさい（$x > z$ のとき）。
A30. $(100x + 10y + z) - (100z + 10y + x) = 99x - 99z = 99(x - z)$

Q31. 3けたの自然数と百の位と一の位を入れかえた数の差は何の倍数になるか。
A31. 9の倍数（$99(x - z) = 9 \times 11(x - z)$ なので9の倍数。さらに99の倍数でもある）

## 等式の変形（基本）

Q32. $3x + 2y = 8$ を $y$ について解きなさい。
A32. $y = \frac{8 - 3x}{2}$

Q33. $4x + y = 3$ を $y$ について解きなさい。
A33. $y = 3 - 4x$

Q34. $x + 2y = 6$ を $y$ について解きなさい。
A34. $y = \frac{6 - x}{2}$

Q35. $5x - y = -1$ を $y$ について解きなさい。
A35. $y = 5x + 1$

Q36. $8x - 4y = 16$ を $y$ について解きなさい。
A36. $y = 2x - 4$

Q37. $a = 3b + c$ を $b$ について解きなさい。
A37. $b = \frac{a - c}{3}$

## 等式の変形（公式）

Q38. $S = \frac{1}{2}ab$ を $a$ について解きなさい。
A38. $a = \frac{2S}{b}$

Q39. $abc = 24$ を $a$ について解きなさい。
A39. $a = \frac{24}{bc}$

Q40. $l = \frac{mn}{20}$ を $n$ について解きなさい。
A40. $n = \frac{20l}{m}$

Q41. $V = \pi r^2 h$ を $h$ について解きなさい。
A41. $h = \frac{V}{\pi r^2}$

Q42. $V = \frac{1}{3}x^2 h$ を $h$ について解きなさい。
A42. $h = \frac{3V}{x^2}$

Q43. 長さ $50$ cmのひもで長方形を作るとき、$a + b = 25$ を $a$ について解きなさい。
A43. $a = 25 - b$

Q44. $S = \pi r^2$ を $r^2$ について解きなさい。
A44. $r^2 = \frac{S}{\pi}$

Q45. $y = ax + b$ を $x$ について解きなさい。
A45. $x = \frac{y - b}{a}$

Q46. $2\pi r = l$ を $r$ について解きなさい。
A46. $r = \frac{l}{2\pi}$

Q47. $S = \frac{(a + b)h}{2}$ を $h$ について解きなさい。
A47. $h = \frac{2S}{a + b}$

Q48. $S = \frac{(a + b)h}{2}$ を $a$ について解きなさい。
A48. $a = \frac{2S}{h} - b$
