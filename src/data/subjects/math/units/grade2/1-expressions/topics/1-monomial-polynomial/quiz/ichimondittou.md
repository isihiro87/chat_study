# 一問一答：単項式と多項式

## 単項式と多項式の定義

Q1. 数や文字のかけ算だけでできている式を何というか。
A1. 単項式

Q2. 単項式の和の形になっている式を何というか。
A2. 多項式

Q3. 多項式の中の一つひとつの単項式を何というか。
A3. 項

Q4. $3x$ は単項式と多項式のどちらか。
A4. 単項式

Q5. $5a + 2b - 3$ は単項式と多項式のどちらか。
A5. 多項式

Q6. 数だけの項（例: $-7$）を何というか。
A6. 定数項（数の項）

Q7. $x^2 + 3x - 5$ の定数項は何か。
A7. $-5$

Q8. $-8$ は単項式と多項式のどちらか。
A8. 単項式（数だけでも単項式）

## 項の識別

Q9. $x^2y + 6xy - 8y$ の項をすべて答えなさい。
A9. $x^2y$, $6xy$, $-8y$

Q10. $3a - 2b + 5$ の項をすべて答えなさい。
A10. $3a$, $-2b$, $5$

Q11. $-x^2 + 4x - 1$ の項をすべて答えなさい。
A11. $-x^2$, $4x$, $-1$

Q12. $2a^2b - 3ab + 7a$ の項をすべて答えなさい。
A12. $2a^2b$, $-3ab$, $7a$

Q13. $5x^2 - x + 3$ の項はいくつあるか。
A13. 3つ（$5x^2$, $-x$, $3$）

Q14. $-2y + 8$ の項をすべて答えなさい。
A14. $-2y$, $8$

## 単項式の次数

Q15. 単項式において、かけ合わされている文字の個数を何というか。
A15. 次数

Q16. $7a$ の次数を答えなさい。
A16. 1（文字 $a$ が1個）

Q17. $-a^2$ の次数を答えなさい。
A17. 2（$a$ が2個かけ合わされている）

Q18. $4xy$ の次数を答えなさい。
A18. 2（$x$ が1個と $y$ が1個で合計2個）

Q19. $3x^2y$ の次数を答えなさい。
A19. 3（$x$ が2個と $y$ が1個で合計3個）

Q20. $-5a^2b^3$ の次数を答えなさい。
A20. 5（$a$ が2個と $b$ が3個で合計5個）

Q21. $6$ の次数を答えなさい。
A21. 0（文字が含まれていないので次数は0）

Q22. $-xy^2z$ の次数を答えなさい。
A22. 4（$x$ が1個、$y$ が2個、$z$ が1個で合計4個）

## 係数

Q23. 単項式で、文字にかかっている数を何というか。
A23. 係数

Q24. $5x$ の係数を答えなさい。
A24. $5$

Q25. $-3a^2b$ の係数を答えなさい。
A25. $-3$

Q26. $x^2$ の係数を答えなさい。
A26. $1$（$1 \times x^2$ なので係数は $1$）

Q27. $-y$ の係数を答えなさい。
A27. $-1$

Q28. $\frac{2}{3}ab$ の係数を答えなさい。
A28. $\frac{2}{3}$

## 多項式の次数

Q29. 多項式の次数はどのように求めるか。
A29. 各項の次数のうち、もっとも大きいもの（最高次数）

Q30. $3x^2 + 5x - 1$ の次数を答えなさい。
A30. 2（$3x^2$ の次数が2で最も大きい）

Q31. $2a + 3b - 7$ の次数を答えなさい。
A31. 1（$2a$, $3b$ の次数がともに1で最大）

Q32. $x^2y + 3xy - 2y$ の次数を答えなさい。
A32. 3（$x^2y$ の次数が $2+1=3$ で最も大きい）

Q33. $4a^2 - 2ab + b^2$ の次数を答えなさい。
A33. 2（すべての項の次数が2）

## 同類項

Q34. 文字の部分が同じ項どうしを何というか。
A34. 同類項

Q35. $2x - y + 4y + 5x$ の中で同類項の組を答えなさい。
A35. $2x$ と $5x$、$-y$ と $4y$

Q36. $3a^2 + 2a - a^2 + 5a$ の中で同類項の組を答えなさい。
A36. $3a^2$ と $-a^2$、$2a$ と $5a$

Q37. $3x$ と $3x^2$ は同類項か。
A37. 同類項ではない（$x$ と $x^2$ は文字の部分が異なる）

Q38. $4ab$ と $-2ab$ は同類項か。
A38. 同類項である（文字の部分 $ab$ が同じ）

Q39. $5xy$ と $5yz$ は同類項か。
A39. 同類項ではない（文字の部分が異なる）

## 同類項をまとめる（整数の係数）

Q40. $3x + 5x$ を計算しなさい。
A40. $8x$

Q41. $7a - 3a$ を計算しなさい。
A41. $4a$

Q42. $x^2 + 9x + 3x^2 - 4x$ を計算しなさい。
A42. $4x^2 + 5x$

Q43. $5a - 2b + 3a + 4b$ を計算しなさい。
A43. $8a + 2b$

Q44. $2x^2 - 3x + 5x^2 + x$ を計算しなさい。
A44. $7x^2 - 2x$

Q45. $-3y + 7 + 5y - 2$ を計算しなさい。
A45. $2y + 5$

## 同類項をまとめる（分数の係数）

Q46. $\frac{1}{2}a + \frac{1}{3}a$ を計算しなさい。
A46. $\frac{5}{6}a$（通分して $\frac{3}{6}a + \frac{2}{6}a = \frac{5}{6}a$）

Q47. $\frac{1}{2}a + b - \frac{1}{3}a + 2b$ を計算しなさい。
A47. $\frac{1}{6}a + 3b$

Q48. $\frac{2}{3}x - \frac{1}{4}x$ を計算しなさい。
A48. $\frac{5}{12}x$（通分して $\frac{8}{12}x - \frac{3}{12}x = \frac{5}{12}x$）

Q49. $\frac{3}{4}y + \frac{1}{2}y - \frac{1}{4}y$ を計算しなさい。
A49. $y$（$\frac{3}{4}y + \frac{2}{4}y - \frac{1}{4}y = \frac{4}{4}y = y$）

Q50. $\frac{1}{3}x^2 + 2x - \frac{2}{3}x^2 + x$ を計算しなさい。
A50. $-\frac{1}{3}x^2 + 3x$
