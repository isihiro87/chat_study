# 単項式の乗法・除法 発展問題

## 問1: 累乗を含む複合計算

次の計算をしなさい。

(1) $3a \times (-4a)^2$
→ $(-4a)^2 = 16a^2$。$3a \times 16a^2 = 48a^3$。

(2) $(2a)^2 \times ab \div (-4a^2)$
→ $(2a)^2 = 4a^2$。$4a^2 \times ab = 4a^3b$。$4a^3b \div (-4a^2) = -ab$。

(3) $(-3x)^2 \times (-8x) \div 6x^2$
→ $(-3x)^2 = 9x^2$。$9x^2 \times (-8x) = -72x^3$。$-72x^3 \div 6x^2 = -12x$。

(4) $(-2a)^3 \times 3a \div (-4a^2)$
→ $(-2a)^3 = -8a^3$。$-8a^3 \times 3a = -24a^4$。$-24a^4 \div (-4a^2) = 6a^2$。

## 問2: 分数係数の多段階計算

次の計算をしなさい。

(1) $\dfrac{2}{3}a^2b \times \left(-\dfrac{9}{4}ab^2\right) \div \dfrac{3}{2}a^2b$
→ 分子: $\dfrac{2}{3}a^2b \times \left(-\dfrac{9}{4}ab^2\right) = -\dfrac{3}{2}a^3b^3$。$-\dfrac{3}{2}a^3b^3 \div \dfrac{3}{2}a^2b = -\dfrac{3}{2}a^3b^3 \times \dfrac{2}{3a^2b} = -ab^2$。

(2) $\left(-\dfrac{3}{5}x^2y\right) \times 10xy \div \left(-\dfrac{2}{3}x^2y\right)$
→ $-\dfrac{3}{5}x^2y \times 10xy = -6x^3y^2$。$-6x^3y^2 \div \left(-\dfrac{2}{3}x^2y\right) = -6x^3y^2 \times \left(-\dfrac{3}{2x^2y}\right) = 9xy$。

(3) $\dfrac{5}{6}a^2 \times \left(-\dfrac{3}{10}ab\right) \div \dfrac{1}{4}a^2b$
→ $\dfrac{5}{6}a^2 \times \left(-\dfrac{3}{10}ab\right) = -\dfrac{1}{4}a^3b$。$-\dfrac{1}{4}a^3b \div \dfrac{1}{4}a^2b = -\dfrac{1}{4}a^3b \times \dfrac{4}{a^2b} = -a$。

(4) $\left(-\dfrac{4}{3}x\right)^2 \times \dfrac{3}{8}x \div \dfrac{2}{3}x^2$
→ $\left(-\dfrac{4}{3}x\right)^2 = \dfrac{16}{9}x^2$。$\dfrac{16}{9}x^2 \times \dfrac{3}{8}x = \dfrac{2}{3}x^3$。$\dfrac{2}{3}x^3 \div \dfrac{2}{3}x^2 = x$。

## 問3: 間違い探し

次の計算には間違いがある。間違いを見つけ、正しい答えを求めなさい。

(1) $(-3a)^2 = -9a^2$
→ 間違い。$(-3a)^2 = (-3)^2 \times a^2 = 9a^2$。括弧全体を2乗しているので偶数乗で係数は正になる。$-9a^2$ となるのは $-(3a)^2$ の場合。

(2) $x^2 \times x^3 = x^6$
→ 間違い。同じ底の累乗のかけ算では指数を足す。$x^2 \times x^3 = x^{2+3} = x^5$。指数をかけるのは $(x^2)^3$ の場合で、このとき $x^6$ になる。

(3) $12a^2b \div 4ab = 3a^2$
→ 間違い。$\dfrac{12a^2b}{4ab} = \dfrac{12}{4} \times \dfrac{a^2}{a} \times \dfrac{b}{b} = 3a$。$a^2 \div a = a^{2-1} = a$ であって $a^2$ ではない。

(4) $-(-2x)^3 = -8x^3$
→ 間違い。$(-2x)^3 = (-2)^3 \times x^3 = -8x^3$（奇数乗で負）。前のマイナスをつけて $-(-8x^3) = 8x^3$。正しい答えは $8x^3$。

## 問4: 指数法則の総合

次の計算をしなさい。

(1) $(-2x)^2 \times (-3x)^2$
→ $(-2x)^2 = 4x^2$, $(-3x)^2 = 9x^2$。$4x^2 \times 9x^2 = 36x^4$。

(2) $(-ab)^3 \div (-ab)^2$
→ $(-ab)^3 = -a^3b^3$, $(-ab)^2 = a^2b^2$。$\dfrac{-a^3b^3}{a^2b^2} = -ab$。

(3) $-7a \times (-3ab) \times 2b^2 \div (-6a^2b^2)$
→ $(-7) \times (-3) \times 2 = 42$、$a \times ab \times b^2 = a^2b^3$。$\dfrac{42a^2b^3}{-6a^2b^2} = -7b$。

(4) $(-2x)^3 \times 3x^2 \div (-6x^4)$
→ $(-2x)^3 = -8x^3$。$-8x^3 \times 3x^2 = -24x^5$。$-24x^5 \div (-6x^4) = 4x$。

## 問5: 文字式を用いた証明的問題

(1) $a = 2, b = -3$ のとき、$(-2a^2b) \times 3ab^2 \div (-6a^2b)$ の値を求めなさい。まず式を簡単にしてから代入すること。
→ 式の簡略化: $\dfrac{(-2a^2b) \times 3ab^2}{-6a^2b} = \dfrac{-6a^3b^3}{-6a^2b} = ab^2$。$a = 2, b = -3$ を代入: $2 \times (-3)^2 = 2 \times 9 = 18$。

(2) $x = -4, y = \dfrac{1}{2}$ のとき、$8x^2y \div (-4xy) \times 2y$ の値を求めなさい。まず式を簡単にしてから代入すること。
→ 式の簡略化: $\dfrac{8x^2y \times 2y}{-4xy} = \dfrac{16x^2y^2}{-4xy} = -4xy$。$x = -4, y = \dfrac{1}{2}$ を代入: $-4 \times (-4) \times \dfrac{1}{2} = 8$。

(3) $a \neq 0, b \neq 0$ のとき、$\dfrac{(-3a^2b)^2}{6a^3b} \div \dfrac{3ab}{2}$ を簡単にしなさい。
→ $(-3a^2b)^2 = 9a^4b^2$。$\dfrac{9a^4b^2}{6a^3b} = \dfrac{3}{2}ab$。$\dfrac{3}{2}ab \div \dfrac{3ab}{2} = \dfrac{3}{2}ab \times \dfrac{2}{3ab} = 1$。
