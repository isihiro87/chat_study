# 大問形式：多項式の加法・減法

## 大問1：多項式の加法・減法（配点20点）

次の計算をしなさい。

(1) $(3x + 2y) + (5x - 3y)$（3点）

(2) $(4a - 3b) + (-2a + 7b)$（3点）

(3) $(5x - 2y) - (3x + 4y)$（3点）

(4) $(2x^2 + 3x - 1) - (x^2 - 5x + 3)$（4点）

(5) 次の2つの多項式を縦書き（筆算）で加法しなさい。$3x^2 - 4x + 2$ と $-x^2 + 7x - 5$（3点）

(6) 次の2つの多項式を縦書き（筆算）で減法しなさい。$(7a - 3b + 1)$ から $(4a + 2b - 3)$ を引く。（4点）

## 大問2：数×多項式、多項式÷数（配点20点）

次の計算をしなさい。

(1) $2(3x - 7y)$（3点）

(2) $-5(-a + 4b)$（3点）

(3) $-4(2x^2 - 3x + 1)$（3点）

(4) $(10x + 6y) \div 2$（3点）

(5) $(24a - 15b) \div (-3)$（4点）

(6) $(-x + 2y) \div \frac{1}{5}$（4点）

## 大問3：かっこがある式の計算（配点20点）

次の計算をしなさい。

(1) $2(x + y) + 6(x - 2y)$（3点）

(2) $3(7a - 2b) - 5(4a - b)$（4点）

(3) $4(2x - y) - 3(x - 3y)$（4点）

(4) $-2(3a + 5b) + 4(2a - b)$（4点）

(5) $\frac{1}{2}(6x - 4y) - \frac{1}{3}(9x - 6y)$（5点）

## 大問4：分数の形の式の計算（配点20点）

次の計算をしなさい。

(1) $\frac{3x - y}{2} + \frac{x + 3y}{2}$（3点）

(2) $\frac{5a + b}{3} - \frac{2a - b}{3}$（3点）

(3) $\frac{4a + b}{3} + \frac{a - 7b}{5}$（4点）

(4) $\frac{3x - y}{2} - \frac{3x - 2y}{3}$（5点）

(5) $\frac{2x + y}{4} - \frac{x - 3y}{6}$（5点）

## 大問5：式の値（配点20点）

次の式を簡単にしてから、値を求めなさい。

(1) $x = 3, y = -2$ のとき、$(5x + 3y) - (2x - y)$ の値を求めなさい。（3点）

(2) $a = -1, b = 4$ のとき、$2(3a - b) + 3(a + 2b)$ の値を求めなさい。（4点）

(3) $x = 4, y = -3$ のとき、$(3x - 4y) - (2x - 5y)$ の値を求めなさい。（4点）

(4) $a = \frac{1}{2}, b = -\frac{1}{3}$ のとき、$6(a + 2b) - 3(2a - b)$ の値を求めなさい。（4点）

(5) $x = -3, y = 5$ のとき、$\frac{3x - y}{2} - \frac{x - 3y}{2}$ の値を求めなさい。（5点）
