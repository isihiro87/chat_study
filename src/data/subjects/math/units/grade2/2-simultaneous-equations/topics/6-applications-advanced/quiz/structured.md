# 速さ・濃度の文章題 大問小問

## 大問1: 速さ・時間・距離の問題（配点 25点）

次の問いに答えなさい。

(1) A 地点から B 地点まで行きは時速 4km、帰りは時速 6km で歩いた。往復にかかった時間が合計 5 時間のとき、A 地点から B 地点までの距離を求めなさい。（5点）
→ 行き $x$ 時間, 帰り $y$ 時間。$\begin{cases} x + y = 5 \\ 4x = 6y \end{cases}$。②: $2x = 3y$, $x = \dfrac{3y}{2}$。①に代入: $\dfrac{3y}{2} + y = 5$, $y = 2$, $x = 3$。距離 $= 4 \times 3 = 12$ km。

(2) 家から公園まで分速 60m で歩き、公園から駅まで分速 80m で歩いたところ、合計 18 分かかった。家から駅までの道のりは 1200m である。家から公園、公園から駅の距離をそれぞれ求めなさい。（5点）
→ 家→公園 $x$ m, 公園→駅 $y$ m。$\begin{cases} x + y = 1200 \\ \dfrac{x}{60} + \dfrac{y}{80} = 18 \end{cases}$。②$\times 240$: $4x + 3y = 4320$。①$\times 3$: $3x + 3y = 3600$。引くと $x = 720$, $y = 480$。

(3) サイクリングで行きは時速 15km、帰りは時速 10km で同じ道を走った。往復にかかった時間は合計 5 時間だった。片道の距離を求めなさい。（5点）
→ 行き $x$ 時間, 帰り $y$ 時間。$\begin{cases} x + y = 5 \\ 15x = 10y \end{cases}$。②: $3x = 2y$, $x = \dfrac{2y}{3}$。①に代入: $\dfrac{2y}{3} + y = 5$, $\dfrac{5y}{3} = 5$, $y = 3$, $x = 2$。距離 $= 15 \times 2 = 30$ km。

(4) A さんは家から学校まで 1490m の道のりを、はじめは分速 70m で歩き、途中から分速 200m で走った。全体で 12 分かかった。歩いた時間と走った時間を求めなさい。（5点）
→ 歩いた時間 $x$ 分, 走った時間 $y$ 分。$\begin{cases} x + y = 12 \\ 70x + 200y = 1490 \end{cases}$。①$\times 70$: $70x + 70y = 840$。引くと $130y = 650$, $y = 5$, $x = 7$。歩き 7 分（490m）、走り 5 分（1000m）。

(5) 家から峠まで上り坂を時速 3km で、峠から学校まで下り坂を時速 5km で歩き、全体で 52 分かかった。全体の道のりが 4km のとき、上り坂と下り坂の道のりをそれぞれ求めなさい。（5点）
→ 上り $x$ km, 下り $y$ km。$\begin{cases} x + y = 4 \\ \dfrac{x}{3} + \dfrac{y}{5} = \dfrac{52}{60} = \dfrac{13}{15} \end{cases}$。②$\times 15$: $5x + 3y = 13$。①$\times 3$: $3x + 3y = 12$。引くと $2x = 1$, $x = 0.5$, $y = 3.5$。上り 0.5km, 下り 3.5km。

## 大問2: 食塩水の濃度の問題（配点 25点）

次の問いに答えなさい。

(1) 5% の食塩水 $x$ g と 10% の食塩水 $y$ g を混ぜて、8% の食塩水 400g を作った。$x$, $y$ の値を求めなさい。（5点）
→ $\begin{cases} x + y = 400 \\ 0.05x + 0.10y = 32 \end{cases}$。②$\times 100$: $5x + 10y = 3200$。①$\times 5$: $5x + 5y = 2000$。引くと $5y = 1200$, $y = 240$, $x = 160$。

(2) 3% の食塩水と 9% の食塩水を混ぜて、6% の食塩水 300g を作りたい。それぞれ何 g 混ぜればよいか。（5点）
→ 3% が $x$ g, 9% が $y$ g。$\begin{cases} x + y = 300 \\ 0.03x + 0.09y = 18 \end{cases}$。②$\times 100$: $3x + 9y = 1800$。①$\times 3$: $3x + 3y = 900$。引くと $6y = 900$, $y = 150$, $x = 150$。

(3) 2% の食塩水 $x$ g と 8% の食塩水 $y$ g を混ぜたら 5% の食塩水が 600g できた。$x$, $y$ を求めなさい。（5点）
→ $\begin{cases} x + y = 600 \\ 0.02x + 0.08y = 30 \end{cases}$。②$\times 100$: $2x + 8y = 3000$。①$\times 2$: $2x + 2y = 1200$。引くと $6y = 1800$, $y = 300$, $x = 300$。

(4) 4% の食塩水 300g に、12% の食塩水を何 g か加えたら 6% の食塩水ができた。加えた 12% の食塩水の量を求めなさい。（5点）
→ 加えた量を $y$ g とする。$0.04 \times 300 + 0.12y = 0.06(300 + y)$。$12 + 0.12y = 18 + 0.06y$。$0.06y = 6$。$y = 100$。100g。

(5) $A$% の食塩水 200g と 10% の食塩水 300g を混ぜたら 7% の食塩水ができた。$A$ の値を求めなさい。（5点）
→ $\dfrac{A}{100} \times 200 + 0.10 \times 300 = 0.07 \times 500$。$2A + 30 = 35$。$2A = 5$。$A = 2.5$。$A = 2.5$%。
