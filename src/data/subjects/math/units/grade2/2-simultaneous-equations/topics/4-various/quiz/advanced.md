# いろいろな連立方程式 発展問題

## 問1: かっこと分数の複合問題

次の連立方程式を解きなさい。

(1) $\begin{cases} \dfrac{2(x + 1)}{3} + y = 5 \\ x - 2y = -1 \end{cases}$
→ ①に $3$ をかけると $2(x+1) + 3y = 15$ → $2x + 3y = 13$。②より $x = 2y - 1$。代入: $2(2y-1) + 3y = 13$ → $7y = 15$ → $y = \dfrac{15}{7}$、$x = \dfrac{23}{7}$。

(2) $\begin{cases} \dfrac{x - 3}{2} = \dfrac{y + 1}{3} \\ x + y = 8 \end{cases}$
→ ①を整理: $3(x-3) = 2(y+1)$ → $3x - 2y = 11$。②と連立: ②×2: $2x + 2y = 16$。足すと $5x = 27$、$x = \dfrac{27}{5}$、$y = \dfrac{13}{5}$。

(3) $\begin{cases} 3(x + y) - 2(x - y) = 13 \\ \dfrac{x}{4} + \dfrac{y}{3} = 2 \end{cases}$
→ ①を整理: $3x + 3y - 2x + 2y = 13$ → $x + 5y = 13$。②に $12$ をかけると $3x + 4y = 24$。①×3: $3x + 15y = 39$。引くと $11y = 15$、$y = \dfrac{15}{11}$、$x = \dfrac{68}{11}$。

(4) $\begin{cases} 0.5(x + 2) - 0.3y = 2.1 \\ \dfrac{x}{3} + \dfrac{y}{6} = 1 \end{cases}$
→ ①を展開: $0.5x + 1 - 0.3y = 2.1$ → $0.5x - 0.3y = 1.1$。$10$ 倍: $5x - 3y = 11$。②に $6$ をかけると $2x + y = 6$ → $y = 6 - 2x$。代入: $5x - 3(6-2x) = 11$ → $11x = 29$、$x = \dfrac{29}{11}$、$y = \dfrac{8}{11}$。

## 問2: A = B = C の応用

次の方程式を解きなさい。

(1) $\dfrac{x + 2}{3} = \dfrac{y - 1}{2} = \dfrac{x + y}{5}$
→ $A = B$: $2(x+2) = 3(y-1)$ → $2x - 3y = -7$。$B = C$: $5(y-1) = 2(x+y)$ → $-2x + 3y = 5$…$A + C$: 足すと $0 = -2$。矛盾なので再計算。$A = C$: $5(x+2) = 3(x+y)$ → $5x + 10 = 3x + 3y$ → $2x - 3y = -10$。$A = B$ と $A = C$: $\begin{cases} 2x - 3y = -7 \\ 2x - 3y = -10 \end{cases}$。矛盾なので解なし。

(2) $2x + y = x + 3y = 3x - y + 4$
→ $\begin{cases} 2x + y = x + 3y \\ x + 3y = 3x - y + 4 \end{cases}$。①: $x - 2y = 0$ → $x = 2y$。②: $-2x + 4y = 4$ → $-x + 2y = 2$。代入: $-2y + 2y = 2$ → $0 = 2$。解なし。再検討: ②: $x + 3y = 3x - y + 4$ → $-2x + 4y = 4$ → $x = 2y - 2$。①に代入: $2y - 2 = 2y$。$-2 = 0$で矛盾。よって $A = B$ と $A = C$ で: ①: $x = 2y$。$A = C$: $2x + y = 3x - y + 4$ → $-x + 2y = 4$。代入: $-2y + 2y = 4$ → $0 = 4$。解なし。

(3) $x + 2y = 3x - y = 10$
→ $\begin{cases} x + 2y = 10 \\ 3x - y = 10 \end{cases}$。②より $y = 3x - 10$。代入: $x + 2(3x-10) = 10$ → $7x = 30$、$x = \dfrac{30}{7}$、$y = \dfrac{20}{7}$。

(4) $\dfrac{x}{2} = \dfrac{y}{3} = \dfrac{x + y}{5}$
→ $A = B$: $3x = 2y$ → $y = \dfrac{3x}{2}$。$A = C$: $5x = 2(x+y) = 2x + 2y$ → $3x = 2y$。同じ式になるため $y = \dfrac{3x}{2}$ のみ。不定（$x$ は任意）。例: $x = 2, y = 3$ は解の1つ。

## 問3: 解と文字の値（応用）

(1) 連立方程式 $\begin{cases} ax + by = 5 \\ bx + ay = 7 \end{cases}$ の解が $x = 3, y = 1$ のとき、$a$ と $b$ の値を求めよ。
→ 代入: $\begin{cases} 3a + b = 5 \\ 3b + a = 7 \end{cases}$。①より $b = 5 - 3a$。代入: $3(5-3a) + a = 7$ → $-8a = -8$ → $a = 1,\ b = 2$。

(2) 連立方程式 $\begin{cases} ax + by = c \\ 2x - y = 1 \end{cases}$ の解が $x = 1, y = 1$ で、$a + b + c = 10$ のとき、$a, b, c$ を求めよ。
→ ②の検算: $2 - 1 = 1$。①に代入: $a + b = c$。$a + b + c = 10$ かつ $c = a + b$ → $2c = 10$ → $c = 5$。$a + b = 5$。$a, b$ は $a + b = 5$ を満たす任意の組。追加条件がないため不定。

(3) $\begin{cases} 2x + 3y = a \\ x - y = b \end{cases}$ の解が $x = p, y = p + 1$（$p$ は正の整数）で、$a = 20$ のとき、$p$ と $b$ の値を求めよ。
→ ①に代入: $2p + 3(p+1) = 20$ → $5p + 3 = 20$ → $p = \dfrac{17}{5}$。$p$ は正の整数ではないため、条件を再確認。$a = 20$ の場合、$p = \dfrac{17}{5}$ で整数にならない。$a = 18$ なら $5p = 15$、$p = 3$、$y = 4$。$b = 3 - 4 = -1$。

(4) 連立方程式 $\begin{cases} ax + 2y = 11 \\ x + by = 7 \end{cases}$ の解と $\begin{cases} 3x - y = 5 \\ x + 2y = 4 \end{cases}$ の解が同じとき、$a$ と $b$ の値を求めよ。
→ まず第2の連立方程式を解く: ①×2: $6x - 2y = 10$。②: $x + 2y = 4$。足すと $7x = 14$、$x = 2$、$y = 1$。第1の連立方程式に代入: $2a + 2 = 11$ → $a = \dfrac{9}{2}$。$2 + b = 7$ → $b = 5$。

## 問4: 複雑な連立方程式

(1) $\begin{cases} \dfrac{x + 2y}{3} - \dfrac{2x - y}{4} = 1 \\ \dfrac{x}{2} + y = 5 \end{cases}$
→ ①に $12$ をかけると $4(x+2y) - 3(2x-y) = 12$ → $4x + 8y - 6x + 3y = 12$ → $-2x + 11y = 12$。②に $2$ をかけると $x + 2y = 10$。②'×2: $2x + 4y = 20$。①'+②'×2 ではなく①'と②': $-2x + 11y = 12$、$x + 2y = 10$ → ②'×2: $2x + 4y = 20$。足すと $15y = 32$、$y = \dfrac{32}{15}$、$x = 10 - \dfrac{64}{15} = \dfrac{86}{15}$。

(2) $\begin{cases} 0.2(x - 3) + 0.5y = 1.4 \\ 0.1x + 0.3(y + 1) = 1.1 \end{cases}$
→ ①を展開: $0.2x - 0.6 + 0.5y = 1.4$ → $0.2x + 0.5y = 2$。$10$ 倍: $2x + 5y = 20$。②を展開: $0.1x + 0.3y + 0.3 = 1.1$ → $0.1x + 0.3y = 0.8$。$10$ 倍: $x + 3y = 8$。②'×2: $2x + 6y = 16$。①'から引くと $-y = 4$、$y = -4$… 検算: $x = 8 - 3(-4) = 20$。①確認: $2(20) + 5(-4) = 40 - 20 = 20$。正しくは $x = 20, y = -4$。

(3) $\begin{cases} \dfrac{3x - 1}{2} = \dfrac{2y + 3}{5} \\ 0.4x - 0.6y = -1.0 \end{cases}$
→ ①を整理: $5(3x-1) = 2(2y+3)$ → $15x - 5 = 4y + 6$ → $15x - 4y = 11$。②を $10$ 倍: $4x - 6y = -10$ → $2x - 3y = -5$。②'×4: $8x - 12y = -20$。①×3: $45x - 12y = 33$。引くと $37x = 53$、$x = \dfrac{53}{37}$。

## 問5: 思考力問題

(1) 連立方程式 $\begin{cases} 2x + 3y = 12 \\ 4x + 6y = k \end{cases}$ が解をもつための $k$ の値を求めよ。
→ ①を $2$ 倍すると $4x + 6y = 24$。②と一致するためには $k = 24$。$k \neq 24$ のとき解なし。

(2) ある連立方程式の解が $x = a + 1, y = 2a - 3$ であり、$x + y = 6$ を満たすとき、$a$ の値と解を求めよ。
→ $(a+1) + (2a-3) = 6$ → $3a - 2 = 6$ → $a = \dfrac{8}{3}$。$x = \dfrac{11}{3},\ y = \dfrac{7}{3}$。

(3) $\begin{cases} x + y + z = 10 \\ x - y = 2 \\ y + z = 7 \end{cases}$ を解け（3元1次方程式の導入）。
→ ②より $x = y + 2$。③より $z = 7 - y$。①に代入: $(y+2) + y + (7-y) = 10$ → $y + 9 = 10$ → $y = 1$。$x = 3,\ z = 6$。
