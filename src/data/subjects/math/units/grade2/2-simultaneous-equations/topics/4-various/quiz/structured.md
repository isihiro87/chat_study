# いろいろな連立方程式 大問小問

## 大問1: かっこがある連立方程式（配点 20点）

次の連立方程式を解きなさい。

(1) $\begin{cases} 2(x + 3) + y = 11 \\ x - y = 1 \end{cases}$（3点）
→ $x = 2,\ y = 1$。①を整理: $2x + y = 5$。②と足すと $3x = 6$、$x = 2$、$y = 1$。

(2) $\begin{cases} 3(x - 2) - y = 1 \\ x + 2y = 7 \end{cases}$（3点）
→ $x = 3,\ y = 2$。①を整理: $3x - y = 7$。②と①を連立: ①+②で $y$ を消す → ①×2: $6x - 2y = 14$、②: $x + 2y = 7$。足すと $7x = 21$、$x = 3$、$y = 2$。

(3) $\begin{cases} 4(x + 1) + 3(y - 2) = 8 \\ 2x - y = 0 \end{cases}$（4点）
→ $x = 1,\ y = 2$。①を整理: $4x + 3y = 10$。②より $y = 2x$。代入: $4x + 6x = 10$、$x = 1$、$y = 2$。

(4) $\begin{cases} 5(x - 1) = 3(y + 1) \\ x + y = 6 \end{cases}$（4点）
→ $x = 4,\ y = 2$。①を整理: $5x - 3y = 8$。②を $3$ 倍して足すと $8x = 26$… ②×3: $3x + 3y = 18$。①+: $8x = 26$、$x = \dfrac{13}{4}$。検算: $\dfrac{13}{4} + y = 6$ → $y = \dfrac{11}{4}$。①確認: $5 \cdot \dfrac{13}{4} - 5 = \dfrac{45}{4}$、$3 \cdot \dfrac{11}{4} + 3 = \dfrac{45}{4}$。正しくは $x = \dfrac{13}{4},\ y = \dfrac{11}{4}$。

(5) $\begin{cases} 2(3x - y) + 5 = 3(x + 2) \\ x + 2y = 9 \end{cases}$（3点）
→ ①を整理: $6x - 2y + 5 = 3x + 6$ → $3x - 2y = 1$。②×2: $2x + 4y = 18$。①×2: $6x - 4y = 2$。足すと $8x = 20$、$x = \dfrac{5}{2}$、$y = \dfrac{13}{4}$。

(6) $\begin{cases} -(x + 4) + 3y = 5 \\ 2x + y = 7 \end{cases}$（3点）
→ ①を整理: $-x + 3y = 9$。②と足すと $x + 4y = 16$…②×1: $2x + y = 7$。①+②×1 ではなく①: $-x + 3y = 9$、②: $2x + y = 7$。①×2: $-2x + 6y = 18$。②と足すと $7y = 25$、$y = \dfrac{25}{7}$、$x = \dfrac{24}{7} - \dfrac{25}{7} = \dfrac{-1}{7}$… $2x = 7 - \dfrac{25}{7} = \dfrac{24}{7}$、$x = \dfrac{12}{7}$。

## 大問2: 分数を含む連立方程式（配点 20点）

次の連立方程式を解きなさい。

(1) $\begin{cases} \dfrac{x}{2} + \dfrac{y}{3} = 2 \\ x + y = 5 \end{cases}$（3点）
→ $x = 2,\ y = 3$。①に $6$ をかけると $3x + 2y = 12$。②を $2$ 倍: $2x + 2y = 10$。引くと $x = 2$、$y = 3$。

(2) $\begin{cases} \dfrac{x}{4} + \dfrac{y}{2} = 3 \\ x - y = 2 \end{cases}$（3点）
→ ①に $4$ をかけると $x + 2y = 12$。②: $x - y = 2$。引くと $3y = 10$、$y = \dfrac{10}{3}$、$x = \dfrac{16}{3}$。

(3) $\begin{cases} \dfrac{2x}{3} - \dfrac{y}{6} = 1 \\ x + y = 4 \end{cases}$（4点）
→ ①に $6$ をかけると $4x - y = 6$。②: $x + y = 4$。足すと $5x = 10$、$x = 2$、$y = 2$。

(4) $\begin{cases} \dfrac{x - 1}{3} + \dfrac{y}{2} = 2 \\ 2x + y = 9 \end{cases}$（4点）
→ ①に $6$ をかけると $2(x-1) + 3y = 12$ → $2x + 3y = 14$。②: $2x + y = 9$。引くと $2y = 5$、$y = \dfrac{5}{2}$、$x = \dfrac{13}{4}$。

(5) $\begin{cases} \dfrac{x}{5} + \dfrac{y}{2} = 1 \\ \dfrac{x}{2} - \dfrac{y}{3} = 2 \end{cases}$（3点）
→ ①に $10$ をかけると $2x + 5y = 10$。②に $6$ をかけると $3x - 2y = 12$。①×2: $4x + 10y = 20$。②×5: $15x - 10y = 60$。足すと $19x = 80$、$x = \dfrac{80}{19}$、$y = \dfrac{30}{19}$。

(6) $\begin{cases} \dfrac{x + y}{2} = 3 \\ \dfrac{x - y}{3} = 1 \end{cases}$（3点）
→ ①より $x + y = 6$。②より $x - y = 3$。足すと $2x = 9$、$x = \dfrac{9}{2}$、$y = \dfrac{3}{2}$。

## 大問3: 小数を含む連立方程式（配点 20点）

次の連立方程式を解きなさい。

(1) $\begin{cases} 0.2x + 0.5y = 1.9 \\ x + y = 5 \end{cases}$（3点）
→ $x = 2,\ y = 3$。①を $10$ 倍: $2x + 5y = 19$。②を $2$ 倍: $2x + 2y = 10$。引くと $3y = 9$、$y = 3$、$x = 2$。

(2) $\begin{cases} 0.3x + 0.4y = 1.7 \\ x - y = 1 \end{cases}$（3点）
→ $x = 3,\ y = 2$。①を $10$ 倍: $3x + 4y = 17$。②より $x = y + 1$。代入: $3(y+1) + 4y = 17$ → $7y = 14$、$y = 2$、$x = 3$。

(3) $\begin{cases} 0.5x - 0.2y = 1.1 \\ 0.3x + 0.1y = 1.0 \end{cases}$（4点）
→ ①を $10$ 倍: $5x - 2y = 11$。②を $10$ 倍: $3x + y = 10$ → $y = 10 - 3x$。代入: $5x - 2(10-3x) = 11$ → $11x = 31$、$x = \dfrac{31}{11}$。$y = 10 - \dfrac{93}{11} = \dfrac{17}{11}$。

(4) $\begin{cases} 1.2x - 0.3y = 0.9 \\ 0.4x + 0.5y = 2.3 \end{cases}$（4点）
→ ①を $10$ 倍: $12x - 3y = 9$ → $4x - y = 3$。②を $10$ 倍: $4x + 5y = 23$。引くと $-6y = -20$、$y = \dfrac{10}{3}$。$x = \dfrac{3 + \frac{10}{3}}{4} = \dfrac{19}{12}$。

(5) $\begin{cases} 0.1x + 0.2y = 1.0 \\ 0.4x - 0.1y = 0.5 \end{cases}$（3点）
→ ①を $10$ 倍: $x + 2y = 10$。②を $10$ 倍: $4x - y = 5$ → $y = 4x - 5$。代入: $x + 2(4x-5) = 10$ → $9x = 20$、$x = \dfrac{20}{9}$、$y = \dfrac{35}{9}$。

(6) $\begin{cases} 0.02x + 0.05y = 0.16 \\ x + y = 5 \end{cases}$（3点）
→ ①を $100$ 倍: $2x + 5y = 16$。②を $2$ 倍: $2x + 2y = 10$。引くと $3y = 6$、$y = 2$、$x = 3$。

## 大問4: A = B = C の形（配点 20点）

次の方程式を解きなさい。

(1) $2x + 1 = 3y - 2 = 7$（3点）
→ $x = 3,\ y = 3$。$2x + 1 = 7$ → $x = 3$。$3y - 2 = 7$ → $y = 3$。

(2) $x + y = 2x - 1 = 5$（3点）
→ $x = 3,\ y = 2$。$2x - 1 = 5$ → $x = 3$。$x + y = 5$ → $y = 2$。

(3) $3x + y = 2x + 3y = 10$（4点）
→ $\begin{cases} 3x + y = 10 \\ 2x + 3y = 10 \end{cases}$。①×3: $9x + 3y = 30$。引くと $7x = 20$、$x = \dfrac{20}{7}$、$y = \dfrac{10}{7}$。

(4) $x - y = 3x - 5y = 2$（4点）
→ $\begin{cases} x - y = 2 \\ 3x - 5y = 2 \end{cases}$。①より $x = y + 2$。代入: $3(y+2) - 5y = 2$ → $-2y = -4$ → $y = 2$、$x = 4$。

(5) $\dfrac{x + 1}{2} = \dfrac{y - 1}{3} = 2$（3点）
→ $x = 3,\ y = 7$。$\dfrac{x+1}{2} = 2$ → $x = 3$。$\dfrac{y-1}{3} = 2$ → $y = 7$。

(6) $5x - y = 3x + y = 8$（3点）
→ $\begin{cases} 5x - y = 8 \\ 3x + y = 8 \end{cases}$。足すと $8x = 16$、$x = 2$、$y = 2$。

## 大問5: 解と文字の値（配点 20点）

(1) $\begin{cases} ax + 2y = 7 \\ 3x + by = 8 \end{cases}$ の解が $x = 1, y = 2$ のとき、$a$ と $b$ の値を求めよ。（3点）
→ $a = 3,\ b = \dfrac{5}{2}$。①に代入: $a + 4 = 7$ → $a = 3$。②に代入: $3 + 2b = 8$ → $b = \dfrac{5}{2}$。

(2) $\begin{cases} 2x + ay = 10 \\ bx - y = 1 \end{cases}$ の解が $x = 3, y = 2$ のとき、$a$ と $b$ の値を求めよ。（3点）
→ $a = 2,\ b = 1$。①に代入: $6 + 2a = 10$ → $a = 2$。②に代入: $3b - 2 = 1$ → $b = 1$。

(3) $\begin{cases} ax - by = 1 \\ 2x + 3y = 13 \end{cases}$ の解が $x = 2, y = 3$ のとき、$a$ と $b$ の値を求めよ。（4点）
→ ②の検算: $4 + 9 = 13$。①に代入: $2a - 3b = 1$。$a, b$ の条件が1つなので不定。追加条件が必要。$a = 2, b = 1$ は解の1つ。

(4) $\begin{cases} ax + by = 5 \\ bx + ay = 7 \end{cases}$ の解が $x = 3, y = 1$ のとき、$a$ と $b$ の値を求めよ。（4点）
→ 代入すると $\begin{cases} 3a + b = 5 \\ 3b + a = 7 \end{cases}$。①より $b = 5 - 3a$。代入: $3(5-3a) + a = 7$ → $15 - 9a + a = 7$ → $-8a = -8$ → $a = 1$、$b = 2$。

(5) 連立方程式 $\begin{cases} x + 2y = a \\ 3x - y = b \end{cases}$ の解が $x = 2, y = -1$ のとき、$a$ と $b$ の値を求めよ。（3点）
→ ①に代入: $2 - 2 = a$ → $a = 0$。②に代入: $6 + 1 = b$ → $b = 7$。

(6) 連立方程式 $\begin{cases} ax + by = 8 \\ bx - ay = -1 \end{cases}$ の解が $x = 3, y = 1$ のとき、$a$ と $b$ の値を求めよ。（3点）
→ 代入すると $\begin{cases} 3a + b = 8 \\ 3b - a = -1 \end{cases}$。②より $a = 3b + 1$。代入: $3(3b+1) + b = 8$ → $10b = 5$ → $b = \dfrac{1}{2}$、$a = \dfrac{5}{2}$。
