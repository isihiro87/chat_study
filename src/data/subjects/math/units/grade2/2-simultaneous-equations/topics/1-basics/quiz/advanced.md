# 連立方程式の基本 発展問題

## 問1: 解の条件から方程式を作る

(1) $x = 2, y = 3$ が $ax + y = 7$ の解であるとき、$a$ の値を求めなさい。
→ $2a + 3 = 7$ より $2a = 4$、$a = 2$。

(2) $x = 3, y = -1$ が $2x + by = 5$ の解であるとき、$b$ の値を求めなさい。
→ $2 \times 3 + b \times (-1) = 5$ より $6 - b = 5$、$b = 1$。

(3) $x = 1, y = 4$ が $\begin{cases} ax + y = 6 \\ x + by = 9 \end{cases}$ の解であるとき、$a, b$ の値をそれぞれ求めなさい。
→ 式①: $a + 4 = 6$ より $a = 2$。式②: $1 + 4b = 9$ より $4b = 8$、$b = 2$。

(4) $x = -1, y = 5$ が $\begin{cases} 3x + ay = 7 \\ bx + 2y = 8 \end{cases}$ の解であるとき、$a, b$ の値を求めなさい。
→ 式①: $-3 + 5a = 7$ より $5a = 10$、$a = 2$。式②: $-b + 10 = 8$ より $b = 2$。

## 問2: 共通解の問題

(1) $x + y = 5$ と $2x + y = 8$ と $3x - y = 7$ のうち、$x = 3, y = 2$ を共通の解にもつものをすべて選びなさい。
→ $3 + 2 = 5$ ✓、$6 + 2 = 8$ ✓、$9 - 2 = 7$ ✓。3つすべてが $x = 3, y = 2$ を解にもつ。

(2) 3つの二元一次方程式 $x + y = 6$、$2x - y = 3$、$x + 3y = 12$ をすべて同時に満たす $x, y$ の組を求めなさい。
→ $x + y = 6$ と $2x - y = 3$ を足すと $3x = 9$、$x = 3$、$y = 3$。確認: $3 + 9 = 12$ ✓。よって $x = 3, y = 3$。

(3) 3つの方程式 $x + 2y = 7$、$3x + y = 11$、$2x - y = a$ がすべて同じ解をもつとき、$a$ の値を求めなさい。
→ $x + 2y = 7$ と $3x + y = 11$ から、式②$\times 2$: $6x + 2y = 22$。引くと $5x = 15$、$x = 3$、$y = 2$。$2 \times 3 - 2 = 4$ より $a = 4$。

(4) $\begin{cases} x + y = 8 \\ 2x + 3y = 18 \end{cases}$ と $\begin{cases} x + y = 8 \\ ax - y = 2 \end{cases}$ が同じ解をもつとき、$a$ の値を求めなさい。
→ 最初の連立方程式: $2x + 3y = 18$ と $x + y = 8$ を解く。式②$\times 2$: $2x + 2y = 16$。引くと $y = 2$、$x = 6$。$6a - 2 = 2$ より $a = \dfrac{2}{3}$。

## 問3: 文章題から連立方程式を立てる

(1) ある数 $x$ と $y$ について、和が12で差が4である。$x > y$ として連立方程式を立て、表を使って解を求めなさい。
→ $\begin{cases} x + y = 12 \\ x - y = 4 \end{cases}$。表から共通解は $(8, 4)$。$x = 8, y = 4$。

(2) 1本80円の鉛筆を $x$ 本と、1本120円のボールペンを $y$ 本買ったら、合計で5本、合計金額は480円だった。連立方程式を立てなさい。
→ $\begin{cases} x + y = 5 \\ 80x + 120y = 480 \end{cases}$

(3) (2) の連立方程式で $x = 1, 2, 3, 4$ のときにそれぞれ式を満たすか確かめ、解を求めなさい。
→ $x = 1, y = 4$: $80 + 480 = 560 \neq 480$ ✗。$x = 2, y = 3$: $160 + 360 = 520 \neq 480$ ✗。$x = 3, y = 2$: $240 + 240 = 480$ ✓。$x = 4, y = 1$: $320 + 120 = 440 \neq 480$ ✗。よって $x = 3, y = 2$。鉛筆3本、ボールペン2本。

(4) 兄と弟の年齢の和は28歳、差は4歳である。連立方程式を立てて表で解を求めなさい。
→ $\begin{cases} x + y = 28 \\ x - y = 4 \end{cases}$（$x$: 兄、$y$: 弟）。$x + y = 28$ と $x - y = 4$ の表で、$x = 16, y = 12$ が共通。兄16歳、弟12歳。

## 問4: 解の存在と条件

(1) $\begin{cases} x + y = 5 \\ 2x + 2y = 10 \end{cases}$ は解をもつか。もつ場合、解はいくつあるか。
→ 式②は式①の両辺を2倍したものなので、実質的に1つの式と同じ。解は無限にある（$x + y = 5$ を満たすすべての組が解）。

(2) $\begin{cases} x + y = 5 \\ x + y = 7 \end{cases}$ は解をもつか。理由とともに答えなさい。
→ 解をもたない。$x + y$ が同時に5と7になることはありえないため。

(3) 連立方程式の解がちょうど1組に決まるための条件を、(1)(2)をふまえて説明しなさい。
→ 2つの式が実質的に同じ（一方が他方の定数倍）でなく、かつ矛盾しない（$x + y = 5$ と $x + y = 7$ のような場合でない）とき、解はちょうど1組に決まる。

## 問5: グラフとの関連（発展）

(1) $x + y = 4$ を $y = -x + 4$ と変形した。この式のグラフはどのような形か。
→ 右下がりの直線。傾きが $-1$、切片が $4$。

(2) $x - y = 2$ を $y = x - 2$ と変形した。(1) のグラフとこのグラフの交点は何を意味するか。
→ 交点は $\begin{cases} x + y = 4 \\ x - y = 2 \end{cases}$ の連立方程式の解を表す。2つの直線の交点の座標が連立方程式の解。

(3) (2) の2直線の交点の座標を求めなさい。
→ $y = -x + 4$ と $y = x - 2$ が等しいとき $-x + 4 = x - 2$、$6 = 2x$、$x = 3$、$y = 1$。交点は $(3, 1)$。

(4) 2つの直線が平行であるとき、連立方程式の解はどうなるか。
→ 解をもたない（交点がないため）。2つの直線の傾きが等しく切片が異なる場合に起こる。
