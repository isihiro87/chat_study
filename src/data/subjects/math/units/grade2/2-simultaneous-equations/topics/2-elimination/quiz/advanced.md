# 加減法 発展問題

## 問1: 係数に分数・小数を含む連立方程式

次の連立方程式を加減法で解きなさい。

(1) $\begin{cases} 0.2x + 0.3y = 1.3 \\ x + y = 5 \end{cases}$
→ まず①の両辺を10倍して $2x + 3y = 13$ にする。②×2で $2x + 2y = 10$。①'−②'で $y = 3$。②に代入し $x = 2$。答え: $x = 2, y = 3$。

(2) $\begin{cases} \dfrac{x}{2} + \dfrac{y}{3} = 4 \\ x + y = 10 \end{cases}$
→ ①の両辺に6をかけて $3x + 2y = 24$。②×2で $2x + 2y = 20$。①'−②'で $x = 4$。②に代入し $y = 6$。答え: $x = 4, y = 6$。

(3) $\begin{cases} 0.5x + y = 4 \\ 2x + 3y = 13 \end{cases}$
→ ①の両辺を2倍して $x + 2y = 8$。①'×2で $2x + 4y = 16$。①''−②で $y = 3$。①に代入し $0.5x + 3 = 4$、$x = 2$。答え: $x = 2, y = 3$。

(4) $\begin{cases} \dfrac{x + y}{2} = 4 \\ x - y = 2 \end{cases}$
→ ①より $x + y = 8$。これと②を足すと $2x = 10$、$x = 5$。$y = 3$。答え: $x = 5, y = 3$。

## 問2: 式の値を求める応用

(1) $\begin{cases} 2x + y = 8 \\ x - y = 1 \end{cases}$ を解き、$3x + 2y$ の値を求めよ。
→ 2式を足すと $3x = 9$、$x = 3$。②に代入し $y = 2$。$3x + 2y = 9 + 4 = 13$。

(2) $\begin{cases} x + 2y = 11 \\ 3x - y = 5 \end{cases}$ を解き、$x^2 + y^2$ の値を求めよ。
→ ②×2: $6x - 2y = 10$。①+②'で $7x = 21$、$x = 3$、$y = 4$。$x^2 + y^2 = 9 + 16 = 25$。

(3) $\begin{cases} 3x + 2y = 8 \\ 2x + 5y = 9 \end{cases}$ を解き、$xy$ の値を求めよ。
→ ①×5: $15x + 10y = 40$、②×2: $4x + 10y = 18$。引くと $11x = 22$、$x = 2$、$y = 1$。$xy = 2$。

(4) $\begin{cases} 5x + 3y = 26 \\ 2x + y = 10 \end{cases}$ を解き、$2x - y$ の値を求めよ。
→ ②×3: $6x + 3y = 30$。②'−①で $x = 4$。②に代入し $y = 2$。$2x - y = 8 - 2 = 6$。

## 問3: 加減法と代入法の比較

(1) $\begin{cases} x + y = 7 \\ 2x + 3y = 18 \end{cases}$ を加減法と代入法の両方で解き、答えが一致することを確かめよ。
→ 【加減法】①×2: $2x + 2y = 14$。②−①'で $y = 4$、$x = 3$。
【代入法】①より $x = 7 - y$。②に代入: $2(7 - y) + 3y = 18$、$14 - 2y + 3y = 18$、$y = 4$、$x = 3$。
どちらも $x = 3, y = 4$ で一致。

(2) 次の連立方程式は、加減法と代入法のどちらが解きやすいか。理由も答えよ。
$\begin{cases} 3x + 2y = 13 \\ 5x + 2y = 19 \end{cases}$
→ 加減法が解きやすい。$y$ の係数がどちらも $+2$ で揃っているので、②−①だけで $y$ が消えるから。②−①で $2x = 6$、$x = 3$。①に代入し $9 + 2y = 13$、$y = 2$。

(3) 次の連立方程式は、加減法と代入法のどちらが解きやすいか。理由も答えよ。
$\begin{cases} y = 2x + 1 \\ 3x + 2y = 16 \end{cases}$
→ 代入法が解きやすい。①がすでに $y =$ の形なので、②にそのまま代入できるから。②に代入: $3x + 2(2x + 1) = 16$、$7x + 2 = 16$、$x = 2$、$y = 5$。

## 問4: 間違い探し

(1) 次の解き方の間違いを指摘し、正しく解きなさい。
$\begin{cases} 3x + 2y = 16 \\ x + 2y = 10 \end{cases}$
「①+②をすると $4x + 4y = 26$ …」
→ 間違い: $y$ の係数がどちらも $+2$ なので、足しても消えない。引くべき。①−②で $2x = 6$、$x = 3$。②に代入し $3 + 2y = 10$、$y = \frac{7}{2}$。

(2) 次の解き方の間違いを指摘し、正しく解きなさい。
$\begin{cases} 2x + y = 8 \\ x - 3y = -3 \end{cases}$
「①−②: $2x + y - x - 3y = 8 - (-3)$ → $x - 2y = 11$ …」
→ 間違い: 引くときは $-(x - 3y) = -x + 3y$ としなければならない。正しくは $2x + y - (x - 3y) = 2x + y - x + 3y = x + 4y = 11$。

(3) 次の計算の間違いを指摘し、正しい答えを求めなさい。
$\begin{cases} 5x + 3y = 29 \\ 3x + 2y = 18 \end{cases}$
「①×2、②×3 → $10x + 6y = 58$、$9x + 6y = 54$。足すと $19x = 112$ …」
→ 間違い: 係数を揃えたのだから引くべき。$10x + 6y - (9x + 6y) = 58 - 54$ → $x = 4$。②に代入し $12 + 2y = 18$、$y = 3$。

## 問5: 文章題への応用

(1) ある2つの数の和は12で、差は4である。この2つの数を連立方程式を使って求めなさい。
→ 大きい方を $x$、小さい方を $y$ とすると $\begin{cases} x + y = 12 \\ x - y = 4 \end{cases}$。足すと $2x = 16$、$x = 8$、$y = 4$。

(2) りんご2個とみかん3個で440円、りんご2個とみかん1個で280円である。りんご1個とみかん1個の値段を求めなさい。
→ りんご $x$ 円、みかん $y$ 円。$\begin{cases} 2x + 3y = 440 \\ 2x + y = 280 \end{cases}$。①−②で $2y = 160$、$y = 80$。②に代入し $2x = 200$、$x = 100$。りんご100円、みかん80円。

(3) ノート3冊とペン2本で590円、ノート1冊とペン2本で330円である。ノート1冊とペン1本の値段を求めなさい。
→ ノート $x$ 円、ペン $y$ 円。$\begin{cases} 3x + 2y = 590 \\ x + 2y = 330 \end{cases}$。①−②で $2x = 260$、$x = 130$。②に代入し $130 + 2y = 330$、$y = 100$。ノート130円、ペン100円。
