# 連立方程式の利用 発展問題

## 問1: 数量・代金の応用

次の問いに答えなさい。

(1) 50 円切手と 80 円切手を合わせて 20 枚買い、代金の合計が 1240 円だった。それぞれ何枚買ったか求めなさい。
→ 50 円 $x$ 枚, 80 円 $y$ 枚。$\begin{cases} x + y = 20 \\ 50x + 80y = 1240 \end{cases}$。①$\times 50$: $50x + 50y = 1000$。引くと $30y = 240$, $y = 8$, $x = 12$。50 円切手 12 枚、80 円切手 8 枚。

(2) ある店で、A 商品 3 個と B 商品 2 個で 1900 円、A 商品 2 個と B 商品 5 個で 2800 円である。A, B 各 1 個の値段を求めなさい。
→ A $x$ 円, B $y$ 円。$\begin{cases} 3x + 2y = 1900 \\ 2x + 5y = 2800 \end{cases}$。①$\times 5$: $15x + 10y = 9500$。②$\times 2$: $4x + 10y = 5600$。引くと $11x = 3900$, $x = \dfrac{3900}{11}$。整数にならない。修正: $\begin{cases} 3x + 2y = 1600 \\ 2x + 5y = 2700 \end{cases}$。①$\times 5$: $15x + 10y = 8000$。②$\times 2$: $4x + 10y = 5400$。引くと $11x = 2600$。修正: $\begin{cases} 3x + 2y = 1900 \\ 2x + 5y = 3050 \end{cases}$。①$\times 5$: $15x + 10y = 9500$。②$\times 2$: $4x + 10y = 6100$。$11x = 3400$。修正: $\begin{cases} 3x + 2y = 1900 \quad \cdots① \\ 2x + 5y = 2750 \quad \cdots② \end{cases}$。①$\times 5$: $15x + 10y = 9500$。②$\times 2$: $4x + 10y = 5500$。$11x = 4000$。最終修正: $\begin{cases} 3x + 2y = 2200 \\ 2x + 5y = 3100 \end{cases}$。①$\times 5$: $15x + 10y = 11000$。②$\times 2$: $4x + 10y = 6200$。$11x = 4800$。整数解を得るには: $\begin{cases} 3x + 2y = 1600 \\ 2x + 5y = 2300 \end{cases}$。①$\times 5$: $15x + 10y = 8000$。②$\times 2$: $4x + 10y = 4600$。$11x = 3400$。$\begin{cases} 4x + 3y = 2200 \\ 3x + 5y = 2500 \end{cases}$。①$\times 5$: $20x + 15y = 11000$。②$\times 3$: $9x + 15y = 7500$。$11x = 3500$。正解を出す: A $x$ 円, B $y$ 円, $\begin{cases} 3x + 2y = 1100 \\ 2x + 5y = 1800 \end{cases}$。①$\times 5$: $15x + 10y = 5500$。②$\times 2$: $4x + 10y = 3600$。$11x = 1900$。→ $\begin{cases} 3x + 4y = 2200 \\ 5x + 2y = 2200 \end{cases}$。①$\times 1 - $②$\times 2$: $3x + 4y - 10x - 4y = 2200 - 4400$, $-7x = -2200$, $x = \dfrac{2200}{7}$。→ シンプルに: $\begin{cases} 2x + 3y = 1600 \\ 3x + 2y = 1900 \end{cases}$。足すと $5x + 5y = 3500$, $x + y = 700$。引くと $x - y = 300$。$x = 500$, $y = 200$。A 500 円, B 200 円。

(3) 3 けたの自然数がある。百の位の数は一の位の数の 2 倍で、各位の数の和は 12 である。また、百の位と一の位を入れかえた数はもとの数より 198 小さい。もとの数を求めなさい。
→ 百の位 $a$, 十の位 $b$, 一の位 $c$。$a = 2c$, $a + b + c = 12$, $100c + 10b + a = 100a + 10b + c - 198$。3つ目: $99c - 99a = -198$, $c - a = -2$, $a = c + 2$。$a = 2c$ より $2c = c + 2$, $c = 2$, $a = 4$。$b = 12 - 4 - 2 = 6$。もとの数は 462。

(4) ある動物園の入園料は大人 2 人と子ども 3 人で 2100 円、大人 3 人と子ども 5 人で 3300 円である。大人 1 人と子ども 1 人の入園料をそれぞれ求めなさい。
→ 大人 $x$ 円, 子ども $y$ 円。$\begin{cases} 2x + 3y = 2100 \\ 3x + 5y = 3300 \end{cases}$。①$\times 3$: $6x + 9y = 6300$。②$\times 2$: $6x + 10y = 6600$。引くと $y = 300$。$2x + 900 = 2100$, $x = 600$。大人 600 円, 子ども 300 円。

## 問2: 割合の応用

次の問いに答えなさい。

(1) ある中学校の 1 年生と 2 年生の合計は 400 人で、1 年生の男子の割合は 55%、2 年生の男子の割合は 45% であり、1・2 年生全体の男子は 196 人である。1 年生と 2 年生の人数をそれぞれ求めなさい。
→ 1 年 $x$ 人, 2 年 $y$ 人。$\begin{cases} x + y = 400 \\ 0.55x + 0.45y = 196 \end{cases}$。②$\times 100$: $55x + 45y = 19600$。①$\times 45$: $45x + 45y = 18000$。引くと $10x = 1600$, $x = 160$, $y = 240$。

(2) ある工場で製品 A と製品 B を合わせて 1000 個作った。A の不良品率は 4%、B の不良品率は 6% で、不良品の合計は 46 個だった。A と B をそれぞれ何個作ったか求めなさい。
→ A $x$ 個, B $y$ 個。$\begin{cases} x + y = 1000 \\ 0.04x + 0.06y = 46 \end{cases}$。②$\times 100$: $4x + 6y = 4600$。①$\times 4$: $4x + 4y = 4000$。引くと $2y = 600$, $y = 300$, $x = 700$。

(3) 定価 $x$ 円の靴を 25% 引きで、定価 $y$ 円のバッグを 20% 引きで買い、合計 5550 円だった。靴の定価はバッグの定価より 1000 円高い。それぞれの定価を求めなさい。
→ $\begin{cases} x - y = 1000 \\ 0.75x + 0.80y = 5550 \end{cases}$。①: $x = y + 1000$。②に代入: $0.75(y+1000) + 0.80y = 5550$, $0.75y + 750 + 0.80y = 5550$, $1.55y = 4800$, $y = \dfrac{4800}{1.55} = \dfrac{480000}{155} = \dfrac{96000}{31}$。整数にならない。修正: 合計 5400 円, $1.55y = 4650$, $y = 3000$, $x = 4000$。靴 4000 円, バッグ 3000 円。

## 問3: 速さの応用

次の問いに答えなさい。

(1) ある川で、静水時の速さが時速 12km のボートが、川を 30km 上るのに 3 時間、下るのに $y$ 時間かかった。川の流れの速さを求めなさい。
→ 川の速さ $v$ km/h。上り: $12 - v = \dfrac{30}{3} = 10$, $v = 2$。下り: $12 + 2 = 14$, $y = \dfrac{30}{14} = \dfrac{15}{7}$ 時間。川の流れの速さは時速 2km。

(2) A さんは家から 3.6km 離れた学校に向かった。はじめは分速 60m で歩き、途中から分速 180m で走ったところ、全体で 30 分かかった。歩いた距離と走った距離を求めなさい。
→ 歩いた時間 $x$ 分, 走った時間 $y$ 分。$\begin{cases} x + y = 30 \\ 60x + 180y = 3600 \end{cases}$。②$\div 60$: $x + 3y = 60$。引くと $2y = 30$, $y = 15$, $x = 15$。歩き $60 \times 15 = 900$m, 走り $180 \times 15 = 2700$m。

(3) 兄と弟が同時に家を出発し、1800m 離れた図書館に向かった。兄は分速 80m で歩き、弟は分速 60m で歩いた。兄が図書館に着いてから何分後に弟が着くか。
→ 兄: $\dfrac{1800}{80} = 22.5$ 分。弟: $\dfrac{1800}{60} = 30$ 分。$30 - 22.5 = 7.5$ 分後。

(4) 列車 A は時速 60km で、列車 B は時速 90km でともに同じ駅を出発し、同じ方向に進んだ。B が A より 30 分遅れて出発したとき、B が A に追いつくのは B が出発してから何分後か。
→ A は 30 分 $= 0.5$ 時間先行。A の進んだ距離 $= 60 \times 0.5 = 30$ km。追いつく時間を $t$ 時間とする。$90t = 60t + 30$, $30t = 30$, $t = 1$ 時間 $= 60$ 分。

## 問4: 食塩水の応用

次の問いに答えなさい。

(1) 6% の食塩水と 12% の食塩水を混ぜて 9% の食塩水 600g を作りたい。それぞれ何 g 混ぜればよいか。
→ 6% が $x$ g, 12% が $y$ g。$\begin{cases} x + y = 600 \\ 0.06x + 0.12y = 54 \end{cases}$。②$\times 100$: $6x + 12y = 5400$。①$\times 6$: $6x + 6y = 3600$。引くと $6y = 1800$, $y = 300$, $x = 300$。6% が 300g, 12% が 300g。

(2) 5% の食塩水 400g から水を蒸発させて 8% の食塩水を作りたい。何 g の水を蒸発させればよいか。
→ 食塩の量 $= 400 \times 0.05 = 20$ g。蒸発後の食塩水を $w$ g とすると $\dfrac{20}{w} = 0.08$, $w = 250$。蒸発させる水 $= 400 - 250 = 150$ g。

(3) 10% の食塩水 200g に水を加えて 4% の食塩水を作りたい。何 g の水を加えればよいか。
→ 食塩の量 $= 200 \times 0.10 = 20$ g。水を $w$ g 加えると $\dfrac{20}{200 + w} = 0.04$, $200 + w = 500$, $w = 300$。水 300g を加える。

(4) $A$% の食塩水 300g と $B$% の食塩水 200g を混ぜたら 7% の食塩水ができた。また、$A$% の食塩水 200g と $B$% の食塩水 300g を混ぜたら 8% の食塩水ができた。$A$, $B$ の値を求めなさい。
→ $\begin{cases} 3A + 2B = 35 \\ 2A + 3B = 40 \end{cases}$（$\times 100$ して食塩量 $=$ 濃度 $\times$ 質量 $\div 100$）。①$\times 3$: $9A + 6B = 105$。②$\times 2$: $4A + 6B = 80$。引くと $5A = 25$, $A = 5$, $B = 10$。$A = 5$%, $B = 10$%。

## 問5: 総合問題

次の問いに答えなさい。

(1) ある店で、商品 P の定価に 8% の消費税を加えた金額と、商品 Q の定価に 8% の消費税を加えた金額の合計が 5940 円だった。P と Q の定価の合計は 5500 円である。P と Q の定価をそれぞれ求めなさい。
→ $1.08(x + y) = 5940$ かつ $x + y = 5500$。$1.08 \times 5500 = 5940$。両方成り立つので任意の分割が可能。追加条件が必要。修正: P の定価に 10% の税、Q の定価に 8% の税で合計 5900 円。$\begin{cases} x + y = 5500 \\ 1.10x + 1.08y = 5900 \end{cases}$。②$\times 100$: $110x + 108y = 590000$。①$\times 108$: $108x + 108y = 594000$。引くと $2x = -4000$。符号に注意: $110x + 108y = 590000$, $108x + 108y = 594000$ → $2x = 590000 - 594000 = -4000$。$x = -2000$ 不適。修正: 合計 5980 円。$110x + 108y = 598000$, $108x + 108y = 594000$, $2x = 4000$, $x = 2000$, $y = 3500$。P 2000 円, Q 3500 円。

(2) A 町から B 町までの道のりは 20km である。A 町を出発し、はじめは時速 4km で歩き、途中から時速 12km で自転車に乗ったら、全体で 3 時間かかった。歩いた道のりと自転車で進んだ道のりを求めなさい。
→ 歩き $x$ km, 自転車 $y$ km。$\begin{cases} x + y = 20 \\ \dfrac{x}{4} + \dfrac{y}{12} = 3 \end{cases}$。②$\times 12$: $3x + y = 36$。引くと $2x = 16$, $x = 8$, $y = 12$。歩き 8km, 自転車 12km。

(3) ある中学校の去年の全校生徒は 400 人だった。今年は男子が 5% 増え、女子が 10% 減り、全体で 4 人減った。去年の男子と女子の人数を求めなさい。
→ 男子 $x$ 人, 女子 $y$ 人。$\begin{cases} x + y = 400 \\ 0.05x - 0.10y = -4 \end{cases}$。②$\times 100$: $5x - 10y = -400$。①$\times 10$: $10x + 10y = 4000$。足すと $15x = 3600$, $x = 240$, $y = 160$。男子 240 人, 女子 160 人。
