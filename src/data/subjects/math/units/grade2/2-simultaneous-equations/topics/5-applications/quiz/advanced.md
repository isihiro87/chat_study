# 連立方程式の利用 発展問題

## 問1: 数量・代金の応用

次の問いに答えなさい。

(1) 50 円切手と 80 円切手を合わせて 20 枚買い、代金の合計が 1240 円だった。それぞれ何枚買ったか求めなさい。
→ 50 円 $x$ 枚, 80 円 $y$ 枚。$\begin{cases} x + y = 20 \\ 50x + 80y = 1240 \end{cases}$。①$\times 50$: $50x + 50y = 1000$。引くと $30y = 240$, $y = 8$, $x = 12$。50 円切手 12 枚、80 円切手 8 枚。

(2) 2 けたの自然数の十の位を $x$、一の位を $y$ としたとき、$x + y = 11$ かつ十の位と一の位を入れかえた数がもとの数より 27 大きい。もとの自然数を求めなさい。
→ $\begin{cases} x + y = 11 \\ (10y + x) - (10x + y) = 27 \end{cases}$。②: $9y - 9x = 27$, $y - x = 3$。①+②: $2y = 14$, $y = 7$, $x = 4$。もとの数は 47。

(3) 3 けたの自然数がある。百の位の数は一の位の数の 2 倍で、各位の数の和は 12 である。また、百の位と一の位を入れかえた数はもとの数より 198 小さい。もとの数を求めなさい。
→ 百の位 $a$, 十の位 $b$, 一の位 $c$。$a = 2c$, $a + b + c = 12$, $100c + 10b + a = 100a + 10b + c - 198$。3つ目: $99c - 99a = -198$, $c - a = -2$, $a = c + 2$。$a = 2c$ より $2c = c + 2$, $c = 2$, $a = 4$。$b = 12 - 4 - 2 = 6$。もとの数は 462。

(4) ある動物園の入園料は大人 2 人と子ども 3 人で 2100 円、大人 3 人と子ども 5 人で 3300 円である。大人 1 人と子ども 1 人の入園料をそれぞれ求めなさい。
→ 大人 $x$ 円, 子ども $y$ 円。$\begin{cases} 2x + 3y = 2100 \\ 3x + 5y = 3300 \end{cases}$。①$\times 3$: $6x + 9y = 6300$。②$\times 2$: $6x + 10y = 6600$。引くと $y = 300$。$2x + 900 = 2100$, $x = 600$。大人 600 円, 子ども 300 円。

## 問2: 割合の応用

次の問いに答えなさい。

(1) ある中学校の 1 年生と 2 年生の合計は 400 人で、1 年生の男子の割合は 55%、2 年生の男子の割合は 45% であり、1・2 年生全体の男子は 196 人である。1 年生と 2 年生の人数をそれぞれ求めなさい。
→ 1 年 $x$ 人, 2 年 $y$ 人。$\begin{cases} x + y = 400 \\ 0.55x + 0.45y = 196 \end{cases}$。②$\times 100$: $55x + 45y = 19600$。①$\times 45$: $45x + 45y = 18000$。引くと $10x = 1600$, $x = 160$, $y = 240$。

(2) ある工場で製品 A と製品 B を合わせて 1000 個作った。A の不良品率は 4%、B の不良品率は 6% で、不良品の合計は 46 個だった。A と B をそれぞれ何個作ったか求めなさい。
→ A $x$ 個, B $y$ 個。$\begin{cases} x + y = 1000 \\ 0.04x + 0.06y = 46 \end{cases}$。②$\times 100$: $4x + 6y = 4600$。①$\times 4$: $4x + 4y = 4000$。引くと $2y = 600$, $y = 300$, $x = 700$。

(3) 定価 $x$ 円の靴を 25% 引きで、定価 $y$ 円のバッグを 20% 引きで買い、合計 5400 円だった。靴の定価はバッグの定価より 1000 円高い。それぞれの定価を求めなさい。
→ $\begin{cases} x - y = 1000 \\ 0.75x + 0.80y = 5400 \end{cases}$。①: $x = y + 1000$。②に代入: $0.75(y+1000) + 0.80y = 5400$, $0.75y + 750 + 0.80y = 5400$, $1.55y = 4650$, $y = 3000$, $x = 4000$。靴 4000 円, バッグ 3000 円。
