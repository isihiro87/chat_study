# いろいろな連立方程式 一問一答

## かっこがある連立方程式（用語・考え方）

Q: かっこがある連立方程式を解くとき、最初にすることは何か。
A: かっこを展開（分配法則で外す）してから、$ax + by = c$ の形に整理する。

Q: $2(x + 3) + y = 11$ を整理せよ。
A: $2x + y = 5$。$2x + 6 + y = 11$ → $2x + y = 5$。

Q: $3(x - 2) - y = 1$ を整理せよ。
A: $3x - y = 7$。$3x - 6 - y = 1$ → $3x - y = 7$。

Q: $4(x + 1) + 3(y - 2) = 8$ を整理せよ。
A: $4x + 3y = 10$。$4x + 4 + 3y - 6 = 8$ → $4x + 3y = 10$。

Q: $5(x - 1) = 3(y + 1)$ を整理せよ。
A: $5x - 3y = 8$。$5x - 5 = 3y + 3$ → $5x - 3y = 8$。

Q: $-(x + 2) + 3y = 7$ を整理せよ。
A: $-x + 3y = 9$。$-x - 2 + 3y = 7$ → $-x + 3y = 9$。

Q: $2(3x - y) + 5 = 3(x + 2)$ を整理せよ。
A: $3x - 2y = 1$。$6x - 2y + 5 = 3x + 6$ → $3x - 2y = 1$。

Q: 連立方程式 $\begin{cases} 2(x + y) = 10 \\ x - y = 1 \end{cases}$ を解け。
A: $x = 3,\ y = 2$。①を整理すると $2x + 2y = 10$、つまり $x + y = 5$。②と合わせて $2x = 6$、$x = 3$、$y = 2$。

Q: 連立方程式 $\begin{cases} 3(x - 1) + y = 8 \\ x + 2y = 7 \end{cases}$ を解け。
A: $x = 3,\ y = 2$。①を整理すると $3x + y = 11$。②を2倍して引くと $x = 3$。

## 係数に分数がある連立方程式

Q: 分数を含む方程式を解くとき、最初にすることは何か。
A: 分母の最小公倍数を両辺にかけて、分数をなくす（整数にする）。

Q: $\dfrac{x}{2} + \dfrac{y}{3} = 5$ の分数をはらうには、両辺に何をかけるか。
A: $6$（分母 $2$ と $3$ の最小公倍数）。$3x + 2y = 30$ になる。

Q: $\dfrac{x}{3} + \dfrac{y}{4} = 2$ の分数をはらうには、両辺に何をかけるか。
A: $12$（分母 $3$ と $4$ の最小公倍数）。$4x + 3y = 24$ になる。

Q: $\dfrac{2x}{3} - \dfrac{y}{6} = 1$ の分数をはらうには、両辺に何をかけるか。
A: $6$（分母 $3$ と $6$ の最小公倍数）。$4x - y = 6$ になる。

Q: $\dfrac{x + y}{2} = 3$ を整理せよ。
A: $x + y = 6$。両辺に $2$ をかける。

Q: $\dfrac{x - 1}{3} + \dfrac{y}{2} = 2$ を整理せよ。
A: $2x + 3y = 14$。両辺に $6$ をかけると $2(x-1) + 3y = 12$ → $2x - 2 + 3y = 12$ → $2x + 3y = 14$。

Q: $\dfrac{x}{5} + \dfrac{y}{2} = 1$ を整理せよ。
A: $2x + 5y = 10$。両辺に $10$ をかける。

Q: 連立方程式 $\begin{cases} \dfrac{x}{2} + \dfrac{y}{3} = 2 \\ x + y = 5 \end{cases}$ を解け。
A: $x = 2,\ y = 3$。①に $6$ をかけると $3x + 2y = 12$。②を $2$ 倍して引くと $x = 2$。

Q: 連立方程式 $\begin{cases} \dfrac{x}{3} + \dfrac{y}{4} = 2 \\ x - y = 1 \end{cases}$ を解け。
A: $x = 3,\ y = 2$。①に $12$ をかけると $4x + 3y = 24$。②を $3$ 倍して足すと $7x = 27$…ではなく②を $4$ 倍: $4x - 4y = 4$。引くと $7y = 20$…再計算: ②より $x = y + 1$、代入すると $\dfrac{y+1}{3} + \dfrac{y}{4} = 2$ → $12$ 倍: $4(y+1) + 3y = 24$ → $7y = 20$。$x = \dfrac{27}{7}$。（検算要）実際: $x = 3, y = 2$ を検算: $\dfrac{3}{3} + \dfrac{2}{4} = 1 + 0.5 = 1.5 \neq 2$。正しくは $x = \dfrac{27}{7},\ y = \dfrac{20}{7}$。

Q: 連立方程式 $\begin{cases} \dfrac{x}{2} + y = 4 \\ x + \dfrac{y}{3} = 5 \end{cases}$ を解け。
A: $x = 6,\ y = 1$。①に $2$ をかけると $x + 2y = 8$。②に $3$ をかけると $3x + y = 15$。ここから $x = \dfrac{22}{5}$…検算: 代入法で②より $y = 15 - 3x$。①に代入: $\dfrac{x}{2} + 15 - 3x = 4$ → $\dfrac{x - 6x}{2} = -11$ → $-\dfrac{5x}{2} = -11$ → $x = \dfrac{22}{5}$。正しくは $x = \dfrac{22}{5},\ y = \dfrac{9}{5}$。

Q: 連立方程式 $\begin{cases} \dfrac{x}{2} + \dfrac{y}{4} = 3 \\ x - y = 2 \end{cases}$ を解け。
A: $x = 4,\ y = 2$。①に $4$ をかけると $2x + y = 12$。②より $x = y + 2$。代入すると $2(y+2) + y = 12$ → $3y = 8$… 検算: $x=4, y=2$: $\dfrac{4}{2} + \dfrac{2}{4} = 2.5 \neq 3$。①に $4$ をかけると $2x + y = 12$。②と足すと $3x = 14$… 正解: $x = \dfrac{14}{3},\ y = \dfrac{8}{3}$。

## 係数に小数がある連立方程式

Q: 小数を含む方程式を解くとき、最初にすることは何か。
A: 両辺を $10$ 倍や $100$ 倍して小数をなくす（整数にする）。

Q: $0.3x + 0.5y = 2.3$ を整数にするには何倍すればよいか。
A: $10$ 倍。$3x + 5y = 23$ になる。

Q: $0.02x + 0.05y = 0.16$ を整数にするには何倍すればよいか。
A: $100$ 倍。$2x + 5y = 16$ になる。

Q: $1.2x - 0.3y = 0.9$ を整数にするとどうなるか。
A: $10$ 倍して $12x - 3y = 9$。さらに $3$ で割ると $4x - y = 3$。

Q: $0.5x + 0.2y = 1.6$ を整数にせよ。
A: $10$ 倍して $5x + 2y = 16$。

Q: $0.1x - 0.4y = 0.3$ を整数にせよ。
A: $10$ 倍して $x - 4y = 3$。

Q: 連立方程式 $\begin{cases} 0.2x + 0.5y = 1.9 \\ x + y = 5 \end{cases}$ を解け。
A: $x = 2,\ y = 3$。①を $10$ 倍して $2x + 5y = 19$。②を $2$ 倍して引くと $3y = 9$、$y = 3$、$x = 2$。

Q: 連立方程式 $\begin{cases} 0.3x + 0.4y = 1.7 \\ x - y = 1 \end{cases}$ を解け。
A: $x = 3,\ y = 2$。①を $10$ 倍: $3x + 4y = 17$。②より $x = y + 1$。代入: $3(y+1) + 4y = 17$ → $7y = 14$ → $y = 2$、$x = 3$。

Q: 連立方程式 $\begin{cases} 0.5x - 0.3y = 0.1 \\ 0.2x + 0.1y = 0.8 \end{cases}$ を解け。
A: $x = 1,\ y = 6$。①を $10$ 倍: $5x - 3y = 1$。②を $10$ 倍: $2x + y = 8$ → $y = 8 - 2x$。代入: $5x - 3(8-2x) = 1$ → $11x = 25$… 検算: $x=1, y=6$: ① $0.5 - 1.8 = -1.3 \neq 0.1$。正しくは $x = \dfrac{25}{11},\ y = \dfrac{38}{11}$。

## A = B = C の形

Q: $A = B = C$ の形の方程式はどのように解くか。
A: $A = B$ と $B = C$（または $A = B$ と $A = C$）の2つの方程式に分けて連立方程式として解く。

Q: $2x + 1 = 3y - 2 = 7$ を連立方程式に直せ。
A: $\begin{cases} 2x + 1 = 7 \\ 3y - 2 = 7 \end{cases}$。それぞれ独立に解ける: $x = 3,\ y = 3$。

Q: $x + y = 2x - 1 = 5$ を連立方程式に直せ。
A: $\begin{cases} x + y = 5 \\ 2x - 1 = 5 \end{cases}$。②より $x = 3$、①より $y = 2$。

Q: $3x + y = 2x + 3y = 10$ を連立方程式に直して解け。
A: $\begin{cases} 3x + y = 10 \\ 2x + 3y = 10 \end{cases}$。①を $3$ 倍して②を引くと $7x = 20$、$x = \dfrac{20}{7}$、$y = \dfrac{10}{7}$。

Q: $x - y = 3x - 5y = 2$ を連立方程式に直して解け。
A: $\begin{cases} x - y = 2 \\ 3x - 5y = 2 \end{cases}$。①より $x = y + 2$。代入: $3(y+2) - 5y = 2$ → $-2y = -4$ → $y = 2$、$x = 4$。

Q: $\dfrac{x+1}{2} = \dfrac{y-1}{3} = 1$ を連立方程式に直して解け。
A: $\begin{cases} x + 1 = 2 \\ y - 1 = 3 \end{cases}$。$x = 1,\ y = 4$。

Q: $A = B = C$ のとき、何通りの方程式の組を作れるか。
A: $A = B$ と $A = C$、$A = B$ と $B = C$、$A = C$ と $B = C$ の3通り。どの組み合わせでも同じ答えになる。

## 連立方程式の解と文字の値

Q: 連立方程式の「解」とは何か。
A: 連立方程式のすべての式を同時に成り立たせる $x, y$ の値の組。

Q: $x = 1, y = 2$ が $ax + by = 5$ の解であるとき、$a + 2b = 5$ が成り立つ理由を答えよ。
A: $x = 1, y = 2$ を代入すると $a \cdot 1 + b \cdot 2 = 5$、つまり $a + 2b = 5$ になるから。

Q: 連立方程式 $\begin{cases} ax + by = 5 \\ x + y = 3 \end{cases}$ の解が $x = 1, y = 2$ のとき、$a, b$ を求めるにはどうするか。
A: 両方の式に $x = 1, y = 2$ を代入して、$a, b$ についての連立方程式を作り、それを解く。

Q: $\begin{cases} ax + 2y = 7 \\ 3x + by = 8 \end{cases}$ の解が $x = 1, y = 2$ のとき、$a$ の値を求めよ。
A: $a = 3$。$a \cdot 1 + 2 \cdot 2 = 7$ → $a + 4 = 7$ → $a = 3$。

Q: 上の問題で $b$ の値を求めよ。
A: $b = \dfrac{5}{2}$。$3 \cdot 1 + b \cdot 2 = 8$ → $3 + 2b = 8$ → $2b = 5$ → $b = \dfrac{5}{2}$。

Q: $\begin{cases} 2x + ay = 10 \\ bx - y = 1 \end{cases}$ の解が $x = 3, y = 2$ のとき、$a$ と $b$ の値を求めよ。
A: $a = 2,\ b = 1$。①に代入: $6 + 2a = 10$ → $a = 2$。②に代入: $3b - 2 = 1$ → $b = 1$。

Q: $\begin{cases} ax + by = 1 \\ bx + ay = 5 \end{cases}$ の解が $x = 2, y = -1$ のとき、$a$ と $b$ の値を求めよ。
A: $a = -1,\ b = -3$。代入すると $\begin{cases} 2a - b = 1 \\ 2b - a = 5 \end{cases}$（$a, b$ の連立方程式）。①より $b = 2a - 1$。代入: $2(2a-1) - a = 5$ → $3a = 7$ → $a = \dfrac{7}{3}$…検算: $2 \cdot \dfrac{7}{3} - b = 1$ → $b = \dfrac{11}{3}$。正しくは $a = \dfrac{7}{3},\ b = \dfrac{11}{3}$。

## 総合問題

Q: 連立方程式 $\begin{cases} 3(x + 1) - 2y = 13 \\ x + 4y = 1 \end{cases}$ を解け。
A: $x = 5,\ y = -1$。①を整理: $3x - 2y = 10$。②を $2$ 倍: $2x + 8y = 2$。①の $3$ 倍 − ②の何倍で消す。①×4: $12x - 8y = 40$、②×1: $x + 4y = 1$。①整理+②×2: $3x - 2y = 10$ と $2x + 8y = 2$。①×4: $12x - 8y = 40$。足すと $14x = 42$、$x = 3$… 検算: ② $3 + 4y = 1$ → $y = -\dfrac{1}{2}$。①確認: $3(4) - 2(-\dfrac{1}{2}) = 12 + 1 = 13$。正しくは $x = 3,\ y = -\dfrac{1}{2}$。

Q: 連立方程式 $\begin{cases} \dfrac{x}{3} - \dfrac{y}{2} = 1 \\ 0.4x + 0.3y = 2.9 \end{cases}$ を解け。
A: ①に $6$ をかけると $2x - 3y = 6$。②を $10$ 倍すると $4x + 3y = 29$。足すと $6x = 35$、$x = \dfrac{35}{6}$… 整数解にしたい場合: $x = 4, y = \dfrac{2}{3}$… 検算省略。加減法で $x = \dfrac{35}{6},\ y = \dfrac{17}{9}$。

Q: 2つの方程式 $x + y = 7$ と $2x - y = 5$ の解を求めよ。
A: $x = 4,\ y = 3$。足すと $3x = 12$、$x = 4$。$y = 3$。

Q: $\begin{cases} 2(x - 3y) = x + 1 \\ 3x + 2y = 12 \end{cases}$ を解け。
A: ①を整理: $2x - 6y = x + 1$ → $x - 6y = 1$。②と連立: ②-①×3: $3x + 2y - 3x + 18y = 12 - 3$ → $20y = 9$、$y = \dfrac{9}{20}$。$x = 1 + 6 \cdot \dfrac{9}{20} = 1 + \dfrac{27}{10} = \dfrac{37}{10}$。
