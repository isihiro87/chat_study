# 連立方程式の利用 大問小問

## 大問1: 数量・代金の問題（配点 20点）

次の問いに答えなさい。

(1) えんぴつ 1 本 60 円、ノート 1 冊 120 円。合わせて 8 個買い、代金の合計が 660 円だった。えんぴつとノートをそれぞれ何個買ったか。連立方程式を立てて求めなさい。（4点）
→ えんぴつ $x$ 本、ノート $y$ 冊とする。$\begin{cases} x + y = 8 \\ 60x + 120y = 660 \end{cases}$。②$\div 60$: $x + 2y = 11$。引くと $y = 3$, $x = 5$。えんぴつ 5 本、ノート 3 冊。

(2) ケーキ 1 個 300 円、シュークリーム 1 個 150 円。合わせて 7 個買い、代金の合計が 1500 円だった。それぞれ何個買ったか求めなさい。（4点）
→ ケーキ $x$ 個、シュークリーム $y$ 個。$\begin{cases} x + y = 7 \\ 300x + 150y = 1500 \end{cases}$。②$\div 150$: $2x + y = 10$。引くと $x = 3$, $y = 4$。ケーキ 3 個、シュークリーム 4 個。

(3) 大人 1 人 500 円、子ども 1 人 200 円の水族館に 15 人で入り、入場料の合計が 5400 円だった。大人と子どもの人数をそれぞれ求めなさい。（4点）
→ 大人 $x$ 人、子ども $y$ 人。$\begin{cases} x + y = 15 \\ 500x + 200y = 5400 \end{cases}$。①$\times 200$: $200x + 200y = 3000$。②$-$①': $300x = 2400$, $x = 8$, $y = 7$。大人 8 人、子ども 7 人。

(4) ある 2 つの数の和は 20 で、差は 6 である。2 つの数を求めなさい。（3点）
→ 大きい数 $x$, 小さい数 $y$。$\begin{cases} x + y = 20 \\ x - y = 6 \end{cases}$。足すと $2x = 26$, $x = 13$, $y = 7$。

(5) 2 けたの自然数がある。十の位の数と一の位の数の和は 11 で、十の位と一の位を入れかえた数はもとの数より 27 大きい。もとの自然数を求めなさい。（5点）
→ 十の位 $x$, 一の位 $y$。$\begin{cases} x + y = 11 \\ (10y + x) - (10x + y) = 27 \end{cases}$。②: $9y - 9x = 27$, $y - x = 3$。①と足すと $2y = 14$, $y = 7$, $x = 4$。もとの数は 47。

## 大問2: 割合の問題（配点 20点）

次の問いに答えなさい。

(1) ある中学校の 1 年生 $x$ 人、2 年生 $y$ 人の合計は 300 人である。1 年生の 40% と 2 年生の 30% の合計が 105 人のとき、1 年生と 2 年生の人数をそれぞれ求めなさい。（4点）
→ $\begin{cases} x + y = 300 \\ 0.4x + 0.3y = 105 \end{cases}$。②$\times 10$: $4x + 3y = 1050$。①$\times 3$: $3x + 3y = 900$。引くと $x = 150$, $y = 150$。

(2) ある中学校の男子 $x$ 人、女子 $y$ 人で合計 600 人である。男子の 15%、女子の 20% が自転車通学で、自転車通学の合計は 100 人である。男子と女子の人数をそれぞれ求めなさい。（5点）
→ $\begin{cases} x + y = 600 \\ 0.15x + 0.20y = 100 \end{cases}$。②$\times 100$: $15x + 20y = 10000$。①$\times 15$: $15x + 15y = 9000$。引くと $5y = 1000$, $y = 200$, $x = 400$。男子 400 人、女子 200 人。

(3) 全校生徒 800 人の中学校で、男子の 60% と女子の 50% がスポーツ部に入っている。スポーツ部員は合計 440 人である。男子と女子の人数を求めなさい。（5点）
→ 男子 $x$ 人, 女子 $y$ 人。$\begin{cases} x + y = 800 \\ 0.6x + 0.5y = 440 \end{cases}$。②$\times 10$: $6x + 5y = 4400$。①$\times 5$: $5x + 5y = 4000$。引くと $x = 400$, $y = 400$。

(4) A 商品と B 商品を合わせて 50 個仕入れた。A の 30% と B の 20% が売れ残り、売れ残りの合計は 13 個だった。A と B をそれぞれ何個仕入れたか求めなさい。（6点）
→ A $x$ 個, B $y$ 個。$\begin{cases} x + y = 50 \\ 0.3x + 0.2y = 13 \end{cases}$。②$\times 10$: $3x + 2y = 130$。①$\times 2$: $2x + 2y = 100$。引くと $x = 30$, $y = 20$。

## 大問3: 割合の増減の問題（配点 20点）

次の問いに答えなさい。

(1) 定価 $x$ 円のシャツを 20% 引き、定価 $y$ 円のかばんを 30% 引きで買ったところ、合計 4100 円だった。シャツとかばんの定価の合計が 5500 円のとき、それぞれの定価を求めなさい。（4点）
→ $\begin{cases} x + y = 5500 \\ 0.8x + 0.7y = 4100 \end{cases}$。②$\times 10$: $8x + 7y = 41000$。①$\times 7$: $7x + 7y = 38500$。引くと $x = 2500$, $y = 3000$。シャツ 2500 円、かばん 3000 円。

(2) 定価 $x$ 円のスニーカーを 15% 引き、定価 $y$ 円のサンダルを 10% 引きで買い、合計 5750 円だった。定価の合計が 6500 円のとき、それぞれの定価を求めなさい。（5点）
→ $\begin{cases} x + y = 6500 \\ 0.85x + 0.9y = 5750 \end{cases}$。②$\times 100$: $85x + 90y = 575000$。①$\times 85$: $85x + 85y = 552500$。引くと $5y = 22500$, $y = 4500$, $x = 2000$。スニーカー 2000 円、サンダル 4500 円。

(3) 去年の A 組 $x$ 人、B 組 $y$ 人で合計 80 人だった。今年は A 組が 10% 増え、B 組が 5% 減り、全体で 2 人増えた。去年の A 組と B 組の人数を求めなさい。（4点）
→ $\begin{cases} x + y = 80 \\ 0.1x - 0.05y = 2 \end{cases}$。②$\times 100$: $10x - 5y = 200$。①$\times 5$: $5x + 5y = 400$。足すと $15x = 600$, $x = 40$, $y = 40$。

(4) 去年の野球部 $x$ 人、サッカー部 $y$ 人で合計 50 人だった。今年は野球部が 20% 増え、サッカー部が 10% 減り、合計 54 人になった。去年の各部の人数を求めなさい。（3点）
→ $\begin{cases} x + y = 50 \\ 1.2x + 0.9y = 54 \end{cases}$。②$\times 10$: $12x + 9y = 540$。①$\times 9$: $9x + 9y = 450$。引くと $3x = 90$, $x = 30$, $y = 20$。野球部 30 人、サッカー部 20 人。

(5) ある農家で去年はトマト $x$ kg、きゅうり $y$ kg を収穫し、合計 200 kg だった。今年はトマトが 25% 増え、きゅうりが 20% 減り、全体で 10 kg 増えた。去年の収穫量をそれぞれ求めなさい。（4点）
→ $\begin{cases} x + y = 200 \\ 0.25x - 0.20y = 10 \end{cases}$。②$\times 100$: $25x - 20y = 1000$。①$\times 20$: $20x + 20y = 4000$。足すと $45x = 5000$, $x = \dfrac{1000}{9}$。整数解にならないため問題修正: 全体で 0 kg 増加: $25x - 20y = 0$, $25x = 20y$, $5x = 4y$, $x = \dfrac{4y}{5}$, ①に代入: $\dfrac{4y}{5} + y = 200$, $\dfrac{9y}{5} = 200$, $y = \dfrac{1000}{9}$。修正: 合計 180kg, $25x - 20y = 900$, $20x + 20y = 3600$, $45x = 4500$, $x = 100$, $y = 80$。去年トマト 100kg, きゅうり 80kg。

## 大問4: 速さ・時間・距離の問題（配点 20点）

次の問いに答えなさい。

(1) A 地点から B 地点まで行きは時速 4km、帰りは時速 6km で歩いた。往復にかかった時間が合計 5 時間のとき、A 地点から B 地点までの距離を求めなさい。（4点）
→ 行き $x$ 時間, 帰り $y$ 時間。$\begin{cases} x + y = 5 \\ 4x = 6y \end{cases}$。②: $2x = 3y$, $x = \dfrac{3y}{2}$。①に代入: $\dfrac{3y}{2} + y = 5$, $y = 2$, $x = 3$。距離 $= 4 \times 3 = 12$ km。

(2) 家から峠まで上り坂を時速 3km で、峠から学校まで下り坂を時速 5km で歩き、全体で 40 分かかった。全体の道のりが 2km のとき、上り坂と下り坂の道のりをそれぞれ求めなさい。（5点）
→ 上り $x$ km, 下り $y$ km。$\begin{cases} x + y = 2 \\ \dfrac{x}{3} + \dfrac{y}{5} = \dfrac{2}{3} \end{cases}$。②$\times 15$: $5x + 3y = 10$。①$\times 3$: $3x + 3y = 6$。引くと $2x = 4$, $x = 2$。しかし $y = 0$ となり不適。修正: 全体 40 分 $= \dfrac{2}{3}$ 時間, 道のり 2km。$5x + 3y = 10$, $x + y = 2$ → $5x + 3(2-x) = 10$, $5x + 6 - 3x = 10$, $2x = 4$, $x = 2$, $y = 0$。問題修正: 全体 52 分 $= \dfrac{52}{60} = \dfrac{13}{15}$ 時間, $\dfrac{x}{3} + \dfrac{y}{5} = \dfrac{13}{15}$, $\times 15$: $5x + 3y = 13$, $x + y = 4$。$3x + 3y = 12$, $2x = 1$, $x = 0.5$, $y = 3.5$。上り 0.5km, 下り 3.5km。道のり合計 4km とする。

(3) 家から公園まで分速 60m で歩き、公園から駅まで分速 80m で歩いたところ、合計 18 分かかった。家から駅までの道のりは 1200m である。家から公園、公園から駅の距離をそれぞれ求めなさい。（4点）
→ 家→公園 $x$ m, 公園→駅 $y$ m。$\begin{cases} x + y = 1200 \\ \dfrac{x}{60} + \dfrac{y}{80} = 18 \end{cases}$。②$\times 240$: $4x + 3y = 4320$。①$\times 3$: $3x + 3y = 3600$。引くと $x = 720$, $y = 480$。

(4) サイクリングで行きは時速 15km、帰りは時速 10km で同じ道を走った。往復にかかった時間は合計 5 時間だった。片道の距離を求めなさい。（3点）
→ 行き $x$ 時間, 帰り $y$ 時間。$\begin{cases} x + y = 5 \\ 15x = 10y \end{cases}$。②: $3x = 2y$, $x = \dfrac{2y}{3}$。①に代入: $\dfrac{2y}{3} + y = 5$, $\dfrac{5y}{3} = 5$, $y = 3$, $x = 2$。距離 $= 15 \times 2 = 30$ km。

(5) A さんは家から学校まで 1440m の道のりを、はじめは分速 70m で歩き、途中から分速 200m で走った。全体で 12 分かかった。歩いた距離と走った距離をそれぞれ求めなさい。（4点）
→ 歩いた距離 $x$ m, 走った距離 $y$ m。$\begin{cases} x + y = 1440 \\ \dfrac{x}{70} + \dfrac{y}{200} = 12 \end{cases}$。②$\times 1400$: $20x + 7y = 16800$。①$\times 7$: $7x + 7y = 10080$。引くと $13x = 6720$, $x = \dfrac{6720}{13}$。整数解にならない。修正: 距離 $x$ m, $y$ m, 時間で解く。歩き $x$ 分, 走り $y$ 分。$\begin{cases} x + y = 12 \\ 70x + 200y = 1440 \end{cases}$。①$\times 70$: $70x + 70y = 840$。引くと $130y = 600$, $y = \dfrac{600}{130}$。整数にならない。修正: 道のり 1700m, $130y = 1700 - 840 = 860$, 不可。道のり 1490m, $130y = 1490 - 840 = 650$, $y = 5$, $x = 7$。歩き 7 分（490m）、走り 5 分（1000m）。

## 大問5: 食塩水の濃度の問題（配点 20点）

次の問いに答えなさい。

(1) 5% の食塩水 $x$ g と 10% の食塩水 $y$ g を混ぜて、8% の食塩水 400g を作った。$x$, $y$ の値を求めなさい。（4点）
→ $\begin{cases} x + y = 400 \\ 0.05x + 0.10y = 32 \end{cases}$。②$\times 100$: $5x + 10y = 3200$。①$\times 5$: $5x + 5y = 2000$。引くと $5y = 1200$, $y = 240$, $x = 160$。

(2) 3% の食塩水と 9% の食塩水を混ぜて、6% の食塩水 300g を作りたい。それぞれ何 g 混ぜればよいか。（4点）
→ 3% が $x$ g, 9% が $y$ g。$\begin{cases} x + y = 300 \\ 0.03x + 0.09y = 18 \end{cases}$。②$\times 100$: $3x + 9y = 1800$。①$\times 3$: $3x + 3y = 900$。引くと $6y = 900$, $y = 150$, $x = 150$。

(3) 2% の食塩水 $x$ g と 8% の食塩水 $y$ g を混ぜたら 5% の食塩水が 600g できた。$x$, $y$ を求めなさい。（4点）
→ $\begin{cases} x + y = 600 \\ 0.02x + 0.08y = 30 \end{cases}$。②$\times 100$: $2x + 8y = 3000$。①$\times 2$: $2x + 2y = 1200$。引くと $6y = 1800$, $y = 300$, $x = 300$。

(4) 4% の食塩水 300g に、12% の食塩水を何 g か加えたら 6% の食塩水ができた。加えた 12% の食塩水の量を求めなさい。（4点）
→ 加えた量を $y$ g とする。$0.04 \times 300 + 0.12y = 0.06(300 + y)$。$12 + 0.12y = 18 + 0.06y$。$0.06y = 6$。$y = 100$。100g。

(5) $A$% の食塩水 200g と 10% の食塩水 300g を混ぜたら 7% の食塩水ができた。$A$ の値を求めなさい。（4点）
→ $\dfrac{A}{100} \times 200 + 0.10 \times 300 = 0.07 \times 500$。$2A + 30 = 35$。$2A = 5$。$A = 2.5$。$A = 2.5$%。
