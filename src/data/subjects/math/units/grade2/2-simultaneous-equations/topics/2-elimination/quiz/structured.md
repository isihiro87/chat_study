# 加減法 大問小問

## 大問1: 基本の加減法（係数がそのまま消える）（配点 20点）

次の連立方程式を加減法で解きなさい。

(1) $\begin{cases} x + y = 8 \\ x - y = 2 \end{cases}$（3点）
→ $x = 5, y = 3$。2式を足すと $2x = 10$、$x = 5$。①に代入し $y = 3$。

(2) $\begin{cases} 3x + y = 10 \\ x + y = 4 \end{cases}$（3点）
→ $x = 3, y = 1$。①−②で $2x = 6$、$x = 3$。②に代入し $y = 1$。

(3) $\begin{cases} 5x + 3y = 4 \\ x - 3y = -10 \end{cases}$（4点）
→ $x = -1, y = 3$。2式を足すと $6x = -6$、$x = -1$。①に代入し $-5 + 3y = 4$、$y = 3$。

(4) $\begin{cases} 4x + y = 11 \\ 4x - y = 5 \end{cases}$（4点）
→ $x = 2, y = 3$。2式を足すと $8x = 16$、$x = 2$。①に代入し $8 + y = 11$、$y = 3$。

(5) $\begin{cases} 7x + 2y = 20 \\ 3x + 2y = 12 \end{cases}$（3点）
→ $x = 2, y = 3$。①−②で $4x = 8$、$x = 2$。②に代入し $6 + 2y = 12$、$y = 3$。

(6) $\begin{cases} x + 2y = 9 \\ x - 2y = 1 \end{cases}$（3点）
→ $x = 5, y = 2$。2式を足すと $2x = 10$、$x = 5$。①に代入し $5 + 2y = 9$、$y = 2$。

## 大問2: 片方の式を定数倍してから加減（配点 20点）

次の連立方程式を加減法で解きなさい。

(1) $\begin{cases} 2x + 3y = 16 \\ x + y = 6 \end{cases}$（3点）
→ $x = 2, y = 4$。②×2で $2x + 2y = 12$。①−②'で $y = 4$。②に代入し $x = 2$。

(2) $\begin{cases} 3x + y = 11 \\ x + 2y = 7 \end{cases}$（4点）
→ $x = 3, y = 2$。②×3で $3x + 6y = 21$。②'−①で $5y = 10$、$y = 2$。①に代入し $3x = 9$、$x = 3$。

(3) $\begin{cases} x + 3y = 11 \\ 2x + y = 7 \end{cases}$（4点）
→ $x = 2, y = 3$。①×2で $2x + 6y = 22$。①'−②で $5y = 15$、$y = 3$。②に代入し $2x = 4$、$x = 2$。

(4) $\begin{cases} 4x + y = 14 \\ x + 3y = 9 \end{cases}$（4点）
→ $x = 3, y = 2$。②×4で $4x + 12y = 36$。②'−①で $11y = 22$、$y = 2$。②に代入し $x + 6 = 9$、$x = 3$。

(5) $\begin{cases} x + 2y = 11 \\ 3x - y = 5 \end{cases}$（5点）
→ $x = 3, y = 4$。②×2で $6x - 2y = 10$。①+②'で $7x = 21$、$x = 3$。①に代入し $3 + 2y = 11$、$y = 4$。

## 大問3: 両方の式を定数倍してから加減（配点 20点）

次の連立方程式を加減法で解きなさい。

(1) $\begin{cases} 2x + 3y = 12 \\ 3x + 2y = 13 \end{cases}$（4点）
→ $x = 3, y = 2$。①×3で $6x + 9y = 36$、②×2で $6x + 4y = 26$。引くと $5y = 10$、$y = 2$。②に代入し $3x = 9$、$x = 3$。

(2) $\begin{cases} 3x + 4y = 18 \\ 2x + 3y = 13 \end{cases}$（4点）
→ $x = 2, y = 3$。①×3で $9x + 12y = 54$、②×4で $8x + 12y = 52$。引くと $x = 2$。②に代入し $4 + 3y = 13$、$y = 3$。

(3) $\begin{cases} 5x + 3y = 29 \\ 3x + 2y = 18 \end{cases}$（4点）
→ $x = 4, y = 3$。①×2で $10x + 6y = 58$、②×3で $9x + 6y = 54$。引くと $x = 4$。②に代入し $12 + 2y = 18$、$y = 3$。

(4) $\begin{cases} 3x + 2y = 8 \\ 2x + 5y = 9 \end{cases}$（4点）
→ $x = 2, y = 1$。①×5で $15x + 10y = 40$、②×2で $4x + 10y = 18$。引くと $11x = 22$、$x = 2$。②に代入し $4 + 5y = 9$、$y = 1$。

(5) $\begin{cases} 2x + 3y = 13 \\ 5x + 4y = 22 \end{cases}$（4点）
→ $x = 2, y = 3$。①×4で $8x + 12y = 52$、②×3で $15x + 12y = 66$。引くと $7x = 14$、$x = 2$。①に代入し $4 + 3y = 13$、$y = 3$。

## 大問4: 解の確認（代入して検算）（配点 20点）

次の連立方程式を加減法で解き、求めた解を両方の式に代入して正しいことを確かめなさい。

(1) $\begin{cases} x + y = 5 \\ x - y = 1 \end{cases}$（4点）
→ 2式を足すと $2x = 6$、$x = 3$。$y = 2$。検算: ①$3 + 2 = 5$ ○、②$3 - 2 = 1$ ○。

(2) $\begin{cases} 2x + y = 7 \\ x + 3y = 11 \end{cases}$（5点）
→ ①×3で $6x + 3y = 21$。①'−②で $5x = 10$、$x = 2$、$y = 3$。検算: ①$4 + 3 = 7$ ○、②$2 + 9 = 11$ ○。

(3) $\begin{cases} 3x + 2y = 19 \\ x + y = 8 \end{cases}$（5点）
→ ②×2で $2x + 2y = 16$。①−②'で $x = 3$。②より $y = 5$。検算: ①$9 + 10 = 19$ ○、②$3 + 5 = 8$ ○。

(4) $\begin{cases} 4x - y = 7 \\ 2x + 3y = 21 \end{cases}$（6点）
→ ①×3で $12x - 3y = 21$。①'+②で $14x = 42$、$x = 3$、$y = 5$。検算: ①$12 - 5 = 7$ ○、②$6 + 15 = 21$ ○。

## 大問5: 混合問題（配点 20点）

次の連立方程式を加減法で解きなさい。最適な方法を自分で考えよう。

(1) $\begin{cases} -x + y = 1 \\ x + y = 5 \end{cases}$（3点）
→ $x = 2, y = 3$。2式を足すと $2y = 6$、$y = 3$。②に代入し $x = 2$。

(2) $\begin{cases} x + y = 10 \\ 3x - y = 6 \end{cases}$（3点）
→ $x = 4, y = 6$。2式を足すと $4x = 16$、$x = 4$。①に代入し $y = 6$。

(3) $\begin{cases} 2x + y = 8 \\ x - 3y = -3 \end{cases}$（4点）
→ $x = 3, y = 2$。②×2で $2x - 6y = -6$。①−②'で $7y = 14$、$y = 2$。①に代入し $2x = 6$、$x = 3$。

(4) $\begin{cases} 3x + 5y = 21 \\ 2x + 3y = 13 \end{cases}$（5点）
→ $x = 2, y = 3$。①×2: $6x + 10y = 42$、②×3: $6x + 9y = 39$。引くと $y = 3$。②に代入し $2x + 9 = 13$、$x = 2$。

(5) $\begin{cases} 4x + 3y = 25 \\ 4x - y = 13 \end{cases}$（5点）
→ $x = 4, y = 3$。①−②で $4y = 12$、$y = 3$。②に代入し $4x = 16$、$x = 4$。
