# 一問一答：比例・反比例の利用

## 比例の利用（重さと個数）

Q1. クリップ1個の重さが0.8gのとき、x個の重さをygとすると、yをxの式で表すと？
A1. y = 0.8x

Q2. クリップ50個の重さが40gだった。クリップ1個の重さは？
A2. 40÷50 = 0.8g

Q3. クリップ1個0.8gのとき、全体の重さが120gなら何個ある？
A3. 120÷0.8 = 150個

Q4. 針金1mの重さが15gのとき、xmの重さygを式で表すと？
A4. y = 15x

Q5. 針金の重さが75gだった。長さは何m？（1mあたり15g）
A5. 75÷15 = 5m

Q6. バネに10gのおもりをつるすと2cmのびる。xgつるしたときのびycmを式で表すと？
A6. y = 0.2x（比例定数 = 2÷10 = 0.2）

Q7. バネののびが5cmのとき、つるしたおもりは何g？（10gで2cmのびるバネ）
A7. 5÷0.2 = 25g

Q8. 厚さが一定の鉄板で、面積10cm²の重さが78gだった。面積xcm²の重さygの式は？
A8. y = 7.8x

## 比例の利用（速さと時間）

Q9. 分速80mで歩く人が、x分後にym進む式は？
A9. y = 80x

Q10. 分速80mの人が600m先の駅に着くのは何分後？
A10. 600÷80 = 7.5分後

Q11. 分速60mのAさんと分速80mのBさんが同時に同じ場所を出発した。x分後の2人の距離の差ymを式で表すと？
A11. y = 20x（80x − 60x = 20x）

Q12. 分速60mのAさんが先に出発し、5分後にBさんが分速100mで追いかけた。BさんがAさんに追いつくのは何分後？
A12. 60×5 = 300mの差。1分で40mずつ縮まるから 300÷40 = 7.5分後

Q13. 家から1200m先の駅へ、父は分速80m、子は分速60mで同時に出発した。父が駅に着くのは何分後？
A13. 1200÷80 = 15分後

Q14. 同じ条件で子が駅に着くのは何分後？
A14. 1200÷60 = 20分後

Q15. 父と子が同時に出発して、2人の距離が200m離れるのは何分後？（父：分速80m、子：分速60m）
A15. 200÷(80−60) = 10分後

## グラフの読み取り

Q16. 比例のグラフが点(3, 9)を通るとき、比例の式は？
A16. a = 9÷3 = 3 より y = 3x

Q17. 反比例のグラフが点(4, 6)を通るとき、反比例の式は？
A17. a = 4×6 = 24 より y = 24/x

Q18. y = 2xのグラフ上で、y = 10となるxの値は？
A18. x = 10÷2 = 5

Q19. y = 12/xのグラフ上で、y = 3となるxの値は？
A19. x = 12÷3 = 4

Q20. y = 50xとy = 80xの2つのグラフの交点の座標は？
A20. 原点(0, 0)（比例のグラフは必ず原点を通る）

Q21. 2つの比例のグラフy = 50xとy = 80xで、x = 5のときのyの値の差は？
A21. 80×5 − 50×5 = 400 − 250 = 150

Q22. グラフ上の2点(0, 0)と(5, 300)を結ぶ直線が表す比例の式は？
A22. a = 300÷5 = 60 より y = 60x

Q23. 反比例のグラフy = 24/xで、x = 8のときのyの値は？
A23. y = 24÷8 = 3

## 反比例の利用（てこ・てんびん）

Q24. てんびんで、左に30gのおもりを支点から8cmの位置に置いた。右側のxcmの位置にygのおもりを置いてつり合わせるとき、yをxの式で表すと？
A24. y = 240/x（30×8 = 240 = xy）

Q25. 上の問題で、支点から12cmの位置につり合うおもりの重さは？
A25. 240÷12 = 20g

Q26. 支点から6cmの位置に40gのおもりがある。反対側の10cmの位置に何gのおもりを置けばつり合う？
A26. 6×40÷10 = 24g

Q27. てんびんで左20g×15cm = 右xg×ycm。右側のおもりが60gのとき、支点からの距離は？
A27. 20×15÷60 = 5cm

## 反比例の利用（歯車）

Q28. 歯数36の歯車Aと歯数12の歯車Bがかみ合っている。Aが5回転するとBは何回転？
A28. 36×5÷12 = 15回転

Q29. 歯数24の歯車Aがx回転するとき、歯数yの歯車Bが8回転する。yをxの式で表すと？
A29. 24x = 8y より y = 3x（これは比例）。※歯車の関係: 歯数A×回転数A = 歯数B×回転数B

Q30. 歯数40の歯車とかみ合う歯車の歯数をx、回転数をyとすると、歯車が10回転したとき、yをxの式で表すと？
A30. 40×10 = xy より y = 400/x

Q31. 歯数20の歯車が毎分15回転している。かみ合う歯数30の歯車の毎分の回転数は？
A31. 20×15÷30 = 10回転

## 反比例の利用（面積一定）

Q32. 面積が36cm²の長方形の縦xcmと横ycmの関係を式で表すと？
A32. y = 36/x

Q33. 面積36cm²の長方形で縦が9cmのとき、横は何cm？
A33. 36÷9 = 4cm

Q34. 底辺xcm、高さycmの三角形の面積が常に20cm²であるとき、yをxの式で表すと？
A34. xy/2 = 20 より xy = 40 より y = 40/x

## 比例・反比例の判定

Q35. 1個80円のりんごをx個買ったときの代金y円。比例？反比例？
A35. 比例（y = 80x、y/x = 80で一定）

Q36. 面積が24cm²の長方形で、縦xcm、横ycm。比例？反比例？
A36. 反比例（xy = 24で一定）

Q37. 1000円でx円の品物を買ったときのおつりy円。比例？反比例？
A37. どちらでもない（y = 1000 − xは一次関数）

Q38. 時速xkmでy時間走ったとき120km進む。比例？反比例？
A38. 反比例（xy = 120で一定）

Q39. 毎分xLずつ水を入れるとy分で水槽がいっぱいになる。比例？反比例？
A39. 反比例（xy = 一定）

Q40. 表：x = 1,2,3,4 / y = 5,10,15,20。比例？反比例？
A40. 比例（y/x = 5で一定、y = 5x）

Q41. 表：x = 2,3,6,9 / y = 18,12,6,4。比例？反比例？
A41. 反比例（xy = 36で一定、y = 36/x）

## 日常問題への応用

Q42. 水を毎分4Lずつ入れる水槽がある。x分後にyLたまるとき、yをxの式で表すと？
A42. y = 4x

Q43. 上の水槽で50Lためるのに何分かかる？
A43. 50÷4 = 12.5分

Q44. 1Lの重さが0.9kgの液体がある。xLの重さykgの式は？
A44. y = 0.9x

Q45. 200ページの本をx分でy ページ読む。1日30分読むと10日で何ページ読める？（毎分2ページずつ読む場合）
A45. 2×30×10 = 600ページ（200ページを超えるので、読み終えている）
