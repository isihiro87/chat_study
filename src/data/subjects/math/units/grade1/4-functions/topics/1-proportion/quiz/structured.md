# 大問形式：比例

## 大問1 関数の判定と変域（各3点×6＝18点）

次のそれぞれについて、yはxの関数であるかどうか答えなさい。

(1) 正方形の1辺の長さを x cm、周の長さを y cm とする。
(2) 長方形の面積が 20 cm² で、縦の長さを x cm、横の長さを y cm とする。
(3) xの値を決めても、yの値が2つ以上考えられる関係がある。
(4) 底辺が x cm、高さが 6 cm の三角形の面積を y cm² とする。

次の変域を不等号を使って表しなさい。

(5) xは −4 以上 3 以下
(6) xは 0 より大きく 10 未満

### 解答
(1) 関数である（y = 4x で、xを決めるとyがただ1つ決まる）
(2) 関数である（y = 20/x で、xを決めるとyがただ1つ決まる）
(3) 関数ではない（xに対してyがただ1つに決まらない）
(4) 関数である（y = (1/2) × x × 6 = 3x で、xを決めるとyがただ1つ決まる）
(5) −4 ≤ x ≤ 3
(6) 0 < x < 10

## 大問2 比例の式を求める（各4点×6＝24点）

(1) yはxに比例し、x = 3 のとき y = −9 である。yをxの式で表しなさい。

(2) yはxに比例し、x = −4 のとき y = 8 である。yをxの式で表しなさい。

(3) yはxに比例し、x = 6 のとき y = 2 である。yをxの式で表しなさい。

(4) (3)の式で、x = −12 のときの y の値を求めなさい。

(5) yはxに比例し、x = −2 のとき y = −10 である。yをxの式で表しなさい。

(6) (5)の式で、y = 35 のときの x の値を求めなさい。

### 解答
(1) a = −9/3 = −3 より、y = −3x
(2) a = 8/(−4) = −2 より、y = −2x
(3) a = 2/6 = 1/3 より、y = (1/3)x
(4) y = (1/3) × (−12) = −4
(5) a = −10/(−2) = 5 より、y = 5x
(6) 35 = 5x より x = 7

## 大問3 座標の読み取りと記入（各3点×6＝18点）

次の各点の座標を答えなさい。

(1) 原点から右に5、上に3の位置にある点A
(2) 原点から左に2、上に4の位置にある点B
(3) 原点から左に3、下に1の位置にある点C
(4) 原点から右に4、下に6の位置にある点D

次の点を座標平面上にとりなさい。

(5) E(−1, 5)
(6) F(3, −4)

### 解答
(1) A(5, 3)
(2) B(−2, 4)
(3) C(−3, −1)
(4) D(4, −6)
(5) 原点から左に1、上に5の位置に点Eをとる
(6) 原点から右に3、下に4の位置に点Fをとる

## 大問4 比例のグラフをかく・読み取る（各4点×6＝24点）

(1) y = 2x について、x = −3, −2, −1, 0, 1, 2, 3 のときの y の値を求めなさい。

(2) (1)の結果をもとに、y = 2x のグラフをかきなさい。グラフはどんな直線になるか？

(3) y = −(1/2)x について、x = −4, −2, 0, 2, 4 のときの y の値を求めなさい。

(4) (3)のグラフはどんな直線になるか？理由もあわせて答えなさい。

(5) y = 3x と y = (1/2)x のグラフでは、どちらが急な直線になるか？理由もあわせて答えなさい。

(6) 比例のグラフが原点を通ることを、y = ax の式を使って説明しなさい。

### 解答
(1) x = −3 → y = −6、x = −2 → y = −4、x = −1 → y = −2、x = 0 → y = 0、x = 1 → y = 2、x = 2 → y = 4、x = 3 → y = 6
(2) 原点を通る右上がりの直線（a = 2 > 0 なので右上がり）
(3) x = −4 → y = 2、x = −2 → y = 1、x = 0 → y = 0、x = 2 → y = −1、x = 4 → y = −2
(4) 原点を通る右下がりの直線。a = −1/2 < 0 なので右下がりになる。
(5) y = 3x の方が急。|3| > |1/2| で、比例定数の絶対値が大きいほど急になる。
(6) y = ax に x = 0 を代入すると y = a × 0 = 0 となり、必ず点 (0, 0)（原点）を通る。

## 大問5 グラフから式を求める（各4点×4＝16点）

(1) 比例のグラフが点 (2, 6) を通る。この比例の式を求めなさい。

(2) 比例のグラフが点 (−3, 12) を通る。この比例の式を求めなさい。

(3) 比例のグラフが点 (4, −3) を通る。この比例の式を求めなさい。

(4) ある比例のグラフが点 (−5, −10) を通る。この比例の式を求め、x = 8 のときの y の値を求めなさい。

### 解答
(1) a = 6/2 = 3 より、y = 3x
(2) a = 12/(−3) = −4 より、y = −4x
(3) a = −3/4 より、y = −(3/4)x
(4) a = −10/(−5) = 2 より y = 2x。x = 8 のとき y = 2 × 8 = 16
