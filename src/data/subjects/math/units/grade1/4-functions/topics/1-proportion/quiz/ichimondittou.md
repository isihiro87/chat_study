# 一問一答：比例

## 関数の定義

Q1. ともなって変わる2つの量 x と y があって、xの値を決めると yの値がただ1つ決まるとき、yはxの何であるというか？
A1. 関数

Q2. 関数の定義で大切な条件は「xの値を決めるとyの値が（　）決まる」である。（　）に入る言葉は？
A2. ただ1つ

Q3. 関数において、いろいろな値をとる文字（xやy）を何というか？
A3. 変数

Q4. 変数のとりうる値の範囲を何というか？
A4. 変域

Q5. xの変域が3以上6以下であることを、不等号を使って表すと？
A5. 3 ≤ x ≤ 6

Q6. xの変域が −2 より大きく 5 未満であることを、不等号を使って表すと？
A6. −2 < x < 5

Q7. 1 ≤ x ≤ 8 を日本語で読むと？
A7. xは1以上8以下

Q8. −3 < x < 4 を日本語で読むと？
A8. xは −3 より大きく 4 未満（4 より小さい）

## 比例の式

Q9. yがxに比例するとき、yをxの式で表すとどうなるか？
A9. y = ax（aは0でない定数）

Q10. y = ax の a のことを何というか？
A10. 比例定数

Q11. 比例 y = ax で、xが2倍、3倍になるとyはどうなるか？
A11. yも2倍、3倍になる

Q12. 比例 y = ax で、y ÷ x の値はいつも何になるか？
A12. 比例定数 a（一定の値）

Q13. 比例定数 a を求める式は？
A13. a = y/x

Q14. y = 5x の比例定数はいくつか？
A14. 5

Q15. y = −3x の比例定数はいくつか？
A15. −3

Q16. y = (2/3)x の比例定数はいくつか？
A16. 2/3

## 比例の判定と式を求める

Q17. y = 4x + 1 は比例か？
A17. 比例ではない（y = ax の形ではなく +1 がある）

Q18. y = −7x は比例か？
A18. 比例である（y = ax の形で、a = −7）

Q19. xが2のとき y = 10 となる比例の式を求めよ。
A19. a = 10/2 = 5 より、y = 5x

Q20. xが −3 のとき y = 9 となる比例の式を求めよ。
A20. a = 9/(−3) = −3 より、y = −3x

Q21. xが4のとき y = −2 となる比例の式を求めよ。
A21. a = −2/4 = −1/2 より、y = −(1/2)x

Q22. xが −5 のとき y = −15 となる比例の式を求めよ。
A22. a = −15/(−5) = 3 より、y = 3x

## 比例の計算

Q23. y = 4x で、x = 3 のとき y はいくつか？
A23. y = 4 × 3 = 12

Q24. y = −2x で、x = 6 のとき y はいくつか？
A24. y = −2 × 6 = −12

Q25. y = (1/2)x で、x = 8 のとき y はいくつか？
A25. y = (1/2) × 8 = 4

Q26. y = −3x で、x = −4 のとき y はいくつか？
A26. y = −3 × (−4) = 12

Q27. y = 6x で、y = 30 のとき x はいくつか？
A27. 30 = 6x より x = 5

Q28. y = −4x で、y = 20 のとき x はいくつか？
A28. 20 = −4x より x = −5

## 座標平面

Q29. 座標平面で横の軸を何というか？
A29. x軸

Q30. 座標平面で縦の軸を何というか？
A30. y軸

Q31. x軸とy軸が交わる点を何というか？
A31. 原点（O）

Q32. 原点の座標は？
A32. (0, 0)

Q33. 点の位置を (x, y) の形で表したものを何というか？
A33. 座標

Q34. x軸上にある点のy座標はいくつか？
A34. 0

Q35. y軸上にある点のx座標はいくつか？
A35. 0

Q36. 点 (3, −2) の x座標はいくつか？
A36. 3

Q37. 点 (3, −2) の y座標はいくつか？
A37. −2

Q38. 右に4、上に5の位置にある点の座標は？
A38. (4, 5)

Q39. 左に2、下に3の位置にある点の座標は？
A39. (−2, −3)

## 比例のグラフ

Q40. 比例のグラフはどんな図形になるか？
A40. 原点を通る直線

Q41. 比例のグラフが必ず通る点は？
A41. 原点 (0, 0)

Q42. y = ax で a > 0 のとき、グラフは右上がりか右下がりか？
A42. 右上がり

Q43. y = ax で a < 0 のとき、グラフは右上がりか右下がりか？
A43. 右下がり

Q44. y = 2x のグラフと y = 5x のグラフでは、どちらが急な直線か？
A44. y = 5x（|a| が大きいほど急）

Q45. 比例のグラフが原点と点 (1, −4) を通るとき、比例の式は？
A45. a = −4/1 = −4 より、y = −4x

Q46. 比例のグラフが原点と点 (3, 6) を通るとき、比例の式は？
A46. a = 6/3 = 2 より、y = 2x

Q47. 比例のグラフが原点と点 (−2, 8) を通るとき、比例の式は？
A47. a = 8/(−2) = −4 より、y = −4x

Q48. 比例のグラフをかくには、原点ともう何点をとればよいか？
A48. もう1点（合計2点を結んで直線をひく）
