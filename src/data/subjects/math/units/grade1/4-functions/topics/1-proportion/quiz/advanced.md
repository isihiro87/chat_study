# 発展問題：比例

## 発展1 比例の性質と変域の応用（各5点×4＝20点）

(1) y = 3x で、xの変域が −2 ≤ x ≤ 4 のとき、yの変域を求めなさい。

(2) y = −2x で、xの変域が 1 ≤ x ≤ 5 のとき、yの変域を求めなさい。

(3) y = ax（a > 0）で、xの変域が −3 ≤ x ≤ 6 のとき、yの変域が −12 ≤ y ≤ 24 であった。比例定数 a の値を求めなさい。

(4) y = ax で、xの変域が −4 ≤ x ≤ 2 のとき、yの変域が −6 ≤ y ≤ 12 であった。比例定数 a の値を求めなさい。

### 解答
(1) x = −2 のとき y = −6、x = 4 のとき y = 12 より、−6 ≤ y ≤ 12
(2) x = 1 のとき y = −2、x = 5 のとき y = −10 より、−10 ≤ y ≤ −2
(3) x = 6 のとき y = 24 より a = 24/6 = 4。確認: x = −3 のとき y = 4 × (−3) = −12 で一致。よって a = 4
(4) a > 0 なら x = 2 のとき y = 2a、x = −4 のとき y = −4a。y = 12 = −4a とすると a = −3 で矛盾。a < 0 なら x = −4 のとき y = −4a = 12 より a = −3。確認: x = 2 のとき y = −3 × 2 = −6 で一致。よって a = −3

## 発展2 グラフの読み取りと条件（各5点×4＝20点）

(1) 2つの比例 y = 2x と y = −x について、x = 3 のときのそれぞれの y の値を求め、2つの y の値の差を求めなさい。

(2) 比例 y = ax のグラフが点 (−2, 5) を通るとき、このグラフは点 (6, b) も通る。b の値を求めなさい。

(3) 比例 y = ax のグラフが第2象限（x < 0, y > 0 の部分）と第4象限（x > 0, y < 0 の部分）を通る。比例定数 a について何がいえるか？

(4) 比例のグラフが2点 A(2, 8) と B(−1, c) を通る。c の値を求めなさい。

### 解答
(1) y = 2 × 3 = 6、y = −1 × 3 = −3。差は 6 − (−3) = 9
(2) a = 5/(−2) = −5/2 より y = −(5/2)x。x = 6 のとき b = −(5/2) × 6 = −15
(3) a < 0（右下がりのグラフなので、比例定数は負）
(4) 比例の式は y = ax に A(2, 8) を代入して a = 8/2 = 4。y = 4x に x = −1 を代入して c = 4 × (−1) = −4

## 発展3 日常生活での比例の応用（各5点×3＝15点）

(1) 1本の重さが 15g のくぎがある。くぎ x 本の重さを yg とするとき、yをxの式で表しなさい。また、くぎの重さが 360g のとき、くぎは何本あるか求めなさい。

(2) 水そうに毎分 3L ずつ水を入れる。x 分後に入った水の量を yL とするとき、yをxの式で表しなさい。また、水が 45L たまるのは何分後か求めなさい。

(3) ある印刷機は 1分間に 50 枚のプリントを印刷できる。x 分間に印刷できる枚数を y 枚とする。yをxの式で表し、300 枚印刷するのにかかる時間を求めなさい。

### 解答
(1) y = 15x。360 = 15x より x = 24（本）
(2) y = 3x。45 = 3x より x = 15（分後）
(3) y = 50x。300 = 50x より x = 6（分）

## 発展4 比例の総合問題（各5点×3＝15点）

(1) y は x に比例し、x = 2 のとき y = −8 である。x = −3 のとき y の値を求めなさい。また、y = 20 のときの x の値を求めなさい。

(2) 比例 y = ax のグラフと y = bx のグラフがある。a = 3, b = −1 のとき、x = 2 における2つのグラフ上の点の間の距離（y座標の差）を求めなさい。

(3) y = ax のグラフが点 (4, −6) を通る。このグラフ上で x座標が −8 の点の y座標を求めなさい。また、y座標が 9 の点の x座標を求めなさい。

### 解答
(1) a = −8/2 = −4 より y = −4x。x = −3 のとき y = −4 × (−3) = 12。y = 20 のとき 20 = −4x より x = −5
(2) x = 2 のとき、y = 3 × 2 = 6 と y = −1 × 2 = −2。距離は |6 − (−2)| = 8
(3) a = −6/4 = −3/2 より y = −(3/2)x。x = −8 のとき y = −(3/2) × (−8) = 12。y = 9 のとき 9 = −(3/2)x より x = −6

## 発展5 比例の性質を利用した問題（各5点×3＝15点）

(1) y = ax のグラフが y = 2x のグラフよりも急で、y = 6x のグラフよりもゆるやかである。a の値として考えられる正の整数をすべて答えなさい。

(2) ばねに重りをつるすと、ばねの伸びは重りの重さに比例する。20g の重りをつるすと 4cm 伸びるばねがある。このばねを 7cm 伸ばすには何 g の重りが必要か。

(3) ある自動車は 1L のガソリンで 12km 走ることができる。x L のガソリンで走れる距離を ykm とする。yをxの式で表し、180km の距離を走るのに必要なガソリンの量を求めなさい。

### 解答
(1) |a| > 2 かつ |a| < 6 で、a は正の整数なので a = 3, 4, 5
(2) 伸びを ycm、重さを xg とすると y = ax。x = 20, y = 4 より a = 4/20 = 1/5。y = 7 のとき 7 = (1/5)x より x = 35（g）
(3) y = 12x。180 = 12x より x = 15（L）
