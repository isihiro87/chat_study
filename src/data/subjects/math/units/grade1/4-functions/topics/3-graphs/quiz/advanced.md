# 発展問題：比例・反比例の利用

## 発展1 点の移動と面積の変化（各5点×4＝20点）

1辺6cmの正方形ABCDで、点Pが頂点Aを出発して辺AB上をBに向かって毎秒1cmで動く。出発からx秒後の△APDの面積をycm²とする。

(1) yをxの式で表しなさい。
(2) yはxに比例するか。理由も答えなさい。
(3) 面積が9cm²になるのは出発から何秒後か。
(4) 点PがAからBまで動く間のxの変域を求め、yの変域も求めなさい。

### 解答
(1) △APDの底辺AP = xcm、高さAD = 6cmなので y = x×6÷2 = 3x
(2) 比例する。y = 3xでy/x = 3が一定だから。
(3) 3x = 9 → x = 3秒後
(4) 0≦x≦6、0≦y≦18

## 発展2 直線と双曲線の交点（各5点×4＝20点）

比例y = 2xのグラフと反比例y = a/xのグラフが点P(3, b)で交わっている。

(1) bの値を求めなさい。
(2) aの値を求めなさい。
(3) 反比例のグラフ上で、x = 2のときのyの値を求めなさい。
(4) 反比例のグラフ上の点で、x座標とy座標がともに整数である点をすべて求めなさい（x > 0の範囲）。

### 解答
(1) y = 2xにx = 3を代入: b = 2×3 = 6
(2) 反比例のグラフが点(3, 6)を通るから a = 3×6 = 18
(3) y = 18/2 = 9
(4) y = 18/xで、18の正の約数は1,2,3,6,9,18。よって(1,18),(2,9),(3,6),(6,3),(9,2),(18,1)の6個。

## 発展3 歯車の組み合わせ（各5点×3＝15点）

歯車A（歯数48）、歯車B（歯数x）、歯車C（歯数16）がA→B→Cの順にかみ合っている。歯車Aが毎分2回転している。

(1) 歯車Bの歯数がx = 24のとき、歯車Bの毎分の回転数を求めなさい。
(2) (1)のとき、歯車Cの毎分の回転数を求めなさい。
(3) 歯車Cの毎分の回転数が12回転になるとき、歯車Bの歯数xを求めなさい。

### 解答
(1) 48×2 = 24×y → y = 4回転
(2) 24×4 = 16×y → y = 6回転
(3) A→B: 48×2 = x×(Bの回転数) → Bの回転数 = 96/x。B→C: x×(96/x) = 16×12 → 96 = 192。これは矛盾。
正しい考え方: A→Cの伝達では 歯数A×回転数A = 歯数C×回転数C が成り立つ（中間歯車の歯数は関係ない）。48×2 = 16×回転数C → 回転数C = 6。よって歯車Cの回転数は常に6回転であり、12回転にすることはBの歯数では調整できない。

## 発展4 グラフの交点から条件を求める（各5点×4＝20点）

Aさんは家から900m離れた図書館に向かって分速amで歩き始めた。Aさんが出発してから5分後にBさんが同じ家を出発し、分速bm(b > a)で同じ図書館に向かった。BさんはAさんが図書館に着いた時刻と同時に図書館に着いた。

(1) Aさんが図書館に着くまでの時間が15分のとき、aの値を求めなさい。
(2) Bさんの歩いた時間は何分間か。
(3) bの値を求めなさい。
(4) BさんがAさんに追いつくのは、Aさんが出発してから何分後か。

### 解答
(1) a = 900÷15 = 60（分速60m）
(2) Aさんが15分で着くので、5分遅れのBさんは15−5 = 10分間
(3) b = 900÷10 = 90（分速90m）
(4) Aさん出発からx分後とすると、60x = 90(x−5) → 60x = 90x−450 → 30x = 450 → x = 15分後。ただしこれはAさんが図書館に着く時刻と同じ。つまり追いつく地点は図書館（900m地点）である。

## 発展5 平行四辺形の面積と反比例（各5点×5＝25点）

底辺の長さがxcm、高さがycmの平行四辺形の面積がつねに48cm²である。

(1) yをxの式で表しなさい。
(2) x = 8のときのyの値を求めなさい。
(3) x、yがともに正の整数となる組をすべて求めなさい。
(4) この平行四辺形で、底辺の長さを3倍にしたとき、面積を48cm²に保つには高さをどうすればよいか。
(5) 底辺の長さが1cm増えると高さはどう変わるか。x = 6のときとx = 7のときで比べなさい。

### 解答
(1) xy = 48 より y = 48/x
(2) y = 48/8 = 6cm
(3) 48の正の約数の組: (1,48),(2,24),(3,16),(4,12),(6,8),(8,6),(12,4),(16,3),(24,2),(48,1) の10組
(4) 高さを1/3倍にする（底辺×高さ = 一定なので、底辺が3倍なら高さは1/3倍）
(5) x = 6のとき y = 48/6 = 8、x = 7のとき y = 48/7 ≒ 6.86。高さは8−48/7 = 8/7 ≒ 1.14cm減る。反比例では一定の割合で減るのではなく、xが大きくなるほど変化量が小さくなる。
