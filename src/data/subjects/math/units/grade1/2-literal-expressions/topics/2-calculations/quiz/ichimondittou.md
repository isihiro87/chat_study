# 一問一答：文字式の計算

## 項と係数

Q1. $3x - 5y + 7$ の項をすべて答えなさい。
A1. $3x$, $-5y$, $7$

Q2. $3x$ の係数は？
A2. $3$

Q3. $-5y$ の係数は？
A3. $-5$

Q4. $3x - 5y + 7$ の定数項は？
A4. $7$

Q5. $-a + 4b - 9$ の項をすべて答えなさい。
A5. $-a$, $4b$, $-9$

Q6. $-a$ の係数は？
A6. $-1$

Q7. $x$ の係数は？
A7. $1$

Q8. 数だけの項を何と呼ぶか？
A8. 定数項

Q9. $2x - 8$ の項をすべて答えなさい。
A9. $2x$, $-8$

Q10. $-7a + 3b + 1$ で $b$ の係数は？
A10. $3$

## 一次式の識別

Q11. 一次式とはどんな式か？
A11. 文字の次数が1の項だけを含む式

Q12. $3x + 2$ は一次式か？
A12. 一次式である

Q13. $x^2 + 3x$ は一次式か？
A13. 一次式ではない（$x^2$ は2次の項）

Q14. $5$ は一次式か？
A14. 一次式ではない（文字を含まない定数）

Q15. 1次の項とは何か？
A15. 文字が1つだけかけられている項（例: $3x$, $-2a$）

## 同類項のまとめ

Q16. $5x + 3x$ を計算しなさい。
A16. $8x$

Q17. $9a - 4a$ を計算しなさい。
A17. $5a$

Q18. $3x + 7 - x + 2$ を計算しなさい。
A18. $2x + 9$

Q19. $6a - 3 + 2a - 5$ を計算しなさい。
A19. $8a - 8$

Q20. $-2x + 5 + 8x - 1$ を計算しなさい。
A20. $6x + 4$

Q21. $4y - 9 - 7y + 3$ を計算しなさい。
A21. $-3y - 6$

Q22. $10a - 6a + 2 - 7$ を計算しなさい。
A22. $4a - 5$

## かっこのはずし方

Q23. $+(4x - 3)$ のかっこを外しなさい。
A23. $4x - 3$（そのまま）

Q24. $-(4x - 3)$ のかっこを外しなさい。
A24. $-4x + 3$（符号がすべて変わる）

Q25. $+(2a + 7)$ のかっこを外しなさい。
A25. $2a + 7$

Q26. $-(2a + 7)$ のかっこを外しなさい。
A26. $-2a - 7$

Q27. $-(- 5x + 1)$ のかっこを外しなさい。
A27. $5x - 1$

## 式の加減

Q28. $(3x + 2) + (5x - 4)$ を計算しなさい。
A28. $8x - 2$

Q29. $(7a - 1) - (3a + 5)$ を計算しなさい。
A29. $4a - 6$

Q30. $(4x + 9) + (- x - 3)$ を計算しなさい。
A30. $3x + 6$

Q31. $(6a - 8) - (- 2a + 4)$ を計算しなさい。
A31. $8a - 12$

## 文字式と数の乗法・除法

Q32. $4x \times 6$ を計算しなさい。
A32. $24x$

Q33. $(-3a) \times 5$ を計算しなさい。
A33. $-15a$

Q34. $(-42x) \div (-6)$ を計算しなさい。
A34. $7x$

Q35. $18a \div 3$ を計算しなさい。
A35. $6a$

Q36. $(-8x) \div 4$ を計算しなさい。
A36. $-2x$

## 分配法則

Q37. $3(2x - 5)$ を計算しなさい。
A37. $6x - 15$

Q38. $-4(3a + 2)$ を計算しなさい。
A38. $-12a - 8$

Q39. $(4x - 7) \times (-3)$ を計算しなさい。
A39. $-12x + 21$

Q40. $(8x - 2) \times \dfrac{3}{5}$ を計算しなさい。
A40. $\dfrac{24x - 6}{5}$

Q41. $(6a + 9) \div 3$ を計算しなさい。
A41. $2a + 3$

Q42. $(10x - 15) \div 5$ を計算しなさい。
A42. $2x - 3$

## 分数の式に数をかける

Q43. $\dfrac{3x + 1}{2} \times 4$ を計算しなさい。
A43. $2(3x + 1) = 6x + 2$

Q44. $\dfrac{5a - 3}{3} \times 6$ を計算しなさい。
A44. $2(5a - 3) = 10a - 6$

Q45. $\dfrac{2x - 4}{4} \times 8$ を計算しなさい。
A45. $2(2x - 4) = 4x - 8$

## 複合計算

Q46. $2(4x - 5) + 6(x + 8)$ を計算しなさい。
A46. $8x - 10 + 6x + 48 = 14x + 38$

Q47. $5(2a + 3) - 3(4a - 1)$ を計算しなさい。
A47. $10a + 15 - 12a + 3 = -2a + 18$

Q48. $4(x - 2) + 3(2x + 1)$ を計算しなさい。
A48. $4x - 8 + 6x + 3 = 10x - 5$

Q49. $3(5a - 4) - 2(3a + 7)$ を計算しなさい。
A49. $15a - 12 - 6a - 14 = 9a - 26$

Q50. $-2(x + 6) + 4(3x - 1)$ を計算しなさい。
A50. $-2x - 12 + 12x - 4 = 10x - 16$
