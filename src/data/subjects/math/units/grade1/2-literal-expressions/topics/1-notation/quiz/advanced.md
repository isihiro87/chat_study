# 文字式の表し方 発展問題

## 発展1: 複雑な文字式の表し方（×÷混合）

次の式を文字式の表し方のルールにしたがって書きなさい。

(1) $a \times b \div c \times d$
→ $\dfrac{abd}{c}$

(2) $x \div 2 \div y \times 3$
→ $\dfrac{3x}{2y}$

(3) $a \times (-2) \times a \div b \times b$
→ $-2a^2$（$\dfrac{-2a^2 b}{b} = -2a^2$、$b$ が約分される）

(4) $(a + b) \times 3 \div (a - b)$
→ $\dfrac{3(a + b)}{a - b}$

## 発展2: 数量を文字式で表す（速さ・割合・面積）

(1) A 地点から B 地点まで $a$ km ある。行きは時速 $x$ km、帰りは時速 $y$ km で往復した。往復にかかった時間の合計を文字式で表しなさい。
→ $\dfrac{a}{x} + \dfrac{a}{y}$ 時間（道のり÷速さ＝時間を行き・帰りそれぞれ求めて足す）

(2) $a$ 円の品物に消費税 10% を加えた税込価格を文字式で表しなさい。
→ $1.1a$ 円（$a + 0.1a = 1.1a$）

(3) 縦 $a$ cm、横 $b$ cm、高さ $h$ cm の直方体の体積と表面積をそれぞれ文字式で表しなさい。
→ 体積：$abh$ cm³　表面積：$2(ab + bh + ah)$ cm²

(4) 半径 $r$ cm の円の周の長さと面積を、円周率 $\pi$ を使って文字式で表しなさい。
→ 周の長さ：$2\pi r$ cm　面積：$\pi r^2$ cm²

## 発展3: 2変数の代入

(1) $3x - 2y + 1$ に $x = -2, y = 3$ を代入して式の値を求めよ。
→ $3 \times (-2) - 2 \times 3 + 1 = -6 - 6 + 1 = -11$

(2) $\dfrac{a + b}{a - b}$ に $a = 5, b = -3$ を代入して式の値を求めよ。
→ $\dfrac{5 + (-3)}{5 - (-3)} = \dfrac{2}{8} = \dfrac{1}{4}$

(3) $a^2 + 2ab + b^2$ に $a = 3, b = -1$ を代入して式の値を求めよ。
→ $3^2 + 2 \times 3 \times (-1) + (-1)^2 = 9 - 6 + 1 = 4$

(4) $ab - \dfrac{a}{b}$ に $a = 6, b = -\dfrac{2}{3}$ を代入して式の値を求めよ。
→ $6 \times \left(-\dfrac{2}{3}\right) - \dfrac{6}{-\dfrac{2}{3}} = -4 - (-9) = -4 + 9 = 5$

## 発展4: 式の意味を文章で説明する

次の式がどんな数量を表しているか、文章で説明しなさい。

(1) 1個 $a$ 円のりんごと1個 $b$ 円のみかんについて、$5a + 3b$ はどんな数量を表すか。
→ りんごを5個、みかんを3個買ったときの代金の合計（円）

(2) 分速 $a$ m で歩く人について、$\dfrac{1000}{a}$ はどんな数量を表すか。
→ 1000 m（1 km）を歩くのにかかる時間（分）

(3) $x$ 人のクラスで $a$ 点のテストについて、$\dfrac{a}{x}$ はどんな数量を表すか。ただし $a$ はクラス全員の合計点とする。
→ クラスのテストの平均点（点）

(4) ある品物の原価が $a$ 円のとき、$1.3a - b$ はどんな数量を表すか説明しなさい。
→ 原価 $a$ 円の品物に30%の利益を乗せた定価（$1.3a$ 円）から $b$ 円を値引きしたときの売値（円）

## 発展5: 総合問題

(1) $-a^2$ と $(-a)^2$ の違いを説明し、$a = 3$ を代入してそれぞれの式の値を求めなさい。
→ $-a^2$ は「$a^2$ にマイナスをつけたもの」で $-(3^2) = -9$。$(-a)^2$ は「$-a$ を2乗したもの」で $(-3)^2 = 9$。符号のつき方が異なり、結果が正負逆になる。

(2) $a = -\dfrac{3}{4}$ のとき、$8a^2 - 4a + 1$ の式の値を求めなさい。
→ $8 \times \left(-\dfrac{3}{4}\right)^2 - 4 \times \left(-\dfrac{3}{4}\right) + 1 = 8 \times \dfrac{9}{16} + 3 + 1 = \dfrac{9}{2} + 4 = \dfrac{9}{2} + \dfrac{8}{2} = \dfrac{17}{2}$

(3) ある長方形の縦を $a$ cm、横を $b$ cm とする。縦を 2 cm 長くし、横を 3 cm 短くした新しい長方形の面積を文字式で表しなさい。また、$a = 8, b = 10$ のとき、もとの長方形と新しい長方形の面積の差を求めなさい。
→ 新しい長方形の面積：$(a + 2)(b - 3)$ cm²。$a = 8, b = 10$ のとき、もとの面積 $= 8 \times 10 = 80$ cm²、新しい面積 $= (8 + 2)(10 - 3) = 10 \times 7 = 70$ cm²。差は $80 - 70 = 10$ cm²。

(4) 兄は家から $a$ km 離れた駅まで時速 $x$ km で歩き、弟は同じ道を時速 $y$ km で自転車で行く。2人が同時に出発したとき、弟が駅に着いた時点で兄は駅まであと何 km の地点にいるか、文字式で表しなさい（ただし $x < y$ とする）。
→ 弟が駅に着くまでの時間は $\dfrac{a}{y}$ 時間。その間に兄が歩いた道のりは $x \times \dfrac{a}{y} = \dfrac{ax}{y}$ km。よって兄と駅の残りの距離は $a - \dfrac{ax}{y} = \dfrac{a(y - x)}{y}$ km。
