# 一問一答：方程式の基本と移項

## 方程式の定義と解

Q1. まだわからない数（未知数）を含む等式を何というか？
A1. 方程式

Q2. 方程式を成り立たせる未知数の値を何というか？
A2. 方程式の解

Q3. 方程式の解を求めることを何というか？
A3. 方程式を解く

Q4. $x + 5 = 8$ は方程式か？
A4. 方程式である（未知数 $x$ を含む等式だから）

Q5. $3 + 5 = 8$ は方程式か？
A5. 方程式ではない（未知数を含まないから）

Q6. $x = 2$ が $x + 6 = 8$ の解であるかどうか確かめるには、どうすればよいか？
A6. $x = 2$ を方程式に代入して、等式が成り立つか確認する

Q7. $x + 6 = 8$ に $x = 2$ を代入すると、左辺はいくつになるか？
A7. $2 + 6 = 8$ で左辺は $8$。右辺 $= 8$ と等しいので解である

Q8. $x + 6 = 8$ に $x = 3$ を代入すると、等式は成り立つか？
A8. $3 + 6 = 9 \neq 8$ なので成り立たない。$x = 3$ は解ではない

## 等式の性質

Q9. 等式の性質を4つ答えよ。
A9. 両辺に同じ数を加えても、引いても、かけても、0以外の同じ数で割っても等式は成り立つ

Q10. $A = B$ のとき、$A + C =$ ？
A10. $B + C$（両辺に同じ数 $C$ を加えても等式は成り立つ）

Q11. $A = B$ のとき、$A - C =$ ？
A11. $B - C$（両辺から同じ数 $C$ を引いても等式は成り立つ）

Q12. $A = B$ のとき、$A \times C =$ ？
A12. $B \times C$（両辺に同じ数 $C$ をかけても等式は成り立つ）

Q13. $A = B$ のとき、$\dfrac{A}{C} =$ ？（ただし $C \neq 0$）
A13. $\dfrac{B}{C}$（両辺を0以外の同じ数 $C$ で割っても等式は成り立つ）

Q14. 等式の性質で、割り算の場合に注意すべきことは何か？
A14. 0で割ることはできない（$C \neq 0$）

Q15. $x + 4 = 9$ を等式の性質を使って解くには、両辺から何を引けばよいか？
A15. 両辺から $4$ を引く → $x = 5$

Q16. $x - 3 = 7$ を等式の性質を使って解くには、両辺に何を加えればよいか？
A16. 両辺に $3$ を加える → $x = 10$

Q17. $5x = 20$ を等式の性質を使って解くには、両辺を何で割ればよいか？
A17. 両辺を $5$ で割る → $x = 4$

Q18. $\dfrac{x}{4} = 3$ を等式の性質を使って解くには、両辺に何をかければよいか？
A18. 両辺に $4$ をかける → $x = 12$

## 移項

Q19. 等式の一方の辺の項を、符号を変えて他方の辺に移すことを何というか？
A19. 移項

Q20. 移項するとき、符号はどう変わるか？
A20. $+$ の項は $-$ に、$-$ の項は $+$ に変わる

Q21. $x + 5 = 9$ で $+5$ を右辺に移項すると？
A21. $x = 9 - 5 = 4$

Q22. $x - 7 = 2$ で $-7$ を右辺に移項すると？
A22. $x = 2 + 7 = 9$

Q23. 移項は等式のどの性質を利用しているか？
A23. 両辺に同じ数を加える（または引く）性質

## 基本的な方程式を解く（x+a=b型, ax=b型）

Q24. $x + 9 = 14$ の解は？
A24. $x = 5$（$+9$ を移項して $x = 14 - 9$）

Q25. $x - 4 = 11$ の解は？
A25. $x = 15$（$-4$ を移項して $x = 11 + 4$）

Q26. $x + 12 = 5$ の解は？
A26. $x = -7$（$+12$ を移項して $x = 5 - 12$）

Q27. $4x = 28$ の解は？
A27. $x = 7$（両辺を $4$ で割る）

Q28. $-6x = 18$ の解は？
A28. $x = -3$（両辺を $-6$ で割る）

Q29. $7x = -21$ の解は？
A29. $x = -3$（両辺を $7$ で割る）

## ax+b=c 型の方程式

Q30. $3x + 2 = 14$ の解は？
A30. $x = 4$（$+2$ を移項して $3x = 12$、両辺を $3$ で割る）

Q31. $5x - 8 = 17$ の解は？
A31. $x = 5$（$-8$ を移項して $5x = 25$、両辺を $5$ で割る）

Q32. $-2x + 9 = 3$ の解は？
A32. $x = 3$（$+9$ を移項して $-2x = -6$、両辺を $-2$ で割る）

Q33. $4x + 7 = -5$ の解は？
A33. $x = -3$（$+7$ を移項して $4x = -12$、両辺を $4$ で割る）

## 両辺にxがある方程式（ax+b=cx+d型）

Q34. $6x + 3 = 4x + 9$ の解は？
A34. $x = 3$（$4x$ と $+3$ を移項して $2x = 6$、$x = 3$）

Q35. $7x - 5 = 3x + 7$ の解は？
A35. $x = 3$（$3x$ と $-5$ を移項して $4x = 12$、$x = 3$）

Q36. $2x + 10 = 5x - 2$ の解は？
A36. $x = 4$（$5x$ と $+10$ を移項して $-3x = -12$、$x = 4$）

Q37. $8x - 1 = 5x + 11$ の解は？
A37. $x = 4$（$5x$ と $-1$ を移項して $3x = 12$、$x = 4$）

Q38. 両辺に $x$ がある方程式を解くとき、まず何をするか？
A38. $x$ の項を左辺に、数の項を右辺にそれぞれ移項してまとめる

## かっこがある方程式

Q39. $2(x + 3) = 14$ を解くとき、最初に何をするか？
A39. 分配法則でかっこを展開する → $2x + 6 = 14$

Q40. $2(x + 3) = 14$ の解は？
A40. $x = 4$（展開して $2x + 6 = 14$、$2x = 8$、$x = 4$）

Q41. $3(x - 4) = 9$ の解は？
A41. $x = 7$（展開して $3x - 12 = 9$、$3x = 21$、$x = 7$）

Q42. $5(x + 2) = 3x + 16$ の解は？
A42. $x = 3$（展開して $5x + 10 = 3x + 16$、$2x = 6$、$x = 3$）

Q43. $4(x - 1) = 2(x + 3)$ の解は？
A43. $x = 5$（展開して $4x - 4 = 2x + 6$、$2x = 10$、$x = 5$）

Q44. $-(x + 5) = 2x - 8$ の解は？
A44. $x = 1$（展開して $-x - 5 = 2x - 8$、$-3x = -3$、$x = 1$）

## 解の確かめ

Q45. 方程式 $3x + 1 = 10$ の解 $x = 3$ が正しいか確かめよ。
A45. 左辺 $= 3 \times 3 + 1 = 10$、右辺 $= 10$。左辺 $=$ 右辺なので正しい

Q46. 方程式 $4(x - 2) = 12$ の解 $x = 5$ が正しいか確かめよ。
A46. 左辺 $= 4(5 - 2) = 4 \times 3 = 12$、右辺 $= 12$。左辺 $=$ 右辺なので正しい
