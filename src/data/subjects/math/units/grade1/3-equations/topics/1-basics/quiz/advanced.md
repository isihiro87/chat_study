# 発展問題：方程式の基本と移項

## 発展1: 解から係数を求める（各5点×4＝20点）

(1) 方程式 $ax + 3 = 15$ の解が $x = 4$ のとき、$a$ の値を求めなさい。
答え: $a = 3$
解説: $x = 4$ を代入すると $4a + 3 = 15$、$4a = 12$、$a = 3$

(2) 方程式 $2x + a = 11$ の解が $x = 3$ のとき、$a$ の値を求めなさい。
答え: $a = 5$
解説: $x = 3$ を代入すると $6 + a = 11$、$a = 5$

(3) 方程式 $ax - 2 = x + 10$ の解が $x = 4$ のとき、$a$ の値を求めなさい。
答え: $a = 4$
解説: $x = 4$ を代入すると $4a - 2 = 14$、$4a = 16$、$a = 4$

(4) 方程式 $3(x + a) = 21$ の解が $x = 4$ のとき、$a$ の値を求めなさい。
答え: $a = 3$
解説: $x = 4$ を代入すると $3(4 + a) = 21$、$4 + a = 7$、$a = 3$

## 発展2: 複雑なかっこの方程式（各5点×4＝20点）

(1) $3(2x - 1) = 5(x + 2)$ を解きなさい。
答え: $x = 13$
解説: 展開して $6x - 3 = 5x + 10$、$x = 13$

(2) $2(3x + 4) = 4(x - 1) + 20$ を解きなさい。
答え: $x = 4$
解説: 展開して $6x + 8 = 4x - 4 + 20 = 4x + 16$、$2x = 8$、$x = 4$

(3) $-3(x - 5) = 2(x + 1) + 3$ を解きなさい。
答え: $x = 2$
解説: 展開して $-3x + 15 = 2x + 2 + 3 = 2x + 5$、$-5x = -10$、$x = 2$

(4) $4(x + 3) - 2(x - 1) = 18$ を解きなさい。
答え: $x = 2$
解説: 展開して $4x + 12 - 2x + 2 = 18$、$2x + 14 = 18$、$2x = 4$、$x = 2$

## 発展3: 方程式の性質を説明する問題（各5点×3＝15点）

(1) 等式の性質「$A = B$ ならば $A + C = B + C$」を使って、$x - 5 = 8$ の解き方を説明しなさい。
答え: 両辺に $5$ を加えると $x - 5 + 5 = 8 + 5$、つまり $x = 13$
解説: 等式の「両辺に同じ数を加えても等式は成り立つ」性質を使います。

(2) 「移項」と「等式の性質」の関係を説明しなさい。
答え: 移項は等式の性質（両辺に同じ数を加える・引く）を簡単にまとめた操作である。例えば $x + 3 = 7$ で両辺から $3$ を引くと $x = 7 - 3$ となるが、これは $+3$ を右辺に移項して符号を変えることと同じ。
解説: 移項は等式の性質の省略表現と理解できます。

(3) 方程式を解いた後に「解の確かめ」をする意味を説明しなさい。
答え: 求めた解を元の方程式に代入して等式が成り立つか確認することで、計算ミスがないかチェックできる。
解説: 特に移項の符号ミスや計算ミスを発見できるので、確かめは重要です。

## 発展4: 文章から方程式を立てる（各5点×4＝20点）

(1) ある数 $x$ に $5$ を加えて $3$ 倍すると $27$ になる。$x$ の値を求めなさい。
答え: $x = 4$
解説: $3(x + 5) = 27$、$x + 5 = 9$、$x = 4$

(2) ある数 $x$ の $4$ 倍から $6$ を引くと、$x$ の $2$ 倍に $8$ を加えたものと等しい。$x$ を求めなさい。
答え: $x = 7$
解説: $4x - 6 = 2x + 8$、$2x = 14$、$x = 7$

(3) ある数 $x$ の $3$ 倍に $1$ を加えると $x$ の $5$ 倍から $9$ を引いたものと等しい。$x$ を求めなさい。
答え: $x = 5$
解説: $3x + 1 = 5x - 9$、$-2x = -10$、$x = 5$

(4) ある数 $x$ を $2$ 倍して $3$ を引いた数と、$x$ に $9$ を加えた数が等しい。$x$ を求めなさい。
答え: $x = 12$
解説: $2x - 3 = x + 9$、$x = 12$

## 発展5: 総合問題（各5点×3＝15点）

(1) 方程式 $7x - 4 = 3(x + 4)$ を解き、解の確かめもしなさい。
答え: $x = 4$
解説: 展開して $7x - 4 = 3x + 12$、$4x = 16$、$x = 4$。確かめ: 左辺 $= 7 \times 4 - 4 = 24$、右辺 $= 3(4 + 4) = 24$。左辺 $=$ 右辺 ✓

(2) 方程式 $2(4x - 3) = 5(x + 3)$ を解きなさい。
答え: $x = 7$
解説: 展開して $8x - 6 = 5x + 15$、$3x = 21$、$x = 7$

(3) 方程式 $a(x - 1) = 2x + 4$ の解が $x = 3$ のとき、$a$ の値を求めなさい。
答え: $a = 5$
解説: $x = 3$ を代入すると $a(3 - 1) = 6 + 4$、$2a = 10$、$a = 5$
