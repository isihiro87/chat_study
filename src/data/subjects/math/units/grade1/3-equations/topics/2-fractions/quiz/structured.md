# 大問形式：小数・分数の方程式

## 大問1 小数をふくむ方程式（10倍で解く）（各4点×6＝24点）

次の方程式を解きなさい。

(1) $0.6x = 3$
(2) $0.4x + 0.8 = 2$
(3) $0.3x - 0.9 = 0$
(4) $0.7x + 1.4 = 5.6$
(5) $0.8x - 0.2 = 3$
(6) $1.2x - 0.6 = 3$

### 解答
(1) 10倍: $6x = 30$、$x = 5$
(2) 10倍: $4x + 8 = 20$、$4x = 12$、$x = 3$
(3) 10倍: $3x - 9 = 0$、$3x = 9$、$x = 3$
(4) 10倍: $7x + 14 = 56$、$7x = 42$、$x = 6$
(5) 10倍: $8x - 2 = 30$、$8x = 32$、$x = 4$
(6) 10倍: $12x - 6 = 30$、$12x = 36$、$x = 3$

## 大問3 分数をふくむ方程式（分母が2つ）（各4点×6＝24点）

次の方程式を解きなさい。

(1) $\dfrac{x}{2} + \dfrac{x}{3} = 10$
(2) $\dfrac{x}{3} - \dfrac{x}{5} = 4$
(3) $\dfrac{x + 1}{2} = \dfrac{x - 1}{3}$
(4) $\dfrac{3x + 1}{4} = \dfrac{x + 3}{2}$
(5) $\dfrac{x}{4} + \dfrac{x}{6} = 5$
(6) $\dfrac{3x}{2} - \dfrac{x}{3} = 7$

### 解答
(1) 最小公倍数6: $3x + 2x = 60$、$5x = 60$、$x = 12$
(2) 最小公倍数15: $5x - 3x = 60$、$2x = 60$、$x = 30$
(3) 最小公倍数6: $3(x+1) = 2(x-1)$、$3x+3 = 2x-2$、$x = -5$
(4) 最小公倍数4: $3x+1 = 2(x+3)$、$3x+1 = 2x+6$、$x = 5$
(5) 最小公倍数12: $3x + 2x = 60$、$5x = 60$、$x = 12$
(6) 最小公倍数6: $9x - 2x = 42$、$7x = 42$、$x = 6$

## 大問4 分数をふくむ方程式（複雑）（各5点×5＝25点）

次の方程式を解きなさい。

(1) $\dfrac{x}{2} + \dfrac{x}{3} + \dfrac{x}{6} = 6$
(2) $\dfrac{x-2}{3} + \dfrac{x+1}{4} = 2$
(3) $\dfrac{2x-3}{6} = \dfrac{x+1}{4}$
(4) $\dfrac{x+3}{2} - \dfrac{x-1}{5} = 3$
(5) $\dfrac{3x-1}{4} - \dfrac{x+2}{6} = 1$

### 解答
(1) 最小公倍数6: $3x + 2x + x = 36$、$6x = 36$、$x = 6$
(2) 最小公倍数12: $4(x-2) + 3(x+1) = 24$、$4x-8+3x+3 = 24$、$7x - 5 = 24$、$7x = 29$、$x = \dfrac{29}{7}$
(3) 最小公倍数12: $2(2x-3) = 3(x+1)$、$4x-6 = 3x+3$、$x = 9$
(4) 最小公倍数10: $5(x+3) - 2(x-1) = 30$、$5x+15-2x+2 = 30$、$3x + 17 = 30$、$3x = 13$、$x = \dfrac{13}{3}$
(5) 最小公倍数12: $3(3x-1) - 2(x+2) = 12$、$9x-3-2x-4 = 12$、$7x - 7 = 12$、$7x = 19$、$x = \dfrac{19}{7}$

## 大問5 小数・分数の混合問題（各5点×5＝25点）

次の方程式を解きなさい。

(1) $0.5x + \dfrac{x}{4} = 6$
(2) $\dfrac{x}{3} - 0.4x = 1$
(3) $0.2(x + 5) = \dfrac{x}{3}$
(4) $\dfrac{0.6x - 1}{2} = 1$
(5) $0.3x + \dfrac{x - 1}{5} = 2$

### 解答
(1) $0.5x = \dfrac{x}{2}$ なので $\dfrac{x}{2} + \dfrac{x}{4} = 6$、最小公倍数4: $2x + x = 24$、$3x = 24$、$x = 8$
(2) $0.4x = \dfrac{2x}{5}$ なので $\dfrac{x}{3} - \dfrac{2x}{5} = 1$、最小公倍数15: $5x - 6x = 15$、$-x = 15$、$x = -15$
(3) 左辺を展開: $0.2x + 1 = \dfrac{x}{3}$、$\dfrac{x}{5} + 1 = \dfrac{x}{3}$、最小公倍数15: $3x + 15 = 5x$、$2x = 15$、$x = \dfrac{15}{2}$
(4) 両辺に2をかけて $0.6x - 1 = 2$、$0.6x = 3$、10倍: $6x = 30$、$x = 5$
(5) $0.3x = \dfrac{3x}{10}$ なので $\dfrac{3x}{10} + \dfrac{x-1}{5} = 2$、最小公倍数10: $3x + 2(x-1) = 20$、$3x + 2x - 2 = 20$、$5x = 22$、$x = \dfrac{22}{5}$
