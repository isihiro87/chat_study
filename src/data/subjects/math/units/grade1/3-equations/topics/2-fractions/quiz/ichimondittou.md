# 一問一答：小数・分数の方程式

## 小数の方程式の基本

Q1. 小数をふくむ方程式を解くとき、最初にやることは？
A1. 両辺を10倍や100倍して、係数を整数にする

Q2. 小数第1位までの小数をふくむ方程式では、両辺を何倍するか？
A2. 10倍

Q3. 小数第2位までの小数をふくむ方程式では、両辺を何倍するか？
A3. 100倍

Q4. $0.3x = 1.5$ を解くとき、両辺を何倍すればよいか？
A4. 10倍（$3x = 15$ になる）

Q5. $0.05x = 2$ を解くとき、両辺を何倍すればよいか？
A5. 100倍（$5x = 200$ になる）

Q6. なぜ小数を整数にしてから解くのか？
A6. 整数のほうが計算ミスが少なく、解きやすいから

## 小数の方程式を解く

Q7. $0.4x = 2$ の解は？
A7. $x = 5$（10倍して $4x = 20$、$x = 5$）

Q8. $0.7x = 2.1$ の解は？
A8. $x = 3$（10倍して $7x = 21$、$x = 3$）

Q9. $0.6x - 1.2 = 0$ の解は？
A9. $x = 2$（10倍して $6x - 12 = 0$、$6x = 12$）

Q10. $0.2x + 0.6 = 1.4$ の解は？
A10. $x = 4$（10倍して $2x + 6 = 14$、$2x = 8$）

Q11. $0.5x - 0.3 = 0.7$ の解は？
A11. $x = 2$（10倍して $5x - 3 = 7$、$5x = 10$）

Q12. $1.5x - 3 = 0$ の解は？
A12. $x = 2$（10倍して $15x - 30 = 0$、$15x = 30$）

Q13. $0.3x + 0.1 = 1$ の解は？
A13. $x = 3$（10倍して $3x + 1 = 10$、$3x = 9$）

Q14. $0.4x - 0.8 = 1.2$ の解は？
A14. $x = 5$（10倍して $4x - 8 = 12$、$4x = 20$）

Q15. $0.25x = 3$ の解は？
A15. $x = 12$（100倍して $25x = 300$、$x = 12$）

Q16. $0.15x + 0.6 = 1.2$ の解は？
A16. $x = 4$（100倍して $15x + 60 = 120$、$15x = 60$）

## 分数の方程式の基本

Q17. 分数をふくむ方程式で、分母をなくす操作を何というか？
A17. 分母をはらう

Q18. 分母をはらうには何をかけるか？
A18. 分母の最小公倍数を両辺にかける

Q19. 2 と 3 の最小公倍数は？
A19. 6

Q20. 3 と 4 の最小公倍数は？
A20. 12

Q21. 4 と 6 の最小公倍数は？
A21. 12

Q22. 2, 3, 6 の最小公倍数は？
A22. 6

Q23. 3, 4, 6 の最小公倍数は？
A23. 12

Q24. 2, 5 の最小公倍数は？
A24. 10

Q25. 分母をはらうとき、もっとも注意すべきことは？
A25. すべての項にもれなく最小公倍数をかけること

## 分数の方程式を解く

Q26. $\dfrac{x}{5} = 3$ の解は？
A26. $x = 15$（両辺に5をかけて $x = 15$）

Q27. $\dfrac{x}{3} + 1 = 3$ の解は？
A27. $x = 6$（両辺に3をかけて $x + 3 = 9$、$x = 6$）

Q28. $\dfrac{x}{2} + \dfrac{x}{5} = 7$ の解は？
A28. $x = 10$（最小公倍数10をかけて $5x + 2x = 70$、$7x = 70$）

Q29. $\dfrac{x}{3} - \dfrac{x}{4} = 1$ の解は？
A29. $x = 12$（最小公倍数12をかけて $4x - 3x = 12$）

Q30. $\dfrac{2x}{3} = 4$ の解は？
A30. $x = 6$（両辺に3をかけて $2x = 12$、$x = 6$）

Q31. $\dfrac{x + 1}{2} = 3$ の解は？
A31. $x = 5$（両辺に2をかけて $x + 1 = 6$、$x = 5$）

Q32. $\dfrac{x - 3}{4} = 2$ の解は？
A32. $x = 11$（両辺に4をかけて $x - 3 = 8$、$x = 11$）

Q33. $\dfrac{3x - 1}{2} = 4$ の解は？
A33. $x = 3$（両辺に2をかけて $3x - 1 = 8$、$3x = 9$）

Q34. $\dfrac{x}{2} - \dfrac{x}{3} = 2$ の解は？
A34. $x = 12$（最小公倍数6をかけて $3x - 2x = 12$）

Q35. $\dfrac{x + 2}{3} = \dfrac{x - 1}{2}$ の解は？
A35. $x = 7$（最小公倍数6をかけて $2(x+2) = 3(x-1)$、$2x+4 = 3x-3$、$x = 7$）

## 一次方程式を解く手順

Q36. 一次方程式を解く手順を5つ順に答えよ。
A36. (1)かっこをはずす (2)分母をはらう (3)移項して整理 (4)$ax = b$ の形にする (5)両辺を $a$ で割る

Q37. 「移項」とは何か？
A37. 等式の一方の辺にある項を、符号を変えて他方の辺に移すこと

Q38. $ax = b$ の形から $x$ を求めるには？
A38. 両辺を $a$ で割って $x = \dfrac{b}{a}$

## 小数と分数が混在する方程式

Q39. $0.5x + \dfrac{x}{4} = 3$ の解は？
A39. $x = 4$（$0.5x = \dfrac{x}{2}$ なので $\dfrac{x}{2} + \dfrac{x}{4} = 3$、最小公倍数4をかけて $2x + x = 12$、$3x = 12$）

Q40. $\dfrac{x}{3} + 0.5 = 2.5$ の解は？
A40. $x = 6$（$\dfrac{x}{3} = 2$ より $x = 6$）

Q41. $0.2x - \dfrac{x}{10} = 1$ の解は？
A41. $x = 10$（$0.2x = \dfrac{2x}{10}$ なので $\dfrac{2x}{10} - \dfrac{x}{10} = 1$、$\dfrac{x}{10} = 1$、$x = 10$）

Q42. $\dfrac{3x + 1}{5} = \dfrac{x + 3}{3}$ の解は？
A42. $x = 6$（最小公倍数15をかけて $3(3x+1) = 5(x+3)$、$9x+3 = 5x+15$、$4x = 12$）

Q43. $0.3(x + 2) = 1.8$ の解は？
A43. $x = 4$（10倍して $3(x+2) = 18$、$3x + 6 = 18$、$3x = 12$）

Q44. $\dfrac{2(x-1)}{3} = \dfrac{x+1}{2}$ の解は？
A44. $x = 7$（最小公倍数6をかけて $4(x-1) = 3(x+1)$、$4x-4 = 3x+3$、$x = 7$）

Q45. $\dfrac{x}{6} + \dfrac{x}{4} = \dfrac{5}{12}$ の解は？
A45. $x = 1$（最小公倍数12をかけて $2x + 3x = 5$、$5x = 5$）
