# 一問一答：比例式と方程式の利用

## 比の値と比例式の基本

Q1. 比 $a : b$ の比の値とは何か？
A1. $\dfrac{a}{b}$ のこと（前の数を後の数でわった値）

Q2. 比 $6 : 4$ の比の値を求めよ。
A2. $\dfrac{6}{4} = \dfrac{3}{2}$

Q3. 比 $15 : 25$ の比の値を求めよ。
A3. $\dfrac{15}{25} = \dfrac{3}{5}$

Q4. 比の値が等しい2つの比を結んだ式を何というか？
A4. 比例式

Q5. 比例式の形を書け。
A5. $a : b = c : d$

Q6. 比例式 $a : b = c : d$ で、$b$ と $c$ を何というか？
A6. 内項

Q7. 比例式 $a : b = c : d$ で、$a$ と $d$ を何というか？
A7. 外項

Q8. 比例式 $a : b = c : d$ のとき成り立つ等式は何か？
A8. $ad = bc$（内項の積 $=$ 外項の積）

## 比例式の解法

Q9. 比例式 $x : 3 = 8 : 6$ を解け。
A9. $6x = 24$ より $x = 4$

Q10. 比例式 $4 : 7 = 12 : x$ を解け。
A10. $4x = 84$ より $x = 21$

Q11. 比例式 $5 : x = 3 : 9$ を解け。
A11. $3x = 45$ より $x = 15$

Q12. 比例式 $x : 6 = 7 : 3$ を解け。
A12. $3x = 42$ より $x = 14$

Q13. 比例式 $2 : 3 = (x+1) : 12$ を解け。
A13. $3(x+1) = 24$ より $3x + 3 = 24$、$x = 7$

Q14. 比例式 $0.3 : 0.5 = x : 10$ を解け。
A14. $0.5x = 3$ より $x = 6$

Q15. 比 $3 : 5$ と比の値が等しい比を、$x : 20$ の形で表すと $x$ はいくつか？
A15. $\dfrac{3}{5} = \dfrac{x}{20}$ より $x = 12$

## 方程式の文章題の手順

Q16. 方程式の文章題を解く手順の第1ステップは？
A16. 求めたいものを $x$ とおく

Q17. 第2ステップは？
A17. 問題文の条件から等しい関係を見つけて方程式を立てる

Q18. 第3ステップは？
A18. 方程式を解く

Q19. 第4ステップは？
A19. 解の吟味（求めた解が問題の条件に合っているか確かめる）

Q20. 解の吟味とは何か？
A20. 求めた解が問題の条件に合っているか確認すること

Q21. 解の吟味で「人数」を求める問題で $x = -2$ が出たら、どう判断するか？
A21. 人数は正の整数だから不適。条件に合わない

## 代金の問題

Q22. 1冊 $150$ 円のノートを $x$ 冊と $80$ 円の消しゴム 1 個で合計 $680$ 円だった。$x$ を求めよ。
A22. $150x + 80 = 680$ より $150x = 600$、$x = 4$

Q23. 1個 $200$ 円のりんごと1個 $150$ 円のみかんを合わせて $10$ 個買い、代金は $1700$ 円だった。りんごは何個か？
A23. りんご $x$ 個とすると $200x + 150(10 - x) = 1700$、$50x = 200$、$x = 4$

Q24. ある品物を定価の $2$ 割引きで買ったら $960$ 円だった。定価はいくらか？
A24. 定価を $x$ 円とすると $0.8x = 960$、$x = 1200$ 円

Q25. $100$ 円の品物と $60$ 円の品物を合わせて $15$ 個買い、代金は $1260$ 円だった。$100$ 円の品物は何個か？
A25. $100$ 円の品物を $x$ 個とすると $100x + 60(15-x) = 1260$、$40x = 360$、$x = 9$

## 過不足の問題

Q26. 過不足の問題で立式するときのポイントは？
A26. 「配るものの総数」が一定なので、2通りの配り方の式を等しくおく

Q27. あめを何人かに配る。1人4個ずつだと6個余り、5個ずつだと3個不足する。人数を求めよ。
A27. 人数を $x$ とすると $4x + 6 = 5x - 3$、$x = 9$。9人

Q28. 折り紙を子どもに配る。1人3枚ずつだと10枚余り、5枚ずつだと2枚不足する。子どもの人数は？
A28. $3x + 10 = 5x - 2$ より $x = 6$。6人

Q29. 問27で、あめの総数は何個か？
A29. $4 \times 9 + 6 = 42$ 個

Q30. ノートを生徒に配る。1人2冊ずつだと8冊余り、3冊ずつだとちょうどなくなる。生徒は何人か？
A30. $2x + 8 = 3x$ より $x = 8$。8人

## 速さ・時間・道のりの問題

Q31. 速さの公式「道のり」を速さと時間で表すと？
A31. 道のり $=$ 速さ $\times$ 時間

Q32. 速さの公式「時間」を道のりと速さで表すと？
A32. 時間 $= \dfrac{\text{道のり}}{\text{速さ}}$

Q33. 家から学校まで分速 $60$ m で歩くと $20$ 分かかる。分速 $80$ m で歩くと何分かかるか？
A33. 道のりは $60 \times 20 = 1200$ m。$\dfrac{1200}{80} = 15$ 分

Q34. 家から駅まで時速 $4$ km で歩くと $30$ 分、時速 $12$ km の自転車だと何分かかるか？
A34. 道のりは $4 \times 0.5 = 2$ km。$\dfrac{2}{12} = \dfrac{1}{6}$ 時間 $= 10$ 分

## 追いつき問題

Q35. 追いつき問題で「追いついた」とはどういうことか？
A35. 2人の進んだ道のりが等しくなること

Q36. 兄が分速 $60$ m で出発し、$10$ 分後に弟が分速 $100$ m で追いかける。弟が出発してから何分で追いつくか？
A36. $60(x + 10) = 100x$ より $60x + 600 = 100x$、$40x = 600$、$x = 15$。15分後

Q37. 問36で、追いついた地点は家から何 m の場所か？
A37. $100 \times 15 = 1500$ m

Q38. A さんは分速 $70$ m で、B さんは分速 $50$ m で同時に同じ方向に出発する。2人が $800$ m 離れるのは何分後か？
A38. $(70 - 50)x = 800$ より $20x = 800$、$x = 40$。40分後

## 比例式の利用

Q39. $200$ g の砂糖水に砂糖が $30$ g 溶けている。砂糖と砂糖水の質量の比を最も簡単な整数の比で表すと？
A39. $30 : 200 = 3 : 20$

Q40. この砂糖水と同じ濃さの砂糖水を $500$ g 作るには、砂糖は何 g 必要か？
A40. $3 : 20 = x : 500$ より $20x = 1500$、$x = 75$ g

Q41. カードを A と B に $3 : 5$ の比で分ける。全部で $40$ 枚のとき、A は何枚もらえるか？
A41. A の枚数は $40 \times \dfrac{3}{8} = 15$ 枚

Q42. $720$ 円を兄と弟で $5 : 4$ の比に分ける。兄はいくらもらえるか？
A42. 兄は $720 \times \dfrac{5}{9} = 400$ 円

Q43. ある量を $2 : 3$ の比に分けたとき、大きい方が $21$ だった。小さい方はいくつか？
A43. $2 : 3 = x : 21$ より $3x = 42$、$x = 14$

Q44. 地図上の $3$ cm が実際の $600$ m を表す。地図上の $5$ cm は実際の何 m か？
A44. $3 : 600 = 5 : x$ より $3x = 3000$、$x = 1000$ m

Q45. 長さ $12$ m のロープを $3 : 1$ に分けると、長い方は何 m か？
A45. $12 \times \dfrac{3}{4} = 9$ m
