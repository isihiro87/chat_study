# 発展問題：小数・分数の方程式

## 発展1 複雑な分数方程式（各5点×4＝20点）

次の方程式を解きなさい。

(1) $\dfrac{2x+1}{3} - \dfrac{3x-2}{4} = \dfrac{1}{6}$

(2) $\dfrac{x-3}{2} + \dfrac{2x+1}{3} - \dfrac{x}{6} = 4$

(3) $\dfrac{5x-1}{6} - \dfrac{x+3}{4} = \dfrac{x-2}{3}$

(4) $\dfrac{x+1}{2} - \dfrac{2(x-3)}{5} = 1$

### 解答
(1) 最小公倍数12: $4(2x+1) - 3(3x-2) = 2$、$8x+4-9x+6 = 2$、$-x + 10 = 2$、$-x = -8$、$x = 8$

(2) 最小公倍数6: $3(x-3) + 2(2x+1) - x = 24$、$3x-9+4x+2-x = 24$、$6x - 7 = 24$、$6x = 31$、$x = \dfrac{31}{6}$

(3) 最小公倍数12: $2(5x-1) - 3(x+3) = 4(x-2)$、$10x-2-3x-9 = 4x-8$、$7x - 11 = 4x - 8$、$3x = 3$、$x = 1$

(4) 最小公倍数10: $5(x+1) - 4(x-3) = 10$、$5x+5-4x+12 = 10$、$x + 17 = 10$、$x = -7$

## 発展2 係数決定問題（各6点×4＝24点）

(1) 方程式 $\dfrac{x+a}{3} = 2$ の解が $x = 4$ であるとき、$a$ の値を求めなさい。

(2) 方程式 $0.5x - a = 1.5$ の解が $x = 7$ であるとき、$a$ の値を求めなさい。

(3) 方程式 $\dfrac{ax}{2} + \dfrac{x}{3} = 5$ の解が $x = 6$ であるとき、$a$ の値を求めなさい。

(4) 方程式 $\dfrac{x-1}{a} = \dfrac{x+3}{6}$ の解が $x = 5$ であるとき、$a$ の値を求めなさい。

### 解答
(1) $x = 4$ を代入: $\dfrac{4+a}{3} = 2$、$4 + a = 6$、$a = 2$

(2) $x = 7$ を代入: $0.5 \times 7 - a = 1.5$、$3.5 - a = 1.5$、$a = 2$

(3) $x = 6$ を代入: $\dfrac{6a}{2} + \dfrac{6}{3} = 5$、$3a + 2 = 5$、$3a = 3$、$a = 1$

(4) $x = 5$ を代入: $\dfrac{5-1}{a} = \dfrac{5+3}{6}$、$\dfrac{4}{a} = \dfrac{8}{6}$、$\dfrac{4}{a} = \dfrac{4}{3}$、$a = 3$

## 発展3 小数と分数が混在する方程式（各5点×4＝20点）

次の方程式を解きなさい。

(1) $0.5x - \dfrac{x+1}{3} = 1$

(2) $\dfrac{0.4x + 1}{2} = \dfrac{x - 0.5}{3}$

(3) $0.75x - \dfrac{2x-1}{4} = 2$

(4) $\dfrac{1.2x - 3}{5} + 0.2x = 1$

### 解答
(1) $0.5x = \dfrac{x}{2}$ なので $\dfrac{x}{2} - \dfrac{x+1}{3} = 1$、最小公倍数6: $3x - 2(x+1) = 6$、$3x - 2x - 2 = 6$、$x = 8$

(2) まず10倍の考え方: $0.4x = \dfrac{2x}{5}$、$0.5 = \dfrac{1}{2}$ なので $\dfrac{\frac{2x}{5}+1}{2} = \dfrac{x - \frac{1}{2}}{3}$。分子分母を整理して、全体に30をかけると: $15 \cdot \dfrac{2x+5}{5} = 10 \cdot \dfrac{2x-1}{2}$、$3(2x+5) = 5(2x-1)$、$6x+15 = 10x-5$、$-4x = -20$、$x = 5$

(3) $0.75x = \dfrac{3x}{4}$ なので $\dfrac{3x}{4} - \dfrac{2x-1}{4} = 2$、$\dfrac{3x - (2x-1)}{4} = 2$、$\dfrac{x+1}{4} = 2$、$x + 1 = 8$、$x = 7$

(4) $1.2x = \dfrac{6x}{5}$ なので $\dfrac{\frac{6x}{5} - 3}{5} + \dfrac{x}{5} = 1$、$\dfrac{6x-15}{25} + \dfrac{x}{5} = 1$、両辺に25: $6x - 15 + 5x = 25$、$11x = 40$、$x = \dfrac{40}{11}$

## 発展4 かっこと分数の複合問題（各5点×4＝20点）

次の方程式を解きなさい。

(1) $\dfrac{3(x-2)}{4} - \dfrac{2(x+1)}{3} = -1$

(2) $\dfrac{2(x+3)}{5} = \dfrac{3(x-1)}{4}$

(3) $\dfrac{x - 2(x-1)}{3} = \dfrac{x+4}{6}$

(4) $\dfrac{4(x+1) - 3}{6} = \dfrac{2x - 1}{4}$

### 解答
(1) 最小公倍数12: $9(x-2) - 8(x+1) = -12$、$9x-18-8x-8 = -12$、$x - 26 = -12$、$x = 14$

(2) 最小公倍数20: $4 \cdot 2(x+3) = 5 \cdot 3(x-1)$、$8(x+3) = 15(x-1)$、$8x+24 = 15x-15$、$-7x = -39$、$x = \dfrac{39}{7}$

(3) 左辺の分子を整理: $\dfrac{x - 2x + 2}{3} = \dfrac{x+4}{6}$、$\dfrac{-x+2}{3} = \dfrac{x+4}{6}$、最小公倍数6: $2(-x+2) = x+4$、$-2x+4 = x+4$、$-3x = 0$、$x = 0$

(4) 左辺の分子を整理: $\dfrac{4x+4-3}{6} = \dfrac{2x-1}{4}$、$\dfrac{4x+1}{6} = \dfrac{2x-1}{4}$、最小公倍数12: $2(4x+1) = 3(2x-1)$、$8x+2 = 6x-3$、$2x = -5$、$x = -\dfrac{5}{2}$

## 発展5 総合応用問題（各4点×4＝16点）

(1) $\dfrac{x}{2} = \dfrac{x}{3} + 1$ と $\dfrac{x}{4} = \dfrac{x}{6} + a$ が同じ解をもつとき、$a$ の値を求めなさい。

(2) 方程式 $\dfrac{x+a}{2} = \dfrac{x-a}{3} + 1$ の解が $x = a + 4$ であるとき、$a$ の値を求めなさい。

(3) 方程式 $0.1x + \dfrac{x-3}{5} = 0.7$ を解きなさい。

(4) ある数 $x$ について、$x$ の $\dfrac{1}{3}$ と $x$ の $\dfrac{1}{4}$ の和が $7$ になるとき、$x$ の値を求めなさい。

### 解答
(1) 第1式: 最小公倍数6で $3x = 2x + 6$、$x = 6$。$x = 6$ を第2式に代入: $\dfrac{6}{4} = \dfrac{6}{6} + a$、$\dfrac{3}{2} = 1 + a$、$a = \dfrac{1}{2}$

(2) $x = a + 4$ を代入: $\dfrac{a+4+a}{2} = \dfrac{a+4-a}{3} + 1$、$\dfrac{2a+4}{2} = \dfrac{4}{3} + 1$、$a + 2 = \dfrac{4}{3} + 1$、$a + 2 = \dfrac{7}{3}$、$a = \dfrac{7}{3} - 2 = \dfrac{1}{3}$

(3) $0.1x = \dfrac{x}{10}$ なので $\dfrac{x}{10} + \dfrac{x-3}{5} = \dfrac{7}{10}$、最小公倍数10: $x + 2(x-3) = 7$、$x + 2x - 6 = 7$、$3x = 13$、$x = \dfrac{13}{3}$

(4) $\dfrac{x}{3} + \dfrac{x}{4} = 7$、最小公倍数12: $4x + 3x = 84$、$7x = 84$、$x = 12$
