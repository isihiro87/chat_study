# 大問小問形式：方程式の基本と移項

## 大問1: 方程式の解の判定（配点20点）

次の方程式について、[ ]内の値が解であるかどうか調べなさい。

(1) [4点] $x + 7 = 12$ [$x = 5$]
答え: 解である（左辺 $= 5 + 7 = 12 =$ 右辺）
解説: $x = 5$ を代入して左辺と右辺が等しくなるか確かめます。

(2) [4点] $3x = 18$ [$x = 5$]
答え: 解ではない（左辺 $= 3 \times 5 = 15 \neq 18 =$ 右辺）
解説: $x = 5$ を代入すると左辺は $15$ で右辺の $18$ と等しくないので解ではありません。

(3) [4点] $2x - 3 = 11$ [$x = 7$]
答え: 解である（左辺 $= 2 \times 7 - 3 = 11 =$ 右辺）
解説: $x = 7$ を代入すると $14 - 3 = 11$ で等式が成り立ちます。

(4) [4点] $5x + 4 = 3x + 10$ [$x = 3$]
答え: 解である（左辺 $= 19$、右辺 $= 19$）
解説: 左辺 $= 5 \times 3 + 4 = 19$、右辺 $= 3 \times 3 + 10 = 19$ で等しい。

(5) [4点] $4(x - 2) = 20$ [$x = 6$]
答え: 解ではない（左辺 $= 4 \times 4 = 16 \neq 20$）
解説: $x = 6$ を代入すると左辺 $= 4(6 - 2) = 16$ で右辺 $20$ と等しくありません。解は $x = 7$ です。

## 大問2: 等式の性質を使って解く（配点20点）

次の方程式を、等式の性質を使って解きなさい。

(1) [4点] $x + 8 = 13$
答え: $x = 5$
解説: 両辺から $8$ を引くと $x = 13 - 8 = 5$

(2) [4点] $x - 6 = 4$
答え: $x = 10$
解説: 両辺に $6$ を加えると $x = 4 + 6 = 10$

(3) [4点] $6x = 42$
答え: $x = 7$
解説: 両辺を $6$ で割ると $x = 42 \div 6 = 7$

(4) [4点] $\dfrac{x}{3} = 5$
答え: $x = 15$
解説: 両辺に $3$ をかけると $x = 5 \times 3 = 15$

(5) [4点] $-4x = 24$
答え: $x = -6$
解説: 両辺を $-4$ で割ると $x = 24 \div (-4) = -6$

## 大問3: 移項を使った方程式の解法（配点20点）

次の方程式を移項を使って解きなさい。

(1) [3点] $x + 11 = 4$
答え: $x = -7$
解説: $+11$ を移項して $x = 4 - 11 = -7$

(2) [3点] $x - 9 = -2$
答え: $x = 7$
解説: $-9$ を移項して $x = -2 + 9 = 7$

(3) [4点] $3x + 5 = 20$
答え: $x = 5$
解説: $+5$ を移項して $3x = 15$、両辺を $3$ で割って $x = 5$

(4) [4点] $7x - 3 = 18$
答え: $x = 3$
解説: $-3$ を移項して $7x = 21$、両辺を $7$ で割って $x = 3$

(5) [3点] $-2x + 8 = 14$
答え: $x = -3$
解説: $+8$ を移項して $-2x = 6$、両辺を $-2$ で割って $x = -3$

(6) [3点] $4x + 3 = -9$
答え: $x = -3$
解説: $+3$ を移項して $4x = -12$、両辺を $4$ で割って $x = -3$

## 大問4: 両辺にxがある方程式（配点20点）

次の方程式を解きなさい。

(1) [4点] $5x + 2 = 2x + 14$
答え: $x = 4$
解説: $2x$ を左辺に、$+2$ を右辺に移項して $3x = 12$、$x = 4$

(2) [4点] $8x - 3 = 5x + 9$
答え: $x = 4$
解説: $5x$ を左辺に、$-3$ を右辺に移項して $3x = 12$、$x = 4$

(3) [4点] $3x + 7 = 7x - 5$
答え: $x = 3$
解説: $7x$ を左辺に、$+7$ を右辺に移項して $-4x = -12$、$x = 3$

(4) [4点] $9x - 6 = 4x + 14$
答え: $x = 4$
解説: $4x$ を左辺に、$-6$ を右辺に移項して $5x = 20$、$x = 4$

(5) [4点] $x + 15 = 6x - 10$
答え: $x = 5$
解説: $6x$ を左辺に、$+15$ を右辺に移項して $-5x = -25$、$x = 5$

## 大問5: かっこがある方程式（配点20点）

次の方程式を解きなさい。

(1) [4点] $3(x + 2) = 21$
答え: $x = 5$
解説: 展開して $3x + 6 = 21$、$3x = 15$、$x = 5$

(2) [4点] $4(x - 3) = 8$
答え: $x = 5$
解説: 展開して $4x - 12 = 8$、$4x = 20$、$x = 5$

(3) [4点] $2(x + 5) = x + 13$
答え: $x = 3$
解説: 展開して $2x + 10 = x + 13$、$x = 3$

(4) [4点] $5(x - 1) = 3(x + 3)$
答え: $x = 7$
解説: 展開して $5x - 5 = 3x + 9$、$2x = 14$、$x = 7$

(5) [4点] $-(2x + 4) = 3x - 19$
答え: $x = 3$
解説: 展開して $-2x - 4 = 3x - 19$、$-5x = -15$、$x = 3$
