# 大問形式：比例式と方程式の利用

## 大問1 比の値と比例式（各4点×6＝24点）

次の問いに答えなさい。

(1) 比 $8 : 12$ の比の値を求めよ。

(2) 比 $14 : 21$ の比の値を求めよ。

(3) 比例式 $x : 4 = 9 : 12$ を解け。

(4) 比例式 $3 : x = 5 : 15$ を解け。

(5) 比例式 $(x - 2) : 4 = 3 : 2$ を解け。

(6) 比例式 $0.4 : 0.6 = x : 15$ を解け。

### 解答
(1) $\dfrac{8}{12} = \dfrac{2}{3}$

(2) $\dfrac{14}{21} = \dfrac{2}{3}$

(3) $12x = 36$ より $x = 3$

(4) $5x = 45$ より $x = 9$

(5) $2(x - 2) = 12$ より $2x - 4 = 12$、$2x = 16$、$x = 8$

(6) $0.6x = 6$ より $x = 10$

## 大問2 代金の問題（各5点×5＝25点）

次の文章を読んで方程式を立て、答えを求めなさい。

(1) $1$ 本 $90$ 円のペンを $x$ 本と $200$ 円のペンケースを $1$ つ買ったら合計 $740$ 円だった。ペンは何本買ったか。

(2) $80$ 円の切手と $120$ 円の切手を合わせて $12$ 枚買い、代金は $1120$ 円だった。$80$ 円の切手は何枚か。

(3) A店ではある品物が $800$ 円で、B店ではそれより $3$ 割安い。B店の値段はいくらか。

(4) ある品物を定価の $15\%$ 引きで買ったら $1020$ 円だった。定価はいくらか。

(5) $1$ 個 $a$ 円のケーキを $3$ 個と $1$ 個 $b$ 円のプリンを $2$ 個買うと $1100$ 円、ケーキ $1$ 個とプリン $4$ 個だと $900$ 円。ケーキ $1$ 個の値段を求めよ。

### 解答
(1) $90x + 200 = 740$ より $90x = 540$、$x = 6$。6本

(2) $80$ 円を $x$ 枚とすると $80x + 120(12 - x) = 1120$、$80x + 1440 - 120x = 1120$、$-40x = -320$、$x = 8$。8枚

(3) $800 \times (1 - 0.3) = 800 \times 0.7 = 560$ 円

(4) 定価を $x$ 円とすると $0.85x = 1020$、$x = 1200$。1200円

(5) $3a + 2b = 1100 \cdots ①$、$a + 4b = 900 \cdots ②$。②より $a = 900 - 4b$、①に代入して $3(900 - 4b) + 2b = 1100$、$2700 - 12b + 2b = 1100$、$-10b = -1600$、$b = 160$。$a = 900 - 640 = 260$。ケーキ1個 $260$ 円

## 大問3 過不足の問題（各5点×5＝25点）

(1) 色紙を何人かの生徒に配る。$1$ 人 $4$ 枚ずつだと $7$ 枚余り、$6$ 枚ずつだと $5$ 枚不足する。生徒の人数を求めよ。

(2) (1) のとき、色紙は全部で何枚あるか。

(3) りんごを箱に入れる。$1$ 箱に $8$ 個ずつ入れると $3$ 個余り、$10$ 個ずつ入れるとちょうどなくなる。箱の数を求めよ。

(4) えんぴつを何人かに配る。$1$ 人 $5$ 本ずつ配ると $12$ 本余り、$7$ 本ずつ配ると $4$ 本余る。人数を求めよ。

(5) 長いすに生徒が座る。$1$ 脚に $4$ 人ずつ座ると $5$ 人座れず、$5$ 人ずつ座ると最後の $1$ 脚は $2$ 人だけになる。長いすの数を求めよ。

### 解答
(1) $4x + 7 = 6x - 5$ より $-2x = -12$、$x = 6$。6人

(2) $4 \times 6 + 7 = 31$ 枚

(3) 箱の数を $x$ とすると $8x + 3 = 10x$、$2x = 3$。$x = 1.5$ だが箱数は整数なので条件を再確認。$10x = 8x + 3$ は不成立。正しくは $8x + 3$ がりんごの総数で $10$ 個ずつ入れるとちょうどなので、りんごの総数が $10$ の倍数かつ $8x + 3$ の形。$8x + 3 = 10x$ より $x = 1.5$…この問題は「$10$ 個ずつ入れると最後の箱に $3$ 個入る」と読み替えると、箱の数 $x$ は $8x + 3 = 10(x-1) + 3$ で不定。条件を見直し: $8x + 3 = 10 \times \dfrac{8x+3}{10}$。正解の立式は次の通り。りんごの数は $8x + 3$ 個、これを $10$ 個ずつ箱に入れるとちょうど $\dfrac{8x+3}{10}$ 箱に入るから $8x + 3$ は $10$ の倍数。$8x + 3 \equiv 0 \pmod{10}$ より $8x \equiv 7 \pmod{10}$、$x$ の最小正の整数は…。

別解: 箱の数を $y$ とし、りんごの総数を $n$ とすると $n = 8x + 3 = 10y$。「箱の数」が同じなら $x = y$ で $8x + 3 = 10x$、$x = 1.5$ で不適。「箱の数が変わる」問題なので、りんごの総数が等しいことだけ利用。$n = 8x + 3$（$x$ 箱）$= 10y$（$y$ 箱）。追加条件がないと不定なので、ここでは「箱の数が同じ」として立式: 不適切な問題設定でした。

**訂正**: りんごを箱に入れる。1箱 $8$ 個ずつ入れると $6$ 個余り、$10$ 個ずつ入れるとちょうどなくなる。箱の数を求めよ。

$8x + 6 = 10x$ より $2x = 6$、$x = 3$。3箱

(4) $5x + 12 = 7x + 4$ より $-2x = -8$、$x = 4$。4人

(5) 長いすの数を $x$ 脚とし、生徒の人数を $n$ 人とする。$n = 4x + 5$（$4$ 人ずつだと $5$ 人座れない）、$n = 5(x - 1) + 2$（最後の $1$ 脚だけ $2$ 人）。$4x + 5 = 5x - 5 + 2$ より $4x + 5 = 5x - 3$、$x = 8$。8脚

## 大問4 速さの問題（各5点×5＝25点）

(1) 家から公園まで分速 $80$ m で歩くと $15$ 分かかる。分速 $200$ m の自転車だと何分かかるか。

(2) A町からB町まで行きは時速 $3$ km で歩き、帰りは時速 $5$ km で走った。往復に $4$ 時間かかった。A町からB町までの距離を求めよ。

(3) 姉が家を出て分速 $60$ m で歩き始めた。$15$ 分後に妹が分速 $90$ m で追いかけた。妹が出発してから何分後に追いつくか。

(4) (3)のとき、追いついた地点は家から何 m の地点か。

(5) P地点とQ地点は $3000$ m 離れている。A は P から、B は Q から同時に向かい合って出発した。A の速さは分速 $70$ m、B の速さは分速 $80$ m のとき、出発してから何分後に出会うか。

### 解答
(1) 道のりは $80 \times 15 = 1200$ m。$\dfrac{1200}{200} = 6$ 分

(2) 距離を $x$ km とすると $\dfrac{x}{3} + \dfrac{x}{5} = 4$。両辺に $15$ をかけて $5x + 3x = 60$、$8x = 60$、$x = 7.5$。7.5 km

(3) $60(x + 15) = 90x$ より $60x + 900 = 90x$、$30x = 900$、$x = 30$。30分後

(4) $90 \times 30 = 2700$ m

(5) $(70 + 80)x = 3000$ より $150x = 3000$、$x = 20$。20分後

## 大問5 比例式の利用（各5点×5＝25点）

(1) ある砂糖水では砂糖と水の質量の比が $1 : 9$ である。同じ濃さの砂糖水を $300$ g 作りたい。砂糖は何 g 必要か。

(2) $600$ mL のジュースを A と B で $2 : 3$ の比に分ける。A は何 mL もらえるか。

(3) リボンを $4 : 5$ の比で分けたところ、長い方が $35$ cm だった。短い方は何 cm か。

(4) 地図上で $4$ cm の距離が実際には $2$ km を表している。実際の距離が $3.5$ km の場所は、地図上では何 cm か。

(5) $810$ 円を A、B、C の $3$ 人に $3 : 2 : 4$ の比で分ける。B はいくらもらえるか。

### 解答
(1) 砂糖と砂糖水の比は $1 : 10$。$1 : 10 = x : 300$ より $10x = 300$、$x = 30$ g

(2) $600 \times \dfrac{2}{5} = 240$ mL

(3) $4 : 5 = x : 35$ より $5x = 140$、$x = 28$ cm

(4) $4 : 2 = x : 3.5$ より $2x = 14$、$x = 7$ cm

(5) $B = 810 \times \dfrac{2}{9} = 180$ 円
