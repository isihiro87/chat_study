# 一問一答：加法と減法

## 同符号の加法

Q1. (+2)+(+5) の計算結果は？
A1. +7

Q2. (+1)+(+3) の計算結果は？
A2. +4

Q3. (−3)+(−5) の計算結果は？
A3. −8

Q4. (−2)+(−7) の計算結果は？
A4. −9

Q5. (−9)+(−1) の計算結果は？
A5. −10

Q6. 同符号の加法のルールを言いなさい。
A6. 絶対値の和に共通の符号をつける

Q7. (+0.4)+(+1.3) の計算結果は？
A7. +1.7

Q8. (−0.2)+(−1.3) の計算結果は？
A8. −1.5

Q9. (−1/2)+(−1/4) の計算結果は？
A9. −3/4

Q10. (+2/3)+(+1/3) の計算結果は？
A10. +1

## 異符号の加法

Q11. (+7)+(−3) の計算結果は？
A11. +4

Q12. (+3)+(−7) の計算結果は？

Q13. (−5)+(+4) の計算結果は？
A13. −1

A14. −3

Q15. (−5)+(+8) の計算結果は？

A16. 絶対値の差に、絶対値が大きい方の符号をつける

Q17. (+4)+(−4) の計算結果は？
A17. 0

Q18. (−2.6)+(+3.9) の計算結果は？
A18. +1.3

Q19. (+1.9)+(−4.2) の計算結果は？
A19. −2.3

Q20. (−3/4)+(+1/2) の計算結果は？
A20. −1/4

## 減法→加法への変換

Q21. 減法を加法に変換する方法は？
A21. ひく数の符号を変えて足す

Q22. (−2)−(+3) を加法に直すと？
A22. (−2)+(−3)

Q23. (+4)−(−7) を加法に直すと？
A23. (+4)+(+7)

Q24. (+8)−(+5) を加法に直すと？
A24. (+8)+(−5)

Q25. 0−(+7) の計算結果は？
A25. −7

Q26. (−3)−(−9) の計算結果は？
A26. +6

Q27. (+4)−(−7) の計算結果は？
A27. +11

Q28. (−7)−(−7) の計算結果は？
A28. 0

## 正の数に符号をつけずに表す

Q29. (+2)+(+3) を符号をつけずに表すと？
A29. 2+3 = 5

Q30. 3−(+5) を符号をつけずに表すと？
A30. 3−5 = −2

Q31. 5+(−3) を符号をつけずに表すと？
A31. 5−3 = 2

Q32. (−4)+(−8) を符号をつけずに表すと？
A32. −4−8 = −12

## 項

Q33. 加法だけの式に直したとき、それぞれの数を何というか？
A33. 項（こう）

Q34. 7−12+3 の正の項は？
A34. +7, +3

Q35. 7−12+3 の負の項は？
A35. −12

Q36. −5+8−2+1 の正の項の和は？
A36. +9（+8 と +1）

Q37. −5+8−2+1 の負の項の和は？
A37. −7（−5 と −2）
