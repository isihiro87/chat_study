# 一問一答：正の数・負の数と絶対値

## 正の数・負の数の定義

Q1. 0より大きい数を何というか？
A1. 正の数

Q2. 0より小さい数を何というか？
A2. 負の数

Q3. 正の数につける符号は？
A3. +（プラス、正の符号）

Q4. 負の数につける符号は？
A4. −（マイナス、負の符号）

Q5. 0は正の数か負の数か？
A5. どちらでもない

Q6. +7 は正の数か負の数か？
A6. 正の数

Q7. −3 は正の数か負の数か？
A7. 負の数

Q8. 0より0.2大きい数を正の符号をつけて表すと？
A8. +0.2

Q9. 0より5小さい数を負の符号をつけて表すと？
A9. −5

Q10. 0より 2/3 大きい数を正の符号をつけて表すと？
A10. +2/3

## 整数の分類

Q11. 正の整数を別の言い方で何というか？
A11. 自然数

Q12. 自然数を小さい方から3つ挙げると？
A12. 1, 2, 3

Q13. 整数の中で、正の整数でも負の整数でもない数は？
A13. 0

Q14. −4 は整数か？
A14. 整数である（負の整数）

Q15. +2.5 は整数か？
A15. 整数ではない（小数）

Q16. −3/4 は整数か？
A16. 整数ではない（分数）

Q17. 0 は自然数か？
A17. 自然数ではない

Q18. 整数を3つのグループに分けると何と何と何？
A18. 正の整数（自然数）、0、負の整数

## 反対の性質を持つ量

Q19. 「+500円の収入」の反対は何と表す？
A19. −500円の支出（または−500円の収入）

Q20. 「3kg増えた」を正の数で表すと？
A20. +3kg

Q21. 「5cm短い」を負の数で表すと？
A21. −5cm

Q22. 基準より2℃高いことを+2℃と表すとき、基準より6℃低いことは？
A22. −6℃

Q23. 「300gの増加」を+300gと表すとき、「270gの減少」は？
A23. −270g

Q24. 「東へ3km」を+3kmと表すとき、「西へ7km」は？
A24. −7km

Q25. 「2倍多い」を正の符号で表すと？
A25. +2倍（多い）

Q26. 「200円もうかった」を+200円と表すとき、「150円なくした」は？
A26. −150円

Q27. 「20分後」を+20分と表すとき、「6時間前」は？
A27. −6時間

## 数直線

Q28. 数直線上で0を表す点を何というか？
A28. 原点

Q29. 数直線では右にいくほど数はどうなる？
A29. 大きくなる

Q30. 数直線では左にいくほど数はどうなる？
A30. 小さくなる

Q31. 数直線上で +3 は原点からどちらに何の距離？
A31. 右に3の距離

Q32. 数直線上で −2 は原点からどちらに何の距離？
A32. 左に2の距離

## 絶対値

Q33. 数直線上で原点からある数までの距離を何というか？
A33. 絶対値

Q34. +5 の絶対値は？
A34. 5

Q35. −7 の絶対値は？
A35. 7

Q36. 0 の絶対値は？
A36. 0

Q37. −0.5 の絶対値は？
A37. 0.5

Q38. 絶対値が 4 の数は何と何？
A38. +4 と −4

Q39. 絶対値が等しい2つの数はどのような関係？
A39. 符号が反対で絶対値が同じ（原点から等距離）

## 数の大小

Q40. 正の数と負の数では、どちらが大きい？
A40. 正の数

Q41. 正の数と0ではどちらが大きい？
A41. 正の数

Q42. 負の数と0ではどちらが大きい？
A42. 0

Q43. −5 と −2 ではどちらが大きい？
A43. −2

Q44. 負の数どうしで、絶対値が大きい方はどうなる？
A44. 小さくなる

Q45. −3 と +1 の大小を不等号で表すと？
A45. −3 < +1

Q46. +4 と −7 の大小を不等号で表すと？
A46. −7 < +4

Q47. −9, 0, +6 を小さい順に並べると？
A47. −9, 0, +6

Q48. −1, −8, −3 を小さい順に並べると？
A48. −8, −3, −1
