# 大問形式：正の数・負の数と絶対値

## 大問1 正負の符号をつけて表す（各3点×6＝18点）

(1) 0より4大きい数
(2) 0より3小さい数
(3) 0より0.7小さい数
(4) 0より 5/3 大きい数
(5) 0より0.5大きい数
(6) 0より 5/9 小さい数

### 解答
(1) +4  (2) −3  (3) −0.7  (4) +5/3  (5) +0.5  (6) −5/9

## 大問2 数の分類（各2点×8＝16点）

次の数の中から、それぞれにあてはまる数をすべて選びなさい。
−14, +1.8, −8/9, 5, −7.1, +6/7, 0, +47, 7/5

(1) 整数
(2) 負の数
(3) 自然数
(4) 正の数でも負の数でもない数
(5) 負の整数
(6) 正の整数
(7) 分数
(8) 正の数

### 解答
(1) −14, 5, 0, +47  (2) −14, −8/9, −7.1  (3) 5, +47  (4) 0
(5) −14  (6) 5, +47  (7) −8/9, +6/7, 7/5  (8) +1.8, 5, +6/7, +47, 7/5

## 大問3 数直線の読み取り（各3点×6＝18点）

数直線上で、A, B, Cにあたる数を答えなさい。

(1) 数直線（−5 〜 5、1目盛り1）で A=−3, B=1, C=4 の位置
(2) 数直線（−5 〜 5、1目盛り0.5）で A=−4.5, B=+1.5, C=+3.5 の位置

次の数を、数直線上に表しなさい。
(3) −4
(4) +2.5
(5) +3 1/2
(6) −1.5

### 解答
(1) A=−3, B=1, C=4  (2) A=−4.5, B=+1.5, C=+3.5
(3)〜(6) 数直線上の該当位置に点を打つ

## 大問4 絶対値と符号変え（各3点×8＝24点）

次の数の絶対値を答えなさい。また、符号を変えた数を答えなさい。

|  | 数 | 絶対値 | 符号を変えた数 |
|--|------|--------|----------------|
|(1)| −6 | | |
|(2)| +2.5 | | |
|(3)| −3/4 | | |
|(4)| 0 | | |

### 解答
(1) 絶対値: 6, 符号変え: +6
(2) 絶対値: 2.5, 符号変え: −2.5
(3) 絶対値: 3/4, 符号変え: +3/4
(4) 絶対値: 0, 符号変え: 0

## 大問5 数の大小比較（各3点×8＝24点）

(1) □に不等号を書き入れて、2数の大小を表しなさい。
① 8 □ −4
② −3.1 □ −1.3
③ −5 □ 0
④ +0.25 □ −0.15

(2) 次の3つの数を小さい順に並べなさい。
① −9, 5, 0
② 8.4, −6, 0
③ −2.3, −1.3, −5.2
④ +1/2, −3/4, 0

### 解答
(1) ① > ② < ③ < ④ >
(2) ① −9, 0, 5  ② −6, 0, 8.4  ③ −5.2, −2.3, −1.3  ④ −3/4, 0, +1/2
