# 発展問題：分母の有理化

## 発展1 複雑な分数式の有理化（配点 各5点×4 = 20点）

(1) (3√2 + √12)/(√6) を有理化しなさい。

(2) (√18 − √8)/(√2) を計算しなさい。

(3) (√50 + √32)/(√2) を計算しなさい。

(4) (6√3 − √27)/(√3) を計算しなさい。

### 解答
(1) (3√2 + 2√3)/(√6) = (3√12 + 2√18)/6 = (6√3 + 6√2)/6 = √3 + √2
(2) (3√2 − 2√2)/√2 = √2/√2 = 1
(3) (5√2 + 4√2)/√2 = 9√2/√2 = 9
(4) (6√3 − 3√3)/√3 = 3√3/√3 = 3

---

## 発展2 有理化を利用した式の値（配点 各5点×4 = 20点）

(1) √2 = 1.414 として、(√8 + √2)/√2 の値を求めなさい。

(2) √3 = 1.732 として、√(1/3) + √(4/3) の値を求めなさい。

(3) √5 = 2.236 として、(√20 − √5)/√5 の値を求めなさい。

(4) √2 = 1.414, √3 = 1.732 として、√(8/3) の値を求めなさい。

### 解答
(1) (2√2 + √2)/√2 = 3√2/√2 = 3
(2) √3/3 + 2√3/3 = 3√3/3 = √3 = 1.732
(3) (2√5 − √5)/√5 = √5/√5 = 1
(4) √(8/3) = 2√2/√3 = 2√6/3 = 2 × 2.449/3 ≈ 1.633

---

## 発展3 除法と有理化の混合問題（配点 各5点×4 = 20点）

(1) 8 ÷ 4√6 を計算しなさい。

(2) √15 ÷ √12 × √8 を計算しなさい。

(3) 6√7 ÷ √21 を計算しなさい。

(4) (3/√6) × (√2/3) を計算しなさい。

### 解答
(1) 8/(4√6) = 2/√6 = 2√6/6 = √6/3
(2) √15/(2√3) × 2√2 = √5 × 2√2/2 = √10
(3) 6√(7/21) = 6/√3 = 2√3
(4) 3√2/(3√6) = √2/√6 = √(2/6) = 1/√3 = √3/3

---

## 発展4 有理化の工夫（配点 各5点×3 = 15点）

(1) √(5/12) を有理化しなさい。

(2) (√2 + 1)/√2 を有理化しなさい。

(3) √54 ÷ (12/√6) を計算しなさい。

### 解答
(1) √5/(2√3) = √15/6
(2) (√2 + 1) × √2/(√2 × √2) = (2 + √2)/2 = 1 + √2/2
(3) √54 × √6/12 = 3√6 × √6/12 = 3 × 6/12 = 18/12 = 3/2

---

## 発展5 総合問題（配点 各5点×3 = 15点）

(1) 9 + √18 の分母を有理化した式を求めなさい。（ヒント: 分母ではないが変形する）
※ 訂正: (9 + √18)/√3 を有理化しなさい。

(2) a = 3/√6, b = √6/2 のとき、a + b の値を求めなさい。

(3) √2 = 1.414, √3 = 1.732 として、(√3 − √2)/(√6) の値を求めなさい。

### 解答
(1) (9 + 3√2)/√3 = (9 + 3√2) × √3/3 = (9√3 + 3√6)/3 = 3√3 + √6
(2) a = 3√6/6 = √6/2, b = √6/2 → a + b = √6
(3) (√3 − √2) × √6/(√6 × √6) = (√18 − √12)/6 = (3√2 − 2√3)/6 = (3 × 1.414 − 2 × 1.732)/6 = (4.242 − 3.464)/6 = 0.778/6 ≈ 0.1297
