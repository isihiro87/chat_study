# 発展問題：根号の和差と展開

## 発展1 複雑な展開計算（配点 各5点×4 = 20点）

(1) √12 − (√15 − √5) ÷ √5 を計算しなさい。

(2) √3(4√2 − √48) ÷ √6 を計算しなさい。

(3) (√48 − √18) ÷ (−√6) を計算しなさい。

(4) (2√5 − 1)² − (√5 + 7)(√5 − 4) を計算しなさい。

### 解答
(1) 2√3 − (√15 − √5)/√5 = 2√3 − √3 + 1 = √3 + 1
(2) (4√6 − √144)/√6 = (4√6 − 12)/√6 = 4 − 12/√6 = 4 − 2√6
(3) (4√3 − 3√2)/(−√6) = −4√3/√6 + 3√2/√6 = −4/√2 + 3/√3 = −2√2 + √3
(4) (20 − 4√5 + 1) − (5 − 4√5 + 7√5 − 28) = 21 − 4√5 − (−23 + 3√5) = 21 − 4√5 + 23 − 3√5 = 44 − 7√5

## 発展2 式の値（配点 各5点×4 = 20点）

(1) x = √5 + √3, y = √5 − √3 のとき、x² + 2xy + y² の値を求めなさい。

(2) x = √5 − √3, y = √5 + √3 のとき、x² − y² の値を求めなさい。

(3) x = √7 + 3 のとき、x² − 6x + 8 の値を求めなさい。

(4) x = √5 − 2 のとき、x² + 4x − 1 の値を求めなさい。

### 解答
(1) (x+y)² = (2√5)² = 20
(2) (x+y)(x−y) = 2√5 × (−2√3) = −4√15
(3) (x−3)² − 1 = (√7)² − 1 = 6
(4) x² + 4x − 1 = (√5−2)² + 4(√5−2) − 1 = (5 − 4√5 + 4) + 4√5 − 8 − 1 = 0

## 発展3 √na が自然数になる問題（配点 各5点×4 = 20点）

(1) n < √60 < n+1 となる自然数 n を求めなさい。

(2) √150a が自然数になるような自然数 a のうち、もっとも小さいものを求めなさい。

(3) √(72−4a) が自然数になるような自然数 a をすべて求めなさい。

(4) √(12a) が自然数となるような2けたの自然数 a のうち、もっとも小さいものを求めなさい。

### 解答
(1) 7² = 49, 8² = 64 → 49 < 60 < 64 → n = 7
(2) 150 = 2 × 3 × 5² → a = 6 のとき √900 = 30。答: a = 6
(3) 72−4a > 0 → a ≤ 17。72−4a が完全平方数: 64→a=2, 36→a=9, 16→a=14, 4→a=17。答: a = 2, 9, 14, 17
(4) 12a = 4 × 3 × a → a = 3k² の形が必要。a = 3→√36=6, a = 12→√144=12（2けた最小は a = 12）

## 発展4 √の整数部分と小数部分（配点 各5点×3 = 15点）

(1) √7 の整数部分を a、小数部分を b とするとき、a, b の値を求めなさい。

(2) (1)のとき、b² + 4b + 4 の値を求めなさい。

(3) √15 の整数部分を a、小数部分を b とするとき、a² + 4ab + 4b² の値を求めなさい。

### 解答
(1) 2 < √7 < 3 より a = 2, b = √7 − 2
(2) (b+2)² = (√7−2+2)² = (√7)² = 7
(3) (a+2b)² = (2+2(√15−3))² = (2+2√15−6)² = (2√15−4)² = 60 − 16√15 + 16 = 76 − 16√15

## 発展5 平方根と図形（配点 各5点×4 = 20点）

(1) 丸太から切り口が正方形の角材を切り取ります。切り口の正方形の1辺が 10cm になるように切り取るとしたら、丸太の直径は少なくとも何cm以上あればよいですか。

(2) 半径が 5cm の円と半径が 10cm の円がある。面積の和と等しい面積の円の半径を求めなさい。

(3) 面積が 2つの正方形のそれぞれの面積の和になる円がある。正方形の1辺がそれぞれ √3cm と √5cm のとき、円の半径を求めなさい。

(4) 次の面積の正方形をかきなさい（1マスの1辺は1cm）。(1) 8cm²　(2) 10cm²

### 解答
(1) 対角線 = 10√2 cm → 直径 ≥ 10√2 ≈ 14.14 → 少なくとも 10√2 cm
(2) π(25 + 100) = 125π → r² = 125 → r = 5√5 cm
(3) 面積 = 3 + 5 = 8 → πr² = 8 → r² = 8/π → r = 2√(2/π) cm
(4) 8cm²: 2√2 cm の正方形（斜めに描く）。10cm²: √10 cm の正方形（1×3の長方形の対角線を使う）
