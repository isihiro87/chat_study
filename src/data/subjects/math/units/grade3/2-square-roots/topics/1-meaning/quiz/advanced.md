# 発展問題：平方根の意味と大小

## 発展1 自然数の個数（配点 各5点×3 = 15点）

(1) 4 < √a < 5.2 となる自然数 a の個数を求めなさい。

(2) √(a+4) < 4 となる自然数 a をすべて求めなさい。

(3) √a が自然数となるような2けたの自然数 a をすべて求めなさい。

### 解答
(1) 4² = 16, 5.2² = 27.04 より 17 ≤ a ≤ 27 → 11個
(2) a+4 < 16 より a < 12、a ≥ 1 なので a = 1, 2, 3, ..., 11 → 11個（ただし a+4 が0以上）
(3) √a が自然数 → a は完全平方数。2けたの完全平方数: 16, 25, 36, 49, 64, 81 → 6個

## 発展2 平方根の値（配点 各5点×3 = 15点）

(1) 2.6² = 6.76, 2.7² = 7.29 であることを利用して、√6.76 < √7 < √7.29 より、√7 の小数第1位の数を求めなさい。

(2) 3.2² = 10.24, 3.3² = 10.89 であることを利用して、√10 の小数第1位の数を求めなさい。

(3) √7 = 2.646 として、√700 の値を求めなさい。

### 解答
(1) √6.76 = 2.6, √7.29 = 2.7 なので 2.6 < √7 < 2.7 → 小数第1位は 6
(2) √10.24 = 3.2, √10.89 = 3.3 なので 3.2 < √10 < 3.3 → 小数第1位は1ではなく... 3.1² = 9.61, 3.2² = 10.24 → 3.1 < √10 < 3.2 → 小数第1位は 1
(3) √700 = √(100 × 7) = 10√7 = 10 × 2.646 = 26.46

## 発展3 近似値と誤差の範囲（配点 各5点×3 = 15点）

(1) ある数 a の小数第2位を四捨五入した近似値が 5.7 であるとき、a の範囲を不等号で表しなさい。また、誤差の絶対値はもっとも大きくていくつですか。

(2) スイカ1個の重さをはかり、その小数第2位を四捨五入した近似値が 2.07kg になりました。このスイカの重さの真の値を a kg とするとき、a の範囲を不等号を使って表しなさい。

(3) 家から学校までの道のりの近似値は 1200m で、有効数字が 3けたであるとき、整数部分が 1けたの小数と 10 の何乗の積の形に表しなさい。

### 解答
(1) 5.65 ≤ a < 5.75、誤差の絶対値は最大 0.05
(2) 2.065 ≤ a < 2.075
(3) 1.20 × 10³ m

## 発展4 数の分類（配点 各5点×3 = 15点）

(1) 次の数を、自然数、整数（自然数でないもの）、有理数（整数でないもの）、無理数に分類しなさい。
  −√16, 101, √12, 3/7, −√(1/9)

(2) 次の数は、右の図（数の分類図）の①〜④のどこに入りますか。記号で答えなさい。
  ※ 図: 数全体 ⊃ 有理数 ⊃ 整数 ⊃ 自然数、無理数
  ① −√16　② 101　③ √12　④ 3/7

(3) 「すべての自然数は有理数である」は正しいか。理由も述べよ。

### 解答
(1) 自然数: 101　整数: −√16 = −4　有理数: 3/7, −√(1/9) = −1/3　無理数: √12
(2) ① 整数（−4）　② 自然数　③ 無理数　④ 有理数（整数でない）
(3) 正しい。自然数 n は n/1 と書けるので分数で表せる。よって有理数である。

## 発展5 有効数字の応用（配点 各5点×3 = 15点）

(1) ある品物の重さをはかり、その小数第3位を四捨五入した近似値が 2.07kg になりました。この近似値の有効数字を答えなさい。また、整数部分が1けたの小数と10の何乗の積の形に表しなさい。

(2) 51000m² の近似値について、有効数字が「5, 1」の2けたであるとき、a × 10ⁿ の形に表しなさい。

(3) 有効数字が「5, 1, 0」の3けたであるとき、同じ 51000m² を a × 10ⁿ の形に表しなさい。

### 解答
(1) 有効数字: 2, 0, 7 の3けた。2.07 × 10⁰ kg
(2) 5.1 × 10⁴ m²
(3) 5.10 × 10⁴ m²
