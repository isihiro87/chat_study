## √をふくむ式の除法

## √をふくむ式の値

Q37. √2 = 1.414 として、√2/2 の値を求めよ。
A37. 1.414/2 = 0.707

Q38. √3 = 1.732 として、√3/3 の値を求めよ。
A38. 1.732/3 ≈ 0.5773

Q39. √5 = 2.236 として、3/√5 の値を求めよ。
A39. 3√5/5 = 3 × 2.236/5 = 6.708/5 = 1.3416

Q40. √2 = 1.414 として、10/√2 の値を求めよ。
A40. 10√2/2 = 5√2 = 5 × 1.414 = 7.07

Q41. √6 = 2.449 として、√(2/3) の値を求めよ。
A41. √2/√3 = √6/3 = 2.449/3 ≈ 0.8163

Q42. √3 = 1.732 として、6/√3 の値を求めよ。
A42. 2√3 = 2 × 1.732 = 3.464

Q43. √2 = 1.414 として、√0.02 の値を求めよ。
A43. √(2/100) = √2/10 = 1.414/10 = 0.1414

Q44. 有理化とは何か？
A44. 分母にある √ をなくすこと（分母分子に同じ √ をかける）
