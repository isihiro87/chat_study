# 一問一答：平方根の意味と大小

## 平方根の定義

Q1. 2乗すると a になる数を何というか？
A1. a の平方根

Q2. 9の平方根を答えよ。
A2. ±3

Q3. 25の平方根を答えよ。
A3. ±5

Q4. 64の平方根を答えよ。
A4. ±8

Q5. 0.04の平方根を答えよ。
A5. ±0.2

Q6. 16/25 の平方根を答えよ。
A6. ±4/5

Q7. 0の平方根を答えよ。
A7. 0

Q8. 負の数の平方根は存在するか？
A8. 存在しない（2乗して負になる実数はない）

## 平方根の大小比較

Q26. √11 と √13 の大小を不等号で表せ。
A26. √11 < √13

Q27. −√5 と −√7 の大小を不等号で表せ。
A27. −√7 < −√5（負の数は絶対値が大きいほど小さい）

Q28. 6 と √37 の大小を不等号で表せ。
A28. 6 < √37（6² = 36 < 37）

Q29. √50 と 7 の大小を不等号で表せ。
A29. 7 < √50（7² = 49 < 50）

Q30. −4 と −√12 の大小を不等号で表せ。
A30. −4 < −√12（4² = 16 > 12 なので √16 > √12、よって −4 < −√12 は誤り。正しくは −√12 < −4 ではない。−4 = −√16, √16 > √12 なので −√16 < −√12、つまり −4 < −√12）

Q31. 1 < √a < 2 となる自然数 a をすべて求めよ。
A31. 2, 3（1² < a < 2² より 1 < a < 4）

Q32. 3 < √a < 4 となる自然数 a の個数を求めよ。
A32. 6個（10, 11, 12, 13, 14, 15）

## 有理数と無理数

Q33. 有理数とは何か？
A33. 整数 m と 0 でない整数 n を使って m/n の形に表せる数

Q34. 無理数とは何か？
A34. 分数で表せない数（√2, √3, π など）

Q35. √5 は有理数か無理数か？
A35. 無理数

Q36. √9 は有理数か無理数か？
A36. 有理数（√9 = 3）

Q37. √(4/9) は有理数か無理数か？
A37. 有理数（√(4/9) = 2/3）

Q38. 0.6, −5/8, √10, −√3 のうち無理数をすべて選べ。
A38. √10, −√3

Q39. 円周率 π は有理数か無理数か？
A39. 無理数

## 有限小数と循環小数

Q40. 有限小数とは何か？
A40. 小数点以下が有限で終わる小数（例: 0.125）

Q41. 循環小数とは何か？
A41. ある桁からさきは決まった数字がくり返される小数（例: 0.333...）

Q42. 5/8 を小数で表すと有限小数か循環小数か？
A42. 有限小数（0.625）

Q43. 1/3 を小数で表すと有限小数か循環小数か？
A43. 循環小数（0.333...）

Q44. 有限小数と循環小数は有理数と無理数のどちらか？
A44. 有理数

## 近似値と有効数字

Q45. 近似値とは何か？
A45. 測定して得られた値などのように、真の値に近い値のこと

Q46. 誤差とは何か？
A46. 近似値 − 真の値

Q47. 有効数字とは何か？
A47. 近似値を表す数で、意味のある数字のこと

Q48. 近似値 4507m を有効数字3けたで a × 10ⁿ の形に表せ。
A48. 4.51 × 10³ m

Q49. 近似値 2.96km の有効数字は何けたか？
A49. 3けた（2, 9, 6）

Q50. 有効数字が 4, 5 の2けたである近似値 4500 を a × 10ⁿ の形に表せ。
A50. 4.5 × 10³
