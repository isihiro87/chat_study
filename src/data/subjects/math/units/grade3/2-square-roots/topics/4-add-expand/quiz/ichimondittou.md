# 一問一答：根号の和差と展開

## 同じ√の和差

Q1. 3√2 + 5√2 を計算せよ。
A1. 8√2

Q2. 7√3 − 4√3 を計算せよ。
A2. 3√3

Q3. 2√5 + 6√5 を計算せよ。
A3. 8√5

Q4. 9√7 − √7 を計算せよ。
A4. 8√7

Q5. −8√3 + 5√3 を計算せよ。
A5. −3√3

Q6. 2√3 − 6√3 + 5√3 を計算せよ。
A6. √3

Q7. √2 + √3 は計算できるか？
A7. できない（異なる√はまとめられない）

A8. 2√2 + √2 = 3√2

A9. 3√2 − √2 = 2√2

A10. 2√3 + √3 = 3√3

A11. 3√3 − 2√3 = √3

Q12. √50 + √32 を計算せよ。
A12. 5√2 + 4√2 = 9√2

Q13. √75 − √12 を計算せよ。
A13. 5√3 − 2√3 = 3√3

Q14. √48 + √3 を計算せよ。
A14. 4√3 + √3 = 5√3

Q15. √45 − √20 を計算せよ。
A15. 3√5 − 2√5 = √5

Q16. √18 − √50 を計算せよ。
A16. 3√2 − 5√2 = −2√2

Q17. √72 + √8 − √32 を計算せよ。
A17. 6√2 + 2√2 − 4√2 = 4√2

## 有理化を含む和差

Q18. √12 − 6/√3 を計算せよ。
A18. 2√3 − 2√3 = 0

Q19. √2/3 + 5√6/6 を計算せよ。（ヒント: √(2/3) = √6/3）
A19. √6/3 + 5√6/6 = 2√6/6 + 5√6/6 = 7√6/6

Q20. √(5/3) + 5/√6 を計算せよ。（ヒント: √6を使う）
A20. √15/3 + 5√6/6

Q21. √54 − 12/(√6) を計算せよ。
A21. 3√6 − 12√6/6 = 3√6 − 2√6 = √6

## 分配法則

Q22. √2(√6 + √3) を計算せよ。
A22. √12 + √6 = 2√3 + √6

Q23. √3(√5 − √2) を計算せよ。
A23. √15 − √6

Q24. √5(√20 + √5) を計算せよ。
A24. √100 + √25 = 10 + 5 = 15

Q25. √3(√12 − √3) を計算せよ。
A25. √36 − √9 = 6 − 3 = 3

Q26. √2(√8 − 3) を計算せよ。
A26. √16 − 3√2 = 4 − 3√2

Q27. 2√3(√6 + √3) を計算せよ。
A27. 2√18 + 2√9 = 6√2 + 6

## 一般の展開

Q39. (√2 + 3)(√2 − 1) を計算せよ。
A39. 2 − √2 + 3√2 − 3 = −1 + 2√2

Q40. (√3 + 4)(√3 + 2) を計算せよ。
A40. 3 + 2√3 + 4√3 + 8 = 11 + 6√3

A41. 5 + 3√5 + √5 + 3 = 8 + 4√5

Q42. (√6 + 4)(−4 + √6) を計算せよ。
A42. (√6 + 4)(√6 − 4) = 6 − 16 = −10

## 式の値

Q43. x = √5 + 1 のとき、x² − 2x + 1 の値を求めよ。
A43. (x−1)² = (√5)² = 5

Q44. x = √3 − 2 のとき、x² + 4x + 4 の値を求めよ。
A44. (x+2)² = (√3)² = 3

Q45. x = √7 + √3, y = √7 − √3 のとき、x² − y² の値を求めよ。
A45. (x+y)(x−y) = 2√7 × 2√3 = 4√21

Q46. x = √5 + √3, y = √5 − √3 のとき、xy の値を求めよ。
A46. (√5+√3)(√5−√3) = 5 − 3 = 2

## 平方根の利用

Q47. 対角線の長さが 6cm の正方形の1辺の長さを求めよ。
A47. 1辺 = 6/√2 = 3√2 cm

Q48. 1辺が √2 cm の正方形の面積を求めよ。
A48. (√2)² = 2 cm²

Q49. 縦 6cm、横 14cm の長方形と面積が等しい正方形の1辺の長さを求めよ。
A49. 面積 = 84 → 1辺 = √84 = 2√21 cm

Q50. 半径が 6cm の円と半径が 10cm の円がある。面積の和と等しい面積の円の半径を求めよ。
A50. π × 6² + π × 10² = 136π → r² = 136 → r = √136 = 2√34 cm
