# 発展問題：y = ax² の値の増減と変域

## 発展1 変域から a の範囲を求める（各5点×4＝20点）

(1) y = ax² で −2 ≤ x ≤ 3 のとき、y の最大値が 36 以上になる a の範囲を求めなさい（a > 0）。

(2) y = ax² で 1 ≤ x ≤ 4 のとき、y の最大値が 32 以下になる a の範囲を求めなさい（a > 0）。

(3) y = ax²（a < 0）で −3 ≤ x ≤ 2 のとき、y の最小値が −45 以上になる a の範囲を求めなさい。

(4) y = ax² で −1 ≤ x ≤ 5 のとき、y のとりうる値が −50 以上になるとき、a の範囲を求めなさい（a < 0）。

### 解答
(1) a > 0 かつ x = 0 を含むので、最大値は x = 3 で y = 9a。9a ≥ 36 より a ≥ 4。答え: a ≥ 4
(2) x = 0 を含まないので端の値。最大値は x = 4 で y = 16a。16a ≤ 32 より a ≤ 2。答え: 0 < a ≤ 2
(3) a < 0 で x = 0 を含むので最小値は |x| が大きい x = −3 で y = 9a。9a ≥ −45 より a ≥ −5。答え: −5 ≤ a < 0
(4) a < 0 で x = 0 を含むので最小値は x = 5 で y = 25a。25a ≥ −50 より a ≥ −2。答え: −2 ≤ a < 0

## 発展2 2つの関数の変域比較（各5点×4＝20点）

(1) y = x² と y = 2x² について、−2 ≤ x ≤ 3 のときの y の変域をそれぞれ求め、共通範囲を答えなさい。

(2) y = x² と y = −x² について、1 ≤ x ≤ 2 のときの y の変域をそれぞれ求めなさい。y の変域が重なる部分はあるか。

(3) y = ax² と y = bx² （a > 0, b > 0, a ≠ b）で −1 ≤ x ≤ 2 のとき、2つの関数の y の最大値が等しくなる条件を求めなさい。

(4) y = 2x² と y = ax²（a < 0）で −1 ≤ x ≤ 3 のとき、y の変域の幅（最大値−最小値）が等しくなる a の値を求めなさい。

### 解答
(1) y = x²: 0 ≤ y ≤ 9、y = 2x²: 0 ≤ y ≤ 18。共通範囲: 0 ≤ y ≤ 9
(2) y = x²: 1 ≤ y ≤ 4、y = −x²: −4 ≤ y ≤ −1。重なる部分はない（一方は正、もう一方は負）
(3) x = 0 を含むので、最大値は x = 2 でそれぞれ 4a と 4b。等しくなるには 4a = 4b → a = b だが a ≠ b なので、等しくならない。つまり条件を満たす a, b の組は存在しない。
(4) y = 2x²: 最小値 0、最大値 18、幅 = 18。y = ax²（a < 0）: 最大値 0、最小値 9a（x = 3 で）、幅 = −9a。−9a = 18 → a = −2。答え: a = −2

## 発展3 変域と方程式の融合（各5点×3＝15点）

(1) y = ax² で −2 ≤ x ≤ 3 のとき、y の最大値と最小値の差が 27 であった。a > 0 として a の値を求めなさい。

(2) y = ax² で x の変域が −1 ≤ x ≤ t（t > 0）のとき、y の変域が 0 ≤ y ≤ 12 である。a = 3 のとき、t の値を求めなさい。

(3) y = 2x² で a ≤ x ≤ a + 3（a < 0, a + 3 > 0）のとき、y の最大値が 8 である。a の値を求めなさい。

### 解答
(1) a > 0 で x = 0 を含むので最小値は 0。最大値は x = 3 で y = 9a。差: 9a − 0 = 27 → a = 3。答え: a = 3
(2) a = 3 で x = 0 を含むので最小値は 0（条件と一致）。最大値は |t| と |−1| = 1 の大きい方で決まる。t > 0 より t > 1 のとき最大値 = 3t²。3t² = 12 → t² = 4 → t = 2（t > 0）。t = 2 > 1 なので条件を満たす。答え: t = 2
(3) a < 0, a + 3 > 0 より −3 < a < 0。x = 0 を含むので最小値は 0。最大値は |a| と |a + 3| の大きい方の x での値。|a| > |a + 3| のとき（−3 < a < −3/2）: 最大値 = 2a² = 8 → a² = 4 → a = −2。−3 < −2 < −3/2 なので適する。答え: a = −2

## 発展4 変域の逆問題（各5点×3＝15点）

(1) y = ax² で x の変域が p ≤ x ≤ p + 4（ただし p < 0, p + 4 > 0）のとき、y の最大値が 2a(p + 4)² で表されるための p の条件を求めなさい（a > 0）。

(2) y = x² で a ≤ x ≤ a + 2 のとき、y の最小値が 0 になるための a の範囲を求めなさい。

(3) y = −x² で a ≤ x ≤ a + 3 のとき、y の最大値が 0 になるための a の範囲を求めなさい。

### 解答
(1) a > 0 のとき x = 0 を含むので最小値は 0。最大値が x = p + 4 で実現するには |p + 4| ≥ |p| すなわち p + 4 ≥ −p → p ≥ −2。条件: −2 ≤ p < 0（p < 0 かつ p + 4 > 0 は自動的に満たされる）。答え: −2 ≤ p < 0
(2) y = x² の最小値が 0 になるには x = 0 が変域に含まれる必要がある。a ≤ 0 ≤ a + 2 より a ≤ 0 かつ a ≥ −2。答え: −2 ≤ a ≤ 0
(3) y = −x² で最大値が 0 になるには x = 0 が変域に含まれる必要がある。a ≤ 0 ≤ a + 3 より a ≤ 0 かつ a ≥ −3。答え: −3 ≤ a ≤ 0

## 発展5 実生活への応用（各5点×3＝15点）

(1) ボールを真上に投げ上げたとき、t 秒後の高さ h (m) が h = −5t² + 20t で表される。0 ≤ t ≤ 4 のとき、h の最大値を求めなさい。（ヒント: h = −5(t − 2)² + 20 と変形できる）

(2) ある物体の速さ v (m/s) が時刻 t (s) によって v = 3t² で表される。2 ≤ t ≤ 5 のとき、v の変域を求めなさい。

(3) 正方形の1辺の長さを x cm としたとき、面積 S = x² (cm²) である。辺の長さが 3 cm 以上 7 cm 以下のとき、面積 S の変域を求めなさい。

### 解答
(1) h = −5(t − 2)² + 20 より t = 2 で最大値 h = 20 (m)。答え: 最大値は 20 m
(2) x = 0 を含まないので端の値のみ。t = 2 で v = 12、t = 5 で v = 75。答え: 12 ≤ v ≤ 75
(3) x = 0 を含まないので端の値のみ。x = 3 で S = 9、x = 7 で S = 49。答え: 9 ≤ S ≤ 49
