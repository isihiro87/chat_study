# 大問形式：y=ax²と放物線

## 大問1 y=ax²の基本（配点 各3点×6 = 18点）

(1) 次のうち、y が x の2乗に比例するものをすべて選びなさい。
  ① y = 4x²　② y = 2x + 3　③ y = -x²　④ y = x³　⑤ y = 1/2 x²

(2) y は x の2乗に比例し、x = 3 のとき y = 36 である。y を x の式で表しなさい。

(3) y = 5x² で、x = -2 のときの y の値を求めなさい。

### 解答
(1) ①③⑤
(2) y = 4x²（36 = a × 9 → a = 4）
(3) y = 20（5 × (-2)² = 5 × 4 = 20）

## 大問2 比例定数を求める（配点 各4点×5 = 20点）

次のそれぞれの場合について、y を x の式で表しなさい。

(1) y は x の2乗に比例し、x = 2 のとき y = 12 である。
(2) y は x の2乗に比例し、x = -4 のとき y = -48 である。
(3) y は x の2乗に比例し、x = 5 のとき y = 10 である。
(4) y は x の2乗に比例し、x = -6 のとき y = 18 である。
(5) ボールが坂を転がるとき、転がり始めてから x 秒間に y cm 進む。3秒間で 3.6cm 進んだとき、y を x の式で表しなさい。

### 解答
(1) y = 3x²（12 = 4a → a = 3）
(2) y = -3x²（-48 = 16a → a = -3）
(3) y = 2/5 x²（10 = 25a → a = 2/5）
(4) y = 1/2 x²（18 = 36a → a = 1/2）
(5) y = 0.4x²（3.6 = 9a → a = 0.4）

## 大問3 グラフの特徴（配点 各3点×6 = 18点）

(1) 次の y = ax² のグラフについて、上に開くか下に開くか答えなさい。
  ① y = 2x²　② y = -3x²　③ y = 1/4 x²　④ y = -0.5x²

(2) 次のグラフを、開きが広い順に並べなさい。
  y = x²、y = 4x²、y = 1/3 x²、y = 2x²

### 解答
(1) ① 上に開く　② 下に開く　③ 上に開く　④ 下に開く
(2) y = 1/3 x², y = x², y = 2x², y = 4x²（|a| が小さい順）

## 大問4 値の表とグラフ（配点 各4点×6 = 24点）

(1) y = 2x² について、x = -3, -2, -1, 0, 1, 2, 3 のそれぞれに対する y の値を求めなさい。

(2) y = -x² について、x = -3, -2, -1, 0, 1, 2, 3 のそれぞれに対する y の値を求めなさい。

(3) 上の(1)(2)のグラフについて、次の問いに答えなさい。
  ① 対称軸は何か。
  ② 頂点の座標を答えなさい。
  ③ y = 2x² と y = -x² のグラフの共通点を1つ述べなさい。
  ④ y = 2x² と y = -x² のグラフの相違点を1つ述べなさい。

### 解答
(1) x = -3 → 18, x = -2 → 8, x = -1 → 2, x = 0 → 0, x = 1 → 2, x = 2 → 8, x = 3 → 18
(2) x = -3 → -9, x = -2 → -4, x = -1 → -1, x = 0 → 0, x = 1 → -1, x = 2 → -4, x = 3 → -9
(3) ① y 軸　② 原点(0, 0)　③ どちらも y 軸について対称で、頂点が原点にある　④ y = 2x² は上に開き、y = -x² は下に開く

## 大問5 放物線の式を求める（配点 各4点×5 = 20点）

(1) 放物線 y = ax² が点(3, -18) を通るとき、a の値を求めなさい。

(2) 放物線 y = ax² が点(-4, 8) を通るとき、y を x の式で表しなさい。

(3) 放物線 y = ax² が点(2, 6) を通るとき、x = -4 のときの y の値を求めなさい。

(4) 下に開く放物線 y = ax² が点(-3, -27) を通る。この放物線が点(6, b) を通るとき、b の値を求めなさい。

(5) 右の図で、放物線は y = ax² のグラフである。グラフが点(2, 2) を通るとき、a の値を求め、x = -6 のときの y の値を求めなさい。

### 解答
(1) a = -2（-18 = 9a → a = -2）
(2) y = 1/2 x²（8 = 16a → a = 1/2）
(3) y = 24（6 = 4a → a = 3/2、y = 3/2 × 16 = 24）
(4) b = -108（-27 = 9a → a = -3、y = -3 × 36 = -108）
(5) a = 1/2（2 = 4a → a = 1/2）、x = -6 のとき y = 1/2 × 36 = 18
