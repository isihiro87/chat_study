# 変化の割合 発展問題

## 問1: 変化の割合の応用計算

(1) $y = ax^2$ で $x$ が $-3$ から $5$ に変化するとき、変化の割合が $-4$ である。$a$ の値を求めよ。
→ $a(-3 + 5) = 2a = -4$ より $a = -2$。

(2) $y = 2x^2$ で $x$ が $p$ から $p + 3$ に変化するとき、変化の割合が $10$ である。$p$ の値を求めよ。
→ $2(p + (p + 3)) = 2(2p + 3) = 4p + 6 = 10$ より $4p = 4$、$p = 1$。

(3) $y = ax^2$ で $x$ が $-2$ から $t$ に変化するとき、変化の割合が $y = 5x - 1$ の変化の割合と等しい。$a = 1$ のとき $t$ の値を求めよ。
→ 一次関数の変化の割合は $5$。$1 \times (-2 + t) = t - 2 = 5$ より $t = 7$。

(4) $y = -3x^2$ で、変化の割合が $12$ になる区間 $x = p$ → $x = q$（$q - p = 2$）を求めよ。
→ $-3(p + q) = 12$ より $p + q = -4$。$q = p + 2$ を代入して $p + p + 2 = -4$ → $2p = -6$、$p = -3$、$q = -1$。区間は $x = -3$ → $x = -1$。

## 問2: 平均の速さの応用

ボールが斜面を転がり、$x$ 秒後の移動距離 $y$ cm が $y = 3x^2$ で表される。

(1) $2$ 秒後から $5$ 秒後までの平均の速さを求めよ。
→ 公式より $3(2 + 5) = 21$ cm/秒。

(2) $t$ 秒後から $(t + 1)$ 秒後までの平均の速さを $t$ を使って表せ。
→ $3(t + (t + 1)) = 3(2t + 1) = 6t + 3$ cm/秒。

(3) 上の平均の速さが $27$ cm/秒になるのは何秒後から何秒後までか？
→ $6t + 3 = 27$ より $6t = 24$、$t = 4$。$4$ 秒後から $5$ 秒後まで。

(4) $0$ 秒後から $t$ 秒後までの平均の速さが $12$ cm/秒になる $t$ を求めよ。
→ $3(0 + t) = 3t = 12$ より $t = 4$。

## 問3: 放物線と直線の交点と式の決定

(1) $y = x^2$ と直線 $y = mx + 3$ が点 $(-1, 1)$ で交わる。$m$ の値を求めよ。
→ $1 = -m + 3$ より $m = 2$。直線は $y = 2x + 3$。

(2) 上の直線ともう一つの交点の座標を求めよ。
→ $x^2 = 2x + 3$ → $x^2 - 2x - 3 = 0$ → $(x - 3)(x + 1) = 0$。もう一つの交点は $(3, 9)$。

(3) $y = 2x^2$ 上の2点 $A(1, 2)$、$B(3, 18)$ を通る直線の式を求めよ。
→ 傾き: $(18 - 2) \div (3 - 1) = 16 \div 2 = 8$。$y = 8x + b$ に $(1, 2)$ を代入: $2 = 8 + b$ → $b = -6$。直線: $y = 8x - 6$。

(4) $y = ax^2$ と $y = 6x - 5$ が $x = 1$ で交わる。$a$ の値と、もう一つの交点を求めよ。
→ $x = 1$ のとき $y = 6 - 5 = 1$。$a \times 1^2 = 1$ より $a = 1$。$x^2 = 6x - 5$ → $x^2 - 6x + 5 = 0$ → $(x - 1)(x - 5) = 0$。もう一つの交点は $(5, 25)$。

## 問4: △OABの面積（応用）

$y = x^2$ のグラフと直線 $\ell$ の交点を A, B とし、原点を O とする。

(1) $\ell: y = 2x + 3$ のとき、A, B の座標を求めよ。
→ $x^2 = 2x + 3$ → $x^2 - 2x - 3 = 0$ → $(x - 3)(x + 1) = 0$。$A(-1, 1)$, $B(3, 9)$。

(2) △OAB の面積を、y軸で2つの三角形に分けて求めよ。
→ 直線のy切片 $C(0, 3)$ で分割。△OAC: 底辺 OC $= 3$、高さ $= 1$（A の $|x|$）。面積 $= \dfrac{1}{2} \times 3 \times 1 = \dfrac{3}{2}$。△OCB: 底辺 OC $= 3$、高さ $= 3$（B の $x$）。面積 $= \dfrac{1}{2} \times 3 \times 3 = \dfrac{9}{2}$。合計: $\dfrac{3}{2} + \dfrac{9}{2} = 6$。

(3) $y = x^2$ と $y = -x + 6$ の交点を P, Q として、△OPQ の面積を求めよ。
→ $x^2 = -x + 6$ → $x^2 + x - 6 = 0$ → $(x + 3)(x - 2) = 0$。$P(-3, 9)$, $Q(2, 4)$。y切片 $R(0, 6)$ で分割。△OPR: $\dfrac{1}{2} \times 6 \times 3 = 9$。△OQR: $\dfrac{1}{2} \times 6 \times 2 = 6$。合計: $9 + 6 = 15$。

## 問5: 一次関数と y = ax² の総合比較

(1) 一次関数 $y = 3x + 1$ と $y = x^2$ について、$x$ が $1$ から $4$ に変化するときのそれぞれの変化の割合を求め、比較せよ。
→ 一次関数: 変化の割合 $= 3$（一定）。$y = x^2$: $1(1 + 4) = 5$。同じ区間でも $y = x^2$ のほうが変化の割合が大きい。

(2) $y = x^2$ と $y = 5x - 4$ が同じ変化の割合をもつ区間を求めよ。ただし $x$ が $a$ から $a + 2$ に変化する区間とする。
→ $y = x^2$ の変化の割合: $1 \times (a + (a + 2)) = 2a + 2$。これが $5$ と等しいとき、$2a + 2 = 5$ → $a = \dfrac{3}{2}$。区間は $x = \dfrac{3}{2}$ → $x = \dfrac{7}{2}$。

(3) 下の表の空欄を埋めよ。

|  | 一次関数 $y = ax + b$ | 関数 $y = ax^2$ |
|--|--|--|
| グラフの形 | (ア) | (イ) |
| 変化の割合 | (ウ) | (エ) |
| 対称性 | (オ) | (カ) |

→ (ア) 直線 (イ) 放物線 (ウ) 常に $a$ で一定 (エ) 区間によって変わる（$a(p+q)$）(オ) 対称性なし (カ) y軸について対称

(4) $y = 2x^2$ で、変化の割合が一次関数 $y = 8x + 1$ の変化の割合と等しくなるような区間 $x = p$ → $x = q$（ただし $q = p + 4$）を求めよ。
→ $2(p + p + 4) = 2(2p + 4) = 4p + 8 = 8$ より $4p = 0$、$p = 0$、$q = 4$。区間は $x = 0$ → $x = 4$。
