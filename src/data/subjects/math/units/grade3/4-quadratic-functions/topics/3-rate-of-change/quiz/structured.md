# 変化の割合 大問小問

## 大問1: 変化の割合の基本計算（配点 20点）

次の関数について、指定された区間の変化の割合を求めなさい。

(1) $y = x^2$ で $x$ が $1$ から $5$ に変化するとき（3点）
→ $y$ の増加量: $25 - 1 = 24$、$x$ の増加量: $5 - 1 = 4$。変化の割合 $= 24 \div 4 = 6$。

(2) $y = 2x^2$ で $x$ が $-3$ から $1$ に変化するとき（3点）
→ 公式: $a(p + q) = 2 \times (-3 + 1) = 2 \times (-2) = -4$。

(3) $y = -x^2$ で $x$ が $2$ から $5$ に変化するとき（3点）
→ 公式: $(-1)(2 + 5) = -7$。

(4) $y = 3x^2$ で $x$ が $-2$ から $4$ に変化するとき（4点）
→ 公式: $3(-2 + 4) = 3 \times 2 = 6$。

(5) $y = \dfrac{1}{2}x^2$ で $x$ が $-4$ から $2$ に変化するとき（4点）
→ 公式: $\dfrac{1}{2} \times (-4 + 2) = \dfrac{1}{2} \times (-2) = -1$。

(6) $y = -2x^2$ で $x$ が $-3$ から $-1$ に変化するとき（3点）
→ 公式: $(-2)(-3 + (-1)) = (-2)(-4) = 8$。

## 大問2: 変化の割合からの逆算（配点 20点）

(1) $y = ax^2$ で $x$ が $1$ から $3$ に変化するとき、変化の割合が $8$ である。$a$ の値を求めよ。（4点）
→ $a(1 + 3) = 4a = 8$ より $a = 2$。

(2) $y = ax^2$ で $x$ が $-2$ から $4$ に変化するとき、変化の割合が $-6$ である。$a$ の値を求めよ。（4点）
→ $a(-2 + 4) = 2a = -6$ より $a = -3$。

(3) $y = 2x^2$ で $x$ が $p$ から $5$ に変化するとき、変化の割合が $14$ である。$p$ の値を求めよ。（4点）
→ $2(p + 5) = 14$ より $p + 5 = 7$、$p = 2$。

(4) $y = -x^2$ で $x$ が $-3$ から $q$ に変化するとき、変化の割合が $5$ である。$q$ の値を求めよ。（4点）
→ $-1 \times (-3 + q) = 5$ より $-3 + q = -5$、$q = -2$。

(5) $y = ax^2$ で $x$ が $-1$ から $5$ に変化するとき、変化の割合が一次関数 $y = 12x + 3$ の変化の割合と等しい。$a$ の値を求めよ。（4点）
→ 一次関数の変化の割合は $12$。$a(-1 + 5) = 4a = 12$ より $a = 3$。

## 大問3: 平均の速さ（配点 20点）

ボールが斜面を転がり、$x$ 秒後の移動距離が $y$ cm で表されるとき、次の問いに答えなさい。

(1) $y = 2x^2$ のとき、$1$ 秒後から $3$ 秒後までの移動距離を求めよ。（3点）
→ $2 \times 9 - 2 \times 1 = 18 - 2 = 16$ cm。

(2) 上の問題で、$1$ 秒後から $3$ 秒後までの平均の速さを求めよ。（3点）
→ $16 \div (3 - 1) = 16 \div 2 = 8$ cm/秒。

(3) $y = 3x^2$ のとき、$2$ 秒後から $5$ 秒後までの平均の速さを求めよ。（4点）
→ 移動距離: $3 \times 25 - 3 \times 4 = 75 - 12 = 63$ cm。平均の速さ: $63 \div 3 = 21$ cm/秒。

(4) $y = 5x^2$ のとき、$0$ 秒後から $4$ 秒後までの平均の速さを求めよ。（4点）
→ 移動距離: $5 \times 16 - 0 = 80$ cm。平均の速さ: $80 \div 4 = 20$ cm/秒。

(5) $y = 4x^2$ のとき、$1$ 秒後から $3$ 秒後までの平均の速さと、$3$ 秒後から $5$ 秒後までの平均の速さをそれぞれ求め、比較せよ。（6点）
→ $1$→$3$: $4(1+3) = 16$ cm/秒。$3$→$5$: $4(3+5) = 32$ cm/秒。後の区間のほうが2倍速い。斜面を転がるにつれ加速していることがわかる。

## 大問4: 放物線と直線の交点（配点 20点）

(1) $y = x^2$ と $y = 3x$ の交点の座標を求めよ。（4点）
→ $x^2 = 3x$ → $x^2 - 3x = 0$ → $x(x - 3) = 0$ → $x = 0, 3$。交点: $(0, 0)$, $(3, 9)$。

(2) $y = x^2$ と $y = x + 6$ の交点の座標を求めよ。（4点）
→ $x^2 = x + 6$ → $x^2 - x - 6 = 0$ → $(x - 3)(x + 2) = 0$ → $x = 3, -2$。交点: $(3, 9)$, $(-2, 4)$。

(3) $y = 2x^2$ と $y = 6x - 4$ の交点の座標を求めよ。（4点）
→ $2x^2 = 6x - 4$ → $2x^2 - 6x + 4 = 0$ → $x^2 - 3x + 2 = 0$ → $(x - 1)(x - 2) = 0$ → $x = 1, 2$。交点: $(1, 2)$, $(2, 8)$。

(4) $y = x^2$ と $y = -2x + 3$ の交点の座標を求めよ。（4点）
→ $x^2 = -2x + 3$ → $x^2 + 2x - 3 = 0$ → $(x + 3)(x - 1) = 0$ → $x = -3, 1$。交点: $(-3, 9)$, $(1, 1)$。

(5) $y = x^2$ 上の2点 $(-1, 1)$ と $(3, 9)$ を通る直線の式を求めよ。（4点）
→ 傾き: $(9 - 1) \div (3 - (-1)) = 8 \div 4 = 2$。$y = 2x + b$ に $(3, 9)$ を代入: $9 = 6 + b$ → $b = 3$。直線: $y = 2x + 3$。

## 大問5: △OABの面積（配点 20点）

$y = x^2$ と $y = x + 2$ の交点を A, B とし、原点を O とする。ただし A の $x$ 座標が小さいほうとする。

(1) 交点 A, B の座標を求めよ。（4点）
→ $x^2 = x + 2$ → $x^2 - x - 2 = 0$ → $(x - 2)(x + 1) = 0$。$A(-1, 1)$, $B(2, 4)$。

(2) 直線 AB の式を確認し、y切片を求めよ。（3点）
→ $y = x + 2$ より y切片は $2$。直線は $y$ 軸と点 $C(0, 2)$ で交わる。

(3) △OAB の面積を求めよ。（ヒント: 直線のy切片Cで△OACと△OCBに分割する）（6点）
→ 点 $C(0, 2)$ で分割。OC $= 2$（y軸上の底辺）。△OAC: 底辺 OC $= 2$、高さ = A の $x$ 座標の絶対値 $= 1$。面積 $= \dfrac{1}{2} \times 2 \times 1 = 1$。△OCB: 底辺 OC $= 2$、高さ = B の $x$ 座標 $= 2$。面積 $= \dfrac{1}{2} \times 2 \times 2 = 2$。合計: $1 + 2 = 3$。

(4) $y = x^2$ と $y = 4x$ の交点を P, Q（P の $x$ 座標が小さいほう）として、原点 O と P, Q で作る三角形の面積を求めよ。（7点）
→ $x^2 = 4x$ → $x(x - 4) = 0$ → $P(0, 0)$, $Q(4, 16)$。P と O が同じ点なので三角形はできない（面積 $= 0$）。放物線と原点を通る直線の交点では、一方が原点と一致するため注意。
