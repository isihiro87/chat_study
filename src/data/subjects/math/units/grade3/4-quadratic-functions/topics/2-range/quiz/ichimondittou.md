# 一問一答：y = ax² の値の増減と変域

## 増加・減少の判定（a > 0 の場合）

Q1. y = x² で、x < 0 のとき x が増加すると y はどう変化するか？
A1. y は減少する

Q2. y = x² で、x > 0 のとき x が増加すると y はどう変化するか？
A2. y は増加する

Q3. y = 2x² で、x = 0 のとき y は最大値・最小値のどちらをとるか？
A3. 最小値（y = 0）をとる

Q4. y = 3x² のグラフで、x が負の範囲から正の範囲に変わるとき、y の変化の仕方はどうなるか？
A4. x = 0 を境に、減少から増加に変わる

Q5. y = x² で x = −3 のとき y の値はいくつか？
A5. y = 9

Q6. y = 2x² で x = −2 のとき y の値はいくつか？
A6. y = 8

## 増加・減少の判定（a < 0 の場合）

Q7. y = −x² で、x < 0 のとき x が増加すると y はどう変化するか？
A7. y は増加する

Q8. y = −x² で、x > 0 のとき x が増加すると y はどう変化するか？
A8. y は減少する

Q9. y = −2x² で、x = 0 のとき y は最大値・最小値のどちらをとるか？
A9. 最大値（y = 0）をとる

Q10. y = −3x² のグラフで、x が負の範囲から正の範囲に変わるとき、y の変化の仕方はどうなるか？
A10. x = 0 を境に、増加から減少に変わる

Q11. y = −x² で x = 4 のとき y の値はいくつか？
A11. y = −16

Q12. y = −2x² で x = −3 のとき y の値はいくつか？
A12. y = −18

## 変域の計算（x = 0 を含む場合）

Q13. y = x² で −2 ≤ x ≤ 3 のとき、y の最小値はいくつか？
A13. y = 0（x = 0 を含むので、a > 0 のとき最小値は 0）

Q14. y = x² で −2 ≤ x ≤ 3 のとき、y の最大値はいくつか？
A14. y = 9（|3| > |−2| なので x = 3 のとき最大）

Q15. y = x² で −2 ≤ x ≤ 3 のとき、y の変域を求めよ。
A15. 0 ≤ y ≤ 9

Q16. y = 2x² で −1 ≤ x ≤ 3 のとき、y の変域を求めよ。
A16. 0 ≤ y ≤ 18（x = 0 を含むので最小値は 0、x = 3 のとき y = 18 が最大値）

Q17. y = 3x² で −2 ≤ x ≤ 1 のとき、y の変域を求めよ。
A17. 0 ≤ y ≤ 12（x = 0 を含むので最小値は 0、x = −2 のとき y = 12 が最大値）

Q18. y = −x² で −3 ≤ x ≤ 2 のとき、y の最大値はいくつか？
A18. y = 0（x = 0 を含み、a < 0 なので最大値は 0）

Q19. y = −x² で −3 ≤ x ≤ 2 のとき、y の最小値はいくつか？
A19. y = −9（|−3| > |2| なので x = −3 のとき最小）

Q20. y = −x² で −3 ≤ x ≤ 2 のとき、y の変域を求めよ。
A20. −9 ≤ y ≤ 0

Q21. y = −2x² で −1 ≤ x ≤ 4 のとき、y の変域を求めよ。
A21. −32 ≤ y ≤ 0（x = 0 を含むので最大値は 0、x = 4 のとき y = −32 が最小値）

## 変域の計算（x = 0 を含まない場合）

Q22. y = x² で 1 ≤ x ≤ 3 のとき、y の変域を求めよ。
A22. 1 ≤ y ≤ 9（x = 0 を含まないので端の値だけ比べる）

Q23. y = x² で −4 ≤ x ≤ −1 のとき、y の変域を求めよ。
A23. 1 ≤ y ≤ 16（x = 0 を含まない。x = −1 で y = 1、x = −4 で y = 16）

Q24. y = −x² で 2 ≤ x ≤ 5 のとき、y の変域を求めよ。
A24. −25 ≤ y ≤ −4（x = 0 を含まない。x = 2 で y = −4、x = 5 で y = −25）

Q25. y = −2x² で −3 ≤ x ≤ −1 のとき、y の変域を求めよ。
A25. −18 ≤ y ≤ −2（x = −3 で y = −18、x = −1 で y = −2）

Q26. y = 3x² で 2 ≤ x ≤ 4 のとき、y の変域を求めよ。
A26. 12 ≤ y ≤ 48（x = 2 で y = 12、x = 4 で y = 48）

## 最大値・最小値の求め方

Q27. y = ax²（a > 0）で x の変域が 0 をまたぐとき、y の最小値を与える x の値は？
A27. x = 0（y の最小値は 0）

Q28. y = ax²（a < 0）で x の変域が 0 をまたぐとき、y の最大値を与える x の値は？
A28. x = 0（y の最大値は 0）

Q29. y = ax²（a > 0）で x の変域が 0 をまたぐとき、y の最大値を与えるのは x の変域のどこか？
A29. 変域の端の値のうち、絶対値が大きい方の x の値

Q30. y = x² で −5 ≤ x ≤ 2 のとき、y の最大値を与える x の値は？
A30. x = −5（|−5| > |2| なので）

Q31. y = −x² で −2 ≤ x ≤ 4 のとき、y の最小値を与える x の値は？
A31. x = 4（|4| > |−2| なので、a < 0 のとき |x| が大きい方で最小値をとる）

Q32. y = 2x² で −3 ≤ x ≤ 3 のとき、y の最大値はいくつか？
A32. y = 18（x = 3 でも x = −3 でも y = 18）

## グラフから変域を読み取る

Q33. y = x² のグラフは上に開くか下に開くか？
A33. 上に開く（a = 1 > 0 なので）

Q34. y = −x² のグラフは上に開くか下に開くか？
A34. 下に開く（a = −1 < 0 なので）

Q35. y = x² のグラフで、x = 0 のときの点を何というか？
A35. 頂点（原点 O）

Q36. y = ax² で変域を求めるとき、グラフを描くメリットは何か？
A36. x = 0 を含むかどうかや、最大値・最小値が視覚的にわかりやすくなる

Q37. y = x² で 0 ≤ x ≤ 2 のとき、y の変域を求めよ。
A37. 0 ≤ y ≤ 4

Q38. y = −x² で 0 ≤ x ≤ 3 のとき、y の変域を求めよ。
A38. −9 ≤ y ≤ 0

## 変域から a を逆算する問題

Q39. y = ax² で −2 ≤ x ≤ 4 のとき、y の最大値が 32 である。a の値を求めよ。
A39. a = 2（a > 0 のとき最大値は x = 4 で y = 16a = 32 より a = 2）

Q40. y = ax² で 1 ≤ x ≤ 3 のとき、y の最大値が 27 である。a > 0 とすると a の値は？
A40. a = 3（x = 3 で最大値 y = 9a = 27 より a = 3）

Q41. y = ax² で −3 ≤ x ≤ 1 のとき、y の最小値が −18 である。a の値を求めよ。
A41. a = −2（a < 0 のとき最小値は x = −3 で y = 9a = −18 より a = −2）

Q42. y = ax² で −2 ≤ x ≤ 3 のとき、y の最大値が 18 であった。a の値を求めよ。
A42. a = 2（a > 0 のとき、|3| > |−2| なので x = 3 で最大値 y = 9a = 18 より a = 2）

Q43. y = ax² で −1 ≤ x ≤ 2 のとき、y の最小値が −12 であった。a の値を求めよ。
A43. a = −3（a < 0 のとき、|2| > |−1| なので x = 2 で最小値 y = 4a = −12 より a = −3）

Q44. y = ax² で 2 ≤ x ≤ 5 のとき、y の最大値が 50 である。a > 0 とすると a の値は？
A44. a = 2（x = 5 で最大値 y = 25a = 50 より a = 2）

Q45. y = ax² で −4 ≤ x ≤ −1 のとき、y の変域が −48 ≤ y ≤ −3 のとき、a の値を求めよ。
A45. a = −3（x = −4 で y = 16a = −48 より a = −3、x = −1 で y = a = −3 と一致）
