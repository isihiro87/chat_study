# 一問一答：変化の割合

## 変化の割合の定義

Q1. 変化の割合を求める式を書け。
A1. 変化の割合 = yの増加量 ÷ xの増加量

Q2. y = ax² で、xがpからqに変化するときの変化の割合の公式を書け。
A2. a(p + q)

Q3. 一次関数 y = ax + b の変化の割合は常に何か？
A3. a（傾き）で一定

Q4. y = ax² の変化の割合は一定か？
A4. 一定ではない（区間によって変わる）

Q5. y = ax² のグラフが曲線になる理由を変化の割合で説明せよ。
A5. 区間ごとに変化の割合が異なるため、傾きが一定でなく曲線になる

## 変化の割合の計算（基本）

Q6. y = x² で xが1から3に変化するとき、yの増加量を求めよ。
A6. 9 − 1 = 8

Q7. y = x² で xが1から3に変化するとき、xの増加量を求めよ。
A7. 3 − 1 = 2

Q8. y = x² で xが1から3に変化するとき、変化の割合を求めよ。
A8. 8 ÷ 2 = 4

Q9. y = x² で xが2から4に変化するとき、変化の割合を求めよ。
A9. (16 − 4) ÷ (4 − 2) = 12 ÷ 2 = 6

Q10. y = x² で xが0から5に変化するとき、変化の割合を求めよ。
A10. (25 − 0) ÷ (5 − 0) = 25 ÷ 5 = 5

Q11. y = 2x² で xが1から4に変化するとき、変化の割合を求めよ。
A11. (32 − 2) ÷ (4 − 1) = 30 ÷ 3 = 10

Q12. y = 3x² で xが−2から1に変化するとき、変化の割合を求めよ。
A12. (3 − 12) ÷ (1 − (−2)) = −9 ÷ 3 = −3

Q13. y = −x² で xが1から4に変化するとき、変化の割合を求めよ。
A13. (−16 − (−1)) ÷ (4 − 1) = −15 ÷ 3 = −5

## 公式 a(p+q) の使い方

Q14. y = x² で xが1から3に変化するとき、公式 a(p+q) で変化の割合を求めよ。
A14. 1×(1 + 3) = 4

Q15. y = x² で xが−2から5に変化するとき、公式で変化の割合を求めよ。
A15. 1×(−2 + 5) = 3

Q16. y = 2x² で xが−1から3に変化するとき、公式で変化の割合を求めよ。
A16. 2×(−1 + 3) = 2×2 = 4

Q17. y = 3x² で xが2から5に変化するとき、公式で変化の割合を求めよ。
A17. 3×(2 + 5) = 3×7 = 21

Q18. y = −2x² で xが1から4に変化するとき、公式で変化の割合を求めよ。
A18. −2×(1 + 4) = −2×5 = −10

Q19. y = −x² で xが−3から1に変化するとき、公式で変化の割合を求めよ。
A19. −1×(−3 + 1) = −1×(−2) = 2

Q20. y = ½x² で xが2から6に変化するとき、公式で変化の割合を求めよ。
A20. ½×(2 + 6) = ½×8 = 4

Q21. y = −3x² で xが−4から−1に変化するとき、公式で変化の割合を求めよ。
A21. −3×(−4 + (−1)) = −3×(−5) = 15

Q22. y = 4x² で xが−2から3に変化するとき、公式で変化の割合を求めよ。
A22. 4×(−2 + 3) = 4×1 = 4

Q23. y = ax² で xが−1から5に変化するとき、変化の割合が12だった。aの値を求めよ。
A23. a(−1 + 5) = 4a = 12 より a = 3

Q24. y = ax² で xが2から4に変化するとき、変化の割合が−18だった。aの値を求めよ。
A24. a(2 + 4) = 6a = −18 より a = −3

## 平均の速さ

Q25. 平均の速さを求める式を書け。
A25. 平均の速さ = 移動距離 ÷ かかった時間

Q26. ボールが斜面を転がり、x秒後の移動距離が y = 2x²（cm）で表される。2秒後から5秒後までの移動距離を求めよ。
A26. 2×5² − 2×2² = 50 − 8 = 42 cm

Q27. 上の問題で、2秒後から5秒後までの平均の速さを求めよ。
A27. 42 ÷ (5 − 2) = 42 ÷ 3 = 14 cm/秒

Q28. y = 3x²（cm）で、1秒後から4秒後までの平均の速さを求めよ。
A28. (3×16 − 3×1) ÷ (4 − 1) = (48 − 3) ÷ 3 = 45 ÷ 3 = 15 cm/秒

Q29. y = x²（cm）で、0秒後から3秒後までの平均の速さを求めよ。
A29. (9 − 0) ÷ (3 − 0) = 9 ÷ 3 = 3 cm/秒

Q30. y = 4x²（cm）で、2秒後から4秒後までの平均の速さを求めよ。
A30. (4×16 − 4×4) ÷ (4 − 2) = (64 − 16) ÷ 2 = 48 ÷ 2 = 24 cm/秒

Q31. 平均の速さと変化の割合の関係を説明せよ。
A31. 距離が y = ax² のとき、平均の速さは変化の割合 a(p+q) と一致する

## 一次関数との比較

Q32. y = 3x + 2 の変化の割合を答えよ。
A32. 3（常に一定）

Q33. y = 3x² の xが1から2のときの変化の割合を求めよ。
A33. 3×(1 + 2) = 9

Q34. 一次関数のグラフが直線になる理由を変化の割合で説明せよ。
A34. 変化の割合が常に一定なので、傾きが変わらず直線になる

Q35. y = ax² のグラフで、xの値が大きくなるほど変化の割合はどうなるか（a > 0のとき）？
A35. 大きくなる（放物線の傾きがきつくなる）

Q36. 一次関数 y = ax + b と y = ax² で、増加・減少の仕方の違いを述べよ。
A36. 一次関数は一定の割合で増減する。y = ax² は区間によって増減の速さが変わる

## 放物線と直線の交点

Q37. y = x² と y = 2x の交点を求めるには、どのような連立方程式を解くか？
A37. x² = 2x として x² − 2x = 0 を解く

Q38. y = x² と y = 2x の交点の座標を求めよ。
A38. x² = 2x → x(x − 2) = 0 → x = 0, 2。交点は (0, 0) と (2, 4)

Q39. y = x² と y = x + 2 の交点の座標を求めよ。
A39. x² = x + 2 → x² − x − 2 = 0 → (x − 2)(x + 1) = 0 → x = 2, −1。交点は (2, 4) と (−1, 1)

Q40. y = 2x² と y = 4x の交点の座標を求めよ。
A40. 2x² = 4x → 2x(x − 2) = 0 → x = 0, 2。交点は (0, 0) と (2, 8)

## △OABの面積

Q41. 原点O、点A(−1, 1)、点B(2, 4) からなる△OABの面積を求める方法を説明せよ。
A41. 底辺と高さを設定するか、座標を使った面積の公式を利用する。長方形で囲んで引く方法もある。

Q42. 2点A(−1, 1)、B(3, 9)を通る直線の傾きを求めよ。
A42. (9 − 1) ÷ (3 − (−1)) = 8 ÷ 4 = 2

Q43. 上の直線の切片を求めよ（直線が y = 2x + b の形のとき）。
A43. 点A(−1, 1)を代入して 1 = 2×(−1) + b → b = 3。直線は y = 2x + 3

Q44. y = x² と y = 2x + 3 の交点のx座標を求めよ。
A44. x² = 2x + 3 → x² − 2x − 3 = 0 → (x − 3)(x + 1) = 0 → x = 3, −1

Q45. △OABの面積を求めるとき、底辺ABの長さの求め方を説明せよ。
A45. 2点間の距離の公式または座標からx座標の差・y座標の差を使って求める
