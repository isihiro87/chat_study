# 発展問題：y=ax²と放物線

## 発展1 グラフの読み取りと式の決定（配点 各5点×4 = 20点）

(1) 放物線 y = ax² が点(3, 6)を通る。この放物線上で x = -6 のときの y の値を求めなさい。

(2) 放物線 y = ax² のグラフ上に点 A(2, -8) がある。
  ① a の値を求めなさい。
  ② この放物線上で y = -32 となる x の値を求めなさい。

(3) 放物線 y = ax² が点(-4, 4)を通る。この放物線上にある点のうち、y 座標が 1 になる x の値をすべて求めなさい。

### 解答
(1) 6 = 9a → a = 2/3、y = 2/3 × 36 = 24
(2) ① -8 = 4a → a = -2　② -32 = -2x² → x² = 16 → x = ±4
(3) 4 = 16a → a = 1/4、1 = 1/4 x² → x² = 4 → x = ±2

## 発展2 2つの放物線の比較（配点 各5点×3 = 15点）

(1) y = 2x² と y = ax² のグラフについて、y = ax² の方が開きが広いとき、a の値の範囲を求めなさい（a > 0 とする）。

(2) y = ax² と y = -3x² のグラフが、ともに点 x = 2 での y の値の絶対値が等しいとき、a の値を求めなさい（a > 0 とする）。

(3) 2つの放物線 y = 1/2 x² と y = 2x² がある。x = k（k > 0）のとき、2つのグラフの y 座標の差が 6 になる k の値を求めなさい。

### 解答
(1) 0 < a < 2（|a| < 2 かつ a > 0）
(2) x = 2 のとき y = -3 × 4 = -12、|y| = 12。a × 4 = 12 → a = 3
(3) 2k² - 1/2 k² = 6 → 3/2 k² = 6 → k² = 4 → k = 2（k > 0）

## 発展3 放物線と座標（配点 各5点×4 = 20点）

(1) 放物線 y = ax² 上に点 A(3, 3) と点 B(-6, b) がある。a, b の値をそれぞれ求めなさい。

(2) 放物線 y = 1/3 x² 上に2点 A, B があり、A の x 座標は -3、B の x 座標は 6 である。
  ① A, B の座標を求めなさい。
  ② 線分 AB の中点の座標を求めなさい。

### 解答
(1) 3 = 9a → a = 1/3、b = 1/3 × 36 = 12
(2) ① A(-3, 3)、B(6, 12)　② 中点((-3+6)/2, (3+12)/2) = (3/2, 15/2)

## 発展4 実生活への応用（配点 各5点×4 = 20点）

(1) ボールが斜面を転がるとき、転がり始めてから x 秒間に y cm 進む関係が y = ax² で表される。2秒間で 6cm 進んだとき、次の問いに答えなさい。
  ① y を x の式で表しなさい。
  ② 4秒間で何 cm 進むか。
  ③ 24cm 進むのに何秒かかるか。

(2) 自動車がブレーキをかけてから止まるまでに進む距離（制動距離）は速さの2乗に比例する。時速 30km で 5m のとき、時速 60km での制動距離を求めなさい。

### 解答
(1) ① 6 = 4a → a = 3/2、y = 3/2 x²　② y = 3/2 × 16 = 24cm　③ 24 = 3/2 x² → x² = 16 → x = 4秒
(2) 5 = a × 900 → a = 1/180、y = 1/180 × 3600 = 20m

## 発展5 グラフの対称性と応用（配点 各5点×5 = 25点）

(1) 放物線 y = ax² のグラフが y 軸対称である理由を、式を使って説明しなさい。

(2) y = 2x² のグラフ上の点で、x 座標と y 座標が等しくなる点の座標をすべて求めなさい。

(3) y = ax² のグラフが点(2, -6)を通る。このグラフ上で y ≥ -24 となる x の範囲を求めなさい。

(4) 放物線 y = 1/4 x² 上にある点で、x 座標も y 座標も整数であるような点のうち、-6 ≤ x ≤ 6 の範囲にあるものをすべて求めなさい。

(5) 放物線 y = ax² が点(-2, 1)を通る。この放物線と x 軸で囲まれた部分に、y 座標が整数となる格子点（x, y 座標がともに整数の点）は原点を除いて何個あるか。ただし y ≤ 4 の範囲で考えなさい。

### 解答
(1) y = a(-x)² = a × x² = ax² となり、x を -x に置き換えても式が変わらないため、y 軸に関して対称である。
(2) y = 2x² に y = x を代入すると 2x² = x → 2x² - x = 0 → x(2x - 1) = 0 → x = 0 または x = 1/2。x = 0 のとき y = 0、x = 1/2 のとき y = 1/2。答え: (0, 0) と (1/2, 1/2)
(3) -6 = 4a → a = -3/2。-24 ≤ -3/2 x² → x² ≤ 16 → -4 ≤ x ≤ 4
(4) y = 1/4 x² で y が整数になるには x² が 4 の倍数、つまり x が偶数。x = -6 → y = 9、x = -4 → y = 4、x = -2 → y = 1、x = 0 → y = 0、x = 2 → y = 1、x = 4 → y = 4、x = 6 → y = 9。答え: (-6, 9), (-4, 4), (-2, 1), (0, 0), (2, 1), (4, 4), (6, 9)
(5) a = 1/4。y = 1/4 x² で y ≤ 4 のとき x² ≤ 16、-4 ≤ x ≤ 4。y = 1: x = ±2、y = 2: x² = 8（整数解なし）、y = 3: x² = 12（整数解なし）、y = 4: x = ±4。原点を除いて 4個。
