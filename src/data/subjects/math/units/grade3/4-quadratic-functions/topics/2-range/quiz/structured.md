# 大問形式：y = ax² の値の増減と変域

## 大問1 増加・減少の判定（各3点×6＝18点）

次の関数について、x が増加すると y はどう変化するか答えなさい。

(1) y = x² で x > 0 のとき
(2) y = x² で x < 0 のとき
(3) y = −x² で x > 0 のとき
(4) y = −x² で x < 0 のとき
(5) y = 3x² で x = 0 のとき、y は最大値・最小値のどちらをとるか
(6) y = −2x² で x = 0 のとき、y は最大値・最小値のどちらをとるか

### 解答
(1) y は増加する（a > 0, x > 0）
(2) y は減少する（a > 0, x < 0）
(3) y は減少する（a < 0, x > 0）
(4) y は増加する（a < 0, x < 0）
(5) 最小値（a > 0 なので x = 0 で最小値 0）
(6) 最大値（a < 0 なので x = 0 で最大値 0）

---

## 大問2 変域の計算 ― x = 0 を含む場合（各4点×6＝24点）

次の関数について、指定された x の変域における y の変域を求めなさい。

(1) y = x² で −3 ≤ x ≤ 2
(2) y = x² で −1 ≤ x ≤ 4
(3) y = 2x² で −2 ≤ x ≤ 1
(4) y = 3x² で −1 ≤ x ≤ 2
(5) y = −x² で −2 ≤ x ≤ 3
(6) y = −2x² で −3 ≤ x ≤ 1

### 解答
(1) 0 ≤ y ≤ 9（x = 0 で最小値 0、|−3| > |2| なので x = −3 で最大値 9）
(2) 0 ≤ y ≤ 16（x = 0 で最小値 0、|4| > |−1| なので x = 4 で最大値 16）
(3) 0 ≤ y ≤ 8（x = 0 で最小値 0、|−2| > |1| なので x = −2 で最大値 8）
(4) 0 ≤ y ≤ 12（x = 0 で最小値 0、|2| > |−1| なので x = 2 で最大値 12）
(5) −9 ≤ y ≤ 0（x = 0 で最大値 0、|3| > |−2| なので x = 3 で最小値 −9）
(6) −18 ≤ y ≤ 0（x = 0 で最大値 0、|−3| > |1| なので x = −3 で最小値 −18）

## 大問3 変域の計算 ― x = 0 を含まない場合（各4点×5＝20点）

次の関数について、指定された x の変域における y の変域を求めなさい。

(1) y = x² で 1 ≤ x ≤ 4
(2) y = x² で −5 ≤ x ≤ −2
(3) y = 2x² で 2 ≤ x ≤ 3
(4) y = −x² で 1 ≤ x ≤ 3
(5) y = −3x² で −4 ≤ x ≤ −1

### 解答
(1) 1 ≤ y ≤ 16（端の値のみ。x = 1 で y = 1、x = 4 で y = 16）
(2) 4 ≤ y ≤ 25（x = −2 で y = 4、x = −5 で y = 25）
(3) 8 ≤ y ≤ 18（x = 2 で y = 8、x = 3 で y = 18）
(4) −9 ≤ y ≤ −1（x = 3 で y = −9、x = 1 で y = −1）
(5) −48 ≤ y ≤ −3（x = −4 で y = −48、x = −1 で y = −3）

## 大問4 変域から a を求める（各5点×5＝25点）

次の条件を満たす a の値を求めなさい。

(1) y = ax² で 1 ≤ x ≤ 3 のとき、y の最大値が 18 である（a > 0）。
(2) y = ax² で −2 ≤ x ≤ 4 のとき、y の最大値が 48 である（a > 0）。
(3) y = ax² で −3 ≤ x ≤ 2 のとき、y の最小値が −27 である（a < 0）。
(4) y = ax² で −1 ≤ x ≤ 5 のとき、y の最大値が 50 である（a > 0）。
(5) y = ax² で −4 ≤ x ≤ −2 のとき、y の変域が −32 ≤ y ≤ −8 である。

### 解答
(1) a = 2（x = 3 で最大値 9a = 18 → a = 2）
(2) a = 3（x = 4 で最大値 16a = 48 → a = 3）
(3) a = −3（x = −3 で最小値 9a = −27 → a = −3）
(4) a = 2（x = 5 で最大値 25a = 50 → a = 2）
(5) a = −2（x = −4 で y = 16a = −32 → a = −2、x = −2 で y = 4a = −8 と一致）

## 大問5 総合問題（各3点×5 + 4点×2＝23点）

y = ax² について次の問いに答えなさい。

(1) a > 0 のとき、x が負の範囲で x の値が増加すると y はどう変化するか。（3点）
(2) a < 0 のとき、x が正の範囲で x の値が増加すると y はどう変化するか。（3点）
(3) y = 2x² で −3 ≤ x ≤ 5 のとき、y の変域を求めなさい。（3点）
(4) y = −x² で −4 ≤ x ≤ 1 のとき、y の変域を求めなさい。（3点）
(5) y = x² で −2 ≤ x ≤ 1 のとき、y = 4 となるのは x がいくつのときか。（3点）
(6) y = ax² で −2 ≤ x ≤ 3 のとき、y の変域が −27 ≤ y ≤ 0 である。a の値を求めなさい。（4点）
(7) y = ax² で −1 ≤ x ≤ 4 のとき、y の変域が p ≤ y ≤ 24 である。a と p の値を求めなさい。（4点）

### 解答
(1) y は減少する
(2) y は減少する
(3) 0 ≤ y ≤ 50（x = 0 を含むので最小値 0、|5| > |−3| なので x = 5 で最大値 50）
(4) −16 ≤ y ≤ 0（x = 0 を含むので最大値 0、|−4| > |1| なので x = −4 で最小値 −16）
(5) x = −2（−2 ≤ x ≤ 1 の範囲で x² = 4 を満たすのは x = −2）
(6) a = −3（a < 0、|3| > |−2| なので x = 3 で最小値 9a = −27 → a = −3）
(7) a = 3/2, p = 0（a > 0 のとき |4| > |−1| なので x = 4 で最大値 16a = 24 → a = 3/2。x = 0 を含むので最小値は y = 0。よって p = 0, a = 3/2）
