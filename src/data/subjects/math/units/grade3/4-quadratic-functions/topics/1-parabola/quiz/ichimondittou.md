# 一問一答：y=ax²と放物線

## y=ax²の定義と意味

Q1. y が x の2乗に比例するとき、y を x の式で表すとどうなるか？
A1. y = ax²（a ≠ 0）

Q2. y = ax² の a のことを何というか？
A2. 比例定数

Q3. y = ax² で、x が2倍になると y は何倍になるか？
A3. 4倍（2² = 4）

Q4. y = ax² で、x が3倍になると y は何倍になるか？
A4. 9倍（3² = 9）

Q5. y = ax² で、x が5倍になると y は何倍になるか？
A5. 25倍（5² = 25）

Q6. y = ax² の関係を「y は x の○○に比例する」という。○○に入る言葉は？
A6. 2乗

Q7. y = ax² のグラフの名前は何か？
A7. 放物線

## x²に比例するかどうかの判定

Q8. y = 5x² は y が x の2乗に比例しているか？
A8. している（a = 5）

Q9. y = 3x + 1 は y が x の2乗に比例しているか？
A9. していない（一次関数である）

Q10. y = -2x² は y が x の2乗に比例しているか？
A10. している（a = -2）

Q11. y = x²/4 は y が x の2乗に比例しているか？
A11. している（a = 1/4）

Q12. ボールが転がる実験で y = 0.4x² となった。y は x の2乗に比例しているか？
A12. している（a = 0.4）

Q13. 正方形の1辺の長さを x cm、面積を y cm² とするとき、y は x の2乗に比例するか？
A13. する（y = x² で a = 1）

Q14. 円の半径を r cm、面積を S cm² とするとき、S は r の2乗に比例するか？
A14. する（S = πr² で a = π）

## 比例定数 a の求め方

Q15. y は x の2乗に比例し、x = 2 のとき y = 8 である。比例定数 a を求めよ。
A15. a = 2（8 = a × 2² → 8 = 4a → a = 2）

Q16. y は x の2乗に比例し、x = 3 のとき y = 18 である。比例定数 a を求めよ。
A16. a = 2（18 = a × 3² → 18 = 9a → a = 2）

Q17. y は x の2乗に比例し、x = 4 のとき y = -32 である。比例定数 a を求めよ。
A17. a = -2（-32 = a × 4² → -32 = 16a → a = -2）

Q18. y は x の2乗に比例し、x = 5 のとき y = 50 である。y を x の式で表せ。
A18. y = 2x²（50 = a × 25 → a = 2）

Q19. y は x の2乗に比例し、x = -3 のとき y = 27 である。y を x の式で表せ。
A19. y = 3x²（27 = a × 9 → a = 3）

Q20. y は x の2乗に比例し、x = 6 のとき y = -12 である。y を x の式で表せ。
A20. y = -1/3 x²（-12 = a × 36 → a = -1/3）

## グラフの値の計算

Q21. y = 3x² で x = 2 のとき y の値を求めよ。
A21. y = 12（3 × 4 = 12）

Q22. y = -x² で x = 4 のとき y の値を求めよ。
A22. y = -16（-1 × 16 = -16）

Q23. y = 2x² で x = -3 のとき y の値を求めよ。
A23. y = 18（2 × 9 = 18）

Q24. y = -3x² で x = -2 のとき y の値を求めよ。
A24. y = -12（-3 × 4 = -12）

Q25. y = 1/2 x² で x = 6 のとき y の値を求めよ。
A25. y = 18（1/2 × 36 = 18）

Q26. y = -1/4 x² で x = -4 のとき y の値を求めよ。
A26. y = -4（-1/4 × 16 = -4）

Q27. y = x² で x = 0 のとき y の値を求めよ。
A27. y = 0

## グラフの特徴（開き方・対称軸・頂点）

Q28. y = ax² のグラフの対称軸は何か？
A28. y 軸

Q29. y = ax² のグラフの頂点はどこか？
A29. 原点（0, 0）

Q30. a > 0 のとき、y = ax² のグラフはどちら向きに開くか？
A30. 上に開く（U字型）

Q31. a < 0 のとき、y = ax² のグラフはどちら向きに開くか？
A31. 下に開く（∩字型）

Q32. y = 5x² のグラフは上に開くか下に開くか？
A32. 上に開く（a = 5 > 0）

Q33. y = -4x² のグラフは上に開くか下に開くか？
A33. 下に開く（a = -4 < 0）

Q34. y = 1/3 x² のグラフは上に開くか下に開くか？
A34. 上に開く（a = 1/3 > 0）

Q35. y = -0.5x² のグラフは上に開くか下に開くか？
A35. 下に開く（a = -0.5 < 0）

Q36. y = ax² のグラフが上に開くための条件は？
A36. a > 0

Q37. y = ax² のグラフが下に開くための条件は？
A37. a < 0

## |a|の大小とグラフの幅

Q38. |a| が大きいほどグラフの開き方はどうなるか？
A38. 細くなる（急になる）

Q39. |a| が小さいほどグラフの開き方はどうなるか？
A39. 広くなる（ゆるやかになる）

Q40. y = x² と y = 3x² ではどちらのグラフが細いか？
A40. y = 3x²（|3| > |1|）

Q41. y = 2x² と y = 1/2 x² ではどちらのグラフが広いか？
A41. y = 1/2 x²（|1/2| < |2|）

Q42. y = -x² と y = -5x² ではどちらのグラフが細いか？
A42. y = -5x²（|-5| > |-1|）

Q43. y = x², y = 2x², y = 1/3 x² を開きの広い順に並べよ。
A43. y = 1/3 x², y = x², y = 2x²

## 放物線の式を求める

Q44. 放物線が点(1, 3)を通るとき、y = ax² の式を求めよ。
A44. y = 3x²（3 = a × 1 → a = 3）

Q45. 放物線が点(2, -8)を通るとき、y = ax² の式を求めよ。
A45. y = -2x²（-8 = a × 4 → a = -2）

Q46. 放物線が点(-3, 9)を通るとき、y = ax² の式を求めよ。
A46. y = x²（9 = a × 9 → a = 1）

Q47. 放物線が点(4, 8)を通るとき、y = ax² の式を求めよ。
A47. y = 1/2 x²（8 = a × 16 → a = 1/2）

Q48. 放物線が点(-2, -12)を通るとき、y = ax² の式を求めよ。
A48. y = -3x²（-12 = a × 4 → a = -3）

Q49. 放物線が点(6, 12)を通るとき、y = ax² の式を求めよ。
A49. y = 1/3 x²（12 = a × 36 → a = 1/3）

Q50. 放物線が点(-5, -50)を通るとき、y = ax² の式を求めよ。
A50. y = -2x²（-50 = a × 25 → a = -2）
