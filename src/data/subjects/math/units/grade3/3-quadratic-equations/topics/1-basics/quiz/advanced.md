# 発展問題：二次方程式の基本

## 発展1 係数に文字を含む問題（配点 各5点×3 = 15点）

(1) x² + ax − 8 = 0 の解の1つが x = 2 であるとき、a の値を求めなさい。

(2) x² − 6x + k = 0 の解の1つが x = 1 であるとき、k の値を求めなさい。また、もう1つの解も求めなさい。

(3) ax² − 12 = 0 の解の1つが x = 2 であるとき、a の値を求めなさい。また、もう1つの解も求めなさい。

### 解答
(1) x = 2 を代入: 4 + 2a − 8 = 0, 2a = 4 より a = 2
(2) x = 1 を代入: 1 − 6 + k = 0, k = 5。x² − 6x + 5 = 0 を因数分解すると (x − 1)(x − 5) = 0 よりもう1つの解は x = 5
(3) x = 2 を代入: 4a − 12 = 0, a = 3。3x² − 12 = 0, x² = 4 よりもう1つの解は x = −2

## 発展2 解の条件からの逆算（配点 各5点×4 = 20点）

(1) (x − a)² = b の解が x = 3 ± √7 であるとき、a と b の値を求めなさい。

(2) 二次方程式 px² − 36 = 0 の解が x = ±3 であるとき、p の値を求めなさい。

(3) (x + m)² = n の解が x = −4 ± √10 であるとき、m と n の値を求めなさい。

(4) ax² + c = 0 の解が x = ±2√5 であるとき、a = 3 のときの c の値を求めなさい。

### 解答
(1) (x − a)² = b の解は x = a ± √b なので、a = 3, b = 7
(2) x = ±3 を代入: 9p − 36 = 0 より p = 4
(3) (x + m)² = n の解は x = −m ± √n なので、m = 4, n = 10
(4) x = ±2√5 より x² = 20。3×20 + c = 0 より c = −60

## 発展3 √を含む解の問題（配点 各5点×3 = 15点）

(1) 2x² − 10 = 0 を解き、解を簡単な形にしなさい。

(2) 3x² = 24 を解き、解を a√b の形（b はできるだけ小さい自然数）で表しなさい。

(3) (x − 1)² = 12 を解き、解を簡単な形にしなさい。

### 解答
(1) x² = 5 より x = ±√5
(2) x² = 8 より x = ±√8 = ±2√2
(3) x − 1 = ±√12 = ±2√3 より x = 1 ± 2√3

## 発展4 二次方程式の解の個数と判別（配点 各5点×4 = 20点）

(1) x² = k について、k の値によって解の個数がどう変わるか説明しなさい。

(2) (x − 3)² = k について、解が2つ存在するための k の条件を答えなさい。

(3) x² + 4 = 0 に実数の解が存在しない理由を説明しなさい。

(4) (x + a)² = 0 の解の個数を答え、その解を a を使って表しなさい。

### 解答
(1) k > 0 のとき解は2つ（x = ±√k）、k = 0 のとき解は1つ（x = 0）、k < 0 のとき解はない
(2) k > 0
(3) x² = −4 となるが、x² は常に0以上なので、−4（負の数）になることはない。よって実数の解は存在しない
(4) 解は1つで、x = −a

## 発展5 応用的な ax² = b, (x + m)² = n（配点 各6点×3 = 18点）

(1) 4(x + 1)² = 49 を解きなさい。

(2) 2(x − 3)² − 18 = 0 を解きなさい。

(3) ある正方形の面積が 50 cm² である。この正方形の1辺の長さを求めなさい。（ただし、長さは正の値とする）

### 解答
(1) (x + 1)² = 49/4, x + 1 = ±7/2 より x = 5/2 または x = −9/2
(2) 2(x − 3)² = 18, (x − 3)² = 9, x − 3 = ±3 より x = 6 または x = 0
(3) 1辺を x cm とすると x² = 50。x > 0 より x = √50 = 5√2。よって1辺の長さは 5√2 cm
