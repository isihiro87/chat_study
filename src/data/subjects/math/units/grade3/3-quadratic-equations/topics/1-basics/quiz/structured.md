# 大問形式：二次方程式の基本

## 大問1 二次方程式の判別（配点 各4点×5 = 20点）

次の方程式のうち、二次方程式であるものをすべて選び、二次方程式であるものについては a, b, c の値を答えなさい（標準形 ax² + bx + c = 0 として）。

(1) x² − 6x + 8 = 0
(2) 3x + 7 = 0
(3) 5x² − 20 = 0
(4) x(x + 4) = 0
(5) x³ − x = 0

### 解答
(1) 二次方程式。a = 1, b = −6, c = 8
(2) 二次方程式ではない（一次方程式）
(3) 二次方程式。a = 5, b = 0, c = −20
(4) 二次方程式。展開すると x² + 4x = 0 なので a = 1, b = 4, c = 0
(5) 二次方程式ではない（三次方程式）

## 大問2 ax² = b 型の計算（配点 各4点×5 = 20点）

次の二次方程式を解きなさい。

(1) x² = 49
(2) 3x² = 75
(3) 2x² − 32 = 0
(4) 5x² = 15
(5) 4x² − 9 = 0

### 解答
(1) x = ±7
(2) x² = 25 より x = ±5
(3) 2x² = 32, x² = 16 より x = ±4
(4) x² = 3 より x = ±√3
(5) 4x² = 9, x² = 9/4 より x = ±3/2

## 大問3 (x + m)² = n 型の計算（配点 各4点×5 = 20点）

次の二次方程式を解きなさい。

(1) (x + 2)² = 9
(2) (x − 4)² = 16
(3) (x − 3)² = 5
(4) (x + 1)² − 8 = 0
(5) (x − 6)² = 18

### 解答
(1) x + 2 = ±3 より x = 1 または x = −5
(2) x − 4 = ±4 より x = 8 または x = 0
(3) x − 3 = ±√5 より x = 3 ± √5
(4) (x + 1)² = 8, x + 1 = ±2√2 より x = −1 ± 2√2
(5) x − 6 = ±3√2 より x = 6 ± 3√2

## 大問4 解の代入と確認（配点 各4点×5 = 20点）

(1) x = 3 は x² − 5x + 6 = 0 の解であるか。代入して確かめなさい。

(2) x = −2 は x² + 3x + 2 = 0 の解であるか。代入して確かめなさい。

(3) x² − 2x − 8 = 0 の解の1つは x = 4 である。もう1つの解を求めなさい。

(4) 2x² − 50 = 0 の2つの解をそれぞれ求め、代入して確かめなさい。

(5) (x − 1)² = 3 の解を求め、x = 1 + √3 が解であることを代入して確かめなさい。

### 解答
(1) 3² − 5×3 + 6 = 9 − 15 + 6 = 0 ✓ よって解である
(2) (−2)² + 3×(−2) + 2 = 4 − 6 + 2 = 0 ✓ よって解である
(3) x = 4 を代入: 16 − 8 − 8 = 0 ✓。式を因数分解すると (x − 4)(x + 2) = 0 よりもう1つの解は x = −2
(4) x² = 25 より x = ±5。確認: 2×25 − 50 = 50 − 50 = 0 ✓（x = 5 も x = −5 も成立）
(5) x = 1 ± √3。確認: (1 + √3 − 1)² = (√3)² = 3 ✓

## 大問5 混合問題（配点 各4点×5 = 20点）

次の二次方程式を解きなさい。

(1) x² = 12

(2) 3x² − 12 = 0

(3) (x + 5)² = 49

(4) (x − 2)² − 10 = 0

(5) ある二次方程式の解が x = ±√11 であるとき、この二次方程式を x² = k の形で表しなさい。

### 解答
(1) x = ±2√3
(2) x² = 4 より x = ±2
(3) x + 5 = ±7 より x = 2 または x = −12
(4) (x − 2)² = 10, x − 2 = ±√10 より x = 2 ± √10
(5) x = ±√11 より x² = 11
