# 一問一答：因数分解による二次方程式の解法

## A×B=0の原則の確認

Q: $A \times B = 0$ のとき、何が言えるか。
A: $A = 0$ または $B = 0$（少なくともどちらか一方が0）。

Q: $(x - 5)(x + 2) = 0$ のとき、どんな2つの式が成り立つか。
A: $x - 5 = 0$ または $x + 2 = 0$。$A \times B = 0$ の原則を使う。

Q: $x(x - 7) = 0$ のとき、$x = 0$ は解になるか。
A: なる。$A \times B = 0$ で $A = x = 0$ も解の1つ。

Q: 二次方程式を因数分解で解くとき、最初にやるべきことは何か。
A: 右辺を0にする（移項して整理する）。

Q: $A \times B = 0$ の原則を使うために、左辺はどんな形にする必要があるか。
A: 因数（かけ算の形）に分解された形。積の形にしてから各因数 $= 0$ とする。

## (x-a)(x-b)=0 型の基本

Q: $(x - 4)(x + 1) = 0$ を解け。
A: $x = 4, x = -1$。$x - 4 = 0$ → $x = 4$、$x + 1 = 0$ → $x = -1$。

Q: $(x + 3)(x + 5) = 0$ を解け。
A: $x = -3, x = -5$。$x + 3 = 0$ → $x = -3$、$x + 5 = 0$ → $x = -5$。

Q: $(x - 7)(x + 2) = 0$ を解け。
A: $x = 7, x = -2$。$x - 7 = 0$ → $x = 7$、$x + 2 = 0$ → $x = -2$。

Q: $(x + 6)(x - 6) = 0$ を解け。
A: $x = -6, x = 6$。$x + 6 = 0$ → $x = -6$、$x - 6 = 0$ → $x = 6$。

## x(x-a)=0 型（x=0を忘れない）

Q: $x(x - 3) = 0$ を解け。
A: $x = 0, x = 3$。$x = 0$ を忘れないこと！

Q: $x(x + 8) = 0$ を解け。
A: $x = 0, x = -8$。$x = 0$ も解になる。

Q: $x^2 - 6x = 0$ を解け。
A: $x = 0, x = 6$。$x(x - 6) = 0$ と因数分解してから解く。

Q: $x^2 + 4x = 0$ を解け。
A: $x = 0, x = -4$。$x(x + 4) = 0$ と因数分解。

Q: $x^2 - 6x = 0$ で両辺を $x$ で割って $x - 6 = 0$ → $x = 6$ としてよいか。
A: ダメ。$x = 0$ の解が消えてしまう。必ず因数分解してから解く。

## x²+bx+c=0 型の因数分解→解

Q: $x^2 - 5x + 6 = 0$ を解け。
A: $x = 2, x = 3$。$(x - 2)(x - 3) = 0$。かけて $6$、足して $-5$ → $-2$ と $-3$。

Q: $x^2 + x - 12 = 0$ を解け。
A: $x = 3, x = -4$。$(x - 3)(x + 4) = 0$。かけて $-12$、足して $1$ → $-3$ と $4$。

Q: $x^2 - 7x + 10 = 0$ を解け。
A: $x = 2, x = 5$。$(x - 2)(x - 5) = 0$。かけて $10$、足して $-7$ → $-2$ と $-5$。

Q: $x^2 + 3x - 10 = 0$ を解け。
A: $x = 2, x = -5$。$(x - 2)(x + 5) = 0$。かけて $-10$、足して $3$ → $-2$ と $5$。

Q: $x^2 - 8x + 15 = 0$ を解け。
A: $x = 3, x = 5$。$(x - 3)(x - 5) = 0$。かけて $15$、足して $-8$ → $-3$ と $-5$。

Q: $x^2 + 9x + 20 = 0$ を解け。
A: $x = -4, x = -5$。$(x + 4)(x + 5) = 0$。かけて $20$、足して $9$ → $4$ と $5$。

Q: $x^2 - 2x - 15 = 0$ を解け。
A: $x = 5, x = -3$。$(x - 5)(x + 3) = 0$。かけて $-15$、足して $-2$ → $-5$ と $3$。

Q: $x^2 + 6x + 5 = 0$ を解け。
A: $x = -1, x = -5$。$(x + 1)(x + 5) = 0$。かけて $5$、足して $6$ → $1$ と $5$。

Q: $x^2 - x - 20 = 0$ を解け。
A: $x = 5, x = -4$。$(x - 5)(x + 4) = 0$。かけて $-20$、足して $-1$ → $-5$ と $4$。

## 共通因数でくくり出す（ax²+bx=0 型）

Q: $2x^2 - 5x = 0$ を解け。
A: $x = 0, x = \dfrac{5}{2}$。$x(2x - 5) = 0$。$x = 0$ または $2x - 5 = 0$。

Q: $3x^2 + 12x = 0$ を解け。
A: $x = 0, x = -4$。$3x(x + 4) = 0$。$x = 0$ または $x + 4 = 0$。

Q: $5x^2 - 10x = 0$ を解け。
A: $x = 0, x = 2$。$5x(x - 2) = 0$。$x = 0$ または $x - 2 = 0$。

Q: $4x^2 + 6x = 0$ を解け。
A: $x = 0, x = -\dfrac{3}{2}$。$2x(2x + 3) = 0$。$x = 0$ または $2x + 3 = 0$。

## 完全平方式（重解）

Q: 重解とは何か。
A: 二次方程式の2つの解が同じ値になること。例えば $(x - 3)^2 = 0$ → $x = 3$（重解）。

Q: $x^2 - 2x + 1 = 0$ を解け。
A: $x = 1$（重解）。$(x - 1)^2 = 0$。$x^2 - 2 \times 1 \times x + 1^2$ の形。

Q: $x^2 + 6x + 9 = 0$ を解け。
A: $x = -3$（重解）。$(x + 3)^2 = 0$。$x^2 + 2 \times 3 \times x + 3^2$ の形。

Q: $x^2 - 10x + 25 = 0$ を解け。
A: $x = 5$（重解）。$(x - 5)^2 = 0$。$x^2 - 2 \times 5 \times x + 5^2$ の形。

Q: $x^2 + 8x + 16 = 0$ を解け。
A: $x = -4$（重解）。$(x + 4)^2 = 0$。$x^2 + 2 \times 4 \times x + 4^2$ の形。

Q: $x^2 - 14x + 49 = 0$ を解け。
A: $x = 7$（重解）。$(x - 7)^2 = 0$。$x^2 - 2 \times 7 \times x + 7^2$ の形。

## 差の平方（x²-a²=0 型）

Q: $x^2 - 9 = 0$ を解け。
A: $x = 3, x = -3$。$(x + 3)(x - 3) = 0$。$x^2 - 3^2$ の形。

Q: $x^2 - 25 = 0$ を解け。
A: $x = 5, x = -5$。$(x + 5)(x - 5) = 0$。$x^2 - 5^2$ の形。

Q: $x^2 - 16 = 0$ を解け。
A: $x = 4, x = -4$。$(x + 4)(x - 4) = 0$。$x^2 - 4^2$ の形。

Q: $x^2 - 49 = 0$ を解け。
A: $x = 7, x = -7$。$(x + 7)(x - 7) = 0$。$x^2 - 7^2$ の形。

Q: $x^2 = 36$ を因数分解で解け。
A: $x = 6, x = -6$。$x^2 - 36 = 0$ → $(x + 6)(x - 6) = 0$。

## 係数をそろえる（両辺を割る・符号を変える）

Q: $-x^2 + 5x - 6 = 0$ を解け。
A: $x = 2, x = 3$。両辺に $-1$ をかけて $x^2 - 5x + 6 = 0$ → $(x - 2)(x - 3) = 0$。

Q: $-x^2 - 2x + 15 = 0$ を解け。
A: $x = 3, x = -5$。両辺に $-1$ をかけて $x^2 + 2x - 15 = 0$ → $(x + 5)(x - 3) = 0$。

Q: $2x^2 - 10x + 12 = 0$ を解け。
A: $x = 2, x = 3$。両辺を $2$ で割って $x^2 - 5x + 6 = 0$ → $(x - 2)(x - 3) = 0$。

Q: $-4x^2 + 8x - 4 = 0$ を解け。
A: $x = 1$（重解）。$-4$ で割って $x^2 - 2x + 1 = 0$ → $(x - 1)^2 = 0$。

Q: $3x^2 - 3x - 18 = 0$ を解け。
A: $x = 3, x = -2$。$3$ で割って $x^2 - x - 6 = 0$ → $(x - 3)(x + 2) = 0$。

## 展開してから因数分解

Q: $(x - 3)(x - 4) = 2$ を解け。
A: $x = 2, x = 5$。展開して $x^2 - 7x + 12 = 2$ → $x^2 - 7x + 10 = 0$ → $(x - 2)(x - 5) = 0$。

Q: $(x + 1)(x - 2) = 4$ を解け。
A: $x = 3, x = -2$。展開して $x^2 - x - 2 = 4$ → $x^2 - x - 6 = 0$ → $(x - 3)(x + 2) = 0$。

Q: $(x + 2)(x - 1) = 4$ を解け。
A: $x = 2, x = -3$。展開して $x^2 + x - 2 = 4$ → $x^2 + x - 6 = 0$ → $(x + 3)(x - 2) = 0$。

Q: $(x - 1)^2 = 9$ を因数分解で解け。
A: $x = 4, x = -2$。$(x - 1)^2 - 9 = 0$ → $\{(x-1)+3\}\{(x-1)-3\} = 0$ → $(x + 2)(x - 4) = 0$。

Q: $x(x + 5) = 14$ を解け。
A: $x = 2, x = -7$。展開して $x^2 + 5x = 14$ → $x^2 + 5x - 14 = 0$ → $(x - 2)(x + 7) = 0$。

Q: $(x + 3)^2 = 4x + 12$ を解け。
A: $x = -3, x = 1$。展開して $x^2 + 6x + 9 = 4x + 12$ → $x^2 + 2x - 3 = 0$ → $(x + 3)(x - 1) = 0$。
