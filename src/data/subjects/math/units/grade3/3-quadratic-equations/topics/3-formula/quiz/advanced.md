# 発展問題：平方完成と解の公式

## 発展1 解の公式の導出過程の穴埋め（配点 各4点×5 = 20点）

解の公式を導出する過程の空欄を埋めなさい。

ax² + bx + c = 0　（a ≠ 0）

(1) 両辺を a で割ると：x² + (b/a)x + c/a = 0

(2) c/a を移項すると：x² + (b/a)x = 【　ア　】

(3) 左辺を平方完成：(x + 【　イ　】)² − (b/2a)² = −c/a

(4) 整理すると：(x + b/2a)² = 【　ウ　】

(5) 両辺の平方根をとると：x + b/2a = ± √(【　ウ　】)
    よって x = 【　エ　】

### 解答
ア: −c/a
イ: b/(2a)
ウ: (b²−4ac) / (4a²)
エ: (−b ± √(b²−4ac)) / (2a)

## 発展2 複雑な係数の問題（配点 各5点×3 = 15点）

次の二次方程式を解の公式で解きなさい。

(1) 4x² − 3x − 2 = 0

(2) 5x² + 6x − 1 = 0

(3) 3x² − 10x + 2 = 0

### 解答
(1) a=4, b=−3, c=−2
    x = (3 ± √(9+32)) / 8 = (3 ± √41) / 8

(2) a=5, b=6, c=−1
    x = (−6 ± √(36+20)) / 10 = (−6 ± √56) / 10 = (−6 ± 2√14) / 10 = (−3 ± √14) / 5

(3) a=3, b=−10, c=2
    x = (10 ± √(100−24)) / 6 = (10 ± √76) / 6 = (10 ± 2√19) / 6 = (5 ± √19) / 3

## 発展3 整理してから公式適用（配点 各5点×4 = 20点）

次の方程式を整理してから解きなさい。

(1) 3x² + 8x + 12 = 6(x + 3)

(2) (x + 2)(x − 3) = 2x − 1

(3) 2x(x − 1) = 3(x + 2)

(4) (2x + 1)² = 5x + 7

### 解答
(1) 3x² + 8x + 12 = 6x + 18 → 3x² + 2x − 6 = 0
    x = (−2 ± √(4+72)) / 6 = (−2 ± √76) / 6 = (−2 ± 2√19) / 6 = (−1 ± √19) / 3

(2) x² − x − 6 = 2x − 1 → x² − 3x − 5 = 0
    x = (3 ± √(9+20)) / 2 = (3 ± √29) / 2

(3) 2x² − 2x = 3x + 6 → 2x² − 5x − 6 = 0
    x = (5 ± √(25+48)) / 4 = (5 ± √73) / 4

(4) 4x² + 4x + 1 = 5x + 7 → 4x² − x − 6 = 0
    x = (1 ± √(1+96)) / 8 = (1 ± √97) / 8

## 発展4 解が1つ（重解）になる条件（配点 各5点×3 = 15点）

(1) 二次方程式 x² + kx + 16 = 0 が重解をもつとき、k の値をすべて求めなさい。

(2) 二次方程式 2x² − 6x + k = 0 が重解をもつとき、k の値を求め、その重解を求めなさい。

(3) 二次方程式 kx² + 4x + 1 = 0 が重解をもつとき、k の値を求めなさい（k ≠ 0）。

### 解答
(1) 重解の条件: b²−4ac = 0
    k² − 64 = 0 → k = ±8

(2) b²−4ac = 36 − 8k = 0 → k = 9/2
    x = 6/4 = 3/2（重解）

(3) b²−4ac = 16 − 4k = 0 → k = 4
    x = −4/8 = −1/2（重解）

## 発展5 √を含む解の約分・整理（配点 各5点×4 = 20点）

次の二次方程式を解き、解を最も簡単な形に整理しなさい。

(1) 2x² + 6x + 1 = 0

(2) 3x² − 6x − 1 = 0

(3) 4x² + 8x − 3 = 0

(4) 6x² − 4x − 1 = 0

### 解答
(1) x = (−6 ± √(36−8)) / 4 = (−6 ± √28) / 4 = (−6 ± 2√7) / 4 = (−3 ± √7) / 2

(2) x = (6 ± √(36+12)) / 6 = (6 ± √48) / 6 = (6 ± 4√3) / 6 = (3 ± 2√3) / 3

(3) x = (−8 ± √(64+48)) / 8 = (−8 ± √112) / 8 = (−8 ± 4√7) / 8 = (−2 ± √7) / 2

(4) x = (4 ± √(16+24)) / 12 = (4 ± √40) / 12 = (4 ± 2√10) / 12 = (2 ± √10) / 6
