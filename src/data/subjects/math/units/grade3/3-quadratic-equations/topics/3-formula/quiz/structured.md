# 大問形式：平方完成と解の公式

## 大問1 平方完成の基礎（配点 各4点×5 = 20点）

次の式を平方完成しなさい。

(1) x² + 6x
(2) x² − 10x
(3) x² + 4x + 1
(4) x² − 8x + 3
(5) x² + 3x

### 解答
(1) (x+3)² − 9
(2) (x−5)² − 25
(3) (x+2)² − 4 + 1 = (x+2)² − 3
(4) (x−4)² − 16 + 3 = (x−4)² − 13
(5) (x+3/2)² − 9/4

---

## 大問2 平方完成で二次方程式を解く（配点 各4点×5 = 20点）

次の二次方程式を平方完成を使って解きなさい。

(1) x² + 6x + 5 = 0
(2) x² − 4x − 12 = 0
(3) x² + 2x − 5 = 0
(4) x² − 8x + 7 = 0
(5) x² + 10x + 18 = 0

### 解答
(1) (x+3)² = 4 → x+3 = ±2 → x = −1, −5
(2) (x−2)² = 16 → x−2 = ±4 → x = 6, −2
(3) (x+1)² = 6 → x = −1 ± √6
(4) (x−4)² = 9 → x−4 = ±3 → x = 7, 1
(5) (x+5)² = 7 → x = −5 ± √7

## 大問3 解の公式の適用（配点 各4点×5 = 20点）

次の二次方程式を解の公式を使って解きなさい。

(1) x² + 3x − 2 = 0
(2) x² − 5x + 1 = 0
(3) 2x² + 5x − 3 = 0
(4) 3x² − x − 4 = 0
(5) x² + 7x + 3 = 0

### 解答
(1) x = (−3 ± √(9+8)) / 2 = (−3 ± √17) / 2
(2) x = (5 ± √(25−4)) / 2 = (5 ± √21) / 2
(3) x = (−5 ± √(25+24)) / 4 = (−5 ± 7) / 4 → x = 1/2, −3
(4) x = (1 ± √(1+48)) / 6 = (1 ± 7) / 6 → x = 4/3, −1
(5) x = (−7 ± √(49−12)) / 2 = (−7 ± √37) / 2

## 大問4 係数が分数・小数の場合（配点 各5点×4 = 20点）

次の二次方程式を解きなさい。（ヒント：両辺に適当な数をかけて整数にしてから解こう）

(1) x² + 0.6x − 1.6 = 0
(2) (1/2)x² + x − 3/2 = 0
(3) 0.3x² − 1.2x + 0.9 = 0
(4) (2/3)x² − x − 1/3 = 0

### 解答
(1) 両辺×10: 10x² + 6x − 16 = 0 → 5x² + 3x − 8 = 0
    x = (−3 ± √(9+160)) / 10 = (−3 ± 13) / 10 → x = 1, −8/5
(2) 両辺×2: x² + 2x − 3 = 0 → (x+3)(x−1) = 0 → x = −3, 1
(3) 両辺×10: 3x² − 12x + 9 = 0 → x² − 4x + 3 = 0
    (x−1)(x−3) = 0 → x = 1, 3
(4) 両辺×3: 2x² − 3x − 1 = 0
    x = (3 ± √(9+8)) / 4 = (3 ± √17) / 4

## 大問5 判別式と解の個数（配点 各4点×5 = 20点）

(1) 次の二次方程式の判別式 b²−4ac の値を求め、解の個数を答えなさい。
  ① x² − 6x + 9 = 0
  ② x² + 4x + 5 = 0
  ③ 2x² − 3x − 1 = 0

(2) 二次方程式 x² − 4x + k = 0 が重解をもつとき、k の値を求めなさい。

(3) 二次方程式 x² + 6x + k = 0 が異なる2つの実数解をもつとき、k の範囲を求めなさい。

### 解答
(1) ① b²−4ac = 36−36 = 0 → 解は1つ（重解）
    ② b²−4ac = 16−20 = −4 → 解なし
    ③ b²−4ac = 9+8 = 17 > 0 → 解は2つ

(2) 重解の条件: b²−4ac = 0
    16−4k = 0 → k = 4

(3) 異なる2つの実数解の条件: b²−4ac > 0
    36−4k > 0 → k < 9
