# 一問一答：平方完成と解の公式

## 平方完成の基本

Q1. x² + bx を平方完成するとき、何を足して引くか？
A1. (b/2)² を足して引く

Q2. x² + 6x を平方完成せよ。
A2. (x+3)² − 9

Q3. x² + 10x を平方完成せよ。
A3. (x+5)² − 25

Q4. x² − 8x を平方完成せよ。
A4. (x−4)² − 16

Q5. x² + 2x を平方完成せよ。
A5. (x+1)² − 1

Q6. x² − 12x を平方完成せよ。
A6. (x−6)² − 36

Q7. x² + 14x を平方完成せよ。
A7. (x+7)² − 49

Q8. x² − 4x を平方完成せよ。
A8. (x−2)² − 4

Q9. x² + x を平方完成せよ。
A9. (x+1/2)² − 1/4

Q10. x² − 3x を平方完成せよ。
A10. (x−3/2)² − 9/4

## 平方完成で二次方程式を解く

Q11. x² + 4x − 5 = 0 を平方完成で解け。
A11. (x+2)² = 9 より x+2 = ±3、x = 1, −5

Q12. x² + 6x + 5 = 0 を平方完成で解け。
A12. (x+3)² = 4 より x+3 = ±2、x = −1, −5

Q13. x² − 2x − 8 = 0 を平方完成で解け。
A13. (x−1)² = 9 より x−1 = ±3、x = 4, −2

Q14. x² + 8x + 7 = 0 を平方完成で解け。
A14. (x+4)² = 9 より x+4 = ±3、x = −1, −7

Q15. x² − 6x + 4 = 0 を平方完成で解け。
A15. (x−3)² = 5 より x = 3±√5

Q16. x² + 2x − 5 = 0 を平方完成で解け。
A16. (x+1)² = 6 より x = −1±√6

## 解の公式の暗記

Q17. 解の公式を書け。
A17. x = (−b ± √(b²−4ac)) / 2a

Q18. 解の公式の分母は何か？
A18. 2a

Q19. 解の公式の分子のルートの中身は何か？
A19. b² − 4ac

Q20. 解の公式の分子で、bにつく符号は？
A20. マイナス（−b）

Q21. b² − 4ac を何と呼ぶか？
A21. 判別式

## 解の公式での a, b, c の読み取り

Q22. x² + 5x + 3 = 0 で a, b, c の値は？
A22. a = 1, b = 5, c = 3

Q23. 2x² − 7x + 1 = 0 で a, b, c の値は？
A23. a = 2, b = −7, c = 1

Q24. 3x² + x − 4 = 0 で a, b, c の値は？
A24. a = 3, b = 1, c = −4

Q25. x² − 3x − 10 = 0 で a, b, c の値は？
A25. a = 1, b = −3, c = −10

Q26. 4x² − 3x − 2 = 0 で a, b, c の値は？
A26. a = 4, b = −3, c = −2

## 解の公式で解く

Q27. x² + 5x + 3 = 0 を解の公式で解け。
A27. x = (−5 ± √(25−12)) / 2 = (−5 ± √13) / 2

Q28. x² + x − 4 = 0 を解の公式で解け。
A28. x = (−1 ± √(1+16)) / 2 = (−1 ± √17) / 2

Q29. x² − 5x − 5 = 0 を解の公式で解け。
A29. x = (5 ± √(25+20)) / 2 = (5 ± √45) / 2 = (5 ± 3√5) / 2

Q30. x² + 2x − 3 = 0 を解の公式で解け。
A30. x = (−2 ± √(4+12)) / 2 = (−2 ± 4) / 2 → x = 1, −3

Q31. 2x² + 5x − 3 = 0 を解の公式で解け。
A31. x = (−5 ± √(25+24)) / 4 = (−5 ± 7) / 4 → x = 1/2, −3

Q32. 3x² − 7x − 1 = 0 を解の公式で解け。
A32. x = (7 ± √(49+12)) / 6 = (7 ± √61) / 6

Q33. 4x² − 3x − 2 = 0 を解の公式で解け。
A33. x = (3 ± √(9+32)) / 8 = (3 ± √41) / 8

Q34. x² − 4x + 1 = 0 を解の公式で解け。
A34. x = (4 ± √(16−4)) / 2 = (4 ± √12) / 2 = 2 ± √3

Q35. 2x² + 3x − 1 = 0 を解の公式で解け。
A35. x = (−3 ± √(9+8)) / 4 = (−3 ± √17) / 4

## 判別式と解の個数

Q36. b² − 4ac > 0 のとき、解はいくつあるか？
A36. 2つ（異なる2つの実数解）

Q37. b² − 4ac = 0 のとき、解はいくつあるか？
A37. 1つ（重解）

Q38. b² − 4ac < 0 のとき、解はいくつあるか？
A38. 0個（実数の範囲では解なし）

Q39. x² − 6x + 9 = 0 の判別式の値を求め、解の個数を答えよ。
A39. b²−4ac = 36−36 = 0 → 解は1つ（重解 x = 3）

Q40. x² + 3x + 5 = 0 の判別式の値を求め、解の個数を答えよ。
A40. b²−4ac = 9−20 = −11 < 0 → 解なし

Q41. x² − 2x − 8 = 0 の判別式の値を求め、解の個数を答えよ。
A41. b²−4ac = 4+32 = 36 > 0 → 解は2つ

Q42. 2x² + 4x + 2 = 0 の判別式の値を求め、解の個数を答えよ。
A42. b²−4ac = 16−16 = 0 → 解は1つ（重解 x = −1）

## √が残る解の処理

Q43. x = (6 ± √12) / 2 を簡単にせよ。
A43. √12 = 2√3 だから x = (6 ± 2√3) / 2 = 3 ± √3

Q44. x = (4 ± √8) / 4 を簡単にせよ。
A44. √8 = 2√2 だから x = (4 ± 2√2) / 4 = (2 ± √2) / 2

Q45. x = (−6 ± √20) / 2 を簡単にせよ。
A45. √20 = 2√5 だから x = (−6 ± 2√5) / 2 = −3 ± √5

Q46. x = (10 ± √50) / 10 を簡単にせよ。
A46. √50 = 5√2 だから x = (10 ± 5√2) / 10 = (2 ± √2) / 2
