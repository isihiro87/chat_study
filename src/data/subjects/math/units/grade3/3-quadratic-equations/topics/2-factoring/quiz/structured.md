# 因数分解による二次方程式の解法 大問小問

## 大問1: A×B=0の原則と基本（配点 20点）

次の二次方程式を解きなさい。

(1) $(x - 4)(x + 1) = 0$（3点）
→ $x = 4, x = -1$。$x - 4 = 0$ → $x = 4$、$x + 1 = 0$ → $x = -1$。

(2) $(x + 3)(x - 7) = 0$（3点）
→ $x = -3, x = 7$。$x + 3 = 0$ → $x = -3$、$x - 7 = 0$ → $x = 7$。

(3) $x(x - 5) = 0$（4点）
→ $x = 0, x = 5$。$x = 0$ も解になることに注意。

(4) $x(x + 9) = 0$（4点）
→ $x = 0, x = -9$。$x = 0$ を忘れないこと。

(5) $x^2 - 6x = 0$（3点）
→ $x = 0, x = 6$。$x(x - 6) = 0$ と因数分解する。

(6) $x^2 + 4x = 0$（3点）
→ $x = 0, x = -4$。$x(x + 4) = 0$ と因数分解する。

## 大問2: x²+bx+c=0 型の因数分解（配点 20点）

次の二次方程式を因数分解して解きなさい。

(1) $x^2 - 5x + 6 = 0$（3点）
→ $x = 2, x = 3$。$(x - 2)(x - 3) = 0$。かけて $6$、足して $-5$ → $-2$ と $-3$。

(2) $x^2 + x - 12 = 0$（3点）
→ $x = 3, x = -4$。$(x - 3)(x + 4) = 0$。かけて $-12$、足して $1$ → $-3$ と $4$。

(3) $x^2 - 7x + 10 = 0$（4点）
→ $x = 2, x = 5$。$(x - 2)(x - 5) = 0$。かけて $10$、足して $-7$ → $-2$ と $-5$。

(4) $x^2 + 3x - 10 = 0$（4点）
→ $x = 2, x = -5$。$(x - 2)(x + 5) = 0$。かけて $-10$、足して $3$ → $-2$ と $5$。

(5) $x^2 - 2x - 15 = 0$（3点）
→ $x = 5, x = -3$。$(x - 5)(x + 3) = 0$。かけて $-15$、足して $-2$ → $-5$ と $3$。

(6) $x^2 + 9x + 20 = 0$（3点）
→ $x = -4, x = -5$。$(x + 4)(x + 5) = 0$。かけて $20$、足して $9$ → $4$ と $5$。

## 大問3: 共通因数・完全平方式・差の平方（配点 20点）

次の二次方程式を解きなさい。

(1) $2x^2 - 5x = 0$（3点）
→ $x = 0, x = \dfrac{5}{2}$。$x(2x - 5) = 0$。共通因数 $x$ でくくる。

(2) $3x^2 + 12x = 0$（3点）
→ $x = 0, x = -4$。$3x(x + 4) = 0$。共通因数 $3x$ でくくる。

(3) $x^2 - 2x + 1 = 0$（4点）
→ $x = 1$（重解）。$(x - 1)^2 = 0$。完全平方式。

(4) $x^2 + 6x + 9 = 0$（4点）
→ $x = -3$（重解）。$(x + 3)^2 = 0$。完全平方式。

(5) $x^2 - 9 = 0$（3点）
→ $x = 3, x = -3$。$(x + 3)(x - 3) = 0$。$a^2 - b^2 = (a+b)(a-b)$。

(6) $x^2 - 25 = 0$（3点）
→ $x = 5, x = -5$。$(x + 5)(x - 5) = 0$。$a^2 - b^2 = (a+b)(a-b)$。

## 大問4: 係数をそろえてから因数分解（配点 20点）

次の二次方程式を解きなさい。

(1) $-x^2 + 5x - 6 = 0$（3点）
→ $x = 2, x = 3$。$-1$ をかけて $x^2 - 5x + 6 = 0$ → $(x - 2)(x - 3) = 0$。

(2) $-x^2 - 2x + 15 = 0$（4点）
→ $x = 3, x = -5$。$-1$ をかけて $x^2 + 2x - 15 = 0$ → $(x + 5)(x - 3) = 0$。

(3) $2x^2 - 10x + 12 = 0$（3点）
→ $x = 2, x = 3$。$2$ で割って $x^2 - 5x + 6 = 0$ → $(x - 2)(x - 3) = 0$。

(4) $-4x^2 + 8x - 4 = 0$（4点）
→ $x = 1$（重解）。$-4$ で割って $x^2 - 2x + 1 = 0$ → $(x - 1)^2 = 0$。

(5) $3x^2 - 3x - 18 = 0$（3点）
→ $x = 3, x = -2$。$3$ で割って $x^2 - x - 6 = 0$ → $(x - 3)(x + 2) = 0$。

(6) $-2x^2 + 18 = 0$（3点）
→ $x = 3, x = -3$。$-2$ で割って $x^2 - 9 = 0$ → $(x + 3)(x - 3) = 0$。

## 大問5: 展開→整理→因数分解の総合問題（配点 20点）

次の方程式を解きなさい。

(1) $(x - 3)(x - 4) = 2$（4点）
→ $x = 2, x = 5$。展開：$x^2 - 7x + 12 = 2$ → $x^2 - 7x + 10 = 0$ → $(x - 2)(x - 5) = 0$。

(2) $(x + 2)(x - 1) = 4$（4点）
→ $x = 2, x = -3$。展開：$x^2 + x - 2 = 4$ → $x^2 + x - 6 = 0$ → $(x + 3)(x - 2) = 0$。

(3) $x(x + 5) = 14$（4点）
→ $x = 2, x = -7$。展開：$x^2 + 5x = 14$ → $x^2 + 5x - 14 = 0$ → $(x - 2)(x + 7) = 0$。

(4) $(x - 1)^2 = 9$（4点）
→ $x = 4, x = -2$。$(x - 1)^2 - 9 = 0$ → $(x + 2)(x - 4) = 0$。差の平方を利用。

(5) $(x + 3)^2 = 4x + 12$（4点）
→ $x = -3, x = 1$。展開：$x^2 + 6x + 9 = 4x + 12$ → $x^2 + 2x - 3 = 0$ → $(x + 3)(x - 1) = 0$。
