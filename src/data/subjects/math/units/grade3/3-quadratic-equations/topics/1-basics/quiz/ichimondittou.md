# 一問一答：二次方程式の基本

## 二次方程式の定義

Q1. xの2乗を含む方程式を何というか？
A1. 二次方程式

Q2. 二次方程式の標準形を書け。
A2. ax² + bx + c = 0（a ≠ 0）

Q3. 二次方程式 ax² + bx + c = 0 で、a の条件は何か？
A3. a ≠ 0（a は0でない）

Q4. a = 0 のとき、ax² + bx + c = 0 はどうなるか？
A4. 一次方程式（bx + c = 0）になる

Q5. 二次方程式において、x の最高次数はいくつか？
A5. 2

Q6. x² − 7x + 10 = 0 の a, b, c の値をそれぞれ答えよ。
A6. a = 1, b = −7, c = 10

Q7. 3x² + 5x = 0 の a, b, c の値をそれぞれ答えよ。
A7. a = 3, b = 5, c = 0

Q8. −2x² + 8 = 0 の a, b, c の値をそれぞれ答えよ。
A8. a = −2, b = 0, c = 8

## 二次方程式かどうかの判別

Q9. 5x + 3 = 0 は二次方程式か？
A9. いいえ（一次方程式）

Q10. x² − 4 = 0 は二次方程式か？
A10. はい（a = 1, b = 0, c = −4）

Q11. x³ + x² − 2 = 0 は二次方程式か？
A11. いいえ（三次方程式）

Q12. 4x² = 0 は二次方程式か？
A12. はい（a = 4, b = 0, c = 0）

Q13. (x + 1)² = 0 を展開すると二次方程式になるか？
A13. はい（x² + 2x + 1 = 0）

Q14. x(x − 3) = 0 を展開すると何次方程式になるか？
A14. 二次方程式（x² − 3x = 0）

## 解の意味

Q15. 二次方程式の「解」とは何か？
A15. 方程式を成り立たせる x の値

Q16. x² − 9 = 0 に x = 3 を代入すると成り立つか？
A16. 成り立つ（3² − 9 = 9 − 9 = 0）

Q17. x² − 9 = 0 に x = −3 を代入すると成り立つか？
A17. 成り立つ（(−3)² − 9 = 9 − 9 = 0）

Q18. x² − 9 = 0 に x = 4 を代入すると成り立つか？
A18. 成り立たない（4² − 9 = 16 − 9 = 7 ≠ 0）

Q19. x² − 5x + 6 = 0 の解の1つは x = 2 である。確認せよ。
A19. 2² − 5×2 + 6 = 4 − 10 + 6 = 0 で成り立つ

## 解の個数

Q20. 二次方程式の解は最大いくつか？
A20. 最大2つ

Q21. x² = 0 の解はいくつあるか？
A21. 1つ（x = 0 のみ）

Q22. x² = −4 の解はいくつあるか？
A22. 0個（2乗して負になる実数はない）

Q23. x² = 25 の解はいくつあるか？
A23. 2つ（x = 5 と x = −5）

## ax² = b 型の解法

Q24. x² = 36 を解け。
A24. x = ±6

Q25. x² = 7 を解け。
A25. x = ±√7

Q26. 2x² = 18 を解け。
A26. x = ±3（x² = 9 より）

Q27. 3x² = 48 を解け。
A27. x = ±4（x² = 16 より）

Q28. 5x² − 45 = 0 を解け。
A28. x = ±3（5x² = 45, x² = 9 より）

Q29. 2x² − 54 = 0 を解け。
A29. x = ±3√3（x² = 27 より）

Q30. 4x² = 1 を解け。
A30. x = ±1/2（x² = 1/4 より）

Q31. x²/3 = 12 を解け。
A31. x = ±6（x² = 36 より）

Q32. −x² + 5 = 0 を解け。
A32. x = ±√5（x² = 5 より）

Q33. 6x² − 24 = 0 を解け。
A33. x = ±2（x² = 4 より）

Q34. x² = 0.25 を解け。
A34. x = ±0.5

## (x + m)² = n 型の解法

Q35. (x + 1)² = 4 を解け。
A35. x + 1 = ±2 より x = 1 または x = −3

Q36. (x − 3)² = 9 を解け。
A36. x − 3 = ±3 より x = 6 または x = 0

Q37. (x − 3)² = 2 を解け。
A37. x − 3 = ±√2 より x = 3 ± √2

Q38. (x + 5)² = 25 を解け。
A38. x + 5 = ±5 より x = 0 または x = −10

Q39. (x − 2)² = 7 を解け。
A39. x − 2 = ±√7 より x = 2 ± √7

Q40. (x + 4)² = 16 を解け。
A40. x + 4 = ±4 より x = 0 または x = −8

Q41. (x + 1)² − 3 = 0 を解け。
A41. (x + 1)² = 3, x + 1 = ±√3 より x = −1 ± √3

Q42. (x − 5)² = 1 を解け。
A42. x − 5 = ±1 より x = 6 または x = 4

Q43. (x + 3)² = 12 を解け。
A43. x + 3 = ±2√3 より x = −3 ± 2√3

Q44. (x − 1)² − 9 = 0 を解け。
A44. (x − 1)² = 9, x − 1 = ±3 より x = 4 または x = −2

Q45. (x + 2)² = 5 を解け。
A45. x + 2 = ±√5 より x = −2 ± √5
