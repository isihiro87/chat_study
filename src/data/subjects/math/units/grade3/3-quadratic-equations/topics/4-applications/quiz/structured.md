# 二次方程式の利用 大問小問

## 大問1: 整数の問題（配点 20点）

次の問いに答えなさい。

(1) 連続する2つの正の整数の積が72のとき、2つの整数を求めなさい。（4点）
→ 小さい方を $n$ とすると $n(n+1) = 72$, $n^2 + n - 72 = 0$, $(n+9)(n-8) = 0$。$n = 8$（$n = -9$ は正の整数でないので不適）。答え：8と9。

(2) 連続する2つの正の整数の積が110のとき、2つの整数を求めなさい。（4点）
→ $n(n+1) = 110$, $n^2 + n - 110 = 0$, $(n+11)(n-10) = 0$。$n = 10$（$n = -11$ は不適）。答え：10と11。

(3) ある正の整数の2乗から、その数の3倍を引くと10になる。この整数を求めなさい。（4点）
→ $x^2 - 3x = 10$, $x^2 - 3x - 10 = 0$, $(x-5)(x+2) = 0$。正の整数なので $x = 5$。

(4) ある正の整数に、その数より6大きい数をかけると72になる。もとの整数を求めなさい。（4点）
→ $x(x+6) = 72$, $x^2 + 6x - 72 = 0$, $(x+12)(x-6) = 0$。正の整数なので $x = 6$。

(5) 2つの正の整数の差が5で、積が84のとき、2つの整数を求めなさい。（4点）
→ 小さい方を $x$ とすると $x(x+5) = 84$, $x^2 + 5x - 84 = 0$, $(x+12)(x-7) = 0$。$x = 7$（$x = -12$ は不適）。答え：7と12。

## 大問2: 図形の面積（配点 20点）

次の問いに答えなさい。

(1) 長方形の縦が横より4cm短く、面積が60cm²のとき、横の長さを求めなさい。（4点）
→ 横を $x$ cm とすると、縦は $(x-4)$ cm。$x(x-4) = 60$, $x^2 - 4x - 60 = 0$, $(x-10)(x+6) = 0$。$x = 10$（$x = -6$ は不適）。横 10cm。

(2) 正方形の縦を3cm伸ばし横を2cm短くしたら面積が50cm²の長方形になった。もとの正方形の1辺の長さを求めなさい。（4点）
→ 1辺を $x$ cm とすると $(x+3)(x-2) = 50$, $x^2 + x - 6 = 50$, $x^2 + x - 56 = 0$, $(x+8)(x-7) = 0$。$x = 7$（$x = -8$ は不適）。もとの1辺は 7cm。検算：$(7+3)(7-2) = 10 \times 5 = 50$。

(3) 縦12m、横16mの長方形の土地に同じ幅 $x$ mの道を縦と横に1本ずつ作り、残りの面積を140m²にしたい。道幅を求めなさい。（4点）
→ 道を端に寄せて考えると $(12 - x)(16 - x) = 140$。$192 - 28x + x^2 = 140$, $x^2 - 28x + 52 = 0$, $(x - 2)(x - 26) = 0$。$x = 2$（$x = 26$ は12mを超えるので不適）。道幅 2m。

(4) 縦20m、横30mの長方形の土地に幅 $x$ mの道を縦と横に1本ずつ作り、残りの面積を504m²にしたい。道幅を求めなさい。（4点）
→ $(20 - x)(30 - x) = 504$, $600 - 50x + x^2 = 504$, $x^2 - 50x + 96 = 0$, $(x-2)(x-48) = 0$。$x = 2$（$x = 48$ は不適）。道幅 2m。

(5) 正方形の花壇の周りに幅1mの道を作ったところ、道の面積が36m²になった。花壇の1辺の長さを求めなさい。（4点）
→ 花壇の1辺を $x$ m とすると $(x+2)^2 - x^2 = 36$, $4x + 4 = 36$, $4x = 32$, $x = 8$。1辺 8m。

## 大問3: 容積の問題（配点 20点）

次の問いに答えなさい。

(1) 1辺が16cmの正方形の厚紙の四隅から、1辺 $x$ cmの正方形を切り取ってふたのない箱を作る。底面の1辺の長さと高さをそれぞれ $x$ を使って表しなさい。（3点）
→ 底面の1辺 $= (16 - 2x)$ cm、高さ $= x$ cm。

(2) 上の箱の容積を $x$ を使った式で表しなさい。（3点）
→ $V = x(16 - 2x)^2$ cm³。

(3) 上の箱で $x$ のとりうる値の範囲を不等式で表しなさい。（3点）
→ $0 < x < 8$。底面の1辺 $(16 - 2x) > 0$ より $x < 8$、また $x > 0$。

(4) 縦30cm、横20cmの長方形の厚紙の四隅から1辺 $x$ cmの正方形を切り取って箱を作る。底面の面積が200cm²になるとき、$x$ の値を求めなさい。（5点）
→ $(30-2x)(20-2x) = 200$, $600 - 100x + 4x^2 = 200$, $4x^2 - 100x + 400 = 0$, $x^2 - 25x + 100 = 0$, $(x-5)(x-20) = 0$。$x = 5$（$x = 20$ は $20 - 40 < 0$ で不適）。$x = 5$ cm。

(5) 1辺12cmの正方形の厚紙の四隅から1辺 $x$ cmの正方形を切り取って箱を作る。容積が128cm³になるとき、$x$ の値を求めなさい。（6点）
→ $x(12-2x)^2 = 128$。$x = 2$ のとき $2 \times (12-4)^2 = 2 \times 64 = 128$。$0 < x < 6$ を満たすので $x = 2$ cm。

## 大問4: 動く点の問題（配点 20点）

長方形ABCDで AB = 10cm, BC = 6cm とする。点PはAを出発して辺AB上をBの方向に毎秒2cmで動く。出発してから $x$ 秒後について、次の問いに答えなさい。

(1) AP の長さを $x$ を使って表しなさい。（3点）
→ $\text{AP} = 2x$ cm。

(2) △APDの面積を $x$ を使って表しなさい。（3点）
→ 底辺AP $= 2x$ cm、高さAD $= 6$ cm。面積 $= \dfrac{1}{2} \times 2x \times 6 = 6x$（cm²）。

(3) △APDの面積が24cm²になるのは何秒後か求めなさい。（4点）
→ $6x = 24$, $x = 4$。4秒後。検算：AP $= 8$ cm, $\dfrac{1}{2} \times 8 \times 6 = 24$。

(4) $x$ のとりうる値の範囲を求めなさい。（3点）
→ PがAB上にあるので $0 < 2x \leq 10$, すなわち $0 < x \leq 5$。

正方形ABCDで1辺が12cmとする。点PはAを出発して辺AB上をBの方向に毎秒1cmで動き、同時に点QはBを出発して辺BC上をCの方向に毎秒1cmで動く。出発してから $x$ 秒後について、次の問いに答えなさい。

(5) △BPQの面積を $x$ を使って表しなさい。（3点）
→ BP $= (12 - x)$ cm、BQ $= x$ cm。面積 $= \dfrac{1}{2}(12-x) \cdot x = \dfrac{x(12-x)}{2}$（cm²）。

(6) △BPQの面積が16cm²になるのは何秒後か求めなさい。（4点）
→ $\dfrac{x(12-x)}{2} = 16$, $x(12-x) = 32$, $12x - x^2 = 32$, $x^2 - 12x + 32 = 0$, $(x-4)(x-8) = 0$。$x = 4$ または $x = 8$。ともに $0 < x < 12$ を満たすので、4秒後と8秒後。

## 大問5: 総合問題（配点 20点）

次の問いに答えなさい。

(1) 長方形の周の長さが26cmで、面積が42cm²のとき、縦と横の長さをそれぞれ求めなさい。（5点）
→ 横を $x$ cm とすると、縦は $(13 - x)$ cm（周 $= 2(x + \text{縦}) = 26$ より）。$x(13-x) = 42$, $13x - x^2 = 42$, $x^2 - 13x + 42 = 0$, $(x-6)(x-7) = 0$。$x = 6$ または $x = 7$。答え：縦6cm、横7cm（または縦7cm、横6cm）。

(2) ある数の2乗と、その数に1を加えた数の2乗の和が61になる。この数が正の整数のとき、その数を求めなさい。（5点）
→ $x^2 + (x+1)^2 = 61$, $2x^2 + 2x + 1 = 61$, $2x^2 + 2x - 60 = 0$, $x^2 + x - 30 = 0$, $(x+6)(x-5) = 0$。正の整数なので $x = 5$。

(3) 正方形の土地の1辺を3m伸ばして長方形にすると、面積がもとの2倍になった。もとの正方形の1辺を求めなさい。（5点）
→ 1辺を $x$ m とすると $x(x+3) = 2x^2$, $x^2 + 3x = 2x^2$, $x^2 - 3x = 0$, $x(x-3) = 0$。$x = 3$（$x = 0$ は不適）。1辺3m。

(4) 縦10m、横15mの長方形の花壇がある。縦と横を同じ長さだけ伸ばして面積を234m²にしたい。何mずつ伸ばせばよいか。（5点）
→ 伸ばす長さを $x$ m とすると $(10+x)(15+x) = 234$, $150 + 25x + x^2 = 234$, $x^2 + 25x - 84 = 0$, $(x+28)(x-3) = 0$。$x = 3$（$x = -28$ は不適）。3mずつ伸ばす。
