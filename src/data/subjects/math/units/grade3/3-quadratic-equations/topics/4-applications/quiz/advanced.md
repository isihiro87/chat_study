# 二次方程式の利用 発展問題

## 発展1: 正方形の動点P, Q

正方形ABCDの1辺の長さは10cmである。点PはAを出発して辺AB上をBの方向に毎秒1cmで動き、同時に点QはBを出発して辺BC上をCの方向に毎秒1cmで動く。出発してから $x$ 秒後について、次の問いに答えなさい。

(1) AP, BQ の長さをそれぞれ $x$ を使って表しなさい。
→ AP $= x$ cm、BQ $= x$ cm。

(2) BP の長さを $x$ を使って表しなさい。
→ BP $= (10 - x)$ cm。

(3) △BPQの面積を $x$ を使って表しなさい。
→ $\dfrac{1}{2} \times (10 - x) \times x = \dfrac{x(10 - x)}{2}$（cm²）。

(4) △BPQの面積が12cm²になるのは何秒後か求めなさい。
→ $\dfrac{x(10-x)}{2} = 12$, $x(10-x) = 24$, $10x - x^2 = 24$, $x^2 - 10x + 24 = 0$, $(x-4)(x-6) = 0$。$x = 4$ または $x = 6$。ともに $0 < x < 10$ を満たすので、4秒後と6秒後。

(5) △BPQの面積が正方形ABCDの面積の $\dfrac{1}{8}$ になるのは何秒後か求めなさい。
→ 正方形の面積 $= 100$ cm²。$\dfrac{1}{8}$ は $12.5$ cm²。$\dfrac{x(10-x)}{2} = 12.5$, $x(10-x) = 25$, $x^2 - 10x + 25 = 0$, $(x-5)^2 = 0$, $x = 5$。5秒後。

## 発展2: 直角三角形の動点問題

直角三角形ABCで ∠B = 90°、AB = 8cm、BC = 6cm とする。点PはAを出発して辺AB上をBの方向に毎秒2cmで動く。出発してから $x$ 秒後について、次の問いに答えなさい。

(1) AP の長さを $x$ を使って表しなさい。
→ AP $= 2x$ cm。

(2) BP の長さを $x$ を使って表しなさい。
→ BP $= (8 - 2x)$ cm。

(3) △PBCの面積を $x$ を使って表しなさい。
→ $\dfrac{1}{2} \times (8 - 2x) \times 6 = 3(8 - 2x) = 24 - 6x$（cm²）。

(4) △PBCの面積が△ABCの面積の $\dfrac{1}{4}$ になるのは何秒後か求めなさい。
→ △ABCの面積 $= \dfrac{1}{2} \times 8 \times 6 = 24$ cm²。$\dfrac{1}{4}$ は 6cm²。$24 - 6x = 6$, $6x = 18$, $x = 3$。3秒後。検算：BP $= 8 - 6 = 2$, $\dfrac{1}{2} \times 2 \times 6 = 6$。

## 発展3: 鉄板の折り曲げ問題

幅20cmの長方形の鉄板を、長辺に平行に両端を同じ高さだけ直角に折り曲げて、断面が長方形の「とい（樋）」を作る。折り曲げる高さを $x$ cmとして、次の問いに答えなさい。

(1) 底面の幅を $x$ を使って表しなさい。
→ $(20 - 2x)$ cm。

(2) 断面積を $x$ を使って表しなさい。
→ $x(20 - 2x)$ cm²。

(3) 断面積が48cm²になるとき、折り曲げる高さを求めなさい。
→ $x(20 - 2x) = 48$, $20x - 2x^2 = 48$, $2x^2 - 20x + 48 = 0$, $x^2 - 10x + 24 = 0$, $(x-4)(x-6) = 0$。$x = 4$ または $x = 6$。ともに $0 < x < 10$ を満たすので、4cmまたは6cm。

(4) 断面積が最大になるのは $x$ がいくらのときか。また、そのときの断面積を求めなさい。
→ $S = x(20-2x) = -2x^2 + 20x = -2(x^2 - 10x) = -2(x-5)^2 + 50$。$x = 5$ のとき最大値 50cm²。

## 発展4: 関数 $y = ax^2$ と面積

関数 $y = x^2$ のグラフ上に2点 A, B がある。Aの $x$ 座標は $-2$、Bの $x$ 座標は $t$（$t > 0$）とする。次の問いに答えなさい。

(1) 点Aの座標を求めなさい。
→ $y = (-2)^2 = 4$。A$(-2, 4)$。

(2) 点Bの座標を $t$ を使って表しなさい。
→ B$(t, t^2)$。

(3) 直線ABが $x$ 軸と平行になるとき、$t$ の値を求めなさい。
→ Aの $y$ 座標とBの $y$ 座標が等しいので $t^2 = 4$, $t = 2$（$t > 0$）。

(4) 原点をOとし、△OABの面積が10になるとき、$t$ の値を求めなさい。
→ 座標の面積公式より $\text{面積} = \dfrac{1}{2}|x_A y_B - x_B y_A| = \dfrac{1}{2}|(-2) \cdot t^2 - t \cdot 4| = \dfrac{1}{2}(2t^2 + 4t) = t^2 + 2t$（$t > 0$）。$t^2 + 2t = 10$, $t^2 + 2t - 10 = 0$。解の公式より $t = \dfrac{-2 + \sqrt{44}}{2} = -1 + \sqrt{11}$（$t > 0$ なので正の解のみ）。

## 発展5: カレンダーの問題

次の問いに答えなさい。

(1) カレンダーで横に並ぶ連続する3つの数を $n$, $n+1$, $n+2$ とするとき、両端の数の積が真ん中の数の2乗より1小さいことを示しなさい。
→ $n(n+2) = n^2 + 2n$。$(n+1)^2 = n^2 + 2n + 1$。$n(n+2) = (n+1)^2 - 1$。よって両端の積は真ん中の2乗より1小さい。

(2) カレンダーで縦に並ぶ3つの数（差は7ずつ）を $n$, $n+7$, $n+14$ とするとき、両端の数の積が真ん中の数の2乗より49小さいことを示しなさい。
→ $n(n+14) = n^2 + 14n$。$(n+7)^2 = n^2 + 14n + 49$。$n(n+14) = (n+7)^2 - 49$。よって示された。

(3) カレンダーで、ある日と翌日の日付の積が182のとき、その2日を求めなさい。
→ $n(n+1) = 182$, $n^2 + n - 182 = 0$, $(n+14)(n-13) = 0$。$n = 13$（$n = -14$ は不適）。13日と14日。

(4) カレンダーで、縦に並ぶ2つの数の積が144のとき、その2つの日付を求めなさい。
→ $n(n+7) = 144$, $n^2 + 7n - 144 = 0$, $(n+16)(n-9) = 0$。$n = 9$（$n = -16$ は不適）。9日と16日。
