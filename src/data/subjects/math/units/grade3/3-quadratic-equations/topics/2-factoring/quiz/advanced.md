# 発展問題：因数分解による二次方程式の解法

## 発展1: 展開→整理→因数分解の応用（配点20点）

次の方程式を解きなさい。

(1) $(x + 5)(x - 3) = 7x - 1$（4点）
→ $x = 7, x = -2$。展開：$x^2 + 2x - 15 = 7x - 1$ → $x^2 - 5x - 14 = 0$ → $(x - 7)(x + 2) = 0$。

(2) $(x - 2)(x + 4) = 2x + 1$（4点）
→ $x = -3, x = 3$。展開：$x^2 + 2x - 8 = 2x + 1$ → $x^2 - 9 = 0$ → $(x + 3)(x - 3) = 0$。

(3) $(x + 1)(x - 3) = 3x - 9$ を解け。（6点）
→ $x = 2, x = 3$。展開：$x^2 - 2x - 3 = 3x - 9$ → $x^2 - 5x + 6 = 0$ → $(x - 2)(x - 3) = 0$。

(4) $(x - 4)^2 = 3(x - 4)$（6点）
→ $x = 4, x = 7$。$(x-4)^2 - 3(x-4) = 0$ → $(x-4)\{(x-4) - 3\} = 0$ → $(x-4)(x-7) = 0$。

## 発展2: 重解の問題（配点20点）

(1) $x^2 + 10x + 25 = 0$ を解け。（4点）
→ $x = -5$（重解）。$(x + 5)^2 = 0$。$x^2 + 2 \times 5 \times x + 5^2$ の形。

(2) $4x^2 - 12x + 9 = 0$ を解け。（4点）
→ $x = \dfrac{3}{2}$（重解）。$(2x - 3)^2 = 0$。$(2x)^2 - 2 \times 2x \times 3 + 3^2$ の形。

(3) $x^2 - 2ax + a^2 = 0$（$a$ は定数）の解を求めよ。（6点）
→ $x = a$（重解）。$(x - a)^2 = 0$。

(4) $x^2 + kx + 9 = 0$ が重解をもつとき、定数 $k$ の値を求めよ。（6点）
→ $k = 6$ または $k = -6$。重解の条件は $(x \pm 3)^2 = 0$ の形 → $k = \pm 6$。

## 発展3: 共通因数でくくってから解く（配点20点）

(1) $2x^2 + 8x + 6 = 0$ を解け。（4点）
→ $x = -1, x = -3$。$2$ で割って $x^2 + 4x + 3 = 0$ → $(x + 1)(x + 3) = 0$。

(2) $3x^2 - 27 = 0$ を解け。（4点）
→ $x = 3, x = -3$。$3$ で割って $x^2 - 9 = 0$ → $(x + 3)(x - 3) = 0$。

(3) $-5x^2 + 20x - 15 = 0$ を解け。（6点）
→ $x = 1, x = 3$。$-5$ で割って $x^2 - 4x + 3 = 0$ → $(x - 1)(x - 3) = 0$。

(4) $6x^2 + 18x + 12 = 0$ を解け。（6点）
→ $x = -1, x = -2$。$6$ で割って $x^2 + 3x + 2 = 0$ → $(x + 1)(x + 2) = 0$。

## 発展4: 置換型の二次方程式（配点20点）

(1) $(x + 2)^2 + 3(x + 2) - 10 = 0$ を解け。（4点）
→ $x = 0, x = -7$。$M = x + 2$ とおくと $M^2 + 3M - 10 = 0$ → $(M + 5)(M - 2) = 0$ → $M = -5, 2$。$x + 2 = -5$ → $x = -7$、$x + 2 = 2$ → $x = 0$。

(2) $(x - 3)^2 - 5(x - 3) + 6 = 0$ を解け。（4点）
→ $x = 5, x = 6$。$M = x - 3$ とおくと $M^2 - 5M + 6 = 0$ → $(M - 2)(M - 3) = 0$ → $M = 2, 3$。$x = 5, 6$。

(3) $(x - 1)^2 - 4(x - 1) - 5 = 0$ を解け。（6点）
→ $x = 6, x = 0$。$M = x - 1$ とおくと $M^2 - 4M - 5 = 0$ → $(M - 5)(M + 1) = 0$ → $M = 5, -1$。$x = 6, 0$。

(4) $(2x - 1)^2 - (2x - 1) - 6 = 0$ を解け。（6点）
→ $x = 2, x = -\dfrac{1}{2}$。$M = 2x - 1$ とおくと $M^2 - M - 6 = 0$ → $(M - 3)(M + 2) = 0$ → $M = 3, -2$。$2x - 1 = 3$ → $x = 2$、$2x - 1 = -2$ → $x = -\dfrac{1}{2}$。

## 発展5: 解と係数の関係の基本（配点20点）

(1) $x^2 - 7x + 12 = 0$ の2つの解の和と積を、方程式を解かずに求めよ。（4点）
→ 和 $= 7$、積 $= 12$。$x^2 - (和)x + (積) = 0$ の形と比較する。

(2) 2つの解が $x = 3, x = -5$ である二次方程式を $x^2 + bx + c = 0$ の形で作れ。（4点）
→ $x^2 + 2x - 15 = 0$。和 $= 3 + (-5) = -2$、積 $= 3 \times (-5) = -15$。$x^2 - (-2)x + (-15) = 0$。

(3) $x^2 + ax + b = 0$ の1つの解が $x = 2$ で、もう1つの解が $x = -6$ のとき、$a, b$ の値を求めよ。（6点）
→ $a = 4, b = -12$。和 $= 2 + (-6) = -4$ → $-a = -4$ → $a = 4$。積 $= 2 \times (-6) = -12$ → $b = -12$。

(4) $x^2 - 8x + k = 0$ の1つの解が $x = 3$ のとき、定数 $k$ ともう1つの解を求めよ。（6点）
→ $k = 15$、もう1つの解は $x = 5$。$x = 3$ を代入：$9 - 24 + k = 0$ → $k = 15$。$x^2 - 8x + 15 = 0$ → $(x - 3)(x - 5) = 0$ → $x = 5$。
