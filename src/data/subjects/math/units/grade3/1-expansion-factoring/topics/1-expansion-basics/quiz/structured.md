# 式の展開の基本 大問小問

## 大問1: 分配法則による展開（配点 20点）

次の式を展開しなさい。

(1) $3(x + 4)$（3点）
→ $3x + 12$。$3 \times x + 3 \times 4 = 3x + 12$。

(2) $5(2x - 3)$（3点）
→ $10x - 15$。$5 \times 2x + 5 \times (-3) = 10x - 15$。

(3) $2x(x + 3)$（3点）
→ $2x^2 + 6x$。$2x \times x + 2x \times 3 = 2x^2 + 6x$。

(4) $-3x(5x - 2)$（4点）
→ $-15x^2 + 6x$。$(-3x) \times 5x + (-3x) \times (-2) = -15x^2 + 6x$。符号に注意。

(5) $(3x + 5y) \times (-2y)$（4点）
→ $-6xy - 10y^2$。$3x \times (-2y) + 5y \times (-2y) = -6xy - 10y^2$。

(6) $(6a - 15) \times \dfrac{2}{3}a$（3点）
→ $4a^2 - 10a$。$6a \times \dfrac{2}{3}a + (-15) \times \dfrac{2}{3}a = 4a^2 - 10a$。

## 大問2: 多項式÷単項式（配点 20点）

次の計算をしなさい。

(1) $(8x^2 + 4x) \div 2x$（3点）
→ $4x + 2$。$\dfrac{8x^2}{2x} + \dfrac{4x}{2x} = 4x + 2$。

(2) $(6a^2 + 9a) \div 3a$（3点）
→ $2a + 3$。$\dfrac{6a^2}{3a} + \dfrac{9a}{3a} = 2a + 3$。

(3) $(12x^2 - 8x) \div 4x$（3点）
→ $3x - 2$。$\dfrac{12x^2}{4x} - \dfrac{8x}{4x} = 3x - 2$。

(4) $(8x^2 + 4x) \div x$（4点）
→ $8x + 4$。$\dfrac{8x^2}{x} + \dfrac{4x}{x} = 8x + 4$。

(5) $(6a^2b - 12ab^2) \div (-3ab)$（4点）
→ $-2a + 4b$。$\dfrac{6a^2b}{-3ab} + \dfrac{-12ab^2}{-3ab} = -2a + 4b$。

(6) $(4x^3 - 2x^2 + 6x) \div 2x$（3点）
→ $2x^2 - x + 3$。各項を $2x$ で割る。

## 大問3: 多項式×多項式（異なる文字）（配点 20点）

次の式を展開しなさい。

(1) $(a + 2)(b + 5)$（3点）
→ $ab + 5a + 2b + 10$。$a \times b + a \times 5 + 2 \times b + 2 \times 5$。

(2) $(x + 3)(y + 7)$（3点）
→ $xy + 7x + 3y + 21$。4つの積を作る。

(3) $(a - 4)(b + 3)$（4点）
→ $ab + 3a - 4b - 12$。$a \times b + a \times 3 + (-4) \times b + (-4) \times 3$。

(4) $(x + 5)(y - 1)$（4点）
→ $xy - x + 5y - 5$。$x \times y + x \times (-1) + 5 \times y + 5 \times (-1)$。

(5) $(a - 2)(3 + b)$（3点）
→ $3a + ab - 6 - 2b$。$a \times 3 + a \times b + (-2) \times 3 + (-2) \times b$。

(6) $(x - a)(y + 3a)$（3点）
→ $xy + 3ax - ay - 3a^2$。$x \times y + x \times 3a + (-a) \times y + (-a) \times 3a$。

## 大問4: (x+a)(x+b) 型の展開（配点 20点）

次の式を展開しなさい。

(1) $(x + 2)(x + 3)$（3点）
→ $x^2 + 5x + 6$。$x^2 + 3x + 2x + 6 = x^2 + 5x + 6$。

(2) $(x + 4)(x + 5)$（3点）
→ $x^2 + 9x + 20$。$x^2 + 5x + 4x + 20 = x^2 + 9x + 20$。

(3) $(x - 3)(x + 7)$（4点）
→ $x^2 + 4x - 21$。$x^2 + 7x - 3x - 21 = x^2 + 4x - 21$。

(4) $(x + 1)(x - 5)$（4点）
→ $x^2 - 4x - 5$。$x^2 - 5x + x - 5 = x^2 - 4x - 5$。

(5) $(x - 2)(x - 6)$（3点）
→ $x^2 - 8x + 12$。$x^2 - 6x - 2x + 12 = x^2 - 8x + 12$。

(6) $(x - 4)(x + 8)$（3点）
→ $x^2 + 4x - 32$。$x^2 + 8x - 4x - 32 = x^2 + 4x - 32$。

## 大問5: 混合問題（配点 20点）

次の計算をしなさい。

(1) $-2(3x^2 - 4x + 1)$（3点）
→ $-6x^2 + 8x - 2$。各項に $-2$ をかける。

(2) $(10x^2y - 15xy^2) \div 5xy$（4点）
→ $2x - 3y$。$\dfrac{10x^2y}{5xy} - \dfrac{15xy^2}{5xy} = 2x - 3y$。

(3) $(x + 6)(x - 1)$（3点）
→ $x^2 + 5x - 6$。$x^2 - x + 6x - 6 = x^2 + 5x - 6$。

(4) $(x + 3)(x - 3)$（4点）
→ $x^2 - 9$。$x^2 - 3x + 3x - 9 = x^2 - 9$。同類項が打ち消し合う。

(5) $-5a(2a - 3b) + (9a^2 + 6ab) \div 3a$（6点）
→ $-10a^2 + 15ab + 3a + 2b$。第1項: $-10a^2 + 15ab$。第2項: $3a + 2b$。合わせて $-10a^2 + 15ab + 3a + 2b$。
