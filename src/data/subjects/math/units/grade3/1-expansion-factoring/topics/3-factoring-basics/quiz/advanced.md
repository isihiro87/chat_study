# 発展問題：因数分解の基本

## 発展1：共通因数＋公式の組み合わせ（配点20点）

次の式を因数分解しなさい。（まず共通因数をくくり出してから公式を使うこと）

(1) $2x^2 - 50$ を因数分解しなさい。（4点）

(2) $3a^2 - 12$ を因数分解しなさい。（4点）

(3) $5x^2 + 20x + 20$ を因数分解しなさい。（6点）

(4) $2x^2y - 8y$ を因数分解しなさい。（6点）

## 発展2：因数分解の公式の見分け（配点20点）

次の式はどの因数分解の公式にあてはまるか、公式名を答えてから因数分解しなさい。

(1) $x^2 - 12x + 36$（4点）

(2) $49a^2 - 16b^2$（4点）

(3) $4x^2 + 12xy + 9y^2$（4点）

(4) $81 - x^2$（4点）

(5) $64a^2 - 48ab + 9b^2$（4点）

## 発展3：間違い訂正問題（配点20点）

次の因数分解にはそれぞれ間違いがある。間違いを指摘し、正しく因数分解しなさい。

(1) 「$x^2 - 16 = (x - 4)^2$」（4点）

(2) 「$4x^2 + 4x + 1 = (4x + 1)^2$」（5点）

(3) 「$6x^2 - 12x = 6(x^2 - 2x)$」（5点）

(4) 「$x^2 + 10x + 25 = (x + 5)(x - 5)$」（6点）

## 発展4：置き換えによる因数分解（配点20点）

(1) $(x + y)^2 - 9$ を因数分解しなさい。（ヒント: $x + y = A$ と置く）（5点）

(2) $(2a - b)^2 - 16$ を因数分解しなさい。（5点）

(3) $(x + 3)^2 - (x - 1)^2$ を因数分解しなさい。（5点）

(4) $a^2 - 4a + 4 - b^2$ を因数分解しなさい。（ヒント: 最初の3項をまとめる）（5点）

## 発展5：総合・応用問題（配点20点）

(1) $x = 53$, $y = 47$ のとき、$x^2 - y^2$ の値を因数分解を利用して求めなさい。（5点）

(2) $101^2 - 99^2$ の値を因数分解を利用して暗算で求めなさい。（4点）

(3) $998^2 + 4 \times 998 + 4$ の値を因数分解を利用して求めなさい。（5点）

(4) 正方形の1辺の長さが $(3x + 2)$ cm のとき、面積を展開して求めなさい。また、$9x^2 + 12x + 4$ を因数分解して、1辺の長さを確認しなさい。（6点）
