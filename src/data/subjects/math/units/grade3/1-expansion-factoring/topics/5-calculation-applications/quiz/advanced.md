# 式の計算の利用 発展問題

## 問1: 整数の性質の証明（応用）

(1) 連続する3つの奇数の和が3の倍数であることを証明しなさい。
→ 真ん中の奇数を $2n+1$ とおくと、3つは $2n-1, 2n+1, 2n+3$。和は $(2n-1)+(2n+1)+(2n+3) = 6n+3 = 3(2n+1)$。$2n+1$ は整数だから3の倍数。

(2) 2つの奇数の積が奇数であることを証明しなさい。
→ 2つの奇数を $2m+1, 2n+1$（$m, n$ は整数）とおく。$(2m+1)(2n+1) = 4mn+2m+2n+1 = 2(2mn+m+n)+1$。$2mn+m+n$ は整数だから、これは奇数。

(3) 連続する2つの偶数の2乗の差が、2数の和の2倍であることを証明しなさい。
→ $2n, 2n+2$ とおく。$(2n+2)^2-(2n)^2 = \{(2n+2)+2n\}\{(2n+2)-2n\} = (4n+2) \times 2 = 2(4n+2)$。2数の和は $2n+(2n+2) = 4n+2$ だから、差は2数の和の2倍。

## 問2: 計算の工夫（発展）

(1) $997 \times 1003$ を工夫して計算しなさい。
→ $(1000-3)(1000+3) = 1000^2 - 9 = 999991$。

(2) $10.5^2 - 9.5^2$ を工夫して計算しなさい。
→ $(10.5+9.5)(10.5-9.5) = 20 \times 1 = 20$。

(3) $201^2 - 2 \times 201 + 1$ を工夫して計算しなさい。
→ $(201-1)^2 = 200^2 = 40000$。$a^2-2a+1 = (a-1)^2$ の公式を使う。

## 問3: 式の値（発展）

(1) $x=\dfrac{1}{2}$ のとき、$(2x+3)^2 - (2x-3)^2$ の値を求めよ。
→ 展開: $(2x+3+2x-3)(2x+3-2x+3) = 4x \times 6 = 24x$。$24 \times \dfrac{1}{2} = 12$。

(2) $a+b=7, a-b=3$ のとき、$a^2-b^2$ の値を求めよ。
→ $a^2-b^2 = (a+b)(a-b) = 7 \times 3 = 21$。

(3) $x+y=10, xy=21$ のとき、$x^2+2xy+y^2$ の値を求めよ。
→ $(x+y)^2 = 10^2 = 100$。

(4) $x+y=10, xy=21$ のとき、$x^2-2xy+y^2$ の値を求めよ。
→ $(x-y)^2 = (x+y)^2-4xy = 100-84 = 16$。

## 問4: 図形への応用（発展）

(1) 半径 $R$ の円から半径 $r$ の円をくり抜いたドーナツ形の面積を因数分解で表しなさい（$R > r$）。
→ $\pi R^2 - \pi r^2 = \pi(R^2-r^2) = \pi(R+r)(R-r)$。

(2) 1辺が $a$ の正方形の4つの角から、1辺が $b$ の正方形をくり抜いた面積を求めよ（$a > 2b$）。
→ $a^2 - 4b^2 = (a+2b)(a-2b)$。

(3) たて $(x+5)$、よこ $(x-5)$ の長方形と、1辺 $x$ の正方形の面積の差を求めよ。
→ $(x+5)(x-5) - x^2 = x^2-25-x^2 = -25$。長方形の方が25だけ小さい。

## 問5: 総合問題

(1) $n$ を自然数とするとき、$n^2+2n+1$ はどんな数の2乗か答え、理由を説明しなさい。
→ $(n+1)^2$。$n$ が自然数のとき $n+1$ も自然数だから、$(n+1)$ の2乗。つまり、ある自然数の次の自然数の2乗である。

(2) $99^2+2 \times 99+1$ を工夫して計算しなさい。
→ $(99+1)^2 = 100^2 = 10000$。$a^2+2a+1=(a+1)^2$。

(3) 2けたの自然数の十の位を $a$、一の位を $b$ とするとき、この数と十の位・一の位を入れ替えた数の差が9の倍数であることを証明しなさい。
→ 元の数は $10a+b$、入れ替えた数は $10b+a$。差は $(10a+b)-(10b+a) = 9a-9b = 9(a-b)$。$a-b$ は整数だから9の倍数。
