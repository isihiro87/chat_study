# 式の展開の基本 一問一答

## 展開の意味（用語）

Q: 「展開」とは何か。
A: かっこを外して、積の形の式をひとつの多項式に変形すること。

Q: 「分配法則」とは何か。
A: $a(b+c) = ab + ac$ のように、かっこの外の項をかっこの中の各項にかける法則。

Q: 「同類項」とは何か。
A: 文字の部分がまったく同じ項のこと。例: $3x$ と $5x$、$2x^2$ と $-7x^2$。

Q: 同類項をまとめるとはどういうことか。
A: 文字の部分が同じ項の係数どうしを足し引きすること。例: $3x + 5x = 8x$。

Q: $(a+b)(c+d)$ を展開すると、何個の項の積ができるか。
A: 4個。$ac, ad, bc, bd$ の4つの積ができる。

## 分配法則 a(b+c)（基本）

Q: $3(x + 4)$ を展開せよ。
A: $3x + 12$。$3 \times x + 3 \times 4 = 3x + 12$。

Q: $5(2x + 3)$ を展開せよ。
A: $10x + 15$。$5 \times 2x + 5 \times 3 = 10x + 15$。

Q: $4(3a - 2)$ を展開せよ。
A: $12a - 8$。$4 \times 3a + 4 \times (-2) = 12a - 8$。

Q: $2x(x + 3)$ を展開せよ。
A: $2x^2 + 6x$。$2x \times x + 2x \times 3 = 2x^2 + 6x$。

Q: $3a(2a + 5)$ を展開せよ。
A: $6a^2 + 15a$。$3a \times 2a + 3a \times 5 = 6a^2 + 15a$。

Q: $x(x + 7)$ を展開せよ。
A: $x^2 + 7x$。$x \times x + x \times 7 = x^2 + 7x$。

Q: $(3x + 5y) \times 2$ を展開せよ。
A: $6x + 10y$。$3x \times 2 + 5y \times 2 = 6x + 10y$。

Q: $4x(x - 5)$ を展開せよ。
A: $4x^2 - 20x$。$4x \times x + 4x \times (-5) = 4x^2 - 20x$。

## 符号に注意する展開 -a(b-c)

Q: $-2(x + 3)$ を展開せよ。
A: $-2x - 6$。$(-2) \times x + (-2) \times 3 = -2x - 6$。

Q: $-3(2x - 5)$ を展開せよ。
A: $-6x + 15$。$(-3) \times 2x + (-3) \times (-5) = -6x + 15$。

Q: $-x(x - 4)$ を展開せよ。
A: $-x^2 + 4x$。$(-x) \times x + (-x) \times (-4) = -x^2 + 4x$。

Q: $-3x(5x - 2)$ を展開せよ。
A: $-15x^2 + 6x$。$(-3x) \times 5x + (-3x) \times (-2) = -15x^2 + 6x$。

Q: $(3x + 5y) \times (-2y)$ を展開せよ。
A: $-6xy - 10y^2$。$3x \times (-2y) + 5y \times (-2y) = -6xy - 10y^2$。

Q: $-4a(3a + 2b)$ を展開せよ。
A: $-12a^2 - 8ab$。$(-4a) \times 3a + (-4a) \times 2b = -12a^2 - 8ab$。

Q: $-5(x^2 - 3x + 1)$ を展開せよ。
A: $-5x^2 + 15x - 5$。各項に $-5$ をかける。

Q: $(6a - 15) \times \dfrac{2}{3}a$ を展開せよ。
A: $4a^2 - 10a$。$6a \times \dfrac{2}{3}a + (-15) \times \dfrac{2}{3}a = 4a^2 - 10a$。

## 多項式÷単項式

Q: $(8x^2 + 4x) \div 2x$ を計算せよ。
A: $4x + 2$。$\dfrac{8x^2}{2x} + \dfrac{4x}{2x} = 4x + 2$。

Q: $(6a^2 + 9a) \div 3a$ を計算せよ。
A: $2a + 3$。$\dfrac{6a^2}{3a} + \dfrac{9a}{3a} = 2a + 3$。

Q: $(12x^2 - 8x) \div 4x$ を計算せよ。
A: $3x - 2$。$\dfrac{12x^2}{4x} - \dfrac{8x}{4x} = 3x - 2$。

Q: $(8x^2 + 4x) \div x$ を計算せよ。
A: $8x + 4$。$\dfrac{8x^2}{x} + \dfrac{4x}{x} = 8x + 4$。

Q: $(6a^2b - 12ab^2) \div (-3ab)$ を計算せよ。
A: $-2a + 4b$。$\dfrac{6a^2b}{-3ab} + \dfrac{-12ab^2}{-3ab} = -2a + 4b$。

Q: $(10x^2y - 15xy^2) \div 5xy$ を計算せよ。
A: $2x - 3y$。$\dfrac{10x^2y}{5xy} - \dfrac{15xy^2}{5xy} = 2x - 3y$。

Q: $(9a^2 - 6a) \div (-3a)$ を計算せよ。
A: $-3a + 2$。$\dfrac{9a^2}{-3a} + \dfrac{-6a}{-3a} = -3a + 2$。

Q: $(4x^3 - 2x^2 + 6x) \div 2x$ を計算せよ。
A: $2x^2 - x + 3$。各項を $2x$ で割る。

## 多項式×多項式（異なる文字）

Q: $(a + 2)(b + 5)$ を展開せよ。
A: $ab + 5a + 2b + 10$。$a \times b + a \times 5 + 2 \times b + 2 \times 5 = ab + 5a + 2b + 10$。

Q: $(x + 3)(y + 7)$ を展開せよ。
A: $xy + 7x + 3y + 21$。4つの積を作る。

Q: $(a - 4)(b + 3)$ を展開せよ。
A: $ab + 3a - 4b - 12$。$a \times b + a \times 3 + (-4) \times b + (-4) \times 3 = ab + 3a - 4b - 12$。

Q: $(x + 5)(y - 1)$ を展開せよ。
A: $xy - x + 5y - 5$。$x \times y + x \times (-1) + 5 \times y + 5 \times (-1) = xy - x + 5y - 5$。

Q: $(a - 2)(3 + b)$ を展開せよ。
A: $3a + ab - 6 - 2b$。$a \times 3 + a \times b + (-2) \times 3 + (-2) \times b = 3a + ab - 6 - 2b$。

Q: $(x - a)(y + 3a)$ を展開せよ。
A: $xy + 3ax - ay - 3a^2$。$x \times y + x \times 3a + (-a) \times y + (-a) \times 3a = xy + 3ax - ay - 3a^2$。

## 同類項のまとめ（(x+a)(x+b) 型）

Q: $(x + 2)(x + 3)$ を展開せよ。
A: $x^2 + 5x + 6$。$x^2 + 3x + 2x + 6 = x^2 + 5x + 6$。

Q: $(x + 4)(x + 5)$ を展開せよ。
A: $x^2 + 9x + 20$。$x^2 + 5x + 4x + 20 = x^2 + 9x + 20$。

Q: $(x - 3)(x + 7)$ を展開せよ。
A: $x^2 + 4x - 21$。$x^2 + 7x - 3x - 21 = x^2 + 4x - 21$。

Q: $(x + 1)(x - 5)$ を展開せよ。
A: $x^2 - 4x - 5$。$x^2 - 5x + x - 5 = x^2 - 4x - 5$。

Q: $(x - 2)(x - 6)$ を展開せよ。
A: $x^2 - 8x + 12$。$x^2 - 6x - 2x + 12 = x^2 - 8x + 12$。

Q: $(x + 3)(x - 3)$ を展開せよ。
A: $x^2 - 9$。$x^2 - 3x + 3x - 9 = x^2 - 9$。同類項が消える。

Q: $(x - 4)(x + 8)$ を展開せよ。
A: $x^2 + 4x - 32$。$x^2 + 8x - 4x - 32 = x^2 + 4x - 32$。

Q: $(x + 6)(x - 1)$ を展開せよ。
A: $x^2 + 5x - 6$。$x^2 - x + 6x - 6 = x^2 + 5x - 6$。
