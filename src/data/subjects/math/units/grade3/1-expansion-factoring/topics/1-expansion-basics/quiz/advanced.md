# 式の展開の基本 発展問題

## 問1: 複数の展開と同類項のまとめ

次の計算をしなさい。

(1) $3(x + 2) + 2(x - 5)$
→ $3x + 6 + 2x - 10 = 5x - 4$。それぞれ展開してから同類項をまとめる。

(2) $5(2a - 1) - 3(a + 4)$
→ $10a - 5 - 3a - 12 = 7a - 17$。マイナスの分配に注意。

(3) $2x(x + 3) - x(2x - 1)$
→ $2x^2 + 6x - 2x^2 + x = 7x$。$2x^2$ が打ち消し合う。

(4) $-3a(2a - 5) + 2a(3a + 1)$
→ $-6a^2 + 15a + 6a^2 + 2a = 17a$。$a^2$ の項が消え $a$ の項だけ残る。

## 問2: 多項式÷単項式の応用

次の計算をしなさい。

(1) $(9a^2b - 6ab^2 + 3ab) \div 3ab$
→ $3a - 2b + 1$。3項ともそれぞれ $3ab$ で割る。

(2) $(8x^3 - 12x^2 + 4x) \div (-2x)$
→ $-4x^2 + 6x - 2$。各項を $-2x$ で割る。符号が反転する。

(3) $(15a^2b^2 - 10ab^3) \div 5ab^2$
→ $3a - 2b$。$\dfrac{15a^2b^2}{5ab^2} - \dfrac{10ab^3}{5ab^2} = 3a - 2b$。

(4) $(18x^2y - 24xy^2 + 6xy) \div (-6xy)$
→ $-3x + 4y - 1$。各項を $-6xy$ で割る。

## 問3: 異なる文字の展開（応用）

次の式を展開しなさい。

(1) $(2a + 3)(b - 4)$
→ $2ab - 8a + 3b - 12$。$2a \times b + 2a \times (-4) + 3 \times b + 3 \times (-4)$。

(2) $(x - 2y)(3x + y)$
→ $3x^2 + xy - 6xy - 2y^2 = 3x^2 - 5xy - 2y^2$。同類項をまとめる。

(3) $(a + b)(a - 2b)$
→ $a^2 - 2ab + ab - 2b^2 = a^2 - ab - 2b^2$。同類項をまとめる。

(4) $(3x + 2y)(x - 4y)$
→ $3x^2 - 12xy + 2xy - 8y^2 = 3x^2 - 10xy - 8y^2$。

## 問4: 展開と除法の混合

次の計算をしなさい。

(1) $(x + 3)(x + 2) - (x + 1)(x + 4)$
→ $(x^2 + 5x + 6) - (x^2 + 5x + 4) = 2$。展開してから引く。

(2) $(x + 5)(x - 2) - (x - 3)(x + 4)$
→ $(x^2 + 3x - 10) - (x^2 + x - 12) = 2x + 2$。

(3) $2x(x + 4) + (6x^2 - 4x) \div 2x$
→ $(2x^2 + 8x) + (3x - 2) = 2x^2 + 11x - 2$。展開と除法をそれぞれ計算してまとめる。

(4) $(x + 2)(x - 5) + (8x^2 + 4x) \div 2x$
→ $(x^2 - 3x - 10) + (4x + 2) = x^2 + x - 8$。

## 問5: 間違い探しと証明的問題

(1) 次の計算には間違いがある。間違いを見つけ、正しい答えを求めなさい。
$-2(3x - 5) = -6x - 10$
→ 間違い。$(-2) \times (-5) = +10$ なので、正しくは $-6x + 10$。マイナスどうしの積は正になる。

(2) 次の計算には間違いがある。間違いを見つけ、正しい答えを求めなさい。
$(6a^2b - 12ab^2) \div (-3ab) = 2a + 4b$
→ 間違い。$\dfrac{6a^2b}{-3ab} = -2a$（符号に注意）。正しくは $-2a + 4b$。

(3) $(x + a)(x - a)$ を展開すると $x^2 - a^2$ になることを示しなさい。
→ $(x + a)(x - a) = x^2 - ax + ax - a^2 = x^2 - a^2$。$-ax$ と $+ax$ が打ち消し合うため、中間の項が消えて $x^2 - a^2$ になる。

(4) $a = 3, b = -2$ のとき、$(a + b)(a - 2b) - a(a - 3b)$ の値を求めなさい。まず式を簡単にしてから代入すること。
→ 展開: $(a^2 - 2ab + ab - 2b^2) - (a^2 - 3ab) = a^2 - ab - 2b^2 - a^2 + 3ab = 2ab - 2b^2$。$a = 3, b = -2$ を代入: $2 \times 3 \times (-2) - 2 \times (-2)^2 = -12 - 8 = -20$。
