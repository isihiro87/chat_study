# 公式を使った因数分解 大問小問

## 大問1: 積が+のパターン（配点 20点）

次の式を因数分解しなさい。

(1) $x^2 + 6x + 8$（3点）
→ $(x+2)(x+4)$。かけて $8$、足して $6$ → $2$ と $4$。

(2) $y^2 + 13y + 42$（3点）
→ $(y+6)(y+7)$。かけて $42$、足して $13$ → $6$ と $7$。

(3) $x^2 + 12x + 27$（3点）
→ $(x+3)(x+9)$。かけて $27$、足して $12$ → $3$ と $9$。

(4) $t^2 - 8t + 15$（4点）
→ $(t-3)(t-5)$。かけて $15$、足して $-8$ → $-3$ と $-5$。両方マイナス。

(5) $a^2 - 10a + 24$（4点）
→ $(a-4)(a-6)$。かけて $24$、足して $-10$ → $-4$ と $-6$。

(6) $x^2 - 16x + 63$（3点）
→ $(x-7)(x-9)$。かけて $63$、足して $-16$ → $-7$ と $-9$。

## 大問2: 積が−のパターン（配点 20点）

次の式を因数分解しなさい。

(1) $x^2 + 11x - 12$（3点）
→ $(x-1)(x+12)$。かけて $-12$、足して $11$ → $-1$ と $12$。

(2) $a^2 + 3a - 28$（3点）
→ $(a-4)(a+7)$。かけて $-28$、足して $3$ → $-4$ と $7$。

(3) $x^2 + 6x - 72$（4点）
→ $(x-6)(x+12)$。かけて $-72$、足して $6$ → $-6$ と $12$。

(4) $y^2 - 3y - 18$（4点）
→ $(y-6)(y+3)$。かけて $-18$、足して $-3$ → $-6$ と $3$。

(5) $x^2 - 4x - 45$（3点）
→ $(x-9)(x+5)$。かけて $-45$、足して $-4$ → $-9$ と $5$。

(6) $x^2 - 7x - 120$（3点）
→ $(x-15)(x+8)$。かけて $-120$、足して $-7$ → $-15$ と $8$。

## 大問3: 公式の選択（配点 20点）

次の式を因数分解しなさい。どの公式を使うか判断すること。

(1) $9x^2 - 25$（3点）
→ $(3x+5)(3x-5)$。$a^2-b^2=(a+b)(a-b)$ を使用。$(3x)^2-5^2$。

(2) $a^2 + 12a + 36$（3点）
→ $(a+6)^2$。$a^2+2ab+b^2=(a+b)^2$ を使用。$a^2+2 \times 6 \times a+6^2$。

(3) $x^2 - 14x + 49$（4点）
→ $(x-7)^2$。$a^2-2ab+b^2=(a-b)^2$ を使用。$x^2-2 \times 7 \times x+7^2$。

(4) $x^2 + 9x + 14$（3点）
→ $(x+2)(x+7)$。$x^2+(a+b)x+ab=(x+a)(x+b)$ を使用。

(5) $4a^2 - 9b^2$（4点）
→ $(2a+3b)(2a-3b)$。$(2a)^2-(3b)^2$ で和と差の積。

(6) $2ax + 2ay - 6a$（3点）
→ $2a(x+y-3)$。共通因数 $2a$ でくくる。

## 大問4: 共通因数→公式の2段階因数分解（配点 20点）

次の式を因数分解しなさい。

(1) $3x^2 + 9x + 6$（3点）
→ $3(x+1)(x+2)$。$3$ でくくると $3(x^2+3x+2)$。かけて $2$、足して $3$。

(2) $2x^2 + 6x - 8$（4点）
→ $2(x-1)(x+4)$。$2$ でくくると $2(x^2+3x-4)$。かけて $-4$、足して $3$。

(3) $-5a^2 + 10a - 5$（4点）
→ $-5(a-1)^2$。$-5$ でくくると $-5(a^2-2a+1)$。$(a-1)^2$ の形。

(4) $a^2b - b$（3点）
→ $b(a+1)(a-1)$。$b$ でくくると $b(a^2-1)$。$a^2-1^2$ で和と差の積。

(5) $ax^2 - 9ax + 18a$（3点）
→ $a(x-3)(x-6)$。$a$ でくくると $a(x^2-9x+18)$。かけて $18$、足して $-9$。

(6) $ax^2 - 8ax + 16a$（3点）
→ $a(x-4)^2$。$a$ でくくると $a(x^2-8x+16)$。$(x-4)^2$ の形。

## 大問5: 置き換えを使った因数分解（配点 20点）

次の式を因数分解しなさい。

(1) $y(x-3) + 4(x-3)$（3点）
→ $(x-3)(y+4)$。$(x-3)$ が共通因数。

(2) $x(y+5) - (y+5)$（3点）
→ $(y+5)(x-1)$。$(y+5)$ が共通因数。$-(y+5) = (-1)(y+5)$ に注意。

(3) $ax - ay + bx - by$（4点）
→ $(a+b)(x-y)$。$a(x-y)+b(x-y)$ とグループ分けする。

(4) $(x+2)^2 + 5(x+2) + 6$（4点）
→ $(x+4)(x+5)$。$M=x+2$ とおくと $M^2+5M+6=(M+2)(M+3)$。戻して $(x+4)(x+5)$。

(5) $(x-6)^2 + 2(x-6) - 24$（3点）
→ $(x-10)(x-2)$。$M=x-6$ とおくと $M^2+2M-24=(M-4)(M+6)$。戻して $(x-10)(x-2)$。
