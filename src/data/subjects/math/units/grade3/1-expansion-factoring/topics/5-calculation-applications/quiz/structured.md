# 式の計算の利用 大問小問

## 大問1: 整数の性質の証明（配点 20点）

次の問いに答えなさい。

(1) 連続する2つの奇数を、整数 $n$ を使って表しなさい。（3点）
→ $2n-1, 2n+1$。奇数は偶数 $\pm 1$ で表す。

(2) 連続する2つの奇数の2乗の和から2をひいた式を、$n$ を使って立てなさい。（3点）
→ $(2n-1)^2 + (2n+1)^2 - 2$。

(3) (2)の式を展開・整理しなさい。（4点）
→ $(4n^2-4n+1) + (4n^2+4n+1) - 2 = 8n^2$。

(4) (3)の結果から、「連続する2つの奇数の2乗の和から2をひいた数は8の倍数である」ことを説明しなさい。（4点）
→ $8n^2 = 8 \times n^2$。$n^2$ は整数だから、$8n^2$ は8の倍数である。

(5) 連続する3つの整数を $n-1, n, n+1$ とおくとき、最大の数の2乗から最小の数の2乗をひいた差を求め、真ん中の数の何倍になるか答えなさい。（6点）
→ $(n+1)^2-(n-1)^2 = \{(n+1)+(n-1)\}\{(n+1)-(n-1)\} = 2n \times 2 = 4n$。真ん中の数 $n$ の4倍。

## 大問2: 因数分解を利用した数の計算（配点 20点）

因数分解の公式を利用して、次の計算をしなさい。

(1) $35^2 - 25^2$（3点）
→ $(35+25)(35-25) = 60 \times 10 = 600$。

(2) $52^2 - 48^2$（3点）
→ $(52+48)(52-48) = 100 \times 4 = 400$。

(3) $59 \times 61$（4点）
→ $(60-1)(60+1) = 60^2 - 1^2 = 3600 - 1 = 3599$。

(4) $51 \times 49$（4点）
→ $(50+1)(50-1) = 50^2 - 1^2 = 2500 - 1 = 2499$。

(5) $10.3 \times 9.7$（6点）
→ $(10+0.3)(10-0.3) = 10^2 - 0.3^2 = 100 - 0.09 = 99.91$。

## 大問3: 展開の公式を利用した数の計算（配点 20点）

展開の公式を利用して、次の計算をしなさい。

(1) $101^2$（3点）
→ $(100+1)^2 = 10000 + 200 + 1 = 10201$。

(2) $99^2$（3点）
→ $(100-1)^2 = 10000 - 200 + 1 = 9801$。

(3) $39^2$（4点）
→ $(40-1)^2 = 1600 - 80 + 1 = 1521$。

(4) $202^2$（4点）
→ $(200+2)^2 = 40000 + 800 + 4 = 40804$。

(5) $998^2$（6点）
→ $(1000-2)^2 = 1000000 - 4000 + 4 = 996004$。

## 大問4: 式の値（配点 20点）

次の式の値を求めなさい。

(1) $x=48, y=38$ のとき、$x^2-y^2$ の値（4点）
→ $(x+y)(x-y) = (48+38)(48-38) = 86 \times 10 = 860$。

(2) $x=53, y=47$ のとき、$x^2-y^2$ の値（4点）
→ $(x+y)(x-y) = (53+47)(53-47) = 100 \times 6 = 600$。

(3) $x=96$ のとき、$x^2+8x+16$ の値（4点）
→ $(x+4)^2 = (96+4)^2 = 100^2 = 10000$。

(4) $x=104$ のとき、$x^2-8x+16$ の値（4点）
→ $(x-4)^2 = (104-4)^2 = 100^2 = 10000$。

(5) $x=15$ のとき、$(x+3)(x-6)-x(x-5)$ の値（4点）
→ 展開: $x^2-3x-18-x^2+5x = 2x-18$。$2 \times 15 - 18 = 12$。

## 大問5: 図形への応用（配点 20点）

(1) 1辺が $(a+b)$ の正方形の面積から、1辺が $(a-b)$ の正方形の面積を引くといくらか。（4点）
→ $(a+b)^2 - (a-b)^2 = (a^2+2ab+b^2)-(a^2-2ab+b^2) = 4ab$。

(2) 幅 $a$ の道が、たて $p$、よこ $q$ の長方形の花壇のまわりにあるとき、道の面積を求めなさい。（5点）
→ 外側の長方形は $(p+2a) \times (q+2a)$、内側は $pq$。差は $(p+2a)(q+2a)-pq = 2a(p+q+2a) = 2a(p+q)+4a^2$。道の真ん中を通る線の長さ $\ell = 2(p+a)+2(q+a) = 2(p+q+2a)$ とすると $S = a\ell$。

(3) 半径 $R$ のおうぎ形（中心角 $90°$）の面積から、半径 $r$ のおうぎ形（中心角 $90°$）の面積を引いた値を因数分解しなさい（$R > r$）。（4点）
→ $\dfrac{1}{4}\pi R^2 - \dfrac{1}{4}\pi r^2 = \dfrac{\pi}{4}(R^2-r^2) = \dfrac{\pi}{4}(R+r)(R-r)$。

(4) 外側の正方形の1辺が $2a+b$、内側の正方形の1辺が $b$ であるとき、色つき部分の面積を求めなさい。（3点）
→ $(2a+b)^2 - b^2 = (2a+b+b)(2a+b-b) = (2a+2b)(2a) = 4a(a+b)$。

(5) 1辺 $x$ の正方形の各辺を $3$ ずつ長くした正方形と、元の正方形の面積の差を求めなさい。（4点）
→ $(x+3)^2 - x^2 = x^2+6x+9-x^2 = 6x+9 = 3(2x+3)$。
