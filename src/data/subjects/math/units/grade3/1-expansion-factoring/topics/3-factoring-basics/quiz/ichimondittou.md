# 一問一答：因数分解の基本

## 因数と因数分解の意味

Q1. 一つの式がいくつかの式の積の形に表されるとき、その一つひとつの式を何というか。
A1. 因数

Q2. 多項式をいくつかの因数の積の形に表すことを何というか。
A2. 因数分解

Q3. 因数分解は何の逆の作業か。
A3. 展開の逆の作業

Q4. $a(b + c) = ab + ac$ で、展開はどちらの向きか。
A4. 左辺 → 右辺（かっこを外す向き）

Q5. $ab + ac = a(b + c)$ で、因数分解はどちらの向きか。
A5. 左辺 → 右辺（かっこにまとめる向き）

Q6. 因数分解した結果が正しいか確かめるには、どうすればよいか。
A6. 展開して元の式に戻るか確認する

Q7. $3(x + 5)$ の因数をすべて答えなさい。
A7. $3$ と $(x + 5)$

Q8. $(x + 2)(x - 3)$ の因数をすべて答えなさい。
A8. $(x + 2)$ と $(x - 3)$

## 共通因数の見つけ方

Q9. すべての項に共通してふくまれている因数を何というか。
A9. 共通因数

Q10. $6x + 9$ の共通因数は何か。
A10. $3$

Q11. $x^2 + 3x$ の共通因数は何か。
A11. $x$

Q12. $4x^2 + 8x$ の共通因数は何か。
A12. $4x$

Q13. $2x^2y - 6xy^2$ の共通因数は何か。
A13. $2xy$

Q14. $6a^2b + 9ab^2 - 3ab$ の共通因数は何か。
A14. $3ab$

## 共通因数のくくり出し（数の共通因数）

Q15. $6x + 9$ を因数分解しなさい。
A15. $3(2x + 3)$

Q16. $5ax + 10xy$ を因数分解しなさい。
A16. $5x(a + 2y)$

Q17. $6x^2 - 3x$ を因数分解しなさい。
A17. $3x(2x - 1)$

Q18. $8x^2y - 12xy^2$ を因数分解しなさい。
A18. $4xy(2x - 3y)$

Q19. $2x^2 + 4xy - 6x$ を因数分解しなさい。
A19. $2x(x + 2y - 3)$

Q20. $8x^2y - 12xy^2 + 6xy$ を因数分解しなさい。
A20. $2xy(4x - 6y + 3)$

## 共通因数のくくり出し（文字の共通因数）

Q21. $3x^2 + 6x$ を因数分解しなさい。
A21. $3x(x + 2)$

Q22. $x^2 - 3xy + 7x$ を因数分解しなさい。
A22. $x(x - 3y + 7)$

Q23. $2x^2y - 6xy^2$ を因数分解しなさい。
A23. $2xy(x - 3y)$

Q24. $a^2b + ab^2$ を因数分解しなさい。
A24. $ab(a + b)$

## 和と差の積の因数分解 $a^2 - b^2$

Q25. $a^2 - b^2$ を因数分解する公式を答えなさい。
A25. $a^2 - b^2 = (a + b)(a - b)$

Q26. $x^2 - 25$ を因数分解しなさい。
A26. $(x + 5)(x - 5)$

Q27. $a^2 - 49$ を因数分解しなさい。
A27. $(a + 7)(a - 7)$

Q28. $a^2 - 64$ を因数分解しなさい。
A28. $(a + 8)(a - 8)$

Q29. $4 - y^2$ を因数分解しなさい。
A29. $(2 + y)(2 - y)$

Q30. $100 - y^2$ を因数分解しなさい。
A30. $(10 + y)(10 - y)$

Q31. $x^2 - \frac{b^2}{9}$ を因数分解しなさい。
A31. $(x + \frac{b}{3})(x - \frac{b}{3})$

Q32. $4a^2 - 81$ を因数分解しなさい。
A32. $(2a + 9)(2a - 9)$

Q33. $25x^2 - 4$ を因数分解しなさい。
A33. $(5x + 2)(5x - 2)$

Q34. $9x^2 - 25$ を因数分解しなさい。
A34. $(3x + 5)(3x - 5)$

Q35. $36b^2 - 25a^2$ を因数分解しなさい。
A35. $(6b + 5a)(6b - 5a)$

Q36. $16b^2 - 9a^2$ を因数分解しなさい。
A36. $(4b + 3a)(4b - 3a)$

## 平方の公式による因数分解

Q37. $a^2 + 2ab + b^2$ を因数分解する公式を答えなさい。
A37. $a^2 + 2ab + b^2 = (a + b)^2$

Q38. $a^2 - 2ab + b^2$ を因数分解する公式を答えなさい。
A38. $a^2 - 2ab + b^2 = (a - b)^2$

Q39. $a^2 + 10a + 25$ を因数分解しなさい。
A39. $(a + 5)^2$

Q40. $x^2 + 16x + 64$ を因数分解しなさい。
A40. $(x + 8)^2$

Q41. $a^2 - 6a + 9$ を因数分解しなさい。
A41. $(a - 3)^2$

Q42. $y^2 - 18y + 81$ を因数分解しなさい。
A42. $(y - 9)^2$

Q43. $x^2 + 14xy + 49y^2$ を因数分解しなさい。
A43. $(x + 7y)^2$

## 係数付きの因数分解

Q44. $9x^2 + 6x + 1$ を因数分解しなさい。
A44. $(3x + 1)^2$

Q45. $4a^2 - 20ab + 25b^2$ を因数分解しなさい。
A45. $(2a - 5b)^2$

Q46. $4y^2 - 4y + 1$ を因数分解しなさい。
A46. $(2y - 1)^2$

Q47. $16x^2 - 24xy + 9y^2$ を因数分解しなさい。
A47. $(4x - 3y)^2$

Q48. $25a^2 + 30ab + 9b^2$ を因数分解しなさい。
A48. $(5a + 3b)^2$

## 因数分解の検算

Q49. $x^2 - 9 = (x + 3)(x - 3)$ は正しいか。展開して確かめなさい。
A49. $(x + 3)(x - 3) = x^2 - 9$ となるので正しい。

Q50. $x^2 + 6x + 9 = (x + 3)^2$ は正しいか。展開して確かめなさい。
A50. $(x + 3)^2 = x^2 + 6x + 9$ となるので正しい。
