# 公式を使った因数分解 一問一答

## 「かけて○足して△」テクニック（用語・考え方）

Q: $x^2+bx+c$ を因数分解するとき、どんな2つの数を探すか。
A: かけて $c$（定数項）、足して $b$（$x$ の係数）になる2つの数を探す。

Q: $x^2+(a+b)x+ab$ を因数分解する公式を書け。
A: $x^2+(a+b)x+ab = (x+a)(x+b)$。

Q: 「かけて12、足して7」になる2つの数は何か。
A: $3$ と $4$。$3 \times 4 = 12$、$3 + 4 = 7$。

Q: 「かけて18、足して11」になる2つの数は何か。
A: $2$ と $9$。$2 \times 9 = 18$、$2 + 9 = 11$。

Q: 「かけて20、足して9」になる2つの数は何か。
A: $4$ と $5$。$4 \times 5 = 20$、$4 + 5 = 9$。

## 積が+、和が+のパターン

Q: $x^2 + 6x + 8$ を因数分解せよ。
A: $(x+2)(x+4)$。かけて $8$、足して $6$ → $2$ と $4$。

Q: $y^2 + 13y + 42$ を因数分解せよ。
A: $(y+6)(y+7)$。かけて $42$、足して $13$ → $6$ と $7$。

Q: $x^2 + 12x + 27$ を因数分解せよ。
A: $(x+3)(x+9)$。かけて $27$、足して $12$ → $3$ と $9$。

Q: $x^2 + 7x + 12$ を因数分解せよ。
A: $(x+3)(x+4)$。かけて $12$、足して $7$ → $3$ と $4$。

Q: $x^2 + 9x + 14$ を因数分解せよ。
A: $(x+2)(x+7)$。かけて $14$、足して $9$ → $2$ と $7$。

Q: $a^2 + 11a + 30$ を因数分解せよ。
A: $(a+5)(a+6)$。かけて $30$、足して $11$ → $5$ と $6$。

## 積が+、和が−のパターン

Q: 定数項が+で $x$ の係数が−のとき、2つの数の符号はどうなるか。
A: 2つとも負の数。かけてプラス・足してマイナスだから。

Q: $t^2 - 8t + 15$ を因数分解せよ。
A: $(t-3)(t-5)$。かけて $15$、足して $-8$ → $-3$ と $-5$。

Q: $a^2 - 10a + 24$ を因数分解せよ。
A: $(a-4)(a-6)$。かけて $24$、足して $-10$ → $-4$ と $-6$。

Q: $x^2 - 16x + 63$ を因数分解せよ。
A: $(x-7)(x-9)$。かけて $63$、足して $-16$ → $-7$ と $-9$。

Q: $x^2 - 11x + 28$ を因数分解せよ。
A: $(x-4)(x-7)$。かけて $28$、足して $-11$ → $-4$ と $-7$。

Q: $y^2 - 9y + 20$ を因数分解せよ。
A: $(y-4)(y-5)$。かけて $20$、足して $-9$ → $-4$ と $-5$。

## 積が−のパターン

Q: 定数項が−のとき、2つの数の符号はどうなるか。
A: 片方がプラス、片方がマイナス。かけてマイナスになるから。

Q: $x^2 + 11x - 12$ を因数分解せよ。
A: $(x-1)(x+12)$。かけて $-12$、足して $11$ → $-1$ と $12$。

Q: $a^2 + 3a - 28$ を因数分解せよ。
A: $(a-4)(a+7)$。かけて $-28$、足して $3$ → $-4$ と $7$。

Q: $x^2 + 6x - 72$ を因数分解せよ。
A: $(x-6)(x+12)$。かけて $-72$、足して $6$ → $-6$ と $12$。

Q: $y^2 - 3y - 18$ を因数分解せよ。
A: $(y-6)(y+3)$。かけて $-18$、足して $-3$ → $-6$ と $3$。

Q: $x^2 - 4x - 45$ を因数分解せよ。
A: $(x-9)(x+5)$。かけて $-45$、足して $-4$ → $-9$ と $5$。

Q: $x^2 - 7x - 120$ を因数分解せよ。
A: $(x-15)(x+8)$。かけて $-120$、足して $-7$ → $-15$ と $8$。

Q: $x^2 + 2x - 35$ を因数分解せよ。
A: $(x-5)(x+7)$。かけて $-35$、足して $2$ → $-5$ と $7$。

## 因数分解の公式選択

Q: $9x^2 - 25$ はどの公式で因数分解するか。
A: $a^2-b^2=(a+b)(a-b)$（和と差の積）。$(3x)^2 - 5^2 = (3x+5)(3x-5)$。

Q: $a^2 + 12a + 36$ はどの公式で因数分解するか。
A: $a^2+2ab+b^2=(a+b)^2$。$a^2+2 \times 6 \times a+6^2 = (a+6)^2$。

Q: $x^2 + 9x + 14$ はどの公式で因数分解するか。
A: $x^2+(a+b)x+ab=(x+a)(x+b)$。かけて $14$、足して $9$ → $(x+2)(x+7)$。

Q: $x^2 - 14x + 49$ はどの公式で因数分解するか。
A: $a^2-2ab+b^2=(a-b)^2$。$x^2-2 \times 7 \times x+7^2 = (x-7)^2$。

Q: $4a^2 - 9b^2$ を因数分解せよ。
A: $(2a+3b)(2a-3b)$。$(2a)^2-(3b)^2$ の形で和と差の積を使う。

Q: $x^2 + 20x + 100$ を因数分解せよ。
A: $(x+10)^2$。$x^2+2 \times 10 \times x+10^2$ の形。

Q: $2ax + 2ay - 6a$ を因数分解せよ。
A: $2a(x+y-3)$。共通因数 $2a$ でくくる。

## 共通因数→公式の2段階因数分解

Q: $3x^2 + 9x + 6$ を因数分解せよ。
A: $3(x+1)(x+2)$。まず $3$ でくくると $3(x^2+3x+2)$。かけて $2$、足して $3$ → $(x+1)(x+2)$。

Q: $2x^2 + 6x - 8$ を因数分解せよ。
A: $2(x-1)(x+4)$。まず $2$ でくくると $2(x^2+3x-4)$。かけて $-4$、足して $3$ → $(x-1)(x+4)$。

Q: $-5a^2 + 10a - 5$ を因数分解せよ。
A: $-5(a-1)^2$。まず $-5$ でくくると $-5(a^2-2a+1)$。$(a-1)^2$ の形。

Q: $a^2b - b$ を因数分解せよ。
A: $b(a+1)(a-1)$。まず $b$ でくくると $b(a^2-1)$。$a^2-1^2=(a+1)(a-1)$。

Q: $ax^2 - 9ax + 18a$ を因数分解せよ。
A: $a(x-3)(x-6)$。まず $a$ でくくると $a(x^2-9x+18)$。かけて $18$、足して $-9$ → $(x-3)(x-6)$。

Q: $ax^2 - 8ax + 16a$ を因数分解せよ。
A: $a(x-4)^2$。まず $a$ でくくると $a(x^2-8x+16)$。$x^2-2 \times 4 \times x+4^2=(x-4)^2$。

Q: 2段階因数分解で最初にやるべきことは何か。
A: 全ての項に共通する因数（共通因数）をくくり出すこと。

## 置き換えを使った因数分解

Q: $y(x-3) + 4(x-3)$ を因数分解せよ。
A: $(x-3)(y+4)$。$(x-3)$ が共通因数。

Q: $x(y+5) - (y+5)$ を因数分解せよ。
A: $(y+5)(x-1)$。$(y+5)$ が共通因数。

Q: $ax - ay + bx - by$ を因数分解せよ。
A: $(a+b)(x-y)$。$a(x-y)+b(x-y)=(a+b)(x-y)$。グループ分けして共通因数を見つける。

Q: $(x+2)^2 + 5(x+2) + 6$ を因数分解せよ。
A: $(x+4)(x+5)$。$M=x+2$ とおくと $M^2+5M+6=(M+2)(M+3)$。$M$ を戻して $(x+4)(x+5)$。

Q: $(x-6)^2 + 2(x-6) - 24$ を因数分解せよ。
A: $(x-10)(x-2)$。$M=x-6$ とおくと $M^2+2M-24=(M-4)(M+6)$。$M$ を戻して $(x-10)(x-2)$。

Q: $(x+y)^2 - 6(x+y) + 9$ を因数分解せよ。
A: $(x+y-3)^2$。$M=x+y$ とおくと $M^2-6M+9=(M-3)^2$。$M$ を戻して $(x+y-3)^2$。

Q: $ax - x + a - 1$ を因数分解せよ。
A: $(a-1)(x+1)$。$(ax-x)+(a-1)=x(a-1)+(a-1)=(a-1)(x+1)$。

Q: 置き換えを使う因数分解で、置き換えた後に必ずやるべきことは何か。
A: 因数分解が終わったら、置き換えた文字をもとの式に戻すこと。
