# 一問一答：乗法の公式

## 公式の名前と式

Q1. $(x+a)(x+b)$ を展開する公式を答えなさい。
A1. $x^2 + (a+b)x + ab$

Q2. $(a+b)^2$ を展開する公式を答えなさい。
A2. $a^2 + 2ab + b^2$

Q3. $(a-b)^2$ を展開する公式を答えなさい。
A3. $a^2 - 2ab + b^2$

Q4. $(a+b)(a-b)$ を展開する公式を答えなさい。
A4. $a^2 - b^2$

Q5. $a^2 - b^2$ という結果になる展開公式は何か。
A5. $(a+b)(a-b) = a^2 - b^2$（和と差の積）

Q6. $(a+b)^2$ の展開で、真ん中の項は何か。
A6. $2ab$

Q7. $(a-b)^2$ の展開で、最後の項 $b^2$ の符号は＋か−か。
A7. ＋（$(-b)^2 = b^2$ だから）

Q8. $(x+a)(x+b)$ の展開で、$x$ の係数はどう求めるか。
A8. $a + b$（$a$ と $b$ を足す）

## 公式1: (x+a)(x+b) の展開

Q9. $(x+2)(x+5)$ を展開しなさい。
A9. $x^2 + 7x + 10$（足して7、かけて10）

Q10. $(x+7)(x-3)$ を展開しなさい。
A10. $x^2 + 4x - 21$（足して4、かけて−21）

Q11. $(x-4)(x+9)$ を展開しなさい。
A11. $x^2 + 5x - 36$（足して5、かけて−36）

Q12. $(x-6)(x-2)$ を展開しなさい。
A12. $x^2 - 8x + 12$（足して−8、かけて12）

Q13. $(x+3)(x-8)$ を展開しなさい。
A13. $x^2 - 5x - 24$（足して−5、かけて−24）

Q14. $(x-1)(x+7)$ を展開しなさい。
A14. $x^2 + 6x - 7$（足して6、かけて−7）

## 公式2: (a+b)² の展開

Q15. $(x+2)^2$ を展開しなさい。
A15. $x^2 + 4x + 4$

Q16. $(y+6)^2$ を展開しなさい。
A16. $y^2 + 12y + 36$

Q17. $(b+9)^2$ を展開しなさい。
A17. $b^2 + 18b + 81$

Q18. $(x+1)^2$ を展開しなさい。
A18. $x^2 + 2x + 1$

Q19. $(a+10)^2$ を展開しなさい。
A19. $a^2 + 20a + 100$

## 公式3: (a-b)² の展開

Q20. $(x-5)^2$ を展開しなさい。
A20. $x^2 - 10x + 25$

Q21. $(a-8)^2$ を展開しなさい。
A21. $a^2 - 16a + 64$

Q22. $(4-a)^2$ を展開しなさい。
A22. $16 - 8a + a^2$（$a = 4$, $b = a$ とおく）

Q23. $(x-7)^2$ を展開しなさい。
A23. $x^2 - 14x + 49$

Q24. $(y-3)^2$ を展開しなさい。
A24. $y^2 - 6y + 9$

## 公式4: (a+b)(a-b) の展開

Q25. $(x+3)(x-3)$ を展開しなさい。
A25. $x^2 - 9$

Q26. $(x-9)(x+9)$ を展開しなさい。
A26. $x^2 - 81$

Q27. $(6+a)(6-a)$ を展開しなさい。
A27. $36 - a^2$

Q28. $(2a-7b)(2a+7b)$ を展開しなさい。
A28. $4a^2 - 49b^2$

Q29. $(x+11)(x-11)$ を展開しなさい。
A29. $x^2 - 121$

## 係数付きの展開

Q30. $(4x-1)^2$ を展開しなさい。
A30. $16x^2 - 8x + 1$（$a=4x$, $b=1$ とおく）

Q31. $(3a+5)^2$ を展開しなさい。
A31. $9a^2 + 30a + 25$

Q32. $(6x+y)^2$ を展開しなさい。
A32. $36x^2 + 12xy + y^2$

Q33. $(a-3b)^2$ を展開しなさい。
A33. $a^2 - 6ab + 9b^2$

Q34. $(5x-2y)^2$ を展開しなさい。
A34. $25x^2 - 20xy + 4y^2$

Q35. $(2x+3)(2x-3)$ を展開しなさい。
A35. $4x^2 - 9$

## 分数を含む展開

Q36. $(a - \frac{1}{2}b)^2$ を展開しなさい。
A36. $a^2 - ab + \frac{1}{4}b^2$

Q37. $(x + \frac{3}{5})(x + \frac{2}{5})$ を展開しなさい。
A37. $x^2 + x + \frac{6}{25}$（足して $\frac{3}{5}+\frac{2}{5}=1$、かけて $\frac{6}{25}$）

Q38. $(x + \frac{1}{3})^2$ を展開しなさい。
A38. $x^2 + \frac{2}{3}x + \frac{1}{9}$

Q39. $(x - \frac{1}{2})(x + \frac{1}{2})$ を展開しなさい。
A39. $x^2 - \frac{1}{4}$

## 置き換えの考え方

Q40. $(3x+1)^2$ を展開するとき、何をおきかえると公式が使えるか。
A40. $M = 3x$ とおくと $(M+1)^2$ の形になる

Q41. $(3x+1)^2$ を展開しなさい。
A41. $9x^2 + 6x + 1$

Q42. $(2a-5b)^2$ を展開するとき、$a$, $b$ にあたるのは何か。
A42. $a = 2a$, $b = 5b$（公式の $a$, $b$ にそれぞれ代入）

## よくある間違い

Q43. $(a+b)^2 = a^2 + b^2$ は正しいか。
A43. 正しくない。正しくは $(a+b)^2 = a^2 + 2ab + b^2$（$2ab$ の項を忘れている）

Q44. $(x+3)^2 = x^2 + 9$ は正しいか。正しくなければ正しい答えを書きなさい。
A44. 正しくない。正しくは $x^2 + 6x + 9$（$2 \times x \times 3 = 6x$ が必要）

Q45. $(a-b)^2 = a^2 - b^2$ は正しいか。正しくなければ正しい答えを書きなさい。
A45. 正しくない。$a^2 - b^2 = (a+b)(a-b)$ であり、$(a-b)^2 = a^2 - 2ab + b^2$

Q46. $(x-4)^2 = x^2 - 8x - 16$ は正しいか。正しくなければ正しい答えを書きなさい。
A46. 正しくない。正しくは $x^2 - 8x + 16$（最後の項は $(-4)^2 = +16$）

Q47. $(2x+3)^2 = 4x^2 + 6x + 9$ は正しいか。正しくなければ正しい答えを書きなさい。
A47. 正しくない。正しくは $4x^2 + 12x + 9$（$2ab = 2 \times 2x \times 3 = 12x$）

## 公式の見分け方

Q48. $(x+5)(x+3)$ にはどの公式を使うか。
A48. 公式1: $(x+a)(x+b) = x^2 + (a+b)x + ab$

Q49. $(x+4)^2$ にはどの公式を使うか。
A49. 公式2: $(a+b)^2 = a^2 + 2ab + b^2$

Q50. $(x+6)(x-6)$ にはどの公式を使うか。
A50. 公式4: $(a+b)(a-b) = a^2 - b^2$（和と差の積）
