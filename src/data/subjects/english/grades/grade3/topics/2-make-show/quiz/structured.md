# make・showの応用 大問小問

## 大問1: 空欄補充 — make + O + C の基本（配点 15点）

次の文の（　）に入る適切な語を書きなさい。

(1) The news made me (　　).（そのニュースは私をうれしくさせました。）（2点）
→ happy。

(2) The movie made her (　　).（その映画は彼女を悲しくさせました。）（2点）
→ sad。

(3) His words (　　) me angry.（彼のことばは私を怒らせました。）（2点）
→ made。（過去の話なので make の過去形。）

(4) Anime songs (　　) me excited.（アニメの曲は私をワクワクさせます。）（3点）
→ make。（現在の話で主語は複数形。）

(5) The long lesson made (　　) tired.（その長い授業は私たちを疲れさせました。）（3点）
→ us。（目的格。we ではなく us。）

(6) Your performance made (　　) excited.（あなたの演技は彼らをワクワクさせました。）（3点）
→ them。（目的格。they ではなく them。）

## 大問2: 空欄補充 — show / tell + 人 + that の基本（配点 15点）

次の文の（　）に入る適切な語を書きなさい。

(1) He showed me (　　) it was true.（彼は私にそれが本当だと示しました。）（2点）
→ that。

(2) She (　　) me that she was busy.（彼女は私に忙しいと言いました。）（2点）
→ told。（tell の過去形。）

(3) The story (　　) us that anything is possible.（その物語は私たちに何でも可能だと示しています。）（3点）
→ shows。（主語が三人称単数で現在の話。）

(4) I told (　　) that I had a headache.（私は彼らに頭が痛いと言いました。）（3点）
→ them。（目的格。）

(5) This movie shows us (　　) good music is necessary.（この映画は私たちによい音楽は必要だと示しています。）（2点）
→ that。

(6) The teacher (　　) us that the test was next Monday.（先生は私たちにテストは来週の月曜日だと言いました。）（3点）
→ told。

## 大問3: 語順整序（配点 20点）

日本語の意味に合うように、語句を並べかえて英文を作りなさい。

(1) そのニュースは私をうれしくさせました。
（ happy / me / made / the / news ）（3点）
→ The news made me happy.

(2) その映画は彼女を悲しくさせました。
（ sad / made / the / movie / her ）（3点）
→ The movie made her sad.

(3) 彼は私にそれが本当だと示しました。
（ showed / that / me / true / he / it / was ）（4点）
→ He showed me that it was true.

(4) 彼女は私に英語が好きだと言いました。
（ me / liked / she / that / told / English / she ）（5点）
→ She told me that she liked English.

(5) この曲は私たちを元気にさせます。
（ us / makes / this / happy / song ）（5点）
→ This song makes us happy.

## 大問4: 選択問題（配点 15点）

次の文の（　）内から正しいものを選びなさい。

(1) The music made me ( happy / happily / happiness ).（3点）
→ happy。（make + O + C の C には形容詞が入る。）

(2) ( Her / She / Hers ) songs make me happy.（2点）
→ Her。（songs を修飾する所有格。）

(3) The data ( show / shows / showed ) us that the plan worked.（3点）
→ showed。（worked が過去形なので、主節も過去形。）

(4) He told me ( what / that / which ) he was a student.（3点）
→ that。（tell + 人 + that + 文。）

(5) Using a graph can ( name / call / make ) your slides more interesting.（4点）
→ make。（make + O + C で「〜をもっと面白くする」。）

## 大問5: 英作文（配点 15点）

次の日本語を英語に直しなさい。

(1) その知らせは彼をびっくりさせました。（3点）
→ The news made him surprised.

(2) 先生は私たちに英語は大切だと言いました。（3点）
→ The teacher told us that English was important.

(3) この本は私に夢は大切だということを示しています。（4点）
→ This book shows me that dreams are important.

(4) ジョギングは私を疲れさせます。（2点）
→ Jogging makes me tired.

(5) 彼女は私に自分は医者だと言いました。（3点）
→ She told me that she was a doctor.
