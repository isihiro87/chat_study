## 1. 過去分詞による後置修飾

### 扱う範囲（chat 全範囲を必ず網羅）

- **基本**: 名詞 + 過去分詞 + (語句) で「〜された名詞」を表す
  - This is a book written in English.
  - That is a car made in Japan.
  - Look at the broken window.（前置の場合は単独過去分詞）
- **by 〜 との組み合わせ**:
  - This is a book written by Soseki.
  - The picture taken by my father is beautiful.
  - The song sung by her is famous.
- **不規則動詞の過去分詞**:
  - write → written, take → taken, sing → sung, speak → spoken, make → made, build → built, give → given, send → sent など
- **受け身との関連**:
  - This is a book. + The book was written by Soseki. → This is a book written by Soseki.

### 例文数

15個（基本7・by 〜 付き6・短い修飾2 程度）— chat の全範囲を網羅

### 問題数

25題（4択13題＋並べ替え12題、basic / standard / advanced 混在）

### 問題作成上の注意

- 名詞 + 過去分詞 + (語句) の語順を確認
- 過去分詞の暗記を促す（grade2-6 受け身と同じ動詞を使い回せる）
- by 〜 との組み合わせを多めに含める
- 単独の過去分詞は名詞の前（broken window）、語句を伴う過去分詞は名詞の後（a window broken by the boy）の使い分けを扱う

### 引っかけ・ワンパターン回避

全問題のうち **15〜25%** を引っかけにする。

- **過去分詞 vs 過去形**（→ 8-past-tense 既習）
  - I broke the window. / This is the broken window. — 動詞の働きで判断
- **後置修飾 vs 受け身**（→ grade2-6 既習）
  - This book was written by Soseki. (受け身) / This is a book written by Soseki. (後置修飾) — be動詞の有無
- **規則動詞の同形性**（過去形と過去分詞が同じ）
  - This is a card made by my mother. (made は過去分詞) — 文脈で判断
- **修飾語句を伴う過去分詞は後ろから**
  - the broken window (単独・前置) / the window broken by him (語句あり・後置)
- 引っかけの explanation には「修飾語句の有無」「動詞の働き（述語動詞か後置修飾か）」を明記
