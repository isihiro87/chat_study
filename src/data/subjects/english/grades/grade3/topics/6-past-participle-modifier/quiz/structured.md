# 大問小問形式：過去分詞による後置修飾

## 大問1: 空欄補充（過去分詞の選択）（3点×6＝18点）

次の英文の（　）内から適切な語を選びなさい。

(1) This is a story ( write / writing / written ) by a famous author.
**答え: written（「書かれた話」なので過去分詞 written）**

(2) The bike ( park / parked / parking ) near the station is mine.
**答え: parked（「駐輪している自転車」→ 受け身の関係なので過去分詞 parked）**

(3) There is a festival ( call / calling / called ) Tanabata in Japan.
**答え: called（「〜と呼ばれている」は called）**

(4) The students ( invite / invited / inviting ) to the party looked happy.
**答え: invited（「招待された生徒たち」なので過去分詞 invited）**

(5) A letter ( send / sent / sending ) last week arrived today.
**答え: sent（「先週送られた手紙」なので過去分詞 sent）**

(6) She is wearing a dress ( make / made / making ) in Italy.
**答え: made（「イタリアで作られたドレス」なので過去分詞 made）**

## 大問2: 2文→1文の書きかえ（4点×6＝24点）

次の2文を、過去分詞の後置修飾を使って1文にしなさい。

(1) This is a novel. It was written by a British author.
**答え: This is a novel written by a British author.**
（「イギリスの著者によって書かれた小説」）

(2) There is a lake. It is known for its clear water.
**答え: There is a lake known for its clear water.**
（「透き通った水で知られている湖」）

(3) The dog is very friendly. The dog is called Max.
**答え: The dog called Max is very friendly.**
（「マックスと呼ばれている犬はとてもなつきやすい」）

(4) The players were excited. They were selected for the team.
**答え: The players selected for the team were excited.**
（「チームに選ばれた選手たちは興奮していた」）

(5) I found a wallet. It was left on the bench.
**答え: I found a wallet left on the bench.**
（「ベンチの上に置かれていた財布を見つけた」）

(6) This is a castle. It was built about 400 years ago.
**答え: This is a castle built about 400 years ago.**
（「約400年前に建てられた城」）

## 大問3: 語順整序（4点×6＝24点）

次の日本文にあう英文になるように、（ ）内の語を並べかえなさい。

(1) 彼女によって撮られた写真はとてもきれいです。
( her / by / taken / the / beautiful / photo / is / very ).
**答え: The photo taken by her is very beautiful.**

(2) 先生に送られたメールを読みましたか。
( the / by / you / sent / read / did / teacher / email )?
**答え: Did you read the email sent by the teacher?**

(3) あの山で見つかった動物は珍しいです。
( found / the / that / rare / animal / mountain / is / on ).
**答え: The animal found on that mountain is rare.**

(4) 100年前に建てられたその図書館は今もまだ使われています。
( the library / is / still / built / ago / 100 years / used ).
**答え: The library built 100 years ago is still used.**

(5) 私は子どもたちに愛されているキャラクターを知っています。
( a character / children / know / loved / by / I ).
**答え: I know a character loved by children.**

(6) 東京で売られている野菜のいくつかは九州から来ています。
( sold / some of / Tokyo / the vegetables / come from Kyushu / in ).
**答え: Some of the vegetables sold in Tokyo come from Kyushu.**

## 大問4: 英作文（5点×4＝20点）

次の日本文を英語にしなさい。

(1) 「これは多くの国で話されている言語です。」
**答え: This is a language spoken in many countries.**
※ spoken（speak の過去分詞）を language のうしろに置く。

(2) 「彼女はユキと呼ばれている女の子を探しています。」
**答え: She is looking for a girl called Yuki.**
※ called を使って「〜と呼ばれている女の子」を表す。

(3) 「壁にかかっているその絵はとても古い。」（hang の過去分詞: hung）
**答え: The picture hung on the wall is very old.**
※ hung on the wall で「壁にかかっている」を表す。

(4) 「外国で作られた製品が日本でも人気です。」
**答え: Products made in foreign countries are popular in Japan too.**
※ made in foreign countries で「外国で作られた」を表す。

## 大問5: 現在分詞 vs 過去分詞の判断問題（3点×5＝15点）

次の（ ）内の動詞を、現在分詞（-ing）か過去分詞のどちらかに変えて書きなさい。

(1) The girl ( play ) the violin on the stage is my sister.
**答え: playing（「ヴァイオリンを弾いている少女」→ 能動の関係なので現在分詞）**

(2) The cup ( break ) by the baby is on the floor.
**答え: broken（「赤ちゃんに割られたカップ」→ 受け身の関係なので過去分詞）**

(3) Look at those clouds ( float ) in the sky.
**答え: floating（「空に浮かんでいる雲」→ 能動の関係なので現在分詞）**

(4) The shoes ( make ) in Spain are expensive.
**答え: made（「スペインで作られた靴」→ 受け身の関係なので過去分詞）**

(5) The child ( sit ) on the bench is my cousin.
**答え: sitting（「ベンチに座っている子ども」→ 能動の関係なので現在分詞）**
