# 大問小問形式：現在完了進行形（have been + ~ing）

## 大問1: 空欄補充（基本）（3点×6＝18点）

次の日本文にあう英文になるように、___に適する語を書きなさい。

(1) 私は1時間ずっと歩いています。
I ___ ___ ___ for an hour.
**答え: have been walking**

(2) 彼女は今朝からずっと料理をしています。
She ___ ___ ___ since this morning.
**答え: has been cooking**

(3) 彼らは2時間ずっとサッカーをしています。
They ___ ___ ___ soccer for two hours.
**答え: have been playing**

(4) 彼は午前10時からずっと泳いでいます。
He ___ ___ ___ since 10 a.m.
**答え: has been swimming**

(5) 昨夜からずっと雪が降り続いています。
It ___ ___ ___ since last night.
**答え: has been snowing**

(6) 私はずっとあなたを探しています。
I ___ ___ ___ for you.
**答え: have been looking**

## 大問2: for / since の選択（3点×5＝15点）

次の文の（ ）に for か since のどちらかを入れなさい。

(1) I have been studying (   ) two hours.
**答え: for（two hours は期間）**

(2) She has been dancing (   ) this morning.
**答え: since（this morning は起点）**

(3) He has been playing the guitar (   ) last Sunday.
**答え: since（last Sunday は起点）**

(4) We have been waiting (   ) thirty minutes.
**答え: for（thirty minutes は期間）**

(5) It has been raining (   ) yesterday afternoon.
**答え: since（yesterday afternoon は起点）**

## 大問3: 現在完了 vs 現在完了進行形（4点×5＝20点）

次の（ ）内の語を適切な形にしなさい。

(1) Toru has been ___ for an hour. (swim)
**答え: swimming（swim は動作 → 現在完了進行形。m を重ねて swimming）**

(2) How long have you been ___ this computer? (use)
**答え: using（use は動作 → 現在完了進行形。e を取って using）**

(3) I have ___ him since I was a child. (know)
**答え: known（know は状態 → 現在完了。過去分詞 known）**

(4) Matcha sweets have been ___ popular around the world. (get)
**答え: getting（get は動作 → 現在完了進行形。t を重ねて getting）**

(5) She has ___ in Nagoya for ten years. (live)
**答え: lived（live は状態 → 現在完了。過去分詞 lived）**

## 大問4: 語順整序（4点×5＝20点）

次の日本文にあう英文になるように、（ ）内の語を並べかえなさい。

(1) 私はずっと英語を勉強しています。
( studying / been / have / I ) English.
**答え: I have been studying English.**

(2) 彼女は2時間ずっとピアノを弾いています。
( playing / has / been / she ) the piano for two hours.
**答え: She has been playing the piano for two hours.**

(3) あなたはどのくらい待っていますか。
( waiting / long / you / have / how / been )?
**答え: How long have you been waiting?**

(4) 彼は午前中ずっとその本を読んでいます。
( the book / been / he / reading / has ) since this morning.
**答え: He has been reading the book since this morning.**

(5) 彼らは30分間ずっとバスを待っています。
( been / they / for / waiting / have ) the bus for thirty minutes.
**答え: They have been waiting for the bus for thirty minutes.**

## 大問5: 英作文（5点×5＝25点）

次の日本文を英語にしなさい。

(1) 私は今朝からずっとかばんを探しています。
**答え: I have been looking for my bag since this morning.**

(2) 彼女は20年間ずっとバイオリンを弾いています。
**答え: She has been playing the violin for twenty years.**

(3) あなたはどのくらいここで待っていますか。
**答え: How long have you been waiting here?**

(4) 私は10年間この町に住んでいます。（liveは状態動詞）
**答え: I have lived in this town for ten years.**

(5) 彼はけさからずっと部屋でゲームをし続けています。
**答え: He has been playing the game in his room since this morning.**
