## 既習内容（このトピックで使ってよい知識）

このトピックは grade1 の8番目です。以下が前トピックで習い済み。例文・問題で使う語彙・文法はこの範囲内に限定してください。

### 既習文法

- **be動詞**（1-be-verbs）: am / is / are、否定・疑問・短縮形、Yes/No 答え方
- **一般動詞**（2-general-verbs）: I/you/we/they の肯定・否定 (don't)・疑問 (Do you ~?)
- **名詞**（3-nouns / 3b-nouns-advanced）: a/an、規則複数 -s、不規則複数 (children, men, women, knives)、不可算 (water, milk, money)、some / any、How many、this/that、these/those、There is/are
- **疑問詞**（4-question-words / 4b-question-patterns）: What / Who / Where / When / Why / How + be動詞 / 一般動詞 / does
- **三人称単数**（5-third-person-singular）: he/she/it のとき動詞に s/es/ies、has、Does ~?、doesn't、頻度の副詞 (often/usually/sometimes)、人名・家族主語 (Ken, my sister)
- **既習の所有格**: my, your, his, her（名詞の前に置く形のみ。本トピックで体系化）

### 既習語彙の例

- 動詞: like, love, want, have / has, play, watch / watches, read, draw, eat, drink, cook, make, study / studies, speak, join, go / goes, get up, wash / washes, live, know, take, use, see, put, teach / teaches
- 名詞: book, pen, dog, cat, bag, teacher, student, notebook, school, library, park, classroom, kitchen, station, store, picture, song, brother, sister, mother, father, friend, breakfast, lunch, music, math, English, Japanese, soccer, tennis, baseball, piano, guitar, water, milk, money, ball, knife, child, woman, etc.
- 副詞: every day, on weekends, often, usually, sometimes, in the morning, after school

### 使ってはいけない（後のトピックで扱う）内容

- 現在進行形 (be + ing)（→ 7-present-progressive）
- 過去形 (-ed / went / had / was / were)（→ 8-past-tense）
- 命令文 (Open the window. / Don't run.)（→ 9-imperatives）
- 再帰代名詞 myself / yourself / himself / herself
- 関係代名詞 who / which / that（疑問詞の which は本トピックで扱うが、関係代名詞は除外）
- 不定代名詞 someone / anything / nobody
- 指示代名詞の応用的用法 (e.g. those who)
- can / must / will / should などの助動詞
- 受け身 (be + 過去分詞)
- 現在完了 (have + 過去分詞)
- to不定詞、動名詞

## 1. 代名詞（主格・所有格・目的格・所有代名詞）

### 扱う範囲

- 主格: I / you / he / she / it / we / they
- 所有格: my / your / his / her / its / our / their
- 目的格: me / you / him / her / it / us / them
- 所有代名詞: mine / yours / his / hers / ours / theirs
- 文中での代名詞の使い分け（主語・動詞の目的語・所有関係）
- 名詞のくり返しを避けるための代名詞（This is Ken. He is my friend. のような流れ）

### 除外する内容

- 再帰代名詞 myself / yourself など
- 関係代名詞 who / which / that
- 指示代名詞の応用的用法
- 不定代名詞 someone / anything など
- 難しい所有代名詞の慣用表現

### 例文数

20個（主格3・所有格5・目的格5・所有代名詞4・混在3 程度のバランス）

### 問題数

30題（4択15題＋並べ替え15題、basic / standard / advanced の3段階を混在）

### 問題作成上の注意

- 主格・所有格・目的格・所有代名詞をバランスよく扱う（偏らせない）
- 「彼を」「彼の」「彼のもの」など、日本語で混同しやすい表現を入れる
- **her が「彼女を」と「彼女の」の両方**で使われることに注意した問題を入れる
- **his が「彼の」と「彼のもの」の両方**で使われることに注意した問題を入れる
- 文の中でどの位置に入るか（主語・目的語・所有）を考えさせる問題を入れる
- 4択では、he / his / him、she / her / hers、they / their / them / theirs などの混同を選択肢に含める
- カッコは1単語のみ。複数語が必要なら短縮形か空欄を2つに
- 別解がある場合は explanation に明記

### 引っかけ・ワンパターン回避

学習中の代名詞ルールが「常に当てはまる」設定にせず、対比される既習文法を一部混ぜて判断力を問う。全問題のうち **15〜25%（4〜7問）** を以下の引っかけにし、standard / advanced に配置する。

- **三人称単数の動詞変化との混在**（→ 5-third-person-singular 既習）
  - He likes her. / She has his book. — 主語の三人称単数 s と代名詞の選択を同時に判断
  - His sister plays tennis. — 「his sister」が he と等価で動詞に s が必要
- **be動詞文との混在**（→ 1-be-verbs 既習）
  - We are friends. / They are students. — 主格 + be動詞の組み合わせ判断
- **a/an・複数名詞との混在**（→ 3-nouns 既習）
  - This is her bag. / These are our books. — 単複に応じた所有格の使い分け
- **疑問詞 What/Who との混在**（→ 4-question-words 既習）
  - Who is he? — He is Ken. — 疑問文と代名詞の組み合わせ
- 引っかけ問題の explanation には「日本語のどの助詞か（〜は/〜の/〜を）」「文中のどの位置か」など**判断基準を必ず書く**

## 2. whose を使う疑問文

### chat.ts での扱い

chat では以下のみを扱っている：
- `Whose + 名詞 + is/are + 主語?` の形（例: Whose bag is this?）
- 答え方: 所有代名詞（It's mine. / It's hers.）
- 所有代名詞リスト: mine / yours / his / hers / ours / theirs

### 扱う範囲

- Whose + 名詞 + is this? / Whose + 名詞 + are these? の疑問文（chat 既出）
- 所有者をたずねる疑問文
- 答え方: It is mine. / They are hers. など、所有代名詞での答え方（chat 既出）
- this / that / these / those と is / are の対応

### 扱う範囲（chat 未収録の補足。導入時は短い解説を例文側に添えること）

- Whose is this? / Whose are these?（noun を省略する形）
- 答え方で `Tom's / my brother's` のような **名詞 + 's** で所有を表す形

### 除外する内容

- 関係代名詞 whose
- 間接疑問文
- 過去形
- 現在完了
- 複雑な所有表現
- 長すぎる会話文

### 例文数

15個（基本5・標準5・発展5 程度）。うち chat 未収録の補足（noun 省略 / 's 所有）は 3〜4個まで。

### 問題数

30題（4択15題＋並べ替え15題、basic / standard / advanced の3段階を混在）

### 問題作成上の注意

- whose は「だれの」という意味であることを確認できる問題を入れる
- whose の後ろに名詞が続く形（Whose pen is this?）を中心に出題する
- noun 省略の形（Whose is this?）も入れるが、chat 未収録のため例文の解説で補う
- this / that / these / those と is / are の対応を意識させる（these → are、that → is）
- 答え方では mine / yours / his / hers / ours / theirs を中心に扱う
- 's 所有（Tom's / my brother's）も少量含めるが、chat 未収録のため解説で補う
- who と whose の違いを問う問題を入れてもよい
- 4択は長さ・情報量を揃え、「のみ」等の限定表現を使わない

### 引っかけ・ワンパターン回避

全問題のうち **15〜25%（4〜7問）** を以下の引っかけにする。

- **who と whose の混同**（→ 4-question-words 既習）
  - Who is he? / Whose pen is this? — 主語をたずねる who と、所有者をたずねる whose の使い分け
- **be動詞 is/are の選択**（→ 1-be-verbs / 3-nouns 既習）
  - Whose pen is this? / Whose pens are these? — 単複に応じた is/are の判断
- **所有格 vs 所有代名詞の答え方**（→ 本トピック1 既習）
  - It is my pen. / It is mine. — どちらの形でも答えられることを示しつつ、混乱しやすい使い分けを問う
- 引っかけ問題の explanation では「単数か複数か」「答えに名詞が続くか続かないか」を明記

## 3. which を使う疑問文

### chat.ts での扱い

chat では以下のみを扱っている：
- `Which do you like, A or B?`（2つから選ぶ）の形
- `Which + 名詞 + do you ~?`（例: Which sport do you play?）の形
- 答え方: 直接の名詞や代名詞（I like cats. など）

### 扱う範囲

- Which do you like, A or B? の形（chat 既出）
- Which + 名詞 + do you ~? の形（chat 既出）
- Which + 名詞 + does + 主語 + 動詞? の三人称単数版（5-third-person-singular 既習を組み合わせ）

### 扱う範囲（chat 未収録の補足。導入時は短い解説を例文側に添えること）

- Which is your bag? / Which are your shoes?（be動詞文との組み合わせ）
- 答え方で `this one / that one / the blue one` などの one を使った答え方

### 除外する内容

- 関係代名詞 which
- 間接疑問文
- 過去形
- 現在完了
- 比較級・最上級を使う複雑な文
- 長すぎる会話文

### 例文数

15個（基本5・標準5・発展5 程度）。うち chat 未収録の補足（Which is/are / one を使った答え）は 3〜4個まで。

### 問題数

30題（4択15題＋並べ替え15題、basic / standard / advanced の3段階を混在）

### 問題作成上の注意

- which は「どちら」「どれ」という意味であることを確認できる問題を入れる
- which の後ろに名詞が続く形（Which color do you like?）を多めに扱う
- which と what の違いを意識できる問題を入れてもよい（限られた選択肢から → which、不特定 → what）
- **Which do you ~?** と **Which is ~?** の語順の違いを扱う（後者は chat 未収録の補足扱い）
- 三人称単数と組み合わせる場合は does を使い、動詞は原形にする（Which does she like?）
- 答え方では直接の答え（I like cats.）を中心に、one を使った答え方（this one / the blue one）も補足扱いで含める
- 4択は長さ・情報量を揃える

### 引っかけ・ワンパターン回避

全問題のうち **15〜25%（4〜7問）** を以下の引っかけにする。

- **what と which の混同**（→ 4-question-words 既習）
  - What do you like? / Which do you like? — 不特定 vs 限られた選択肢の判断
- **三人称単数 do / does の判断**（→ 5-third-person-singular 既習）
  - Which do you like? / Which does she like? — 主語に応じた do/does の選択と、動詞を原形にすること
- **be動詞 is/are の選択**（→ 1-be-verbs / 3-nouns 既習）
  - Which is your bag? / Which are your shoes? — 単複に応じた is/are の判断
- **所有代名詞での答え方**（→ 本トピック1 既習）
  - The blue one is hers. / These are mine. — one や所有代名詞を組み合わせた答え方の自然さ
- 引っかけ問題の explanation では「選択肢が限られているか不特定か」「主語の人称・数」「単複の判断」を明記

## 出力形式（全セクション共通）

- 例文: 番号付き、「日本語」改行「英文」の順、レベル別 (基本/標準/発展) にセクション分け
- 問題: 4択と並べ替えをセクション分け、各問題に番号と難易度を明記、解説と別解の有無も
- 末尾に難易度配分のサマリー (basic / standard / advanced の例文数・問題数)
- カッコは1単語のみ。複数語が必要な場合は短縮形を使うか、空欄を2つ並べる
