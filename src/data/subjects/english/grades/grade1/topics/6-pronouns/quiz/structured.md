# 代名詞 大問小問

## 大問1: 代名詞の格変化 — 表を完成させなさい（配点 21点）

次の代名詞の表を完成させなさい。

(1) I — my — (　　) — mine（3点）
→ me。（I の目的格は me）

(2) you — your — you — (　　)（3点）
→ yours。（you の所有代名詞は yours）

(3) he — (　　) — him — his（3点）
→ his。（he の所有格は his）

(4) she — her — (　　) — hers（3点）
→ her。（she の目的格は her）

(5) we — our — (　　) — ours（3点）
→ us。（we の目的格は us）

(6) they — (　　) — them — theirs（3点）
→ their。（they の所有格は their）

(7) it — its — it — (　　)（3点）
→ なし（×）。（it には所有代名詞がない）

## 大問2: 適語補充（目的格・所有代名詞） — （　）に適する語を書きなさい（配点 21点）

(1) I have a sister. I like (　　) very much.（3点）
→ her。（「彼女を」は目的格の her）

(2) Tom is my friend. Do you know (　　)?（3点）
→ him。（「彼を」は目的格の him）

(3) We are in this class. Please teach (　　) English.（3点）
→ us。（「私たちに」は目的格の us）

(4) This is my pen. It's (　　).（3点）
→ mine。（「私のもの」は所有代名詞の mine）

(5) A: Is this your book? B: No. It isn't (　　).（3点）
→ mine。（「私のもの」は mine）

(6) Those cats are cute. I like (　　).（3点）
→ them。（「それらを」は目的格の them）

(7) This bag is not my bag. It's (　　) bag.（3点）
→ her / his / your など。（文脈に合う所有格を入れる。例: her）

## 大問3: Whose / Which の疑問文 — 次の対話文の（　）に適する語を書きなさい（配点 20点）

(1) A: (　　) pencil is this?
    B: It's mine.（3点）
→ Whose。（「だれの」をたずねる疑問詞）

(2) A: Whose umbrella is that?
    B: It's (　　).（ケンのものです。）（3点）
→ Ken's。（所有を表す 's をつける）

(3) A: (　　) do you like, math or English?
    B: I like English.（3点）
→ Which。（2つのうちどちらかを選ぶ疑問詞）

(4) A: (　　) sport do you play, soccer or tennis?
    B: I play soccer.（4点）
→ Which。（Which + 名詞 で「どちらのスポーツ」）

(5) A: Whose notebook is this?
    B: It's not (　　). Ask Yuki.（私のものではありません。）（3点）
→ mine。（所有代名詞で「私のもの」）

(6) A: Whose guitar is that?
    B: It's (　　). She plays it every day.（彼女のものです。）（4点）
→ hers。（所有代名詞で「彼女のもの」）

## 大問4: 書き換え・語順整序 — 次の問いに答えなさい（配点 18点）

(1) This is my bag. → 所有代名詞を使って同じ意味の文に書きかえなさい。（3点）
→ This bag is mine.（my bag → mine）

(2) That is her piano. → 所有代名詞を使って同じ意味の文に書きかえなさい。（3点）
→ That piano is hers.（her piano → hers）

(3) 次の語句を正しく並べかえて英文を作りなさい。（4点）
    「私たちは彼らを知っています。」
    （ them / know / we ）
→ We know them.

(4) 次の語句を正しく並べかえて英文を作りなさい。（4点）
    「これはだれの帽子ですか。」
    （ is / whose / cap / this ）?
→ Whose cap is this?

(5) 次の語句を正しく並べかえて英文を作りなさい。（4点）
    「あなたは犬と猫、どちらが好きですか。」
    （ you / like / do / which ）, dogs or cats?
→ Which do you like, dogs or cats?

## 大問5: 英作文 — 次の日本文を英語にしなさい（配点 20点）

(1) 私は彼女に毎日会います。（4点）
→ I see her every day.（「彼女に」は目的格の her）

(2) このペンはだれのものですか。 — 彼のものです。（4点）
→ Whose pen is this? — It's his.

(3) あなたは赤と青、どちらの色が好きですか。（4点）
→ Which color do you like, red or blue?

(4) あの自転車は私たちのものです。（4点）
→ That bike is ours.（所有代名詞 ours を使う）

(5) 彼らを手伝ってあげてください。（4点）
→ Please help them.（「彼らを」は目的格の them）
