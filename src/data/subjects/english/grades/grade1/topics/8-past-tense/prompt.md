## 1. be動詞の過去形（was / were）

### 扱う範囲

- I / he / she / it + was
- you / we / they + 複数主語 + were
- 否定: was not / wasn't / were not / weren't
- 疑問: Was ~? / Were ~? と Yes/No 答え方
- 過去を表す副詞: yesterday, last night, last week, two days ago, then

### 例文数 / 問題数

- 例文: 10個 / 問題: 15題（4択8 + 並べ替え7、basic / standard / advanced 混在）

### 問題作成上の注意

- I/he/she/it → was、you/we/they/複数 → were の選択を問う
- 過去を示す副詞（yesterday, last ~, ~ ago）の位置（文末が基本）を確認
- 別解（was not = wasn't、were not = weren't）は explanation に明記

---

## 2. 一般動詞の過去形（規則動詞 -ed / 不規則動詞）

### 扱う範囲

- 規則動詞: -ed / -d / 子音重ね -ed（played, liked, studied, stopped）
- y → ied（study → studied, cry → cried）
- 不規則動詞: go → went, have → had, see → saw, eat → ate, take → took, come → came, run → ran, write → wrote, read → read, get → got, make → made, do → did, say → said など
- 主語に関係なく動詞は一律で過去形（三単現の s は過去形には影響しない）

### 例文数 / 問題数

- 例文: 12個 / 問題: 20題（4択10 + 並べ替え10、basic / standard / advanced 混在）

### 問題作成上の注意

- 規則動詞と不規則動詞をバランスよく扱う（規則:不規則 ≒ 6:6）
- y → ied、子音重ね -ed のルールを各々問う
- 三単現の s と過去形の混同（× He played**s**.）を防ぐ問題を入れる

---

## 3. 否定文・疑問文（didn't / Did）

### 扱う範囲

- 否定: did not / didn't + 動詞のもとの形
- 疑問: Did + 主語 + 動詞のもとの形 ~?
- 答え方: Yes, I did. / No, I didn't.
- 主語が he/she/it でも did/didn't を使う（doesn't ではない）

### 例文数 / 問題数

- 例文: 8個 / 問題: 15題（4択8 + 並べ替え7）

### 問題作成上の注意

- didn't / Did の後ろは動詞のもとの形（× didn't played）
- 主語が三人称単数でも did/didn't は変わらない（× doesn't / × didnt's）
- 別解（did not = didn't）を explanation に明記

---

## 4. 過去進行形（was/were + ing）

### 扱う範囲

- 肯定: I was playing tennis. / They were watching TV.
- 否定: was not / were not + ing
- 疑問: Was/Were + 主語 + ing ~? と Yes/No 答え
- 「そのとき」を表す副詞: then, at that time, at 7 last night

### 例文数 / 問題数

- 例文: 8個 / 問題: 10題（4択5 + 並べ替え5）

### 問題作成上の注意

- be動詞は was / were で主語に応じて使い分け
- ing 形のルール（7-present-progressive 既習）はそのまま流用できる
- 「今〜している」(現在進行形) と「そのとき〜していた」(過去進行形) の対比を問う

## 引っかけ・ワンパターン回避（全節共通）

全問題のうち **15〜25%** を引っかけにし、standard / advanced に配置する。

- **現在形 vs 過去形**（→ 5-third-person-singular / 2-general-verbs 既習）
  - I play tennis every day. / I played tennis yesterday. — 時制副詞で判断
- **現在進行形 vs 過去進行形**（→ 7-present-progressive 既習）
  - She is studying now. / She was studying then. — be動詞の現在/過去で判断
- **三単現の s と過去形の混同**（→ 5-third-person-singular 既習）
  - He plays / He played — 三単現の s は過去形には残らない
- **be動詞文と一般動詞文の混在**（→ 1-be-verbs 既習）
  - He was kind. / He played soccer. — be動詞の過去 vs 一般動詞の過去
- 引っかけ問題の explanation には「時制を表す副詞」「主語の人称・数」「be動詞か一般動詞か」など判断基準を明記

## カッコ・別解ルール

- カッコは1単語のみ。複数語が必要なら短縮形（wasn't / weren't / didn't）を使うか、空欄を2つ
- 別解（was not = wasn't 等）は explanation に明記
