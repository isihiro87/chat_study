# 名詞 大問小問

## 大問1: 複数形の作り方 — 次の名詞を複数形にしなさい（配点 18点）

(1) book（2点）
→ books。（ふつうは語尾に s をつける）

(2) box（2点）
→ boxes。（x で終わる語には es をつける）

(3) watch（2点）
→ watches。（ch で終わる語には es をつける）

(4) city（3点）
→ cities。（子音字 + y で終わる語は y を i にかえて es をつける）

(5) child（3点）
→ children。（不規則変化）

(6) sheep（3点）
→ sheep。（単数形と複数形が同じ）

(7) dish（3点）
→ dishes。（sh で終わる語には es をつける）

## 大問2: a / an の使い分け — （　）に a か an を書きなさい（配点 15点）

(1) I have （　） cat.（2点）
→ a。（c の音ではじまるので a）

(2) This is （　） apple.（2点）
→ an。（a の音ではじまるので an）

(3) That is （　） umbrella.（3点）
→ an。（u の音ではじまるので an）

(4) She has （　） egg.（3点）
→ an。（e の音ではじまるので an）

(5) He is （　） English teacher.（2点）
→ an。（E の音ではじまるので an）

(6) I want （　） new bike.（3点）
→ a。（n の音ではじまるので a。new がつくので a new bike）

## 大問3: this / that の疑問文と応答 — 次の対話文の（　）に適する語を書きなさい（配点 20点）

(1) A: Is this a pen?
    B: Yes, （　　　）.（3点）
→ it is。（Yes, it is.）

(2) A: Is that your notebook?
    B: No, （　　　）. It's Yuki's notebook.（3点）
→ it isn't。（No, it isn't.）

(3) 「This is a dog.」を疑問文にしなさい。（3点）
→ Is this a dog?

(4) 「That is your guitar.」を否定文にしなさい。（3点）
→ That is not your guitar.（または That isn't your guitar.）

(5) A: （　　　）?
    B: No, it isn't. It's a hospital.（4点）
→ Is that a school?（答えが No で hospital だから、別のものをたずねる疑問文。例: Is that a school?）

(6) A: What is this?
    B: （　　　）.（4点）
→ It is a map.（または It's a map. など、具体的にものを答える文）

## 大問4: How many を使った疑問文 — 次の日本文にあう英文になるように（　）に適する語を書きなさい（配点 22点）

(1) あなたはマンガを何冊もっていますか。（4点）
    （　　）（　　）comics （　　） you have?
→ How many comics do you have?

(2) (1)に対して「10冊もっています。」（3点）
    I （　　）（　　） comics.
→ I have ten comics.

(3) あなたはいくつの星が見えますか。（4点）
    How （　　）（　　） can you （　　）?
→ How many stars can you see?

(4) (3)に対して「星が1つ見えます。」（3点）
    I can （　　）（　　）（　　）.
→ I can see one star.

(5) あなたはかばんをいくつもっていますか。（4点）
    （　　）（　　）（　　） do you have?
→ How many bags do you have?

(6) (5)に対して「3つもっています。」（4点）
    I have （　　）（　　）.
→ I have three bags.

## 大問5: 語順並べ替え — 日本語の意味に合うように、語句を並べかえて英文を作りなさい（配点 25点）

(1) これは私のペンです。（4点）
    （ my / this / pen / is ）
→ This is my pen.

(2) あれはあなたの自転車ですか。（4点）
    （ is / that / bike / your ）?
→ Is that your bike?

(3) 私はリンゴを5つもっています。（5点）
    （ five / have / I / apples ）
→ I have five apples.

(4) あなたはネコを何匹飼っていますか。（6点）
    （ many / how / you / do / cats / have ）?
→ How many cats do you have?

(5) これらは私の本です。（6点）
    （ are / my / these / books ）
→ These are my books.
