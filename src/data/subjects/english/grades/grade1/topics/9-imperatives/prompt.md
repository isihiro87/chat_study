## 1. 命令文（基本・Please・否定・Let's）

### 扱う範囲（chat 全範囲を必ず網羅）

- **基本の命令文**: 主語なし + 動詞のもとの形（Open the window. / Close the door. / Come here. / Wash your hands. / Stand up.）
- **Please をつけて丁寧に**: Please + 命令文（Please read this book. / Please listen to me. / Please show me your notebook.）
- **否定の命令文**: Do not / Don't + 動詞のもとの形（Don't run here. / Please don't use that box.）
- **Let's**: Let's + 動詞のもとの形（Let's speak English. / Let's go to the library. / Let's sing this song together.）
- 短縮形: Don't = Do not、Let's = Let us

### 例文数

15個（基本5・標準5・発展5）— chat の4パターンをすべて含める

### 問題数

30題（4択15題＋並べ替え15題、basic / standard / advanced を混在、4パターンをバランスよく出題）

### 問題作成上の注意

- 4パターン（基本／Please／Don't／Let's）をバランスよく出題（各6〜8問程度）
- **動詞のもとの形を使う**こと（× Opens / × To open / × Opening）を確認できる問題を入れる
- Please の位置は文頭が基本（chat に従う）
- 別解（Do not = Don't、Let us = Let's）は explanation に明記
- カッコは1単語のみ。複数語が必要なら短縮形を使うか、空欄を2つ並べる

### 引っかけ・ワンパターン回避

全問題のうち **15〜25%（4〜7問）** を以下の引っかけにし、standard / advanced に配置する。

- **三人称単数 s との混同**（→ 5-third-person-singular 既習）
  - He plays soccer. / Play soccer. — 主語ありの三単現と命令文（主語なし＋原形）の対比
- **be動詞文との混同**（→ 1-be-verbs 既習）
  - 命令文では動詞の原形を使うが、be動詞主体の文 (You are kind.) は命令文ではない
- **Don't と doesn't / don't の使い分け**（→ 2-general-verbs / 5-third-person-singular 既習）
  - Don't run. (命令) / He doesn't run. (三単現否定) / They don't run. (一般否定) — 主語の有無で判断
- **Let's と let / 一般動詞の混同**
  - Let's go. (勧誘) / I go. (普通の文) — 主語の有無
- 引っかけの explanation には「主語があるか」「動詞の形」「文の意図（命令か事実か）」を明記
