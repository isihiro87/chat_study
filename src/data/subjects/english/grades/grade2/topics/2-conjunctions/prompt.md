## 1. 接続詞 when / if / because / that

### 扱う範囲（chat 全範囲を必ず網羅）

- **when**: 「〜のとき」（When I was a child, I lived in Osaka. / I was reading when you called me.）
- **if**: 「もし〜なら」（If it rains tomorrow, I will stay home. / If you are tired, take a break.）
  - 注: if 節内は未来でも現在形（× If it will rain → ◯ If it rains）
- **because**: 「〜だから」（I like him because he is kind. / I stayed home because it was raining.）
- **that**: 「〜ということ」（I think that he is right. / I know that you are busy.）
  - that の省略可（I think he is right.）
- 接続詞節の前後の位置（文頭 / 文中）と、文頭のときのカンマの有無

### 例文数

20個（when 5・if 5・because 5・that 5）— chat の4語をすべて網羅

### 問題数

30題（4択15題＋並べ替え15題、basic / standard / advanced 混在）

### 問題作成上の注意

- 4つの接続詞をバランスよく出題（各6〜8問）
- 「いつ → when、もし → if、なぜ → because、〜ということ → that」の意味判別を問う
- if 節内が現在形（will にしない）ことを確認する問題を入れる
- that の省略可・不可を扱う
- 文頭の接続詞節 + カンマ vs 文中の接続詞節（カンマなし）を扱う

### 引っかけ・ワンパターン回避

全問題のうち **15〜25%** を引っかけにする。

- **接続詞同士の混同**
  - when / if / because / that のうち、文意に合うものを選ぶ
- **if 節内の時制**（→ 8-past-tense 既習）
  - If it rains / If it will rain — if 節内は未来でも現在形
- **that と疑問詞節**（→ 4b-question-patterns 既習）
  - I think that he is right. / I know what he likes. — 後者は疑問詞、本トピックでは扱わない
- 引っかけの explanation には「文意の関係（時 / 条件 / 理由 / 内容）」「if 節の時制ルール」を明記
