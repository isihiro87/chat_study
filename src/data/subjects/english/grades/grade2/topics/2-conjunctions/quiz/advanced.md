# 発展問題：接続詞（when / if / because / that）

## 大問1: 対話文読解（4点×5＝20点）

次の対話文を読んで、あとの問いに答えなさい。

Jun: There is a good Sapporo ramen restaurant near my house.
    It's famous for delicious miso ramen.
Anna: Really? I want to try miso ramen sometime.
Jun: Riko, Eddy, and I will go to the restaurant this Sunday.
    ( ① ) you have time, you can come with us.
Anna: I'd love to! I think ( ② ) Japanese ramen is amazing.
Jun: ( ③ ) I go there, I always order the special miso ramen
    ( ④ ) the soup is very rich and tasty.

(1) ①に入る最も適切な接続詞を選びなさい。
ア When  イ If  ウ Because  エ That
**答え: イ If（「もし時間があるなら」の条件）**

(2) ②に入る最も適切な接続詞を選びなさい。
ア when  イ if  ウ because  エ that
**答え: エ that（thinkの後ろで「〜ということ」）**

(3) ③に入る最も適切な接続詞を選びなさい。
ア When  イ If  ウ Because  エ That
**答え: ア When（「行くときはいつも」）**

(4) ④に入る最も適切な接続詞を選びなさい。
ア when  イ if  ウ because  エ that
**答え: ウ because（「スープが濃いので」の理由）**

(5) 本文の内容に合うものを選びなさい。
ア Anna often goes to the ramen restaurant.
イ Jun always orders the special miso ramen at the restaurant.
ウ Jun and Anna will go to the restaurant on Saturday.
エ Anna doesn't like Japanese food.
**答え: イ（Junはいつも特製味噌ラーメンを注文すると言っている）**

## 大問2: 誤文訂正（4点×5＝20点）

次の英文にはそれぞれ1か所まちがいがあります。まちがいを指摘し、正しく書きかえなさい。

(1) If it will be sunny tomorrow, we will go to the beach.
**答え: will be → is（if節では未来のことでも現在形を使う）**

(2) When I am child, I lived in Hokkaido.
**答え: am → was（過去の話なので過去形）**

(3) I like this town. Because it is beautiful.
**答え: ピリオドをカンマに → I like this town because it is beautiful.（Becauseで始まる文は不完全。前の文とつなげる）**

(4) I think that she don't like natto.
**答え: I think that → I don't think that（英語ではthinkを否定する方が自然）。または don't → doesn't（she は三人称単数）。**

(5) Because I was tired, so I went to bed early.
**答え: so を削除 → Because I was tired, I went to bed early.（becauseとsoは同時に使えない）**

## 大問3: 長文読解（4点×5＝20点）

次の英文を読んで、あとの問いに答えなさい。

Sapporo ramen is famous in Hokkaido. Many people love it because the soup is rich and warm. When it is cold in winter, a hot bowl of ramen makes you feel better.

If you visit Hokkaido, you should try different kinds of ramen. I think that Sapporo ramen, Asahikawa ramen, and Hakata ramen are all delicious. Each ramen has a special flavor because the chefs use different ingredients.

When I traveled to Sapporo last year, I ate miso ramen at a famous shop. I was surprised because there were so many people waiting in line. I think that good food brings people together.

(1) 多くの人が札幌ラーメンを好む理由を日本語で書きなさい。
**答え: スープが濃厚で温かいから。**

(2) 下線部 "a hot bowl of ramen makes you feel better" を日本語にしなさい。
**答え: 温かい一杯のラーメンはあなたの気分をよくしてくれます。**

(3) 筆者が札幌で驚いた理由を日本語で書きなさい。
**答え: たくさんの人が行列に並んでいたから。**

(4) 本文中で使われている接続詞 when, if, because, that をそれぞれ1つずつ含む文を本文から抜き出しなさい。
**答え:（例）when: When it is cold in winter... / if: If you visit Hokkaido... / because: ...because the soup is rich and warm. / that: I think that Sapporo ramen...**

(5) 本文の内容に合わないものを選びなさい。
ア The writer thinks ramen chefs all use the same ingredients.
イ Sapporo ramen is popular in Hokkaido.
ウ The writer visited a ramen shop in Sapporo.
エ The writer thinks good food connects people.
**答え: ア（シェフは異なる材料を使うと書かれている）**

## 大問4: 条件英作文（5点×4＝20点）

次の条件にあう英文を書きなさい。

(1) whenを使って、「暇なときにすること」を表す文を書きなさい。
**答え例: When I am free, I play video games. / I read books when I have free time.**

(2) ifを使って、「もし〜なら」の条件を含む文を書きなさい。
**答え例: If you like sushi, you should visit this restaurant. / I will help you if you need me.**

(3) becauseを使って、「自分が好きなもの＋その理由」を表す文を書きなさい。
**答え例: I like summer because I can swim in the sea. / I like this book because the story is exciting.**

(4) I don't think that を使って、否定の意見を表す文を書きなさい。
**答え例: I don't think that math is difficult. / I don't think that we need homework every day.**

## 大問5: 文の結合問題（4点×5＝20点）

次の2つの文を、指定された接続詞を使って1文にしなさい。

(1) I was walking to school. It started raining.（whenを使って）
**答え: When I was walking to school, it started raining. / It started raining when I was walking to school.**

(2) You are interested in history. You should visit this museum.（ifを使って）
**答え: If you are interested in history, you should visit this museum.**

(3) She practices piano every day. She wants to be a pianist.（becauseを使って）
**答え: She practices piano every day because she wants to be a pianist.**

(4) I believe so. He will pass the exam.（thatを使って）
**答え: I believe (that) he will pass the exam.**

(5) I don't think so. This question is easy.（I don't think thatを使って）
**答え: I don't think (that) this question is easy.**
