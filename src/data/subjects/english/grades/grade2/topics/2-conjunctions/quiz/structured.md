# 大問小問形式：接続詞（when / if / because / that）

## 大問1: 空欄補充（when / if / because / that の選択）（3点×6＝18点）

次の英文の（ ）に入る最も適切な接続詞を when / if / because / that から選びなさい。

(1) ( ) I was a child, I often played in the river.
**答え: When（「子どものとき」）**

(2) I eat instant noodles ( ) I don't have time to cook.
**答え: when / because（「料理する時間がないとき/ないので」どちらも可）**

(3) ( ) you are free, can you help me?
**答え: If（「もし暇なら」）**

(4) I think ( ) this restaurant is the best in the city.
**答え: that（「〜ということ」）**

(5) She was happy ( ) she got a good grade.
**答え: because（「良い成績をとったので」）**

(6) I hope ( ) many people will come to our concert.
**答え: that（「〜ということ」。hopeの後ろ）**

## 大問2: 書きかえ問題（4点×5＝20点）

次の文を（ ）内の指示にしたがって書きかえなさい。

(1) I was reading a book. My father came home.（whenを使って1文に）
**答え: I was reading a book when my father came home. / When my father came home, I was reading a book.**

(2) When I am sad, I listen to music.（接続詞の部分を文の後ろに移して）
**答え: I listen to music when I am sad.（カンマは不要）**

(3) The soup is delicious. I like this ramen.（becauseを使って1文に）
**答え: I like this ramen because the soup is delicious.**

(4) If it rains tomorrow, we will cancel the trip.（接続詞の部分を文の後ろに移して）
**答え: We will cancel the trip if it rains tomorrow.（カンマは不要）**

(5) I think so. He is honest.（thatを使って1文に）
**答え: I think (that) he is honest.**

## 大問3: 正誤問題（3点×6＝18点）

次の英文が正しければ○、まちがいがあれば×をつけ、正しく書きかえなさい。

(1) If it will rain tomorrow, I will stay home.
**答え: ×　→ If it rains tomorrow, I will stay home.（ifの後ろではwillを使わない）**

(2) When she happy, she sings.
**答え: ×　→ When she is happy, she sings.（be動詞が必要）**

(3) I stayed home because it was raining.
**答え: ○**

(4) Because I was tired. I went to bed early.
**答え: ×　→ Because I was tired, I went to bed early.（ピリオドではなくカンマ。becauseだけでは文にならない）**

(5) I don't think that he is wrong.
**答え: ○（英語では自然な否定の形）**

(6) I know that she can speaks English.
**答え: ×　→ I know that she can speak English.（canの後ろは動詞の原形）**

## 大問4: 語順整序（4点×6＝24点）

次の日本文にあう英文になるように、（ ）内の語を並べかえなさい。

(1) 私がラーメン店に行くとき、いつも味噌ラーメンを注文します。
( go / when / a ramen restaurant / I / to ), I always order miso ramen.
**答え: When I go to a ramen restaurant, I always order miso ramen.**

(2) もし時間があれば、サッカーの試合を見に行きましょう。
( have / if / time / you ), let's go and see a soccer game.
**答え: If you have time, let's go and see a soccer game.**

(3) 私はこのカレーが好きです。なぜならとてもおいしいからです。
I like this curry ( it / because / delicious / very / is ).
**答え: I like this curry because it is very delicious.**

(4) 私は彼女がその試合に勝てると思います。
( think / that / I / win / she / can ) the game.
**答え: I think that she can win the game.**

(5) 彼が日本語を話せると知っていますか。
( know / you / do / he / that / can ) speak Japanese?
**答え: Do you know that he can speak Japanese?**

(6) もしあなたが日本食に興味があるなら、この店に行ってみてください。
( Japanese food / you / if / are / in / interested ), please try this restaurant.
**答え: If you are interested in Japanese food, please try this restaurant.**

## 大問5: 英作文（4点×5＝20点）

次の日本文の意味を英文で表しなさい。指定された接続詞を使うこと。

(1) 私は暇なとき、よくマンガを読みます。（whenを使って）
**答え: When I am free, I often read manga. / I often read manga when I am free.**

(2) もし明日天気がよければ、自転車で学校に行きます。（ifを使って）
**答え: If it is sunny tomorrow, I will go to school by bike.**

(3) 宿題がたくさんあったので、遊べませんでした。（becauseを使って）
**答え: I couldn't play because I had a lot of homework.**

(4) 私は東京はわくわくする街だと思います。（thatを使って）
**答え: I think (that) Tokyo is an exciting city.**

(5) 私はその映画はおもしろくないと思います。（I don't thinkを使って）
**答え: I don't think (that) the movie is interesting.**
