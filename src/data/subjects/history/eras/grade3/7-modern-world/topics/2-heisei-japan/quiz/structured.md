# 平成の日本 大問小問

## 大問1: バブル経済とその崩壊（配点 20点）

1980年代後半から1990年代にかけて、日本経済は大きな転換期を迎えた。以下の問いに答えなさい。

(1) 1980年代後半に株価や地価が大幅に上昇した好景気を何というか。
→ バブル経済

(2) バブル経済が崩壊した後の長期不況は何と呼ばれるか。
→ 「失われた10年」（平成不況）

(3) バブル崩壊後、銀行が大量に抱えた問題を何というか。
→ 不良債権

(4) バブル経済の崩壊が日本社会に与えた影響を2つ述べなさい。
→ 企業の倒産・リストラが増加した。就職難が深刻化し「就職氷河期」と呼ばれた。

## 大問2: 55年体制の崩壊と政権交代（配点 20点）

戦後日本の政治体制は、1990年代に大きな変化を迎えた。以下の問いに答えなさい。

(1) 1955年から続いた自民党の長期政権体制を何というか。
→ 55年体制

(2) 1993年に非自民の連立内閣として誕生した内閣の首相は誰か。
→ 細川護熙

(3) 2009年に自民党から政権を奪った政党はどこか。
→ 民主党

(4) 55年体制が崩壊した背景として、国民が自民党政治に不満を持った理由を述べなさい。
→ 政治腐敗や汚職事件が相次ぎ、国民の政治不信が高まったため

## 大問3: 自然災害と日本社会（配点 20点）

平成時代には大きな自然災害が相次いだ。以下の問いに答えなさい。

(1) 1995年1月に発生した大震災の名称を答えなさい。
→ 阪神・淡路大震災

(2) 阪神・淡路大震災で死亡した人数はおよそ何人か。
→ 6400人以上

(3) この震災を機に重要性が認識され、活動が広がったものは何か。
→ ボランティア活動（NPOの活動）

(4) 2011年3月に発生した大震災の名称を答えなさい。
→ 東日本大震災

(5) 東日本大震災で事故を起こした原子力発電所の名称を答えなさい。
→ 福島第一原子力発電所

## 大問4: 国際貢献と外交課題（配点 20点）

冷戦終結後、日本の国際的な役割が問われるようになった。以下の問いに答えなさい。

(1) 1992年に成立した、自衛隊の海外派遣を可能にした法律は何か。
→ PKO協力法（国際平和協力法）

(2) PKO協力法に基づき自衛隊が初めて派遣された国はどこか。
→ カンボジア

(3) 日本が抱える領土問題を3つ挙げなさい。
→ 北方領土問題、竹島問題、尖閣諸島問題

(4) 北朝鮮との間で大きな外交課題となっている問題は何か。
→ 拉致問題（北朝鮮が日本人を不法に連れ去った問題）
