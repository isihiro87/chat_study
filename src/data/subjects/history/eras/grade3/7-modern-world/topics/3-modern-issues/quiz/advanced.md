# 現代の課題 発展問題

## 問1: グローバル化の光と影

現代ではグローバル化が進み、世界が一体化している。情報やヒト・モノ・カネが国境を越えて移動し、国際的な結びつきが強まっている。

(1) グローバル化が人々の生活にもたらした「利点」と「課題」をそれぞれ2つずつ挙げなさい。
→ 利点：世界中の商品やサービスを利用できるようになった、異文化交流が活発になった。課題：経済格差が拡大した、安い労働力を求めて工場が海外に移転し国内の雇用が減少した。

(2) 情報化社会の進展がグローバル化をどのように加速させたか、「インターネット」に着目して説明しなさい。
→ インターネットの普及により、世界中の情報にリアルタイムでアクセスできるようになり、国境を越えた取引や交流が飛躍的に容易になった。電子商取引やSNSを通じた情報発信が、経済のグローバル化と文化の国際化を加速させた。

(3) グローバル化に対して「反グローバリズム」の動きが生まれている理由を説明しなさい。
→ グローバル化により自国の産業が衰退したり、移民の増加により雇用が脅かされると感じる人々が増えた。また、多国籍企業が利益を独占し地域社会との格差が広がるなど、グローバル化の恩恵が均等に行き渡らないことへの不満が高まっている。

## 問2: 地球環境問題と国際協力

地球温暖化は現代世界の最大の課題の一つであり、国際社会は京都議定書やパリ協定などを通じて対策を進めてきた。

(1) 京都議定書（1997年）の限界と、パリ協定（2015年）がそれをどのように克服しようとしたかを説明しなさい。
→ 京都議定書は先進国のみに温室効果ガスの削減義務を課したため、中国やインドなどの新興国が対象外となり、世界全体の排出量削減につながらなかった。パリ協定では発展途上国も含めた全ての国が自主的な削減目標を定めることとし、世界全体での取り組みを実現しようとした。

(2) SDGs（持続可能な開発目標）が、環境問題だけでなく貧困・教育・ジェンダーなど幅広い分野を含んでいる理由を説明しなさい。
→ 環境問題は貧困や教育、経済格差などと密接に関連しており、環境だけを切り離して解決することはできない。例えば、貧困層は環境破壊の影響を受けやすく、教育の普及は環境意識の向上につながる。そのため、SDGsは持続可能な社会の実現に必要な課題を包括的に扱っている。

## 問3: 日本社会の課題と国際的役割

(1) 少子高齢化が日本の社会保障制度に与える影響を、「年金」と「医療費」の観点から説明しなさい。
→ 少子高齢化により現役世代（年金の支え手）が減少し高齢者（受給者）が増加するため、年金制度の維持が困難になる。また、高齢者の増加に伴い医療費が増大し、国や自治体の財政を圧迫する。現役世代一人当たりの負担が増え続けるという構造的な問題を抱えている。

(2) 核軍縮・核不拡散が進まない理由を、「核抑止論」の観点から説明しなさい。
→ 核保有国は、核兵器を持つことで他国からの攻撃を抑止できるという「核抑止論」に基づいて核兵器を保有している。核兵器を手放すことで安全保障上の優位性を失うことを恐れており、核軍縮の交渉が進みにくい。一方で、新たに核保有を目指す国も現れ、核不拡散の取り組みも困難な状況が続いている。

(3) 世界金融危機（2008年）が国際社会に与えた影響と、その後の対応策を述べなさい。
→ アメリカの住宅バブル崩壊に端を発した世界金融危機は、グローバル化した金融市場を通じて世界中に波及し、各国で景気後退や失業率の上昇を引き起こした。対応策として、各国の中央銀行が金融緩和を実施し、G20サミットが開催されて国際的な金融規制の強化が議論された。
