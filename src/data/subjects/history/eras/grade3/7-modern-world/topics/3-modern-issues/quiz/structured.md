# 現代の課題 大問小問

## 大問1: グローバル化と情報化（配点 20点）

現代の世界ではグローバル化と情報化が急速に進んでいる。以下の問いに答えなさい。

(1) 情報やヒト・モノ・カネが国境を越えて移動する現象を何というか。
→ グローバル化

(2) グローバル化の進展により拡大した、関税などの障壁を減らす貿易を何というか。
→ 自由貿易

(3) 携帯電話やインターネットの普及により進んだ社会の変化を何というか。
→ 情報化（情報化社会）

(4) グローバル化がもたらす課題を2つ挙げなさい。
→ 経済格差の拡大、文化の画一化（感染症の世界的流行なども可）

## 大問2: 地球環境問題への取り組み（配点 25点）

地球温暖化への対策として、国際社会はさまざまな取り組みを行ってきた。以下の問いに答えなさい。

(1) 温室効果ガスの排出により起こる地球規模の環境問題は何か。
→ 地球温暖化

(2) 1997年に採択された、先進国に温室効果ガスの削減義務を課した文書は何か。
→ 京都議定書

(3) 2015年に結ばれた、発展途上国も含めた温暖化対策の国際合意は何か。
→ パリ協定

(4) 京都議定書とパリ協定の最も大きな違いを述べなさい。
→ 京都議定書は先進国のみに削減義務を課したが、パリ協定は発展途上国も含めた全ての国が削減目標を定めた

(5) 東日本大震災後に導入が進んだ太陽光や風力などのエネルギーを何というか。
→ 再生可能エネルギー

## 大問3: SDGsと持続可能な社会（配点 20点）

2015年に国連で新たな国際目標が採択された。以下の問いに答えなさい。

(1) 2015年に国連で採択された、2030年までの達成を目指す国際目標を何というか。
→ SDGs（持続可能な開発目標）

(2) SDGsには何のゴールが定められているか。
→ 17のゴール

(3) 将来の世代のことも考えた社会づくりを何というか。
→ 持続可能な社会

(4) SDGsが目指す分野を3つ挙げなさい。
→ 貧困の撲滅、質の高い教育、気候変動への対策（他にも健康・ジェンダー平等・クリーンエネルギーなど）

## 大問4: 国際協力と日本の課題（配点 20点）

現代の日本と世界にはさまざまな課題がある。以下の問いに答えなさい。

(1) 日本で急速に進んでいる、子供が減り高齢者の割合が高まる現象を何というか。
→ 少子高齢化

(2) 国連が行う紛争地域での平和維持のための活動を何というか。
→ 平和維持活動（PKO）

(3) 民間の立場で国際協力を行う組織を何というか。
→ NGO（非政府組織）

(4) 2008年に起きた世界的な経済の混乱を何というか。
→ 世界金融危機

(5) 核兵器に関する世界的な課題を2つの観点から述べなさい。
→ 核軍縮（核兵器の削減）と核不拡散（核兵器の広がりを防ぐ）
