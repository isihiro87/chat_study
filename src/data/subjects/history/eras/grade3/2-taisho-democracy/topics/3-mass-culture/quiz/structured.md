# 大正の文化・生活 大問小問

## 大問1: メディアの発達と大衆文化（配点 20点）

大正時代には情報の伝え方が大きく変化した。以下の問いに答えなさい。

(1) 1925年に日本で始まった新しいメディアは何か。
→ ラジオ放送

(2) 大正時代に大衆娯楽として人気を集めた映像メディアを何というか。
→ 活動写真（映画）

(3) 1冊1円で販売され、読書の大衆化に貢献した文学全集を何というか。
→ 円本

(4) 大正時代に「大衆文化」が広がった要因を、メディアの発達に着目して2つ挙げなさい。
→ 新聞・雑誌が広く読まれるようになったこと、ラジオ放送が始まり全国の家庭に情報が届くようになったこと。

(5) 大正時代の大衆文化が明治時代と異なる点を説明しなさい。
→ 明治時代は文化が一部の知識人や上流層に限られていたが、大正時代にはメディアの発達により文化が広く大衆に広まった。

## 大問2: 生活の変化と関東大震災（配点 20点）

大正時代には都市の生活が大きく変化した。以下の問いに答えなさい。

(1) 大正時代に都市部で広まった洋風の住宅を何というか。
→ 文化住宅

(2) 大正時代に庶民の間に定着した洋食を2つ挙げなさい。
→ カレーライス、コロッケ

(3) 1923年9月1日に発生した大災害を何というか。
→ 関東大震災

(4) 関東大震災で特に大きな被害を受けた2つの都市を答えなさい。
→ 東京、横浜

(5) 関東大震災の死者・行方不明者はおよそ何人か。
→ 約10万5千人

(6) 関東大震災の後、東京にはどのような変化が起きたか。
→ 復興を通じて都市の近代化がさらに進んだ。

## 大問3: 大正の文学（配点 20点）

大正時代にはさまざまな文学作品が生まれた。以下の問いに答えなさい。

(1) 「羅生門」「蜘蛛の糸」を書いた大正時代の代表的な作家は誰か。
→ 芥川龍之介

(2) 「蟹工船」を書き、プロレタリア文学を代表する作家は誰か。
→ 小林多喜二

(3) プロレタリア文学とはどのような文学か説明しなさい。
→ 労働者や農民の苦しい生活を描き、社会の矛盾を告発する文学。

(4) 日用品の中に美を見出す「民芸運動」を提唱した人物は誰か。
→ 柳宗悦

(5) 「暗夜行路」を書き、「小説の神様」と呼ばれた作家は誰か。
→ 志賀直哉
