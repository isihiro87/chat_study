# 大戦後の国際秩序 発展問題

## 問1: 国際連盟の理想と限界

第一次世界大戦後、ウィルソン大統領の提案により国際連盟が設立された。しかし、アメリカ自身は参加せず、国際連盟にはいくつかの弱点があった。

(1) 国際連盟の設立の目的と、アメリカが不参加となった理由を説明しなさい。
→ 目的は世界平和の維持と国際紛争の平和的解決。アメリカが不参加となったのは、議会がヨーロッパの問題に巻き込まれることに反対し、孤立主義的な外交方針を重視したため。

(2) 国際連盟にアメリカが参加しなかったことが、その後の国際政治にどのような影響を与えたか推測しなさい。
→ 世界最大の経済力・軍事力を持つアメリカの不参加により、国際連盟の制裁力が弱くなった。後に日本やドイツが侵略行為を行った際にも有効な対応がとれず、第二次世界大戦を防ぐことができなかった。

## 問2: 民族自決の矛盾

ウィルソンが提唱した「民族自決」の原則は、東ヨーロッパでは多くの国の独立をもたらしたが、アジア・アフリカの植民地には適用されなかった。

(1) 民族自決がアジア・アフリカに適用されなかった理由を、列強の利害に着目して説明しなさい。
→ イギリス・フランスなどの列強はアジア・アフリカに広大な植民地を持っており、民族自決を適用すれば植民地の独立を認めることになり、自国の経済的・政治的利益を失うことになるため。

(2) 民族自決がアジアに適用されなかったことが、五・四運動や三・一独立運動にどのようにつながったか、因果関係を説明しなさい。
→ パリ講和会議で民族自決がアジアに適用されず、逆に山東省の権益が日本に渡るなど、アジアの民族が不当に扱われたことに対する失望と怒りが、中国での五・四運動や朝鮮での三・一独立運動を引き起こした。

## 問3: ワシントン会議と東アジアの秩序

1921年にワシントン会議が開かれ、太平洋地域の国際秩序が取り決められた。

(1) ワシントン会議で日英同盟が解消された背景を、アメリカの立場から説明しなさい。
→ アメリカは日英同盟が日本のアジア・太平洋での勢力拡大を後押ししていると見ており、これを解消させることで日本の勢力を抑えようとした。

(2) ワシントン会議の決定（太平洋地域の現状維持、中国の主権尊重）が日本にとってどのような意味を持ったか説明しなさい。
→ 日本にとっては、中国大陸での勢力拡大に歯止めをかけられ、日英同盟という外交上の後ろ盾も失うことになった。国際的な枠組みの中で協調外交を行う必要に迫られた。
