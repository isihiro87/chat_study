# ロシア革命 大問小問

## 大問1: ロシア革命の背景と経過（配点 20点）

1917年、ロシアで大きな革命が起こった。以下の問いに答えなさい。

(1) この革命を何というか。
→ ロシア革命

(2) ロシア革命を指導した人物は誰か。
→ レーニン

(3) 革命で各地に結成された、労働者や兵士の代表会議を何というか。
→ ソビエト

(4) ロシア革命が起きた背景を、「第一次世界大戦」という語を用いて説明しなさい。
→ 第一次世界大戦による疲弊と食糧不足で国民の不満が高まり、レーニンの指導のもと革命が起きた。

(5) ロシア革命後に誕生した政府はどのような点で世界史上画期的だったか。
→ 世界初の社会主義政府が誕生した点。

## 大問2: シベリア出兵（配点 16点）

ロシア革命の後、列強がシベリアに軍隊を送った。以下の問いに答えなさい。

(1) この出来事を何というか。
→ シベリア出兵

(2) シベリア出兵に参加した国を2つ挙げなさい。
→ 日本、アメリカ（イギリス、フランスなども可）

(3) 列強がシベリアに軍隊を送った目的を説明しなさい。
→ 社会主義革命の影響が自国に広がることを恐れ、革命政府に干渉するため。

(4) 日本がシベリアから撤兵した年は何年か。
→ 1922年

## 大問3: ソ連の成立とスターリン（配点 24点）

ロシア革命後、新しい国家が建設された。以下の問いに答えなさい。

(1) 1922年に成立した国家の正式名称は何か。
→ ソビエト社会主義共和国連邦

(2) レーニンの死後、ソ連の指導者となった人物は誰か。
→ スターリン

(3) スターリンが実施した経済計画を何というか。
→ 五か年計画

(4) 五か年計画の2つの柱を挙げなさい。
→ 重工業の発展と農業の集団化

(5) 農業の集団化とはどのような政策か説明しなさい。
→ 個人の農地を国や共同体の管理にまとめ、大規模な集団農場にする政策。

(6) スターリンの政治の問題点を述べなさい。
→ 強権的な独裁体制のもとで多くの犠牲者が出た。
