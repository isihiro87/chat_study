# 第一次世界大戦の勃発 大問小問

## 大問1: ヨーロッパの対立構造（配点 20点）

20世紀初めのヨーロッパでは、2つの大きな陣営が対立していた。以下の問いに答えなさい。

(1) ドイツ・オーストリア・イタリアが結んだ軍事同盟を何というか。
→ 三国同盟

(2) イギリス・フランス・ロシアが結んだ協力関係を何というか。
→ 三国協商

(3) 2つの陣営の対立が深まった原因を2つ挙げなさい。
→ 植民地や勢力圏をめぐる対立、軍備の拡大競争

(4) バルカン半島は当時何と呼ばれていたか。また、その理由を述べなさい。
→ 「ヨーロッパの火薬庫」。多くの民族が住み、列強の利害が絡み合って紛争が絶えず、いつ大きな戦争に発展してもおかしくない状態だったため。

## 大問2: サラエボ事件と第一次世界大戦の開戦（配点 20点）

1914年にある事件をきっかけに、ヨーロッパ全体を巻き込む大戦争が始まった。以下の問いに答えなさい。

(1) 1914年にバルカン半島のサラエボで起きた事件を何というか。
→ サラエボ事件

(2) サラエボ事件で暗殺されたのは誰か。
→ オーストリアの皇位継承者（皇太子）夫妻

(3) この事件をきっかけに始まった戦争を何というか。
→ 第一次世界大戦

(4) 各国が次々と参戦した理由を「同盟」という語を用いて説明しなさい。
→ 三国同盟と三国協商という同盟関係で結ばれていた各国が、味方の国を支援するために連鎖的に参戦したため。

## 大問3: 総力戦と新兵器（配点 20点）

第一次世界大戦は、それまでの戦争とは大きく異なる性質を持っていた。以下の問いに答えなさい。

(1) 第一次世界大戦のように、国の全ての力を動員して行う戦争を何というか。
→ 総力戦

(2) 「国の全ての力」とは具体的に何を指すか。3つ挙げなさい。
→ 軍事力・経済力・国民の労働力（科学技術なども可）

(3) 第一次世界大戦で初めて実戦に投入された新兵器を3つ挙げなさい。
→ 戦車・飛行機・毒ガス（潜水艦も可）

(4) 三国同盟に参加していたにもかかわらず、途中で連合国側に移った国はどこか。
→ イタリア

(5) 第一次世界大戦は何年間続いたか。
→ 4年間（1914年〜1918年）
