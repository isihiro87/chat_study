# 大戦後の国際秩序 大問小問

## 大問1: パリ講和会議とベルサイユ条約（配点 20点）

1919年、第一次世界大戦の講和のためにある国際会議が開かれた。以下の問いに答えなさい。

(1) この国際会議を何というか。
→ パリ講和会議

(2) この会議で結ばれた講和条約を何というか。
→ ベルサイユ条約

(3) この条約でドイツに課された条件を3つ挙げなさい。
→ 巨額の賠償金、植民地の没収、軍備の縮小

(4) この条約がドイツに厳しい内容だったことが、後のドイツにどのような影響を与えたか推測しなさい。
→ ドイツ国民の不満が高まり、後のナチスの台頭やヒトラーの登場につながった。

## 大問2: 国際連盟とワシントン会議（配点 24点）

第一次世界大戦後、国際平和を目指す新しい仕組みが作られた。以下の問いに答えなさい。

(1) 1920年に設立された国際平和機関を何というか。
→ 国際連盟

(2) 国際連盟の設立を提案した人物の名前と国名を答えなさい。
→ ウィルソン大統領（アメリカ）

(3) 国際連盟の本部が置かれた都市はどこか。
→ ジュネーブ

(4) 国際連盟で日本は何という立場だったか。
→ 常任理事国

(5) アメリカが国際連盟に参加しなかった理由を述べなさい。
→ アメリカ議会の反対があり、ヨーロッパの問題に巻き込まれたくないという考え（孤立主義）が強かったため。

(6) 1921年のワシントン会議で解消された同盟は何か。
→ 日英同盟

## 大問3: アジアの民族運動（配点 20点）

第一次世界大戦後、アジア各地で民族運動が起きた。以下の問いに答えなさい。

(1) ウィルソン大統領が提唱した、民族が自分たちの運命を決める権利の考え方を何というか。
→ 民族自決

(2) 1919年に中国で起きた反日・反帝国主義運動を何というか。また、そのきっかけを述べなさい。
→ 五・四運動。パリ講和会議で山東省のドイツ権益が日本に渡ることが決まったことがきっかけ。

(3) 1919年に朝鮮で起きた独立運動を何というか。
→ 三・一独立運動

(4) インドで非暴力・不服従運動を指導した人物は誰か。
→ ガンディー

(5) ドイツで制定された、男女普通選挙や労働者の権利を定めた憲法を何というか。
→ ワイマール憲法
