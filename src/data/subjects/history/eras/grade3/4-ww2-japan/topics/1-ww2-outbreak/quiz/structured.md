# 第二次世界大戦の始まり 大問小問

## 大問1: 第二次世界大戦の開始（配点 18点）

1939年、ヨーロッパで大規模な戦争が始まった。以下の問いに答えなさい。

(1) 1939年にドイツとソ連が結んだ条約を何というか。
→ 独ソ不可侵条約

(2) 1939年9月にドイツが侵攻した国はどこか。
→ ポーランド

(3) ドイツのポーランド侵攻に対して宣戦布告した2つの国を答えなさい。
→ イギリスとフランス

(4) ドイツが素早い攻撃で次々とヨーロッパ諸国を占領した戦法を何というか。
→ 電撃戦

(5) 1940年にドイツに占領された大国はどこか。
→ フランス

(6) ヨーロッパ各地でドイツの支配に対して行われた抵抗運動を何というか。
→ レジスタンス

## 大問2: 枢軸国と連合国（配点 15点）

第二次世界大戦では、二つの陣営が対立した。以下の問いに答えなさい。

(1) 1940年に日本・ドイツ・イタリアが結んだ軍事同盟を何というか。
→ 日独伊三国同盟

(2) 日本・ドイツ・イタリアを中心とする陣営を何というか。
→ 枢軸国

(3) イギリス・アメリカ・ソ連・中国などを中心とする陣営を何というか。
→ 連合国

(4) ナチス・ドイツによるユダヤ人の大量虐殺を何というか。
→ ホロコースト

(5) ユダヤ人が送られた施設として最も有名なものの名前を答えなさい。
→ アウシュビッツ（強制収容所）

## 大問3: 日本の南進とABCD包囲陣（配点 17点）

日本は東南アジアへの南進を進めたが、各国の経済封鎖に直面した。以下の問いに答えなさい。

(1) 日本が東南アジアに進出した目的は何か。
→ 石油などの資源を確保すること

(2) 日本がフランス領インドシナ南部に進駐したことへの対応として、アメリカなどが行った経済封鎖を何というか。
→ ABCD包囲陣（ABCD包囲網）

(3) ABCDの4文字はそれぞれどの国を指すか、すべて答えなさい。
→ A=America（アメリカ）、B=Britain（イギリス）、C=China（中国）、D=Dutch（オランダ）

(4) ABCD包囲陣で日本に対して禁止されたものは何か。
→ 石油の輸出

(5) 資源を断たれた日本は、その後どのような決断をしたか。
→ 開戦を決意した（太平洋戦争の開始）
