# 戦時下の生活 大問小問

## 大問1: 学徒出陣と勤労動員（配点 15点）

戦争が長引くにつれ、兵力も労働力も不足した。以下の問いに答えなさい。

(1) 1943年から大学生が戦場に送られたことを何というか。
→ 学徒出陣

(2) 学徒出陣の対象となったのはどのような大学生か。
→ 文科系の大学生

(3) 中学生や女学生が軍需工場で働かされたことを何というか。
→ 勤労動員

(4) 学徒出陣や勤労動員が行われた理由を答えなさい。
→ 戦争の長期化により兵力と労働力が不足したから

(5) 国民すべてが戦争に動員される体制を何というか。
→ 総動員体制

## 大問2: 疎開と金属供出（配点 15点）

空襲が激しくなると、国民の生活はさらに制限された。以下の問いに答えなさい。

(1) 都市部の子どもたちが地方に移り住むことを何というか。
→ 疎開

(2) 疎開の目的は何か。
→ 空襲から子どもたちを守るため

(3) 家庭の鍋や寺の鐘を政府に差し出すことを何というか。
→ 金属供出

(4) 金属供出が行われた理由を答えなさい。
→ 兵器を作るための金属が不足したから

(5) 空襲時に夜間に明かりを消すことを何というか。
→ 灯火管制

## 大問3: 配給制と国民生活の統制（配点 15点）

戦時中、国民の日常生活は厳しく統制された。以下の問いに答えなさい。

(1) 食料や日用品を国が管理して分配した制度を何というか。
→ 配給制

(2) 配給で品物を受け取るために必要だったものは何か。
→ 切符（配給切符）

(3) 国民に我慢を求めた有名なスローガンを答えなさい。
→ 「欲しがりません勝つまでは」

(4) 地域の相互監視や防空活動を行った組織を何というか。
→ 隣組

(5) 朝鮮半島や中国から日本に連れてこられ、炭鉱や工場で過酷な労働を強いられた人々について、このような行為を何というか。
→ 強制連行
