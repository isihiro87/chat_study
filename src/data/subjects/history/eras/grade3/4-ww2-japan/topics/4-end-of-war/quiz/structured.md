# 戦争の終結 大問小問

## 大問1: ヨーロッパ戦線の展開（配点 15点）

第二次世界大戦のヨーロッパ戦線について、以下の問いに答えなさい。

(1) 1943年に枢軸国で最初に降伏した国はどこか。
→ イタリア

(2) ソ連がドイツに反撃し勝利した戦いを何というか。
→ スターリングラードの戦い

(3) 1944年に連合国がフランス北部に上陸した作戦を何というか。
→ ノルマンディー上陸作戦

(4) ドイツが降伏したのは西暦何年何月か。
→ 1945年5月

(5) ノルマンディー上陸作戦の意義を答えなさい。
→ 西部戦線が開かれてドイツは東西から挟撃され、連合国の反攻が本格化した

## 大問2: 日本本土への攻撃（配点 18点）

太平洋戦争末期、日本本土への攻撃が激しくなった。以下の問いに答えなさい。

(1) 1944年に陥落し、日本本土空襲の拠点となった島はどこか。
→ サイパン島

(2) アメリカ軍が日本本土空襲に使用した大型爆撃機は何か。
→ B-29

(3) 1945年3月10日に東京が受けた大規模な空襲を何というか。
→ 東京大空襲

(4) 1945年にアメリカ軍が上陸し、住民を巻き込む激しい地上戦が行われた場所はどこか。
→ 沖縄

(5) 沖縄戦ではおよそ何万人が犠牲になったか。
→ 約20万人

(6) 沖縄戦の特徴として、通常の戦闘と異なる点を答えなさい。
→ 住民を巻き込んだ地上戦であり、住民の集団自決なども起きた

## 大問3: 原爆投下と終戦（配点 22点）

1945年夏、第二次世界大戦は終結を迎えた。以下の問いに答えなさい。

(1) 1945年8月6日に原子爆弾が投下された都市はどこか。
→ 広島

(2) 1945年8月9日に原子爆弾が投下された都市はどこか。
→ 長崎

(3) 1945年8月8日に日ソ中立条約を破って満州に侵攻した国はどこか。
→ ソ連

(4) ソ連の対日参戦を事前に決めた秘密協定が結ばれた会談を何というか。
→ ヤルタ会談

(5) 連合国が日本に無条件降伏を求めた宣言を何というか。
→ ポツダム宣言

(6) 1945年8月15日に天皇がラジオで国民に終戦を伝えた放送を何というか。
→ 玉音放送

(7) 日本が降伏を決定した直接的な理由を2つ答えなさい。
→ 広島・長崎への原爆投下とソ連の参戦
