# 日本国憲法 構造化問題

## 大問1: 日本国憲法の基本

(1) 日本国憲法が公布された日と施行された日を答えよ。
→ 公布：1946年11月3日、施行：1947年5月3日

(2) 日本国憲法の三大原則を3つ答えよ。
→ 国民主権・基本的人権の尊重・平和主義

(3) 日本国憲法で天皇はどのような地位と定められたか。
→ 日本国と日本国民統合の象徴

(4) 第9条で定められている内容を2つ答えよ。
→ 戦争放棄と戦力不保持

## 大問2: 大日本帝国憲法との比較

(1) 大日本帝国憲法では主権は誰にあったか。
→ 天皇（天皇主権）

(2) 日本国憲法では主権は誰にあるか。
→ 国民（国民主権）

(3) 大日本帝国憲法での天皇の地位と、日本国憲法での天皇の地位の違いを述べよ。
→ 大日本帝国憲法では天皇は国の元首で統治権を持つ主権者であったが、日本国憲法では天皇は「象徴」となり、政治的権力を持たない。

(4) 日本国憲法で新たに定められた、大日本帝国憲法にはなかった原則は何か。
→ 平和主義（戦争の放棄）

## 大問3: 戦後の社会改革

(1) 農地改革の内容と結果を説明せよ。
→ 政府が地主の農地を買い上げて小作人に安く売り渡した。その結果、自作農が大幅に増加した。

(2) 財閥解体はなぜ行われたか。
→ 三井・三菱などの巨大資本家集団が日本経済を支配していたため、経済の民主化を進めるために解体された。

(3) 教育基本法の目的は何か。
→ 民主主義の教育の基本を定め、民主的な社会を担う人材を育てること。

(4) 労働組合法と労働基準法の違いを説明せよ。
→ 労働組合法は労働者の団結権・団体交渉権・争議権を保障する法律。労働基準法は賃金・労働時間などの労働条件の最低基準を定めた法律。

## 大問4: 憲法と社会への影響

(1) 選挙法改正で実現した女性参政権の意義を説明せよ。
→ 戦前は男子のみに選挙権があったが、戦後は満20歳以上の男女に選挙権が与えられ、女性の政治参加が実現した。

(2) 地方自治法が制定された目的を説明せよ。
→ 地方の政治を住民が自主的に行えるようにし、中央集権的な戦前の体制を改めて民主的な地方行政を実現するため。

(3) 日本国憲法の制定が日本社会に与えた影響を、政治・権利の2つの面から述べよ。
→ 政治面では国民主権が確立され、国民が主権者として政治に参加する民主政治が実現した。権利の面では基本的人権の尊重により、国民の自由と権利が保障されるようになった。
