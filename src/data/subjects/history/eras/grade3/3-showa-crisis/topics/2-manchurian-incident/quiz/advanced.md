# ファシズムの台頭と満州事変 発展問題

## 問1: ファシズム台頭の背景

イタリアやドイツでファシズムが台頭した。以下の問いに答えなさい。

(1) ファシズムの特徴を3つ答えなさい。
→ 個人の自由を否定、民主主義を否定、強い指導者による独裁政治

(2) 世界恐慌後にファシズムが支持を集めた理由を、経済状況と国民心理の両面から説明しなさい。
→ 世界恐慌で経済が混乱し大量の失業者が出る中、民主主義では問題を解決できないという不満が広がった。ファシズムの指導者たちは強いリーダーシップで国を立て直すと約束し、経済的苦境に絶望した国民の支持を集めた。

(3) イタリアのファシズムとドイツのナチズムの共通点と相違点をそれぞれ1つ答えなさい。
→ 共通点：民主主義を否定する全体主義で対外侵略を推進した。相違点：ドイツのナチズムはユダヤ人に対する人種差別・迫害を政策の中心に据えたが、イタリアのファシズムにはそのような特徴は薄かった。

## 問2: 満州事変から国際連盟脱退への経緯

満州事変から国際連盟脱退に至る流れについて、以下の問いに答えなさい。

(1) 柳条湖事件を「自作自演」と呼ぶのはなぜか説明しなさい。
→ 関東軍が自ら南満州鉄道の線路を爆破しておきながら、それを中国側の仕業として軍事行動を開始したため。自分で事件を起こして相手のせいにした。

(2) 満州国は「独立国」とされていたが実態はどうだったか説明しなさい。
→ 清の最後の皇帝・溥儀を元首に据えたが、実際の権力は日本の関東軍が握っており、日本が支配する傀儡国家だった。

(3) 日本の国際連盟脱退がその後の日本にどのような影響を与えたか説明しなさい。
→ 国際的に孤立し、国際社会の歯止めがなくなったことで、軍部の暴走がさらに加速した。やがてドイツ・イタリアと結びつき、第二次世界大戦への道を進むことになった。

## 問3: 軍部の台頭と政党政治の崩壊

五・一五事件と二・二六事件について、以下の問いに答えなさい。

(1) 五・一五事件と二・二六事件を起こしたのがそれぞれ「海軍」と「陸軍」の青年将校であったことから、当時の日本の軍部全体についてどのようなことが言えるか。
→ 海軍・陸軍の両方に政党政治に不満を持つ勢力がいたことから、軍部全体に政治への介入や既存の政治体制の打倒を目指す動きが広がっていたことがわかる。

(2) 五・一五事件で政党内閣が終わった後、日本の政治はどのように変化したか説明しなさい。
→ 政党出身の首相に代わり、軍人や官僚が首相に任命されるようになった。テロを恐れた政治家が軍部に逆らえなくなり、軍部主導の政治が行われるようになった。

(3) 「話せばわかる」という犬養毅の言葉は歴史的にどのような意味を持つか説明しなさい。
→ 対話と民主主義の力を信じた政党政治家の最後の訴えであり、暴力によってそれが断ち切られたことは、日本の民主主義が軍事力の前に屈した象徴的な出来事とされている。
