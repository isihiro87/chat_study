# 日中戦争 発展問題

## 問1: 日中戦争の長期化

日中戦争は日本が短期間で終わらせるつもりだったが、泥沼化した。以下の問いに答えなさい。

(1) 日中戦争が泥沼化した原因を、中国側の対応に注目して説明しなさい。
→ 国民党と共産党が対立を一時的にやめて抗日民族統一戦線を結成し、協力して日本に抵抗した。さらに蔣介石が首都を内陸の重慶に移し、広大な中国の国土を利用した長期抗戦を行ったため。

(2) 日中戦争の長期化が日本国内にどのような影響を与えたか、政治と国民生活の両面から説明しなさい。
→ 政治面：国家総動員法の制定により政府が議会を通さず人や物資を動員する権限を持ち、大政翼賛会で政党が解散するなど民主主義が形骸化した。国民生活面：配給制や切符制が導入され、軍需品の生産が優先されたことで日用品や食料が不足した。

(3) 盧溝橋事件と柳条湖事件の共通点と相違点を答えなさい。
→ 共通点：どちらも日本と中国の軍事衝突が戦争や事変の発端となった。相違点：柳条湖事件は関東軍の自作自演だったが、盧溝橋事件は偶発的な衝突がきっかけだったとされる。

## 問2: 国家総動員法と戦時体制

国家総動員法の制定とその後の戦時体制について、以下の問いに答えなさい。

(1) 国家総動員法が「議会の承認なしに」動員できると定めた意味を、民主主義の観点から説明しなさい。
→ 通常は政府が国民の権利を制限するには議会の審議と承認が必要だが、国家総動員法は政府に白紙委任状を与えるような法律で、議会制民主主義が事実上無力化された。

(2) 大政翼賛会の結成が日本の政治にもたらした影響を説明しなさい。
→ 全ての政党が解散して大政翼賛会に統合されたことで、政府への批判や反対意見を述べる場がなくなった。一党独裁のような体制となり、国民を戦争に向けて一元的に動員する翼賛体制が完成した。

## 問3: 皇民化政策の問題点

朝鮮や台湾で行われた皇民化政策について、以下の問いに答えなさい。

(1) 皇民化政策の具体的な内容を3つ答えなさい。
→ 日本語の使用の強制、創氏改名（日本式の姓名への変更の強制）、神社参拝の強制

(2) 創氏改名が植民地の人々にとってどのような意味を持ったか説明しなさい。
→ 先祖代々受け継いできた名前を捨てさせられることは、民族のアイデンティティや誇りを否定されることを意味し、精神的に大きな苦痛を伴った。

(3) 皇民化政策は何を目的としていたか説明しなさい。
→ 植民地の人々を日本人として同化させ、戦争に動員するための忠実な臣民にすることを目的としていた。
