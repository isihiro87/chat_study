# 日中戦争 大問小問

## 大問1: 盧溝橋事件と日中戦争（配点 18点）

1937年、北京郊外で日本軍と中国軍が衝突する事件が起き、全面戦争へと発展した。以下の問いに答えなさい。

(1) この事件を何というか。
→ 盧溝橋事件

(2) この事件をきっかけに始まった戦争を何というか。
→ 日中戦争

(3) 日本軍が占領した中国の首都はどこか。
→ 南京

(4) 中国側が日本の侵略に対抗するため国民党と共産党が結成した組織を何というか。
→ 抗日民族統一戦線

(5) 蔣介石が首都を移した内陸の都市はどこか。
→ 重慶

(6) 日本が短期間で戦争を終わらせられなかった理由を答えなさい。
→ 抗日民族統一戦線の結成により中国側の抵抗が強まり、蔣介石が重慶に移って長期抗戦を続けたから

## 大問2: 戦時体制の整備（配点 20点）

日中戦争が長引くなか、日本政府は国民を戦争に動員する体制を整えた。以下の問いに答えなさい。

(1) 1938年に制定された、議会の承認なしに人や物資を動員できる法律を何というか。
→ 国家総動員法

(2) 1940年に全ての政党が解散して統合された組織を何というか。
→ 大政翼賛会

(3) 戦時中に食料や日用品を国が管理して分配した制度を何というか。
→ 配給制

(4) 配給で品物を受け取るために必要だったものは何か。
→ 切符（配給切符）

(5) 軍需品の生産を優先するため、国民の生活にどのような影響があったか。
→ 日用品や食料が不足し、配給制や切符制で制限された

## 大問3: 植民地政策と国際関係（配点 12点）

日中戦争期の日本の植民地政策と国際関係について、以下の問いに答えなさい。

(1) 朝鮮や台湾で日本語の使用や日本式の名前を強制した政策を何というか。
→ 皇民化政策

(2) 朝鮮の人々に日本式の姓名に変えさせたことを何というか。
→ 創氏改名

(3) 1936年に日本とドイツが結んだ協定を何というか。
→ 日独防共協定

(4) この協定にのちに加わった国はどこか。
→ イタリア
