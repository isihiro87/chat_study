# 世界恐慌 大問小問

## 大問1: 世界恐慌とアメリカの対応（配点 20点）

1929年、アメリカのある場所で株価が大暴落し、世界的な経済危機が始まった。以下の問いに答えなさい。

(1) この経済危機を何というか。
→ 世界恐慌

(2) 株価の大暴落が起きた場所を答えなさい。
→ ニューヨーク株式市場

(3) この経済危機に対して、アメリカのルーズベルト大統領が行った経済政策を何というか。
→ ニューディール政策

(4) この政策では、どのような方法で失業者を救おうとしたか。
→ 公共事業を拡大して雇用を生み出した

(5) この政策では農業や工業に対してどのような対策をとったか。
→ 生産調整を行い、価格の安定を図った

## 大問2: ブロック経済と「持たざる国」（配点 18点）

世界恐慌に対して、イギリスやフランスは独自の経済政策で対応した。以下の問いに答えなさい。

(1) イギリスやフランスが採用した、自国と植民地の間だけで貿易を行う経済政策を何というか。
→ ブロック経済

(2) ブロック経済では、外国の商品に対してどのような対応をしたか。
→ 高い関税をかけて締め出した

(3) イギリスやフランスがブロック経済を採用できた理由を答えなさい。
→ 広大な植民地を持っていたから

(4) ブロック経済によって打撃を受けた国を3つ答えなさい。
→ 日本・ドイツ・イタリア

(5) これらの国が「持たざる国」と呼ばれた理由を答えなさい。
→ 植民地が少なく、資源や市場を確保する経済圏を持たなかったから

## 大問3: ソ連と日本の状況（配点 12点）

世界恐慌の影響は世界中に広がったが、ソ連はほとんど影響を受けなかった。一方、日本にも大きな影響が及んだ。以下の問いに答えなさい。

(1) ソ連が世界恐慌の影響を受けなかった理由として、どのような経済政策を行っていたからか。
→ 五か年計画（計画経済）

(2) 五か年計画によりソ連はどのような地位を得たか。
→ アメリカに次ぐ工業国に成長した

(3) 世界恐慌の影響で1930年に日本が陥った不況を何というか。
→ 昭和恐慌

(4) 昭和恐慌が日本の農村にどのような影響を与えたか、2つ答えなさい。
→ 「身売り」（娘が売られる）、「欠食児童」（食事ができない子ども）が発生した
