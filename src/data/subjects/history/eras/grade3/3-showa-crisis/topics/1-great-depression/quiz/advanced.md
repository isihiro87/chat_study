# 世界恐慌 発展問題

## 問1: 世界恐慌への各国の対応比較

世界恐慌に対して、各国はそれぞれ異なる対応をとった。以下の問いに答えなさい。

(1) アメリカ・イギリスとフランス・ソ連の3つの対応策をそれぞれ答えなさい。
→ アメリカ：ニューディール政策（公共事業で雇用を創出）、イギリス・フランス：ブロック経済（植民地との経済圏を形成）、ソ連：五か年計画（計画経済で工業化を推進）

(2) ブロック経済が結果的に第二次世界大戦の原因の一つとなった理由を説明しなさい。
→ ブロック経済により植民地を持たない日本・ドイツ・イタリアが資源や市場を確保できなくなり、これらの国が資源を求めて対外侵略に向かったため。

(3) ソ連が世界恐慌の影響を受けなかったのはなぜか、経済体制の違いから説明しなさい。
→ ソ連は資本主義国と異なり計画経済（社会主義経済）を採用しており、世界市場との結びつきが弱かったため、資本主義圏の恐慌の影響を直接受けなかった。

## 問2: 昭和恐慌と日本社会の変化

世界恐慌は日本にも大きな影響を与え、昭和恐慌と呼ばれる深刻な不況をもたらした。以下の問いに答えなさい。

(1) 昭和恐慌が日本の農村にもたらした影響を具体的に2つ答えなさい。
→ 農産物の価格が暴落して農家が困窮し、「身売り」（娘が都会に売られる）や「欠食児童」（食事ができない子ども）が発生した。

(2) 昭和恐慌による社会不安が、その後の日本の政治にどのような影響を与えたか説明しなさい。
→ 経済的苦境に対する国民の不満が高まり、政党政治への不信感が広がった。その結果、軍部が国民の支持を集め、やがて五・一五事件や二・二六事件を経て軍部主導の政治へと転換していった。

(3) 「持たざる国」であった日本が世界恐慌後にとった行動を、満州事変と関連づけて説明しなさい。
→ ブロック経済により資源や市場の確保が困難になった日本は、中国東北部（満州）の資源と市場に目をつけ、1931年に関東軍が柳条湖事件を起こして満州事変を開始した。資源不足を軍事的な侵略で解決しようとした。

## 問3: ニューディール政策の意義

ルーズベルト大統領のニューディール政策について、以下の問いに答えなさい。

(1) ニューディール政策の具体的な内容を2つ答えなさい。
→ 公共事業（ダムや道路の建設など）で失業者に仕事を与えたこと、農業・工業の生産調整を行い価格の安定を図ったこと。

(2) ニューディール政策は、それまでの資本主義のあり方とどのように異なっていたか説明しなさい。
→ それまでの資本主義は政府が経済にあまり介入しない自由放任主義が基本だったが、ニューディール政策では政府が積極的に経済に介入して景気回復を図った点が画期的だった。
