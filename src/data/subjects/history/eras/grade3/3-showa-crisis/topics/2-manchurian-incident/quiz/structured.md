# ファシズムの台頭と満州事変 大問小問

## 大問1: ファシズムの台頭（配点 15点）

世界恐慌後、ヨーロッパで新しい政治体制が台頭した。以下の問いに答えなさい。

(1) 個人の自由や民主主義を否定し、独裁的な指導者のもとで国民を統制する政治体制を何というか。
→ ファシズム

(2) イタリアでファシスト党を率いて独裁政治を行った人物は誰か。
→ ムッソリーニ

(3) ドイツで政権を握り独裁政治を行った人物と、その人物が率いた政党名を答えなさい。
→ ヒトラー、ナチス（国民社会主義ドイツ労働者党）

(4) ヒトラーが迫害した民族は何か。
→ ユダヤ人

(5) ムッソリーニのイタリアが侵攻したアフリカの国はどこか。
→ エチオピア

## 大問2: 満州事変と国際的孤立（配点 20点）

1931年、日本の関東軍が中国東北部で軍事行動を起こした。以下の問いに答えなさい。

(1) 関東軍が南満州鉄道の線路を爆破した事件を何というか。
→ 柳条湖事件

(2) (1)をきっかけに始まった事変を何というか。
→ 満州事変

(3) 1932年に建国された国の名前と、その元首に据えられた人物を答えなさい。
→ 満州国、溥儀（清の最後の皇帝）

(4) 国際連盟が派遣した調査団の名前を答えなさい。
→ リットン調査団

(5) この調査団の勧告はどのような内容だったか。
→ 満州国を認めず、日本に撤退を勧告した

(6) 日本は1933年にどのような行動をとったか。
→ 国際連盟を脱退した

## 大問3: 五・一五事件と二・二六事件（配点 15点）

1930年代に日本国内で軍部によるテロ事件が相次いだ。以下の問いに答えなさい。

(1) 1932年5月15日に起きた事件名と、暗殺された人物の名前を答えなさい。
→ 五・一五事件、犬養毅首相

(2) 五・一五事件を起こしたのはどのような人々か。
→ 海軍の青年将校

(3) 五・一五事件の結果、日本の政治にどのような変化が起きたか。
→ 政党内閣が終わりを迎えた

(4) 1936年2月26日に起きた事件名と、起こした人々を答えなさい。
→ 二・二六事件、陸軍の青年将校

(5) 二・二六事件の結果、日本の政治にどのような変化が起きたか。
→ 軍部の政治的発言力がさらに強まった
