# 富国強兵・殖産興業 一問一答

Q1. 欧米に対抗するための国策スローガンを何というか。
A1. 富国強兵
Q2. 産業を育成する政策を何というか。
A2. 殖産興業
Q3. 1872年に開通した鉄道はどこの区間か。
A3. 新橋-横浜間
Q4. 群馬県に設立された官営の模範工場は何か。
A4. 富岡製糸場
Q5. 富岡製糸場でどの国の技術が導入されたか。
A5. フランス
Q6. 統一貨幣制度で使われた単位は何か。
A6. 円・銭・厘
Q7. 欧米の文化を積極的に取り入れる動きを何というか。
A7. 文明開化
Q8. 『学問のすゝめ』を著した人物は誰か。
A8. 福沢諭吉
Q9. 福沢諭吉が説いた考え方を答えなさい。
A9. 「天は人の上に人を造らず」（人間の平等と学問の大切さ）
Q10. 太陽暦が採用されたのは何年か。
A10. 1873年（明治5年の末）
Q11. 文明開化で広まった生活の変化を3つ答えなさい。
A11. ランプ・洋服・牛鍋（ガス灯・パン・靴なども可）
Q12. ルソーの思想を紹介した人物は誰か。
A12. 中江兆民
Q13. 郵便制度を整備した人物は誰か。
A13. 前島密
Q14. 官営模範工場の目的は何か。
A14. 政府が率先して近代的な工場を建設し、民間の産業発展を促すこと
