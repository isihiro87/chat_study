# 幕藩体制の仕組み 一問一答

## 幕府の政治
Q1. 第5代将軍で生類憐みの令を出した人物は誰か。
A1. 徳川綱吉

Q2. 生類憐みの令とはどのような法令か。
A2. 動物の殺生を禁止した法令（特に犬を保護）

Q3. 綱吉が学問を奨励するために建てた施設は何か。
A3. 湯島聖堂

Q4. 綱吉が重んじた学問は何か。
A4. 朱子学（儒学）

Q5. 綱吉の時代に行われた貨幣改鋳の結果どうなったか。
A5. 貨幣の質が低下し、物価が上昇した

Q6. 第6・7代将軍に仕えた学者で、正徳の治を行った人物は誰か。
A6. 新井白石

Q7. 新井白石が行った政治を何というか。
A7. 正徳の治

Q8. 新井白石が貨幣について行った政策は何か。
A8. 貨幣の質を元に戻した（良質の貨幣を鋳造）

Q9. 新井白石が長崎貿易について行った政策は何か。
A9. 金銀の海外流出を防ぐために長崎貿易を制限した

Q10. 幕府の財政が次第に悪化した主な原因は何か。
A10. 米価の下落、出費の増大、貨幣経済の浸透

Q11. 石高制とはどのような制度か。
A11. 土地の生産力を米の収穫量（石高）で表し、これを基準に年貢や大名の格式を決める制度

Q12. 幕府と藩がそれぞれの領地を支配する政治体制を何というか。
A12. 幕藩体制
