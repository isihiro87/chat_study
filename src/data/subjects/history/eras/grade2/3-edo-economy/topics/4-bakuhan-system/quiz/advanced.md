# 幕藩体制の仕組み 発展問題

## 発展1: 綱吉と白石の政治比較
(1) 綱吉の政治と新井白石の政治を比較し、共通点と相違点を説明しなさい。
→ 共通点：ともに儒学を重視し文治政治を行った。相違点：綱吉は生類憐みの令のような極端な政策と貨幣改鋳による経済混乱を招いたが、白石は堅実な財政再建と貨幣の質の回復に努めた。

(2) 幕府の財政悪化の構造的な原因を、石高制と貨幣経済の関係から説明しなさい。
→ 幕府・武士の収入は米の石高で固定されていたが、貨幣経済の発達で支出は貨幣で行われるようになった。米価が下落すると実質的な収入が減少し、財政が悪化した。

## 発展2: 幕藩体制の矛盾
(1) 18世紀以降、幕藩体制が揺らいでいった経済的要因を論じなさい。
→ 商品経済の発達で商人が経済力をつけた一方、武士は米価低迷で窮乏した。百姓も商品作物栽培で貨幣経済に巻き込まれ、貧富の差が拡大した。身分制度の実態と建前の乖離が進んだ。
