# 四つの窓口 大問小問

## 大問1: 四つの窓口の対応（配点 20点）
次の表の空欄を埋めなさい。

| 窓口 | 担当した藩 | 交易・外交の相手 |
|------|-----------|----------------|
| 長崎 | 幕府直轄 | （1）と（2） |
| （3） | 宗氏 | 朝鮮 |
| 薩摩 | （4） | （5） |
| （6） | 松前氏 | アイヌ |

(1) → オランダ
(2) → 中国（清）

## 大問2: 朝鮮通信使（配点 18点）
(1) 朝鮮通信使とは何か、簡潔に説明しなさい。
→ 朝鮮が日本に派遣した外交使節

(2) 朝鮮通信使が派遣された主な目的は何か。
→ 将軍の代替わりの祝賀

(3) 朝鮮通信使の人数は約何人規模だったか。
→ 約400〜500人

(4) 朝鮮通信使が日本の文化に与えた影響を1つ述べなさい。
→ 朝鮮の学問・文化・技術が日本に伝えられた（儒学や医学の交流など）

(5) 対馬藩が朝鮮との関係で果たした役割を説明しなさい。
→ 幕府と朝鮮の間の外交仲介役を務め、釜山の倭館を拠点に貿易を行った

## 大問3: 琉球とアイヌ（配点 22点）
(1) 薩摩藩が琉球に侵攻した年を答えなさい。
→ 1609年

(2) 薩摩藩の支配下でも琉球が独立国の体裁を保った理由を説明しなさい。
→ 中国（清）との朝貢貿易を続けさせることで、薩摩藩が間接的に中国との交易利益を得るため

(3) アイヌの人々が松前藩に対して起こした戦いの名前を答えなさい。
→ シャクシャインの戦い

(4) シャクシャインの戦いが起きた原因を説明しなさい。
→ 松前藩がアイヌとの交易で不当に安い価格で取引を強いたことへの不満

(5) シャクシャインの戦いの結果、アイヌの人々にどのような影響があったか。
→ 松前藩による支配がさらに強化され、交易条件はより不利になった

(6) 「四つの窓口」全体を通じて、鎖国体制下の日本の対外関係の特徴を説明しなさい。
→ 幕府が対外関係を独占的に管理し、限定された窓口を通じてのみ交流を許可することで、国内の安定と幕府の権威を維持した
