# 朱印船貿易と禁教 大問小問

## 大問1: 朱印船貿易（配点 18点）
(1) 徳川家康が海外貿易を奨励するために商人に与えた証明書を何というか。
→ 朱印状

(2) 朱印船が主に貿易を行った地域はどこか。
→ 東南アジア

(3) 東南アジア各地にできた日本人の居住地を何というか。
→ 日本町

(4) 日本町が特に多く形成された国を1つ答えなさい。
→ タイ（シャム）、ベトナム、フィリピンなどから1つ

(5) 家康が貿易を奨励した目的を説明しなさい。
→ 貿易による利益を得て幕府の財政を豊かにするため

## 大問2: 禁教から鎖国へ（配点 24点）
次の年表を見て、問いに答えなさい。
1612年→（ア）、1635年→（イ）、1637年→（ウ）、1639年→（エ）、1641年→（オ）

(1) アにあてはまる出来事を答えなさい。
→ 禁教令が出される

(2) イにあてはまる出来事を答えなさい。
→ 日本人の海外渡航・帰国の禁止

(3) ウにあてはまる出来事を答えなさい。
→ 島原・天草一揆

(4) エにあてはまる出来事を答えなさい。
→ ポルトガル船の来航禁止

(5) オにあてはまる出来事を答えなさい。
→ オランダ商館を長崎の出島に移す（鎖国の完成）

(6) ウの一揆の中心人物は誰か。
→ 天草四郎

(7) 鎖国後も貿易が許されたヨーロッパの国とアジアの国をそれぞれ答えなさい。
→ ヨーロッパ：オランダ、アジア：中国（清）

(8) 幕府が禁教政策を進めた理由を説明しなさい。
→ キリスト教の平等思想が幕府の身分制度を脅かすことを恐れ、また信者が外国勢力と結びつくことを警戒したため

## 大問3: 鎖国体制の仕組み（配点 18点）
(1) 鎖国体制のもとで、オランダ人が居住を許された場所を何というか。
→ 出島

(2) 出島はどこにあったか。
→ 長崎

(3) オランダ商館長が幕府に提出した海外情報の報告書を何というか。
→ 風説書

(4) キリシタンを発見するために行われた行事を何というか。
→ 踏み絵（絵踏）

(5) すべての住民を寺院に所属させた制度を何というか。
→ 寺請制度

(6) 鎖国体制のもとでの「四つの窓口」のうち、長崎以外の3つを答えなさい。
→ 対馬藩（朝鮮）、薩摩藩（琉球）、松前藩（アイヌ）
