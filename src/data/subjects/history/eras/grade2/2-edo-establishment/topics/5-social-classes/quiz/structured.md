# 江戸時代の身分制度 大問小問

## 大問1: 身分制度の概要（配点 20点）
(1) 江戸時代の社会は大きくどのような身分に分けられていたか。
→ 武士・百姓・町人

(2) 全人口に占める百姓の割合は約何パーセントか。
→ 約85%

(3) 武士が持っていた特権を2つ答えなさい。
→ 名字を名乗ること、刀を差すこと（帯刀）

(4) 身分制度を維持するために幕府が定めた、身分ごとの規則を何というか。
→ 諸法度（百姓には慶安の御触書など）

(5) 身分は原則として変えることができたか。
→ できなかった（世襲制）

## 大問2: 百姓の生活（配点 22点）
(1) 百姓が幕府や藩に納める税を何というか。
→ 年貢

(2) 「四公六民」とはどのような負担割合か。
→ 収穫の4割を年貢として納め、6割を自分のものとする

(3) 農村で5軒ずつ組を作り連帯責任を負わせた制度を何というか。
→ 五人組

(4) 五人組の目的を2つ答えなさい。
→ 年貢の連帯納入とキリシタンの相互監視

(5) 百姓が年貢の軽減などを求めて起こす集団行動を何というか。
→ 百姓一揆

(6) 百姓一揆の指導者が処罰されることが多かった理由を説明しなさい。
→ 幕府は秩序を乱す行為を厳しく取り締まり、一揆の首謀者は見せしめとして処刑されたため

## 大問3: 差別された身分（配点 18点）
(1) 武士・百姓・町人の身分の外に置かれた人々を何と呼んだか。
→ えた・ひにん

(2) えたの人々が担わされた仕事を答えなさい。
→ 皮革業（死んだ牛馬の処理・皮なめし）

(3) 差別された人々の住む場所はどのように制限されていたか。
→ 決められた地域に住むことを強制され、他の身分の人々との交流が制限された

(4) こうした差別は法的に正当化されていたか。
→ 幕府の制度として差別的な扱いが定められていた

(5) 差別を受けながらも、えたの人々が社会に貢献していた分野を答えなさい。
→ 皮革製品の生産（武具の材料）や治安維持の補助
