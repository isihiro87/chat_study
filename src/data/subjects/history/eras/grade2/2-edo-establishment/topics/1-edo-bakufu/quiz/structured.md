# 江戸幕府の成立 大問小問

## 大問1: 江戸幕府の成立（配点 20点）

次の文を読んで、あとの問いに答えなさい。

豊臣秀吉の死後、関東に領地を持つ（ ア ）が勢力を拡大した。1600年の（ イ ）で（ ウ ）らの大名を破り、（ エ ）年に征夷大将軍に任じられて（ オ ）を開いた。

(1) ア〜オにあてはまる語句や数字をそれぞれ答えなさい。（各2点）
→ ア：徳川家康、イ：関ヶ原の戦い、ウ：石田三成、エ：1603、オ：江戸幕府

(2) 1615年に家康が豊臣氏を滅ぼした戦いを何というか。（2点）
→ 大坂の陣。

(3) 関ヶ原の戦いで、家康が率いた軍を何と呼ぶか。（2点）
→ 東軍。

(4) 征夷大将軍に任じられることにはどのような意味があったか、簡潔に答えなさい。（4点）
→ 朝廷から全国の武士を支配する権限を認められたことを意味した。

## 大問2: 大名の種類と配置（配点 20点）

次の表を見て、あとの問いに答えなさい。

| 種類 | 特徴 |
|------|------|
| A | 徳川家の一族 |
| B | 関ヶ原以前から従っていた |
| C | 関ヶ原以後に従った |

(1) A〜Cにあてはまる大名の種類をそれぞれ答えなさい。（各2点）
→ A：親藩、B：譜代大名、C：外様大名

(2) 幕府の要職に就くことが多かった大名はA〜Cのどれか。（2点）
→ B（譜代大名）

(3) 江戸から遠い地方に配置されたのはA〜Cのどれか。また、その理由を説明しなさい。（4点）
→ C（外様大名）。関ヶ原以後に従ったため信用が低く、幕府を脅かさないよう遠方に配置された。

(4) 将軍から1万石以上の領地を与えられた武士を何と呼ぶか。（2点）
→ 大名。

(5) 将軍と大名がそれぞれの領地を支配するしくみを何というか。（2点）
→ 幕藩体制。

(6) 幕府と将軍の家臣の領地合計は約何万石で、全国のおよそ何分の1にあたるか。（2点）
→ 約700万石で、全国の約4分の1にあたる。

## 大問3: 大名統制と幕府のしくみ（配点 20点）

次の問いに答えなさい。

(1) 幕府が大名を統制するために定めた法律を何というか。（2点）
→ 武家諸法度。

(2) 武家諸法度で禁じられた事項を2つ答えなさい。（4点）
→ 城の無断修築、大名同士の無断婚姻。

(3) 大名が1年おきに江戸と領地を往復する制度は何か。（2点）
→ 参勤交代。

(4) 参勤交代を制度化した将軍は第何代のだれか。（2点）
→ 第3代将軍・徳川家光。

(5) 参勤交代が大名の力を弱めた理由を説明しなさい。（4点）
→ 大名行列を組んでの往復に莫大な費用がかかり、経済力を弱めて反乱を防ぐ効果があったため。

(6) 天皇や公家の行動を制限した法律を何というか。（2点）
→ 禁中並公家諸法度。

(7) 幕府の最高職を何というか。（2点）
→ 老中。

(8) 将軍直属の家臣で将軍に会えるものを何というか。（2点）
→ 旗本。

## 大問4: 幕府の直轄地と権威（配点 15点）

次の問いに答えなさい。

(1) 幕府が直接支配した土地を何と呼ぶか。（2点）
→ 幕領（天領）。

(2) 幕府が直接支配したものを2つ答えなさい。（4点）
→ 主要な鉱山、主要都市（京都・大阪・長崎など）。

(3) 家康をまつる日光の建物は何か。（2点）
→ 日光東照宮。

(4) 将軍直属で1万石未満の家臣のうち、将軍に会えない者を何というか。（2点）
→ 御家人。

(5) 江戸幕府の支配体制が約何年続いたか。（2点）
→ 約260年。
