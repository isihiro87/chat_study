# キリスト教とイスラム教の歴史 大問小問

## 大問1: ローマ帝国の分裂とキリスト教（配点 18点）

4世紀にローマ帝国が東西に分裂し、キリスト教の世界にも大きな変化が訪れた。以下の問いに答えなさい。

(1) 5世紀に滅亡したのは東西どちらのローマ帝国か。
→ 西ローマ帝国

(2) 西ヨーロッパのキリスト教はどのような指導者を中心にまとまったか。
→ ローマ教皇

(3) ローマ教皇を中心とするキリスト教の教会を何というか。
→ カトリック教会

(4) 東ローマ帝国と結びついたキリスト教の教会を何というか。
→ 正教会

(5) 15世紀にビザンツ帝国（東ローマ帝国）を滅ぼしたイスラム教の大国は何か。
→ オスマン帝国

## 大問2: イスラム教の成立と十字軍（配点 20点）

7世紀にアラビア半島で新しい宗教が誕生し、急速に勢力を拡大した。以下の問いに答えなさい。

(1) アラビア半島でイスラム教を開いた人物の名前を答えなさい。
→ ムハンマド

(2) イスラム勢力が8世紀に進出したヨーロッパの半島はどこか。
→ イベリア半島

(3) 11世紀にイスラム勢力が征服した聖地の名前を答えなさい。
→ エルサレム

(4) ローマ教皇が聖地を奪還するために呼びかけた遠征を何というか。
→ 十字軍

(5) 十字軍の遠征は最終的にどのような結果になったか、簡潔に答えなさい。
→ 最終的に失敗に終わった

(6) 16世紀にインドに成立したイスラム教の帝国の名前を答えなさい。
→ ムガル帝国

## 大問3: 宗教改革とその影響（配点 22点）

16世紀のヨーロッパでは、カトリック教会に対する批判から大きな宗教的変革が起こった。以下の問いに答えなさい。

(1) カトリック教会が資金集めのために販売し、批判を受けたものは何か。
→ 免罪符（贖宥状）

(2) 免罪符の販売を批判し、宗教改革を始めたドイツの人物は誰か。
→ ルター

(3) ルターの宗教改革を支持したスイスの改革者の名前を答えなさい。
→ カルバン

(4) ローマ教皇に抗議した人々は何と呼ばれるようになったか。
→ プロテスタント（抗議する者）

(5) カトリック教会が改革に対抗するために設立した修道会の名前を答えなさい。
→ イエズス会

(6) イエズス会が海外で行った活動を簡潔に説明しなさい。
→ アジアや南米に宣教師を派遣して布教活動を行った

## 大問4: ルネサンス（配点 20点）

14世紀のイタリアから始まった文化運動について、以下の問いに答えなさい。

(1) 古代ギリシャ・ローマの文化を手本に、人間の個性や自由を重視した文化運動を何というか。
→ ルネサンス（文芸復興）

(2) ルネサンスを代表する芸術家を2人答えなさい。
→ レオナルド・ダ・ヴィンチ、ミケランジェロ

(3) ルネサンスの時代に発明され、書物の普及を助けた技術は何か。
→ 活版印刷

(4) ルネサンスの時代に改良された航海用具は何か。また、それはどのような影響を与えたか。
→ 羅針盤。遠洋航海を可能にし、大航海時代につながった。
