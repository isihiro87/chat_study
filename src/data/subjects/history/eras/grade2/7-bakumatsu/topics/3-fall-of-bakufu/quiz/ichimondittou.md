# 幕府の滅亡 一問一答

Q1. 1863年に薩摩藩とイギリスが戦った戦争を何というか。
A1. 薩英戦争
Q2. 1863-64年に長州藩が外国艦隊と戦った戦争を何というか。
A2. 下関戦争（四国艦隊下関砲撃事件）
Q3. 薩英戦争・下関戦争の結果、薩摩・長州はどのように考えを変えたか。
A3. 攘夷は不可能と悟り、開国・倒幕に方針を転換した
Q4. 1866年に薩摩藩と長州藩が結んだ同盟を何というか。
A4. 薩長同盟
Q5. 薩長同盟を仲介した土佐藩出身の人物は誰か。
A5. 坂本龍馬
Q6. 1867年に大政奉還を行った最後の将軍は誰か。
A6. 徳川慶喜
Q7. 大政奉還とは何か。
A7. 将軍が政権を天皇に返上すること
Q8. 大政奉還の後に出された、天皇中心の新政府樹立の宣言を何というか。
A8. 王政復古の大号令
Q9. 旧幕府軍と新政府軍の内戦を何というか。
A9. 戊辰戦争
Q10. 戊辰戦争はいつ起きたか。
A10. 1868-1869年
Q11. 戊辰戦争の最初の戦いはどこで起きたか。
A11. 鳥羽・伏見（京都近郊）
Q12. 最後の将軍徳川慶喜は第何代か。
A12. 第15代
Q13. 薩長同盟が成立した年を答えなさい。
A13. 1866年
Q14. 大政奉還が行われた年を答えなさい。
A14. 1867年
