# 東山文化と民衆の文化 大問小問

## 大問1: 東山文化（配点 20点）

次の文を読んで、あとの問いに答えなさい。

第（ ア ）代将軍（ イ ）は京都の東山に（ ウ ）を建てた。この建物には（ エ ）の建築様式が用いられ、現在の和室の原型となっている。この時代の簡素で深みのある文化を（ オ ）と呼ぶ。

(1) ア〜オにあてはまる語句を答えなさい。（各2点）
→ ア：8、イ：足利義政、ウ：銀閣、エ：書院造、オ：東山文化

(2) 銀閣の正式な寺の名前を答えなさい。（2点）
→ 慈照寺（じしょうじ）。

(3) 書院造に使われる建築要素を3つ挙げなさい。（3点）
→ 畳・障子・ふすま（床の間・違い棚も可）。

(4) 北山文化と東山文化の雰囲気の違いを説明しなさい。（3点）
→ 北山文化は金閣に象徴される華やかで豪華な文化、東山文化は銀閣に象徴される簡素で落ち着いた深みのある文化。

(5) 東山文化に禅宗が与えた影響を説明しなさい。（2点）
→ 禅宗の「無駄を省き本質を追求する」精神が、書院造や枯山水、水墨画などの簡素で深みのある芸術を生み出した。

## 大問2: 芸術と庭園（配点 20点）

次の問いに答えなさい。

(1) 墨一色で自然を表現する絵画を何というか。（2点）
→ 水墨画。

(2) 水墨画を大成した室町時代の画家は誰か。（2点）
→ 雪舟。

(3) 雪舟の代表作を答えなさい。（2点）
→ 「秋冬山水図」。

(4) 雪舟が水墨画の技法を学ぶために渡った国はどこか。（2点）
→ 明（中国）。

(5) 水を使わず石や砂で山水の風景を表現する庭園様式を何というか。（2点）
→ 枯山水。

(6) 枯山水の代表的な庭園がある寺の名前を答えなさい。（2点）
→ 龍安寺（りょうあんじ）。

(7) 室町時代に始まった、わびの精神を大切にする茶の文化を何というか。（2点）
→ 茶の湯（侘び茶）。

(8) 室町時代に始まった、花を器に生けて飾る芸術は何か。（2点）
→ 生け花。

(9) 水墨画と枯山水に共通する芸術的な特徴を、「禅宗」の語句を使って説明しなさい。（4点）
→ どちらも禅宗の影響を受けており、限られた素材（墨だけ、石と砂だけ）で自然の本質を表現しようとする点が共通している。余分なものを省き、精神性を重視する禅宗の教えが反映されている。

## 大問3: 民衆の文化と学問（配点 20点）

次の問いに答えなさい。

(1) 室町時代に庶民の間で流行した、挿絵入りの物語本を何というか。（2点）
→ 御伽草子。

(2) 御伽草子の代表的な作品名を1つ答えなさい。（2点）
→ 「一寸法師」（ほかに「浦島太郎」「ものぐさ太郎」など）。

(3) 御伽草子が庶民に広く親しまれた理由を説明しなさい。（4点）
→ 挿絵が付いていて分かりやすく、文字を読めない人にも内容が伝わった。庶民の生活や夢を題材にした親しみやすい内容であった。

(4) 栃木県にあった学問の中心を何というか。（2点）
→ 足利学校。

(5) 足利学校で主に教えられた学問は何か。（2点）
→ 儒学。

(6) 東山文化から現代に受け継がれている文化を3つ挙げ、それぞれの現代での名称を答えなさい。（4点）
→ 書院造→和室、茶の湯→茶道、生け花→華道。

(7) 室町時代の文化が民衆にまで広がった要因を2つ述べなさい。（4点）
→ 1. 商業の発展で庶民の経済力が向上し、文化を楽しむ余裕が生まれた。2. 御伽草子のような挿絵入りの分かりやすい作品が作られ、文化へのアクセスが容易になった。
