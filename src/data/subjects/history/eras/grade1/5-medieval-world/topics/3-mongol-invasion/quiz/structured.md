# 元寇と鎌倉幕府の衰退 大問小問

## 大問1: 元寇（配点 20点）

次の文を読んで、あとの問いに答えなさい。

（ ア ）は日本に使者を送り服属を求めたが、執権（ イ ）はこれを退けた。1274年の（ ウ ）では、元と（ エ ）の連合軍が対馬・壱岐を攻め、九州北部に上陸した。元軍は（ オ ）戦法や火薬を使った武器（ カ ）で日本の武士を苦しめた。

(1) ア〜カにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：フビライ・ハン、イ：北条時宗、ウ：文永の役、エ：高麗、オ：集団、カ：てつはう

(2) 文永の役の後、幕府が博多湾に築いた防衛施設は何か。（2点）
→ 石の防壁（石塁）。

(3) 1281年の元軍の2度目の襲来を何というか。（2点）
→ 弘安の役。

(4) 弘安の役で元軍が大打撃を受けた原因を答えなさい。（2点）
→ 暴風雨が襲い、元軍の船が大きな被害を受けたため。

(5) 二度の元寇で日本の防衛を指揮した北条時宗の役職名を答えなさい。（2点）
→ 執権（しっけん）。

## 大問2: 鎌倉幕府の衰退（配点 20点）

次の問いに答えなさい。

(1) 元寇後、幕府が御家人に十分な恩賞を与えられなかった理由を説明しなさい。（4点）
→ 元寇は防衛戦であり、新しい領地を獲得できなかったため、御家人に分け与える恩賞がなかった。

(2) 御家人の領地が減少していった原因となった相続の方法を何というか。（2点）
→ 分割相続。

(3) 1297年に幕府が出した、御家人の借金を帳消しにする法令を何というか。（2点）
→ 永仁の徳政令。

(4) 徳政令の効果が一時的だった理由を説明しなさい。（4点）
→ 借金を帳消しにしたことで、以後金貸しが御家人に金を貸さなくなり、かえって経済が混乱して御家人の困窮が解消されなかった。

(5) 鎌倉時代末期に幕府に従わず武力を行使した者たちを何と呼ぶか。（2点）
→ 悪党。

(6) 元寇の様子や恩賞を求める御家人の姿を描いた絵巻の名前を答えなさい。（3点）
→ 蒙古襲来絵詞（もうこしゅうらいえことば）。

(7) 元寇から幕府滅亡までの流れを「恩賞」「不満」の語句を用いて説明しなさい。（3点）
→ 元寇は防衛戦で新しい領地がなく十分な恩賞が出せなかったため、戦いで疲弊した御家人の不満が高まり、幕府の支配力が弱まっていった。

## 大問3: 鎌倉幕府の滅亡（配点 20点）

次の文を読んで、あとの問いに答えなさい。

朝廷に政治の実権を取り戻そうとした（ ア ）は倒幕を企て、各地の武士に呼びかけた。（ イ ）は京都の（ ウ ）を攻め落とし、（ エ ）は鎌倉を攻めた。こうして（ オ ）年に鎌倉幕府は滅亡した。

(1) ア〜オにあてはまる語句や数字を答えなさい。（各2点）
→ ア：後醍醐天皇、イ：足利尊氏、ウ：六波羅探題、エ：新田義貞、オ：1333

(2) 六波羅探題とはどのような機関か説明しなさい。（3点）
→ 鎌倉幕府が京都に置いた機関で、朝廷の監視と西国の統治を担当した。

(3) 鎌倉幕府が倒された背景にある北条氏の問題を説明しなさい。（3点）
→ 北条氏に権力が集中し、他の有力御家人が政治に参加できないことへの不満が高まっていた。

(4) 鎌倉幕府滅亡後、後醍醐天皇が始めた政治を何というか。（2点）
→ 建武の新政。

(5) 建武の新政が短期間で崩壊した理由を説明しなさい。（3点）
→ 公家中心の政治で武士の功績を正しく評価しなかったため、武士の不満が高まり足利尊氏が反旗を翻した。

## 大問4: 元寇の戦い方の比較（配点 20点）

次の表を完成させ、あとの問いに答えなさい。

【資料】元軍と日本の武士の戦い方の比較

| | 元軍 | 日本の武士 |
|---|---|---|
| 戦法 | （ ア ） | （ イ ） |
| 武器の特徴 | （ ウ ） | 弓・刀 |

(1) ア〜ウにあてはまる語句を答えなさい。（各2点）
→ ア：集団戦法、イ：一騎打ち、ウ：てつはう（火薬を使った武器）

(2) 元軍の戦い方が日本の武士にとって不利だった理由を説明しなさい。（4点）
→ 日本の武士は名乗りを上げてから一騎打ちで戦う方法だったが、元軍は集団で統制された攻撃を行い、さらに火薬を使った武器（てつはう）で攻撃したため、日本の武士は経験したことのない戦い方に苦戦した。

(3) 弘安の役に備えて幕府がとった防衛策を1つ述べなさい。（3点）
→ 博多湾に石の防壁（石塁）を築き、元軍の上陸を防ぐ備えをした。

(4) 文永の役と弘安の役の年号をそれぞれ答えなさい。（各2点）
→ 文永の役：1274年、弘安の役：1281年。

(5) 二度の元寇で日本が撃退に成功した要因を2つ挙げなさい。（4点）
→ 博多湾に築いた石の防壁による防衛と、暴風雨が元軍に大打撃を与えたこと。

## 大問5: 総合問題（配点 20点）

次の年表を見て、あとの問いに答えなさい。

【資料】
- 1274年：文永の役
- 1281年：弘安の役
- 1297年：永仁の徳政令
- 1333年：鎌倉幕府滅亡
- 1334年：建武の新政

(1) 文永の役から弘安の役までの期間は何年か。（2点）
→ 7年間。

(2) 弘安の役から永仁の徳政令が出されるまでの期間は何年か。（2点）
→ 16年間。

(3) 永仁の徳政令から鎌倉幕府滅亡までの期間は何年か。（2点）
→ 36年間。

(4) 元寇（1274年）から鎌倉幕府滅亡（1333年）まで約60年かかっている。この間に幕府が弱体化していった過程を「恩賞」「分割相続」「徳政令」の3つの語句を用いて説明しなさい。（6点）
→ 元寇は防衛戦で新領地がなく恩賞を与えられなかった。また分割相続のくり返しで御家人の領地は減少し、生活が困窮した。幕府は徳政令を出して借金を帳消しにしたが効果は一時的で、かえって経済が混乱し、御家人の不満はますます高まった。

(5) 鎌倉幕府が滅亡した後に始まった建武の新政が崩壊した理由と、その後の展開を述べなさい。（4点）
→ 建武の新政は公家中心で武士の功績を正しく評価しなかったため、武士の不満が高まった。足利尊氏が反旗を翻して京都に新しい天皇を立て、室町幕府を開いた。

(6) 「元寇は日本の歴史の転換点であった」という意見について、その理由を鎌倉幕府の衰退と関連づけて述べなさい。（4点）
→ 元寇は日本を外敵から守ることに成功したが、その代償として御家人に恩賞を与えられなくなり、幕府と御家人の主従関係が揺らいだ。これが幕府の衰退・滅亡につながり、鎌倉時代から室町時代への転換を引き起こした。
