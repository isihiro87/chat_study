# 室町幕府の成立 大問小問

## 大問1: 建武の新政と南北朝時代（配点 20点）

次の文を読んで、あとの問いに答えなさい。

鎌倉幕府を倒した（ ア ）は天皇中心の（ イ ）を始めた。しかし（ ウ ）を優遇して武士の功績を正しく評価しなかったため、（ エ ）が反旗を翻した。（ エ ）は京都に新たな天皇を立て（ オ ）とし、（ ア ）は吉野に逃れて（ カ ）を開いた。この対立は約（ キ ）年間続いた。

(1) ア〜キにあてはまる語句や数字を答えなさい。（各2点）
→ ア：後醍醐天皇、イ：建武の新政、ウ：公家、エ：足利尊氏、オ：北朝、カ：南朝、キ：60

(2) 南北朝時代を統一した将軍はだれか。（2点）
→ 足利義満（第3代将軍）。

(3) 南北朝が統一された年を答えなさい。（2点）
→ 1392年。

(4) 吉野はどの現在のどの県にあるか。（2点）
→ 奈良県。

## 大問2: 室町幕府の成立（配点 20点）

次の問いに答えなさい。

(1) 足利尊氏が征夷大将軍に任命された年を答えなさい。（2点）
→ 1338年。

(2) 足利尊氏が開いた幕府を何と呼ぶか。（2点）
→ 室町幕府。

(3) 室町幕府の名前の由来を説明しなさい。（4点）
→ 第3代将軍足利義満が京都の室町に「花の御所」と呼ばれる豪華な邸宅を建てたことに由来する。

(4) 足利義満が朝廷で就いた最高の官職は何か。（2点）
→ 太政大臣。

(5) 足利義満が太政大臣にもなったことの意味を説明しなさい。（4点）
→ 武士の棟梁である将軍でありながら朝廷の最高位にも就くことで、武家と公家の両方の権威を一手に握ったことを意味する。

(6) 室町幕府はおよそ何年間存続したか。（2点）
→ 約230年間。

(7) 室町幕府が開かれた都市はどこか。鎌倉幕府との違いを含めて答えなさい。（4点）
→ 京都に開かれた。鎌倉幕府は鎌倉（現在の神奈川県）に開かれたが、室町幕府は京都という朝廷のある都に幕府を置いた点が異なる。

## 大問3: 室町幕府の仕組み（配点 20点）

次の表を完成させ、あとの問いに答えなさい。

| 役職・機関 | 役割 |
|---|---|
| （ ア ） | 将軍を補佐する最高職 |
| 侍所 | （ イ ） |
| （ ウ ） | 財政・事務を管理 |
| 問注所 | （ エ ） |
| （ オ ） | 関東の統治 |

(1) ア〜オにあてはまる語句を答えなさい。（各2点）
→ ア：管領、イ：軍事・警察（京都の治安維持）、ウ：政所、エ：裁判・訴訟、オ：鎌倉府

(2) 管領に就任したのはどのような家柄の武士か。（3点）
→ 細川・斯波・畠山の三家の有力な守護大名が交代で就任した。

(3) 鎌倉幕府で将軍を補佐した役職の名前を答え、管領との違いを述べなさい。（4点）
→ 執権（しっけん）。執権は北条氏が独占したが、管領は三家が交代で就任した点が異なる。

(4) 京都で金貸しを行い、幕府の財源となった業者を2つ答えなさい。（3点）
→ 土倉と酒屋。

## 大問4: 守護大名の成長（配点 20点）

次の問いに答えなさい。

(1) 守護大名とはどのような存在か説明しなさい。（4点）
→ 南北朝の動乱の中で領地を拡大し、地方の武士を家来化して、一国を支配するほどの強い権限を持つようになった守護のこと。

(2) 鎌倉時代の守護と室町時代の守護大名の権限の違いを説明しなさい。（4点）
→ 鎌倉時代の守護は主に軍事・警察の権限のみだったが、室町時代の守護大名は土地の支配や年貢の徴収など行政的な権限も持ち、一国全体を支配する存在となった。

(3) 守護が守護大名に成長できた歴史的背景を「南北朝の動乱」と関連づけて説明しなさい。（4点）
→ 南北朝の動乱において、守護は自らの軍事力で勢力を拡大し、戦乱の中で地方の武士を家来として組織化した。約60年にわたる混乱の中で各地の守護が独自の力を蓄え、守護大名へと成長した。

(4) 守護大名の存在が室町幕府にとってどのような影響を持ったか、プラス面とマイナス面を述べなさい。（4点）
→ プラス面は、各地の統治を担うことで幕府の地方支配を支えたこと。マイナス面は、守護大名が強大な力を持ちすぎると幕府の統制が効かなくなり、将軍の権威が弱まるおそれがあったこと。

(5) 足利義満が南北朝を統一できた理由を、「守護大名」「権威」という語句を用いて述べなさい。（4点）
→ 足利義満は有力な守護大名を抑えて将軍の権威を高め、南朝に対しても強い立場で交渉を進めることができた。こうして約60年にわたる南北朝の対立を統一することに成功した。

## 大問5: 総合問題（配点 20点）

次の年表を見て、あとの問いに答えなさい。

【資料】
- 1333年：鎌倉幕府滅亡
- 1334年：建武の新政
- 1336年頃：南北朝時代の始まり
- 1338年：足利尊氏が征夷大将軍に
- 1392年：足利義満が南北朝を統一

(1) 鎌倉幕府滅亡から建武の新政開始まで何年か。（2点）
→ 約1年。

(2) 建武の新政から南北朝時代の始まりまで何年か。（2点）
→ 約2年。

(3) 南北朝時代の始まりから統一までおよそ何年続いたか。（2点）
→ 約56年間（約60年）。

(4) 鎌倉幕府の滅亡から室町幕府の全盛期（南北朝統一）に至るまでの流れを、以下の語句をすべて用いて説明しなさい。
【後醍醐天皇、建武の新政、足利尊氏、南北朝、足利義満】（6点）
→ 鎌倉幕府を倒した後醍醐天皇は建武の新政を始めたが、公家中心の政治に武士が不満を持ち、足利尊氏が反旗を翻した。尊氏は京都に北朝を立て室町幕府を開き、後醍醐天皇は吉野に南朝を開いて南北朝時代が始まった。第3代将軍足利義満が1392年に南北朝を統一し、幕府の全盛期を築いた。

(5) 室町幕府と鎌倉幕府の違いを、「場所」「将軍の補佐役」「地方支配」の3つの観点から比較しなさい。（6点）
→ 場所：鎌倉幕府は鎌倉、室町幕府は京都に置かれた。将軍の補佐役：鎌倉幕府は執権（北条氏が独占）、室町幕府は管領（三家が交代で就任）。地方支配：鎌倉幕府は守護・地頭を配置、室町幕府では守護大名がより強い権限で各地を支配した。
