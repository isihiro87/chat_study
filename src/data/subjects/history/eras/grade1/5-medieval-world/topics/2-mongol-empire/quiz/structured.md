# モンゴル帝国 大問小問

## 大問1: モンゴル帝国の成立と拡大（配点 20点）

次の文を読んで、あとの問いに答えなさい。

13世紀の初め、モンゴル高原の（ ア ）を統一した（ イ ）がモンゴル帝国を建てた。（ イ ）は強力な（ ウ ）を率いて東西に領土を広げ、ユーラシア大陸にまたがる大帝国を築いた。その子孫の第5代皇帝（ エ ）は国号を（ オ ）と定め、（ カ ）に都を置いた。

(1) ア〜カにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：遊牧民、イ：チンギス・ハン、ウ：騎兵、エ：フビライ・ハン、オ：元、カ：大都

(2) カの都市は現在のどの都市にあたるか。（2点）
→ 北京（ペキン）。

(3) モンゴル帝国が他民族を統治する際にとった方針の特徴を説明しなさい。（3点）
→ 他民族の宗教や言語を認める柔軟な統治方針をとった。

(4) フビライ・ハンが従えた朝鮮半島の国の名前を答えなさい。（3点）
→ 高麗（こうらい）。

## 大問2: 東西交流の活発化（配点 20点）

次の問いに答えなさい。

(1) モンゴル帝国が整備し、東西交流を促進したものは何か。（2点）
→ 交通路。

(2) フビライ・ハンに仕えたイタリア人の名前を答えなさい。（2点）
→ マルコ・ポーロ。

(3) マルコ・ポーロが帰国後にまとめた書物の名前を答えなさい。（2点）
→ 「世界の記述」（東方見聞録）。

(4) その書物の中で日本はどのように紹介されたか。（3点）
→ 「黄金の国ジパング」と紹介された。

(5) モンゴル帝国の広大な領域によって、東西の交流がどのように変化したか説明しなさい。（4点）
→ 交通路が整備されたことで人・物・文化の移動が活発になり、外国の人や宗教師が帝国内を自由に行き来できるようになった。

(6) インド洋交易の主要な担い手となった人々を何と呼ぶか。（3点）
→ ムスリム商人。

(7) 13世紀末にモンゴル帝国が分裂した後、トルコで成立した国の名前を答えなさい。（4点）
→ オスマン帝国。

## 大問3: マルコ・ポーロと「世界の記述」（配点 20点）

次の資料を読んで、あとの問いに答えなさい。

【資料】イタリア人のAは、陸路で元の都に向かい、皇帝Bに仕えた。帰国後にまとめた書物の中で、日本を「C」と紹介し、ヨーロッパの人々にアジアへの興味を抱かせた。

(1) A〜Cにあてはまる語句をそれぞれ答えなさい。（各2点）
→ A：マルコ・ポーロ、B：フビライ・ハン、C：黄金の国ジパング

(2) Aが著した書物の正式な名前を答えなさい。（3点）
→ 「世界の記述」（東方見聞録）。

(3) Aは実際に日本を訪れたか。理由とともに答えなさい。（4点）
→ 実際には訪れていない。伝聞をもとに日本の様子を記述した。

(4) この書物がヨーロッパに与えた影響を説明しなさい。（4点）
→ ヨーロッパの人々にアジアへの興味を抱かせ、のちの大航海時代のきっかけの一つとなった。

(5) Bが元の都とした場所の名前と、その現在の都市名を答えなさい。（3点）
→ 大都（だいと）。現在の北京（ペキン）。

## 大問4: モンゴル帝国の影響（配点 20点）

次の問いに答えなさい。

(1) モンゴル帝国の騎兵が他国の軍を圧倒できた理由を説明しなさい。（4点）
→ 幼い頃から馬に乗り慣れた遊牧民で構成され、機動力に優れていたため。

(2) モンゴル帝国が東西交流に果たした役割を「交通路」という語を用いて説明しなさい。（4点）
→ 帝国内の交通路を整備することで、東西のヒト・モノ・文化の交流を活発にした。

(3) フビライ・ハンが高麗を従えた後、日本に対してどのような要求をしたか。（4点）
→ 使者を送り、日本にも従うよう（朝貢・貿易を）求めた。

(4) 次のア〜エの出来事を古い順に並べ替えなさい。（4点）
ア. フビライ・ハンが国号を元と定めた
イ. マルコ・ポーロが元を訪れた
ウ. チンギス・ハンがモンゴル帝国を建てた
エ. モンゴル帝国が分裂した
→ ウ → ア → イ → エ

(5) モンゴル帝国が歴史的に果たした最大の意義を、「東西」という語を用いて述べなさい。（4点）
→ ユーラシア大陸の広大な領域を統一し交通路を整備することで、東西の文化・経済の交流を飛躍的に活発にしたこと。

## 大問5: 総合問題（配点 20点）

次の地図を見て、あとの問いに答えなさい。

【資料】モンゴル帝国とその周辺（13世紀）
- モンゴル高原を中心に東西に広がる帝国の領域
- 都：大都（現在の北京）
- 朝鮮半島の高麗は元に服属

(1) モンゴル帝国が成立する以前、モンゴル高原の人々はどのような生活をしていたか。（3点）
→ 遊牧生活。家畜を連れて草原を移動しながら暮らしていた。

(2) チンギス・ハンからフビライ・ハンに至るまでに、帝国はどのように発展したか。2つの点から述べなさい。（4点）
→ 領土がユーラシア大陸の大部分にまで拡大した。フビライ・ハンが国号を元と定め、大都に都を置いて中国全土を支配する体制を整えた。

(3) 元が高麗に対して行ったことを2つ答えなさい。（4点）
→ 高麗を服属させたこと。貿易や朝貢を求めたこと。

(4) モンゴル帝国の統治方針が「柔軟」と評価される理由を説明しなさい。（4点）
→ 征服した地域の民族の宗教や言語をそのまま認め、強制的に変えようとしなかったため。

(5) 東方見聞録の内容が当時のヨーロッパ社会に与えた影響について述べなさい。（5点）
→ アジアの富や文化についての情報がヨーロッパに広まり、人々のアジアへの関心が高まった。これがやがて大航海時代において、アジアへの航路を開拓しようとする動きにつながった。
