# 室町時代の民衆の力 大問小問

## 大問1: 農民の自治と惣（配点 20点）

次の文を読んで、あとの問いに答えなさい。

室町時代には、村の有力な農民を中心に（ ア ）という自治組織が成立した。（ ア ）では（ イ ）を開いて村の運営を話し合い、村の規約である（ ウ ）を定めた。

(1) ア〜ウにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：惣（そう）、イ：寄合（よりあい）、ウ：おきて（掟）

(2) 惣が共同で管理・利用した山林や野原を何というか。（2点）
→ 入会地（いりあいち）。

(3) 惣が用水の管理を共同で行った理由を説明しなさい。（4点）
→ 農業に不可欠な水を公平に分配し、村全体の農業生産を維持するため。

(4) 惣の運営方法から、室町時代の農民がどのような力を持つようになったといえるか。（4点）
→ 領主に頼らず自分たちで村を運営する自治の力を持つようになった。寄合での合議制による意思決定は民主的な運営であった。

(5) 惣の発展が土一揆の発生と関係する理由を説明しなさい。（4点）
→ 惣によって農民の団結力が高まり、不満があるときに組織的に行動する力を身につけた。この団結力が土一揆という大規模な運動を可能にした。

## 大問2: 土一揆と民衆運動（配点 20点）

次の問いに答えなさい。

(1) 農民や馬借が団結して蜂起した運動を何というか。（2点）
→ 土一揆（つちいっき）。

(2) 1428年に起こった日本で最初の大規模な土一揆の名称を答えなさい。（2点）
→ 正長の土一揆。

(3) 土一揆で農民たちが幕府に求めたものは何か。（2点）
→ 徳政令（借金の帳消し）。

(4) 土一揆の攻撃対象となった業者を2つ挙げ、それぞれどのような業者か説明しなさい。（4点）
→ 土倉：質物を取って金を貸す金融業者。酒屋：酒造業と金融業を兼ねた業者。

(5) 正長の土一揆で農民たちが行った具体的な行動を説明しなさい。（4点）
→ 近畿地方一帯で土倉や酒屋を襲撃し、借金の証文を奪って破棄した。幕府に対して徳政令の発布を求めた。

(6) 1441年の嘉吉の土一揆が歴史的に重要である理由を説明しなさい。（4点）
→ 幕府が初めて正式に徳政令を出した一揆であり、民衆の要求が幕府の政策に直接影響を与えた点で画期的であった。

(7) 土一揆が起こった背景にある社会問題を説明しなさい。（2点）
→ 土倉や酒屋の高利貸しにより農民が多額の借金を抱え、生活が困窮していたこと。

## 大問3: 都市の自治（配点 20点）

次の問いに答えなさい。

(1) 京都で自治を担った裕福な商工業者を何というか。（2点）
→ 町衆（まちしゅう）。

(2) 町衆が復興・運営した京都の有名な祭りを答えなさい。（2点）
→ 祇園祭。

(3) 堺（大阪府）で自治を行った有力商人の組織を何というか。（2点）
→ 会合衆（えごうしゅう）。

(4) 堺が「自由都市」と呼ばれた理由を説明しなさい。（4点）
→ 有力商人が合議制で自治を行い、周囲に濠をめぐらせて外敵から防御し、大名の支配を受けずに独立した運営を行ったため。

(5) 博多（福岡県）で自治を行った組織を何というか。（2点）
→ 年行司（ねんぎょうじ）。

(6) 堺と博多に共通する特徴を、「貿易」「自治」の2つの語句を使って説明しなさい。（4点）
→ どちらも貿易で栄えた港町であり、有力商人による自治が行われた。大名の支配を受けず、独立した都市運営を行った。

(7) 室町時代の社会における農村の惣と都市の町衆の共通点を述べなさい。（4点）
→ どちらも民衆が自主的に組織を作り、領主に頼らず自治を行った点で共通している。室町時代は民衆の力が大きく成長した時代であった。
