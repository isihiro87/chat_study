# 元寇と鎌倉幕府の衰退 発展問題

## 問1: 元寇と日本の防衛

(1) 北条時宗がフビライ・ハンの要求を退けた判断について、そのリスクと意義を両面から評価しなさい。
→ リスクとしては、当時ユーラシア大陸の大部分を支配する元に逆らうことで、軍事的な報復を受ける危険があった。一方で意義としては、日本の独立と主権を守る決断であり、元に服属すれば朝貢や軍事協力を強いられ、日本の自主性が失われる可能性があった。時宗は国防の準備を整えつつ要求を拒否するという毅然とした態度をとった。

(2) 文永の役で日本の武士が苦戦した原因を、「戦い方の違い」と「武器の違い」の2つの観点から詳しく説明しなさい。
→ 戦い方の違いとして、日本の武士は名乗りを上げてから一騎打ちで戦う個人戦が基本だったが、元軍は指揮系統のもと集団で統制された攻撃を行った。武器の違いとして、日本の武士は弓と刀を主な武器としたが、元軍は「てつはう」と呼ばれる火薬を使った爆発物を用いた。日本の武士にとって初めて目にする兵器であり、爆音と炎に馬が驚いて混乱した。

(3) 弘安の役で幕府が石の防壁（石塁）を築いた判断について、文永の役の経験がどのように活かされたか考察しなさい。
→ 文永の役で元軍が博多湾に上陸して九州北部を攻撃した経験から、幕府は上陸地点を防御することが最も重要だと学んだ。石の防壁を海岸線に築くことで、元軍の船から兵士が上陸するのを物理的に阻止した。これは文永の役の反省に基づく具体的な防衛策であり、実際に弘安の役では元軍の上陸を効果的に防いだ。

(4) 二度の元寇で日本が撃退に成功した要因を、「人的要因」と「自然的要因」に分けて分析しなさい。
→ 人的要因としては、北条時宗の指導のもと全国の御家人を動員して防衛体制を整えたこと、石の防壁を築いて計画的に備えたこと、武士たちの奮闘がある。自然的要因としては、二度とも暴風雨が元軍に大きな打撃を与えたことが挙げられる。ただし自然的要因だけでなく、防壁による防御で元軍が海上に留まらざるを得なかったことが暴風雨の被害を拡大させたという見方もある。

## 問2: 鎌倉幕府の衰退と御家人の困窮

(1) 元寇が鎌倉幕府の衰退につながった因果関係を、以下の語句をすべて用いて説明しなさい。
【防衛戦、恩賞、分割相続、徳政令、不満】
→ 元寇は防衛戦であったため新しい領地を獲得できず、幕府は戦いで疲弊した御家人に十分な恩賞を与えることができなかった。もともと分割相続のくり返しで御家人の領地は減少していたが、元寇後はさらに生活が困窮した。幕府は永仁の徳政令を出して借金を帳消しにしたが効果は一時的で、かえって経済が混乱した。こうして御家人の不満が高まり、幕府の支配力は弱まっていった。

(2) 永仁の徳政令が「一時的な効果しかなかった」理由を、経済の仕組みの観点から詳しく説明しなさい。
→ 徳政令により御家人の借金は帳消しにされたが、これは借金を貸していた側（金貸しや商人）にとって大きな損失となった。その結果、金貸しは今後御家人に金を貸すことを拒否するようになった。御家人は急な出費の際に借金ができなくなり、かえって生活が困難になった。つまり徳政令は一時的に借金をなくしたが、信用制度そのものを破壊してしまい、長期的には逆効果となった。

(3) 分割相続が御家人の弱体化につながったメカニズムを、具体的な数値例を用いて説明しなさい。
→ 例えば100の広さの領地を持つ御家人に4人の子がいた場合、分割相続により各自25の領地を受け継ぐ。さらに次の世代で4人の子がいれば、一人あたり約6の領地になる。このようにわずか2世代で領地は25分の1に縮小する。領地が減れば収入も減り、武具や馬を維持する費用も捻出できなくなり、武士としての活動自体が困難になった。

## 問3: 鎌倉幕府の滅亡と建武の新政

(1) 後醍醐天皇の倒幕が成功した要因を、「幕府内部の問題」と「外部からの攻撃」の2つの面から分析しなさい。
→ 幕府内部の問題として、北条氏に権力が集中し他の有力御家人が不満を持っていたこと、元寇後の恩賞不足で御家人全体の忠誠心が低下していたことがある。外部からの攻撃として、後醍醐天皇の呼びかけに応じた足利尊氏が京都の六波羅探題を攻め落とし、新田義貞が鎌倉を直接攻撃したことで、幕府は東西から挟み撃ちにされて崩壊した。

(2) 建武の新政が失敗した原因について、「鎌倉幕府を倒す段階」と「新しい政治を行う段階」で後醍醐天皇の判断がどう変化したか比較して分析しなさい。
→ 鎌倉幕府を倒す段階では、後醍醐天皇は武士の力を積極的に利用し、足利尊氏や新田義貞らの武力に頼った。しかし新しい政治を行う段階では、公家中心の政治に切り替え、倒幕に功績があった武士の恩賞を適切に行わなかった。つまり倒幕には武士の力を借りながら、政権運営では武士を軽視するという矛盾した態度をとったことが失敗の原因である。

(3) 足利尊氏と新田義貞の役割の違いを、鎌倉幕府滅亡の過程から説明しなさい。
→ 足利尊氏は京都において六波羅探題を攻め落とし、幕府の西国における拠点を壊滅させた。一方新田義貞は東国において鎌倉を直接攻撃し、幕府の本拠地を陥落させた。二人は東西に分かれてそれぞれ幕府の重要拠点を攻略するという役割分担で、幕府を追い詰めた。

## 問4: 資料分析問題

次の資料を読んで、あとの問いに答えなさい。

【資料A】蒙古襲来絵詞の一場面
御家人の竹崎季長が、元寇での自身の戦功を幕府に訴え、恩賞を求めている場面。

【資料B】永仁の徳政令（1297年）の要旨
「御家人が質入れした土地は、代金の返済なしに取り戻すことができる。金銭の貸し借りについては裁判を行わない。」

(1) 資料Aから、元寇後の御家人がどのような状況に置かれていたか読み取りなさい。
→ 元寇で実際に戦った御家人が、自ら恩賞を求めて幕府に訴えなければならないほど、恩賞の分配が不十分だったことがわかる。防衛戦で得た領地がなく、幕府が御家人の期待に応えられていなかった。

(2) 資料Bの徳政令が、御家人にとっての「短期的な利点」と「長期的な問題点」をそれぞれ説明しなさい。
→ 短期的な利点は、借金が帳消しになり、質入れした土地も取り戻せるため、一時的に経済的な負担が軽減されたこと。長期的な問題点は、金貸しが今後の貸し出しを拒否するようになり、御家人が資金を調達できなくなったこと。また「裁判を行わない」ことで経済取引の秩序が崩れた。

(3) 資料AとBを関連づけて、元寇が鎌倉幕府の滅亡につながった過程を論述しなさい（100字程度）。
→ 元寇後、資料Aのように御家人は恩賞不足に苦しんだ。幕府は資料Bの徳政令で救済を試みたが効果は一時的で、御家人の困窮と不満は解消されなかった。こうして幕府と御家人の主従関係が崩れ、幕府滅亡への道が開かれた。（100字）

## 問5: 論述問題

(1) 「鎌倉幕府は、元寇に勝利したからこそ滅亡した」という逆説的な見方がある。この見方について、あなたの考えを根拠とともに述べなさい。
→ この見方は一定の妥当性がある。もし元寇に敗北していれば幕府は存続していた可能性は低いが、勝利したことで特有の問題が生じた。防衛戦の勝利は新領地の獲得を伴わず、幕府は御家人に恩賞を与えられなかった。鎌倉幕府の統治は「御恩と奉公」の関係に基づいていたため、恩賞が出せなくなることは制度の根幹を揺るがすものだった。つまり「勝利したが恩賞が出せない」という矛盾が、幕府と御家人の信頼関係を崩壊させた。

(2) 鎌倉幕府の滅亡から建武の新政の失敗に至る過程は、「政権交代の難しさ」を示している。どのような教訓が読み取れるか、あなたの考えを述べなさい。
→ 鎌倉幕府を倒すために武士の力を借りた後醍醐天皇が、政権を握ると公家中心の政治を行い武士を軽視したことは、協力者への配慮を欠いた政権運営の失敗例である。新しい政権を安定させるためには、旧政権を倒す際に協力した勢力の利益も適切に考慮する必要がある。武力で政権を獲得しても、それを支えた人々の期待に応えなければ政権は長続きしないという教訓が読み取れる。
