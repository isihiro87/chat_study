# 東アジアとの交流 大問小問

## 大問1: 日明貿易と倭寇（配点 20点）

次の文を読んで、あとの問いに答えなさい。

中国では（ ア ）民族が（ イ ）を建国し、モンゴル民族を北方に退けた。日本では（ ウ ）と呼ばれる海賊が朝鮮や中国の沿岸を荒らしていた。室町幕府の3代将軍（ エ ）は、正式な貿易船と（ ウ ）を区別するため（ オ ）という合い札を用いて、（ イ ）との間で日明貿易を始めた。

(1) ア〜オにあてはまる語句を答えなさい。（各2点）
→ ア：漢、イ：明、ウ：倭寇、エ：足利義満、オ：勘合

(2) 日明貿易の別名を答えなさい。（2点）
→ 勘合貿易。

(3) 勘合の仕組みを説明しなさい。（4点）
→ 一枚の札を二つに割って日本と明で半分ずつ持ち、貿易船が来た時にぴったり合えば正式な船であることを証明できる仕組み。

(4) 日明貿易で日本から輸出された品物と、明から輸入された品物をそれぞれ2つずつ答えなさい。（4点）
→ 輸出：銅・刀剣（硫黄）。輸入：銅銭・生糸（絹織物・書画）。

## 大問2: 朝鮮国（配点 20点）

次の問いに答えなさい。

(1) 1392年に高麗を滅ぼして朝鮮国を建てた人物はだれか。（2点）
→ 李成桂。

(2) 朝鮮国で作られた独自の文字の名前と正式名称を答えなさい。（3点）
→ ハングル。正式名称は訓民正音（くんみんせいおん）。

(3) 朝鮮国がハングルを作った目的を説明しなさい。（4点）
→ 漢字を読めない民衆にも学問を広めるため、民衆の教育に利用する目的で作られた。

(4) 朝鮮国と日本の関係を、「倭寇」という語を用いて説明しなさい。（4点）
→ 朝鮮国は日本に倭寇の取り締まりを求めつつ、正式な貿易関係も築いた。倭寇の被害を受けながらも、日本との交流は行われていた。

(5) 朝鮮国が建国される前に朝鮮半島にあった国の名前と、その国とモンゴル帝国の関係を述べなさい。（4点）
→ 高麗。高麗はモンゴル帝国（元）に服属し、元寇の際には元とともに日本を攻撃した。

(6) ハングルが朝鮮の歴史において重要である理由を述べなさい。（3点）
→ それまで漢字しかなかった朝鮮に独自の文字をもたらし、庶民にも読み書きが広まる基盤を作ったため。

## 大問3: 琉球王国（配点 20点）

次の問いに答えなさい。

(1) 15世紀前半に沖縄島を統一して琉球王国を建てた一族はだれか。（2点）
→ 尚氏。

(2) 琉球王国の都はどこに置かれたか。（2点）
→ 首里。

(3) 琉球王国が行い、大いに繁栄した貿易の形態を何というか。（2点）
→ 中継貿易。

(4) 中継貿易とはどのような貿易か、琉球王国の具体例を用いて説明しなさい。（4点）
→ 日本・中国（明）・東南アジアなど周辺諸国の産物を集めて、別の地域に転売する貿易のこと。琉球は各地域の間に位置する地理的利点を活かして仲介役を果たした。

(5) 琉球王国が中継貿易で繁栄できた理由を「地理的条件」の観点から説明しなさい。（4点）
→ 琉球は日本・中国・東南アジアのほぼ中間に位置しており、各地域を結ぶ海上交通の要所にあった。この地理的な利点を活かして各地の産物を効率的に仲介できた。

(6) 琉球王国が交易相手として結んでいた地域を3つ以上挙げなさい。（3点）
→ 日本、中国（明）、東南アジア、朝鮮。

(7) 首里は現在のどの県のどの市にあるか。（3点）
→ 沖縄県那覇市。

## 大問4: アイヌ民族（配点 20点）

次の問いに答えなさい。

(1) 北海道で独自の文化を営んでいた民族の名前を答えなさい。（2点）
→ アイヌ民族。

(2) アイヌ民族はどのようにして生活していたか、2つの方法を答えなさい。（4点）
→ 狩りと漁。独自の交易活動も行っていた。

(3) 15世紀にアイヌ民族の首長コシャマインが戦いを起こした原因を説明しなさい。（4点）
→ 本州からの和人が蝦夷地（北海道）に進出し、アイヌ民族に対して不公平な扱いをしたことへの反発。

(4) アイヌ民族との交易を仲立ちしていた一族の名前と、その拠点を答えなさい。（3点）
→ 安藤氏。拠点は青森県の十三湊（とさみなと）。

(5) アイヌ民族と和人の関係がどのように変化したか、「交易」と「衝突」という語を用いて説明しなさい。（4点）
→ もともとは安藤氏の仲立ちでアイヌ民族と和人は交易関係にあったが、和人の北海道への進出が進むにつれて不公平な扱いが生じ、コシャマインの戦いのような衝突が起きるようになった。

(6) 蝦夷地とは現在のどの地域を指すか答えなさい。（3点）
→ 北海道。

## 大問5: 総合問題（配点 20点）

次の表を見て、あとの問いに答えなさい。

| 地域 | 国・民族 | 特徴 |
|---|---|---|
| 中国 | 明 | 日明貿易（勘合貿易） |
| 朝鮮半島 | 朝鮮国 | ハングルの制定 |
| 沖縄 | 琉球王国 | 中継貿易 |
| 北海道 | アイヌ民族 | 独自の交易と文化 |

(1) この表の4つの地域・民族に共通する活動は何か。（3点）
→ 交易（貿易）。どの地域でも交易が活発に行われていた。

(2) 日明貿易と琉球王国の中継貿易の違いを説明しなさい。（4点）
→ 日明貿易は日本と明の二国間で勘合を用いて行う公式の貿易。中継貿易は琉球が複数の国・地域の間で産物を仲介する貿易で、特定の二国間に限らない広範な交易ネットワークだった。

(3) 15世紀の東アジアで交易が活発になった背景を、「明の建国」と「室町幕府の政策」の観点から述べなさい。（4点）
→ 明の建国により中国が安定し、国際的な貿易関係が再構築された。室町幕府の足利義満は明との正式な貿易（日明貿易）を開始し、東アジアの交易ネットワークに日本を組み込んだ。

(4) 倭寇の存在が東アジアの外交関係にどのような影響を与えたか説明しなさい。（4点）
→ 倭寇は朝鮮や中国の沿岸に被害を与え、日本との外交問題となった。明や朝鮮は日本に倭寇の取り締まりを要求し、これが日明貿易における勘合制度の導入の背景となった。倭寇問題を解決することが正式な外交・貿易関係を築くきっかけとなった。

(5) 室町時代の東アジア交流が現代の日本に残した影響について、あなたの考えを述べなさい。（5点）
→ 日明貿易で輸入された銅銭は日本国内の貨幣経済の発展を促した。朝鮮国のハングル創制のように各地で独自の文化が発展し、現在の東アジア各国の文化の基盤となった。琉球王国の中継貿易は沖縄の独自の文化を形成し、現在も沖縄の歴史的アイデンティティの重要な要素となっている。
