# 国風文化 大問小問

## 大問1: 仮名文字と文学（配点 20点）

遣唐使の中止後、（ア）をもとにして（イ）が生まれ、日本語の繊細な感情を自由に表現できるようになった。以下の問いに答えなさい。

(1) 空欄（ア）に入る文字を答えなさい。
→ 漢字

(2) 空欄（イ）に入る文字の名称を答えなさい。
→ 仮名文字

(3) 仮名文字の2種類の名称を答えなさい。
→ ひらがなとカタカナ

(4) 紫式部が書いた世界最古級の長編小説の名称を答えなさい。
→ 源氏物語

(5) 清少納言が書いた随筆の名称を答えなさい。
→ 枕草子

(6) 天皇の命令で編纂された最初の勅撰和歌集の名称と、その選者を答えなさい。
→ 古今和歌集。選者は紀貫之。

## 大問2: 源氏物語と枕草子の比較（配点 20点）

国風文化を代表する2つの文学作品について、以下の問いに答えなさい。

(1) 源氏物語の作者は誰か。
→ 紫式部

(2) 源氏物語の主人公は誰か。
→ 光源氏

(3) 枕草子の作者は誰か。
→ 清少納言

(4) 枕草子はどのような種類の文学作品か。
→ 随筆

(5) 紫式部と清少納言がそれぞれ仕えていた人物（后）を答えなさい。
→ 紫式部は藤原道長の娘・彰子に仕えた。清少納言は一条天皇の中宮・定子に仕えた。

(6) 源氏物語と枕草子が仮名文字で書かれたことの文学史上の意義を説明しなさい。
→ 漢文ではなく仮名文字で書かれたことで、日本語による繊細な感情表現や情景描写が可能になり、日本独自の文学が確立した。

## 大問3: 浄土信仰と文化（配点 20点）

平安時代後期には、来世での救いを求める信仰が広まった。以下の問いに答えなさい。

(1) 念仏を唱えて極楽浄土への往生を願う信仰を何というか。
→ 浄土信仰

(2) 浄土信仰で信仰の対象とされた仏は何か。
→ 阿弥陀如来

(3) 浄土信仰が広まった背景にある、仏教の教えが衰える時代に入るという考え方を何というか。
→ 末法思想

(4) 1053年に藤原頼通が宇治に建立した、浄土信仰を表す建物は何か。
→ 平等院鳳凰堂

(5) 平等院鳳凰堂はどのような世界を地上に表現しようとしたか。
→ 阿弥陀如来がいる極楽浄土の世界

(6) 天台宗を開いた僧と真言宗を開いた僧をそれぞれ答え、浄土信仰との違いを述べなさい。
→ 天台宗は最澄、真言宗は空海。天台宗・真言宗は山奥での厳しい修行を重視するが、浄土信仰は念仏を唱えることで極楽浄土に往生することを願う、より平易な信仰である。

## 大問4: 美術と生活文化（配点 20点）

国風文化は文学だけでなく、美術や生活様式にも独自の特色が見られた。以下の問いに答えなさい。

(1) 日本の風景や貴族の生活を描いた日本独自の絵画を何というか。
→ 大和絵

(2) 大和絵が中国の絵画（唐絵）と異なる特徴を述べなさい。
→ 中国の山水画のように中国の風景を描くのではなく、日本の四季の風景や貴族の日常生活を繊細な色彩で描いた。

(3) 平安貴族の女性が着用した正装を何というか。
→ 十二単

(4) 平安貴族の住居の建築様式を何というか。
→ 寝殿造

(5) 寝殿造の特徴を3つ挙げなさい。
→ 中央に寝殿を配置。渡り廊下で対屋を結ぶ。庭に池や築山を設けた庭園がある。

(6) 国風文化の美術・建築・衣服などに共通する特徴を述べなさい。
→ 唐の文化をもとにしながらも、日本の風土や感性に合わせて独自に発展させた、繊細で優雅な美意識が特徴である。

## 大問5: 国風文化の総合理解（配点 20点）

次の文化要素を、飛鳥文化・天平文化・国風文化のいずれに分類しなさい。また、あとの問いに答えなさい。

ア：法隆寺　イ：源氏物語　ウ：正倉院　エ：仮名文字　オ：万葉集　カ：大和絵　キ：釈迦三尊像

(1) 飛鳥文化に分類されるものをすべて選びなさい。
→ ア、キ

(2) 天平文化に分類されるものをすべて選びなさい。
→ ウ、オ

(3) 国風文化に分類されるものをすべて選びなさい。
→ イ、エ、カ

(4) 飛鳥文化→天平文化→国風文化という変遷を、外国文化との関係性の変化に着目して説明しなさい。
→ 飛鳥文化は朝鮮半島や中国南北朝の文化を直接受容した時期、天平文化は唐の文化を積極的に吸収した時期、国風文化は遣唐使の中止後にそれまで吸収した文化を消化して日本独自に発展させた時期である。

(5) 国風文化が「日本文化の基礎」とされる理由を、仮名文字・文学・美術のいずれかに触れて説明しなさい。
→ 仮名文字の発明により日本語での文学表現が可能になり、源氏物語や枕草子のような作品が生まれた。この日本語による文学の伝統は現代まで続いており、国風文化は日本文化の基礎を築いたといえる。
