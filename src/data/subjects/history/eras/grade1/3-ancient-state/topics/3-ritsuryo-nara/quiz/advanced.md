# 律令国家と奈良時代 発展問題

## 問1: 律令制度の仕組みと唐との比較

大宝律令は唐の律令制度を手本にして作られた。以下の問いに答えなさい。

(1) 唐の律令制度を手本にしたにもかかわらず、日本の律令制度には独自の要素がある。神祇官がその一つであるが、唐にはない神祇官を日本が設けた理由を説明しなさい。
→ 日本には天皇を中心とした神道の祭祀が古くからあり、仏教だけでなく日本固有の神々への祭祀も国家の重要な役割であったため、唐にはない神祇官を設けた。

(2) 国司と郡司の任命方法の違いが、地方統治にどのような影響を与えたか説明しなさい。
→ 国司は中央から派遣される役人であるため天皇の意向を直接地方に伝えることができたが、土地の事情に疎い面があった。郡司は地元の豪族が任命されるため地域の実情をよく知っていたが、中央の意向と地元の利害が対立することもあった。

(3) 律令制のもとで戸籍が6年ごとに作成された理由を、班田収授法と関連づけて説明しなさい。
→ 班田収授法では6歳以上の人々に口分田を与え、死後に返還させる仕組みであったため、人口の変動を正確に把握して土地の配分を更新する必要があった。

(4) 日本が唐の律令を完全にそのまま導入せず、修正を加えた理由を考察しなさい。
→ 唐は巨大な大陸国家で日本とは地理・規模・社会構造が大きく異なるため、そのまま導入すると実情に合わない面があった。日本の豪族社会や神道の伝統などを考慮して修正を加える必要があった。

## 問2: 平城京の都市計画

平城京は唐の長安をモデルにして計画的に造られた都市である。以下の問いに答えなさい。

(1) 平城京が碁盤の目のような区画に整備されたことのメリットを2つ挙げなさい。
→ 行政管理がしやすく、住民の把握や課税が容易になった。道路が整然としており、交通の便がよく物資の運搬にも便利であった。

(2) 平城京に東市と西市が置かれた理由を、律令国家の経済政策と関連づけて説明しなさい。
→ 国家が商業活動を管理・統制するため、決められた場所に市を設置し、和同開珎などの貨幣を流通させて経済活動を活性化させようとした。

(3) 和同開珎を広めるために蓄銭叙位令が出されたことから、当時の貨幣流通の状況について何がわかるか。
→ 法律で位を与えるほどの奨励をしなければ貨幣が流通しなかったことから、当時の人々はまだ物々交換が中心で、貨幣経済が十分に浸透していなかったことがわかる。

(4) 平城京が約70年間しか都として機能しなかった理由を考察しなさい。
→ 奈良の大寺院が政治に介入するようになったこと、人口増加による生活環境の悪化、疫病の流行などが挙げられる。桓武天皇はこれらの問題を解決するために長岡京、さらに平安京への遷都を行った。

## 問3: 律令国家における身分制度

律令国家には明確な身分制度が存在した。以下の問いに答えなさい。

(1) 律令制のもとでの身分を大きく2つに分け、それぞれの名称を答えなさい。
→ 良民と賤民

(2) 貴族と一般の農民の生活の違いを、税の負担という観点から説明しなさい。
→ 貴族は官職について給与や特権を受け、税の負担が免除または軽減されていた。一方、農民は租・調・庸などの重い税や防人などの兵役を課せられていた。

(3) 奴婢が売買の対象となっていたことは、律令国家の「公地・公民」の理念とどのように矛盾するか説明しなさい。
→ 公地・公民は全ての土地と人民が国家のものであるという理念だが、奴婢は個人の所有物として売買されており、人民を国家が直接管理するという原則に反していた。

## 問4: 遣唐使と国際交流

律令国家は遣唐使を通じて唐との交流を続けた。以下の問いに答えなさい。

(1) 遣唐使の航海が危険であった理由を、当時の航海技術と関連づけて説明しなさい。
→ 当時の船は木造で小さく、航海技術も未熟であったため、東シナ海の荒波や台風に耐えられず、難破する船が多かった。また、より安全な朝鮮半島沿いの航路が新羅との関係悪化で使えなくなり、危険な南路を取らざるを得なかったため。

(2) 遣唐使が日本にもたらした文化や制度を3つ以上挙げなさい。
→ 律令制度、仏教の新しい教え、漢詩・書道、医学・暦学の知識、建築技術、陶磁器の製法など。

(3) 唐の長安には世界各地からの商人や使者が集まっていた。このことが日本にもたらした間接的な影響を説明しなさい。
→ 遣唐使は唐を通じてシルクロード経由の西アジアやインドの文化にも触れることができ、正倉院の宝物に見られるような国際色豊かな文化が日本にもたらされた。

(4) 律令国家の建設において、遣唐使と渡来人はそれぞれどのような役割を果たしたか比較して説明しなさい。
→ 遣唐使は日本から唐に赴き最新の制度・文化・学問を直接学んで持ち帰る役割を果たした。渡来人は大陸から日本に定住し、技術者・学者として日常的に技術や知識を伝える役割を果たした。
