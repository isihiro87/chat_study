# 大化の改新 大問小問

## 大問1: 乙巳の変と大化の改新（配点 20点）

645年、（ア）皇子と（イ）は、朝廷で権力をほしいままにしていた蘇我入鹿を暗殺し、政治改革を始めた。この改革を（ウ）という。以下の問いに答えなさい。

(1) 空欄（ア）に入る人物名を答えなさい。
→ 中大兄

(2) 空欄（イ）に入る人物名を答えなさい。
→ 中臣鎌足

(3) 空欄（ウ）に入る政治改革の名称を答えなさい。
→ 大化の改新

(4) この改革で定められた、土地と人民を国家のものとする方針を何というか。
→ 公地・公民

(5) 「大化」は日本で初めて定められたあるものである。それは何か。
→ 年号（元号）

(6) 中大兄皇子がのちに即位して名乗った天皇名を答えなさい。
→ 天智天皇

## 大問2: 白村江の戦いとその影響（配点 20点）

663年、日本は朝鮮半島の百済を助けるため軍を送ったが、唐・新羅の連合軍に大敗した。以下の問いに答えなさい。

(1) この戦いの名称を答えなさい。
→ 白村江の戦い

(2) 日本が助けようとした国はどこか。
→ 百済

(3) 日本を破った連合軍を構成していた2つの国を答えなさい。
→ 唐と新羅

(4) 敗戦後に九州に築かれた防衛施設の名称を答えなさい。
→ 水城

(5) 敗戦後に設置された、九州の外交と防衛を担った役所を何というか。
→ 大宰府

(6) 白村江の戦いの敗北が日本の国づくりに与えた影響を説明しなさい。
→ 外国からの攻撃に備えて国防を強化するとともに、唐にならった律令制度の整備を急ぐきっかけとなった。

## 大問3: 壬申の乱と天武天皇（配点 20点）

天智天皇の死後、672年に皇位継承をめぐる内乱が起きた。以下の問いに答えなさい。

(1) この内乱の名称を答えなさい。
→ 壬申の乱

(2) この乱で勝利した人物は誰か。
→ 大海人皇子

(3) (2)の人物が即位して名乗った天皇名を答えなさい。
→ 天武天皇

(4) この乱で敗れた大友皇子は、天智天皇とどのような関係だったか。
→ 天智天皇の子（息子）

(5) 天武天皇の政治の特徴を簡潔に説明しなさい。
→ 天皇自らが強い権力を握り、豪族を抑えて天皇中心の政治を推進した。

## 大問4: 藤原京と持統天皇（配点 20点）

天武天皇の死後、その皇后が即位し、日本初の本格的な都城が造られた。以下の問いに答えなさい。

(1) 天武天皇の死後に即位した天皇は誰か。
→ 持統天皇

(2) 694年に造営された日本初の本格的な都城の名称を答えなさい。
→ 藤原京

(3) 藤原京はどの国の都をモデルにして造られたか。
→ 唐（長安）

(4) 藤原京が置かれた場所は現在のどの都道府県か。
→ 奈良県

(5) 持統天皇と天武天皇の関係を答えなさい。
→ 持統天皇は天武天皇の皇后（妻）

## 大問5: 大化の改新から律令国家への歩み（配点 20点）

次のできごとを年代順に並べ替え、あとの問いに答えなさい。
ア：壬申の乱　イ：大化の改新　ウ：白村江の戦い　エ：藤原京の造営

(1) 年代順に並べ替えなさい。
→ イ → ウ → ア → エ

(2) イの年を答えなさい。
→ 645年

(3) ウの年を答えなさい。
→ 663年

(4) アの年を答えなさい。
→ 672年

(5) エの年を答えなさい。
→ 694年

(6) これらの出来事の流れが最終的に目指した国家体制とは何か。「律令」という語を用いて答えなさい。
→ 唐にならった律令に基づく天皇中心の中央集権国家（律令国家）。
