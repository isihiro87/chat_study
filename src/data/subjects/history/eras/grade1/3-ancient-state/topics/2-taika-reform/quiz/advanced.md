# 大化の改新 発展問題

## 問1: 大化の改新の意義と限界

大化の改新は天皇中心の国づくりを目指した画期的な改革であった。以下の問いに答えなさい。

(1) 公地・公民の制度は、それまでの豪族の支配とどのように異なるか。「土地」「人民」「国家」の語を用いて説明しなさい。
→ それまでは豪族が土地と人民を私有し支配していたが、公地・公民では土地と人民はすべて国家（天皇）のものとされ、豪族の私的な支配を否定した。

(2) 大化の改新は一度の政変で完成したものではない。改革が完成するまでにどれくらいの期間を要したか、大宝律令の完成（701年）までの年数を求めなさい。
→ 約56年（645年〜701年）

(3) 中臣鎌足が藤原姓を賜ったことは、のちの日本の政治にどのような影響を与えたか。
→ 藤原氏の祖となり、平安時代には藤原氏が摂関政治を行い、天皇の外戚として長期にわたり政治の実権を握ることとなった。

(4) 大化の改新が中国の制度を手本としたことのメリットとデメリットをそれぞれ説明しなさい。
→ メリット：唐の律令制度は高度に整備されており、効率的な中央集権体制を短期間で導入できた。デメリット：中国と日本では社会の仕組みが異なるため、公地・公民制度が日本の実情に完全には合わず、のちに崩壊していった。

## 問2: 白村江の戦いの影響

白村江の戦いは、日本の国家形成に大きな転機をもたらした。以下の問いに答えなさい。

(1) 日本が百済を支援しようとした理由を、東アジアの国際関係をふまえて説明しなさい。
→ 百済は古くから日本と友好関係にあり、文化や技術の交流が深かった。百済が唐・新羅に滅ぼされれば、朝鮮半島における日本の影響力が失われ、さらに唐・新羅が日本に攻めてくる危険性もあったため。

(2) 白村江の敗戦後、天智天皇が都を近江大津宮に移した理由を推測しなさい。
→ 九州に近い飛鳥よりも内陸の近江のほうが唐・新羅の攻撃から安全であり、また琵琶湖の水運を利用できる地理的優位性もあったため。

(3) 白村江の戦い以後、朝鮮半島を統一した新羅と日本の関係はどう変化したか。
→ 新羅が朝鮮半島を統一したことで百済・高句麗との関係は断たれ、新羅との間でも緊張関係が続いた。一方で、新羅を通じた文化交流は一定程度続いた。

(4) 白村江の敗戦から藤原京の造営までの約30年間に起きた主な出来事を3つ挙げ、それぞれが律令国家の建設にどう貢献したか説明しなさい。
→ (1)国防の強化（水城・大宰府の設置）→外敵からの防衛体制の確立。(2)壬申の乱（672年）→天武天皇による強力な天皇中心の政治の推進。(3)藤原京の造営（694年）→唐の長安にならった本格的な都城の完成で、律令国家としての体裁が整った。

## 問3: 壬申の乱と天皇権力の強化

壬申の乱は古代日本最大の内乱であり、勝利した天武天皇の政治は大きな転換点となった。以下の問いに答えなさい。

(1) 壬申の乱が起きた背景を、天智天皇の皇位継承問題と関連づけて説明しなさい。
→ 天智天皇は弟の大海人皇子ではなく、自分の子である大友皇子に皇位を継がせようとした。大海人皇子は一度出家して身を引いたが、天智天皇の死後に挙兵して大友皇子を破った。

(2) 天武天皇は豪族の力を抑えるためにどのような政策を行ったか。2つ以上挙げなさい。
→ 八色の姓を制定して身分秩序を再編した。豪族の私有民を制限した。律令の編纂を進めた。皇族を重要な地位に登用した。

(3) 天武天皇が歴史書の編纂を命じた目的は何か。
→ 天皇家の正統性を示し、天皇を中心とした国家の歴史的根拠を確立するため。

(4) 壬申の乱で天武天皇が勝利できた理由として、地方豪族や農民の支持を得たことが挙げられる。なぜ彼らは大海人皇子を支持したのか推測しなさい。
→ 天智天皇の政治が重い負担を民衆に課していたことへの不満があり、大海人皇子に新しい政治への期待があった。また大海人皇子が東国の豪族と結びつきを持っていたことも大きい。

## 問4: 大化の改新から律令国家への流れ

次の年表を見て、あとの問いに答えなさい。

| 年 | できごと |
|:-:|:-:|
| 645年 | 大化の改新 |
| 663年 | 白村江の戦い |
| 672年 | 壬申の乱 |
| 694年 | 藤原京の造営 |
| 701年 | 大宝律令の完成 |
| 710年 | 平城京への遷都 |

(1) 645年から701年までの約56年間に一貫して目指された国家体制は何か。
→ 天皇を中心とした律令国家

(2) 白村江の戦いの敗北が、律令国家の建設を加速させた理由を説明しなさい。
→ 外国（唐・新羅）からの脅威が現実のものとなり、国防と国内体制の整備が急務となったため。唐の強さを実感したことで、唐にならった律令制度を早急に導入する動きが加速した。

(3) 藤原京から平城京への遷都までわずか16年しかない理由として考えられることを述べなさい。
→ 藤原京は周囲を山に囲まれた狭い土地にあり、人口増加や政治機能の拡大に対応しきれなかったため。また大宝律令による本格的な律令国家にふさわしい、より大規模な都が必要になったため。

(4) この年表の流れ全体を通して、天皇の権力がどのように変化したか説明しなさい。
→ 大化の改新で天皇中心の国づくりが始まり、白村江の敗戦で国の危機感が高まった。壬申の乱で天武天皇が強力な権力を握り、藤原京の造営と大宝律令の完成によって天皇を頂点とする律令国家体制が確立した。
