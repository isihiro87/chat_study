# 聖徳太子と飛鳥時代 発展問題

## 問1: 聖徳太子の国づくり構想

聖徳太子は、冠位十二階・十七条の憲法・遣隋使の派遣という3つの政策を行った。以下の問いに答えなさい。

(1) 冠位十二階と十七条の憲法の共通する目的を、「豪族」「天皇」という語を用いて説明しなさい。
→ 豪族の力を抑え、天皇を中心とした中央集権的な政治体制を確立すること。

(2) 冠位十二階は主に豪族のどのような特権を弱めることを狙ったものか。具体的に説明しなさい。
→ 豪族が家柄によって自動的に高い地位を得る特権を弱め、才能や功績のある者を登用できるようにした。

(3) 遣隋使の派遣が国内の政治改革にとってどのような意味を持ったか説明しなさい。
→ 隋の進んだ律令制度や政治の仕組みを学び、日本の天皇中心の国づくりに活かすことができた。

(4) 聖徳太子が「日出づる処の天子」と称した意図を、当時の東アジアの国際秩序をふまえて説明しなさい。
→ 当時、隋は周辺国を臣下として扱う冊封体制をとっていたが、聖徳太子は日本の天皇を隋の皇帝と対等な立場に位置づけ、独立した国家としての立場を主張しようとした。

## 問2: 蘇我氏と仏教受容

6世紀に百済から仏教が伝わった際、朝廷内で受け入れをめぐる対立が起きた。以下の問いに答えなさい。

(1) 仏教の受け入れに賛成した豪族と反対した豪族をそれぞれ答えなさい。
→ 賛成：蘇我氏、反対：物部氏

(2) 物部氏が仏教の受け入れに反対した理由を推測して説明しなさい。
→ 日本古来の神々を祀る立場にあった物部氏は、外来の仏教が広まれば自分たちの宗教的な権威が弱まると考えたため。

(3) 蘇我氏が仏教を支持した背景には、政治的な利益もあったと考えられる。どのような利益があったか説明しなさい。
→ 仏教という新しい文化を取り入れることで他の豪族に対する優位性を確保し、渡来人とのつながりを強めて政治的な力を拡大できた。

(4) 聖徳太子と蘇我馬子の協力関係が飛鳥文化に与えた影響を説明しなさい。
→ 聖徳太子が仏教を政治理念に取り入れ、蘇我馬子が豊富な財力で寺院建立を支援したことで、法隆寺や飛鳥寺など仏教中心の飛鳥文化が花開いた。

## 問3: 飛鳥文化の国際性

飛鳥文化は日本初の仏教文化であるが、その特徴には国際的な要素が多く含まれている。以下の問いに答えなさい。

(1) 飛鳥文化が影響を受けた地域を2つ以上挙げなさい。
→ 中国（隋・南北朝）、朝鮮半島（百済・高句麗）、さらにインドや西アジアの影響も間接的に受けている。

(2) 法隆寺の釈迦三尊像の表情には、中国南北朝時代の仏像の特徴が見られる。飛鳥文化がこのような特徴をもつ理由を説明しなさい。
→ 朝鮮半島や中国から渡来した人々（渡来人）が仏像の制作に携わり、大陸の様式を直接伝えたため。

(3) 飛鳥文化と後の天平文化を比較したとき、飛鳥文化の特徴として正しいものを選びなさい。
ア：唐の文化の影響が強い イ：朝鮮半島や中国南北朝の影響が強い ウ：日本独自の文化が中心
→ イ

(4) 飛鳥文化の発展において、渡来人が果たした役割を具体的に説明しなさい。
→ 渡来人は仏教の経典や仏像の制作技術、建築技術、暦法、漢字の知識などを日本に伝え、飛鳥文化の発展を技術面から支えた。

## 問4: 十七条の憲法の内容分析

十七条の憲法の条文の一部を読み、あとの問いに答えなさい。

第一条「和を以て貴しとなす」
第二条「篤く三宝を敬へ」
第三条「詔を承りては必ず謹め」
第十二条「国に二君なし、民に両主なし」

(1) 第一条から第三条までに示された考え方を、それぞれ要約しなさい。
→ 第一条：人々が協調し和を大切にすること。第二条：仏教（三宝）を敬うこと。第三条：天皇の命令に従うこと。

(2) 第十二条と第三条に共通する政治思想は何か。
→ 天皇を唯一の君主とし、天皇の権威のもとに国を統治するという考え方。

(3) 十七条の憲法は現代の「憲法」とは性格が異なる。どのように異なるか説明しなさい。
→ 現代の憲法は国家権力を制限し国民の権利を保障する最高法規であるが、十七条の憲法は役人の心構えや道徳的な規範を示したもので、法的な拘束力を持つ法律ではなかった。

(4) 聖徳太子がこの法を定めた歴史的背景を、当時の政治状況をふまえて説明しなさい。
→ 蘇我氏をはじめとする有力豪族が大きな力を持ち、天皇の権威が十分に確立されていなかった。聖徳太子は豪族たちに協調と天皇への服従を求め、天皇中心の統一的な政治体制を築こうとした。

## 問5: 遣隋使と東アジア外交

次の資料は、聖徳太子が隋の煬帝に送った国書の一部である。あとの問いに答えなさい。

「日出づる処の天子、書を日没する処の天子に致す。恙なきや。」

(1) この国書を届けた人物は誰か。
→ 小野妹子

(2) 「日出づる処の天子」は誰を指しているか。
→ 日本の天皇（推古天皇）

(3) 「日没する処の天子」は誰を指しているか。
→ 隋の煬帝

(4) 隋の煬帝はこの国書を読んで怒ったとされる。その理由を説明しなさい。
→ 隋の皇帝は自らを天下唯一の天子と考えていたが、日本が自国の天皇を「天子」と称して隋と対等の立場を主張したため、煬帝はこれを無礼と感じた。
