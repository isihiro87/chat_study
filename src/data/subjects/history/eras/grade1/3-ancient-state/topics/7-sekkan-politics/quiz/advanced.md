# 摂関政治 発展問題

## 問1: 摂関政治の権力構造

藤原氏は約200年にわたって摂関政治を行った。以下の問いに答えなさい。

(1) 藤原氏が権力を維持するために用いた「外戚」という仕組みの長所と短所をそれぞれ説明しなさい。
→ 長所：天皇の母方の親族として正統な権威を保ちながら実権を握れた。武力に頼らず平和的に権力を維持できた。短所：娘がいない場合や天皇との間に男子が生まれない場合は外戚の立場を維持できず、権力基盤が不安定になった。

(2) 摂関政治が「天皇を廃位させる」のではなく「天皇を利用する」という方法をとった理由を考察しなさい。
→ 天皇は日本の政治体制の頂点であり、神話に由来する神聖な存在とされていた。天皇を廃して自ら王位につくことは正統性を失い、他の貴族や民衆の支持を得られなかった。天皇の権威を利用しながら実質的な権力を握ることが最も効果的であった。

(3) 藤原道長が「この世をば…」と詠んだ時期は、実際にはどのような政治状況であったか。4人の娘を天皇に嫁がせた状況と関連づけて説明しなさい。
→ 道長は4人の娘をそれぞれ天皇の后にすることに成功し、3人の天皇の外祖父という前例のない地位に立っていた。朝廷の主要な人事や政策を思いのままに決定でき、反対する勢力もなかった。

(4) 摂関政治を、聖徳太子の時代の豪族の政治介入と比較し、共通点と相違点を述べなさい。
→ 共通点：ともに天皇以外の有力者が政治の実権を握った。相違点：飛鳥時代の蘇我氏は武力で他の豪族を排除したが、藤原氏は婚姻関係を通じて平和的に権力を掌握した。また蘇我氏は天皇権力を脅かそうとしたが、藤原氏は天皇の権威を利用する形で権力を維持した。

## 問2: 浄土信仰と末法思想

平安時代後期に浄土信仰が広まった背景には、末法思想の影響があった。以下の問いに答えなさい。

(1) 末法思想とは何か説明しなさい。
→ 釈迦の死後、正法（正しい教えが行われる時代）→像法（形だけが残る時代）→末法（教えが衰え乱れる時代）と移り変わるという仏教の考え方。当時の日本では1052年に末法の時代に入ると信じられていた。

(2) 末法思想が貴族社会に広まったことと、当時の社会状況の関係を説明しなさい。
→ 11世紀頃には自然災害、疫病、戦乱が相次ぎ、社会不安が高まっていた。末法の時代に入るという恐怖と相まって、来世での救いを求める浄土信仰が貴族を中心に急速に広まった。

(3) 藤原頼通が平等院鳳凰堂を建てた1053年という年が末法思想とどのように関係しているか。
→ 当時の計算で1052年に末法の時代に入るとされており、まさにその翌年に建てられた。末法の世にあっても阿弥陀如来の力で極楽往生できることを願い、極楽浄土を地上に再現しようとして建立された。

(4) 浄土信仰が、のちの鎌倉時代の仏教（法然の浄土宗、親鸞の浄土真宗）にどのようにつながったか説明しなさい。
→ 平安時代の浄土信仰は主に貴族が寺院建立や写経によって往生を願うものであったが、法然は「ただ念仏を唱えれば誰でも往生できる」と説き、親鸞はさらに「阿弥陀如来の慈悲を信じるだけで救われる」と唱えて、浄土信仰を庶民にも広く開放した。

## 問3: 摂関政治の衰退と院政への移行

摂関政治はやがて衰退し、新たな政治形態が生まれた。以下の問いに答えなさい。

(1) 藤原頼通以降、摂関政治が衰退した直接的な原因を説明しなさい。
→ 頼通には天皇のきさきとなる娘がおらず、後三条天皇のように藤原氏と外戚関係のない天皇が即位した。外戚としての権力基盤を失った藤原氏は実権を維持できなくなった。

(2) 後三条天皇が藤原氏の影響力を弱めるために行った政策を1つ挙げなさい。
→ 延久の荘園整理令を出し、不正な荘園を整理して藤原氏の経済基盤を弱めた。

(3) 摂関政治から院政への移行は、権力の所在がどのように変化したことを意味するか。
→ 天皇の外戚である藤原氏が実権を握る構造から、退位した天皇（上皇）自身が直接政治の実権を握る構造へと変化した。

(4) 摂関政治→院政→武家政治（鎌倉幕府）という権力の移り変わりを、「誰が実権を握ったか」という観点からまとめなさい。
→ 摂関政治：藤原氏（天皇の外戚の貴族）が実権を握った。院政：上皇（退位した天皇自身）が実権を握った。武家政治：武士（将軍や執権）が実権を握った。天皇の権威は維持されつつも、実際の政治権力は貴族→上皇→武士へと移り変わった。

## 問4: 摂関政治期の文化と社会

摂関政治の時代は日本独自の国風文化が花開いた時代でもあった。以下の問いに答えなさい。

(1) 貴族の生活が華やかであった一方、農民の生活はどのようなものであったか。両者の格差を説明しなさい。
→ 貴族は寝殿造の邸宅で優雅に暮らし、和歌や音楽を楽しみ、荘園からの収入で裕福に生活していた。一方、農民は簡素な住居に住み、租・調・庸の税に加えて荘園領主への年貢も負担し、生活は厳しかった。

(2) 寝殿造の建築様式が、貴族の文化活動とどのように結びついていたか考察しなさい。
→ 寝殿造の広い庭園は和歌の会や管弦の宴の場として使われ、渡り廊下からは月見や花見を楽しむことができた。開放的な建築は自然との調和を重視する国風文化の美意識を反映していた。

(3) 日宋貿易が行われていたにもかかわらず、この時代の文化が「国風文化」と呼ばれる理由を説明しなさい。
→ 日宋貿易で中国の物品は輸入されたが、正式な国家間の使節は派遣されておらず、唐のように制度や文化を直接学ぶ関係ではなかった。むしろ遣唐使時代に吸収した文化を消化し、仮名文字、大和絵、十二単など日本独自の表現に昇華させたため「国風文化」と呼ばれる。
