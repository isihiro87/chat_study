# 班田収授法と税制度 発展問題

## 問1: 班田収授法の理念と現実

班田収授法は公地・公民の理念に基づく制度であったが、実際にはさまざまな問題が生じた。以下の問いに答えなさい。

(1) 班田収授法の理念を、「土地」「国家」「人民」の語を用いて説明しなさい。
→ すべての土地は国家のものであり、人民に口分田として一時的に貸し与え、死後に国家に返還させることで、土地と人民を国家が管理するという理念。

(2) 戸籍に性別を偽って登録する農民がいたが、なぜ男性が女性として登録したのか。班田収授法と税制度の仕組みから説明しなさい。
→ 口分田は女性のほうが面積が少なく、調・庸・兵役は男性に課されるものが多かった。男性を女性として登録すれば口分田は減るが、税や兵役の負担を大幅に減らすことができた。

(3) 班田収授法が人口増加に対応できなかった理由を、制度の仕組みから説明しなさい。
→ 人口が増えると口分田の需要も増えるが、国が管理する耕地には限りがある。新たに耕地を開墾しなければ口分田が不足するが、班田収授法のもとでは開墾しても土地は私有できなかったため、開墾する動機が生まれにくかった。

(4) 唐の均田制と日本の班田収授法を比較し、共通点と相違点を1つずつ述べなさい。
→ 共通点：国家が人民に土地を分配し、死後に返還させる制度である。相違点：唐の均田制は一部の土地について私有を認めていたが、日本の班田収授法では口分田の私有を認めていなかった。

## 問2: 税制度と農民の抵抗

律令国家の税制度は農民に大きな負担をかけ、さまざまな形で抵抗が見られた。以下の問いに答えなさい。

(1) 租・調・庸の3つの税について、それぞれの特徴を比較し、農民にとってもっとも負担が大きかったものを選び、理由を述べなさい。
→ 租は収穫の約3%で比較的軽い。調は特産物、庸は布を都まで自費で運ぶ必要があり、運搬の負担が大きい。もっとも負担が大きかったのは調・庸で、都までの往復に何十日もかかり、その間の食料も自弁であったため。

(2) 防人の制度が農民に与えた影響を、農業生産と家族生活の面から説明しなさい。
→ 農業生産面では、家族の主要な働き手が3年間も不在となるため農作業に支障が出た。家族生活面では、遠い九州まで行かされるため家族との離別が避けられず、精神的苦痛も大きかった。旅費・食料も自弁で経済的負担も重かった。

(3) 万葉集に収められた防人の歌から、当時の農民の気持ちを推測しなさい。
→ 故郷の家族を思う切ない気持ちや、いつ帰れるかわからない不安、望郷の念が強く表れている。自分の意思ではなく国の命令で遠い九州に送られる悲しみが詠まれている。

(4) 農民が逃亡以外にとった税への抵抗の方法を2つ挙げ、それぞれ説明しなさい。
→ 戸籍の偽装（性別や年齢を偽って登録し、税負担を軽くした）。浮浪・逃散（土地を捨てて別の地域に移り住み、税を逃れた）。

## 問3: 土地制度の変遷と荘園

班田収授法→三世一身法→墾田永年私財法→荘園拡大という流れは、律令国家の根幹を揺るがす大きな変化であった。以下の問いに答えなさい。

(1) 朝廷が三世一身法（723年）を出さなければならなかった社会的背景を説明しなさい。
→ 農民の逃亡が増加し、口分田として配分できる土地が不足した。税収も減少し国家財政が悪化したため、新しい土地の開墾を奨励する必要に迫られた。

(2) 三世一身法から墾田永年私財法への転換が、わずか20年で起きた理由を考察しなさい。
→ 三世一身法では3代で土地を返還しなければならなかったため、返還期限が近づくと土地の手入れを怠る者が現れ、開墾の奨励効果が十分に発揮されなかった。そのため朝廷はより強い動機づけとして永久私有を認めざるを得なかった。

(3) 墾田永年私財法が「律令制度を崩壊させる法律」と評されることがある。この評価の根拠を、公地・公民の原則との矛盾を指摘しながら説明しなさい。
→ 律令制度の根幹は公地・公民であり、すべての土地は国家のものであるという原則に立つ。しかし墾田永年私財法は土地の永久私有を公式に認めたため、この原則と真っ向から矛盾する。結果として貴族や寺院の荘園が拡大し、国が管理する公地は減少し、律令制度は内部から崩壊していった。

(4) 荘園の拡大は、のちの武士の台頭とどのように関係しているか推測しなさい。
→ 荘園が拡大すると、その管理や防衛のために武力を持つ者が必要となった。荘園を守るための武装した管理者がやがて武士へと成長し、地方で力を蓄えていった。

## 問4: 律令制度の変容を読み解く

次の年表を参考に、あとの問いに答えなさい。

| 年 | できごと |
|:-:|:-:|
| 701年 | 大宝律令の完成 |
| 710年 | 平城京への遷都 |
| 723年 | 三世一身法 |
| 743年 | 墾田永年私財法 |

(1) 大宝律令が完成してからわずか22年後に三世一身法が出されたことから、律令制度の運用にどのような問題があったと考えられるか。
→ 大宝律令で定められた班田収授法が理想通りには機能せず、口分田の不足や農民の逃亡といった問題が早くから生じていたことがわかる。

(2) 701年から743年までの42年間で、土地に対する国の方針がどのように変わったかを、「公有」「私有」の語を用いてまとめなさい。
→ 701年の大宝律令では土地は国の公有が原則であったが、723年の三世一身法で限定的な私有を認め、743年の墾田永年私財法で完全な私有を認めるようになった。公有から私有へと大きく方針が転換した。

(3) この変化が起きた根本的な原因と、もたらした結果を、社会の変化と関連づけて論述しなさい。
→ 根本的原因は、人口増加と重い税負担による農民の逃亡で、班田収授法が機能不全に陥ったことである。朝廷は耕地不足を解消するため私有を認めざるを得なくなったが、結果として財力のある貴族や寺院が荘園を拡大し、律令国家の基盤である公地・公民制が崩壊した。やがて荘園を守る武力集団が成長し、武士が台頭する遠因となった。
