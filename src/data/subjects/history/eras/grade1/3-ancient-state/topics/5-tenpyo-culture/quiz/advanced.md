# 天平文化 発展問題

## 問1: 聖武天皇の仏教政策の意図と効果

聖武天皇が国分寺建立や大仏造立を行った背景には、当時の社会状況があった。以下の問いに答えなさい。

(1) 聖武天皇の時代に社会が不安定であった具体的な要因を3つ挙げなさい。
→ 天然痘などの疫病の大流行、藤原広嗣の乱などの貴族の反乱、飢饉による農民の困窮。

(2) 「鎮護国家」という考え方を説明し、聖武天皇の政策との関係を述べなさい。
→ 仏教の力で国家を守り安定させるという考え方。聖武天皇は各国に国分寺を建て、都に東大寺と大仏を造ることで、仏教の力による国家の安定を図ろうとした。

(3) 行基は当初、朝廷から弾圧を受けていたが、のちに大仏造立に協力を求められた。この変化の理由を考察しなさい。
→ 行基は朝廷の許可なく民間で布教活動を行っていたため弾圧されていた。しかし行基には多くの民衆の支持があり、大仏造立という大事業には民衆の労働力が不可欠であった。そのため朝廷は行基の影響力を利用して民衆の協力を得ようとした。

(4) 大仏造立に多くの人々が動員されたことが、当時の農業生産にどのような影響を与えたか推測しなさい。
→ 大仏造立には延べ260万人もの人員が動員されたとされ、農繁期にも労働力が取られたため、各地の農業生産が低下し、農民の生活がさらに圧迫された可能性がある。

## 問2: 鑑真と国際交流

鑑真の来日は、当時の東アジアにおける文化交流の重要な事例である。以下の問いに答えなさい。

(1) 鑑真が5度の失敗にもかかわらず渡日を諦めなかった理由を、当時の日本の仏教界の状況をふまえて説明しなさい。
→ 当時の日本では正式な戒律を授けることができる高僧がおらず、僧侶の資格が不明確であった。鑑真は日本の仏教界を正しく整備するために、強い使命感をもって渡日を試みた。

(2) 鑑真の来日は民間外交の一例ともいえる。遣唐使という公的な外交と比較して、民間レベルの交流の意義を述べなさい。
→ 遣唐使は国家間の公式な交流で制度や政策面の影響が大きいが、鑑真のような個人の来日は宗教・文化面で直接的な知識や技術を伝え、日本社会の根本的な変化をもたらした。

(3) 鑑真が唐招提寺を建立したことの意義を、日本の仏教史の観点から説明しなさい。
→ 唐招提寺は日本で初めて正式な戒律を授ける場となり、僧侶の資格認定制度が確立された。これにより日本の仏教界の規律が整備され、仏教がより体系的に発展する基盤ができた。

## 問3: 古事記・日本書紀の編纂意図

天平文化の時代に歴史書が編纂されたことには政治的な意図があった。以下の問いに答えなさい。

(1) 古事記と日本書紀が編纂された政治的な目的を説明しなさい。
→ 天皇家の正統性を神話の時代にまで遡って証明し、天皇を中心とした国家体制を歴史的に正当化するため。

(2) 風土記が各地で編纂されたことは、律令国家の統治とどのように関係しているか。
→ 各地の地理・産物・伝承を把握することで、中央政府が地方を効率的に統治するための基礎情報を集める目的があった。

(3) 万葉集には天皇から農民まで幅広い層の歌が収められている。このことが可能であった文化的背景を説明しなさい。
→ 万葉仮名という漢字を音にあてはめた表記法が発達し、貴族だけでなく一般の人々も自分の思いを歌に詠んで残すことができるようになった。また、防人の歌のように国の制度を通じて各地の歌が都に集められる仕組みもあった。

(4) 古事記（712年）と日本書紀（720年）が近い時期に編纂された理由を考察しなさい。
→ 古事記は物語的で日本語（和文）の要素が強い歴史書、日本書紀は漢文で書かれた正式な国史であり、目的と対象読者が異なる。古事記は国内向けに天皇家の由来を語り、日本書紀は中国を意識した正式な歴史書として、それぞれ異なる役割で天皇の正統性を示した。

## 問5: 天平文化と正倉院の宝物

正倉院の宝物は天平文化の国際性を示す貴重な資料である。以下の問いに答えなさい。

(1) 正倉院にシルクロード経由の品物が収められている理由を、当時の東アジアの交易ルートと関連づけて説明しなさい。
→ 唐の都・長安はシルクロードの東端に位置し、ペルシアやインドなど西方の品物が集まる国際都市であった。遣唐使がこれらの品物を持ち帰り、また唐の商人を通じて日本にもたらされた。

(2) 校倉造が宝物の保存に適していた理由を、建築の特徴から説明しなさい。
→ 三角形の断面をもつ木材を組み上げる構造で、木材が湿度に応じて膨張・収縮することで室内の湿度が自然に調節され、宝物の劣化を防いだとされる。

(3) 正倉院の宝物が約1300年にわたって保存されてきたことの歴史的・文化的意義を述べなさい。
→ 8世紀の東アジア・西アジアの工芸技術や文化交流の実態を直接示す資料として、世界的にも類をみない貴重な文化遺産である。当時の国際的な文化交流の広がりと、日本文化の豊かさを後世に伝えている。

(4) 天平文化の国際性は、のちの国風文化と比較してどのような特徴があるか。両者を比較して説明しなさい。
→ 天平文化は唐やシルクロードの影響を直接受けた国際色豊かな文化であるのに対し、国風文化は遣唐使の中止後に唐の影響をもとにしながらも日本独自に発展させた文化である。天平文化が「受容」の文化であるとすれば、国風文化は「消化・独自化」の文化といえる。
