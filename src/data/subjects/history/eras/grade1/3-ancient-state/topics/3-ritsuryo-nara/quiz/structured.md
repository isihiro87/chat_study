# 律令国家と奈良時代 大問小問

## 大問1: 大宝律令と律令制度（配点 20点）

701年に（ア）が完成し、唐にならった法律に基づく国家体制が整った。「律」は（イ）に関するきまり、「令」は（ウ）に関するきまりである。以下の問いに答えなさい。

(1) 空欄（ア）に入る法律の名称を答えなさい。
→ 大宝律令

(2) 空欄（イ）に入る語句を答えなさい。
→ 刑罰

(3) 空欄（ウ）に入る語句を答えなさい。
→ 政治

(4) この法律のお手本とした国はどこか。
→ 唐

(5) 大宝律令の編纂に中心的に関わった人物を1人答えなさい。
→ 藤原不比等（または刑部親王）

## 大問2: 中央と地方の政治組織（配点 20点）

律令国家では、中央と地方にそれぞれ役所を設けて全国を統治した。以下の問いに答えなさい。

(1) 行政を行う中央の最高機関を何というか。
→ 太政官

(2) 神事や祭祀を司り、太政官と並ぶ機関を何というか。
→ 神祇官

(3) 都から地方の「国」に派遣された役人を何というか。
→ 国司

(4) 地方の「郡」を治めるために、地元の有力豪族が任命された役職は何か。
→ 郡司

(5) 国司と郡司の違いを、任命の方法に着目して説明しなさい。
→ 国司は都から派遣される中央の役人であるのに対し、郡司は地元の有力豪族から任命された。

(6) 九州の外交・防衛を担った役所の名称を答えなさい。
→ 大宰府

## 大問3: 平城京（配点 20点）

710年に都が奈良の平城京に移された。以下の問いに答えなさい。

(1) 平城京への遷都を行った天皇は誰か。
→ 元明天皇

(2) 平城京のモデルとなった唐の都を答えなさい。
→ 長安

(3) 平城京の中央を南北に走る大通りの名前を答えなさい。
→ 朱雀大路

(4) 平城京の北端に置かれた、天皇の住居と政治の中心を何というか。
→ 平城宮

(5) 平城京の町並みの特徴を説明しなさい。
→ 碁盤の目のように区画が整備され、朱雀大路を中心に左京と右京に分かれていた。

(6) 平城京に都が置かれていた時代を何時代というか。
→ 奈良時代

## 大問4: 貨幣と経済（配点 20点）

律令国家では貨幣経済の導入も進められた。以下の問いに答えなさい。

(1) 708年に鋳造された日本最古の流通貨幣を何というか。
→ 和同開珎

(2) 和同開珎の鋳造が命じられた年を答えなさい。
→ 708年

(3) 銅銭を貯めた者に位を与える法を何というか。
→ 蓄銭叙位令

(4) (3)の法が定められた目的を説明しなさい。
→ 人々に貨幣を使わせ、流通させるため。

(5) 平城京で開かれた商品売買の場を何というか。
→ 市（東市・西市）

## 大問5: 律令国家の全体像（配点 20点）

次の図は律令国家の仕組みを示したものである。あとの問いに答えなさい。

天皇 → 二官（A・B） → 八省 → 国 → 郡 → 里

(1) Aにあたる機関名を答えなさい。
→ 太政官

(2) Bにあたる機関名を答えなさい。
→ 神祇官

(3) 「国」に派遣される役人と「郡」を治める役人の名称をそれぞれ答えなさい。
→ 国に派遣されるのは国司、郡を治めるのは郡司

(4) 東北地方の政治拠点として設置された城の名称を答えなさい。
→ 多賀城

(5) 律令国家が目指した政治体制の特徴を、「天皇」「法律」という語を用いて説明しなさい。
→ 天皇を頂点に、法律（律令）に基づいて全国を統一的に支配する中央集権体制。
