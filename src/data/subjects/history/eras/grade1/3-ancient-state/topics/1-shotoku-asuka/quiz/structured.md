# 聖徳太子と飛鳥時代 大問小問

## 大問1: 聖徳太子の政治改革（配点 20点）

6世紀末、中国では（ア）が全土を統一し、強大な国家となった。日本では593年に（イ）天皇のもとで（ウ）が摂政となり、天皇中心の政治を目指す改革を進めた。以下の問いに答えなさい。

(1) 空欄（ア）に入る王朝名を答えなさい。
→ 隋

(2) 空欄（イ）に入る天皇名を答えなさい。
→ 推古天皇

(3) 空欄（ウ）に入る人物名を答えなさい。
→ 聖徳太子（厩戸皇子）

(4) 聖徳太子とともに仏教を広め、政治を行った有力豪族は誰か。
→ 蘇我馬子

(5) 仏教の受け入れをめぐり蘇我氏と対立し、滅ぼされた豪族は何氏か。
→ 物部氏

## 大問2: 冠位十二階と十七条の憲法（配点 20点）

聖徳太子は新しい制度を設けて天皇中心の政治を整えようとした。以下の問いに答えなさい。

(1) 家柄ではなく才能や功績で役人の地位を決める制度を何というか。
→ 冠位十二階

(2) (1)の制度を定めた目的を、「豪族」「天皇」という語を用いて簡潔に説明しなさい。
→ 豪族の力を抑え、天皇を中心とした政治を行うため。

(3) 役人の心構えを示した法を何というか。
→ 十七条の憲法

(4) 十七条の憲法の第一条で示された、人々の協調を大切にする考えを漢文で答えなさい。
→ 「和を以て貴しとなす」

(5) 十七条の憲法の第二条で敬うべきとされた「三宝」とは何か。
→ 仏・法・僧

## 大問3: 遣隋使と外交（配点 20点）

聖徳太子は中国との外交にも力を入れた。以下の問いに答えなさい。

(1) 聖徳太子が中国に送った使者を何というか。
→ 遣隋使

(2) 遣隋使として派遣された人物は誰か。
→ 小野妹子

(3) 聖徳太子が隋の皇帝に送った手紙で、日本の天皇をどのように表現したか。
→ 「日出づる処の天子」

(4) この手紙で聖徳太子が隋に対して主張したことは何か。
→ 隋と日本が対等な関係であること。

(5) 遣隋使を送った主な目的を2つ答えなさい。
→ 隋の進んだ制度や文化を学ぶこと。隋と対等な外交関係を築くこと。

## 大問4: 飛鳥文化（配点 20点）

聖徳太子の時代に、日本初の仏教文化が栄えた。以下の問いに答えなさい。

(1) この時代に栄えた日本初の仏教文化を何というか。
→ 飛鳥文化

(2) 聖徳太子が斑鳩に建立した、世界最古の木造建築として知られる寺院は何か。
→ 法隆寺

(3) 法隆寺金堂に安置されている、飛鳥文化を代表する仏像は何か。
→ 釈迦三尊像

(4) 飛鳥文化はどの地域の文化の影響を強く受けているか。
→ 中国や朝鮮半島

(5) 蘇我馬子が建てた寺院は何か。
→ 飛鳥寺

(6) 飛鳥文化が仏教を中心とした文化である理由を、聖徳太子の政策と関連づけて説明しなさい。
→ 聖徳太子が十七条の憲法で三宝を敬うことを定め、仏教を政治に取り入れて国を治めようとしたため、仏教中心の文化が発展した。

## 大問5: 聖徳太子の政治の総合理解（配点 20点）

次の資料を読み、あとの問いに答えなさい。

「一に曰く、和を以て貴しとなす。（中略）二に曰く、篤く三宝を敬へ。三宝とは仏・法・僧なり。（中略）三に曰く、詔を承りては必ず謹め。」

(1) この資料は何という法の一部か。
→ 十七条の憲法

(2) この法を定めた人物は誰か。
→ 聖徳太子（厩戸皇子）

(3) 第一条・第二条・第三条の内容から、聖徳太子が目指した政治の理想を説明しなさい。
→ 人々が和を大切にし、仏教を敬い、天皇の命令に従うことで、天皇を中心とした秩序ある国家を作ること。

(4) この法の対象は一般の民衆か、それとも役人か。
→ 役人

(5) 聖徳太子が同じ時期に定めた、才能で役人を登用するための制度を何というか。
→ 冠位十二階
