# 平安京と律令国家の変化 大問小問

## 大問1: 桓武天皇と遷都（配点 20点）

奈良時代の末期、（ア）天皇は政治を立て直すため、784年に（イ）に、794年にはさらに（ウ）に都を移した。以下の問いに答えなさい。

(1) 空欄（ア）に入る天皇名を答えなさい。
→ 桓武天皇

(2) 空欄（イ）に入る都の名称を答えなさい。
→ 長岡京

(3) 空欄（ウ）に入る都の名称を答えなさい。
→ 平安京

(4) 桓武天皇が奈良から都を移した主な理由を説明しなさい。
→ 奈良の大寺院が政治に介入するようになったことを嫌い、新しい都で政治を一新しようとした。

(5) 長岡京からさらに平安京に遷都した理由を述べなさい。
→ 長岡京の造営中に藤原種継の暗殺事件が起き、不吉とされたため。

(6) 平安京に都が置かれた期間を何時代というか。
→ 平安時代

## 大問2: 蝦夷征伐（配点 20点）

桓武天皇は東北地方の蝦夷を朝廷の支配下に置くために軍を送った。以下の問いに答えなさい。

(1) 桓武天皇が蝦夷征伐のために任命した役職を何というか。
→ 征夷大将軍

(2) この役職に任命された人物は誰か。
→ 坂上田村麻呂

(3) 蝦夷の指導者として朝廷に激しく抵抗した人物は誰か。
→ アテルイ

(4) 東北地方の政治・軍事拠点として設置された城は何か。
→ 多賀城

(5) 蝦夷征伐を行った桓武天皇の目的を説明しなさい。
→ 東北地方を朝廷の支配下に置き、律令国家の領域を拡大するため。

(6) 「征夷大将軍」はのちの時代にどのような意味を持つ役職となったか。
→ 鎌倉時代以降、武家政権の長としての役職となった。

## 大問3: 最澄と空海（配点 20点）

唐から帰国した2人の僧が日本に新しい仏教を伝えた。以下の問いに答えなさい。

(1) 天台宗を開いた僧は誰か。
→ 最澄

(2) 最澄が比叡山に建てた寺の名称を答えなさい。
→ 延暦寺

(3) 真言宗を開いた僧は誰か。
→ 空海

(4) 空海が高野山に建てた寺の名称を答えなさい。
→ 金剛峯寺

(5) 最澄と空海が開いた仏教に共通する特徴を、修行の場所に着目して答えなさい。
→ ともに山奥に寺を建てて修行を行う山岳仏教であること。

(6) 奈良時代の仏教と比較して、平安初期の新しい仏教の特徴を説明しなさい。
→ 奈良時代は国家のための仏教（鎮護国家）が中心であったが、平安初期は山奥での厳しい修行を通じた個人の悟りを重視する山岳仏教が広まった。

## 大問4: 遣唐使の中止（配点 20点）

9世紀末、唐への使節の派遣が中止された。以下の問いに答えなさい。

(1) 894年に遣唐使の派遣中止を提案した人物は誰か。
→ 菅原道真

(2) 遣唐使の中止が提案された理由を2つ答えなさい。
→ 唐が衰退していたこと。航海が危険であったこと。

(3) 遣唐使の中止後、日本ではどのような文化が発展したか。
→ 日本独自の国風文化

(4) 菅原道真はのちにどこに左遷されたか。
→ 太宰府（大宰府）

(5) 菅原道真が左遷された原因は何か。
→ 藤原氏の陰謀により無実の罪で左遷された。

(6) 遣唐使の中止と国風文化の発展の関係を説明しなさい。
→ 遣唐使が中止されたことで唐の文化の直接的な流入がなくなり、それまで吸収してきた唐の文化を日本独自のものへと発展させる国風文化が花開いた。

## 大問5: 平安時代初期の総合問題（配点 20点）

次のできごとを年代順に並べ替え、あとの問いに答えなさい。
ア：遣唐使の中止　イ：平安京への遷都　ウ：坂上田村麻呂が征夷大将軍に　エ：最澄・空海が唐へ

(1) 年代順に並べ替えなさい。
→ イ → エ → ウ → ア

(2) イの年を答えなさい。
→ 794年

(3) アの年を答えなさい。
→ 894年

(4) 桓武天皇の時代に行われたできごとをすべて選びなさい。
→ イ、ウ、エ

(5) 平安時代初期の約100年間（794年〜894年）に起きた大きな変化を、仏教と外交の2つの観点からまとめなさい。
→ 仏教面では奈良の旧仏教に代わり最澄と空海が山岳仏教を開き、仏教が新しい展開を見せた。外交面では唐の衰退を理由に遣唐使が中止され、日本独自の文化が発展する基盤ができた。
