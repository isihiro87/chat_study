# 摂関政治 大問小問

## 大問1: 摂関政治の仕組み（配点 20点）

藤原氏は娘を天皇の后にして（ア）の立場を得、幼い天皇に代わって政治を行う（イ）や、成長した天皇を補佐する（ウ）の地位を独占した。このような政治を（エ）という。以下の問いに答えなさい。

(1) 空欄（ア）に入る語句を答えなさい。
→ 外戚

(2) 空欄（イ）に入る役職名を答えなさい。
→ 摂政

(3) 空欄（ウ）に入る役職名を答えなさい。
→ 関白

(4) 空欄（エ）に入る政治の名称を答えなさい。
→ 摂関政治

(5) 藤原氏が外戚の立場を得るために行った具体的な方法を説明しなさい。
→ 自分の娘を天皇のきさき（后）にし、生まれた皇子を次の天皇に即位させることで、天皇の母方の親戚（外戚）となった。

(6) 藤原氏の祖となった人物と、その功績を答えなさい。
→ 中臣鎌足。大化の改新で中大兄皇子に協力した功績により藤原姓を賜った。

## 大問2: 藤原道長の栄華（配点 20点）

藤原道長は摂関政治の全盛期を築いた人物である。以下の問いに答えなさい。

(1) 藤原道長は何人の娘を天皇に嫁がせたか。
→ 4人

(2) 藤原道長が自分の栄華を詠んだ歌を書きなさい。
→ 「この世をば わが世とぞ思ふ 望月の 欠けたることも なしと思へば」

(3) この歌の「望月」とは何か。また、この歌で道長が何を表現しようとしたか。
→ 望月は満月のこと。自分の栄華が満月のように欠けることがないほど完全であると表現した。

(4) 藤原道長が政治の実権を握ることができた仕組みを、「娘」「天皇」「外戚」の語を用いて説明しなさい。
→ 娘を天皇のきさきにし、その間に生まれた皇子を天皇にすることで外戚の立場を得て、政治の実権を握った。

(5) 藤原道長は摂政・関白の地位にはあまり就かなかったが、なぜ実権を握れたのか。
→ 外戚という立場自体が最大の権力の源泉であり、摂政・関白の地位がなくても天皇の外祖父として影響力を行使できたため。

## 大問3: 藤原頼通と浄土信仰（配点 20点）

藤原道長の子・藤原頼通は宇治に美しい建物を建立した。以下の問いに答えなさい。

(1) 藤原頼通が1053年に宇治に建立した建物は何か。
→ 平等院鳳凰堂

(2) 平等院鳳凰堂に安置された仏像は何か。
→ 阿弥陀如来像

(3) 平等院鳳凰堂が建てられた宗教的背景にある信仰は何か。
→ 浄土信仰

(4) 浄土信仰とはどのような信仰か説明しなさい。
→ 阿弥陀如来を信仰し、念仏を唱えて死後に極楽浄土に往生することを願う信仰。

(5) 平安時代に浄土信仰が広まった背景を説明しなさい。
→ 当時、末法思想（釈迦の教えが衰えるという考え）が広まり、貴族たちの間で来世への不安が高まったため。

(6) 平等院鳳凰堂のデザインが使われている硬貨は何か。
→ 10円硬貨

## 大問4: 貴族の暮らし（配点 20点）

摂関政治の時代、貴族たちは優雅な暮らしを送っていた。以下の問いに答えなさい。

(1) 平安時代の貴族の住居の建築様式を何というか。
→ 寝殿造

(2) 寝殿造の特徴を説明しなさい。
→ 中央に寝殿を置き、渡り廊下で対屋（たいのや）と結ばれ、庭には池や築山が設けられた。

(3) 貴族の女性が着用した正装を何というか。
→ 十二単

(4) 貴族社会で重要視された教養を2つ挙げなさい。
→ 和歌を詠むこと。音楽（管弦）を演奏すること。

(5) 貴族の生活と一般の農民の生活の違いを、住居と税の面から比較しなさい。
→ 貴族は寝殿造の豪華な邸宅に住み、給与や荘園からの収入で裕福に暮らした。農民は簡素な住居に住み、租・調・庸などの重い税を課せられ厳しい暮らしであった。

## 大問5: 摂関政治と東アジア（配点 20点）

摂関政治の時代、東アジアでは大きな変化が起きていた。以下の問いに答えなさい。

(1) 10世紀に中国を統一した王朝は何か。
→ 宋

(2) 朝鮮半島を統一した国は何か。
→ 高麗

(3) 宋との間で行われた貿易を何というか。
→ 日宋貿易

(4) 日宋貿易で日本が主に輸入したものを2つ答えなさい。
→ 宋銭、陶磁器（書籍、絹織物なども可）

(5) 遣唐使の中止後も日宋貿易が行われたことから、日本の対外関係についてどのようなことがわかるか。
→ 正式な国家間の使節（遣唐使）は中止されたが、民間レベルでの貿易や文化交流は続いており、日本は完全に鎖国していたわけではなかったことがわかる。

(6) 摂関政治が衰退し、やがて院政へと移行した過程を説明しなさい。
→ 藤原頼通以降、天皇のきさきとなる娘が生まれず、外戚の立場を維持できなくなった。藤原氏と外戚関係のない後三条天皇が即位し、さらにその子の白河天皇が退位後も上皇として政治の実権を握る院政を始めた。
