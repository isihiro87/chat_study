# 天平文化 大問小問

## 大問1: 聖武天皇の仏教政策（配点 20点）

聖武天皇は疫病や反乱が続く中、仏教の力で国を安定させようとした。以下の問いに答えなさい。

(1) 聖武天皇が各国に建てさせた寺を何というか。
→ 国分寺

(2) 聖武天皇が都に建立した寺を何というか。
→ 東大寺

(3) 東大寺に安置された巨大な仏像の正式名称を答えなさい。
→ 盧舎那仏

(4) 大仏造立に民衆をまとめて協力した僧は誰か。
→ 行基

(5) 行基が大仏造立に協力する以前から行っていた活動を説明しなさい。
→ 民間で布教活動を行い、橋・道路・池などを造って人々の生活に貢献していた。

(6) 聖武天皇が仏教の力に頼ろうとした社会的背景を説明しなさい。
→ 当時、疫病の流行、貴族の反乱、飢饉などが相次ぎ、社会が不安定であったため。

## 大問2: 鑑真の来日（配点 20点）

唐の高僧が苦難を乗り越えて日本に来た。以下の問いに答えなさい。

(1) 唐から来日し、日本に正式な戒律を伝えた僧は誰か。
→ 鑑真

(2) 鑑真は何回目の渡航で来日に成功したか。
→ 6回目

(3) 鑑真は渡航の苦難の中でどのような身体的障害を負ったか。
→ 失明した

(4) 鑑真が建立した寺の名称を答えなさい。
→ 唐招提寺

(5) 鑑真が日本に伝えた「戒律」とは何か簡潔に説明しなさい。
→ 僧侶が守るべき規則・修行の規範のこと。正式な僧侶になるための授戒の作法を含む。

(6) 鑑真の来日が日本の仏教にとって重要であった理由を説明しなさい。
→ 当時の日本では正式な戒律による授戒ができず、僧侶の質が問題になっていた。鑑真が正しい戒律を伝えたことで、日本の仏教界が整備された。

## 大問3: 正倉院と国際色（配点 20点）

聖武天皇の愛用品は東大寺の境内にある特別な建物に納められている。以下の問いに答えなさい。

(1) この建物の名称を答えなさい。
→ 正倉院

(2) この建物に見られる特徴的な建築様式を何というか。
→ 校倉造

(3) 校倉造の特徴を簡潔に説明しなさい。
→ 三角形の断面をもつ木材を井桁状に組み上げた建築様式で、湿度の調節に優れている。

(4) 正倉院の宝物にはシルクロード経由の品物が含まれている。このことから天平文化のどのような特色がわかるか。
→ 唐を通じてペルシアやインドなど西方の文化の影響を受けた、国際色豊かな文化であったこと。

(5) 正倉院に納められている代表的な宝物を1つ挙げなさい。
→ 螺鈿紫檀五弦琵琶（または、鳥毛立女屏風など）

## 大問4: 書物と文学（配点 20点）

天平文化の時代には重要な書物が編纂された。以下の問いに答えなさい。

(1) 712年に完成した、神話や天皇家の歴史をまとめた書物は何か。
→ 古事記

(2) 720年に完成した、国の正史として編纂された歴史書は何か。
→ 日本書紀

(3) 各地の自然や産物、伝承を記録した報告書を何というか。
→ 風土記

(4) 日本最古の和歌集で、天皇から農民まで幅広い人々の歌を収めたものは何か。
→ 万葉集

(5) 古事記と日本書紀の共通点と相違点を1つずつ述べなさい。
→ 共通点：ともに神話の時代から歴史を記録した書物である。相違点：古事記は物語的な性格が強く、日本書紀は国の正史として漢文で書かれた公式の歴史書である。

(6) 万葉集に収められた歌は、天皇から農民まで幅広い。このことからどのような文化的特徴がわかるか。
→ 天平文化は貴族だけでなく幅広い層の人々が文化活動に参加していたことがわかる。

## 大問5: 飛鳥文化と天平文化の比較（配点 20点）

次の表を完成させ、あとの問いに答えなさい。

| | 飛鳥文化 | 天平文化 |
|:-:|:-:|:-:|
| 時代 | 7世紀前半 | （ア） |
| 代表的な寺院 | （イ） | 東大寺 |
| 代表的な仏像 | 釈迦三尊像 | （ウ） |
| 影響を受けた文化圏 | 朝鮮半島・中国南北朝 | （エ） |

(1) 空欄（ア）に入る時代を答えなさい。
→ 8世紀

(2) 空欄（イ）に入る寺院名を答えなさい。
→ 法隆寺

(3) 空欄（ウ）に入る仏像名を1つ答えなさい。
→ 阿修羅像（または盧舎那仏）

(4) 空欄（エ）に入る文化圏を答えなさい。
→ 唐・シルクロード（西アジア・インドを含む国際的な文化圏）

(5) 飛鳥文化から天平文化への変化を、日本と東アジアの関係の変化と関連づけて説明しなさい。
→ 飛鳥時代には朝鮮半島経由で中国南北朝の文化が伝わったが、遣唐使の派遣により唐と直接交流するようになり、唐の文化やシルクロード経由の国際的な文化の影響がより強くなった。
