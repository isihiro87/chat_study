# 班田収授法と税制度 大問小問

## 大問1: 班田収授法の仕組み（配点 20点）

律令国家では、（ア）に基づいて6歳以上のすべての人々に（イ）が与えられ、死後に国へ返す仕組みであった。以下の問いに答えなさい。

(1) 空欄（ア）に入る制度名を答えなさい。
→ 班田収授法

(2) 空欄（イ）に入る田の名称を答えなさい。
→ 口分田

(3) 口分田を正確に配るために、何年ごとに作られた台帳は何か。
→ 戸籍（6年ごと）

(4) 口分田の面積は男性と女性で同じか。異なる場合、どちらが多いか。
→ 異なる。男性のほうが多い（女性は男性の約3分の2）。

(5) 班田収授法がお手本にした唐の制度を何というか。
→ 均田制

(6) 班田収授法が公地・公民の考え方に基づいていることを説明しなさい。
→ 土地（口分田）は国のものであり、人民に一時的に貸し与え、死後に国へ返させるため、土地の私有を認めない公地・公民の原則に基づいている。

## 大問2: 租・調・庸と農民の負担（配点 20点）

律令国家では、農民にさまざまな税の負担が課された。以下の問いに答えなさい。

(1) 収穫量の約3%を稲で納める税を何というか。
→ 租

(2) 布や各地の特産物を都に納める税を何というか。
→ 調

(3) 都での労役の代わりに布を納める税を何というか。
→ 庸

(4) 地方で年間60日以内の労働をさせられる労役を何というか。
→ 雑徭

(5) 調や庸を納める際、農民が特に苦しんだ理由を、運搬の面から説明しなさい。
→ 調や庸の品物を自費で都まで運ばなければならず、往復に何十日もかかり、その間の食料も自分で用意しなければならなかったため。

(6) 九州北部の防衛にあたった兵士を何というか。また、主にどの地方の農民から選ばれたか。
→ 防人。主に東国の農民から選ばれた。

## 大問3: 三世一身法と墾田永年私財法（配点 20点）

重い税の負担から農民が逃亡し、口分田が不足するようになった。朝廷は耕地を増やすためにいくつかの法を出した。以下の問いに答えなさい。

(1) 723年に出された、開墾した土地を3代まで私有できるとした法は何か。
→ 三世一身法

(2) 三世一身法の効果が薄かった理由を説明しなさい。
→ 3代で土地を国に返さなければならなかったため、開墾の意欲が十分に高まらなかった。

(3) 743年に出された、開墾した土地の永久私有を認めた法は何か。
→ 墾田永年私財法

(4) 墾田永年私財法が出されたことで、広大な私有地を持つようになったのは主にどのような人々か。
→ 貴族や大寺院

(5) (4)の人々が広大な土地を開墾できた理由を説明しなさい。
→ 多くの労働力と資金を持っていたため、大規模な開墾事業を行うことができた。

(6) 三世一身法から墾田永年私財法への変化が示す、朝廷の土地政策の転換を説明しなさい。
→ 土地の私有を一時的に認める政策から、永久に認める政策へと転換し、開墾をさらに促進しようとした。

## 大問4: 荘園の拡大と律令制の崩壊（配点 20点）

墾田永年私財法の施行後、貴族や大寺院による私有地が広がった。以下の問いに答えなさい。

(1) 貴族や大寺院が持つ私有地を何というか。
→ 荘園

(2) 農民が貴族や寺院に自分の土地を寄進した理由を説明しなさい。
→ 荘園にすることで税の負担を逃れることができたため。

(3) 荘園の拡大が公地・公民の原則をどのように崩していったか説明しなさい。
→ 本来すべての土地は国のもの（公地）であるはずが、荘園として私有地が増え、国が管理する土地が減少した。それに伴い国に税を納める人民（公民）も減少した。

(4) 荘園の拡大は国家の財政にどのような影響を与えたか。
→ 荘園は税が免除される場合があったため、国の税収が減少し財政が悪化した。

(5) 班田収授法→三世一身法→墾田永年私財法→荘園拡大という流れから、律令制度が崩壊に向かった過程を説明しなさい。
→ 班田収授法では土地は国のものとされたが、農民の逃亡で口分田が不足し、三世一身法・墾田永年私財法で土地の私有を認めたため、貴族や寺院の荘園が拡大した。これにより公地・公民の原則が崩れ、律令制度は機能しなくなっていった。

## 大問5: 税と農民の暮らし（配点 20点）

次の表は律令制のもとでの税をまとめたものである。あとの問いに答えなさい。

| 税の種類 | 内容 |
|:-:|:-:|
| A | 収穫の約3%を稲で納める |
| B | 布や特産物を都に納める |
| C | 労役の代わりに布を納める |
| D | 地方で年間60日以内の労役 |

(1) A〜Dにあたる税の名称をそれぞれ答えなさい。
→ A：租、B：調、C：庸、D：雑徭

(2) A〜Cのうち、もっとも農民の生活を圧迫したのはどれか。理由もあわせて答えなさい。
→ BまたはC。調・庸は品物を自費で都まで運ぶ必要があり、往復に長い日数がかかったため。

(3) 九州を守る兵士の名称を答え、その任務の大変さを説明しなさい。
→ 防人。主に東国の農民から選ばれ、3年間家族と離れて九州北部の防衛に当たった。旅費や食料も自弁で、家の農作業ができなくなるため家族にも大きな負担がかかった。

(4) 重い税負担から逃れるため、農民がとった行動を2つ答えなさい。
→ 逃亡して別の土地に逃げた。戸籍の性別を偽って登録した（男性が女性として登録し、口分田の面積を減らして税負担を軽くしようとした）。

(5) 農民の逃亡が増加したことが、律令制の崩壊につながった理由を説明しなさい。
→ 逃亡者が増えると口分田を配る人が減り、税収も減少する。朝廷は耕地を増やすために土地の私有を認めざるを得なくなり、結果として公地・公民の原則が崩壊していった。
