# 中国文明の発展 大問小問

## 大問1: 殷と甲骨文字（配点 20点）

次の文を読んで、あとの問いに答えなさい。

（ ア ）・長江の流域で中国文明が発生した。中国最古の王朝とされる（ イ ）では（ ウ ）の甲羅や牛の骨に文字が刻まれた。この文字を（ エ ）という。

(1) ア〜エにあてはまる語句を答えなさい。（各2点）
→ ア：黄河、イ：殷、ウ：亀、エ：甲骨文字。

(2) 甲骨文字は何のために使われたか。（4点）
→ 占いの結果を記録するために使われた。戦争や農業など重要な事項について神の意志を確認した。

(3) 甲骨文字は現在のどの文字のもとになったか。（2点）
→ 漢字。

(4) 殷を滅ぼした国の名前を答えなさい。（2点）
→ 周。

(5) 周の支配が弱まった後の、多くの国が争った時代を何と呼ぶか。（2点）
→ 春秋戦国時代。

## 大問2: 孔子と儒教（配点 20点）

(1) 紀元前6世紀頃に活躍した中国の思想家で、「仁」と「礼」を説いた人物はだれか。（2点）
→ 孔子。

(2) 「仁」とはどのような教えか、説明しなさい。（4点）
→ 人を思いやる心のこと。他者への愛情や慈しみの気持ちを大切にするという教え。

(3) 「礼」とはどのような教えか、説明しなさい。（4点）
→ 社会の秩序や人間関係の決まり・礼儀を守ること。目上の人を敬い、社会の調和を保つという教え。

(4) 孔子の教えがもとになった思想を何というか。（2点）
→ 儒教。

(5) 孔子の言行を弟子たちがまとめた書物を何というか。（2点）
→ 論語。

(6) 儒教がその後の中国の政治にどのような影響を与えたか説明しなさい。（6点）
→ 漢の時代に国の学問として採用され、約2000年にわたって中国の政治や道徳の基盤となった。官僚を選ぶ試験にも儒教の経典が使われた。

## 大問3: 秦の統一（配点 20点）

次の文を読んで、あとの問いに答えなさい。

紀元前（ ア ）年、（ イ ）の王が初めて中国を統一し、自らを（ ウ ）と名乗った。北方の（ エ ）民族の侵入を防ぐため（ オ ）を築いた。

(1) ア〜オにあてはまる語句を答えなさい。（各2点）
→ ア：221、イ：秦、ウ：始皇帝、エ：遊牧、オ：万里の長城。

(2) 始皇帝が中国全土で統一したものを2つ答えなさい。（4点）
→ 文字と度量衡（長さ・重さ・容量の単位）。

(3) 始皇帝が書物を焼き、儒学者を弾圧した政策を何というか。（2点）
→ 焚書坑儒。

(4) 始皇帝の墓のそばから発見された兵士や馬の焼き物を何というか。（2点）
→ 兵馬俑。

(5) 秦がわずか15年で滅亡した理由を説明しなさい。（4点）
→ 始皇帝の厳しい法律や過酷な労役に民が苦しみ、始皇帝の死後に各地で反乱が起きたため。

## 大問4: 漢とシルクロード（配点 20点）

(1) 秦の後に中国を統一した国と、その建国者を答えなさい。（4点）
→ 国：漢、建国者：劉邦。

(2) 漢の時代に国の学問として採用された思想は何か。（2点）
→ 儒教。

(3) 中国と西方を結んだ東西交易路を何と呼ぶか。（2点）
→ シルクロード（絹の道）。

(4) シルクロードという名前の由来を説明しなさい。（4点）
→ 中国の特産品である絹（シルク）がこの交易路を通って西方に運ばれたことに由来する。

(5) シルクロードを通じて中国から西方に運ばれた主な産物を答えなさい。（2点）
→ 絹。

(6) 漢の時代にシルクロードの開拓に貢献し、西域に派遣された人物はだれか。（2点）
→ 張騫。

(7) シルクロードの開通が東西の文明交流にもたらした意義を説明しなさい。（4点）
→ 中国の絹や技術が西方に伝わり、西方のガラス製品や宗教（仏教）が中国に伝わるなど、東西の文化・物資の交流が活発になった。

## 大問5: 総合問題（配点 20点）

次の出来事を年代の古い順に並べ替え、それぞれの特徴を簡潔に述べなさい。

ア. シルクロードによる東西交易
イ. 始皇帝による中国統一
ウ. 甲骨文字の使用
エ. 孔子が「仁」と「礼」を説く

(1) 古い順に並べなさい。（4点）
→ ウ → エ → イ → ア。

(2) ウの出来事の特徴を述べなさい。（4点）
→ 殷の時代に亀の甲羅や牛の骨に文字を刻んで占いの記録をした。この文字が現在の漢字のもとになった。

(3) エの出来事の特徴を述べなさい。（4点）
→ 孔子が紀元前6世紀頃に人を思いやる「仁」と社会の秩序を守る「礼」を説き、儒教の基礎を築いた。

(4) イの出来事の特徴を述べなさい。（4点）
→ 紀元前221年に秦の王が始皇帝を名乗り、文字や度量衡を統一した。万里の長城を築いたが、厳しい政治で秦は短命に終わった。

(5) アの出来事の特徴を述べなさい。（4点）
→ 漢の時代に中国と西方を結ぶシルクロードが開かれ、絹などの交易が行われた。東西の文化交流が活発になった。
