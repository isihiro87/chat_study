# 人類の出現と進化 大問小問

## 大問1: 人類の進化（配点 20点）

次の文を読んで、あとの問いに答えなさい。

約（ ア ）万年前にアフリカで最古の人類である（ イ ）が現れた。やがて約240万年前に（ ウ ）が現れ、（ エ ）を使うようになった。約20万年前には現在の人類の直接の祖先である（ オ ）が出現した。

(1) ア〜オにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：700、イ：猿人、ウ：原人、エ：火、オ：新人（ホモ・サピエンス）

(2) 人類が最初に出現した大陸はどこか。（2点）
→ アフリカ大陸。

(3) 猿人がほかの動物と大きく異なる特徴を1つ答えなさい。（2点）
→ 二足歩行を行ったこと。

(4) 原人の具体例を1つ答えなさい。（2点）
→ 北京原人（またはジャワ原人）。

(5) 新人の学名を答えなさい。（2点）
→ ホモ・サピエンス。

(6) 人類の進化を古い順に正しく並べなさい。（2点）
→ 猿人 → 原人 → 新人。

## 大問2: 旧石器時代（配点 20点）

次の問いに答えなさい。

(1) 石を打ち欠いて作った石器を何というか。（2点）
→ 打製石器。

(2) 打製石器を使い狩りや採集をしていた時代を何と呼ぶか。（2点）
→ 旧石器時代。

(3) 旧石器時代の人々はどのように食料を得ていたか。2つの方法を答えなさい。（4点）
→ 狩猟（狩り）と採集。

(4) 旧石器時代の人々はなぜ定住しなかったのか。理由を説明しなさい。（4点）
→ 狩りや採集で食料を得ていたため、獲物や食べ物を求めて移動する必要があったから。

(5) フランスで発見された旧石器時代の代表的な洞窟壁画の名前を答えなさい。（2点）
→ ラスコーの洞窟壁画。

(6) 洞窟壁画には主に何が描かれていたか。また、描かれた目的は何と考えられるか。（4点）
→ 動物（牛・馬など）が描かれていた。狩りの成功を祈るためと考えられている。

(7) 旧石器時代の地球の気候はどのようであったか。（2点）
→ 氷河時代にあたり、寒冷な時期が繰り返された。

## 大問3: 新石器時代（配点 20点）

次の問いに答えなさい。

(1) 表面をみがいて作った石器を何というか。（2点）
→ 磨製石器。

(2) 磨製石器や土器を使い始めた時代を何と呼ぶか。（2点）
→ 新石器時代。

(3) 新石器時代に始まった食料の新しい生産方法を2つ答えなさい。（4点）
→ 農耕と牧畜。

(4) 土器はどのような用途に使われたか。2つ答えなさい。（4点）
→ 食物の煮炊きと貯蔵。

(5) 新石器時代に人々が定住するようになった理由を説明しなさい。（4点）
→ 農耕や牧畜により食料を安定して得られるようになり、移動する必要がなくなったため。

(6) 農耕が始まったことにより、人口や社会にどのような変化が起きたか。2つ答えなさい。（4点）
→ 人口が増加して集落が大きくなったこと、やがて指導者が生まれ身分の差ができたこと。

## 大問4: 石器の比較（配点 20点）

次の表を完成させ、あとの問いに答えなさい。

| | 旧石器時代 | 新石器時代 |
|---|---|---|
| 石器 | （ア） | （イ） |
| 食料の得方 | （ウ） | （エ） |
| 住まい方 | （オ） | （カ） |

(1) ア〜カにあてはまる語句を答えなさい。（各2点）
→ ア：打製石器、イ：磨製石器、ウ：狩猟・採集、エ：農耕・牧畜、オ：移動生活、カ：定住生活。

(2) 新石器時代に新しく使われるようになった、粘土を焼いて作った容器を何というか。（2点）
→ 土器。

(3) 旧石器時代から新石器時代への変化で、最も大きな転換点は何か。理由とともに説明しなさい。（6点）
→ 農耕・牧畜の開始。食料を自ら生産することで安定した食料供給が可能になり、定住生活・人口増加・社会の複雑化につながったため。

## 大問5: 総合問題（配点 20点）

次の文章を読んで、あとの問いに答えなさい。

人類は長い進化の歴史の中で、さまざまな道具や技術を身につけてきた。最初の人類は（ ア ）を行うことで手が自由になり、やがて道具を作るようになった。（ イ ）の使用は、食生活や居住環境を大きく変えた。旧石器時代には（ ウ ）の洞窟壁画のような芸術も生まれた。新石器時代に入ると（ エ ）の開始により、人類の暮らしは劇的に変化した。

(1) ア〜エにあてはまる語句を答えなさい。（各2点）
→ ア：二足歩行、イ：火、ウ：ラスコー（またはアルタミラ）、エ：農耕（農耕・牧畜）。

(2) 二足歩行が人類の進化にとって重要であった理由を説明しなさい。（4点）
→ 両手が自由になり、道具を作ったり使ったりすることが可能になったため。

(3) 火の使用が人類にもたらした利点を3つ答えなさい。（6点）
→ 暖をとれるようになったこと、食物を加熱して食べやすくなったこと、獣から身を守れるようになったこと。

(4) 旧石器時代と新石器時代の名称の由来を説明しなさい。（2点）
→ 使われていた石器の種類に由来する。旧石器時代は打製石器、新石器時代は磨製石器から名がついた。
