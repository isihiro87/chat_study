# 古代文明のおこり 大問小問

## 大問1: 文明の誕生と特徴（配点 20点）

次の文を読んで、あとの問いに答えなさい。

大きな川のほとりで（ ア ）や文字を持つ社会が発達した。これを（ イ ）という。古代の文明はいずれも大河の流域で発生し、農耕に必要な（ ウ ）と、洪水がもたらす（ エ ）な土壌に恵まれていた。

(1) ア〜エにあてはまる語句を答えなさい。（各2点）
→ ア：都市、イ：文明、ウ：水、エ：肥沃（ひよく）。

(2) 古代文明に共通する3つの特徴を答えなさい。（6点）
→ 大河のほとりに発生したこと、文字が発明されたこと、都市が形成されたこと。

(3) 文明が大河のほとりで発生した理由を、農業との関わりから説明しなさい。（6点）
→ 大きな川の流域は農耕に必要な水が豊富で、洪水によって肥沃な土が運ばれ農業に適していた。安定した食料生産が可能になり、人口が増えて都市が形成された。

## 大問2: メソポタミア文明（配点 20点）

(1) メソポタミア文明が発達した2つの川の名前を答えなさい。（4点）
→ チグリス川とユーフラテス川。

(2) 「メソポタミア」という名前の意味を答えなさい。（2点）
→ 「川の間の土地」という意味。

(3) メソポタミアで粘土板に刻まれた文字を何というか。（2点）
→ くさび形文字。

(4) 「目には目を、歯には歯を」で有名な法典の名前を答えなさい。（2点）
→ ハンムラビ法典。

(5) ハンムラビ法典はどの国の王が作ったか。（2点）
→ バビロニアの王ハンムラビ。

(6) メソポタミア文明で発明された暦と数の数え方をそれぞれ答えなさい。（4点）
→ 太陰暦（月の満ち欠けを基準にした暦）と60進法。

(7) 60進法が現在の生活にどのように残っているか、例を1つ挙げなさい。（4点）
→ 時間の計算（60分＝1時間、60秒＝1分）や角度（360度）に残っている。

## 大問3: エジプト文明（配点 20点）

(1) エジプト文明が発達した川の名前を答えなさい。（2点）
→ ナイル川。

(2) エジプトで王のことを何と呼んだか。（2点）
→ ファラオ。

(3) ファラオの墓として建てられた巨大な建造物を何というか。（2点）
→ ピラミッド。

(4) エジプト文明で使われた文字を何というか。（2点）
→ 象形文字（ヒエログリフ）。

(5) エジプトで太陽暦が作られた理由を説明しなさい。（4点）
→ ナイル川の洪水の時期を正確に予測するために、太陽の動きを基準にした1年365日の暦が必要だったため。

(6) ナイル川の洪水がエジプト文明にとって重要だった理由を説明しなさい。（4点）
→ 毎年の洪水が上流から肥沃な土を運び、豊かな農地を作った。これにより安定した農業が可能になった。

(7) エジプト文明で死者をミイラにした理由を述べなさい。（4点）
→ 来世（死後の世界）で復活するために、体を保存する必要があると信じていたため。

## 大問4: インダス文明・中国文明（配点 20点）

(1) インダス文明が発達した川の名前を答えなさい。（2点）
→ インダス川。

(2) インダス文明の代表的な都市遺跡を1つ答えなさい。（2点）
→ モヘンジョ＝ダロ（またはハラッパー）。

(3) モヘンジョ＝ダロの特徴を2つ述べなさい。（4点）
→ 整然とした街路が整備されていたこと、下水道や排水設備が整った計画都市であったこと。

(4) インダス文明が衰退した後にインドに侵入した民族の名前を答えなさい。（2点）
→ アーリヤ人。

(5) 中国文明が発生した2つの川の名前を答えなさい。（4点）
→ 黄河と長江。

(6) 殷で使われた文字を何というか。また、それはどのようなものに刻まれたか。（4点）
→ 甲骨文字。亀の甲羅や牛の骨に刻まれた。

(7) 甲骨文字は何のために使われたか。（2点）
→ 占いの結果を記録するため。
