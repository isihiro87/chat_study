# ギリシャ・ローマ 大問小問

## 大問1: ギリシャの都市国家（配点 20点）

次の文を読んで、あとの問いに答えなさい。

ギリシャでは（ ア ）と呼ばれる都市国家が形成された。その中心には（ イ ）と呼ばれる丘の上の城塞があり、（ ウ ）などの神殿が建てられた。代表的なポリスである（ エ ）では、市民全員が参加する（ オ ）が行われた。

(1) ア〜オにあてはまる語句を答えなさい。（各2点）
→ ア：ポリス、イ：アクロポリス、ウ：パルテノン神殿、エ：アテネ、オ：民主政。

(2) アテネの民主政に参加できたのはどのような人々か。参加できなかった人々も含めて説明しなさい。（4点）
→ 参加できたのは成年男性の市民のみ。女性や奴隷、外国人は参加できなかった。

(3) アテネの民主政が「直接民主政」と呼ばれる理由を述べなさい。（2点）
→ 市民が代表者を選ぶのではなく、直接集会に参加して政治の決定を行ったため。

(4) もう1つの代表的なポリスであるスパルタの特徴を述べなさい。（2点）
→ 軍事を重視し、市民に厳しい訓練を課す軍事国家であった。

## 大問2: ペルシャ戦争とアレクサンドロス大王（配点 20点）

(1) ギリシャのポリス連合軍が撃退した西アジアの大帝国はどこか。（2点）
→ ペルシャ（ペルシャ帝国）。

(2) この戦いを何と呼ぶか。（2点）
→ ペルシャ戦争。

(3) ペルシャ戦争後、ギリシャのポリスはどのような状態になったか。（4点）
→ アテネを中心とするグループとスパルタを中心とするグループに分かれて争うようになり、ギリシャ全体の力が弱まった。

(4) マケドニアの王で東方遠征を行った人物はだれか。（2点）
→ アレクサンドロス大王。

(5) アレクサンドロス大王の遠征はどの範囲まで及んだか。（2点）
→ エジプトからインド北西部にまで及んだ。

(6) アレクサンドロス大王の東方遠征により生まれた文化を何というか。（2点）
→ ヘレニズム文化。

(7) ヘレニズム文化とはどのような文化か、説明しなさい。（6点）
→ ギリシャ文化とオリエント（東方）文化が融合した文化。ギリシャの芸術・哲学と東方の技術・宗教が混ざり合い、彫刻・建築・天文学などの分野で新しい成果が生まれた。

## 大問3: ローマの政治体制（配点 20点）

(1) ローマで最初に行われた、王を置かない政治の仕組みを何というか。（2点）
→ 共和政。

(2) ローマの共和政の特徴を説明しなさい。（4点）
→ 王を置かず、市民の中から選ばれた代表者（元老院や執政官）が政治を行う仕組み。

(3) ローマが共和政から帝政に変わった理由を述べなさい。（4点）
→ 領土の拡大に伴い内部の権力争いが激化し、強力な指導者が必要となったため。カエサルの独裁を経て、オクタウィアヌスが初代皇帝となった。

(4) 皇帝が支配する政治の仕組みを何というか。（2点）
→ 帝政。

(5) ローマ帝国で4世紀に国教として採用された宗教は何か。（2点）
→ キリスト教。

(6) ローマ帝国が東西に分裂した後、西ローマ帝国が滅亡した年を答えなさい。（2点）
→ 476年。

(7) 西ローマ帝国の滅亡の原因を述べなさい。（4点）
→ ゲルマン民族の大移動によって帝国内に多くの異民族が流入し、国力が衰退して滅亡した。

## 大問4: ローマの文化と建造物（配点 20点）

(1) ローマ市内に造られた、約5万人を収容できた巨大な円形闘技場の名称を答えなさい。（2点）
→ コロッセオ。

(2) ローマ帝国が各地に建設した水を運ぶ橋を何というか。（2点）
→ 水道橋。

(3) 「すべての道はローマに通ず」ということわざが生まれた理由を説明しなさい。（4点）
→ ローマ帝国が広大な領土を効率的に統治するために、ローマを中心とする道路網を帝国各地に張り巡らせたため。

(4) ローマが整備した法律（ローマ法）が後世に与えた影響を述べなさい。（4点）
→ ローマ法はヨーロッパの法律の基礎となり、現在の多くの国の法体系にも影響を与えている。

(5) ローマ帝国が巨大な建造物を各地に建設できた理由を「技術」と「労働力」の観点から述べなさい。（4点）
→ コンクリートやアーチ構造など優れた建築技術を持っていたこと、征服した地域の人々を労働力として動員できたこと。

(6) ローマ帝国が広大な領土を長期間維持できた理由を2つ答えなさい。（4点）
→ 道路網の整備により軍隊の迅速な移動と物資の輸送が可能だったこと、法律を整備して統治の基盤を確立したこと。

## 大問5: ギリシャ・ローマの比較（配点 20点）

次の表を完成させ、あとの問いに答えなさい。

| | ギリシャ | ローマ |
|---|---|---|
| 政治体制 | （ア） | （イ）→（ウ） |
| 代表的建造物 | （エ） | （オ）・（カ） |
| 文化の特徴 | （キ）の発展 | （ク）の整備 |

(1) ア〜クにあてはまる語句を答えなさい。（各2点）
→ ア：民主政（ポリスにおける直接民主政）、イ：共和政、ウ：帝政、エ：パルテノン神殿、オ：コロッセオ、カ：水道橋、キ：哲学・芸術、ク：法律・道路。

(2) ギリシャの政治とローマの政治の最大の違いを説明しなさい。（4点）
→ ギリシャ（アテネ）は市民全員が直接参加する直接民主政であったのに対し、ローマは代表者を選んで政治を行う共和政から始まり、やがて皇帝が統治する帝政に移行した。
