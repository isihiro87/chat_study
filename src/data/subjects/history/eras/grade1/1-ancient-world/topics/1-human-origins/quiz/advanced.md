# 人類の出現と進化 発展問題

## 問1: 人類の進化と環境の関係

(1) 人類が二足歩行を始めた背景には、アフリカの環境変化があったと考えられている。どのような環境変化が起きたか説明しなさい。
→ アフリカの気候が乾燥化し、森林が減少して草原（サバンナ）が広がった。樹上生活が困難になり、草原で生活するために二足歩行が有利になったと考えられている。

(2) 氷河時代に海面が低下したことで、人類の移動にどのような影響があったか、具体例を挙げて説明しなさい。
→ 海面低下により、現在は海で隔てられている地域が陸続きになった。例えばアジア大陸と日本列島が陸橋でつながり、マンモスなどの大型動物とともに人類が日本列島に渡ることが可能になった。

(3) 新人（ホモ・サピエンス）がアフリカから世界各地に広がることができた理由を、道具や言葉の面から2つ以上述べなさい。
→ 精巧な石器や骨角器を作る技術を持っていたこと、言葉を使って仲間と協力し情報を共有できたこと、衣服を作って寒冷な地域にも適応できたこと。

(4) 旧石器時代の人々が洞窟壁画を描いた理由について、「狩りの成功を祈るため」以外にどのような説があるか述べなさい。
→ 狩りの技術や知識を次の世代に伝えるための教育的な目的があった、宗教的・儀式的な意味があった、芸術的な表現欲求に基づくものだった、などの説がある。

## 問2: 農耕革命の影響

(1) 農耕が最初に始まったのはどの地域と考えられているか。また、最初に栽培された作物の例を1つ挙げなさい。
→ 西アジア（メソポタミア周辺）と考えられている。最初に栽培されたのは小麦や大麦である。

(2) 農耕の開始が「革命」と呼ばれるほど大きな変化であった理由を、「食料の獲得方法」「人口」「社会構造」の3つの観点から説明しなさい。
→ 食料の獲得方法が狩猟・採集から生産へと根本的に変わった。安定した食料供給により人口が大幅に増加した。人口増加に伴い集落が大きくなり、指導者や身分の差が生まれて社会が複雑化した。

(3) 農耕・牧畜の開始により、やがて「文明」が生まれた。農耕から文明の誕生に至るまでの流れを順を追って説明しなさい。
→ 農耕の開始 → 食料の安定供給 → 定住生活・人口増加 → 集落の拡大 → 指導者の出現・身分の差 → 大きな川のほとりで灌漑農業の発達 → 都市の形成 → 文字の発明 → 文明の誕生。

(4) 農耕社会では、なぜ文字が必要になったと考えられるか説明しなさい。
→ 収穫物の記録や管理、税の計算、法律や取り決めの記録など、社会の運営に文字による記録が不可欠になったため。

## 問3: 道具の発達と人類の進歩

(1) 打製石器から磨製石器への変化は、人類のどのような能力の発達を示しているか説明しなさい。
→ 石をより精密に加工する技術力の向上、計画的に道具を作る知性の発達、用途に合わせて道具を使い分ける工夫の能力を示している。

(2) 土器の発明が人類の食生活をどのように変えたか、具体的に説明しなさい。
→ 食物を煮炊きできるようになり、生では食べにくかった穀物やイモなどを柔らかく調理して食べられるようになった。また食料の貯蔵が可能になり、余った食料を保存できるようになった。

(3) 以下の出来事を古い順に並べ替えなさい。
ア. 農耕・牧畜の開始　イ. 火の使用　ウ. 磨製石器の使用　エ. 二足歩行の開始　オ. 洞窟壁画の制作
→ エ（約700万年前）→ イ（約240万年前）→ オ（約2万年前）→ ウ（約1万年前）→ ア（約1万年前）。

(4) 「食料を獲得する」生活から「食料を生産する」生活への変化が、現代社会にまで続くどのような影響を与えたか、自分の考えを述べなさい。
→ 食料生産により人口が増え、社会の分業が進んだ。農業に従事しない人々が政治・宗教・技術・芸術などの専門分野を発展させ、それが都市・国家・文明の発展につながった。現代の社会構造の基盤は農耕の開始に遡ることができる。

## 問4: 資料読み取り問題

次の資料を見て、あとの問いに答えなさい。

【資料】人類の進化と道具の年表
- 約700万年前：猿人の出現（アフリカ）
- 約240万年前：原人の出現、打製石器の使用開始
- 約20万年前：新人の出現（アフリカ）
- 約4万年前：新人がヨーロッパに到達
- 約2万年前：ラスコーの洞窟壁画
- 約1万年前：農耕・牧畜の開始、磨製石器・土器の使用

(1) 猿人の出現から新人の出現まで、約何年の歳月がかかったか。（3点）
→ 約680万年（700万年 − 20万年 = 680万年）。

(2) 新人が出現してからヨーロッパに到達するまでにかかった年数を求めなさい。また、この間に新人はどのような経路をたどったと考えられるか。（4点）
→ 約16万年（20万年前 − 4万年前 = 16万年）。アフリカを出て中東（西アジア）を経由してヨーロッパに到達したと考えられる。

(3) 打製石器の使用が始まってから農耕が始まるまでの期間と、農耕が始まってから現在までの期間を比べ、人類の歴史において狩猟・採集の時代がいかに長かったか説明しなさい。（4点）
→ 打製石器から農耕まで約239万年、農耕から現在まで約1万年。人類の道具使用の歴史の99%以上は狩猟・採集の時代であり、農耕の歴史はごくわずかな期間に過ぎない。

(4) この年表から読み取れる「人類の進化の特徴」を1つ述べなさい。（4点）
→ 人類の進化は非常にゆっくり進んだが、新人の出現以降は道具や技術の発達が加速し、特に農耕の開始以降は急速に社会が変化した。技術の発展が人類の歴史を大きく変えたことがわかる。

## 問5: 比較と論述

(1) 旧石器時代の人々と新石器時代の人々の暮らしを、「食料」「住まい」「道具」「社会」の4つの観点から比較しなさい。
→ 食料：旧石器時代は狩猟・採集、新石器時代は農耕・牧畜。住まい：旧石器時代は移動生活（洞穴やテント）、新石器時代は定住生活（集落の形成）。道具：旧石器時代は打製石器、新石器時代は磨製石器・土器。社会：旧石器時代は小集団、新石器時代は大きな集落と指導者の出現。

(2) 「人類は道具を作ることで進化した」という意見と「人類は協力することで進化した」という意見のどちらに賛成か、理由とともに述べなさい。
→ どちらの意見にも根拠がある。道具の面では打製石器から磨製石器、土器へと進歩することで生活が向上した。協力の面では集団での狩猟、農耕のための灌漑作業、集落の防衛など、協力なしには生き延びられなかった。両方が相互に影響し合って人類は進化したと考えるのが妥当である。

(3) 現代の日本に住む私たちの生活と、新石器時代の人々の生活を比較し、新石器時代に生まれた「農耕」が現代にどのようにつながっているか論じなさい。
→ 新石器時代に始まった農耕は、食料の安定供給を可能にし、定住→集落→都市→国家という社会の発展を導いた。現代の食料生産、土地の所有、社会の階層構造、さらには国家という仕組みの根幹は、すべて約1万年前に始まった農耕に遡ることができる。

(4) 旧石器時代の洞窟壁画が「人類最古の芸術」と呼ばれる理由を、壁画の特徴や目的から考察しなさい。
→ ラスコーやアルタミラの壁画は単なる記録ではなく、動物の動きを生き生きと描き、色彩を使い分けるなど高い表現力を持っている。狩りの成功を祈る宗教的・儀式的な目的もあったと考えられ、人類が実用を超えて美や精神世界を表現し始めた最古の証拠として重要である。
