# 中国文明の発展 発展問題

## 問1: 殷から周への変化

(1) 殷の政治の特徴である「神権政治」とはどのような政治のあり方か説明しなさい。また、このような政治が甲骨文字の発達とどのように関わっているか述べなさい。
→ 神権政治とは、占いによって神の意志を確認し、それに基づいて政治的な決定を行う政治のあり方である。占いの結果を正確に記録する必要があったため、亀の甲羅や牛の骨に甲骨文字が刻まれるようになり、文字の発達が促された。

(2) 周の時代には「封建制度」が行われた。これはどのような制度か。また、この制度が後に弱体化した理由を述べなさい。
→ 封建制度とは、王が一族や功臣に土地を与えて治めさせ、その代わりに軍事的な奉仕を求める制度である。やがて地方の諸侯が力を蓄えて独立性を高め、王の権威が低下して周の支配が弱体化した。

(3) 春秋戦国時代には孔子以外にも多くの思想家が現れた。このような状況を何と呼ぶか。また、なぜこの時代に多くの思想家が現れたか理由を述べなさい。
→ 「諸子百家（しょしひゃっか）」と呼ぶ。戦乱の時代にあって、各国の君主が優れた人材を求めたことや、社会が混乱する中で理想的な政治や生き方を模索する思想家が多く現れたため。

## 問2: 始皇帝の政策の意義と問題点

(1) 始皇帝が文字を統一したことの意義を、「情報伝達」と「国家の一体性」の観点から説明しなさい。
→ 情報伝達の観点：統一された文字により、広大な領土内で命令や法律を正確に伝えることが可能になった。国家の一体性の観点：異なる地域の人々が同じ文字を使うことで、一つの国家としての意識が生まれた。

(2) 始皇帝が行った「焚書坑儒」の目的と、その政策がもたらした弊害を述べなさい。
→ 目的は、儒教などの思想によって政治が批判されることを防ぎ、思想統制によって統治を安定させることであった。弊害は、貴重な書物が失われて学問・文化が大きな損害を受けたこと、知識人の弾圧が民心の離反を招いたこと。

(3) 万里の長城の建設と始皇帝陵（兵馬俑）の造営が秦の滅亡にどのように影響したか、「労役」という言葉を使って説明しなさい。
→ 万里の長城の建設や始皇帝陵の造営には膨大な労役が必要で、多くの農民が強制的に動員された。過酷な労役により民衆は疲弊し、始皇帝への不満が高まった。この不満が始皇帝の死後の反乱につながり、秦はわずか15年で滅亡した。

(4) 始皇帝の政策を「功績」と「過ち」に分けて整理しなさい。
→ 功績：中国を統一して戦乱を終わらせたこと、文字・度量衡・車輪の幅を統一して交通や交易を便利にしたこと、万里の長城で北方の脅威に備えたこと。過ち：焚書坑儒で学問・文化を弾圧したこと、過酷な法律と労役で民を苦しめたこと、短期間の滅亡につながる独裁的な政治を行ったこと。

## 問3: シルクロードと東西交流

(1) シルクロードには陸路だけでなく海路もあった。陸のシルクロードと海のシルクロード（海の道）のそれぞれの利点を述べなさい。
→ 陸のシルクロード：中央アジアのオアシス都市を経由して安定した交易が可能であった。海のシルクロード（海の道）：大量の物資を一度に運べ、陸路より短時間で目的地に到達できる利点があった。

(2) シルクロードを通じて中国から西方に伝わったものと、西方から中国に伝わったものを、それぞれ3つ以上挙げなさい。
→ 中国から西方へ：絹、紙、火薬、陶磁器、茶。西方から中国へ：ガラス製品、宝石、葡萄、仏教、音楽・楽器。

(3) 漢の武帝が張騫を西域に派遣した目的は何であったか。また、その結果どのようなことが起きたか述べなさい。
→ 目的は匈奴（きょうど）を挟み撃ちにするため、西方の大月氏（だいげっし）と同盟を結ぶことであった。同盟は実現しなかったが、張騫の報告によって西域の情報がもたらされ、シルクロードを通じた東西交易が活発化するきっかけとなった。

(4) シルクロードによる東西交流が世界史に与えた影響を、「文化」「宗教」「技術」の3つの観点から論じなさい。
→ 文化：東西の芸術・音楽・食文化が互いに影響し合い、ヘレニズム文化の東伝やガンダーラ美術の成立などが起きた。宗教：仏教がインドから中国・東アジアへ伝播し、広大な地域に信仰が広まった。技術：製紙法や養蚕技術が西方に伝わり、西方のガラス製造技術が東方に伝わるなど技術交流が進んだ。

## 問4: 漢の統治と日本への影響

(1) 漢の時代に儒教が国の学問として採用された理由を、秦の政治と比較して説明しなさい。
→ 秦は法律と刑罰による厳しい統治（法家思想）で民の反感を買い滅亡した。漢はその教訓から、人の道徳心に訴える儒教を採用し、仁と礼を基盤とした穏やかな統治を目指した。

(2) 漢の時代に中国と日本との間にはどのような交流があったか。具体的な史実を1つ挙げて説明しなさい。
→ 後漢の時代に倭の奴国の王が使いを送り、皇帝から金印（「漢委奴国王」と刻まれた印）を授けられた。これは日本と中国の外交関係を示す最古の史料の一つである。

(3) 中国文明が日本を含む東アジアに与えた影響を、「文字」「思想」「政治制度」の3つの観点から述べなさい。
→ 文字：漢字が日本・朝鮮半島・ベトナムなどに伝わり、各地で独自に発展した（日本ではひらがな・カタカナが生まれた）。思想：儒教の教えが東アジア各国の道徳や社会規範に大きな影響を与えた。政治制度：中国の律令制度が日本や朝鮮半島にも導入され、中央集権的な国家体制のモデルとなった。

## 問5: 総合論述

(1) 殷・周・秦・漢の各時代の特徴を比較し、中国文明がどのように発展したか、「統治方法の変化」に注目して論じなさい。
→ 殷は神権政治（占いに基づく統治）、周は封建制度（地方に土地を分封）、秦は法家思想による厳格な法治（始皇帝の中央集権）、漢は儒教に基づく徳治へと変化した。つまり中国の統治方法は、神の意志→地方分権→法律による中央集権→道徳による統治と段階的に発展した。

(2) 始皇帝とアレクサンドロス大王はともに大帝国を築いたが、その帝国はいずれも短命だった。二人の共通点と相違点を述べなさい。
→ 共通点：広大な領土を征服・統一したこと、死後まもなく帝国が崩壊したこと。相違点：始皇帝は文字・度量衡の統一など制度面での統一を重視したのに対し、アレクサンドロスはギリシャ文化の東方伝播など文化面の融合を重視した。

(3) 「論語」に「学びて時にこれを習う、亦た説ばしからずや」（学んだことを時々復習するのは、なんと嬉しいことではないか）という言葉がある。この言葉が2500年以上経った現在でも人々に親しまれている理由を考えなさい。
→ 学ぶことの喜びは時代が変わっても普遍的なものであり、知識を身につけ成長する楽しさは人間の本質的な欲求だからである。また、繰り返し学ぶことの大切さは現代の学習法にも通じ、孔子の教えが時代を超えて有効であることを示している。

(4) シルクロードは「古代のグローバル化」と言える。現代のグローバル化と比較して、共通点と相違点を述べなさい。
→ 共通点：異なる文化・文明の間で物資や情報が行き交い、相互に影響し合った点。相違点：シルクロードは陸路中心で移動に数か月かかったが、現代はインターネットや航空機により瞬時に世界とつながることができる。またシルクロードは主に物資と文化の交流であったが、現代のグローバル化は経済・政治・情報など多方面にわたる。
