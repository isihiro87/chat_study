# 古墳時代と大和政権 発展問題

## 問1: 前方後円墳の分布と大和政権の支配

前方後円墳は3世紀後半から東北地方南部から九州地方にかけて各地に造られた。初期の巨大な前方後円墳は奈良盆地に集中しているが、5世紀になると大阪平野に巨大古墳が造られるようになった。各地の豪族の墓にも前方後円墳の形式が採用されたが、大和政権の大王の古墳より小さいものが多い。

(1) 前方後円墳が各地に広がっていることから、大和政権と地方の豪族の関係についてどのようなことが推測できるか。
→ 地方の豪族は大和政権に従属しつつ、前方後円墳の形式を許可されることで大和政権との結びつきを示した。前方後円墳は大和政権の政治的な連合関係を象徴するものだったと考えられる。

(2) 各地の豪族の古墳が大王の古墳より小さい傾向にある理由を考えなさい。
→ 古墳の大きさは権力の大きさを反映するため、地方の豪族は大王より低い地位にあることを示す必要があった。古墳の大きさに序列をつけることで、大和政権内の身分秩序が表されていたと考えられる。

(3) 5世紀に巨大古墳の造営地が奈良盆地から大阪平野に移った背景として、どのような社会変化が考えられるか。
→ 大和政権の中で有力な豪族の勢力関係が変化し、大阪平野を拠点とする有力者が台頭したことが考えられる。また大阪平野は港に近く、朝鮮半島との交流の拠点として重要性が増したことも背景にあると考えられる。

(4) 7世紀になると前方後円墳が造られなくなった。その理由を推測しなさい。
→ 仏教の浸透や大化の改新による政治制度の変革により、権力の象徴としての巨大古墳の造営は衰退し、代わりに寺院の建立が権力を示す手段へと変わっていったため。

## 問2: 稲荷山古墳の鉄剣と大和政権の支配構造

埼玉県の稲荷山古墳から出土した鉄剣には、115文字の銘文が刻まれていた。銘文には「ワカタケル大王に仕えた」という内容が記されており、刀の持ち主の一族が代々大王に仕えてきたことが書かれていた。また、熊本県の江田船山古墳からも同じ大王の名を含む銘文が見つかった。

(1) 鉄剣に銘文を刻むことにはどのような目的があったと考えられるか。
→ 大王との結びつきを証明し、自分の地位や権威を子孫に伝えるため。大王に忠誠を誓った証として残す意図もあったと考えられる。

(2) 関東（埼玉県）と九州（熊本県）の両方からワカタケル大王に関する出土品が見つかったことは、当時の政治体制について何を示しているか。
→ 5世紀後半には、大和政権の支配が関東から九州まで広がっており、各地の豪族が大王を中心とした政治体制に組み込まれていたことを示す。

(3) 銘文に「一族が代々大王に仕えてきた」と記されていることから、大和政権の支配構造についてどのようなことがわかるか。
→ 大和政権は大王を頂点として各地の豪族が代々仕えるという世襲的な従属関係で成り立っており、長期的・安定的な支配体制が構築されていたことがわかる。

## 問3: 渡来人がもたらした変化

次の表は、渡来人が伝えた主な技術・文化をまとめたものである。

| 伝えたもの | 内容 |
|:-:|:-:|
| 須恵器 | 高温で焼く硬い土器の製作技術 |
| 漢字 | 文字による記録の技術 |
| 仏教 | インドに始まる宗教 |
| 儒教 | 礼儀や道徳に関する教え |
| 機織り | 布を織る技術 |
| 土木技術 | 灌漑やため池の建設技術 |

(1) 漢字の伝来が日本の政治にどのような影響を与えたか説明しなさい。
→ 文字による記録が可能になり、法律や行政文書を作成できるようになった。これにより、口伝えだけに頼らない組織的な政治運営が可能になり、国家としての統治体制が整備される基盤となった。

(2) 仏教の伝来が日本の社会に与えた影響を、政治面と文化面の両方から説明しなさい。
→ 政治面では、仏教の受容をめぐって有力豪族の間で争いが起き（蘇我氏と物部氏の対立）、のちの政治体制に影響を与えた。文化面では、寺院建築・仏像彫刻などの新しい芸術が生まれ、日本文化に大きな影響を及ぼした。

(3) 渡来人が伝えた土木技術（灌漑やため池の建設）は、日本の農業にどのような影響を与えたか。
→ 灌漑設備やため池の建設により、安定した水の供給が可能になった。これにより水田の開発が進み、米の生産量が増加して、大和政権の経済的基盤が強化された。

(4) 渡来人が果たした役割を総合的に評価し、日本の「国づくり」にどのように貢献したか論じなさい。
→ 渡来人は先進的な技術と文化を日本に伝え、国家形成に不可欠な要素を提供した。漢字は行政・記録を可能にし、仏教と儒教は精神的・道徳的な基盤を作った。須恵器や機織りは生活水準を向上させ、土木技術は農業生産力を高めた。こうした技術・文化の伝来により、日本は部族社会から組織的な国家へと発展する基礎を築くことができた。

## 問5: 古墳時代の東アジア情勢

古墳時代の朝鮮半島には高句麗・百済・新羅の3国と、南部に伽耶地域があった。大和政権は百済と友好関係を結び、伽耶地域からは鉄の原料を輸入していた。5世紀には倭の五王が中国の宋に使いを送った。

(1) 大和政権が伽耶地域から鉄の原料を輸入した理由と、その鉄の用途を説明しなさい。
→ 当時の日本列島では鉄の資源が乏しかったため、朝鮮半島南部の伽耶地域から輸入する必要があった。鉄は武器・農具・工具の材料として不可欠なものであった。

(2) 倭の五王が中国の宋に使いを送った目的を、朝鮮半島の情勢と関連づけて説明しなさい。
→ 朝鮮半島では高句麗が勢力を拡大しており、百済や伽耶地域との関係を維持するため、中国の皇帝から「倭王」として認められることで国際的な立場を強めようとした。

(3) 大和政権が百済と友好関係を結んだ理由を、東アジアの国際関係の中で説明しなさい。
→ 百済は北方の高句麗の圧力を受けており、大和政権の軍事的支援を必要としていた。一方、大和政権も朝鮮半島での鉄の確保や先進文化の入手のために百済との協力が不可欠だった。両者は共通の利害のもとに同盟関係を結んだ。

(4) 古墳時代の東アジアの国際関係を「鉄」を中心に整理し、大和政権の外交戦略について論じなさい。
→ 鉄は武器や農具の材料として国力の根幹を支える資源であった。大和政権は伽耶地域から鉄を輸入するルートを確保するため、百済と同盟を結んで高句麗に対抗した。さらに中国の宋に朝貢して国際的な権威を得ることで、朝鮮半島での立場を有利にしようとした。このように大和政権は鉄の確保を軸に、東アジアの国際関係の中で巧みな外交戦略を展開していた。
