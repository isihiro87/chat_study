# 旧石器時代と縄文時代 大問小問

## 大問1: 旧石器時代の暮らし（配点 18点）

数万年前、地球全体の気温が大きく下がった時代があった。この時代の日本列島について、以下の問いに答えなさい。

(1) この時代を何というか。
→ 氷河時代（氷期）

(2) この時代、日本列島と大陸はどのような状態だったか。
→ 陸続きだった

(3) 大陸から渡ってきた大型動物を2つ答えなさい。
→ ナウマンゾウ・オオツノジカ

(4) この時代の人々が使っていた、石を打ち欠いて作った石器を何というか。
→ 打製石器

(5) 群馬県で打製石器が発見され、日本にも旧石器時代があったことを証明した遺跡の名前を答えなさい。
→ 岩宿遺跡

(6) この遺跡を発見した人物の名前を答えなさい。
→ 相沢忠洋

## 大問2: 縄文時代の生活（配点 22点）

氷河時代が終わり気候が温暖になると、人々の暮らしは大きく変わった。以下の問いに答えなさい。

(1) 約1万数千年前に始まり、独特の文様がある土器を使い始めた時代を何というか。
→ 縄文時代

(2) この時代の名前の由来となった土器を何というか。また、その表面の特徴を答えなさい。
→ 縄文土器。表面に縄目の文様がある。

(3) 縄文人が地面を掘り下げて柱を立て、屋根をかけた住居を何というか。
→ たて穴住居

(4) 食べ終わった貝殻や魚の骨などが捨てられた場所を何というか。
→ 貝塚

(5) 貝塚が歴史研究で重要とされる理由を説明しなさい。
→ 当時の人々の食生活や暮らしの様子を知る手がかりになるから。

(6) 祈りや魔よけのために作られたと考えられている土製の人形を何というか。
→ 土偶

## 大問3: 縄文時代の遺跡と文化（配点 20点）

次の文章を読んで、以下の問いに答えなさい。

青森県にある遺跡Aは、縄文時代の大規模な集落跡で、大型の建物や多くの土器が発見された。また、この遺跡からは遠方の地域で産出される石Bが発見されており、当時すでに遠くの地域との交流があったことがわかっている。

(1) 遺跡Aの名前を答えなさい。
→ 三内丸山遺跡

(2) 石Bとして考えられる、旧石器時代から石器の材料に使われた黒いガラス質の石の名前を答えなさい。
→ 黒曜石

(3) 遠方の産地の石が発見されることから、縄文時代の人々の生活についてどのようなことがわかるか。
→ 遠く離れた地域どうしで物の交換（交易）が行われていたこと。

(4) 縄文時代の集落で、たて穴住居が円形に並んでいたのはなぜだと考えられているか。
→ 集落の中心に広場を設け、共同生活を営むため。

(5) 縄文時代に死者の手足を折り曲げて埋葬する方法を何というか。
→ 屈葬

## 大問4: 旧石器時代から縄文時代への変化（配点 20点）

旧石器時代から縄文時代への移り変わりについて、以下の問いに答えなさい。

(1) 旧石器時代の人々はどのような生活をしていたか、簡潔に答えなさい。
→ 獲物を追って移動しながらの狩猟・採集生活。

(2) 縄文時代に人々が定住するようになったきっかけとなる環境の変化は何か。
→ 氷河時代が終わり気候が温暖になったこと。

(3) 気候が温暖になると、海面はどのように変化したか。
→ 海面が上昇した。

(4) 海面の上昇により日本列島はどうなったか。
→ 大陸から離れて島になった。

(5) 旧石器時代に使われた石器と、縄文時代に新たに加わった石器をそれぞれ答えなさい。
→ 旧石器時代：打製石器、縄文時代：磨製石器

(6) 縄文時代に土器が作られるようになったことで、食生活にどのような変化があったか。
→ 食べ物を煮炊きできるようになり、これまで食べられなかった食材も調理できるようになった。

## 大問5: 旧石器時代と縄文時代の比較（配点 20点）

旧石器時代と縄文時代について、次の表の空欄を埋めなさい。

| 項目 | 旧石器時代 | 縄文時代 |
|:-:|:-:|:-:|
| 石器 | (1) | (2) |
| 土器 | (3) | (4) |
| 住居 | (5) | (6) |
| 生活 | 移動生活 | 定住生活 |

(1) → 打製石器
(2) → 打製石器＋磨製石器
(3) → なし
(4) → 縄文土器
(5) → 一定の住居なし（洞穴・岩陰など）
(6) → たて穴住居
