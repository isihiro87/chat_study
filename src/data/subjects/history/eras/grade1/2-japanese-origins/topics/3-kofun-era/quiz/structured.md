# 古墳時代と大和政権 大問小問

## 大問1: 古墳と大和政権（配点 20点）

3世紀後半から、日本各地に巨大な墓が造られるようになった。以下の問いに答えなさい。

(1) この時代を何というか。
→ 古墳時代

(2) 奈良盆地を中心に成立した強大な勢力を何というか。
→ 大和政権（ヤマト王権）

(3) 大和政権の頂点に立つ王を何と呼んだか。
→ 大王（おおきみ）

(4) 円形と方形を組み合わせた日本独自の墓を何というか。
→ 前方後円墳

(5) 大阪府堺市にある日本最大級の前方後円墳の名前を答えなさい。
→ 大仙古墳（大山古墳）

(6) 前方後円墳が東北地方から九州地方まで広く分布していることから、大和政権についてどのようなことがわかるか。
→ 大和政権の勢力（影響力）が日本各地に及んでいたこと。

## 大問2: 古墳の副葬品と埴輪（配点 18点）

古墳からはさまざまな副葬品が出土している。以下の問いに答えなさい。

(1) 古墳の表面や周囲に並べられた人や動物の形の焼き物を何というか。
→ 埴輪

(2) 埴輪にはどのような種類があるか。3つ答えなさい。
→ 人物・馬・家（犬・鳥・船なども可）

(3) 古墳に埴輪を並べた目的は何だと考えられているか。
→ 死者の魂を守り、あの世での生活に役立てるため。

(4) 古墳から出土する代表的な副葬品を3つ答えなさい。
→ 鉄剣・銅鏡・勾玉（甲冑・装身具なども可）

(5) 古墳が大きいほど何を示すと考えられているか。
→ 被葬者（墓に埋葬された人物）の権力の大きさ。

## 大問3: ワカタケル大王と大和政権の広がり（配点 22点）

次の資料を読んで、以下の問いに答えなさい。

【資料】埼玉県の稲荷山古墳から鉄剣が出土し、そこにはある大王の名前が刻まれていた。また、熊本県の江田船山古墳からも同じ大王の名前が刻まれた鉄刀が出土した。

(1) 鉄剣に刻まれていた大王の名前を答えなさい。
→ ワカタケル大王

(2) ワカタケル大王は後の何天皇にあたるとされるか。
→ 雄略天皇

(3) 埼玉県（関東）と熊本県（九州）の両方から同じ大王の名が刻まれた出土品が見つかったことから、何がわかるか。
→ 大和政権の勢力が関東から九州まで広範囲に及んでいたこと。

(4) 5世紀ごろ、中国の南朝の宋に使いを送った倭の5人の王を何というか。
→ 倭の五王

(5) 倭の五王の活動が記録されている中国の歴史書は何か。
→ 宋書

(6) 倭の王が中国に使いを送った目的を説明しなさい。
→ 中国の皇帝から倭王として認めてもらい、朝鮮半島での立場を有利にするため。

## 大問4: 渡来人と大陸文化（配点 22点）

大陸や朝鮮半島から多くの人が日本に移り住み、さまざまな技術や文化を伝えた。以下の問いに答えなさい。

(1) 大陸から移り住み技術や文化を伝えた人々を何というか。
→ 渡来人

(2) 渡来人が伝えた高温で焼く硬い土器を何というか。
→ 須恵器

(3) 渡来人が伝えた、記録に使う文字は何か。
→ 漢字

(4) 6世紀半ばに渡来人が伝えた宗教は何か。
→ 仏教

(5) 渡来人は主にどの方面から日本に来たか。
→ 朝鮮半島（百済・伽耶など）

(6) 須恵器がそれまでの土器と比べてすぐれていた点を答えなさい。
→ 高温で焼くため硬くて丈夫であること。

(7) 渡来人が日本に与えた影響を総合的に説明しなさい。
→ 須恵器の製作・漢字・仏教・儒教・機織り・土木技術など先進的な技術と文化を伝え、日本の国づくりと文化の発展に大きく貢献した。

## 大問5: 朝鮮半島との関係（配点 18点）

古墳時代の朝鮮半島との関係について、以下の問いに答えなさい。

(1) 古墳時代に朝鮮半島にあった3つの国を答えなさい。
→ 高句麗・百済・新羅

(2) 大和政権が鉄の原料を輸入していた朝鮮半島南部の地域を何というか。
→ 伽耶地域（任那）

(3) 大和政権が朝鮮半島に進出した主な目的は何か。
→ 鉄の資源を確保するため。

(4) 大和政権は朝鮮半島のどの国と友好関係を結んでいたか。
→ 百済

(5) 朝鮮半島との交流が日本にもたらした文化的な影響を2つ答えなさい。
→ 渡来人による先進技術の伝来（須恵器・機織りなど）、仏教や漢字の伝来。
