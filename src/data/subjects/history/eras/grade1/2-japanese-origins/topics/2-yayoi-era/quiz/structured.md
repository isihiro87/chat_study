# 弥生時代 大問小問

## 大問1: 稲作の伝来と暮らしの変化（配点 20点）

紀元前4世紀ごろ、大陸から日本列島に新しい農業技術が伝わり、人々の生活は大きく変わった。以下の問いに答えなさい。

(1) 大陸から九州北部に伝わった農業技術は何か。
→ 稲作

(2) 収穫した稲を保管するために建てられた建物を何というか。
→ 高床倉庫

(3) (2)の建物の床を高くした理由を2つ答えなさい。
→ 湿気を防ぐため、ねずみの害を防ぐため

(4) 稲作が広まったことで、人々の間にどのような変化が起きたか。
→ 富を多く蓄える人とそうでない人の間に貧富の差が生まれ、身分の違いができた。

(5) 弥生時代の土器は縄文土器と比べてどのような特徴があるか。
→ 薄手で硬く、シンプルな形をしている。

(6) 稲の穂をつみ取るために使われた半月形の石器を何というか。
→ 石包丁

## 大問2: 金属器の伝来（配点 18点）

弥生時代には、大陸から金属器も伝わった。以下の問いに答えなさい。

(1) 大陸から伝わった金属器を大きく2種類に分けると何と何か。
→ 青銅器と鉄器

(2) 武器や農具など実用的に使われたのは、青銅器と鉄器のどちらか。
→ 鉄器

(3) 祭りの宝物として使われたつりがね形の青銅器を何というか。
→ 銅鐸

(4) 祭りや儀式に使われた青銅製の鏡を何というか。
→ 銅鏡

(5) 鉄器が実用品に使われ、青銅器が祭器に使われた理由をそれぞれ説明しなさい。
→ 鉄は硬くて丈夫なため農具や武器に適し、青銅は加工しやすく光沢があるため祭りの道具に適していたから。

## 大問3: 金印と中国との交流（配点 20点）

次の資料を読んで、以下の問いに答えなさい。

【資料】1世紀ごろ、倭の奴国の王が中国の漢の皇帝に使いを送り、ある物を授かった。この物は江戸時代に福岡県の志賀島で発見された。

(1) 奴国の王が漢の皇帝から授かったものは何か。
→ 金印

(2) 金印に刻まれていた文字を答えなさい。
→ 漢委奴国王

(3) 奴国は現在のどの地域にあったとされるか。
→ 福岡県博多付近

(4) 奴国の王が中国の皇帝に使いを送った目的は何か。
→ 中国の皇帝に認めてもらい、自分の権威を高めるため。

(5) 「漢書地理志」によると、1世紀ごろの倭にはいくつの「くに」があったか。
→ 100あまり

## 大問4: 邪馬台国と卑弥呼（配点 22点）

3世紀ごろの日本について、以下の問いに答えなさい。

(1) 邪馬台国を中心に30あまりの国をまとめた女王は誰か。
→ 卑弥呼

(2) 卑弥呼はどのような力で人々をまとめたとされるか。
→ まじない（呪術）の力

(3) 卑弥呼が使いを送った中国の国はどこか。
→ 魏

(4) 卑弥呼が魏の皇帝から授かった称号を答えなさい。
→ 親魏倭王

(5) 卑弥呼のことが記されている中国の歴史書を何というか。
→ 魏志倭人伝

(6) 邪馬台国の場所については2つの有力な説がある。それぞれ答えなさい。
→ 畿内説（近畿地方）と九州説

(7) 卑弥呼が中国に使いを送り朝貢した目的を説明しなさい。
→ 中国の皇帝から称号を得て、自分の立場を他の国々に対して強化するため。

## 大問5: 弥生時代の遺跡（配点 20点）

弥生時代の代表的な遺跡について、以下の問いに答えなさい。

(1) 佐賀県にある、堀や柵を備えた大規模な弥生時代の遺跡の名前を答えなさい。
→ 吉野ヶ里遺跡

(2) 静岡県にある、弥生時代の水田跡が見つかった遺跡の名前を答えなさい。
→ 登呂遺跡

(3) 吉野ヶ里遺跡に堀（環濠）や柵が設けられていた理由を答えなさい。
→ 村同士の争いに備えるため。

(4) 吉野ヶ里遺跡で物見やぐらが建てられていた目的を答えなさい。
→ 敵の侵入を監視するため。

(5) 弥生時代に争いが増えた原因を、稲作との関連から説明しなさい。
→ 稲作が広まると土地や水の確保が重要になり、それをめぐって村同士の争いが起きるようになったため。
