# 院政と平氏の政治 大問小問

## 大問1: 院政のしくみ（配点 20点）

次の文を読んで、あとの問いに答えなさい。

（ ア ）天皇が荘園の増加をおさえる政策を行ったのち、（ イ ）上皇が退位した天皇として初めて政治の実権を握った。この政治のしくみを（ ウ ）という。上皇は院の警備に（ エ ）を置き、武士の力を利用した。

(1) ア〜エにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：後三条、イ：白河、ウ：院政、エ：北面の武士

(2) 院政が始まる前に行われていた藤原氏中心の政治を何というか。（2点）
→ 摂関政治。

(3) 上皇が退位してからも政治の実権を握ることができた理由を説明しなさい。（4点）
→ 天皇の位にいるときの形式的な制約から解放され、上皇として自由に政治を行うことができたため。

(4) 院政期に社会が不安定になった要因を1つ挙げなさい。（4点）
→ 大きな寺社が僧兵を武装させて強訴を行うなど、武力を背景にした要求が増えたこと。

## 大問2: 保元の乱と平治の乱（配点 20点）

次の年表を見て、あとの問いに答えなさい。

| 年 | できごと |
|------|---------|
| 1156年 | （ ア ） |
| 1159年 | （ イ ） |
| 1167年 | 平清盛が（ ウ ）に就任 |

(1) ア〜ウにあてはまる語句を答えなさい。（各2点）
→ ア：保元の乱、イ：平治の乱、ウ：太政大臣

(2) 保元の乱で対立した2人の人物を答えなさい。（4点）
→ 崇徳上皇と後白河天皇。

(3) 平治の乱で勝利した人物と敗れた人物を答えなさい。（4点）
→ 勝利：平清盛、敗北：源義朝。

(4) 保元の乱と平治の乱が武士の歴史に与えた影響を説明しなさい。（4点）
→ 朝廷の内部争いに武士が動員されたことで、武士の力が政治を左右するようになり、平清盛が武士の頂点に立つきっかけとなった。

(5) 平治の乱で敗れた源義朝の子で、のちに平氏を滅ぼす人物の名前を答えなさい。（2点）
→ 源頼朝。

## 大問3: 平清盛の政治と日宋貿易（配点 20点）

次の問いに答えなさい。

(1) 平清盛が就任した官職名と、それが武士にとって初めてであった理由を答えなさい。（4点）
→ 太政大臣。それまで太政大臣は貴族のみが就任する官職であり、武士が就任したのは清盛が初めてであった。

(2) 平清盛が整備した港の名前と、その場所を答えなさい。（4点）
→ 大輪田泊。現在の兵庫県神戸市にある。

(3) 平清盛が推進した貿易の名称を答えなさい。（2点）
→ 日宋貿易。

(4) 平清盛が一族の権力を広げるために用いた方法を2つ答えなさい。（4点）
→ 娘を天皇のきさきにしたこと、一族を政府の要職に就けたこと。

(5) 平清盛の政治の方法は、かつての藤原氏のどのような方法と似ているか説明しなさい。（4点）
→ 藤原氏の摂関政治と同じように、娘を天皇に嫁がせて外戚となり、一族で朝廷の要職を占めるという方法。

(6) 日宋貿易が平氏にもたらした利点を答えなさい。（2点）
→ 貿易による莫大な利益が平氏の経済基盤を支えた。

## 大問4: 平氏の滅亡（配点 20点）

次の問いに答えなさい。

(1) 平氏の政治に不満が高まった理由を2つ答えなさい。（4点）
→ 一族で朝廷の要職を独占したこと、他の武士や貴族の声を無視したこと。

(2) 平氏に対して挙兵した人物と、その本拠地を答えなさい。（4点）
→ 源頼朝。本拠地は鎌倉（神奈川県）。

(3) 1185年に平氏が滅亡した戦いの名前と場所を答えなさい。（4点）
→ 壇ノ浦の戦い。現在の山口県下関市。

(4) 壇ノ浦の戦いで源氏の軍を率いた人物はだれか。（2点）
→ 源義経。

(5) 院政の開始から平氏の滅亡までの流れを、主な出来事を4つ以上含めて時系列で説明しなさい。（6点）
→ 白河上皇が院政を開始 → 1156年の保元の乱で武士が政争に動員される → 1159年の平治の乱で平清盛が勝利 → 1167年に平清盛が太政大臣に就任 → 平氏の横暴に対して源頼朝が挙兵 → 1185年の壇ノ浦の戦いで平氏が滅亡。

## 大問5: 院政期の社会と政治の変化（配点 20点）

次の問いに答えなさい。

(1) 院政期に武士が重用された理由を「朝廷内部の争い」と「武力」のキーワードを使って説明しなさい。（4点）
→ 朝廷内部の争いが激化し、争いを解決するために武力を持つ武士の力が不可欠となったため。

(2) 北面の武士が設置された目的を説明しなさい。（4点）
→ 上皇（院）の身辺を警護し、院の権力を武力で支えるため。

(3) 平清盛の政治が摂関政治と異なる点を1つ挙げなさい。（4点）
→ 平清盛は武力を背景に権力を握った点が異なる。藤原氏は婚姻関係のみで権力を維持したが、清盛は武力と経済力（日宋貿易）も権力の基盤とした。

(4) 「院政→保元の乱・平治の乱→平氏の政権→平氏の滅亡」という流れが、日本の政治体制にどのような変化をもたらしたか説明しなさい。（4点）
→ 貴族中心の政治から武士中心の政治へと移行する過程となった。この流れの結果、源頼朝が鎌倉幕府を開き、武士が政権を握る新しい時代が始まった。

(5) 平氏が約20年で滅亡した要因を複数の観点から分析しなさい。（4点）
→ 一族で要職を独占して広範な反感を買ったこと、武士の支持基盤が西日本に偏っていたこと、源氏が東国武士の支持を得て大規模な挙兵を行ったことが複合的に作用した。
