# 鎌倉幕府の成立 大問小問

## 大問1: 鎌倉幕府のしくみ（配点 20点）

次の文を読んで、あとの問いに答えなさい。

1185年、（ ア ）は国ごとに（ イ ）を、荘園や公領ごとに（ ウ ）を置くことを朝廷に認めさせた。（ イ ）は軍事・警察を担当し、（ ウ ）は年貢の取り立てなどを行った。1192年には（ ア ）が（ エ ）に任命された。

(1) ア〜エにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：源頼朝、イ：守護、ウ：地頭、エ：征夷大将軍

(2) 鎌倉幕府の3つの主要な役所の名前と、それぞれの役割を答えなさい。（6点）
→ 侍所（御家人の統率）、政所（政務・財政）、問注所（裁判）。

(3) 将軍に忠誠をちかった武士を何というか。（2点）
→ 御家人。

(4) 源頼朝が鎌倉を本拠地に選んだ理由を地理的特徴から答えなさい。（4点）
→ 三方を山に囲まれ一方が海に面する天然の要塞で、防衛に適していたため。

## 大問2: 御恩と奉公（配点 20点）

次の図を見て、あとの問いに答えなさい。

【図】将軍 ←→ 御家人
将軍→御家人：（ A ）  御家人→将軍：（ B ）

(1) Aにあてはまる語句を答え、その内容を具体的に説明しなさい。（4点）
→ 御恩。将軍が御家人の先祖伝来の領地を認めたり（本領安堵）、新たな土地をあたえたり（新恩給与）すること。

(2) Bにあてはまる語句を答え、その内容を具体的に説明しなさい。（4点）
→ 奉公。御家人が将軍のために戦場で戦ったり（軍役）、京都や鎌倉の警護（京都大番役・鎌倉番役）を行ったりすること。

(3) 将軍と御家人の関係を成り立たせていたものは何か。（2点）
→ 土地（領地）を仲立ちとした主従関係。

(4) このような土地を仲立ちとした主従関係を基盤とする社会制度を何というか。（2点）
→ 封建制度。

(5) 御恩と奉公の関係がうまく機能しなくなるのはどのような場合か、考えて答えなさい。（4点）
→ 戦いで手柄を立てても与える土地が不足した場合。あたらしい領地がもらえないと、御家人の不満が高まり奉公の意欲が低下する。

(6) 鎌倉時代の御恩と奉公の関係と、現代の雇用関係を比較して類似点を1つ答えなさい。（4点）
→ 将軍が御恩として領地を与え、御家人が奉公で軍役を果たすのは、現代の会社が給料を支払い、社員が労働を提供する関係に似ている。

## 大問3: 北条氏の執権政治（配点 20点）

次の問いに答えなさい。

(1) 源頼朝の死後、幕府の実権を握った一族は何氏か。（2点）
→ 北条氏。

(2) 幕府で将軍を補佐し実権を握った役職を何というか。（2点）
→ 執権。

(3) 初代執権の名前を答えなさい。（2点）
→ 北条時政。

(4) 3代将軍の源実朝が暗殺された結果、どのような変化があったか。（4点）
→ 源氏の将軍は3代で途絶え、以後は京都の貴族や皇族が名目上の将軍に迎えられ、北条氏が執権として幕府の実権を握った。

(5) 北条政子と幕府との関係を説明しなさい。（4点）
→ 北条政子は源頼朝の妻であり、北条氏出身。頼朝の死後は「尼将軍」と呼ばれ、承久の乱では御家人に頼朝の恩を説いて団結を呼びかけ、幕府存続に大きな役割を果たした。

(6) 北条氏が執権として実権を握った期間はいつからいつまでか、おおよそ答えなさい。（2点）
→ 13世紀初めから1333年の鎌倉幕府滅亡まで。約130年間。

(7) 執権政治のしくみを簡潔に説明しなさい。（4点）
→ 将軍は名目上の最高権力者として置かれ、実際の政治は執権である北条氏が行った。合議制によって政策が決定されることもあった。

## 大問4: 承久の乱（配点 20点）

次の問いに答えなさい。

(1) 1221年に幕府をたおそうとした上皇の名前を答えなさい。（2点）
→ 後鳥羽上皇。

(2) この事件の名前を答えなさい。（2点）
→ 承久の乱。

(3) 承久の乱で御家人に団結を呼びかけた人物とその内容を説明しなさい。（4点）
→ 北条政子。「頼朝公の御恩は山よりも高く海よりも深い」と説き、御家人たちに幕府への忠誠を訴えた。

(4) 承久の乱の結果を答えなさい。（2点）
→ 幕府軍が勝利し、後鳥羽上皇は隠岐に流された。

(5) 承久の乱のあと、京都に設置された機関名と、その役割を答えなさい。（4点）
→ 六波羅探題。京都の朝廷を監視し、西国の御家人を統轄する役割。

(6) 承久の乱が幕府の支配に与えた影響を説明しなさい。（6点）
→ 朝廷に対する幕府の優位が確立した。上皇方の貴族や武士の領地が没収され、新たに御家人が地頭として西国にも配置された。これにより幕府の支配が西日本にも広がり、全国的な武家政権となった。

## 大問5: 御成敗式目（配点 20点）

次の問いに答えなさい。

(1) 1232年に定められた武士のための法律の名前を答えなさい。（2点）
→ 御成敗式目（貞永式目）。

(2) この法律を定めた人物と役職を答えなさい。（4点）
→ 北条泰時。第3代執権。

(3) 御成敗式目は何を基準にして定められたか。（2点）
→ 武士の慣習や先例。

(4) 御成敗式目の目的を説明しなさい。（4点）
→ 武士の間の土地争いなどの裁判基準を明確にし、公正な判断を行うこと。

(5) 御成敗式目は全部で何か条からなるか。（2点）
→ 51か条。

(6) 御成敗式目が制定された歴史的意義を説明しなさい。（6点）
→ 武士独自の法体系が初めて確立され、裁判が慣習や力ではなく成文法に基づいて行われるようになった。これにより御家人の権利が保障され、幕府の統治が安定化した。後の室町幕府や戦国大名の法にも影響を与えた。
