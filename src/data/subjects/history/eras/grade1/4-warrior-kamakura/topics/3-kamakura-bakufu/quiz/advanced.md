# 鎌倉幕府の成立 発展問題

## 問1: 鎌倉幕府の統治構造

(1) 源頼朝が守護と地頭を全国に置いたことの歴史的意義を、「朝廷の支配」「武士の支配」の対比から論述しなさい。
→ それまで全国の統治は朝廷が国司を通じて行っていた。頼朝が守護・地頭を朝廷に認めさせたことで、朝廷の支配体制の上に武士の支配体制が重なることになった。これにより二重の統治構造が生まれ、実質的な権力は幕府の守護・地頭が握るようになった。中央集権的な律令国家から、武士を媒介とする封建的な支配構造への転換点となった。

(2) 鎌倉幕府の3つの役所（侍所・政所・問注所）の分業体制が、当時としては画期的であった理由を、律令制との比較から説明しなさい。
→ 律令制では朝廷が一元的に政治・軍事・司法を管轄していたのに対し、鎌倉幕府は軍事（侍所）・行政（政所）・司法（問注所）を分離して設置した。これにより各分野の専門性が高まり、効率的な統治が可能になった。特に問注所による独立した裁判機関の設置は、公正な紛争解決を目指す先進的な試みであった。

(3) 「1185年」と「1192年」のどちらを鎌倉幕府の成立年とすべきか、両方の根拠を示した上で自分の考えを述べなさい。
→ 1185年説の根拠は、この年に頼朝が守護・地頭の設置を認められ、全国的な支配体制を確立したこと。1192年説の根拠は、頼朝が征夷大将軍に任命され、名実ともに武家の長となったこと。現在の歴史学では、実質的な支配権が確立した1185年を重視する見方が有力である。征夷大将軍の称号は象徴的な意味が強く、実際の権力は1185年の守護・地頭設置により確立していたと考えられる。

## 問2: 御恩と奉公の封建関係

(1) 御恩と奉公の関係を西欧の封建制度と比較し、類似点と相違点を述べなさい。
→ 類似点として、どちらも土地を仲立ちとした主従関係であり、主君が土地を与え（御恩／封土）、家臣が軍事奉仕を行う（奉公／軍役）構造を持つ。相違点として、日本では血縁関係が重視され一族が一体となって主君に仕えたのに対し、西欧では個人と主君の契約的関係が強かった。また日本の封建制は天皇・朝廷という別の権威が並存した点が特徴的である。

(2) 御恩と奉公の関係が元寇後に崩れ始めた理由を推測し、論述しなさい。
→ 元寇は外国からの侵略であり、御家人は命がけで戦った。しかし勝利しても敵国の土地を獲得したわけではないため、幕府には御家人に与える新しい領地がなかった。御恩なき奉公を強いられた御家人の不満が高まり、御恩と奉公のバランスが崩れた。これが鎌倉幕府衰退の重要な要因となった。

(3) 御恩と奉公の制度における「本領安堵」と「新恩給与」の違いを説明し、それぞれが御家人にとってどのような意味を持ったか論じなさい。
→ 本領安堵は御家人の先祖伝来の領地を将軍が公式に認めることで、土地の所有権が法的に保障される安心感を与えた。新恩給与は戦功に対して新たな領地を与えることで、御家人の忠誠心と戦意を高める報酬としての意味を持った。本領安堵は現状維持の保障であり、新恩給与は上昇志向の動機づけとして、両者は相互補完的に封建制度を機能させた。

## 問3: 承久の乱の歴史的転換

(1) 北条政子の演説が御家人を動かした理由を、当時の武士の価値観から分析しなさい。
→ 当時の武士は主君への恩義を重んじる文化を持っていた。政子は「頼朝公の御恩は山より高く海より深い」と説き、頼朝が御家人に土地を与えて生活を保障した御恩を想起させた。御恩に報いることが武士の名誉であるという価値観に訴えることで、朝廷への畏れを上回る忠誠心を引き出した。

(2) 承久の乱は日本史上初めて「武士が朝廷に勝利した事件」である。このことの歴史的意義を、「権威」と「権力」の関係から論じなさい。
→ 承久の乱以前は、天皇・朝廷の権威は絶対的なものとされ、武士も朝廷の権威のもとで活動していた。しかし承久の乱で武力（権力）が権威を上回る結果となった。これにより「権威は朝廷に、権力は幕府に」という二元的な統治構造が確立した。天皇の権威は形式的に維持されつつも、実質的な政治権力は幕府が握る体制が以後数百年にわたって続くことになった。

(3) 承久の乱後の六波羅探題の設置と、西国への地頭配置が日本の政治地図をどのように変えたか論述しなさい。
→ 六波羅探題の設置により幕府の監視網が京都に直接及び、朝廷の独自行動が制限された。西国への新たな地頭配置は、それまで東日本中心だった幕府の支配を西日本にも拡大した。これにより幕府は名実ともに全国政権となり、東国武士が西国の荘園を支配する構図が生まれた。

## 問4: 御成敗式目の法制史的意義

(1) 御成敗式目が「武士の慣習」に基づいて定められたことの意義を、律令との比較から論述しなさい。
→ 律令は中国の法制度を模倣した法律であり、日本の実情に合わない部分も多かった。御成敗式目は日本の武士社会の実態に即した慣習法を成文化したもので、現実の紛争解決に直結する実用的な法律であった。これは外来の法体系から日本独自の法体系への転換を意味し、法が社会の実態を反映するべきという法思想の成熟を示している。

(2) 御成敗式目が後の日本の法制度に与えた影響を、室町時代・戦国時代の法との関連から論じなさい。
→ 御成敗式目は後の武家法の基礎となった。室町幕府は御成敗式目を基本法として引き継ぎ、追加法令を加えて運用した。戦国時代の分国法（今川仮名目録、武田信玄の甲州法度之次第など）も、御成敗式目の影響を受けている。武士のための独自の法という概念は、御成敗式目によって確立され、その後の武家社会の法的基盤となった。

(3) 御成敗式目の制定が御家人の権利保障にどのように寄与したか、「裁判の公正性」「領地の保護」の観点から分析しなさい。
→ 裁判の公正性の面では、成文法により裁判基準が明確化され、恣意的な判断が排除された。権力者の意向ではなく法に基づく裁判が行われるようになったことは画期的であった。領地の保護の面では、土地の所有権や相続に関する規定が明文化されたことで、御家人の領地が不当に奪われることを防ぐ制度的保障が確立された。

## 問5: 総合問題

(1) 鎌倉幕府の成立を「日本の政治体制の転換」として捉えたとき、それ以前の政治体制との最大の違いは何か、複数の観点から総合的に論述しなさい。
→ 最大の違いは、政治の担い手が貴族から武士に変わったことである。支配の基盤が血筋や家柄から武力と土地に変化し、政治の中心が京都から鎌倉に移った。朝廷が形式的な権威を保ちつつ、実質的な権力は幕府が握るという二元的統治構造が初めて成立した。また、御恩と奉公という主従関係に基づく封建制度が政治の基本原理となり、法の支配（御成敗式目）が導入されるなど、統治の原理そのものが変革された。

(2) 「源頼朝 → 北条時政 → 北条政子 → 北条泰時」という権力の移り変わりから、鎌倉幕府初期の政治的変遷をまとめなさい。
→ 源頼朝は鎌倉幕府を創設し、守護・地頭の設置により全国的な武家政権を確立した。頼朝の死後、舅の北条時政が初代執権として幕府の実権を握った。北条政子は頼朝の妻として承久の乱で御家人の団結を呼びかけ、幕府の危機を救った。北条泰時は第3代執権として御成敗式目を制定し、法に基づく安定的な統治体制を確立した。源氏から北条氏への権力移行は、個人的カリスマから制度的統治への転換を示している。
