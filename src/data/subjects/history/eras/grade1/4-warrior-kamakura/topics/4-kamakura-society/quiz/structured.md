# 鎌倉時代の社会 大問小問

## 大問1: 農業の発展（配点 20点）

次の文を読んで、あとの問いに答えなさい。

鎌倉時代には農業技術が進歩した。西日本を中心に（ ア ）が広まり、同じ土地で年に（ イ ）回、異なる作物を栽培できるようになった。また（ ウ ）を使って田を耕したり、草木を焼いた灰を（ エ ）にしたりする工夫も広がった。

(1) ア〜エにあてはまる語句をそれぞれ答えなさい。（各2点）
→ ア：二毛作、イ：2、ウ：牛や馬（牛馬）、エ：肥料

(2) 二毛作で主に栽培された2種類の作物を答えなさい。（2点）
→ 米と麦。

(3) 二毛作が主に西日本で広まった理由を答えなさい。（4点）
→ 西日本は気候が温暖で、米を収穫した後に麦を栽培するのに十分な生育期間が確保できたため。

(4) 鎌倉時代に普及した鉄製農具の利点を答えなさい。（4点）
→ 木製や石製の農具に比べて硬くて丈夫であり、土を深く耕すことができたため、耕作の効率が大幅に向上した。

## 大問2: 商業と経済の発展（配点 20点）

次の問いに答えなさい。

(1) 鎌倉時代に寺社の門前や交通の要所で定期的に開かれた市を何というか。（2点）
→ 定期市。

(2) 定期市は一般的に月に何回開かれたか。また、それは何と呼ばれたか。（4点）
→ 月に3回。三斎市と呼ばれた。

(3) 日宋貿易で日本に大量に流入した貨幣を何というか。（2点）
→ 宋銭。

(4) 宋銭の流通により日本の経済にどのような変化が起きたか、「物々交換」の語を使って説明しなさい。（4点）
→ それまでの物々交換中心の経済から、宋銭を使った貨幣経済へと変化した。

(5) 鎌倉時代に馬を使って荷物を運ぶ職業を何というか。（2点）
→ 馬借。

(6) 農業の発展と商業の発展はどのように関係しているか説明しなさい。（6点）
→ 二毛作の普及などにより農業の生産力が向上し、自家消費を超える余剰作物が生まれた。農民はその余剰を定期市で売って貨幣を得るようになり、商業活動が活発化した。また宋銭の流通により、取引が容易になったことも商業発展を後押しした。

## 大問3: 武士の暮らし（配点 20点）

次の問いに答えなさい。

(1) 鎌倉時代の武士の住まいはどのような特徴を持っていたか。（4点）
→ 質素な板葺きの住まいで、周囲に堀をめぐらし、戦いに備えた構造であった。

(2) 鎌倉時代の武士が重んじた武芸の道を何というか。（2点）
→ 弓馬の道。

(3) 鎌倉時代の武士がふだんの生活で行っていたことを2つ答えなさい。（4点）
→ 自分の領地での農業の指導と、弓や馬の武芸の訓練。

(4) 「いざ鎌倉」という言葉の意味を説明しなさい。（4点）
→ 戦いや非常事態が起きた際に、将軍のいる鎌倉へすぐに駆けつけるという御家人の覚悟と義務を表した言葉。

(5) 鎌倉時代の武士が質素な生活を送った理由を考えて答えなさい。（6点）
→ 武士は常に戦いに備える必要があり、武芸の鍛錬を最優先としていた。贅沢な生活は武士としての精神を鈍らせると考えられており、名誉と忠誠を重んじる武士道の精神から質素な暮らしが美徳とされた。

## 大問4: 地頭と荘園領主の対立（配点 20点）

次の問いに答えなさい。

(1) 鎌倉時代に地頭と荘園領主の間で対立が起きた原因を説明しなさい。（4点）
→ 地頭が荘園の管理を任されるうちに、年貢の横領や農民への支配を強めるなど、荘園領主の権利を侵害するようになったため。

(2) 地頭と荘園領主が土地を半分ずつ分けることを何というか。（2点）
→ 下地中分。

(3) 下地中分が行われた理由を説明しなさい。（4点）
→ 地頭と荘園領主の土地争いが裁判でも解決しにくく、土地を分割して双方の支配権を明確にする妥協策がとられた。

(4) 地頭と荘園領主の争いを裁判で解決する役割を担った幕府の機関を答えなさい。（2点）
→ 問注所。

(5) 地頭の勢力拡大が日本の土地制度に与えた影響を説明しなさい。（4点）
→ 地頭が実質的な土地支配者となることで、荘園制度がしだいに崩壊していった。これは後の守護大名による土地支配へとつながった。

(6) 荘園領主の立場から見た下地中分の意味を説明しなさい。（4点）
→ 荘園の半分を失うことになるが、武力を持つ地頭に全てを奪われるよりは、残り半分の支配権を確保できる現実的な妥協策であった。

## 大問5: 鎌倉時代の社会変化（配点 20点）

次の問いに答えなさい。

(1) 鎌倉時代の農業・商業・武士の生活の変化を、それぞれ1つずつキーワードを挙げて説明しなさい。（6点）
→ 農業：二毛作の普及により生産力が向上。商業：定期市の開催と宋銭の流通により貨幣経済が発展。武士：弓馬の道を重んじ、質素な生活を送りながら領地を管理。

(2) 鎌倉時代に宋銭が広く使われるようになった背景を、「日宋貿易」「貨幣」の語を使って説明しなさい。（4点）
→ 平清盛以来の日宋貿易により中国から大量の宋銭が輸入された。日本では独自の貨幣の鋳造が少なかったため、宋銭が広く流通し、取引の手段として定着した。

(3) 鎌倉時代の社会変化を「農業→商業→経済」の流れで整理して説明しなさい。（6点）
→ 二毛作や鉄製農具の普及により農業生産力が向上した。余剰作物が生まれ、定期市で売買されるようになった。宋銭の流通とあいまって貨幣経済が発達し、物々交換中心の経済から貨幣を使った経済活動へと社会全体が変化していった。

(4) 鎌倉時代の社会の変化が、後の室町時代に与えた影響を推測して答えなさい。（4点）
→ 農業技術の向上と貨幣経済の発達は室町時代にさらに進展し、定期市は月6回の六斎市に発展した。また商工業者の同業組合（座）が成長し、都市が発達する基盤となった。
