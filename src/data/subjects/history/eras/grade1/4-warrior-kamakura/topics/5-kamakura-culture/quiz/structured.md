# 鎌倉文化と新しい仏教 大問小問

## 大問1: 鎌倉文化の特徴（配点 20点）

次の問いに答えなさい。

(1) 鎌倉時代の文化の特徴を、平安時代の文化と比較して説明しなさい。（4点）
→ 平安時代の文化が貴族的で優美な作風であったのに対し、鎌倉文化は武士の気風を反映した力強く写実的な作風が特徴である。

(2) 東大寺南大門にある鎌倉文化を代表する彫刻の名前を答えなさい。（2点）
→ 金剛力士像。

(3) 金剛力士像を作った2人の代表的な仏師の名前を答えなさい。（4点）
→ 運慶と快慶。

(4) 金剛力士像の作風の特徴を「写実的」「力強い」のキーワードを使って説明しなさい。（4点）
→ 写実的な筋肉の表現と力強い動きの表現が特徴で、武士の時代にふさわしい迫力ある作品である。

(5) 鎌倉文化が「力強い」と評される理由を、当時の社会背景と結びつけて説明しなさい。（6点）
→ 武士が政治の中心となった時代であり、武士の気質や価値観が文化にも反映された。質実剛健を重んじる武士の精神が、写実的で迫力のある美術や、戦いを題材とする文学を生み出した。

| （ ウ ） | （ オ ） | 随筆 |

→ ア：平家物語、イ：方丈記、ウ：徒然草、エ：軍記物語、オ：兼好法師、カ：後鳥羽上皇

(2) 平家物語の内容を簡潔に説明しなさい。（4点）

## 大問3: 鎌倉新仏教（念仏・題目系）（配点 20点）

次の表を完成させ、あとの問いに答えなさい。

| 宗派 | 開祖 | 教えの特徴 |
|------|------|-----------|
| 浄土宗 | （ ア ） | （ エ ）を唱えれば救われる |
| （ イ ） | 親鸞 | 阿弥陀仏の救いを信じる心 |
| 時宗 | （ ウ ） | （ オ ）で念仏を広める |
| 日蓮宗 | 日蓮 | （ カ ）の題目を唱える |

(1) ア〜カにあてはまる語句を答えなさい。（各2点）
→ ア：法然、イ：浄土真宗、ウ：一遍、エ：念仏（南無阿弥陀仏）、オ：踊り念仏、カ：法華経

(2) 法然と親鸞の関係を答えなさい。（2点）
→ 法然が師匠、親鸞が弟子の師弟関係。

(3) 鎌倉新仏教が平安仏教と異なり、庶民に広まった理由を説明しなさい。（4点）
→ 念仏や題目を唱えるだけで救われるという簡潔でわかりやすい教えであり、難しい修行や学問を必要としなかったため。

(4) 日蓮宗で唱える題目を答えなさい。（2点）
→ 「南無妙法蓮華経」。

## 大問4: 鎌倉新仏教（禅宗）（配点 20点）

次の問いに答えなさい。

(1) 座禅で悟りを開こうとする仏教の宗派をまとめて何というか。（2点）
→ 禅宗。

(2) 禅宗の2つの宗派と、それぞれの開祖を答えなさい。（各2点、計4点）
→ 臨済宗（栄西）と曹洞宗（道元）。

(3) 臨済宗と曹洞宗の教えの違いを説明しなさい。（4点）
→ 臨済宗は座禅に加えて問答（公案）を重視し、曹洞宗はひたすら座禅（只管打坐）を行うことを重視した。

(4) 禅宗が武士に好まれた理由を説明しなさい。（4点）
→ 座禅で精神を集中させ、自らを厳しく鍛えるという教えが、武芸を磨き名誉を重んじる武士の生き方や精神に合っていたため。

(5) 鎌倉新仏教の6宗派を、「念仏系」「題目系」「座禅系」に分類しなさい。（6点）
→ 念仏系：浄土宗（法然）、浄土真宗（親鸞）、時宗（一遍）。題目系：日蓮宗（日蓮）。座禅系：臨済宗（栄西）、曹洞宗（道元）。

## 大問5: 鎌倉文化の総合問題（配点 20点）

次の問いに答えなさい。

(1) 鎌倉時代の美術・文学・宗教をそれぞれ1つずつ代表作品または人物を挙げて特徴を説明しなさい。（6点）
→ 美術：金剛力士像（運慶・快慶が制作した写実的で力強い彫刻）。文学：平家物語（琵琶法師が語り広めた平氏の栄枯盛衰を描く軍記物語）。宗教：法然（念仏を唱えれば救われるという浄土宗を開き、庶民に仏教を広めた）。

(2) 鎌倉新仏教が「新しい」と呼ばれる理由を、平安時代の仏教と比較して説明しなさい。（6点）
→ 平安時代の仏教（天台宗・真言宗）は難しい経典の学習や厳しい修行が必要で、主に貴族や僧侶のためのものであった。鎌倉新仏教は「念仏を唱えるだけ」「題目を唱えるだけ」「座禅をするだけ」という簡単な実践で誰でも救われると説き、庶民に門戸を開いた点が「新しい」とされる。

(3) 鎌倉文化と鎌倉新仏教が生まれた社会的背景を「武士の台頭」「民衆の苦しみ」「戦乱」のキーワードを使って説明しなさい。（4点）
→ 武士の台頭により力強い文化が生まれた。戦乱が続く時代に人々は不安を感じ、民衆の苦しみを救う新しい仏教への需要が高まった。

(4) 鎌倉時代の文化が後の日本文化に与えた影響を1つ挙げて説明しなさい。（4点）
→ 禅宗は武家社会だけでなく、枯山水庭園や茶道、書院造りなど後の日本文化の多くの分野に影響を与えた。禅の精神は日本の美意識の根幹となっている。
