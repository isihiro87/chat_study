# 関東の自然と東京 構造問題

## 大問1 関東地方の自然環境（配点20）

(1) 関東平野は日本最大の平野である。この平野の台地を覆う赤土の層の名称を答えよ。（2点）
→ 関東ローム層

(2) 関東ローム層は何が堆積してできたか。（2点）
→ 火山灰

(3) 関東ローム層が畑作に利用される理由を土壌の性質に触れて述べよ。（3点）
→ 火山灰が堆積した土壌で水はけがよく、水田には不向きだが畑作に適しているから。

(4) 日本最大の流域面積をもつ河川名を答えよ。（2点）
→ 利根川

(5) 冬に関東地方に吹く冷たい北西の季節風を何というか。（2点）
→ からっ風

(6) ヒートアイランド現象の原因と影響を説明せよ。（4点）
→ アスファルトやコンクリートの蓄熱やエアコンの排熱で都市部の気温が郊外より高くなる。猛暑日の増加やゲリラ豪雨の原因にもなっている。

(7) 関東地方の南部が冬でも比較的温暖な理由を述べよ。（3点）
→ 南部には黒潮（暖流）が流れており、その影響で温暖な気候となっている。

(8) 関東ローム層とシラス台地の共通点を1つ答えよ。（2点）
→ どちらも火山灰が堆積してできた土壌で水はけがよい。

## 大問2 世界都市・東京と一極集中（配点20）

(1) 東京は世界都市と呼ばれる。世界都市とはどのような都市か説明せよ。（3点）
→ 政治・経済・文化の国際的な中心機能をもち、世界に大きな影響力を持つ都市。

(2) 東京に集中している国の三権の機関をそれぞれ答えよ。（3点）
→ 国会（立法）、中央省庁（行政）、最高裁判所（司法）

(3) 東京にある2つの国際空港の名称を答えよ。（2点）
→ 羽田空港と成田空港

(4) 東京に外国大使館や外国企業の本社が集まる理由を述べよ。（3点）
→ 日本の政治・経済の中心地であり、国際的な情報発信やビジネス活動に有利な環境があるため。

(5) 東京に金融・情報通信・マスメディアが集中している理由を述べよ。（3点）
→ 政治の中心で最新の情報が集まりやすく、企業や消費者が多いため経済活動にも有利だから。

(6) 一極集中がもたらす問題点を2つ挙げて説明せよ。（4点）
→ ①地方から人口が流出し過疎化が進む。②首都直下型地震が起きた場合に国の機能が一度に失われるリスクがある。

(7) 一極集中の対策として考えられることを1つ述べよ。（2点）
→ 政府機関や企業の地方移転を進め、機能を分散させること。

## 大問3 東京大都市圏（配点20）

(1) 東京大都市圏に含まれる東京以外の3県を答えよ。（3点）
→ 神奈川県、埼玉県、千葉県

(2) 東京大都市圏の人口は約何万人で、日本の人口の約何分の1か。（2点）
→ 約3700万人で、日本の人口の約3分の1

(3) 東京都心で昼間人口が夜間人口を大幅に上回る理由を述べよ。（3点）
→ 周辺の神奈川・埼玉・千葉から多くの人が都心に通勤・通学するため。

(4) 筑波研究学園都市が茨城県に建設された目的を述べよ。（3点）
→ 東京への一極集中を緩和し、研究機関を集約して研究環境を整備するため。

(5) 東京大都市圏で第三次産業が約8割を占める理由を述べよ。（4点）
→ 大都市には多くの消費者と企業が集まるため、サービス業・商業・情報通信業などの第三次産業の需要が大きいから。

(6) ニュータウンが現在抱えている課題を2つ挙げよ。（3点）
→ 住民の高齢化と建物の老朽化

(7) 幕張メッセや東京ディズニーリゾートが千葉県に立地する利点を述べよ。（2点）
→ 東京に近く交通の便がよい一方で、広い用地を確保できるため。
