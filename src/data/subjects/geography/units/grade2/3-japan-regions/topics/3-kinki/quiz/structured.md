# 近畿地方 構造問題

## 大問1 近畿地方の自然環境（配点20）

(1) 若狭湾や志摩半島に見られる複雑な海岸地形を何というか。（2点）
→ リアス海岸

(2) 日本最大の湖の名前と所在する府県を答えよ。（2点）
→ 琵琶湖、滋賀県

(3) 琵琶湖から大阪湾に流れる川の名前を答えよ。（2点）
→ 淀川

(4) 近畿地方の北部で冬に降水量が多い理由を季節風に触れて説明せよ。（4点）
→ 北西の季節風が日本海の水蒸気を含んで吹きつけるため、冬に雪や雨が多くなる。

(5) 近畿地方の中央部の盆地で気温の寒暖差が大きい理由を説明せよ。（4点）
→ 周囲を山地に囲まれているため海の影響を受けにくく、夏は暑く冬は冷え込み、昼と夜の寒暖差も大きくなる。

(6) 近畿地方の南部が温暖な理由を海流に触れて説明せよ。（4点）
→ 南部は黒潮（暖流）の影響を受けるため、冬でも比較的温暖な気候となっている。

(7) 紀伊山地で盛んな林業の代表的な木材を答えよ。（2点）
→ 吉野杉

## 大問2 大阪大都市圏と阪神工業地帯（配点20）

(1) 大阪を中心とする大都市圏の人口は約何万人か。（2点）
→ 約880万人

(2) 大阪湾臨海部に広がる工業地帯の名称を答えよ。（2点）
→ 阪神工業地帯

(3) 阪神工業地帯の特徴を企業の規模と技術力の面から説明せよ。（4点）
→ 中小企業が多く集まっており、高い技術力を持っていることが特徴である。

(4) 東大阪市が「ものづくりの町」と呼ばれる理由を具体例を挙げて述べよ。（4点）
→ 中小企業の町工場が密集しており、協力して人工衛星「まいど1号」を開発するなど高い技術力を持っているから。

(5) 堺市の臨海部の工業の特徴を東大阪市と比較して述べよ。（4点）
→ 東大阪市は中小企業が密集しているのに対し、堺市の臨海部には大型工場が立地している。

(6) 阪神工業地帯が広がる範囲を答えよ。（2点）
→ 大阪から神戸にかけて

(7) 大阪が西日本で果たしている経済的な役割を述べよ。（2点）
→ 西日本最大の経済の中心地として、商業・金融・情報の機能が集まっている。

## 大問3 ニュータウンと都市再開発（配点20）

(1) 1960年代に大阪周辺の郊外にニュータウンが建設された理由を述べよ。（3点）
→ 大阪市への人口集中が進み、住宅不足を解消するために郊外にニュータウンが建設された。

(2) ポートアイランドはどのように造られたか説明せよ。（4点）
→ 神戸市の丘陵を削った土で海を埋め立てて造った人工島である。

(3) 大阪府にある代表的なニュータウンを2つ答えよ。（2点）
→ 千里ニュータウン、泉北ニュータウン

(4) ニュータウンが現在抱えている問題を2つ挙げ、それぞれの内容を説明せよ。（4点）
→ ①老朽化：建設から50年以上経過し建物が古くなっている。②高齢化：若い世代が転出し住民の高齢化が進んでいる。

(5) 大阪市中心部の梅田で進められている取り組みは何か。（2点）
→ 再開発

(6) 京都市の景観条例で規制されている内容を2つ答えよ。（3点）
→ 建物の高さ制限と看板の色・大きさの規制

(7) オーバーツーリズムとは何か説明せよ。（2点）
→ 観光客が増えすぎて交通渋滞やごみ問題など住民生活に悪影響が出る現象。

## 大問4 農山村の過疎化と地域おこし（配点20）

(1) 近畿地方の山間部・沿岸部で進んでいる2つの問題を答えよ。（2点）
→ 過疎化と高齢化

(2) 奈良県川上村の林業が衰退した理由を説明せよ。（4点）
→ 安い外国産木材が大量に輸入されるようになり、国産の吉野杉の需要が減少して林業が衰退した。

(3) 京都府和束町が地域ブランドとして売り出している茶の名称を答えよ。（2点）
→ 和束茶

(4) 和束町の地域振興の取り組みを景観に触れて説明せよ。（4点）
→ 宇治茶の産地として茶畑の景観を観光資源に活用し、地域ブランド「和束茶」として独自のまちづくりを進めている。

(5) 京都府伊根町の舟屋とはどのような建物か説明せよ。（3点）
→ 1階が船の格納庫、2階が住居となっている海に面した伝統的な建物。

(6) 伊根町が舟屋をどのように活用して地域振興しているか述べよ。（3点）
→ 舟屋を民宿やカフェに改装し、伝統的な景観を観光資源として活用している。

(7) 和束町と伊根町の地域振興の共通点を述べよ。（2点）
→ どちらも地域の伝統的な景観や文化を観光資源として活用して過疎化対策に取り組んでいる。

## 大問5 交通と災害（配点20）

(1) 1995年に近畿地方で発生した大地震の名称を答えよ。（2点）
→ 阪神・淡路大震災

(2) 本州と淡路島を結ぶ世界最長級のつり橋の名称を答えよ。（2点）
→ 明石海峡大橋

(3) 関西国際空港が造られた場所とその理由を説明せよ。（4点）
→ 大阪湾の人工島に造られた。騒音問題を避けるためと24時間運用を可能にするため。

(4) 阪神・淡路大震災が近畿地方の都市に与えた影響を述べよ。（4点）
→ 多くの建物が倒壊し交通網が寸断された。震災の教訓から耐震基準の見直しや防災対策が強化された。

(5) ポートアイランドが震災時に被害を受けた原因となった現象は何か。（3点）
→ 液状化現象（埋立地であるため地震時に地盤が液状化した）

(6) 近畿地方の交通網の発達が地域経済に与える影響を述べよ。（5点）
→ 明石海峡大橋や新幹線・高速道路の整備により、大阪を中心に神戸・京都・奈良の大都市圏が一体的に機能し、人や物の移動が活発になって経済圏が広がっている。
