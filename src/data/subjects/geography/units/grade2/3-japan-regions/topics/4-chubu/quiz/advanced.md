# 中部地方 発展問題

## 発展1 日本アルプスと3地域の気候・産業（配点20）

(1) 日本アルプスが中部地方の気候を3つに分ける仕組みを、季節風に触れて説明せよ。（5点）
→ 冬の北西季節風は日本海から水分を含んで北陸に大量の雪を降らせ、山脈を越えると乾燥した風となり東海側は晴天が多い。山脈に挟まれた中央高地は海から遠く標高も高いため寒暖差の大きい内陸性気候となる。

(2) 濃尾平野の輪中が形成された歴史的背景を、木曽三川の特徴に触れて論じよ。（5点）
→ 木曽川・長良川・揖斐川の3本の大河川が合流する低湿地帯では繰り返し洪水が発生した。住民は生活を守るため集落の周囲を堤防で囲む輪中を形成し、水屋や上げ舟などの水防設備を備えて洪水に備えた。

(3) 東海・中央高地・北陸の産業が気候にどのように規定されているか、3地域を比較して論じよ。（10点）
→ 東海は温暖多雨の太平洋側気候で茶の栽培や温室メロンが可能となり、港湾を活かした工業も発達した。中央高地は冷涼な内陸性気候により高原野菜の抑制栽培や寒暖差を活かした果樹栽培が盛んになった。北陸は冬の豪雪が稲作の農閑期を生み、副業として伝統的工芸品が発展する契機となった。各地域の気候特性が農業形態や産業構造を大きく方向づけている。

## 発展2 東海の工業と農業の発展（配点20）

(1) 中京工業地帯が日本最大の製造品出荷額を維持できる理由を、産業集積の観点から論じよ。（6点）
→ トヨタ自動車を頂点に部品メーカーが周辺に多数集積し、効率的なサプライチェーンが構築されている。自動車の完成車だけでなく部品・素材の関連工場も含めた裾野の広い産業集積が出荷額を押し上げている。

(2) 四日市の石油化学コンビナートの立地条件と、かつて発生した公害について説明せよ。（6点）
→ 四日市は港湾に面し原料の石油を海外から輸入しやすい立地にある。高度経済成長期に工場の排煙による大気汚染が深刻化し、四日市ぜんそくとよばれる公害病が発生した。四大公害病の一つとして知られ、環境規制の強化につながった。

(3) 愛知用水の建設が東海地方の農業と工業に与えた影響を多面的に論じよ。（8点）
→ 木曽川の水を知多半島に導く愛知用水の完成により、水不足だった地域に安定した水の供給が可能になった。農業面では畑作や園芸農業が発展し、工業面では工業用水の確保により臨海部の工業化が進んだ。生活用水も含め地域の総合的な発展を支えるインフラとなっている。

## 発展3 中央高地の農業と産業転換（配点20）

(1) 甲府盆地の扇状地で果樹栽培が盛んな理由を、地形・気候・土壌の3つの面から説明せよ。（6点）
→ 扇状地は水はけがよく果樹の根腐れが起きにくい。盆地は日照時間が長く昼夜の寒暖差が大きいため果実の糖度が高まる。砂礫質の土壌は水分を適度に制限し、果樹に凝縮した味わいを生む条件が整っている。

(2) 抑制栽培が市場で有利な理由を、促成栽培と比較しながら経済的観点から論じよ。（6点）
→ 抑制栽培は冷涼な高原で夏に高原野菜を出荷し、平地の産地が暑さで生産しにくい時期に供給できるため高値で取引される。促成栽培は温暖な気候で冬に出荷し同様に競合が少ない。両者とも他産地との出荷時期をずらすことで価格優位性を得る手法だが、利用する気候条件が正反対である。

(3) 諏訪地方が養蚕業から精密機械工業、さらにコンピューター関連産業へと転換した過程を、産業構造の変化として論じよ。（8点）
→ 戦前は養蚕業が主力だったが、化学繊維の普及で衰退。戦後は養蚕で培った精密な手作業の技術と、空気がきれいで水が豊富な環境を活かし時計・カメラなどの精密機械工業が発達し「東洋のスイス」とよばれた。さらに技術革新に伴い電子部品やコンピューター関連産業へ転換し、一貫して「精密さ」を核に産業を高度化させてきた。

## 発展4 北陸の稲作と伝統産業の持続可能性（配点20）

(1) 新潟県が米の生産量日本一である理由を、地形・気候・水資源の面から説明せよ。（6点）
→ 越後平野は日本有数の広大な平野で大規模な水田経営が可能。信濃川などの河川が豊富な水を供給する。夏の高温と十分な降水が稲の生育に適しており、これらの条件が揃うことで日本一の生産量を誇る。

(2) 北陸の伝統的工芸品が冬の副業から発展した経緯を、具体的な工芸品の例を3つ挙げて論じよ。（6点）
→ 冬の豪雪で農作業ができない農閑期に収入を得るための副業として手工業が始まった。石川県の輪島塗は木地に漆を何層も塗り重ねる精緻な技法が磨かれた。九谷焼は鮮やかな色彩で知られる陶磁器に発展した。福井県鯖江市のメガネフレームは農家の副業から始まり現在は国内シェア約96%を占める一大産業に成長した。

(3) 富山市のコンパクトシティ政策がなぜ必要とされているか、少子高齢化と環境問題の観点から考察せよ。（8点）
→ 少子高齢化により車を運転できない高齢者が増加し、郊外に広がった都市では移動困難者が増える。LRTなど公共交通を整備し都市機能を中心部に集約することで、高齢者も暮らしやすいまちになる。また車依存の軽減はCO2排出削減にもつながり、環境負荷の低い持続可能な都市モデルとして注目されている。

## 発展5 中部地方の総合考察（配点20）

(1) 中部地方が「日本の東西を結ぶ要衝」である理由を地理的位置と交通網から論じよ。（6点）
→ 中部地方は本州の中央に位置し、東海道新幹線・東名高速道路が東海地方を東西に貫いている。名古屋は東京と大阪の中間にあり物流と交通の結節点として機能する。北陸新幹線の開通で日本海側との連携も強化され、日本の東西・太平洋側と日本海側を結ぶ交通の要衝となっている。

(2) 東海の工業・中央高地の農業・北陸の伝統産業が、それぞれグローバル化の中でどのような課題と可能性を持つか考察せよ。（8点）
→ 東海の自動車産業はEV化への対応が課題だが技術力を活かした転換が可能。中央高地の高原野菜はブランド化と輸出拡大の可能性がある一方、担い手不足が課題。北陸の伝統工芸品は海外での日本文化への関心を背景にインバウンド需要や輸出の可能性がある反面、後継者不足が深刻で技術継承が課題となっている。

(3) 中部地方の3地域が互いに補完し合う関係にあることを、産業連関の観点から具体例を挙げて論じよ。（6点）
→ 東海の工業地帯は中央高地や北陸からの労働力供給を受け、北陸の米や中央高地の農産物は東海の都市部の消費地に供給される。また東海の工業製品は北陸の港から輸出される場合もあり、交通網を通じて3地域は相互に依存し補完し合う関係にある。
