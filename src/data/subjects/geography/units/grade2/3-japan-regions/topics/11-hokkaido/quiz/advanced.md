# 北海道地方 発展問題

## 発展1 北海道の農業と土地利用（配点20）

(1) 石狩平野の泥炭地が稲作に不向きであった理由と、客土による改良の過程を詳しく説明せよ。（5点）
→ 泥炭地は植物の遺体が分解されずに堆積した酸性の土壌で水はけが悪く、養分も乏しいため稲作に適さなかった。周辺の山地や台地から良質な土を運び入れる客土を行い、排水路の整備と合わせて土壌を改良した。

(2) 十勝平野の輪作が土壌管理に果たす役割を、連作障害と関連づけて論じよ。（5点）
→ 同じ作物を連続して栽培すると特定の養分が消耗し病害虫が増える連作障害が起きる。輪作ではじゃがいも・小麦・てんさい・豆類を順に栽培することで養分消耗を分散させ、豆類の根粒菌による窒素固定で土壌を回復させている。

(3) 北海道の大規模農業と本州の中小規模農業を比較し、それぞれの長所と課題を述べよ。（5点）
→ 北海道は広大な耕地と大型機械で効率的な生産が可能だが、広い土地の維持管理費が大きく後継者確保が課題。本州は少量多品目生産やブランド化で高付加価値を実現できるが、耕地が狭く機械化が難しい面がある。

(4) 北海道の酪農が日本の乳製品供給に果たす役割と、今後の課題を考察せよ。（5点）
→ 北海道は日本の生乳生産量の半分以上を占め、バター・チーズなど乳製品の安定供給に不可欠である。飼料価格の高騰、酪農家の減少、TPPなどによる海外製品との競争激化が課題で、生産性向上とブランド化による差別化が求められている。

## 発展2 アイヌ文化と多文化共生（配点20）

(1) 明治時代以降のアイヌの人々への政策の変遷を時代の流れに沿って説明せよ。（6点）
→ 明治政府は北海道旧土人保護法（1899年）を制定しアイヌを「保護」の名の下に同化政策を推進、土地や漁猟権を奪い文化を抑圧した。1997年にアイヌ文化振興法が制定され文化復興が始まり、2019年のアイヌ施策推進法でアイヌが先住民族と法的に認められ、2020年にウポポイが開設された。

(2) アイヌの自然観が現代の環境保全にどのような示唆を与えるか論じよ。（5点）
→ アイヌは動植物や自然現象を「カムイ（神）」として敬い、自然から必要な分だけを恵みとして受け取る持続可能な暮らしを営んでいた。この自然と共生する考え方は、現代の持続可能な開発や生物多様性保全の理念と共通し、環境問題を考える上で重要な視点を提供する。

(3) ウポポイの設立が日本の多文化共生社会にとって持つ意義を考察せよ。（5点）
→ アイヌ文化を国立施設で紹介することは、先住民族の歴史と文化を国民全体で学ぶ機会を提供する。多様な文化的背景を持つ人々の存在を認め尊重する社会の基盤づくりに貢献し、国際的にも先住民族の権利回復への取り組みとして評価される。

(4) アイヌ語の地名が北海道に多く残っている理由と、その意義を述べよ。（4点）
→ 北海道の地名の多くはアイヌ語に由来し、札幌（サッ・ポロ＝乾いた大きな川）や知床（シレトク＝大地の突端）などがある。これらの地名はアイヌの人々が自然をどう認識していたかを伝える文化遺産であり、保存することでアイヌ文化の記憶を未来に継承できる。

## 発展3 観光とエコツーリズム（配点20）

(1) 知床の世界自然遺産登録の基準となった生態系の特徴を、流氷と関連づけて説明せよ。（5点）
→ オホーツク海の流氷が溶ける際に海中に栄養分を放出し、プランクトンが大量発生する。これを起点に魚類→海鳥・ヒグマへと続く食物連鎖が形成され、海洋と陸上の生態系が流氷を介してつながる世界的にも稀有な環境が評価された。

(2) エコツーリズムと従来の観光開発の違いを、環境への影響と経済効果の面から比較せよ。（5点）
→ 従来の観光開発はホテルや道路の大規模建設で環境を改変し短期的な経済効果を追求する。エコツーリズムは自然環境を保全しつつ少人数のガイドツアーなどで質の高い体験を提供し、入場制限や環境保全費用の徴収で持続可能な経済効果を目指す。

(3) 北海道の「食のブランド化」戦略の成功要因を分析し、他の地域への応用可能性を論じよ。（5点）
→ 北海道ブランドの成功は、広大な自然環境と「安全・安心・おいしい」というイメージの確立による。産地の管理体制や品質基準の統一、ストーリー性のある情報発信が差別化につながった。他地域でも産地の強みを明確にしたブランド戦略と品質管理の徹底で応用が可能である。

(4) 地球温暖化が北海道の観光・農業に与える影響をプラス面とマイナス面から考察せよ。（5点）
→ プラス面：気温上昇により稲作適地が拡大し新たな作物栽培の可能性が広がる。冬の観光シーズンの延長も期待できる。マイナス面：流氷の減少で知床の生態系や冬の観光資源が損なわれる。高温による農作物の品質低下、降雪量変化による酪農への影響も懸念される。

## 発展4 北海道の産業構造と未来（配点20）

(1) 北海道の一次産業（農業・漁業）が二次・三次産業に与える波及効果を具体例を挙げて述べよ。（5点）
→ 十勝産小麦が製粉業やパン製造業を支え、生乳が乳製品工場の原料となる（二次産業）。農産物直売所やファームレストラン、観光牧場などが観光客を呼び込み飲食・宿泊業を活性化する（三次産業）。

(2) 6次産業化が北海道の農漁業にもたらす可能性を論じよ。（5点）
→ 農漁業者が加工・販売まで手がけることで利益率が向上する。十勝のチーズ工房やオホーツクの海産物加工品など、付加価値の高い商品開発が進み、後継者にとって魅力ある産業への転換が期待できる。

(3) 北海道と沖縄を、気候・農業・観光の3点で比較し、それぞれの強みを述べよ。（5点）
→ 北海道は亜寒帯で大規模畑作・酪農が強み、沖縄は亜熱帯でさとうきびやパイナップル。観光は北海道が自然景観・グルメ・ウィンタースポーツ、沖縄はビーチ・サンゴ礁・独自の文化。いずれも本州とは異なる気候を最大の強みとして産業を発展させている。

(4) 北海道の人口減少に対して、農業・漁業・観光の連携で地域を活性化する方策を提案せよ。（5点）
→ スマート農業やICT漁業による省人化で生産性を維持しつつ、農泊・漁業体験・食のツーリズムで観光客を呼び込む。テレワーク移住者に農漁業の副業機会を提供し、関係人口を増やすことで地域コミュニティを維持する方策が有効である。

## 発展5 地図・資料活用（配点20）

(1) 石狩平野・十勝平野・根釧台地の位置と、それぞれで行われている農業を対応させて説明せよ。（6点）
→ 石狩平野は北海道西部に位置し客土で稲作が発展。十勝平野は道東南部に位置し大規模畑作で輪作が行われている。根釧台地は道東の太平洋岸に広がり、冷涼な気候を生かした酪農が盛ん。

(2) 北海道の主要都市（札幌・旭川・函館・釧路・帯広）の位置と産業的特徴を述べよ。（6点）
→ 札幌は道央で政治経済の中心。旭川は上川盆地で農業と食品加工。函館は道南で観光と水産業。釧路は道東で漁業と製紙業。帯広は十勝平野で畑作の中心地。

(3) 北海道の農業・漁業生産額の推移から読み取れる傾向と、その背景を論じよ。（8点）
→ 農業は大規模化と品種改良・ブランド化で生産額を維持している一方、漁業は200海里問題以降の遠洋漁業の縮小で生産額が減少傾向にある。養殖業やさけの放流事業で補っているが、水産資源の管理強化と付加価値向上が今後の鍵となる。
