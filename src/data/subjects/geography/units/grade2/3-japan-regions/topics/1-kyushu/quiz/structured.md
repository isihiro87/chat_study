# 九州地方 構造問題

## 大問1 九州地方の自然環境（配点20）

(1) 阿蘇山の特徴的な地形を答えよ。（2点）
→ カルデラ

(2) シラス台地は何が堆積してできたか。（2点）
→ 火山の噴出物（火山灰）

(3) シラス台地が稲作に向かない理由を簡潔に述べよ。（3点）
→ 水はけがよすぎるため水田に必要な水を保てないから。

(4) 南西諸島の気候を気温と降水量の面から説明せよ。（3点）
→ 一年を通じて気温が高く温暖で降水量も多い亜熱帯の気候である。

(5) 鹿児島市が火山灰対策として配布しているものは何か。（2点）
→ 克灰袋

(6) 筑紫平野を流れる九州最大級の河川名を答えよ。（2点）
→ 筑後川

(7) 土石流を防ぐために河川上流に造られるダムを何というか。（2点）
→ 砂防ダム

(8) 鹿児島湾はカルデラ地形である。この湾の別名を答えよ。（2点）
→ 錦江湾

(9) 九州南西の島々の総称を答えよ。（2点）
→ 南西諸島

## 大問2 九州地方の農業（配点20）

(1) 筑紫平野で盛んな農業は何か。（2点）
→ 稲作

(2) ビニールハウスなどで出荷時期を早める栽培方法を何というか。（2点）
→ 促成栽培

(3) 促成栽培で出荷時期を早められる理由を説明せよ。（4点）
→ 温暖な気候に加えビニールハウスで保温することで冬でも作物の生長を促進でき、通常より早い時期に出荷できるから。

(4) 宮崎平野の促成栽培で栽培される主な野菜を2つ答えよ。（2点）
→ きゅうり、ピーマン

(5) 鹿児島県・宮崎県で飼育されている主な家畜を3つ答えよ。（3点）
→ 肉牛、豚、鶏

(6) 南西諸島で栽培が盛んな砂糖の原料になる作物は何か。（2点）
→ さとうきび

(7) シラス台地で多く栽培されている芋を答えよ。（2点）
→ さつまいも

(8) 有明海の沿岸で盛んに養殖されている海産物は何か。（3点）
→ のり

## 大問3 九州地方の自然エネルギー（配点15）

(1) 火山の地下の高温の熱水や蒸気を利用する発電方法を何というか。（2点）
→ 地熱発電

(2) 大分県にある日本最大級の地熱発電所の名前を答えよ。（2点）
→ 八丁原発電所

(3) 繰り返し利用できるエネルギーの総称を答えよ。（2点）
→ 再生可能エネルギー

(4) 宮崎・鹿児島で日照時間を生かして行われている発電を答えよ。（2点）
→ 太陽光発電

(5) 大分県の日本有数の温泉都市はどこか。（2点）
→ 別府市

(6) 地熱発電が九州で盛んな理由を自然環境に触れて説明せよ。（5点）
→ 火山活動が活発で地下に高温のマグマや熱水がある地域が多く、地熱発電に適した環境が整っているから。

## 大問4 公害と環境の取り組み（配点25）

(1) 1901年に操業開始した北九州市の近代製鉄所を答えよ。（2点）
→ 八幡製鉄所

(2) 北九州市でかつて深刻だった環境問題を2つ答えよ。（4点）
→ 大気汚染、水質汚濁

(3) 北九州市の環境改善のきっかけとなった活動を何というか。（2点）
→ 市民運動

(4) リサイクル産業を集積させた北九州市の事業名を答えよ。（2点）
→ エコタウン事業

(5) 水俣病の原因物質は何か。（3点）
→ メチル水銀

(6) 水俣病以外の四大公害病を2つ答えよ。（4点）
→ 新潟水俣病、四日市ぜんそく、イタイイタイ病のうち2つ

(7) 水俣市が取り組んでいる都市づくりを何というか。（2点）
→ 環境モデル都市

(8) 北九州市のリサイクル率が高い理由を歴史的背景に触れて説明せよ。（6点）
→ 深刻な公害を経験した教訓から環境意識が高く、エコタウン事業でリサイクル産業を集積させ資源の循環利用に積極的に取り組んでいるから。

## 大問5 沖縄と南西諸島（配点20）

(1) 沖縄の伝統的な家屋の特徴とその理由を答えよ。（4点）
→ 鉄筋コンクリート造りで屋根が低い。台風の強風に耐えるため。

(2) 南西諸島の海岸周辺に発達している地形を答えよ。（2点）
→ さんご礁

(3) 沖縄県で最も盛んな産業は何か。（2点）
→ 観光業

(4) 沖縄県の県庁所在地はどこか。（2点）
→ 那覇市

(5) 保養地開発で問題となっている環境問題を答えよ。（3点）
→ さんご礁の減少

(6) 沖縄県が日本に復帰したのは何年か。（2点）
→ 1972年

(7) 九州・沖縄の観光資源を自然環境と結びつけて説明せよ。（5点）
→ 九州は温泉や火山の景観、沖縄はさんご礁の海と亜熱帯の自然が観光資源で、多様な自然環境と温暖な気候が観光客を引きつけている。
