# 中国・四国地方 構造問題

## 大問1 中国・四国地方の自然環境（配点20）

(1) 中国・四国地方を3つの地域に分ける2つの山地を答えよ。（2点）
→ 中国山地と四国山地

(2) 中国山地と四国山地の地形の違いを述べよ。（3点）
→ 中国山地はなだらかで、四国山地は険しい。

(3) 山陰地方の冬の気候の特徴とその原因を説明せよ。（4点）
→ 北西の季節風が日本海から湿った空気を運び、中国山地にぶつかって雪や雨を降らせるため、冬に降水量が多い。

(4) 瀬戸内地方の降水量が少ない理由を山地との関係で説明せよ。（4点）
→ 中国山地と四国山地にはさまれているため、季節風の影響を受けにくく降水量が少ない。

(5) 香川県にため池が多い理由を気候と関連づけて答えよ。（4点）
→ 瀬戸内式気候で降水量が少なく、農業用水が不足しがちなため、水をためるため池が多くつくられた。

(6) 早明浦ダムの役割を答えよ。（3点）
→ 四国の水がめとして、瀬戸内地域などに水を供給する役割がある。

## 大問2 交通網と地域への影響（配点20）

(1) 本州四国連絡橋の3つのルートと橋の名前を答えよ。（6点）
→ 児島・坂出ルート（瀬戸大橋）、神戸・鳴門ルート（明石海峡大橋）、尾道・今治ルート（しまなみ海道）

(2) 瀬戸大橋が開通した年を答えよ。（2点）
→ 1988年

(3) ストロー現象とは何か説明せよ。（4点）
→ 交通網の整備で移動が便利になった結果、地方の人口や消費が大都市に吸い取られる現象。

(4) ストロー現象が地方にもたらす具体的な影響を述べよ。（4点）
→ 四国の住民が大阪や神戸に買い物に行くようになり、地元の商店街が衰退して空き店舗が増えている。

(5) しまなみ海道を活用した地域活性化の取り組みを述べよ。（4点）
→ サイクリングコースとして観光客を誘致し、瀬戸内の島々の魅力を発信して地域の活性化につなげている。

## 大問3 瀬戸内工業地域と産業（配点20）

(1) 瀬戸内海沿岸に工場が立地した理由を海の特徴から説明せよ。（4点）
→ 瀬戸内海は波が穏やかで大型タンカーの航行に適しており、原料の輸入や製品の輸出に便利だから。

(2) 岡山県倉敷市にある代表的な工業施設の名前を答えよ。（2点）
→ 水島コンビナート

(3) 広島県で盛んな工業を答えよ。（2点）
→ 自動車工業

(4) 瀬戸内地方でかつて塩田が盛んだった理由を気候から説明せよ。（4点）
→ 降水量が少なく日照時間が長いため、海水を蒸発させて塩をつくる塩田に適していた。

(5) 高知県で行われている促成栽培の特徴と主な作物を答えよ。（4点）
→ 温暖な気候とビニールハウスを利用して出荷時期を早める栽培方法。主になす・ピーマンが栽培されている。

(6) 鳥取県と愛媛県の代表的な農産物をそれぞれ答えよ。（4点）
→ 鳥取県：なし・らっきょう、愛媛県：みかん

## 大問4 過疎化と地域おこし（配点20）

(1) 中山間地域で過疎化が進む原因を述べよ。（4点）
→ 都市部への若者の流出で人口が減少し、高齢化が進んで地域の産業や公共サービスの維持が困難になっているから。

(2) 限界集落とは何か説明せよ。（3点）
→ 人口の50%以上が65歳以上の高齢者で、集落の維持が困難になっている集落。

(3) 徳島県上勝町の葉っぱビジネスの内容を具体的に説明せよ。（4点）
→ 高齢者がタブレットを使って注文を確認し、料理のつまものに使う葉っぱを出荷して収入を得るICT活用ビジネス。

(4) 島根県松江市の地域おこしの特徴を説明せよ。（4点）
→ プログラミング言語Rubyの開発者が拠点を置き、IT企業の誘致による地域おこしが行われている。

(5) インターネットを活用した過疎地域の活性化策を述べよ。（5点）
→ 移住支援サイトやSNSで地域の魅力を発信し、テレワーク環境の整備で都市部からの移住者を呼び込んでいる。

## 大問5 文化・観光と地域の特色（配点20）

(1) 広島市にある世界遺産を2つ答えよ。（4点）
→ 原爆ドーム、厳島神社

(2) 瀬戸内地方でみかん・レモンの栽培が盛んな理由を気候から説明せよ。（4点）
→ 瀬戸内式気候で降水量が少なく日照時間が長いため、柑橘類の栽培に適している。

(3) 中国・四国地方の農業の特徴を、山陰・瀬戸内・南四国に分けて説明せよ。（6点）
→ 山陰：鳥取のなし・らっきょう。瀬戸内：みかん・レモンなどの柑橘類。南四国：高知の促成栽培（なす・ピーマン）、愛媛のみかん。

(4) 中国・四国地方の過疎化と交通網整備の関係を、ストロー現象にも触れて説明せよ。（6点）
→ 中山間地域では若者の流出で過疎化が進んでいる。本州四国連絡橋の整備で移動が便利になった一方、ストロー現象で四国の消費が大都市に流出し、地方の商店街がさらに衰退する結果となっている。
