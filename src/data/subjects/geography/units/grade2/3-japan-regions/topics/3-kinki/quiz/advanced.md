# 近畿地方 発展問題

## 発展1 近畿地方の自然と気候の多様性（配点20）

(1) 近畿地方の北部・中央部・南部で気候が異なる理由を、それぞれの地域の特徴に触れて説明せよ。（6点）
→ 北部は日本海側で北西の季節風の影響を受け冬に雪や雨が多い。中央部は盆地が多く山に囲まれるため降水量が少なく寒暖差が大きい。南部は黒潮の影響で温暖で降水量も多い。

(2) リアス海岸が若狭湾と志摩半島に形成された地理的背景を説明し、この地形が産業にどう利用されているか論じよ。（6点）
→ 山地が海に迫る地形で河川の浸食による谷が沈降して入り江となった。波が穏やかな入り江は養殖漁業や真珠の養殖に適しており、若狭湾の漁業や志摩半島の真珠養殖に活用されている。

(3) 琵琶湖が近畿地方の生活に果たしている役割を水資源の面から述べよ。（4点）
→ 琵琶湖は日本最大の湖で、淀川を通じて京都・大阪・神戸など近畿地方の約1400万人の水源となっている。

(4) 紀伊山地の吉野杉による林業が衰退した原因と、その対策を考察せよ。（4点）
→ 安い外国産木材の輸入増加で国産材の需要が低下した。対策として吉野杉のブランド価値の向上、木材の新たな活用法の開発、林業の担い手育成が考えられる。

## 発展2 大阪大都市圏の構造と課題（配点20）

(1) 大阪大都市圏が近畿地方の経済に果たす役割を、他の大都市圏と比較しながら論じよ。（6点）
→ 大阪大都市圏は約880万人の人口を抱え西日本最大の経済圏で、商業・金融・情報の機能が集中している。東京大都市圏に次ぐ規模だが、近年は企業の東京移転で相対的に地位が低下している面もある。

(2) 阪神工業地帯の中小企業と堺市の大型工場の違いを、立地と産業構造の面から説明せよ。（6点）
→ 東大阪市など内陸部には町工場型の中小企業が密集し高度な技術力で部品製造などを担う。一方堺市の臨海部には大型工場が立地し、広い用地と港湾を利用した大規模な製造業を展開している。

(3) 東大阪市の中小企業が持つ強みと課題について、「まいど1号」の事例を交えて論じよ。（8点）
→ 強みは個々の町工場が高い専門技術を持ち、企業間の協力で人工衛星「まいど1号」を開発できるほどの総合力がある点。課題は後継者不足や大企業の海外生産移転による受注減少、設備の老朽化などがある。

## 発展3 ニュータウンの過去・現在・未来（配点20）

(1) 1960年代にニュータウンが建設された社会的背景を高度経済成長と関連づけて説明せよ。（5点）
→ 高度経済成長期に農村から都市への人口移動が加速し、大阪市への人口集中で住宅不足が深刻化した。そのため郊外の丘陵地を開発してニュータウンを計画的に建設した。

(2) ポートアイランドの造成方法と、その利点・問題点を述べよ。（5点）
→ 六甲山系の丘陵を削った土で大阪湾を埋め立てて人工島を造成した。利点は新しい都市空間を海上に創出できた点。問題点は地震時の液状化リスクや、阪神・淡路大震災で実際に被害を受けた点。

(3) ニュータウンの老朽化・高齢化問題を解決するための方策を、梅田の再開発と比較しながら提案せよ。（10点）
→ 梅田では商業施設やオフィスビルの大規模再開発で新たな人の流れを生み出した。ニュータウンでは建物のリノベーション、空き店舗の活用、テレワーク拠点の整備で若い世代を呼び込むことが考えられる。また多世代交流施設の設置や地域コミュニティの活性化、公共交通の改善なども重要。

## 発展4 農山村の地域振興モデルの比較（配点20）

(1) 和束町の「和束茶」ブランド化と伊根町の舟屋活用、2つの地域振興モデルを比較し、共通する成功要因を論じよ。（8点）
→ 和束町は宇治茶の産地として茶畑の景観と地域ブランド「和束茶」を売りに観光と特産品販売を結びつけた。伊根町は舟屋を民宿やカフェに改装し伝統的景観を体験型観光に転換した。共通する成功要因は地域固有の景観や文化資源を観光と結びつけ、外部から人を呼び込んでいる点。

(2) 過疎地域が抱える「地域の担い手不足」の問題に対して、近畿地方の事例を参考に解決策を考察せよ。（6点）
→ 移住者の受け入れ体制の整備、リモートワークの推進、地域おこし協力隊の活用が考えられる。和束町や伊根町のように地域の魅力を発信して外部人材を呼び込み、新たな担い手を確保することが重要。

(3) 京都の景観条例と農山村の景観保全の取り組みを比較し、「景観」が地域経済に与える影響を論じよ。（6点）
→ 京都では景観条例で都市の歴史的景観を守り観光資源として高い経済効果を生んでいる。農山村では茶畑や舟屋など自然・伝統的景観が観光客誘致の原動力となっている。いずれも景観そのものが地域のブランド力となり経済活性化につながっている。

## 発展5 災害と都市の持続可能性（配点20）

(1) 阪神・淡路大震災が近畿地方の都市計画・防災に与えた影響を具体的に述べよ。（6点）
→ 耐震基準の大幅な見直し、緊急輸送道路の整備、防災拠点の配置、木造密集市街地の改善が進められた。また住民の防災意識が高まりボランティア文化の発展にもつながった。

(2) 埋立地で造られたポートアイランドや関西国際空港が自然災害に対して抱えるリスクを説明せよ。（6点）
→ 地震時の液状化現象により地盤沈下や建物の傾斜が起こるリスクがある。関西国際空港は台風時の高潮で浸水被害を受けた実績もあり、海面上昇や大規模台風への対策が課題である。

(3) 近畿地方の都市と農山村が共に持続可能な発展を遂げるために必要な取り組みを、本トピックで学んだ内容を踏まえて総合的に論じよ。（8点）
→ 都市ではニュータウンの再生や防災都市づくりを進めつつ景観条例でまちの魅力を守る。農山村では地域資源を生かした観光振興とブランド化で若い世代を呼び込む。都市と農村の交流促進、ICTの活用で地方に住みながら都市の仕事ができる環境整備も重要。大阪大都市圏の経済力を近畿全体に波及させる広域連携が持続可能な発展の鍵となる。
