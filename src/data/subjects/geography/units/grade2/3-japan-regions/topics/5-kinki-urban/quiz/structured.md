# 近畿の都市問題と農山村 構造問題

## 大問1 ニュータウンと都市再開発（配点20）

(1) 1960年代に大阪周辺の郊外にニュータウンが建設された理由を述べよ。（3点）
→ 大阪市への人口集中が進み、住宅不足を解消するために郊外にニュータウンが建設された。

(2) ポートアイランドはどのように造られたか説明せよ。（4点）
→ 神戸市の丘陵を削った土で海を埋め立てて造った人工島である。

(3) 大阪府にある代表的なニュータウンを2つ答えよ。（2点）
→ 千里ニュータウン、泉北ニュータウン

(4) ニュータウンが現在抱えている問題を2つ挙げ、それぞれの内容を説明せよ。（4点）
→ ①老朽化：建設から50年以上経過し建物が古くなっている。②高齢化：若い世代が転出し住民の高齢化が進んでいる。

(5) 大阪市中心部の梅田で進められている取り組みは何か。（2点）
→ 再開発

(6) ポートアイランドが震災時に被害を受けた原因となった現象は何か。（3点）
→ 液状化現象（埋立地であるため地震時に地盤が液状化した）

(7) 1995年に近畿地方で発生した大地震の名称を答えよ。（2点）
→ 阪神・淡路大震災

## 大問2 農山村の過疎化と地域おこし（配点20）

(1) 近畿地方の山間部・沿岸部で進んでいる2つの問題を答えよ。（2点）
→ 過疎化と高齢化

(2) 奈良県川上村の林業が衰退した理由を説明せよ。（4点）
→ 安い外国産木材が大量に輸入されるようになり、国産の吉野杉の需要が減少して林業が衰退した。

(3) 京都府和束町が地域ブランドとして売り出している茶の名称を答えよ。（2点）
→ 和束茶

(4) 和束町の地域振興の取り組みを景観に触れて説明せよ。（4点）
→ 宇治茶の産地として茶畑の景観を観光資源に活用し、地域ブランド「和束茶」として独自のまちづくりを進めている。

(5) 京都府伊根町の舟屋とはどのような建物か説明せよ。（3点）
→ 1階が船の格納庫、2階が住居となっている海に面した伝統的な建物。

(6) 伊根町が舟屋をどのように活用して地域振興しているか述べよ。（3点）
→ 舟屋を民宿やカフェに改装し、伝統的な景観を観光資源として活用している。

(7) 和束町と伊根町の地域振興の共通点を述べよ。（2点）
→ どちらも地域の伝統的な景観や文化を観光資源として活用して過疎化対策に取り組んでいる。
