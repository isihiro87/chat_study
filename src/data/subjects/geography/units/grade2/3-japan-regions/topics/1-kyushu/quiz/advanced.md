# 九州地方 発展問題

## 発展1 シラス台地と農業の関係（配点20）

(1) シラス台地の成り立ちを火山活動と関連づけて説明せよ。（4点）
→ 九州南部の火山が大規模な噴火を繰り返し、噴出した火山灰や軽石が広範囲に堆積してできた台地である。

(2) シラス台地で稲作ではなく畑作・畜産が発展した理由を土壌の性質に触れて論じよ。（5点）
→ 火山灰が主成分で水はけがよく水田に必要な水を保てないため、さつまいもの栽培や広い土地を活用した畜産が発展した。

(3) 促成栽培と抑制栽培の違いを出荷時期と価格の面から説明せよ。（5点）
→ 促成栽培は温暖な気候で出荷時期を早め冬に出荷する。抑制栽培は冷涼な気候で出荷時期を遅らせ秋に出荷する。市場に少ない時期に出荷でき高い価格で販売できる。

(4) 九州南部と北部で異なる農業の特徴を地形と気候の面から比較せよ。（6点）
→ 北部は筑紫平野で筑後川の水を利用した稲作が盛ん。南部はシラス台地で水はけがよいため畜産やさつまいもが中心で、温暖な気候を生かした促成栽培も行われている。

## 発展2 公害と持続可能な社会（配点20）

(1) 北九州市が「公害の街」から「環境都市」へ転換した経緯を時代の流れに沿って説明せよ。（6点）
→ 1901年八幡製鉄所操業で工業都市化。大気汚染・水質汚濁が深刻化。1960年代に市民運動で環境改善。その後エコタウン事業でリサイクル産業を集積させ環境先進都市へ転換した。

(2) 水俣病から環境モデル都市への変化を踏まえ、公害が地域社会に与えた影響を論じよ。（6点）
→ メチル水銀による深刻な健康被害で地域に大打撃を与えたが、経験から環境意識が高まりごみ分別やSDGsの取り組みで環境モデル都市に再生した。公害の教訓が住民主体の環境保全の原動力となった。

(3) 北九州市と水俣市の取り組みの共通点を述べ、他の地域が学べることを考察せよ。（8点）
→ 共通点は深刻な公害を環境保全の先進的取り組みにつなげた点。市民・行政・企業の連携やリサイクル産業・環境教育など、環境保全と経済活動を両立させるモデルとして参考になる。

## 発展3 自然災害と防災（配点20）

(1) 九州で起こりうる自然災害を3つ挙げそれぞれの原因を説明せよ。（6点）
→ ①火山噴火：阿蘇山や桜島など活火山が多い。②土石流：火山灰地帯に大雨で土砂流出。③台風被害：台風の通り道で暴風・大雨被害。

(2) 火山がもたらす恵みと災害の両面を具体例を挙げて論じよ。（6点）
→ 恵み：温泉観光（別府・湯布院）、地熱発電（八丁原）、シラス台地の畜産。災害：桜島噴火の降灰、土石流。克灰袋や砂防ダムで対策しながら恵みを活用している。

(3) 気候変動が九州の農業に与える影響を促成栽培と畜産の観点から考察せよ。（8点）
→ 温暖化で他地域でも冬季栽培が可能になると宮崎平野の価格競争力低下の可能性がある。台風大型化は農地・畜舎への被害リスクを高める。高温による家畜のストレスで生産性低下も懸念され暑熱対策が必要。

## 発展4 九州の産業構造（配点20）

(1) 九州で畜産が盛んな理由を自然環境の面から説明せよ。（5点）
→ シラス台地は水はけがよく稲作に不向きだが広大な土地があり、温暖で飼料作物の栽培にも適しているため畜産が発展した。

(2) 北九州市の工業出荷額が低下し福岡市が成長している理由を考察せよ。（5点）
→ 北九州は重化学工業の衰退が影響。福岡はサービス業や情報通信業が成長しアジアへの近さを生かした国際ビジネス拠点としての役割が高まっているため。

(3) 九州が「シリコンアイランド」と呼ばれた理由を説明せよ。（5点）
→ 1970〜80年代にIC工場が多く進出し半導体産業が集積。空気がきれいで水が豊富な環境が精密な半導体製造に適していた。

(4) 九州と沖縄で観光業が重要な理由を両地域の特色を比較して述べよ。（5点）
→ 九州は温泉や火山の景観、沖縄はさんご礁と亜熱帯の自然や独自文化が観光資源。多様な自然環境と温暖な気候が観光客を引きつけている。

## 発展5 地図・資料活用（配点20）

(1) 九州の県で県名と県庁所在地名が異なるものを全て答えよ。（4点）
→ 沖縄県（那覇市）

(2) 阿蘇山・筑紫平野・シラス台地・桜島の位置関係を説明せよ。（6点）
→ 中央部に阿蘇山がありカルデラを持つ。北西部に筑紫平野が広がり稲作地帯。南部一帯にシラス台地が広がり、鹿児島湾に面して桜島がある。

(3) 九州がアジアに近い地理的位置がもたらす影響を歴史と現在の両面から述べよ。（10点）
→ 古くから大陸との交流の玄関口で博多港は貿易拠点であった。八幡製鉄所も中国からの鉄鉱石輸入に便利な立地で建設された。現在も福岡空港や博多港はアジアとの航空・クルーズ船発着地としてビジネスや観光での結びつきが強い。
