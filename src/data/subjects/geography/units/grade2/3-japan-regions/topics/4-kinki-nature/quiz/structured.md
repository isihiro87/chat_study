# 近畿の自然と大阪大都市圏 構造問題

## 大問1 近畿地方の自然環境（配点20）

(1) 若狭湾や志摩半島に見られる複雑な海岸地形を何というか。（2点）
→ リアス海岸

(2) 日本最大の湖の名前と所在する府県を答えよ。（2点）
→ 琵琶湖、滋賀県

(3) 琵琶湖から大阪湾に流れる川の名前を答えよ。（2点）
→ 淀川

(4) 近畿地方の北部で冬に降水量が多い理由を季節風に触れて説明せよ。（4点）
→ 北西の季節風が日本海の水蒸気を含んで吹きつけるため、冬に雪や雨が多くなる。

(5) 近畿地方の中央部の盆地で気温の寒暖差が大きい理由を説明せよ。（4点）
→ 周囲を山地に囲まれているため海の影響を受けにくく、夏は暑く冬は冷え込み、昼と夜の寒暖差も大きくなる。

(6) 近畿地方の南部が温暖な理由を海流に触れて説明せよ。（4点）
→ 南部は黒潮（暖流）の影響を受けるため、冬でも比較的温暖な気候となっている。

(7) 紀伊山地で盛んな林業の代表的な木材を答えよ。（2点）
→ 吉野杉

## 大問2 大阪大都市圏と阪神工業地帯（配点20）

(1) 大阪を中心とする大都市圏の人口は約何万人か。（2点）
→ 約880万人

(2) 大阪湾臨海部に広がる工業地帯の名称を答えよ。（2点）
→ 阪神工業地帯

(3) 阪神工業地帯の特徴を企業の規模と技術力の面から説明せよ。（4点）
→ 中小企業が多く集まっており、高い技術力を持っていることが特徴である。

(4) 東大阪市が「ものづくりの町」と呼ばれる理由を具体例を挙げて述べよ。（4点）
→ 中小企業の町工場が密集しており、協力して人工衛星「まいど1号」を開発するなど高い技術力を持っているから。

(5) 堺市の臨海部の工業の特徴を東大阪市と比較して述べよ。（4点）
→ 東大阪市は中小企業が密集しているのに対し、堺市の臨海部には大型工場が立地している。

(6) 阪神工業地帯が広がる範囲を答えよ。（2点）
→ 大阪から神戸にかけて

(7) 大阪が西日本で果たしている経済的な役割を述べよ。（2点）
→ 西日本最大の経済の中心地として、商業・金融・情報の機能が集まっている。

## 大問3 古都の景観保全（配点20）

(1) 京都市の景観条例で規制されている内容を2つ答えよ。（3点）
→ 建物の高さ制限と看板の色・大きさの規制

(2) オーバーツーリズムとは何か説明せよ。（3点）
→ 観光客が増えすぎて交通渋滞やごみ問題など住民生活に悪影響が出る現象。

(3) 京都の景観条例が地域経済に与える影響を述べよ。（4点）
→ 歴史的な町並みの景観を守ることで観光資源としての価値を高め、多くの観光客を呼び込んで経済効果を生んでいる。
