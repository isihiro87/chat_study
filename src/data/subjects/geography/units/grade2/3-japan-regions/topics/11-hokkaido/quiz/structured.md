# 北海道地方 構造問題

## 大問1 北海道の自然環境と気候（配点20）

(1) 北海道が属する気候帯を答えよ。（2点）
→ 亜寒帯（冷帯）気候

(2) 北海道の冬の寒さの特徴を具体的な気温を挙げて述べよ。（3点）
→ 冬は非常に寒さが厳しく、内陸部では氷点下20度以下になることもある。

(3) 北海道の住宅で行われている寒さ対策を3つ答えよ。（3点）
→ 二重窓、断熱材の使用、ロードヒーティング（道路融雪装置）

(4) 北海道に梅雨がなく台風の影響が少ない理由を述べよ。（4点）
→ 梅雨前線は本州付近で停滞し北海道まで達しにくい。台風も多くは本州以南で勢力を弱めるか進路が逸れるため、北海道への直接的な影響が少ない。

(5) 知床が世界自然遺産に登録された理由を述べよ。（4点）
→ 流氷が運ぶ栄養分を起点とした海洋と陸上の生態系の相互作用が見られ、希少な動植物が生息する豊かな自然環境が評価された。

(6) オホーツク海沿岸に冬に押し寄せる流氷の近年の変化を述べよ。（4点）
→ 地球温暖化による海水温の上昇で流氷の量が減少傾向にあり、流氷が接岸する期間も短くなっている。

## 大問2 北海道の農業（配点25）

(1) 石狩平野で稲作が発展した経緯を「泥炭地」「客土」「品種改良」の語句を使って説明せよ。（5点）
→ 石狩平野はもともと泥炭地で稲作に適さなかったが、客土で土壌を改良し、品種改良で寒さに強い品種を開発したことで稲作が発展した。

(2) 輪作とは何か、その目的を含めて説明せよ。（3点）
→ 同じ畑で作物を順番に変えて栽培する方法で、連作障害を防ぎ土地の力を回復させる目的がある。

(3) 十勝平野の輪作で栽培される作物を順番に4つ答えよ。（4点）
→ じゃがいも→小麦→てんさい→豆類

(4) てんさいとさとうきびはどちらも砂糖の原料である。それぞれの栽培地域を答えよ。（3点）
→ てんさいは北海道（冷涼な気候）、さとうきびは沖縄・南西諸島（亜熱帯の温暖な気候）

(5) 根釧台地で酪農が発達した理由を気候と地形の面から説明せよ。（4点）
→ 冷涼で霧が多いため畑作に不向きだが、広大で平坦な土地に牧草地が広がり、乳牛の飼育に適した環境であるため。

(6) 北海道の農業が本州と比べて大規模である理由を述べよ。（3点）
→ 広大な土地が利用可能で一戸あたりの耕地面積が本州の10倍以上あり、大型機械を使った効率的な農業が可能なため。

(7) 北海道の農業が抱える課題を2つ挙げよ。（3点）
→ 農業従事者の高齢化と後継者不足、冷害リスクへの対応

## 大問3 北海道の水産業（配点15）

(1) 北海道で水揚げされる主な魚介類を4つ答えよ。（4点）
→ さけ、ほたて、昆布、いか

(2) 北海道の漁獲量が多い漁港を2つ答えよ。（2点）
→ 釧路港、根室港

(3) さけの稚魚放流事業の目的を説明せよ。（3点）
→ 人工的にふ化させた稚魚を川に放流し、成長して戻ってくるさけを捕獲することで資源量を維持しながら漁獲量を確保するため。

(4) 北海道の漁業と農業を比較し、どちらも北海道が全国トップクラスである理由を自然条件から説明せよ。（6点）
→ 広大な土地と冷涼な気候が大規模農業と酪農に適しており、周囲を海に囲まれた地理的条件と寒流が運ぶ豊富な栄養分が水産業を支えている。自然環境の多様さが両方の産業の発展を可能にした。

## 大問4 アイヌ文化と歴史（配点15）

(1) アイヌの人々が持っていた文化の特徴を3つ答えよ。（3点）
→ 独自の言語、独自の文化（儀式・模様など）、自然を敬う自然観

(2) 明治時代以降、アイヌの人々に起こったことを説明せよ。（4点）
→ 明治政府の開拓政策により土地を奪われ、日本語教育の強制や伝統的な生活様式の禁止など、文化や権利が著しく侵害された。

(3) ウポポイ（民族共生象徴空間）の役割を説明せよ。（4点）
→ アイヌの歴史・文化を紹介する展示や体験プログラムを通じて、アイヌ文化の保存・継承・発信を行い、多文化共生社会の実現に貢献する施設。

(4) アイヌ施策推進法（2019年）の意義を述べよ。（4点）
→ アイヌを先住民族として初めて法的に位置づけ、アイヌ文化の振興や差別の禁止、地域振興などを推進することを定めた法律で、多文化共生社会の実現を目指している。

## 大問5 観光とブランド化（配点25）

(1) エコツーリズムとは何か説明せよ。（3点）
→ 自然環境を守りながら、自然の大切さを学び楽しむ観光の形態。

(2) 知床でエコツーリズムが推進されている理由を述べよ。（4点）
→ 世界自然遺産として貴重な生態系を持つ地域であり、観光客の増加による環境への悪影響を防ぎながら、自然の価値を伝える持続可能な観光が求められているため。

(3) 北海道の食のブランド化の具体例を2つ挙げ、その効果を述べよ。（4点）
→ 夕張メロンや十勝産小麦が代表例。ブランド化により付加価値が高まり高価格で取引され、産地の知名度向上と農家の収入増加につながっている。

(4) 北海道の観光業が抱える課題を季節の偏りの観点から述べよ。（4点）
→ 夏のラベンダーや冬のスキー・流氷など季節限定の観光資源が多く、春秋の閑散期に観光客が減少する。通年型の観光コンテンツの開発が課題である。

(5) 北海道のブランド化戦略と観光業の発展が地域経済に与える影響を、農業・漁業との関連から論じよ。（10点）
→ 農産物・水産物のブランド化は「北海道ブランド」として地域全体の魅力を高める。観光客が北海道を訪れることで飲食・宿泊業が潤い、現地での特産品購入が農漁業の販路拡大につながる。食と観光の相乗効果により、一次産業から三次産業まで連動した地域経済の好循環が生まれている。
