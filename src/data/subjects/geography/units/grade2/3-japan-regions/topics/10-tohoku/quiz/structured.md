# 東北地方 構造問題

## 大問1 東北地方の自然環境（配点20）

(1) 東北地方の中央を南北に走る山脈の名前を答えよ。（2点）
→ 奥羽山脈

(2) 奥羽山脈を境にした日本海側の冬の気候の特徴を述べよ。（3点）
→ 北西の季節風が日本海の水蒸気を含んで大量の雪を降らせる豪雪地帯である。

(3) やませとは何か、また農業にどのような影響を与えるか説明せよ。（4点）
→ 東北地方の太平洋側に夏に吹く冷たく湿った北東の風で、霧を発生させて日照不足を招き、稲の生育を妨げて冷害を引き起こす。

(4) やませ対策として行われてきた取り組みを答えよ。（3点）
→ 品種改良で寒さに強い品種（ひとめぼれ、あきたこまちなど）を開発してきた。

(5) 岩手県北部から宮城県北部にかけて広がる高地の名前を答えよ。（2点）
→ 北上高地

(6) 庄内平野はどの県に位置するか答えよ。（2点）
→ 山形県

(7) 東北地方の日本海側と太平洋側で気候が異なる理由を山脈と関連づけて説明せよ。（4点）
→ 奥羽山脈が壁となり、日本海側では季節風が雪を降らせ豪雪地帯となる一方、太平洋側では夏にやませが吹いて冷害をもたらすなど気候条件が異なる。

## 大問2 東北地方の農業と水産業（配点20）

(1) 庄内平野や秋田平野で盛んな農業は何か。（2点）
→ 稲作

(2) 山形県が全国生産量の約7割を占める果物を答えよ。（2点）
→ さくらんぼ（おうとう）

(3) 青森県が全国生産量の半分以上を占める果物を答えよ。（2点）
→ りんご

(4) リアス海岸の入り江を利用して行われている漁業の形態を答えよ。（2点）
→ 養殖漁業

(5) 三陸沖が世界有数の漁場である理由を「暖流」「寒流」の語句を使って説明せよ。（4点）
→ 暖流の黒潮と寒流の親潮がぶつかる潮目でプランクトンが豊富に発生し、それを食べる魚が多く集まるため。

(6) 三陸海岸で養殖されている主な海産物を3つ答えよ。（3点）
→ わかめ、かき、ほたて

(7) 東北地方の稲作が日本全体にとって重要な理由を述べよ。（3点）
→ 庄内平野や秋田平野など広い平野を持ち、日本有数の米の産地として全国に米を供給しているため。

(8) 東北地方で水揚げ量が多い漁港を2つ答えよ。（2点）
→ 石巻港、気仙沼港

## 大問3 東北地方の工業と交通（配点15）

(1) 東北自動車道沿いに多く進出した工場は何か。（2点）
→ IC（集積回路）工場

(2) IC工場が東北地方に進出した理由を交通面から説明せよ。（3点）
→ 東北自動車道や東北新幹線の整備により首都圏へのアクセスが改善され、製品輸送や人材確保に便利になったため。

(3) 東北地方と首都圏を結ぶ高速鉄道の名前を答えよ。（2点）
→ 東北新幹線

(4) 高速交通網の整備が東北地方の産業に与えた影響を説明せよ。（4点）
→ 企業の工場進出が進み雇用が増加した。また農産物や水産物を短時間で大都市に輸送できるようになり、産地としての競争力が高まった。

(5) 東北地方の工業が太平洋側に集中しやすい理由を地理的条件から説明せよ。（4点）
→ 太平洋側は冬の積雪が日本海側より少なく工場の操業に有利であり、太平洋ベルトに近く首都圏への輸送が便利なため。

## 大問4 伝統文化（配点20）

(1) 東北四大祭りの名前と開催地を答えよ。（4点）
→ ねぶた祭（青森）、竿燈まつり（秋田）、七夕まつり（仙台）、花笠まつり（山形）

(2) 東北の夏祭りの多くに共通する起源を答えよ。（2点）
→ 稲作の豊作を願う行事

(3) 南部鉄器はどの県の伝統的工芸品か。（2点）
→ 岩手県

(4) 南部鉄器が近年注目されている理由を述べよ。（3点）
→ 海外でデザイン性が評価され、急須や鉄瓶がグローバル市場で人気を集めているため。

(5) 天童将棋駒はどの県の伝統工芸品か。（2点）
→ 山形県

(6) 東北地方に雪国ならではの保存食文化が発達した理由を説明せよ。（3点）
→ 冬は豪雪で交通が遮断され新鮮な食材の入手が困難だったため、漬物・干し餅・きりたんぽなどの保存食が発達した。

(7) 伝統産業のグローバル化が東北地方にもたらす効果を述べよ。（4点）
→ 海外市場への販路拡大で売上が増加し、後継者の確保や産地の活性化につながる。地域のブランド力向上にも貢献する。

## 大問5 東日本大震災と復興（配点25）

(1) 東日本大震災が発生した年月日を答えよ。（2点）
→ 2011年3月11日

(2) 三陸海岸で津波被害が大きかった地形的な理由を説明せよ。（4点）
→ リアス海岸は入り江が奥に向かって狭くなる地形で、津波が入り江に入ると波が集中して高くなりやすいため。

(3) 震災後に進められている復興まちづくりの取り組みを3つ答えよ。（3点）
→ 防潮堤の建設、高台移転、防災教育の充実

(4) 高台移転とは何か説明せよ。（3点）
→ 津波の被害を受けにくい高い土地に住宅地を移すことで、将来の津波災害による被害を減らす取り組み。

(5) 東日本大震災で起きた原子力発電所の事故について、発電所名と県名を答えよ。（3点）
→ 福島第一原子力発電所、福島県

(6) 防災教育の具体的な内容を2つ挙げよ。（4点）
→ 避難訓練の実施、震災の経験を後世に伝える語り部活動（ハザードマップの活用でも可）

(7) 震災復興が東北地方に与えた変化について、防災と地域づくりの両面から述べよ。（6点）
→ 防潮堤や高台移転で津波への備えが強化された。また、震災の教訓から住民主体の防災教育や地域コミュニティの再構築が進み、災害に強い持続可能なまちづくりが行われている。
