# 中部地方 構造問題

## 大問1 中部地方の自然環境と地域区分（配点20）

(1) 中部地方を3つの地域に分ける山脈の総称を答えよ。（2点）
→ 日本アルプス

(2) 日本アルプスを構成する3つの山脈を答えよ。（3点）
→ 飛驒山脈・木曽山脈・赤石山脈

(3) 木曽川・長良川・揖斐川を合わせた呼び名を答えよ。（2点）
→ 木曽三川

(4) 木曽三川の洪水に備え、周囲を堤防で囲んだ集落を何というか。（2点）
→ 輪中

(5) 東海・中央高地・北陸の気候の特徴をそれぞれ簡潔に述べよ。（6点）
→ 東海は太平洋側の気候で冬は乾燥し夏は高温多雨。中央高地は内陸性の気候で寒暖差が大きい。北陸は日本海側の気候で冬に北西の季節風の影響で大量の雪が降る。

(6) 中部地方を流れ日本海にそそぐ日本最長の河川名を答えよ。（2点）
→ 信濃川

(7) 濃尾平野が低湿地になっている理由を木曽三川に触れて説明せよ。（3点）
→ 木曽川・長良川・揖斐川の3本の大きな川が合流する下流域にあたり、水が集まりやすく排水が困難なため低湿地になっている。

## 大問2 東海地方の産業（配点20）

(1) 愛知県を中心に広がる日本最大の工業地帯の名称を答えよ。（2点）
→ 中京工業地帯

(2) トヨタ自動車の本社がある都市名を答えよ。（2点）
→ 豊田市

(3) 静岡県浜松市で盛んな工業製品を2つ答えよ。（2点）
→ オートバイ・楽器

(4) 三重県四日市市で発達している工場群の形態を答えよ。（2点）
→ 石油化学コンビナート

(5) 中京工業地帯が製造品出荷額で日本最大である理由を説明せよ。（4点）
→ 豊田市を中心にトヨタ自動車の関連工場が多数集積し、自動車産業の規模が非常に大きいため製造品出荷額が日本最大となっている。

(6) 静岡県の牧之原台地で栽培が盛んな農産物を答えよ。（2点）
→ 茶

(7) 愛知県の渥美半島で温室栽培されている作物を答えよ。（2点）
→ メロン（温室メロン）

(8) 愛知用水の目的と供給先を説明せよ。（4点）
→ 木曽川の水を知多半島などに引き、農業・工業・生活用水を安定的に供給するために建設された。

## 大問3 中央高地の農業と工業（配点20）

(1) 涼しい高原の気候を利用して出荷時期を遅らせる栽培方法を何というか。（2点）
→ 抑制栽培

(2) 抑制栽培と促成栽培の違いを、気候と出荷時期の面から説明せよ。（4点）
→ 抑制栽培は冷涼な気候を利用して出荷時期を遅らせる方法で夏に高原野菜を出荷する。促成栽培は温暖な気候やビニールハウスを利用して出荷時期を早める方法で冬に野菜を出荷する。

(3) 甲府盆地で果樹栽培が盛んな理由を地形と気候から説明せよ。（4点）
→ 扇状地で水はけがよく果樹の根が腐りにくい。また盆地のため日照時間が長く寒暖差が大きいことが果実の糖度を高める。

(4) 諏訪地方でかつて盛んだった工業と、そのよび名を答えよ。（2点）
→ 精密機械工業。「東洋のスイス」とよばれた。

(5) 諏訪地方の工業が精密機械からコンピューター関連に変化した背景を述べよ。（4点）
→ 精密機械の技術の蓄積を生かし、電子部品やコンピューター関連機器の製造へと転換が進んだ。きれいな空気と水という精密工業に適した環境が引き続き活用されている。

(6) 八ヶ岳山麓で栽培される代表的な高原野菜を答えよ。（2点）
→ レタス

(7) 中央高地の抑制栽培が有利な理由を市場価格の面から説明せよ。（2点）
→ 他の産地が出荷しにくい夏に出荷できるため、市場に出回る量が少ない時期に高い価格で販売できる。

## 大問4 北陸地方の農業と伝統産業（配点20）

(1) 北陸地方で稲作が盛んな平野を2つ答えよ。（2点）
→ 越後平野・庄内平野

(2) 米の生産量が日本一の都道府県を答えよ。（2点）
→ 新潟県

(3) 北陸地方で伝統的工芸品が発達した背景を説明せよ。（4点）
→ 冬に大量の雪が降る豪雪地帯のため農作業ができず、農閑期の副業として手工業が発達し、それが現在の伝統的工芸品につながった。

(4) 北陸の伝統的工芸品を3つ、都道府県名とともに答えよ。（6点）
→ 石川県の輪島塗、石川県の九谷焼、福井県の越前漆器（他に加賀友禅、新潟県の小千谷ちぢみなどでも可）

(5) 福井県鯖江市で国内シェア約96%を占める産業は何か。（2点）
→ メガネフレーム

(6) 新潟県燕市・三条市で盛んな伝統産業を答えよ。（2点）
→ 金属洋食器

(7) 富山市が目指すコンパクトシティとは何か、LRTに触れて説明せよ。（2点）
→ LRT（次世代型路面電車）を導入して公共交通を中心に都市機能を集約し、車に頼らず暮らせるまちづくりを目指す都市の形態。

## 大問5 中部地方の総合問題（配点20）

(1) 東海・中央高地・北陸それぞれの主要産業を1つずつ答えよ。（3点）
→ 東海：自動車産業（中京工業地帯）。中央高地：高原野菜の抑制栽培。北陸：稲作。

(2) 中部地方の3地域で気候が産業に与えた影響をそれぞれ説明せよ。（6点）
→ 東海は温暖な気候で茶の栽培やメロンの温室栽培が盛ん。中央高地は冷涼な気候を生かした高原野菜の抑制栽培と果樹栽培が発達。北陸は冬の豪雪で農閑期ができ、副業として伝統的工芸品が発展した。

(3) 瀬戸焼・常滑焼と北陸の伝統的工芸品の成り立ちの違いを説明せよ。（5点）
→ 瀬戸焼・常滑焼は良質な粘土が産出される土地の特性から発展した焼き物産業である。一方、北陸の伝統的工芸品は冬の豪雪で農作業ができない農閑期の副業から発展したもので、気候が発達の背景にある。

(4) 中部地方の交通網の特徴を東海道新幹線と北陸新幹線に触れて述べよ。（6点）
→ 東海道新幹線が東海地方を東西に結び名古屋を交通の結節点としている。北陸新幹線は東京と北陸地方を直結し、観光や産業の活性化に貢献している。中部地方は日本の東西を結ぶ交通の要衝としての役割を持つ。
