# 中部地方②（中央高地と北陸） 発展問題

## 発展1 中央高地の農業と産業転換（配点20）

(1) 甲府盆地の扇状地で果樹栽培が盛んな理由を、地形・気候・土壌の3つの面から説明せよ。（6点）
→ 扇状地は水はけがよく果樹の根腐れが起きにくい。盆地は日照時間が長く昼夜の寒暖差が大きいため果実の糖度が高まる。砂礫質の土壌は水分を適度に制限し、果樹に凝縮した味わいを生む条件が整っている。

(2) 抑制栽培が市場で有利な理由を、促成栽培と比較しながら経済的観点から論じよ。（6点）
→ 抑制栽培は冷涼な高原で夏に高原野菜を出荷し、平地の産地が暑さで生産しにくい時期に供給できるため高値で取引される。促成栽培は温暖な気候で冬に出荷し同様に競合が少ない。両者とも他産地との出荷時期をずらすことで価格優位性を得る手法だが、利用する気候条件が正反対である。

(3) 諏訪地方が養蚕業から精密機械工業、さらにコンピューター関連産業へと転換した過程を、産業構造の変化として論じよ。（8点）
→ 戦前は養蚕業が主力だったが、化学繊維の普及で衰退。戦後は養蚕で培った精密な手作業の技術と、空気がきれいで水が豊富な環境を活かし時計・カメラなどの精密機械工業が発達し「東洋のスイス」とよばれた。さらに技術革新に伴い電子部品やコンピューター関連産業へ転換し、一貫して「精密さ」を核に産業を高度化させてきた。

## 発展2 北陸の稲作と伝統産業の持続可能性（配点20）

(1) 新潟県が米の生産量日本一である理由を、地形・気候・水資源の面から説明せよ。（6点）
→ 越後平野は日本有数の広大な平野で大規模な水田経営が可能。信濃川などの河川が豊富な水を供給する。夏の高温と十分な降水が稲の生育に適しており、これらの条件が揃うことで日本一の生産量を誇る。

(2) 北陸の伝統的工芸品が冬の副業から発展した経緯を、具体的な工芸品の例を3つ挙げて論じよ。（6点）
→ 冬の豪雪で農作業ができない農閑期に収入を得るための副業として手工業が始まった。石川県の輪島塗は木地に漆を何層も塗り重ねる精緻な技法が磨かれた。九谷焼は鮮やかな色彩で知られる陶磁器に発展した。福井県鯖江市のメガネフレームは農家の副業から始まり現在は国内シェア約96%を占める一大産業に成長した。

(3) 富山市のコンパクトシティ政策がなぜ必要とされているか、少子高齢化と環境問題の観点から考察せよ。（8点）
→ 少子高齢化により車を運転できない高齢者が増加し、郊外に広がった都市では移動困難者が増える。LRTなど公共交通を整備し都市機能を中心部に集約することで、高齢者も暮らしやすいまちになる。また車依存の軽減はCO2排出削減にもつながり、環境負荷の低い持続可能な都市モデルとして注目されている。

## 発展3 中央高地と北陸の総合考察（配点20）

(1) 中央高地の高原野菜はブランド化と輸出拡大の可能性がある一方で課題もある。課題と可能性を論じよ。（6点）
→ 中央高地の高原野菜は品質の高さからブランド化と輸出拡大の可能性がある。一方で少子高齢化による農業の担い手不足が深刻で、労働力確保と後継者育成が課題となっている。ICT活用による省力化や6次産業化が解決の方向性として期待される。

(2) 北陸の伝統工芸品が海外展開で持つ可能性と課題を考察せよ。（8点）
→ 北陸の伝統工芸品は海外での日本文化への関心を背景にインバウンド需要や輸出の可能性がある。鯖江のメガネはすでにグローバルブランドとして認知されつつある。一方で後継者不足が深刻で技術継承が課題となっている。伝統技術とデザインの融合による新たな付加価値創造が求められている。

(3) 中部地方の3地域が互いに補完し合う関係にあることを、産業連関の観点から具体例を挙げて論じよ。（6点）
→ 東海の工業地帯は中央高地や北陸からの労働力供給を受け、北陸の米や中央高地の農産物は東海の都市部の消費地に供給される。また東海の工業製品は北陸の港から輸出される場合もあり、交通網を通じて3地域は相互に依存し補完し合う関係にある。
