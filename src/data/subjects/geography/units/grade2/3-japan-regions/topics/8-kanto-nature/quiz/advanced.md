# 関東の自然と東京 発展問題

## 発展1 関東平野の自然環境と都市問題（配点20）

(1) 関東ローム層の成り立ちを火山活動と関連づけて説明せよ。（4点）
→ 関東地方周辺の火山（富士山や箱根など）が噴火し、風に運ばれた火山灰が長い年月をかけて堆積して関東ローム層が形成された。

(2) 関東ローム層が畑作に利用され稲作に向かない理由を、九州のシラス台地と比較して論じよ。（5点）
→ どちらも火山灰が堆積した土壌で水はけがよく、水田に必要な水を保てないため稲作に不向き。関東ローム層は畑作に、シラス台地は畜産やさつまいも栽培に利用されている点で共通する。

(3) ヒートアイランド現象が都市住民の生活に与える影響を3つ挙げ、対策を1つ述べよ。（6点）
→ ①猛暑日の増加で熱中症のリスクが高まる。②ゲリラ豪雨の発生頻度が増す。③冷房使用の増加で電力消費が増大する。対策として屋上緑化や保水性舗装の普及がある。

(4) 利根川の流域面積が日本最大である地理的条件を、関東平野の特徴と結びつけて説明せよ。（5点）
→ 日本最大の関東平野が広がっているため集水域が広く、多くの支流が合流する。平野が広大で山地に囲まれた盆地状の地形であるため流域面積が大きくなる。

## 発展2 世界都市・東京と一極集中の課題（配点20）

(1) 東京が世界都市と呼ばれる条件を、政治・経済・国際の3つの面から説明せよ。（6点）
→ 政治面では国会・中央省庁・最高裁判所が置かれ国の統治機能が集中。経済面では大企業の本社や金融機関が集まり証券取引所がある。国際面では外国大使館・外国企業本社・国際空港（羽田・成田）があり世界とつながっている。

(2) 一極集中を是正するための政策として実際に行われている取り組みと、その効果・課題を論じよ。（6点）
→ 筑波研究学園都市の建設や政府関係機関の地方移転が行われている。研究機能の分散には一定の効果があるが、企業本社の東京集中は依然として続いており、十分な分散には至っていない。

(3) 首都直下型地震のリスク集中について、東京一極集中との関係を踏まえて考察せよ。（8点）
→ 国会・省庁・大企業本社・金融機関・マスメディアがすべて東京に集中しているため、首都直下型地震が発生した場合、日本の政治・経済・情報の機能が同時にまひするリスクがある。人口約3700万人の大都市圏では避難や救助も困難になる。機能分散やバックアップ体制の整備が急務である。

## 発展3 東京大都市圏の人口問題（配点20）

(1) 東京都心の昼夜間人口比率が高い理由を、大都市圏の構造と結びつけて説明せよ。（5点）
→ 東京大都市圏は東京都心にオフィスや商業施設が集中し、住宅地は地価の安い神奈川・埼玉・千葉に広がるドーナツ状の構造をしている。このため大量の通勤・通学者が昼間に都心に流入し、昼間人口が夜間人口を大幅に上回る。

(2) ニュータウンの高齢化問題について、建設時の背景と現在の課題を時代の流れに沿って論じよ。（6点）
→ 高度経済成長期に急増する都市人口の受け皿として郊外に大規模ニュータウンが建設された。同世代が一斉に入居したため、現在は住民の高齢化が同時に進行し、若年層の流出・建物の老朽化・商店の撤退が起きている。

(3) 筑波研究学園都市と幕張メッセの立地を、東京大都市圏の機能分散の観点から比較せよ。（4点）
→ 筑波は研究機能を東京から分散させる目的で建設された学術都市。幕張メッセは大型イベント機能を千葉に配置し東京の過密を緩和する役割がある。どちらも東京に近い交通利便性を生かしつつ機能を分散させている。

(4) 第三次産業が約8割を占める東京大都市圏の産業構造の特徴と課題を述べよ。（5点）
→ サービス業・情報通信・金融など第三次産業が中心で、製造業の比重は低い。利点は付加価値の高い産業が集積する効率性だが、製造業の空洞化や地方との経済格差の拡大が課題である。
