# 地形図の読み取り 大問小問

## 大問1: 縮尺と距離の計算（配点 20点）

地形図の縮尺について、次の問いに答えなさい。

(1) 地形図を発行している国の機関の名前を答えよ。（2点）
→ 国土地理院。

(2) 2万5千分の1地形図で、地図上の1cmは実際の何mにあたるか。計算式とともに答えよ。（4点）
→ 1cm × 25,000 = 25,000cm = 250m。

(3) 5万分の1地形図で、地図上の1cmは実際の何mにあたるか。（2点）
→ 500m。

(4) 2万5千分の1地形図で、地図上の6cmの距離は実際の何mか。（4点）
→ 6cm × 25,000 = 150,000cm = 1,500m。

(5) 実際の距離が750mの場合、2万5千分の1地形図では地図上何cmで表されるか。（4点）
→ 750m = 75,000cm、75,000 ÷ 25,000 = 3cm。

(6) 5万分の1地形図で地図上の5cmは実際の何kmか。（4点）
→ 5cm × 50,000 = 250,000cm = 2,500m = 2.5km。

## 大問2: 等高線の読み取り（配点 22点）

等高線について、次の問いに答えなさい。

(1) 等高線とは何か。簡潔に答えよ。（3点）
→ 同じ標高（高さ）の地点を結んだ線。

(2) 2万5千分の1地形図では、主曲線と計曲線はそれぞれ何mごとに引かれるか。（4点）
→ 主曲線は10mごと、計曲線は50mごと。

(3) 5万分の1地形図では、主曲線と計曲線はそれぞれ何mごとに引かれるか。（4点）
→ 主曲線は20mごと、計曲線は100mごと。

(4) 等高線の間隔がせまい場所はどのような地形か。また、間隔が広い場所はどうか。（4点）
→ 間隔がせまい場所は傾斜がきつい急斜面、間隔が広い場所は傾斜がゆるやかな場所。

(5) 「尾根」と「谷」は等高線のどのような形で見分けられるか。それぞれ説明せよ。（4点）
→ 尾根：等高線が低い方に突き出している部分。谷：等高線が高い方に入り込んでいる部分。

(6) 断面図を作成する手順を簡潔に説明せよ。（3点）
→ 2地点間に直線を引き、その線が横切る等高線の標高を読み取って、方眼紙などに高さをプロットしてつなぐ。

## 大問3: 地図記号（配点 20点）

地形図に使われる地図記号について、次の問いに答えなさい。

(1) 次の施設を表す地図記号の名前を答えよ。鳥居の形をした記号。（2点）
→ 神社。

(2) 卍（まんじ）の形をした地図記号は何を表すか。（2点）
→ 寺院。

(3) 〒の記号は何を表すか。（2点）
→ 郵便局。

(4) ×の形をした地図記号は何を表すか。（2点）
→ 交番。

(5) 田と畑の地図記号の違いを説明せよ。（4点）
→ 田は横線と縦線が交差した形（稲の形）、畑はVの字を組み合わせた形。

(6) 広葉樹林と針葉樹林の地図記号の違いを説明せよ。（4点）
→ 広葉樹林は丸い形の樹木の記号、針葉樹林はとがった三角形の樹木の記号。

(7) 病院、消防署、警察署の地図記号をそれぞれ説明せよ。（4点）
→ 病院：十字の形（＋）、消防署：Yの字のような形（さすまた）、警察署：丸の中に×の形。

## 大問4: GISとデジタル地図（配点 18点）

地理情報を活用した技術について、次の問いに答えなさい。

(1) GISの正式名称を日本語と英語で答えよ。（4点）
→ 地理情報システム（Geographic Information System）。

(2) GISを活用するとどのようなことが可能になるか。2つ答えよ。（4点）
→ 複数の地理情報を重ね合わせて分析できる。目的に応じたオリジナルの地図を作成できる。

(3) ドットマップとはどのような地図か。何を表すのに適しているか。（4点）
→ 点（ドット）を打って分布を表す地図。人口や施設などの分布の密度や偏りを表すのに適している。

(4) デジタル地図と紙の地形図を比較し、デジタル地図の利点を2つ答えよ。（4点）
→ 拡大・縮小が自由にできる。複数のデータを重ね合わせて表示できる。

(5) 国土地理院が提供する「地理院地図」で閲覧できるものを2つ答えよ。（2点）
→ 地形図、空中写真、過去の地形図など。

## 大問5: 地形図の比較と読み取り（配点 20点）

新旧の地形図を比較して、次の問いに答えなさい。

(1) 新旧の地形図を比較することで何がわかるか。（3点）
→ その地域の土地利用の変化や都市の発展のようすがわかる。

(2) 以前は「田」の記号が多かった地域に現在は建物が並んでいる場合、どのような変化が起きたと考えられるか。（4点）
→ 農地（水田）が宅地（住宅地）に転用され、都市化が進んだと考えられる。

(3) 地形図上に新たに鉄道の線路が引かれている場合、周辺地域にどのような変化が予想されるか。2つ答えよ。（4点）
→ 駅周辺に住宅や商店が増える。交通の便がよくなり人口が増加する。

(4) 空中写真と地形図を比較する利点を答えよ。（4点）
→ 地図記号だけではわかりにくい地域の実際の景観や土地の様子を、写真で視覚的に確認できる。

(5) 2万5千分の1地形図と5万分の1地形図では、同じ地域を表す場合にどちらがより詳しいか。理由とともに答えよ。（5点）
→ 2万5千分の1地形図の方が詳しい。縮尺が大きい（実際の距離に対する地図上の距離が大きい）ため、より細かな地形や建物を表現できる。
