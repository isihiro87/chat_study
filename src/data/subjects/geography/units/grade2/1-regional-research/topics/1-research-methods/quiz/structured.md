# 地域調査の進め方 大問小問

## 大問1: 調査テーマの設定（配点 18点）

地域調査ではテーマを設定することから始める。次の問いに答えなさい。

(1) 地域調査でテーマを決めるときに用いる7つの視点のうち、4つを答えよ。（4点）
→ 環境問題・環境保全、人口、地域の歴史・景観、産業・土地利用、自然環境・防災、他地域との結びつき、作品・文化のうち4つ。

(2) 調査テーマは広すぎない方がよいとされる。その理由を答えなさい。（4点）
→ テーマが広すぎると調べる内容が多くなりすぎて調べきれず、結論があいまいになるため。

(3) 「この地域では高齢者の割合が増えている」という事実に注目した場合、どの視点からテーマを設定していると考えられるか。（2点）
→ 人口の視点。

(4) 「なぜ駅前に商店が集まっているのか」というテーマは、どの視点から設定されたものか。（2点）
→ 産業・土地利用の視点。

(5) 「新しいバイパス道路の開通で地域がどう変わったか」というテーマは、どの視点から設定されたものか。（2点）
→ 交通の発達や他地域との結びつきの視点。

(6) テーマ設定のために、地域を実際に歩く以外にどのような方法があるか、1つ答えよ。（4点）
→ 地域の地図を見て特徴を探す、統計資料を調べるなど。

## 大問2: 仮説の立て方と検証（配点 16点）

地域調査では仮説を立ててから調査を行う。次の問いに答えなさい。

(1) 地域調査における「仮説」とは何か。簡潔に説明せよ。（4点）
→ 調査結果について根拠をつけて予想すること。「〜だから〜だろう」の形式で立てる。

(2) 仮説を立てることのメリットを2つ答えよ。（4点）
→ 何を調べればよいかが明確になること、調査の方向性が定まること。

(3) 「この地域では農業がさかんである。それは平野が広く水が豊富だからだろう」という文の仮説部分と根拠部分をそれぞれ答えよ。（4点）
→ 仮説：この地域では農業がさかんである。根拠：平野が広く水が豊富だから。

(4) 仮説が調査結果と一致しなかった場合、どのように対処すべきか。（4点）
→ 仮説を修正し、新たな視点で調査を続ける。仮説が違っていたこと自体も重要な発見である。

## 大問3: 調査方法（配点 20点）

地域調査にはさまざまな方法がある。次の問いに答えなさい。

(1) 地域調査の主な調査方法を3つ答えよ。（3点）
→ 野外観察、聞き取り調査、文献調査。

(2) 野外観察で記録する際に有効な方法を2つ答えよ。（4点）
→ メモ・スケッチ、写真撮影など。

(3) 聞き取り調査を行う前に準備しておくべきことを2つ答えよ。（4点）
→ 事前に訪問先へ連絡を入れること、質問項目を準備しておくこと。

(4) 聞き取り調査のマナーとして注意すべきことを3つ答えよ。（3点）
→ 礼儀正しく丁寧に対応する、相手の都合を考慮する、個人情報の扱いに注意する。

(5) 地域の農業について調べたいとき、どのような場所で資料を入手できるか。2つ答えよ。（4点）
→ 農業協同組合（JA）、市区町村役場など。

(6) 地形図や空中写真を入手したいとき、どの機関に問い合わせるのが適切か。（2点）
→ 国土地理院。

## 大問4: グラフの使い分け（配点 20点）

調査結果をまとめるとき、適切なグラフを選ぶことが大切である。次の問いに答えなさい。

(1) 量の大小を比較するのに最も適したグラフの種類を答えよ。（3点）
→ 棒グラフ。

(2) 時間の経過にともなう変化を表すのに適したグラフの種類を答えよ。（3点）
→ 折れ線グラフ。

(3) 全体に対する各項目の割合を1つの図で表すのに適したグラフを答えよ。（3点）
→ 円グラフ。

(4) 帯グラフはどのような場合に使うと効果的か。（4点）
→ 複数のデータの割合を横に並べて比較したい場合に効果的。

(5) 「過去30年間のこの地域の人口の推移」を表したい場合、どのグラフが適切か。理由とともに答えよ。（4点）
→ 折れ線グラフ。時間の経過にともなう変化の推移を見るのに適しているため。

(6) 「この地域の産業別の就業者の割合」を表したい場合に適したグラフを答えよ。（3点）
→ 円グラフ。

## 大問5: 発表の流れ（配点 16点）

地域調査の発表について、次の問いに答えなさい。

(1) 地域調査の発表で望ましい順序を6つの段階で答えよ。（6点）
→ テーマ → 仮説 → 調査方法 → 結果 → 考察 → まとめ。

(2) 発表で最初に伝えるべき内容は何か。その理由も答えよ。（4点）
→ 調査テーマ。聞き手が「何についての発表か」を最初に理解できるようにするため。

(3) 「考察」の段階で述べるべきことを2つ答えよ。（4点）
→ 仮説が正しかったかどうかの検証結果、調査を通じて新たに気づいたこと。

(4) 主題図を使って発表することの利点を答えよ。（2点）
→ テーマに関する地域の特徴が視覚的に伝わりやすくなること。
