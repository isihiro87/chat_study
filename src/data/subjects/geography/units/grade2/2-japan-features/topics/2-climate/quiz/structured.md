# 気候から見た日本の特色 大問小問

## 大問1: 日本の気候の基本（配点 18点）

日本の気候の特徴について、次の問いに答えなさい。

(1) 日本の大部分が属する気候帯と気候区を答えよ。（4点）
→ 温帯の温暖湿潤気候。

(2) 季節風（モンスーン）とは何か。発生する原因にふれて説明しなさい。（4点）
→ 季節によって吹く方向が変わる風のこと。大陸と海洋の温度差によって、夏は海洋から大陸へ、冬は大陸から海洋へ向かって吹く。

(3) 夏の季節風と冬の季節風はそれぞれどの方角から吹くか。（4点）
→ 夏の季節風は南東から、冬の季節風は北西から吹く。

(4) 梅雨とはどのような気象現象か。時期もふくめて答えよ。（3点）
→ 6月から7月にかけて梅雨前線が日本付近に停滞し、長期間にわたって雨が続く現象。

(5) 台風が日本に接近しやすい季節を答えよ。（3点）
→ 夏から秋にかけて（7月～10月ごろ）。

## 大問2: 6つの気候区分（配点 24点）

日本の6つの気候区分について、次の問いに答えなさい。

(1) 日本の6つの気候区分の名前をすべて答えよ。（6点）
→ 北海道の気候・日本海側の気候・太平洋側の気候・中央高地の気候・瀬戸内の気候・南西諸島の気候。

(2) 北海道の気候の特徴を2つ答えよ。（4点）
→ 冬の寒さが厳しい（亜寒帯に属する）。梅雨がない。

(3) 日本海側の気候で冬に大量の雪が降るしくみを、「季節風」「日本海」「山脈」の語を使って説明しなさい。（4点）
→ 冬の北西の季節風が日本海を渡るとき大量の水蒸気を吸収し、山脈にぶつかって上昇することで大量の雪を降らせる。

(4) 太平洋側の気候で冬に晴天が多い理由を答えよ。（3点）
→ 冬の季節風が山脈を越える際に水分を日本海側で落とし、乾燥した空気が太平洋側に吹き下ろすため。

(5) 瀬戸内の気候が年中降水量が少ない理由を、地形にふれて答えよ。（4点）
→ 北側の中国山地と南側の四国山地に挟まれているため、夏の季節風も冬の季節風も山地にさえぎられて、湿った空気が届きにくいから。

(6) 中央高地の気候が「内陸性気候」ともよばれる理由を答えよ。（3点）
→ 海から遠い内陸部に位置しているため、海洋の影響を受けにくく、気温の年較差が大きくなるから。

## 大問3: 雨温図の読み取り（配点 20点）

次の(ア)～(ウ)の雨温図の特徴を読み、それぞれの気候区分を答えなさい。

(ア) 冬の降水量が非常に多く、夏は比較的少ない。1月の気温は約2℃。
(イ) 年間を通じて気温が高く（最寒月でも約17℃）、降水量も年間を通じて多い。
(ウ) 年間降水量が約1100mmと少なく、1月の平均気温は約-4℃。気温の年較差が約25℃と大きい。

(1) (ア)にあてはまる気候区分を答え、その理由を述べよ。（4点）
→ 日本海側の気候。冬に降水量（雪）が多いのは、北西の季節風が日本海で水蒸気を含み山脈にぶつかるため。

(2) (イ)にあてはまる気候区分を答え、その理由を述べよ。（4点）
→ 南西諸島の気候。年中気温が高いのは緯度が低いためで、亜熱帯に近い気候。

(3) (ウ)にあてはまる気候区分を答え、その理由を述べよ。（4点）
→ 北海道の気候。最寒月が氷点下であること、降水量が比較的少ないこと、気温の年較差が大きいことから、亜寒帯（冷帯）に属する北海道の気候。

(4) 夏に降水量が多く冬は乾燥するという特徴がある気候区分を答えよ。（4点）
→ 太平洋側の気候。

(5) (ウ)のような気候区分と、地中海性気候の降水パターンの違いを説明しなさい。（4点）
→ (ウ)の北海道の気候は冬に降雪があり、夏にもある程度の降水がある。地中海性気候は夏に乾燥して降水量が極端に少なく、冬に雨が多いという、季節ごとの降水量の逆転が見られる。

## 大問4: 季節風のしくみ（配点 18点）

季節風と日本の気候の関係について、次の問いに答えなさい。

(1) 大陸と海洋では、暖まりやすさ・冷えやすさにどのような違いがあるか。（3点）
→ 大陸は海洋に比べて暖まりやすく冷えやすい。

(2) 夏に海洋から大陸に向かって風が吹く理由を、気圧にふれて説明しなさい。（4点）
→ 夏は大陸が海洋より先に暖まるため大陸上の空気が上昇して気圧が下がり（低気圧）、相対的に気圧の高い海洋側から大陸に向かって風が吹く。

(3) 冬の季節風が大陸から海洋に向かって吹く理由を説明しなさい。（4点）
→ 冬は大陸が海洋より先に冷えるため大陸上の空気が下降して気圧が上がり（高気圧=シベリア高気圧）、気圧の高い大陸側から低い海洋側に向かって風が吹く。

(4) 日本海側と太平洋側で冬の天気が正反対になるしくみを、山脈の役割にふれて説明しなさい。（4点）
→ 北西の季節風が日本海で水蒸気を含んで山脈にぶつかり、日本海側に大雪を降らせる。水分を失った乾燥した風が山脈を越えて太平洋側に吹き下ろすため、太平洋側は晴天が多くなる。

(5) 山脈がなかった場合、日本の気候はどのように変わると考えられるか推測しなさい。（3点）
→ 山脈がなければ季節風がさえぎられずに通過するため、日本海側と太平洋側の降水量の差が小さくなり、冬の日本海側の豪雪や太平洋側の晴天という対照的な天気パターンは見られなくなると考えられる。

## 大問5: 気候と人間生活（配点 20点）

日本の気候が人々の生活に与える影響について、次の問いに答えなさい。

(1) 日本海側の豪雪地帯に見られる住居の工夫を2つ答えよ。（4点）
→ 屋根の傾斜を急にして積もった雪が自然に落ちるようにする。家の1階部分を高くして雪に埋もれにくくする（高床式の工夫）。

(2) 瀬戸内地方に「ため池」が多い理由を気候にふれて説明しなさい。（4点）
→ 瀬戸内の気候は年間を通じて降水量が少なく水不足になりやすいため、農業用水を確保するためにため池が多くつくられた。

(3) 沖縄県の伝統的な家屋にみられる台風対策を1つ答えよ。（3点）
→ 屋根を低くし石垣やフクギの防風林で家を囲んで強風を防ぐ工夫がある。

(4) 北海道で畑作や酪農が盛んな理由を、気候の特徴と関連づけて説明しなさい。（4点）
→ 北海道は梅雨がなく夏も涼しいため稲作よりも畑作に適している。また、広大な土地と冷涼な気候が牧草の生育に適しているため酪農も盛んである。

(5) 中央高地の長野県や山梨県で高原野菜の栽培やレタスなどの抑制栽培が行われる理由を気候にふれて答えよ。（5点）
→ 中央高地は標高が高く夏でも涼しいため、他の地域で暑くて栽培しにくい時期にレタスやキャベツなどの高原野菜を出荷できる。この涼しい夏の気候を活かした栽培を抑制栽培という。
