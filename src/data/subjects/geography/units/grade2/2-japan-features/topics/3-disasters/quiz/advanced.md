# 自然災害と防災 発展問題

## 問1: 日本の地震メカニズムと被害

日本は4つのプレート（太平洋プレート・フィリピン海プレート・ユーラシアプレート・北アメリカプレート）の境界付近に位置している。

(1) プレートの境界で地震が発生するしくみを「沈み込み」「ひずみ」の語を使って説明しなさい。
→ 海洋プレート（太平洋プレートなど）が大陸プレートの下に沈み込む際にひずみがたまり、そのひずみが限界に達して一気に元に戻ろうとするときに地震が発生する。

(2) 液状化現象が起こりやすい地盤の条件を答え、液状化によってどのような被害が起こるか具体的に述べなさい。
→ 水を多く含んだ砂質の地盤（埋立地や河川沿いなど）で起こりやすい。液状化により建物が傾いたり沈んだりし、マンホールが浮き上がったり、水道管が破損するなどの被害が生じる。

(3) 東日本大震災では津波による被害が甚大であった。津波の被害を軽減するために有効な対策を「ハード面」「ソフト面」からそれぞれ1つずつ答えなさい。
→ ハード面：防潮堤の建設や高台への住宅移転。ソフト面：ハザードマップの整備や避難訓練の実施、緊急地震速報・津波警報の迅速な発表。

(4) 東日本大震災を教訓に、日本社会で見直された防災上の課題を1つ挙げ、その後の取り組みを述べなさい。
→ 「想定外」の災害に備える必要性が認識された。津波の想定高さの見直し、避難場所の見直し、原子力発電所の安全基準の強化などが行われた。

## 問2: 火山災害と恩恵の両面

日本には100以上の活火山があり、噴火による災害のリスクがある一方で、火山は人々の生活に恩恵ももたらしている。

(1) 火砕流・溶岩流・火山灰のそれぞれについて、どのような被害をもたらすか説明しなさい。
→ 火砕流：高温のガスや岩石が高速で流れ下り、通過した場所のすべてを焼き尽くす。溶岩流：高温の溶岩がゆっくり流れ下り、建物や農地を埋め尽くす。火山灰：広範囲に降り積もり、農作物への被害、交通の麻痺、呼吸器への影響を引き起こす。

(2) 火山が人間社会にもたらす恩恵を3つ挙げ、それぞれについて簡潔に説明しなさい。
→ 温泉（観光資源や健康増進に利用）、肥沃な土壌（火山灰が堆積し農業に適した土地を形成）、地熱発電（火山の地下にある熱エネルギーを利用して発電）。

(3) 火山の噴火に備えて、住民がとるべき行動を「噴火前」と「噴火時」に分けて答えなさい。
→ 噴火前：ハザードマップで危険区域を確認し、避難経路と避難場所を把握しておく。噴火時：噴火警戒レベルに応じて速やかに避難し、火口から離れた安全な場所へ移動する。屋外にいる場合はヘルメットなどで頭を守り、火山灰を吸わないようにマスクを着用する。

## 問3: 気象災害の多様性

日本では台風・集中豪雨のほかにも、冷害や干ばつなどさまざまな気象災害が発生する。

(1) 台風がもたらす3つの災害要因（暴風・大雨・高潮）について、それぞれどのような被害が生じるか具体的に述べなさい。
→ 暴風：建物の屋根が飛ばされたり、木が倒れたり、電柱が折れて停電が発生する。大雨：河川の氾濫による洪水、低地の浸水、山間部の土砂崩れが起こる。高潮：海面が異常に上昇し、沿岸部の住宅や農地が浸水する。

(2) 高潮が発生するしくみを「気圧」「風」の語を使って説明しなさい。
→ 台風の中心付近は気圧が非常に低く、低い気圧によって海面が吸い上げられる。さらに、台風の強い風が海岸に向かって吹くことで海水が陸地側に押し寄せ、海面が異常に上昇する。

(3) 東北地方では「やませ」と呼ばれる風が冷害の原因となる。やませとはどのような風か説明し、なぜ冷害を引き起こすか述べなさい。
→ やませは夏にオホーツク海から吹く冷たく湿った北東の風である。やませが吹くと日照不足と低温が続き、稲の生育が妨げられて米の収穫量が大幅に減少する。

(4) 近年、集中豪雨による被害が増加傾向にあるとされる。その背景について「地球温暖化」の語を使って考えを述べなさい。
→ 地球温暖化により大気中の水蒸気量が増加し、短時間に大量の雨が降る集中豪雨が発生しやすくなっていると考えられる。また、都市化によるアスファルト舗装の増加で雨水の浸透が減少し、洪水や浸水のリスクが高まっている。

## 問4: 防災・減災の考え方

防災・減災において「自助・共助・公助」の3つの柱が重要とされている。

(1) 自助・共助・公助をそれぞれ定義し、具体例を1つずつ挙げなさい。
→ 自助：自分の身は自分で守ること。例：家具の転倒防止器具を設置する。共助：地域住民が互いに助け合うこと。例：町内会で防災訓練を実施する。公助：国や自治体が行う支援。例：避難所を開設し、救助隊を派遣する。

(2) 大規模災害が発生した場合、公助による支援が届くまでに時間がかかることがある。その間に「自助」と「共助」が特に重要となる理由を述べなさい。
→ 大規模災害では被害が広範囲に及び、救助隊や行政の対応能力を超える場合がある。公助が届くまでの間、自分で身を守る自助と、近所の人々と助け合う共助によって命を守り、被害を最小限に抑える必要があるため。

(3) ハザードマップが住民に広く活用されるために、自治体はどのような取り組みを行うべきか。2つ提案しなさい。
→ 全世帯にハザードマップを配布し、内容を理解するための説明会を開催する。ハザードマップを使った避難訓練を地域ぐるみで定期的に実施する。

(4) ボランティア活動が被災地の復興において果たす役割を、「行政の支援」と比較して述べなさい。
→ 行政の支援は制度に基づく公的な対応であり一定の手続きや時間を要する一方、ボランティアは被災者の個別のニーズにきめ細やかに対応でき、機動性が高い。がれき撤去、炊き出し、心のケアなど行政の手が届きにくい部分を補う重要な役割を果たす。

## 問5: 災害事例の分析

次の3つの災害事例について考えなさい。

【事例A】1999年の台風18号（九州）
【事例B】2005年の集中豪雨
【事例C】2011年の東日本大震災

(1) 事例A・B・Cを「地震災害」と「気象災害」に分類しなさい。
→ 地震災害：事例C　気象災害：事例A・B

(2) 事例Cは、それまでの防災の想定を大きく超える災害であった。この震災後に「想定外」という言葉が注目された理由を、防波堤の高さなどを例にして説明しなさい。
→ 東日本大震災で発生した津波の高さは、従来の想定を大きく超えるものであった。各地に建設されていた防波堤や防潮堤の高さを超える津波が押し寄せ、想定に基づいて安全とされていた地域にも被害が及んだ。これにより「想定外」の災害にどう備えるかが大きな課題となった。

(3) 事例A〜Cのような災害から得られた教訓をもとに、今後の防災・減災のあり方について、自助・共助・公助の観点を含めて、あなたの考えを100字程度で述べなさい。
→ 過去の災害から学び、公助としてハザードマップの更新や防波堤の整備を進めるとともに、自助として避難経路の確認や備蓄を行い、共助として地域で防災訓練を実施することが重要である。「想定外」を前提に、日頃からの備えを怠らないことが命を守ることにつながる。
