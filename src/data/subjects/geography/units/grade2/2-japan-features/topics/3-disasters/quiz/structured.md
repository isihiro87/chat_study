# 自然災害と防災 大問小問

## 大問1: 地震と津波（配点 18点）

日本は世界でも有数の地震多発国である。次の問いに答えなさい。

(1) 日本で地震が多い理由を「プレート」という語を使って説明しなさい。（4点）
→ 日本列島は複数のプレートの境界付近に位置しており、プレート同士が押し合うことで地震が頻繁に発生するため。

(2) 海底で大きな地震が起こると発生し、沿岸部に甚大な被害をもたらす現象を何というか。（2点）
→ 津波。

(3) 地震の揺れによって、水を含んだ砂地盤が液体のようになる現象を何というか。（2点）
→ 液状化（現象）。

(4) 2011年に発生した東日本大震災について、特に大きな被害をもたらした原因を答えなさい。（4点）
→ 巨大な津波が東北地方の太平洋沿岸を襲い、広範囲にわたる浸水と建物の倒壊をもたらした。

(5) 緊急地震速報が発表される目的を答えなさい。（3点）
→ 強い揺れが届く前に住民に知らせ、身の安全を確保する時間を与えるため。

(6) 地震による津波の被害を防ぐために沿岸部に建設されているものを答えよ。（3点）
→ 防波堤（防潮堤）。

## 大問2: 火山災害（配点 16点）

日本には100以上の活火山がある。次の問いに答えなさい。

(1) 火山噴火で発生する災害を3つ答えよ。（6点）
→ 火砕流・火山灰の降下・溶岩流（火山ガスも可）。

(2) 火砕流が特に危険とされる理由を「速度」「温度」の語を使って説明しなさい。（4点）
→ 火砕流は非常に高い温度（数百度）の火山ガスや岩石の破片が高速（時速100km以上）で流れ下るため、逃げることが極めて難しい。

(3) 火山が人間にもたらす恩恵を2つ答えよ。（4点）
→ 温泉や肥沃な土地（地熱発電の利用、美しい景観なども可）。

(4) 日本が環太平洋造山帯に位置していることは、火山活動とどのような関係があるか。（2点）
→ 環太平洋造山帯はプレートが沈み込む場所にあたり、マグマが発生しやすいため火山活動が活発である。

## 大問3: 気象災害（配点 18点）

台風や豪雨などの気象災害について、次の問いに答えなさい。

(1) 台風が日本に接近・上陸しやすい季節を答えよ。（2点）
→ 夏から秋にかけて。

(2) 台風によって海面が異常に上昇する現象を何というか。（2点）
→ 高潮。

(3) 集中豪雨が原因で発生する災害を3つ答えよ。（6点）
→ 洪水・土砂崩れ（土石流・がけ崩れ）・浸水。

(4) 東北地方で冷害が起こりやすい理由を「やませ」の語を使って説明しなさい。（4点）
→ 夏にやませと呼ばれる冷たく湿った北東の風が吹くことで気温が上がらず、稲などの農作物の生育が妨げられるため。

(5) 干ばつとはどのような災害か説明しなさい。（4点）
→ 長期間にわたり雨が降らず、水不足が深刻化する災害。農業用水や生活用水の確保が困難になる。

## 大問4: ハザードマップと防災の取り組み（配点 24点）

自然災害への備えについて、次の問いに答えなさい。

(1) ハザードマップとは何か。「災害リスク」「避難場所」の語を使って説明しなさい。（4点）
→ 地域ごとの災害リスクや避難場所を示した地図で、自治体が作成し住民に配布するもの。

(2) ハザードマップは主にどこが作成するか。（2点）
→ 自治体（市町村）。

(3) 防災と減災の違いを説明しなさい。（4点）
→ 防災は災害そのものを防ぐことを目指す考え方で、減災は災害が起きたときに被害をできるだけ少なくすることを目指す考え方。

(4) 次の取り組みは自助・共助・公助のどれに当たるか答えよ。（6点）
ア 市が避難所を開設する
イ 家庭で非常持ち出し袋を準備する
ウ 地域住民が協力して避難誘導する
→ ア：公助　イ：自助　ウ：共助

(5) 災害後に被災地で支援活動を行うボランティアの役割を1つ具体的に答えよ。（4点）
→ がれきの撤去、炊き出し、物資の仕分け・配布、避難所の運営補助など（いずれか1つ）。

(6) 避難訓練を日頃から行うことの意義を説明しなさい。（4点）
→ 災害時にパニックにならず、迅速かつ安全に避難できるよう行動を体で覚えておくため。

## 大問5: 総合問題（配点 24点）

次の文を読んで問いに答えなさい。

日本は環太平洋造山帯に位置し、地震・津波・火山噴火のほか、台風や集中豪雨による気象災害も多い国である。こうした自然災害に対して、国や自治体による（ A ）、自分の身は自分で守る（ B ）、地域の住民が助け合う（ C ）を組み合わせた防災・減災が重要である。

(1) 文中の（A）（B）（C）にあてはまる語をそれぞれ答えよ。（6点）
→ A：公助　B：自助　C：共助

(2) 日本列島付近にあるプレートを2つ答えよ。（4点）
→ 太平洋プレート・フィリピン海プレート（ユーラシアプレート・北アメリカプレートも可）。

(3) 1999年に九州に大きな被害をもたらした台風は何号か。（2点）
→ 台風18号。

(4) 東日本大震災が発生した年と月日を答えよ。（4点）
→ 2011年3月11日。

(5) 「減災」の考え方に基づいて、住民が日頃から行えることを2つ具体的に答えなさい。（4点）
→ 避難経路を確認しておくこと、非常持ち出し品（水・食料・懐中電灯など）を準備しておくこと。

(6) ハザードマップを住民が活用する利点を「避難」の語を使って説明しなさい。（4点）
→ 自分の住む地域の災害リスクを事前に把握し、適切な避難経路や避難場所を確認できるため、災害時に迅速に避難行動をとれる。
