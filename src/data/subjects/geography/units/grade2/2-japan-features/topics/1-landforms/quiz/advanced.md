# 地形から見た日本の特色 発展問題

## 問1: プレートと日本の地形

日本列島は、ユーラシアプレート・北アメリカプレート・太平洋プレート・フィリピン海プレートの4つのプレートが接する場所に位置している。次の問いに答えなさい。

(1) プレートが押し合う境界付近で起こりやすい自然現象を2つ答えよ。
→ 火山活動（噴火）と地震。

(2) フォッサマグナは東日本と西日本の地質の境目とされている。フォッサマグナの西の端の境界線として知られる構造線の名前を答えよ。
→ 糸魚川静岡構造線。

(3) 日本アルプス（飛騨山脈・木曽山脈・赤石山脈）がフォッサマグナの東側にそびえている理由を、プレートの動きにふれて説明しなさい。
→ プレート同士が押し合う力によって地殻が隆起し、フォッサマグナの東側に高い山脈が形成された。プレートの圧縮力が大きい場所ほど激しく隆起するため、標高3000m級の山脈が連なっている。

(4) 日本に約111の活火山がある理由を、プレートの沈み込みと関連づけて説明しなさい。
→ 太平洋プレートやフィリピン海プレートが大陸側のプレートの下に沈み込むとき、地下でマグマが発生しやすくなる。このマグマが地表に噴出することで多くの火山が形成されている。

## 問2: 河川の特徴と地形利用の比較

次の資料を見て、問いに答えなさい。

【資料】信濃川（日本最長の河川）：全長367km、流域面積11,900km²
ミシシッピ川（アメリカ）：全長約6,275km、流域面積約298万km²

(1) 日本の河川と大陸の河川を比較して、日本の河川の特徴を3つ答えよ。
→ 距離が短い、流れが急である、流域面積が狭い。

(2) 信濃川の全長はミシシッピ川の約何分の1か。概算で答えよ。
→ 約17分の1。

(3) 日本の河川がこのような特徴をもつ理由を、国土の地形と関連づけて説明しなさい。
→ 日本の国土は南北に細長い島国で、脊梁山脈（中央の山脈）が海のすぐ近くまで迫っている。そのため川の源流から海までの距離が短く、急な傾斜を一気に下るため流れが速くなり、流域面積も狭くなる。

(4) 扇状地と三角州では、それぞれなぜ異なる土地利用がなされるのか。地形の特徴と水の関係から説明しなさい。
→ 扇状地は砂礫が堆積した地形で水はけがよく地表水が少ないため、水田には向かず果樹園に利用される。一方、三角州は細かい泥や砂が堆積した低湿地で水分が豊富なため、水田として利用されてきた。

## 問3: 海岸地形の成り立ちと利用

(1) リアス海岸はどのようにしてできたか。「山地」「沈降」の語を使って説明しなさい。
→ 海面に近い谷の多い山地が沈降（または海面が上昇）することで、谷に海水が入り込み、入り組んだ複雑な海岸線が形成された。

(2) リアス海岸では養殖業が盛んである。その理由を海岸の地形的特徴から説明しなさい。
→ リアス海岸は入り江が深く波が穏やかなため、いかだや網を設置しやすく、養殖業に適している。また、山から栄養分が流れ込むため、養殖の環境としても優れている。

(3) 人工海岸が増加した理由と、それに伴う環境上の問題点を述べなさい。
→ 高度経済成長期以降、工業用地や宅地の確保のために埋め立てが進み、人工海岸が増加した。これにより、干潟や自然の砂浜が失われ、生態系への影響や防災機能の低下が問題となっている。

(4) 大陸棚が漁業にとって重要な理由を、「太陽光」「プランクトン」の語を使って説明しなさい。
→ 大陸棚は水深が浅いため太陽光が海底近くまで届き、植物プランクトンが光合成をしやすい環境にある。プランクトンが豊富になることで、それを食べる魚介類も多く生息し、好漁場となる。

## 問4: 海流と漁場の関係

(1) 日本周辺を流れる4つの海流を暖流・寒流に分けてそれぞれ答えよ。
→ 暖流：黒潮（日本海流）・対馬海流。寒流：親潮（千島海流）・リマン海流。

(2) 潮目ではなぜ魚が多く集まるのか。海水の性質の違いにふれて詳しく説明しなさい。
→ 暖流と寒流がぶつかる潮目では、温度や塩分濃度の異なる海水が混ざり合い、海水が上下にかき混ぜられる。これにより海底の栄養分が表層に運ばれてプランクトンが大量に繁殖し、それを食べる小魚、さらにそれを食べる大型の魚が集まるため、好漁場になる。

(3) 黒潮（暖流）が流れる太平洋側と、親潮（寒流）が流れる北太平洋側では、沿岸の気候にどのような違いが生じるか。
→ 黒潮が流れる太平洋側の沿岸部は比較的温暖な気候になる。一方、親潮が流れる北太平洋側は冷涼な気候になりやすく、特に夏にはやませ（冷たい北東風）が吹いて冷害の原因になることがある。

(4) 海溝が日本の太平洋側に形成される理由を、プレートの沈み込みにふれて説明しなさい。
→ 太平洋プレートやフィリピン海プレートが日本列島側のプレートの下に沈み込む場所が海溝である。プレートが海底に引きずり込まれることで深い溝が形成され、深さ8000m以上に達する海溝が日本の太平洋側に連なっている。

## 問5: 地形と人間生活の総合考察

(1) 日本では国土の約4分の3が山地・丘陵地であるにもかかわらず、人口の大部分は平野部に集中している。この分布の偏りによって生じる問題点を2つ述べよ。
→ 平野部に人口が集中することで、都市部の過密問題（住宅不足・交通渋滞・環境悪化）が起こる。一方、山間部では過疎化が進み、農業の担い手不足や集落の存続が困難になる。

(2) 三陸海岸はリアス海岸として知られているが、津波の被害を受けやすい理由を地形の特徴から説明しなさい。
→ リアス海岸は入り江が狭く奥深い形状をしている。津波が入り江に入ると、奥に行くほど幅が狭くなるため波が集中し、水位が急激に高くなって大きな被害をもたらす。

(3) 日本の地形が多様であることは、自然災害のリスクを高める一方で、人間生活に恩恵をもたらす面もある。地形がもたらす恩恵を3つ挙げよ。
→ 火山の周辺では温泉が湧き観光資源になる。急流の河川は水力発電に利用できる。変化に富んだ海岸線は養殖や漁業に適した漁港をもたらす。

(4) 甲府盆地や長野盆地で果樹栽培が盛んな理由を、盆地の気候的特徴にもふれて説明しなさい。
→ 盆地は周囲を山に囲まれているため降水量が少なく、昼夜の寒暖差が大きい。この寒暖差が果物の糖度を高めるのに適しており、さらに水はけのよい扇状地が盆地の入り口に広がっていることから、ぶどうやもも、りんごなどの果樹栽培が盛んに行われている。
