# 日本の交通・通信と地域区分 大問小問

## 大問1: 交通手段と新幹線（配点 20点）

日本の交通手段について、次の問いに答えなさい。

(1) 近距離・中距離・長距離の移動に適した交通手段をそれぞれ答えよ。（6点）
→ 近距離：鉄道・バス・自動車。中距離（300〜500km）：新幹線。長距離：航空機。

(2) 東海道新幹線が結ぶ主な都市を答えよ。（3点）
→ 東京・名古屋・新大阪（大阪）。

(3) 2015年3月に延伸開業した新幹線の路線名と、延伸先の都市名を答えよ。（4点）
→ 北陸新幹線。金沢まで延伸した。

(4) 新幹線の整備が沿線地域にもたらす効果を2つ答えよ。（4点）
→ 移動時間の短縮による利便性の向上、観光客の増加や企業進出による経済活性化。

(5) 高速道路の整備が内陸部にもたらした変化を「工場」の語を使って説明しなさい。（3点）
→ 高速道路の整備により内陸部でも輸送が便利になり、沿線に多くの工場が進出するようになった。

## 大問2: 貨物輸送（配点 18点）

日本の貨物輸送について、次の問いに答えなさい。

(1) 日本の貨物輸送で最も割合が高い輸送手段は何か。（2点）
→ 自動車（トラック）。

(2) トラック輸送の長所を「個別配送」の語を使って答えよ。（3点）
→ 戸口から戸口への個別配送ができ、時間や場所の融通がきく。

(3) 船舶による輸送に適した貨物の特徴を答え、具体例を2つ挙げよ。（4点）
→ 重くてかさばるもの。具体例：石油（原油）、鉄鉱石（自動車、木材なども可）。

(4) 航空機による輸送に適した貨物の特徴を答え、具体例を2つ挙げよ。（4点）
→ 軽くて高価な精密品。具体例：ICチップ（半導体）、医薬品（電子部品、宝石なども可）。

(5) 輸送手段を選ぶ際に考慮すべき要素を3つ答えよ。（5点）
→ 輸送するものの重さ・大きさ、輸送にかかるコスト、輸送にかかる時間（距離、緊急性なども可）。

## 大問3: 空港と通信（配点 20点）

日本の空港と通信網について、次の問いに答えなさい。

(1) 成田国際空港がある都道府県名を答えよ。（2点）
→ 千葉県。

(2) 関西国際空港がある都道府県名を答え、海上に建設された理由を説明しなさい。（4点）
→ 大阪府。騒音問題を避けるためと、24時間運用を可能にするため。

(3) 光ファイバーによる通信の特徴を答えよ。（3点）
→ 大容量のデータを高速で通信できる。

(4) インターネットの普及が人々の生活に与えた変化を2つ答えよ。（4点）
→ いつでもどこでも情報を入手できるようになったこと、ネットショッピングやオンラインサービスが広がったこと。

(5) 情報格差（デジタルデバイド）とは何か。また、どのような問題が生じるか答えよ。（4点）
→ 情報技術を使える人と使えない人の間に生じる格差。情報の入手やサービスの利用に差が生まれ、社会的・経済的な不平等につながる。

(6) 情報通信網の整備が地方にもたらす効果を「テレワーク」の語を使って答えよ。（3点）
→ 情報通信網が整備されることでテレワークが可能になり、地方に住みながら都市部の企業で働けるようになる。

## 大問4: 7地方区分（配点 22点）

日本の地方区分について、次の問いに答えなさい。

(1) 日本の7つの地方区分をすべて答えよ。（7点）
→ 北海道地方・東北地方・関東地方・中部地方・近畿地方・中国・四国地方・九州地方。

(2) 関東地方に含まれる都県をすべて答えよ。（3点）
→ 東京都・神奈川県・埼玉県・千葉県・茨城県・栃木県・群馬県。

(3) 東北地方に含まれる県をすべて答えよ。（3点）
→ 青森県・岩手県・宮城県・秋田県・山形県・福島県。

(4) 同じ中部地方でも日本海側と太平洋側で異なる特徴を「気候」にふれて説明しなさい。（4点）
→ 中部地方の日本海側は冬に北西の季節風の影響で大雪が降るのに対し、太平洋側は冬に乾燥した晴天が続くなど、気候が大きく異なる。

(5) 7地方区分だけでは地域の特色を十分にとらえられない理由を説明しなさい。（5点）
→ 同じ地方の中でも気候・産業・人口分布などが地域によって大きく異なるため、7つの区分だけでは細かい特色の違いをとらえることが難しい。

## 大問5: 地域区分の方法（配点 20点）

さまざまな視点による地域区分について、次の問いに答えなさい。

(1) 地域区分とは何か。行う目的とあわせて説明しなさい。（4点）
→ 地域区分とは、ある基準にもとづいて地域をいくつかに分けること。各地域の特色や地域間の違いを明確にするために行う。

(2) 第一次産業の割合による地域区分を行うと、どのようなことがわかるか。（3点）
→ 農業や漁業がさかんな地域と、都市化・工業化が進んだ地域の違いがわかる。

(3) 交通・移動時間による地域区分を行うと、どのようなことがわかるか。（3点）
→ 大都市からの交通の便利さ（アクセスのしやすさ）の違いがわかる。

(4) 自然的特徴による地域区分の具体例を2つ挙げ、それぞれ何がわかるか答えよ。（6点）
→ 気候による区分では日本海側と太平洋側の降水パターンの違いがわかる。標高による区分では山地が多い地域と平野が広がる地域の違いがわかる。

(5) 人文的特徴による地域区分の具体例を2つ挙げ、それぞれ何がわかるか答えよ。（4点）
→ 人口密度による区分では都市部と農村部の違いがわかる。産業別就業者数による区分では農業中心の地域と工業・サービス業中心の地域の違いがわかる。
