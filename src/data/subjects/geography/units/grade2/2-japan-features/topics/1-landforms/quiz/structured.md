# 地形から見た日本の特色 大問小問

## 大問1: 造山帯と日本の山地（配点 18点）

日本列島の地形の成り立ちについて、次の問いに答えなさい。

(1) 日本列島が位置する造山帯の名前を答えよ。（2点）
→ 環太平洋造山帯。

(2) この造山帯で火山活動や地震が活発な理由を、「プレート」という語を使って説明しなさい。（4点）
→ 複数のプレートが押し合う境界にあたるため、プレートの動きによって火山活動や地震が頻繁に起こる。

(3) 日本の国土のうち山地・丘陵地が占める割合を答えよ。（2点）
→ 約4分の3（約75%）。

(4) 日本アルプスとよばれる3つの山脈の名前をすべて答えよ。（3点）
→ 飛騨山脈・木曽山脈・赤石山脈。

(5) フォッサマグナとは何か。「地溝帯」「地質」の語を使って説明しなさい。（4点）
→ 本州中央部にある大きな地溝帯であり、東日本と西日本の地質の境目にあたる。

(6) 東北地方の中央を南北に走る山脈の名前を答えよ。（3点）
→ 奥羽山脈。

## 大問2: 河川と地形（配点 20点）

日本の河川の特徴とそれがつくる地形について、次の問いに答えなさい。

(1) 日本の河川は大陸の河川と比べてどのような特徴があるか。3つの観点で答えよ。（6点）
→ 距離が短い、流れが急である、流域面積が狭い。

(2) (1)のような特徴がある理由を、日本の国土の特徴にふれて説明しなさい。（4点）
→ 日本の国土は南北に細長く、山地が海の近くまで迫っているため。

(3) 河川が山地から平地に出るところにできる扇形の地形を何というか。（2点）
→ 扇状地。

(4) 河口付近に土砂が堆積してできる三角形の地形を何というか。（2点）
→ 三角州。

(5) 台地とはどのような地形か。また、台地で畑作が行われることが多い理由を答えよ。（4点）
→ 台地は周囲より一段高い平らな土地で、高い位置にあるため水が得にくく水田に向かないため、畑作が行われることが多い。

(6) 扇状地や盆地で果樹栽培が盛んに行われる理由を答えよ。（2点）
→ 水はけがよく日当たりもよいため、果樹の栽培に適しているから。

## 大問3: 海岸地形（配点 18点）

日本の海岸線の特徴について、次の問いに答えなさい。

(1) リアス海岸とはどのような海岸か。その成り立ちにふれて説明しなさい。（4点）
→ 山地が海に沈んでできた、入り組んだ複雑な海岸線をもつ海岸。

(2) リアス海岸が見られる日本の代表的な場所を2つ答えよ。（4点）
→ 三陸海岸と志摩半島。

(3) 鳥取県にある有名な海岸地形の名前を答えよ。（2点）
→ 鳥取砂丘。

(4) 人工海岸とはどのような海岸か説明しなさい。（4点）
→ 埋め立てや護岸工事などにより人工的につくられた海岸。

(5) 大陸棚とは何か。深さの目安もふくめて答えよ。（4点）
→ 大陸の周辺に広がる深さ約200mまでの浅い海底のこと。

## 大問4: 海流と漁業（配点 22点）

日本列島周辺の海流について、次の問いに答えなさい。

(1) 日本の太平洋側を流れる暖流の名前と別名を答えよ。（3点）
→ 黒潮（日本海流）。

(2) 日本海側を流れる暖流の名前を答えよ。（2点）
→ 対馬海流。

(3) 日本の太平洋側を流れる寒流の名前と別名を答えよ。（3点）
→ 親潮（千島海流）。

(4) 日本海側を流れる寒流の名前を答えよ。（2点）
→ リマン海流。

(5) 潮目とは何か説明しなさい。（4点）
→ 暖流と寒流がぶつかる場所のこと。

(6) 潮目が好漁場になる理由を、「プランクトン」という語を使って説明しなさい。（4点）
→ 暖流と寒流がぶつかることで海水がかき混ぜられてプランクトンが豊富になり、それを求めて多くの魚が集まるため。

(7) 海溝とは何か。日本周辺の海溝を1つ具体的に答えよ。（4点）
→ 海底にある細長く深い溝で、深さ8000m以上に達するものもある。例：伊豆・小笠原海溝。

## 大問5: 日本の地形と土地利用の総合問題（配点 22点）

次の文章を読んで、下の問いに答えなさい。

「日本の平野部では水田が広がり、台地では畑が多い。山のふもとには（ア）がつくられ果樹園が見られ、河口付近には（イ）が広がり水田が多い。また、内陸の（ウ）では昼夜の気温差を生かした果物の栽培が盛んである。海岸線は変化に富み、（エ）海岸では養殖業が営まれている。」

(1) 文中の（ア）～（エ）にあてはまる語をそれぞれ答えよ。（8点）
→ ア：扇状地　イ：三角州　ウ：盆地　エ：リアス

(2) 台地で水田ではなく畑作が行われる理由を答えよ。（4点）
→ 台地は周囲より高い位置にあるため水が得にくく、水田に適さないから。

(3) 日本最大の平野の名前を答え、どの地方にあるか答えよ。（4点）
→ 関東平野。関東地方にある。

(4) 甲府盆地で盛んに栽培されている代表的な果物を1つ答えよ。（2点）
→ ぶどう（もも）。

(5) 桜島・雲仙岳は活火山として知られている。日本に活火山が多い理由を造山帯にふれて説明しなさい。（4点）
→ 日本は環太平洋造山帯に位置し、プレートの境界付近にあるためマグマが上昇しやすく、活火山が多い。
