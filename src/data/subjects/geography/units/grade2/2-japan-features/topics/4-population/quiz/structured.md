# 日本の人口 大問小問

## 大問1: 人口の推移と少子高齢化（配点 20点）

日本の人口の変化について、次の問いに答えなさい。

(1) 1940年代後半に出生数が急増した現象を何というか。（2点）
→ ベビーブーム。

(2) 2019年現在、日本の総人口はおよそ何人か。（2点）
→ 約1億2600万人。

(3) 少子高齢化とはどのような現象か。「出生数」「高齢者」の語を使って説明しなさい。（4点）
→ 出生数が減少して子どもの割合が低下する一方、高齢者の割合が増加する現象。

(4) 少子化の原因を2つ答えよ。（4点）
→ 出生率の低下（晩婚化・未婚化の進行、子育てにかかる経済的負担の増加など）。

(5) 高齢化が進む原因を答えよ。（2点）
→ 医療の発達による平均寿命の延び。

(6) 少子高齢化がもたらす社会的な問題を3つ答えよ。（6点）
→ 労働力不足、社会保障費の増大、地域社会の衰退。

## 大問2: 人口ピラミッド（配点 18点）

人口ピラミッドについて、次の問いに答えなさい。

(1) 人口ピラミッドとは何か説明しなさい。（4点）
→ 人口の年齢構成を男女別に横棒グラフで表した図。横軸が人口、縦軸が年齢層を示す。

(2) 人口ピラミッドの3つの型を、若い型から順に答えよ。（6点）
→ 富士山型→つりがね型→つぼ型。

(3) 富士山型（ピラミッド型）はどのような特徴の人口構成か。（2点）
→ 若年層が多く、高齢層が少ない人口構成。

(4) 現在の日本の人口ピラミッドは何型に近いか。（2点）
→ つぼ型。

(5) 日本の人口ピラミッドがつぼ型に変化した理由を「出生率」「平均寿命」の語を使って説明しなさい。（4点）
→ 出生率が低下して若年層が減少する一方、平均寿命の延びにより高齢層が増加したため。

## 大問3: 都市への人口集中（配点 22点）

日本の都市部への人口集中について、次の問いに答えなさい。

(1) 日本の三大都市圏を答えよ。（3点）
→ 東京圏・大阪圏・名古屋圏。

(2) 地方中枢都市とは何か。代表例を2つ挙げて説明しなさい。（4点）
→ 地方の政治・経済・文化の中心となる大都市。例：札幌・仙台・広島・福岡（うち2つ）。

(3) ドーナツ化現象とは何か。「地価」「郊外」の語を使って説明しなさい。（4点）
→ 都心部の地価が高騰したため、人口が郊外に移り住み、都心の人口が減少する現象。

(4) ニュータウンとは何か説明しなさい。（3点）
→ 都市の郊外に計画的に開発された大規模な住宅地。

(5) 1990年代以降に都心回帰が起こった理由を答えなさい。（4点）
→ バブル崩壊後に都心部の地価が下がり、マンション建設が進んだため、都心に住む人が再び増加した。

(6) 過密地域で生じる問題を2つ答えよ。（4点）
→ 通勤ラッシュ、住宅不足（交通渋滞、ごみ処理問題なども可）。

## 大問4: 過疎化の問題と対策（配点 20点）

農村・山間部・離島の過疎化について、次の問いに答えなさい。

(1) 過疎化とはどのような現象か説明しなさい。（4点）
→ 農村・山間部・離島などで、若者を中心に都市部へ人口が流出し、人口が急減する現象。

(2) 過疎地域で高齢化が特に進む理由を答えなさい。（4点）
→ 若者が進学・就職のために都市に移り、地域に残るのは高齢者が多くなるため。

(3) 過疎化が進むことで起こる問題を3つ答えよ。（6点）
→ 学校や病院の統廃合、公共交通機関の廃止、農地の荒廃。

(4) IターンとUターンの違いを説明しなさい。（4点）
→ Iターンは都市出身者が地方に移住すること。Uターンは地方出身者が都市で働いた後に故郷に戻ること。

(5) 過疎対策として行われている取り組みを1つ具体的に答えよ。（2点）
→ 地域おこし協力隊の活用、移住促進のための補助金制度など（いずれか1つ）。

## 大問5: 人口密度と総合問題（配点 20点）

次の問いに答えなさい。

(1) 人口密度の求め方を答えよ。（2点）
→ 人口÷面積（1km2あたりの人口）。

(2) ある地域の人口が50万人、面積が200km2であるとき、人口密度を求めよ。（4点）
→ 500000÷200＝2500人/km2

(3) 日本の三大都市圏に人口が集中する理由を「仕事」「教育」の語を使って答えなさい。（4点）
→ 大都市圏には仕事の機会が豊富にあり、教育機関も充実しているため、若者を中心に人が集まりやすい。

(4) 過密と過疎の問題を解決するためにはどのような対策が必要か。「地方」「都市」の語を使って述べなさい。（6点）
→ 地方に仕事や教育の場を整備し、都市からの人口移動を促す。また、都市の過密緩和のために企業や行政機関の地方移転を進め、地方と都市のバランスある発展を図る必要がある。

(5) 少子高齢化と人口減少が同時に進むと、社会保障制度にどのような影響があるか述べなさい。（4点）
→ 高齢者の増加により年金や医療費などの社会保障費が増大する一方、それを負担する現役世代（労働力人口）が減少し、一人あたりの負担が重くなる。
