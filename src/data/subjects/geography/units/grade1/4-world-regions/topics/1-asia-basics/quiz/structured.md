# アジア州 大問小問

## 大問1: アジア州の自然環境（配点 20点）

アジア州の自然環境について、次の問いに答えなさい。

(1) アジア州の気候に大きな影響を与える季節風（モンスーン）について、夏と冬それぞれの風向きと特徴を説明しなさい。（4点）
→ 夏は海から大陸に向かって湿った風が吹き雨季をもたらす。冬は大陸から海に向かって乾いた風が吹き乾季となる。

(2) ヒマラヤ山脈やチベット高原を含む地域が「世界の屋根」とよばれる理由を説明しなさい。（4点）
→ ヒマラヤ山脈は世界最高峰のエベレストを含み、チベット高原は標高4000m以上の広大な高原であり、世界で最も標高の高い地域であるから。

(3) アジア州を流れる主要な河川を三つ挙げ、それぞれどの地域を流れるか答えなさい。（6点）
→ 黄河・長江（中国）、ガンジス川（インド）、メコン川（東南アジア）。

(4) 西アジアや中央アジアが乾燥帯に属する理由を「季節風」の語を使って説明しなさい。（3点）
→ 海から遠く季節風の影響が届きにくいため、降水量が極めて少なく乾燥帯となっている。

(5) 季節風の影響を受ける地域で稲作が盛んである理由を説明しなさい。（3点）
→ 夏の季節風が大量の雨をもたらすため、稲作に必要な水が豊富に確保できるから。

## 大問2: 中国の経済と社会（配点 22点）

中国の経済発展と社会問題について、次の問いに答えなさい。

(1) 中国の人口の約90%を占める民族の名称を答えよ。（2点）
→ 漢族。

(2) 中国が人口増加を抑制するために実施していた政策の名称と、その廃止年を答えなさい。（4点）
→ 一人っ子政策。2015年に廃止された。

(3) 経済特区とは何か説明しなさい。また、代表的な経済特区の都市名を一つ答えよ。（5点）
→ 外国企業を積極的に受け入れるために税の優遇や規制の緩和を行った地域のこと。代表的な都市はシェンチェン（深圳）。

(4) 中国が「世界の工場」とよばれるようになった背景を「労働力」の語を使って説明しなさい。（5点）
→ 経済特区を設けて外国企業を受け入れ、豊富で安い労働力を背景に大量の工業製品を生産するようになったから。

(5) 中国で沿岸部と内陸部の間に生じている問題と、政府がこの問題に対して行っている政策をそれぞれ答えなさい。（6点）
→ 沿岸部と内陸部の経済格差が拡大している問題。政府は西部大開発という政策を進めて、内陸部のインフラ整備や産業振興を図っている。

## 大問3: 東南アジアの農業と経済（配点 20点）

東南アジアについて、次の問いに答えなさい。

(1) 東南アジアで季節風を利用して年に2回米を収穫する農法を何というか。（3点）
→ 二期作。

(2) プランテーション農業とは何か説明し、東南アジアで栽培されている代表的な作物を二つ答えなさい。（5点）
→ 大規模な農園で輸出向けに単一の作物を栽培する農業のこと。代表的な作物は天然ゴムとパーム油（油やし）。

(3) ASEAN（東南アジア諸国連合）の加盟国数と主な活動目的を答えなさい。（4点）
→ 10か国が加盟。経済協力や政治的安定の維持を主な目的としている。

(4) 東南アジアの海岸部に広がるマングローブとは何か説明し、減少が問題になっている理由を述べなさい。（4点）
→ 熱帯・亜熱帯の海岸に生育し、潮の満ち引きの影響を受ける森林。えび養殖場への転用などで減少しており、海岸の浸食防止や生態系への悪影響が懸念されている。

(5) 東南アジアの都市部で問題となっているスラムが形成される原因を説明しなさい。（4点）
→ 工業化に伴い農村部から都市部への人口流入が急増し、住居や生活基盤が整わないまま人が集中したため、スラムが形成された。

## 大問4: 南アジアの特徴（配点 18点）

南アジアについて、次の問いに答えなさい。

(1) 南アジアで最も広く信仰されている宗教の名称を答えよ。（2点）
→ ヒンドゥー教。

(2) ガンジス川がヒンドゥー教徒にとって重要な理由を述べなさい。（3点）
→ ガンジス川は聖なる川とされ、沐浴することで罪が清められると信じられているから。

(3) インドでICT産業が急成長している背景を二つ挙げなさい。（4点）
→ 英語力の高さと数学教育の水準が高いこと。アメリカとの時差を活用して24時間体制のサービスが可能であること。

(4) インドのICT産業の中心都市の名前を答えよ。（3点）
→ バンガロール。

(5) ガンジス川の上流域と下流域で降水量に違いがある理由を「季節風」の語を使って説明しなさい。（6点）
→ 季節風は海から吹いてくるため、海に近い下流域では大量の雨をもたらすが、内陸の上流域まで届くころには水分が減少するため降水量が少なくなる。

## 大問5: 西アジア・中央アジアの資源と経済（配点 20点）

西アジア・中央アジアについて、次の問いに答えなさい。

(1) 西アジア・中央アジアで広く信仰されている宗教と、その聖典の名称をそれぞれ答えよ。（4点）
→ イスラム教（イスラーム）。聖典はコーラン。

(2) OPEC（石油輸出国機構）とは何か、その役割を説明しなさい。（4点）
→ 産油国が結成した国際組織で、石油の生産量や価格を調整して産油国の利益を守る役割を持つ。

(3) 石油化学コンビナートとは何か説明しなさい。（4点）
→ 石油を原料にして、プラスチックや化学繊維などさまざまな化学製品を効率よく生産するために、関連工場を一か所に集めた大規模な工業施設。

(4) 西アジア・中央アジアで産出されるレアメタルがなぜ重要視されているか説明しなさい。（4点）
→ スマートフォンや電気自動車などの電子機器に不可欠な材料であり、産出量が限られているため国際的に重要視されている。

(5) ドバイなどの中東の都市が石油に依存しない経済を目指している理由を述べなさい。（4点）
→ 石油は限りある資源であり、将来的に枯渇する可能性があるため、観光業や先端技術産業など石油以外の産業を育成して持続的な経済発展を目指している。
