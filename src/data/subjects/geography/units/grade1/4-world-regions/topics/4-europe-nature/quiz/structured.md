# ヨーロッパ州 大問小問

## 大問1: ヨーロッパの自然環境（配点 20点）

ヨーロッパ州は北海道よりも高い緯度に位置しているが、冬でも比較的温暖な地域が多い。また、地域によって異なる気候や地形の特徴が見られる。

(1) ヨーロッパが高緯度の割に温暖である理由を、「北大西洋海流」「偏西風」の2語を使って説明しなさい。（4点）
→ 暖流の北大西洋海流が沿岸に暖かい空気をもたらし、偏西風がその暖かさを内陸まで運ぶため。

(2) ヨーロッパ南部の地中海沿岸に見られる気候の名称を答えよ。また、その気候の夏と冬の特徴をそれぞれ簡潔に述べよ。（4点）
→ 地中海性気候。夏は高温で乾燥し、冬は比較的温暖で雨が多い。

(3) 地中海性気候の地域で栽培がさかんな作物を2つ答えよ。（4点）
→ オリーブ、小麦（ぶどう、オレンジなども可）。

(4) スカンディナビア半島の海岸線に見られる、氷河によってつくられた地形を何というか。またそれがどのようにしてできたか説明しなさい。（4点）
→ フィヨルド。氷河が山を削ってできたU字形の深い谷に、海水が入り込んでできた入り江。

(5) 家畜の飼育と穀物の栽培を組み合わせた農業を何というか。（4点）
→ 混合農業。

## 大問2: ヨーロッパの文化と多様性（配点 18点）

ヨーロッパには多くの民族と言語が存在し、宗教もさまざまな宗派に分かれている。

(1) ヨーロッパの言語は大きく3つの系統に分けられる。その3つを答えよ。（3点）
→ ゲルマン系、ラテン系、スラブ系。

(2) 次の言語をそれぞれの系統に分類しなさい。（6点）
　英語・フランス語・ロシア語・ドイツ語・スペイン語・ポーランド語
→ ゲルマン系：英語・ドイツ語、ラテン系：フランス語・スペイン語、スラブ系：ロシア語・ポーランド語。

(3) ヨーロッパで広く信仰されているキリスト教の3つの主な宗派を答えよ。（3点）
→ カトリック、プロテスタント、正教会（東方正教会）。

(4) 近年、ヨーロッパでイスラム教の信者が増加している理由を答えよ。（3点）
→ 中東やアフリカなどイスラム教を信仰する地域からの移民が増加したため。

(5) 言語や宗教の多様性がヨーロッパ統合にとってどのような課題をもたらすか、説明しなさい。（3点）
→ 言語や宗教、文化的な背景が異なるため、加盟国間での政策の統一や相互理解が難しくなることがある。

## 大問3: EU統合のしくみ（配点 20点）

二度の世界大戦を経験したヨーロッパでは、平和と経済発展のために地域統合が進められた。

(1) EUの正式名称を日本語で答えよ。また、EUが発足した年を答えよ。（4点）
→ ヨーロッパ連合。1993年。

(2) EUの前身となった組織の略称と正式名称を答えよ。（4点）
→ EC（ヨーロッパ共同体）。

(3) EU加盟国間の統合によって可能になったことを3つ答えよ。（6点）
→ 共通通貨ユーロの使用、関税の撤廃による自由貿易、パスポート不要での国境移動（シェンゲン協定）。

(4) ヨーロッパの複数の国が協力して生産している航空機の名前を答えよ。また、共同生産の利点を説明しなさい。（6点）
→ エアバス。一国では負担しきれない開発費用を各国で分担でき、それぞれの技術力を結集して高品質な製品をつくることができる。

## 大問4: 環境と持続可能な社会（配点 22点）

ヨーロッパでは環境問題に対する意識が高く、さまざまな取り組みが行われている。

(1) ヨーロッパで深刻になった酸性雨は何が原因で発生したか。（3点）
→ 工場や自動車から排出される排気ガス（硫黄酸化物や窒素酸化物）。

(2) 酸性雨による被害を2つ答えよ。（4点）
→ 森林の枯死、湖の生態系の破壊（建造物の腐食なども可）。

(3) ライン川の水質改善が成功した理由を、「国際的」という語を使って説明しなさい。（4点）
→ ライン川の流域にある複数の国が国際的に協力して排水規制を進め、水質改善に取り組んだため。

(4) パークアンドライドの仕組みを説明し、その目的を2つ答えよ。（5点）
→ 郊外の駐車場に車を停めて公共交通機関で都市の中心部に移動する仕組み。目的は交通渋滞の緩和と排気ガスの削減。

(5) ルーラル・ツーリズムとエコツーリズムの違いを説明しなさい。（6点）
→ ルーラル・ツーリズムは農村や田舎に滞在して農業体験や自然とのふれあいを楽しむ観光形態。エコツーリズムは自然環境を保全しながら、その自然を観光資源として活用する観光形態。

## 大問5: EUの課題とロシア（配点 20点）

EU統合は成果を上げる一方で、さまざまな課題も抱えている。また、ヨーロッパとアジアにまたがるロシアも重要な存在である。

(1) EU加盟国間の経済格差について、「西ヨーロッパ」「東ヨーロッパ」の語を使って説明しなさい。（4点）
→ 西ヨーロッパの先進国と東ヨーロッパの新規加盟国の間に所得や生活水準の大きな差があり、GNIの格差は最大で約8倍にもなる。

(2) ヨーロッパ中央銀行（ECB）の本部がある都市名と国名を答えよ。（4点）
→ フランクフルト（ドイツ）。

(3) 2020年にEUを離脱した国名と、その離脱の通称を答えよ。（4点）
→ イギリス。ブレグジット（Brexit）。

(4) ロシアの自然の特徴について、「タイガ」「ステップ」の2語を使って説明しなさい。（4点）
→ ロシアのシベリア地方にはタイガと呼ばれる広大な針葉樹林が広がり、南部にはステップと呼ばれる草原地帯がある。

(5) ロシアが産出する主な鉱産資源を2つ答え、それがヨーロッパに対してどのような役割を果たしているか述べよ。（4点）
→ 石油と天然ガス。パイプラインを通じてヨーロッパの国々にエネルギーを供給する重要な役割を果たしている。
