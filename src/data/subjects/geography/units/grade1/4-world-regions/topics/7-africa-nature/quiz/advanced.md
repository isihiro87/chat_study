# アフリカ州 発展問題

## 問1: アフリカの自然環境と気候の多様性

アフリカ大陸の自然環境について、次の問いに答えなさい。

(1) アフリカの赤道を中心とした気候帯の対称的な分布を、北半球と南半球の両方について説明しなさい。
→ 赤道付近に熱帯雨林が広がり、その外側（南北両方向）にサバナが分布する。さらに外側にはステップがあり、北半球側にはサハラ砂漠、南半球側にはカラハリ砂漠やナミブ砂漠が広がる。北端と南端には温帯（地中海性気候）がみられ、赤道を軸にほぼ対称的な気候分布をなしている。

(2) ナイル川が「南から北に流れる」という特徴と、古代エジプト文明との関係を「氾濫」「肥沃」の語を使って説明しなさい。
→ ナイル川は上流の熱帯地域で雨季に増水し、下流のエジプトまで北に向かって流れていく。定期的な氾濫によって上流から運ばれた土砂が下流の土地を肥沃にし、古代エジプト文明の農業を支える基盤となった。

(3) キリマンジャロの氷河が近年縮小している問題について、「地球温暖化」の語を使って説明し、アフリカの水資源への影響を述べなさい。
→ 地球温暖化の影響で気温が上昇し、キリマンジャロの山頂付近の氷河が急速に縮小している。氷河は周辺地域の水源となっているため、氷河の消失は農業用水や生活用水の不足につながり、ふもとの住民の生活に深刻な影響を及ぼすおそれがある。

(4) 砂漠化の「自然的要因」と「人為的要因」をそれぞれ2つずつ挙げ、砂漠化を防ぐための国際的な取り組みを1つ述べなさい。
→ 自然的要因：①降水量の長期的な減少、②干ばつの頻発。人為的要因：①過放牧による植生の破壊、②過耕作による地力の低下。国際的な取り組みとして、サハラ砂漠の南縁を横断する「アフリカ緑の長城」計画があり、幅15キロメートル・東西約8000キロメートルにわたって植林を行い、砂漠の拡大を防ぐことを目指している。

## 問2: 植民地支配とモノカルチャー経済の構造

アフリカの経済的課題について、次の問いに答えなさい。

(1) 植民地時代にプランテーションで栽培された作物が、独立後も主要輸出品であり続ける理由を「インフラ」「技術」の語を使って説明しなさい。
→ 植民地時代にプランテーション経営のためのインフラ（道路・港湾・加工施設）が整えられ、栽培技術も商品作物に特化して蓄積された。独立後に新たな産業を育てるための資金やインフラ、技術が不足していたため、既存の商品作物の輸出に頼らざるを得ない状況が続いた。

(2) カカオの生産国と消費国の関係を「フェアトレード」の考え方を含めて説明しなさい。
→ カカオはガーナやコートジボワールなどのアフリカの国々で主に生産されるが、チョコレートに加工して消費するのは先進国が中心である。生産者は安い価格で買いたたかれることが多く、貧困から抜け出せない。フェアトレードは生産者に適正な価格を支払い、生活の改善や児童労働の防止を目指す取り組みである。

(3) アフリカのレアメタル資源をめぐる問題点を「紛争鉱物」の観点から説明しなさい。
→ コンゴ民主共和国などでは、レアメタル（コバルト・タンタルなど）の採掘による利益が武装勢力の資金源となり、紛争を長引かせている。こうした鉱物は「紛争鉱物」と呼ばれ、採掘現場では児童労働や劣悪な労働環境も問題となっている。国際社会では紛争鉱物の使用を規制する動きが広がっている。

(4) モノカルチャー経済から脱却するために、アフリカの国々が取り組むべき方策を「産業の多角化」「教育」の語を使って2つ提案しなさい。
→ ①産業の多角化として、農産物の加工業や観光業、ICT産業など新たな産業を国内で育成し、特定の一次産品に依存しない経済構造をつくる。②教育の充実により人材を育成し、技術力や専門知識をもった労働者を増やすことで、付加価値の高い産業を発展させる。

## 問3: ICTの活用と社会変革

アフリカにおけるICT（情報通信技術）の活用について、次の問いに答えなさい。

(1) アフリカで固定電話よりも携帯電話が先に普及した現象を「リープフロッグ（かえる跳び）現象」という。この現象が起きた理由を説明しなさい。
→ 固定電話は電話線を各家庭に敷設する必要があり、広大で人口密度の低いアフリカでは莫大な費用がかかる。一方、携帯電話は基地局を設置するだけで広範囲をカバーできるため、固定電話の段階を飛び越えて一気に携帯電話が普及した。

(2) ケニアの「エムペサ（M-Pesa）」を例に、携帯電話がアフリカの金融サービスをどのように変えたか、「銀行口座」の語を使って説明しなさい。
→ エムペサは携帯電話のSMSを使って送金や支払いができるサービスである。銀行口座を持たない人々（農村部の住民や貧困層）でも携帯電話があれば利用でき、家族への送金や商品の売買の決済が可能になった。銀行の支店が少ない地域でも金融サービスにアクセスできるようになり、経済活動が活性化した。

(3) ルワンダがICT立国を目指した背景と成果を、「紛争」「復興」の語を使って説明しなさい。
→ 1994年の民族紛争で国土が荒廃したルワンダは、復興の柱としてICT産業の育成を選んだ。鉱物資源が少ないルワンダにとって、ICTは大規模な設備投資がなくても発展できる分野であった。全国に光ファイバー網を整備し、IT教育を推進した結果、首都キガリはアフリカ有数のIT都市となり、経済成長を実現している。

## 問4: アフリカの民族紛争と国際社会

アフリカの紛争と国際社会の関わりについて、次の問いに答えなさい。

(1) アフリカの民族紛争が起きる原因を「植民地」「国境」「民族」の3つの語を使って構造的に説明しなさい。
→ 植民地時代にヨーロッパ諸国が民族の居住範囲を無視して直線的な国境を引いたため、独立後に一つの国の中に対立する複数の民族が共存することになった。また、植民地支配の中で特定の民族が優遇されるなどの分断政策がとられたことで、民族間の不信感が深まり、独立後も紛争の火種となった。

(2) ルワンダの和解プロセスから学べることを「対話」「共存」の語を使って述べなさい。
→ ルワンダではガチャチャ裁判などを通じて、加害者と被害者が直接向き合う対話の場を設けた。過去の過ちを認め、真実を明らかにした上で赦しを求めるプロセスにより、異なる民族の共存を可能にした。敵対した人々が同じ社会で暮らすためには、処罰だけでなく対話による相互理解が重要であることを示している。

(3) アフリカ連合（AU）の役割と課題を「紛争解決」「経済統合」の語を使って説明しなさい。
→ AUは加盟55か国の協力により、紛争解決のための平和維持活動や調停を行い、アフリカ大陸の安定を目指している。また、経済統合に向けてアフリカ大陸自由貿易圏（AfCFTA）の設立を推進し、域内の貿易を活性化させようとしている。課題としては、加盟国間の利害対立や活動資金の不足があり、紛争解決の実効性を高めることが求められている。

## 問5: アフリカと日本・世界の関わり

アフリカと国際社会の関わりについて、次の問いに答えなさい。

(1) 日本がTICADを通じてアフリカを支援する意義を、「資源」「市場」「国際社会」の語を使って多角的に説明しなさい。
→ アフリカはレアメタルなどの資源が豊富であり、日本にとって重要な資源の調達先である。また、人口が急増するアフリカは将来の巨大な市場としても期待されている。さらに、国際社会における日本の発言力を高め、信頼関係を築くうえでも、アフリカの開発支援に積極的に関わることは大きな意義がある。

(2) フェアトレード商品の購入が、アフリカの生産者にどのような影響を与えるか、「適正な価格」「児童労働」の語を使って説明しなさい。
→ フェアトレードにより適正な価格で商品が取引されると、生産者の収入が安定し、生活水準が向上する。収入が増えることで子どもを働かせる必要がなくなり、児童労働の減少につながる。また、フェアトレードの基準では児童労働の禁止が求められているため、子どもたちが教育を受ける機会が守られる。

(3) 「アフリカの課題は世界全体の課題である」という考え方について、「人口」「環境」「経済」の語を使ってあなたの意見を述べなさい。
→ アフリカの人口は急増しており、食料や雇用の問題は移民や難民として世界に影響を及ぼす。砂漠化や森林減少などの環境問題は地球温暖化を加速させ、世界全体の気候に影響する。また、経済のグローバル化により、アフリカの不安定な経済は国際市場や資源供給にも影響を与える。こうした相互依存の関係から、アフリカの課題は世界全体で取り組むべき共通の課題であるといえる。
