# 南アメリカ州 大問小問

## 大問1: 南アメリカ州の自然環境（配点 20点）

次の文を読んで、各問いに答えなさい。

南アメリカ大陸の西側には、南北約（ ア ）kmにわたって（ イ ）山脈が連なっている。大陸の北部には世界最大の流域面積を持つ（ ウ ）川が流れ、その流域には（ エ ）とよばれる広大な熱帯雨林が広がっている。また、大陸の南部にはラプラタ川の流域に（ オ ）とよばれる大草原が広がり、牧畜がさかんである。

(1) 文中の（ア）〜（オ）に当てはまる語句をそれぞれ答えよ。（各2点・計10点）
→ ア：7500　イ：アンデス　ウ：アマゾン　エ：セルバ　オ：パンパ

(2) アンデス山脈の最高峰は標高およそ何mを超えるか。（2点）
→ 6000mを超える。

(3) ギアナ高地はどのような地形的特徴を持つか、簡潔に説明せよ。（4点）
→ テーブル状の台地が連なる高地で、南アメリカ大陸の北部に位置する。

(4) パンパが広がる国名を答え、そこで行われている農業の特徴を説明せよ。（4点）
→ アルゼンチン。広大な草原を利用して牛などの牧畜（放牧）がさかんに行われている。

## 大問2: 民族と文化（配点 18点）

南アメリカ州の歴史と文化について、次の問いに答えなさい。

(1) アンデス山脈の高地にかつて栄えた帝国の名前を答えよ。（2点）
→ インカ帝国。

(2) 16世紀以降、南アメリカを植民地として支配した二つのヨーロッパの国を答えよ。（4点）
→ スペインとポルトガル。

(3) 先住民とヨーロッパ系の人々の混血の人々を何とよぶか。（2点）
→ メスティーソ。

(4) ブラジルの公用語がポルトガル語である理由を、歴史的背景にふれて説明せよ。（4点）
→ 16世紀以降、ブラジルはポルトガルの植民地として支配されたため、ポルトガル語が公用語として定着した。

(5) 20世紀初頭以降、日本からブラジルへ多くの人が移住した。この人々を何とよぶか。また、移住の主な目的を答えよ。（4点）
→ 日系移民。コーヒー農園などでの労働力として移住した。

(6) 南アメリカ州で広く信仰されているキリスト教の宗派を答えよ。（2点）
→ カトリック。

## 大問3: 農業と輸出品（配点 20点）

次の表は南アメリカ州の主な農産物とその生産国を示したものである。

| 農産物 | 主な生産国 |
|--------|-----------|
| A | ブラジル |
| 大豆 | B |
| バナナ | エクアドルなど |
| 砂糖の原料 | ブラジル |

(1) 表中のAに当てはまる農産物を答えよ。（2点）
→ コーヒー豆。

(2) 表中のBに当てはまる国名を答えよ。（2点）
→ ブラジル。

(3) 砂糖の原料となる作物の名前を答えよ。（2点）
→ サトウキビ。

(4) アルゼンチンのパンパで行われている農業の形態とその特徴を説明せよ。（4点）
→ 牧畜。パンパの広大な草原を利用して牛の放牧がさかんに行われ、牛肉は重要な輸出品となっている。

(5) ブラジルで大豆の生産量が増加した結果、どのような環境問題が発生しているか説明せよ。（4点）
→ 大豆畑を拡大するために熱帯雨林が大規模に伐採され、森林面積が減少している。

(6) サトウキビから作られる再生可能な燃料の名前を答え、その利用方法を説明せよ。（6点）
→ バイオエタノール。ブラジルでは自動車の燃料として広く利用されており、化石燃料に代わる再生可能エネルギーとして注目されている。

## 大問4: 鉱産資源（配点 16点）

南アメリカ州の鉱産資源について、次の問いに答えなさい。

(1) ブラジルで大量に産出される、鉄の原料となる鉱石の名前を答えよ。（2点）
→ 鉄鉱石。

(2) チリやペルーで多く産出される金属の鉱石を答えよ。（2点）
→ 銅鉱石。

(3) アルミニウムの原料となる鉱石で、南アメリカでも産出されるものは何か。（2点）
→ ボーキサイト。

(4) 携帯電話やコンピュータなど先端技術に不可欠な、産出量の少ない金属の総称を答えよ。（2点）
→ レアメタル。

(5) 鉱産資源の開発が環境に与える影響を2つ答えよ。（4点）
→ 採掘のために森林が伐採される。採掘による土壌や水質の汚染が起きる。

(6) 南アメリカ州が鉱産資源に恵まれている理由として考えられることを、地形の特徴にふれて説明せよ。（4点）
→ アンデス山脈やブラジル高原など大規模な山脈や高原があり、地殻変動や火山活動によって多様な鉱物が地中に形成されたため。

## 大問5: 環境問題と持続可能な開発（配点 26点）

次の文を読んで、各問いに答えなさい。

ブラジルのアマゾン川流域では、近年、大規模な（ ア ）の開発によって熱帯雨林が急速に減少している。一方で、サトウキビから（ イ ）を生産し、自動車の燃料とする取り組みや、森林を保全しながら農作物を育てる（ ウ ）、自然を守りながら観光収入を得る（ エ ）などの持続可能な開発の取り組みも進んでいる。

(1) 文中の（ア）に当てはまる語句を2つ答えよ。（4点）
→ 大豆畑と牧場。

(2) 文中の（イ）〜（エ）に当てはまる語句をそれぞれ答えよ。（各2点・計6点）
→ イ：バイオエタノール　ウ：アグロフォレストリー　エ：エコツーリズム

(3) 焼畑農業の仕組みを「伐採」「灰」「移動」の3語を使って説明せよ。（6点）
→ 森林を伐採して焼き、その灰を肥料として作物を育てる。数年たって地力が落ちると、別の場所に移動して同じことを繰り返す農業である。

(4) 熱帯雨林が減少することで起きる地球規模の問題を2つ答えよ。（4点）
→ 生物多様性が失われる。二酸化炭素の吸収量が減り、地球温暖化が進む。

(5) 「持続可能な開発」とはどのような考え方か、「経済」「環境」の語を使って説明せよ。（6点）
→ 経済の発展を進めながらも環境の保全を両立させ、将来の世代も資源や自然の恩恵を受けられるようにする開発のあり方。
