# オセアニア州 大問小問

## 大問1: オセアニア州の自然環境（配点 20点）

オセアニア州の自然環境について、次の問いに答えなさい。

(1) オーストラリア大陸が「乾燥大陸」とよばれる理由を、降水量に着目して説明しなさい。（4点）
→ 大陸の約3分の2の地域で年間降水量が500mm未満であり、内陸部の大部分が砂漠や草原で占められているため。

(2) 太平洋の島々を大きく三つの海域に分けたとき、それぞれの名称を答えなさい。（3点）
→ ミクロネシア・メラネシア・ポリネシア。

(3) 太平洋の島々に多く見られる二つの種類の島を挙げ、それぞれの特徴を簡潔に説明しなさい。（4点）
→ 火山島は海底火山の噴火によってできた島で比較的標高が高い。サンゴ礁の島はサンゴが堆積してできた島で標高が非常に低い。

(4) オーストラリアの北東部の海に広がるグレートバリアリーフとは何か説明しなさい。（4点）
→ 約2000kmにわたって広がる世界最大のサンゴ礁で、多様な海洋生物が生息する自然環境として知られている。

(5) ニュージーランドの南島と北島の気候の違いについて述べなさい。（5点）
→ 南島は温帯に属し比較的冷涼な気候であるのに対し、北島はやや温暖な気候である。ニュージーランド全体としてオーストラリアに比べて降水量が多く、緑豊かな環境が広がっている。

## 大問2: オセアニア州の文化と先住民（配点 18点）

オセアニア州の文化について、次の問いに答えなさい。

(1) オーストラリアの先住民であるアボリジニと、ニュージーランドの先住民であるマオリについて、それぞれの名称を答えよ。（2点）
→ アボリジニ（オーストラリア）、マオリ（ニュージーランド）。

(2) アボリジニが世界最古の文化の一つとされる根拠を述べなさい。（4点）
→ アボリジニは約5万年前からオーストラリアに暮らしてきたとされており、その長い歴史の中で独自の文化や伝統を受けついできたため。

(3) マオリの伝統文化として知られる「ハカ」とは何か説明し、現在どのような場面で行われているか述べなさい。（4点）
→ ハカはマオリの伝統的な踊りで、体をたたいたり足を踏みならしたりしながら力強く舞う。現在はニュージーランドのラグビーチームが試合前に行うことで世界的に知られている。

(4) イギリスの植民地化がオセアニアの文化にどのような影響を与えたか、「キリスト教」「英語」の語を使って説明しなさい。（4点）
→ イギリスの植民地化によってキリスト教が広まり、英語が公用語として定着した。その一方で先住民の伝統文化や言語が衰退する影響もあった。

(5) ウルル（エアーズロック）がアボリジニにとって重要な理由を述べなさい。（4点）
→ ウルルはアボリジニにとって古くからの聖地であり、精神的・文化的に深い意味を持つ場所である。アボリジニの権利回復の象徴としても重要視されている。

## 大問3: オーストラリアの産業と資源（配点 22点）

オーストラリアの産業について、次の問いに答えなさい。

(1) オーストラリアでさかんな農畜産業を三つ挙げなさい。（3点）
→ 牧羊（羊の放牧）、牧牛（牛の放牧）、小麦の栽培。

(2) オーストラリアで産出される主な鉱産資源を三つ挙げ、それぞれの主な産出地域を答えなさい。（6点）
→ 鉄鉱石（北西部）、石炭（東部）、ボーキサイト（北部など）。

(3) 1960年代と現在のオーストラリアの輸出品第1位をそれぞれ答え、この変化が起こった背景を説明しなさい。（5点）
→ 1960年代は羊毛が第1位、現在は鉄鉱石が第1位。アジア諸国の工業化にともなう鉱産資源の需要増加により、鉄鉱石の輸出が急増したため。

(4) オーストラリアの内陸部の鉱山から港まで鉱産資源を運ぶ仕組みについて説明しなさい。（4点）
→ 内陸部の鉱山から沿岸部の港まで長距離の鉄道が敷設されており、大量の鉱産資源を効率的に運んで輸出している。

(5) オーストラリアの農畜産業が大規模に行われている理由を「土地」という語を使って説明しなさい。（4点）
→ オーストラリアは広大な土地を持ちながら人口が少ないため、一人あたりの土地面積が広く、大規模な放牧や栽培が可能となっている。

## 大問4: 他地域との結びつきと貿易（配点 20点）

オーストラリアと他地域との結びつきについて、次の問いに答えなさい。

(1) APECの正式名称を日本語で答えよ。また、設立された年を答えなさい。（4点）
→ アジア太平洋経済協力会議。1989年に設立された。

(2) オーストラリアの主な貿易相手国が変化した経緯を「イギリス」「アジア」の語を使って説明しなさい。（5点）
→ かつてはイギリスの植民地であったことから貿易相手もイギリスが中心であったが、アジア諸国の経済成長にともない、日本や中国などアジアの国々が主な貿易相手となった。

(3) オーストラリアがアジア太平洋地域との経済的な結びつきを強めている理由を二つ述べなさい。（6点）
→ 地理的にアジア太平洋地域に近く、輸送コストが低いこと。アジア諸国の急速な経済成長にともない、鉄鉱石や石炭などの鉱産資源の需要が大きく増加したこと。

(4) オーストラリアから日本に輸出される主な資源を二つ挙げ、日本側がそれを必要とする理由を述べなさい。（5点）
→ 鉄鉱石と石炭。日本は自国での鉱産資源の産出が少ないため、鉄鋼業や発電に必要な鉄鉱石と石炭をオーストラリアから大量に輸入している。

## 大問5: 移民・多文化社会と環境問題（配点 20点）

オーストラリアの社会変化と環境問題について、次の問いに答えなさい。

(1) 白豪主義とは何か説明し、廃止された時期を答えなさい。（4点）
→ 白豪主義とは、オーストラリアがヨーロッパ系以外の移民を制限していた差別的な政策のこと。1970年代以降に廃止された。

(2) シドニーのチャイナタウンの存在が多文化社会の象徴とされる理由を説明しなさい。（4点）
→ チャイナタウンには中国系移民が多く暮らし、中国の文化や食が根づいている。多様な文化が共存するオーストラリアの多文化社会を象徴する場所である。

(3) 地球温暖化がオセアニアの島国に与える影響を「海面上昇」「ツバル」の語を使って説明しなさい。（4点）
→ 地球温暖化による海面上昇の影響で、サンゴ礁の低い島々からなるツバルなどの国は国土が水没する危機に直面している。

(4) グレートバリアリーフで起きているサンゴ礁の白化現象とは何か。その原因とともに説明しなさい。（4点）
→ サンゴ礁の白化とは、海水温の上昇によってサンゴと共生する藻類がいなくなり、サンゴが白くなって死滅していく現象。地球温暖化が主な原因とされている。

(5) アボリジニの権利回復の取り組みとして行われていることを二つ挙げなさい。（4点）
→ かつて奪われた土地の権利を返還する取り組み。アボリジニの言語や伝統文化を保護し継承する活動。
