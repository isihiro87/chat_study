# アジア州 発展問題

## 問1: 季節風と地形がもたらす気候の多様性

次の文章を読んで、あとの問いに答えなさい。

アジア州は世界で最も広い州であり、ヒマラヤ山脈やチベット高原といった高山地域から、メコン川流域の熱帯低地、西アジアの乾燥地帯まで、極めて多様な気候を持つ。この多様性には（　A　）と呼ばれる季節によって風向きが変わる風と、大きな山脈による地形的な影響が関わっている。

(1) 空欄Aに入る語句を答えなさい。（2点）
→ 季節風（モンスーン）

(2) ヒマラヤ山脈が南アジアの気候に与える影響を「季節風」「水分」の語を使って説明しなさい。（4点）
→ 夏の季節風がインド洋から大量の水分を含んで北上するが、ヒマラヤ山脈にぶつかって上昇し、山脈の南側に大量の雨を降らせる。一方、山脈の北側（チベット高原）には水分が届かず乾燥している。

(3) 同じアジア州でも、東南アジアでは年間を通して高温多湿であるのに対し、西アジアは乾燥帯が広がっている。この違いが生じる理由を、「海からの距離」と「季節風」に着目して説明しなさい。（4点）
→ 東南アジアは海に囲まれているため季節風が大量の水分を運び高温多湿になる。一方、西アジアは海から遠い内陸に位置し季節風の影響が届きにくいため、降水量が少なく乾燥帯が広がっている。

(4) 「世界の屋根」と呼ばれる地域の存在がアジア全体の河川にどのような影響を与えているか、具体的な河川名を二つ挙げて説明しなさい。（5点）
→ ヒマラヤ山脈やチベット高原に降った雪や雨が水源となり、ガンジス川やメコン川などアジアの主要河川が生まれている。これらの河川は下流域に肥沃な平野を形成し、数億人の生活と農業を支えている。

## 問2: 中国の経済発展と格差の構造

次の資料を読んで、あとの問いに答えなさい。

【資料】
- 中国の沿岸部にあるシェンチェンは、1980年に経済特区に指定された当時は小さな漁村だったが、現在は人口1700万人を超える巨大都市に成長した。
- 中国政府は2000年から「西部大開発」政策を進め、内陸部のインフラ整備に取り組んでいる。
- 中国はかつて一人っ子政策を実施していたが、2015年に廃止した。

(1) 経済特区が沿岸部に設置された理由を「貿易」の語を使って説明しなさい。（4点）
→ 沿岸部は港湾に近く、外国との貿易に有利な立地であるため、輸出入の利便性を活かして外国企業の誘致と工業製品の輸出を効率よく行うことができるから。

(2) 「世界の工場」と呼ばれるまでに成長した中国が、近年この称号にふさわしくなくなりつつあると言われる理由を「賃金」の語を使って推測しなさい。（4点）
→ 経済成長に伴い中国の労働者の賃金が上昇し、安い労働力という最大の魅力が薄れているため、東南アジアなどより賃金の安い国に工場が移転する動きが進んでいる。

(3) 西部大開発が必要とされた背景にある問題を説明し、この政策が目指す目標を述べなさい。（4点）
→ 経済特区のある沿岸部が急速に発展した一方、内陸部（西部）は経済発展から取り残され、沿岸部と内陸部の間に大きな経済格差が生まれた。西部大開発はインフラ整備や産業誘致を通じて内陸部の経済発展を促し、格差を縮小することを目標としている。

(4) 一人っ子政策が廃止された背景と、廃止後に予想される社会的影響について説明しなさい。（4点）
→ 一人っ子政策により出生率が低下し、少子高齢化と労働力不足が深刻な問題となったため廃止された。廃止後は出生数の増加による労働力の確保が期待される一方、教育や医療などの社会保障費用の増大が課題となる可能性がある。

## 問3: 東南アジアの開発と環境問題

次の文章を読んで、あとの問いに答えなさい。

東南アジアでは、季節風を利用した稲作や大規模なプランテーション農業が行われている。近年は外国企業の工場進出も相次ぎ、経済成長が著しい。しかし、開発に伴うマングローブの減少や、都市部のスラム問題など、さまざまな課題も生じている。

(1) 東南アジアで二期作が可能な理由を、季節風との関係から説明しなさい。（4点）
→ 年間を通して温暖であり、季節風がもたらす雨季の豊富な降水量を利用することで、年に2回米を栽培・収穫することが可能になっている。

(2) パーム油の生産拡大とマングローブの減少にはどのような関係があるか。「生態系」の語を使って、環境への影響とあわせて説明しなさい。（5点）
→ パーム油の需要増加に伴い、油やしのプランテーションを拡大するために海岸部のマングローブが伐採されている。マングローブは魚介類の繁殖地であり、海岸の浸食を防ぐ役割もあるため、その減少は沿岸部の生態系に深刻な影響を与えている。

(3) ASEANが東南アジアの経済発展に果たしている役割と、ASEAN加盟国に共通する経済的な課題を一つ述べなさい。（4点）
→ ASEANは加盟国間の関税引き下げや人の移動の自由化を進め、域内の経済協力と貿易の活性化に貢献している。共通する課題として、急速な工業化に伴う都市部への人口集中とスラムの形成がある。

(4) 東南アジアに多くの華人が暮らしている歴史的背景と、彼らが経済に与えている影響を説明しなさい。（4点）
→ 古くから中国南部の人々が交易や移住のために東南アジアに渡り定住した歴史がある。華人は商業や金融の分野で大きな影響力を持ち、東南アジア各国の経済発展に重要な役割を果たしている。

## 問4: 南アジアのICT産業と社会変化

次の資料を読んで、あとの問いに答えなさい。

【資料】
- インドのバンガロールは「インドのシリコンバレー」とも呼ばれ、多くのIT企業が集まっている。
- インドの人口は世界最大級で、現在も急増を続けている。
- ガンジス川はヒンドゥー教徒にとって聖なる川であり、沐浴が行われている。
- インドは近年、太陽光発電などの再生可能エネルギーの導入を加速している。

(1) インドでICT産業が急成長できた要因を、「英語」「時差」の二つの語を使って説明しなさい。（4点）
→ インドでは英語が広く使われているため欧米企業との業務連携がしやすく、またアメリカとの時差を活用して24時間体制のソフトウェア開発やサポート業務が可能であることから、ICT産業が急成長した。

(2) インドが再生可能エネルギーの導入を進める理由を、「人口」「経済成長」の語を使って説明しなさい。（4点）
→ 急速な人口の増加と経済成長に伴いエネルギー需要が急増しているが、化石燃料に頼り続けると環境汚染やエネルギーの安定供給に問題が生じるため、太陽光発電など再生可能エネルギーの導入を進めている。

(3) ICT産業の成長がインド社会にもたらすプラスの面とマイナスの面をそれぞれ一つ挙げ、説明しなさい。（4点）
→ プラスの面：ICT産業が雇用を創出し、技術者の高い収入が消費や投資を生んで経済全体を押し上げている。マイナスの面：ICT産業に従事できる高学歴の都市住民と、農村部の貧困層との間で貧富の格差がさらに拡大している。

(4) ガンジス川は聖なる川であると同時に深刻な水質汚染が問題となっている。宗教的な価値と環境保全の両立が難しい理由を考えて説明しなさい。（4点）
→ ヒンドゥー教徒にとって沐浴は重要な宗教的行為であり制限が難しいうえ、川沿いに暮らす膨大な人口の生活排水や工場排水も流入している。宗教的な慣習を尊重しながら環境規制を強化することは社会的な反発を招きかねず、両立が困難になっている。

## 問5: 西アジアの資源と産業転換

次の文章を読んで、あとの問いに答えなさい。

西アジアのペルシャ湾岸諸国は、豊富な石油資源を背景に急速な経済発展を遂げた。OPEC（石油輸出国機構）を通じて世界のエネルギー市場に大きな影響力を持つ一方で、石油資源の枯渇に備えた産業の多角化も進められている。ドバイでは人工知能（AI）の活用や観光業の発展が注目されている。

(1) OPECが石油の生産量を調整できることが、世界経済にどのような影響を与えるか説明しなさい。（4点）
→ OPECが石油の生産量を減らすと供給が減って原油価格が上昇し、世界の輸送コストや工業製品の価格が上がる。逆に生産量を増やすと原油価格が下がり、経済活動が活発になる。このように世界経済はOPECの決定に大きく左右される。

(2) 石油化学コンビナートが産油国に建設されることの利点を、「原料」「輸送」の語を使って説明しなさい。（4点）
→ 産油国に石油化学コンビナートを建設すれば、原料である石油を遠方に輸送する必要がなく、輸送コストを抑えながら石油を高付加価値の化学製品に加工して輸出することができる。

(3) レアメタルの産出が西アジア・中央アジアにとって今後重要になると考えられる理由を、世界的な技術動向と関連づけて説明しなさい。（4点）
→ 電気自動車やスマートフォン、再生可能エネルギー設備などの需要が世界的に急増しており、これらの製品に不可欠なレアメタルの需要も高まっている。産出国である西アジア・中央アジアの国際的な重要性がさらに増すと考えられる。

(4) ドバイが石油依存から脱却するために観光業やAI技術を推進していることについて、この戦略の利点と課題をそれぞれ述べなさい。（4点）
→ 利点：石油が枯渇しても持続的に収入を得られる経済構造を構築できること。課題：観光業は国際情勢や景気変動の影響を受けやすく、AI技術には高度な人材の育成や確保が必要であり、短期間での成果が保証されないこと。
