# 世界のさまざまな国々 大問小問

## 大問1: 国旗と国名（配点 20点）

世界にはさまざまな国旗があり、それぞれの国の歴史や文化が反映されている。次の問いに答えなさい。

(1) オーストラリアの国旗に描かれている2つの特徴的なデザインを答えよ。（4点）
→ ユニオンジャック（イギリスの国旗）と南十字星。

(2) ユニオンジャックが国旗に含まれる理由を、「植民地」という語を使って説明しなさい。（4点）
→ オーストラリアはかつてイギリスの植民地だった歴史があり、ユニオンジャックはそのつながりを示している。

(3) イスラム教を信仰する国の国旗に多く見られるデザインを答えよ。（3点）
→ 三日月と星。

(4) 次の国名の由来をそれぞれ答えよ。（8点）
ア インド　イ フィリピン　ウ コロンビア　エ エクアドル
→ ア：大河を意味するインダス川に由来。イ：スペインの皇太子フェリペに由来。ウ：探検家コロンブスに由来。エ：スペイン語で「赤道」を意味する。

(5) 南十字星が描かれる国旗は、主にどの半球の国に多いか。理由もあわせて答えよ。（3点）（配点調整で1点追加）
→ 南半球の国。南十字星は南半球でよく見える星座であるため。

## 大問2: 国境のしくみ（配点 18点）

国と国の境界線である国境について、次の問いに答えなさい。

(1) 国境には大きく2つの種類がある。それぞれの名称を答えよ。（4点）
→ 自然的国境と人為的国境。

(2) 自然的国境に利用される地形を3つ答えよ。（3点）
→ 山脈・川・海。

(3) 人為的国境とはどのような国境か。「緯線」「経線」の語を使って説明しなさい。（4点）
→ 緯線や経線に沿って人が直線的に引いた国境。

(4) 人為的国境が特に多く残る大陸はどこか。また、その理由を答えなさい。（4点）
→ アフリカ大陸。植民地時代にヨーロッパの国々が、現地の民族や地形を考慮せず緯線・経線に沿って国境を引いたため。

(5) アメリカ合衆国とメキシコの国境には、自然的国境として利用されている川がある。その川は何と呼ばれるか。（3点）
→ リオグランデ川。

## 大問3: 島国と内陸国・面積の比較（配点 20点）

(1) 全体が海に囲まれている国を何というか。漢字2字で答えよ。（2点）
→ 島国。

(2) 海に面していない国を何というか。漢字3字で答えよ。（2点）
→ 内陸国。

(3) 島国と内陸国の例を2つずつ答えよ。（4点）
→ 島国：日本・イギリス（インドネシア、フィリピンなども可）。内陸国：モンゴル・スイス（ネパール、ボリビアなども可）。

(4) 面積が世界最大の国と最小の国の名前を答え、両国の面積の差がおよそ何万倍か答えよ。（4点）
→ 最大：ロシア（約1,710万km²）、最小：バチカン市国（約0.44km²）。差は約3,900万倍。

(5) 日本の面積は約38万km²である。ロシアの面積は日本のおよそ何倍か。（4点）
→ 約1,710万÷38万≒約45倍。

(6) バチカン市国はどこの国の中にあるか。また、何の総本山として知られるか。（4点）
→ イタリア（ローマ市内）にある。カトリック教会の総本山。

## 大問4: 世界の人口と人口密度（配点 22点）

(1) 2023年現在、世界の人口はおよそ何億人か。（2点）
→ 約80億人。

(2) 世界で人口が最も多い2つの国を答えよ。（4点）
→ インドと中国。

(3) 人口密度の求め方を式で答えよ。（3点）
→ 人口密度＝人口÷面積。

(4) A国は人口4,000万人、面積80万km²である。人口密度を計算しなさい。（3点）
→ 4,000万÷80万＝50人/km²。

(5) B国は人口200万人、面積400km²である。人口密度を計算しなさい。（3点）
→ 200万÷400＝5,000人/km²。

(6) (4)のA国と(5)のB国を比べて、人口密度の意味を「混み具合」という語を使って説明しなさい。（4点）
→ A国は50人/km²、B国は5,000人/km²であり、B国はA国の100倍の人口密度がある。人口密度は1km²あたりの人口で、その土地の混み具合を表す指標である。B国の方がはるかに混み合っている。

(7) モナコの人口密度が約2万人/km²と非常に高い理由を推測しなさい。（3点）
→ モナコは面積が約2km²と非常に小さいが、都市国家として多くの人が住み、観光地や高級住宅地としても発展しているため。

## 大問5: 総合問題（配点 20点）

次の文を読み、問いに答えなさい。

「世界には約（ア）の国がある。国と国の境界線を（イ）といい、山脈や川などの地形を利用した（ウ）と、緯線や経線に沿って引いた（エ）がある。海に囲まれた国を（オ）、海に面していない国を（カ）という。面積が世界最大の（キ）と最小の（ク）では、面積の差は約3,900万倍にもなる。」

(1) （ア）～（ク）に入る語句を答えよ。（8点）
→ ア：190　イ：国境　ウ：自然的国境　エ：人為的国境　オ：島国　カ：内陸国　キ：ロシア　ク：バチカン市国。

(2) 世界の人口のうち、アジア州が占める割合がほかの州より圧倒的に多い理由を「農業」「大河」の語を使って説明しなさい。（6点）
→ アジア州には長江やガンジス川などの大河の流域が広がり、古くから稲作を中心とした農業が発達して多くの人口を養うことができたため。

(3) 人口の多い国が必ずしも人口密度が高いとは限らない。この理由を「面積」の語を使って説明しなさい。（6点）
→ 人口密度は人口÷面積で求めるため、人口が多くても面積が大きければ人口密度は低くなる。たとえばロシアは面積が非常に大きいため人口約1.4億人でも人口密度は低い。
