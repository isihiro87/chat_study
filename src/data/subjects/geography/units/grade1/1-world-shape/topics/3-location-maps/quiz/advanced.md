# 地球上の位置と地図 発展問題

## 問1: 緯度・経度と対蹠点の計算

(1) 北緯35度・東経135度の地点Aの対蹠点Bの緯度と経度を求めなさい。
→ 南緯35度・西経45度。緯度は北緯→南緯で同じ数値。経度は180度−135度＝45度で東経→西経。

(2) 地点Bは地球上のどのあたりにあるか。大陸名または海洋名で答えなさい。
→ 南アメリカ大陸のブラジル東部の沖合い（大西洋上）。

(3) 南緯20度・西経60度の地点Cの対蹠点Dの緯度と経度を求めなさい。
→ 北緯20度・東経120度。

(4) 地点Dはどこの国の付近か推測しなさい。
→ フィリピン付近（東経120度・北緯20度はフィリピンのルソン島付近）。

(5) 対蹠点同士を結ぶ直線は地球のどこを通るか答えなさい。
→ 地球の中心を通る。対蹠点は地球の中心をはさんで正反対に位置する。

## 問2: 白夜と極夜のしくみを深く考える

(1) 白夜が起こる北極圏の範囲は「北緯66.6度以北」である。この66.6度という数値と地軸の傾き（約23.4度）の関係を説明しなさい。
→ 90度−23.4度＝66.6度。地軸が23.4度傾いているため、北緯66.6度以北では夏至のとき地球の傾きにより太陽の光が一日中当たり続ける。

(2) 北極点（北緯90度）では、白夜はおよそ何か月間続くか。
→ 約半年間（春分から秋分の約6か月間）。北極点では半年間太陽が沈まない。

(3) 赤道付近の地域では白夜や極夜が起こらない理由を説明しなさい。
→ 赤道付近は緯度が低いため、地軸の傾きの影響が小さく、1年を通じて昼と夜がほぼ均等に訪れる。白夜や極夜は高緯度地域特有の現象である。

(4) 白夜の時期、北極圏に住む人々の生活にどのような影響があるか推測しなさい。
→ 夜も明るいため、睡眠のリズムが乱れやすくなる。一方で、農作業や漁業など屋外活動を長時間行うことができる利点がある。

## 問3: 図法の選択と活用

ある学校の授業で、3種類の世界地図（メルカトル図法・モルワイデ図法・正距方位図法）を使って課題に取り組んだ。

(1) 「世界各国の面積を正確に比較しなさい」という課題に最も適した図法を選び、理由を答えなさい。
→ モルワイデ図法。面積が正しく表されるため、各国の面積を正確に比較することができる。

(2) 「東京からニューヨークまでの最短ルートの方角と距離を調べなさい」という課題に最も適した図法を選び、理由を答えなさい。
→ 正距方位図法（中心を東京にしたもの）。中心からの距離と方位が正しく表されるため、最短ルートの方角と距離を正確に読み取ることができる。

(3) 「船が一定の方角で航海するための方位を調べなさい」という課題に最も適した図法を選び、理由を答えなさい。
→ メルカトル図法。角度が正しく表されるため、地図上の直線が等角航路（一定方位の航路）を示し、コンパスを使った航海の方位を正確に読み取ることができる。

(4) メルカトル図法で東京からニューヨークへの直線を引くと、実際の最短ルートとは異なる。この理由を「球体」「大圏」の語を使って説明しなさい。
→ 地球は球体であるため、2地点間の最短距離は大圏（地球の中心を通る平面で地球を切った円）に沿ったルートになる。メルカトル図法の直線は等角航路を示しているが、球体上での最短距離（大圏航路）とは一致しない。

## 問4: 16方位と地図の読み取り

(1) 8方位では表現できないが16方位で表現できる方位を4つ答えなさい。
→ 北北東・東北東・東南東・南南東（または北北西・西北西・西南西・南南西なども可）。

(2) ある正距方位図法の地図（中心：東京）でニューヨークの方向を16方位で読み取ったところ「北北東」であった。地球は球体であるため、メルカトル図法でニューヨークを見るとほぼ真東にある。この違いが生じる理由を説明しなさい。
→ 正距方位図法は中心からの方位が正確であるため、東京からの実際の方角を正しく表す。地球が球体であるため、東京からニューヨークへの最短経路は北極圏の近くを通る大圏航路になり、出発時の方角は北北東になる。メルカトル図法では緯線と経線が直角に交わるため、同じ緯度帯にある都市は横（東西）方向に見えるが、これは実際の最短方向とは異なる。

(3) 4方位・8方位・16方位は、方位の数をそれぞれ何度刻みで区切っていることになるか。
→ 4方位：90度刻み、8方位：45度刻み、16方位：22.5度刻み。

(4) 32方位という考え方もある。32方位は何度刻みになるか答え、どのような場面で使われるか推測しなさい。
→ 360÷32＝11.25度刻み。船の航海や気象観測など、より精密に方向を指定する必要がある場面で使われる。

## 問5: 総合問題 ─ 緯度経度・図法・方位を使いこなす

地理の授業で、東京を中心とした正距方位図法の地図と、メルカトル図法の世界地図の2枚を使って学習した。

(1) 東京から見て、ロンドン（北緯51度・西経0度）の方向をそれぞれの地図で調べた場合、正距方位図法ではどの方角に見えるか。また、メルカトル図法との違いを説明しなさい。
→ 正距方位図法では北北西の方角に見える。これが実際の最短方向（大圏方向）である。メルカトル図法ではロンドンは東京のほぼ真西に見えるが、これは地図上の見かけの方向であり、実際の最短経路の方向ではない。

(2) 東京から最も近い外国の首都を正距方位図法の地図で調べたところ、ソウル（韓国）であった。東京からソウルまでの距離はおよそ1,200kmである。この距離を地球儀と正距方位図法のどちらで調べるのがより正確か。理由とあわせて答えよ。
→ 地球儀の方がより正確。地球儀は面積・距離・方位・形のすべてを正しく表すため。正距方位図法も中心からの距離は正確だが、中心以外の2地点間の距離は正確とは限らない。ただし中心が東京の場合、東京からソウルまでの距離は正距方位図法でも正確に読める。

(3) 東京の対蹠点を求め、その地点が正距方位図法の地図上のどこに位置するか答えよ。
→ 東京（北緯約36度・東経約140度）の対蹠点は南緯約36度・西経約40度で、南アメリカのウルグアイ沖の大西洋上。正距方位図法の地図上では、中心（東京）から最も遠い地図の外縁部（端）に位置する。

(4) 地球儀・メルカトル図法・モルワイデ図法・正距方位図法の4つを「距離」「面積」「方位」「角度」の正確さで整理し、それぞれの用途を1つずつ答えなさい。
→ 地球儀：距離・面積・方位・角度すべて正確。用途：正確な位置関係の把握。メルカトル図法：角度が正確。用途：航海の方位の確認。モルワイデ図法：面積が正確。用途：各国の面積の比較や分布図。正距方位図法：中心からの距離と方位が正確。用途：航空ルートの計画。
