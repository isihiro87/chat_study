# 地球上の位置と地図 大問小問

## 大問1: 緯度と経度（配点 20点）

地球上の位置を表す方法について、次の問いに答えなさい。

(1) 次の文の空欄を埋めよ。（6点）
「緯度は（ア）を基準（0度）として、北へ向かうのが（イ）、南へ向かうのが（ウ）である。経度は（エ）を基準（0度）として、東へ向かうのが（オ）、西へ向かうのが（カ）である。」
→ ア：赤道　イ：北緯　ウ：南緯　エ：本初子午線　オ：東経　カ：西経。

(2) 緯度は最大で何度まであるか。北と南それぞれ答えよ。（2点）
→ 北緯・南緯ともにそれぞれ90度まで。

(3) 経度は最大で何度まであるか。東と西それぞれ答えよ。（2点）
→ 東経・西経ともにそれぞれ180度まで。

(4) 本初子午線が通る都市名と、その国名を答えよ。（4点）
→ ロンドン、イギリス（旧グリニッジ天文台）。

(5) 東京の位置を、緯度と経度のおおよその値で答えよ。（3点）
→ 北緯約36度・東経約140度。

(6) 緯度と経度を組み合わせることの利点を説明しなさい。（3点）
→ 地球上のどの地点でも2つの数値で正確に位置を表すことができる。

## 大問2: 対蹠点と高緯度の現象（配点 20点）

(1) 地球上のある地点の正反対にある地点を何というか。（3点）
→ 対蹠点（たいせきてん）。

(2) 東京の対蹠点は、地球上のどのあたりにあるか。（3点）
→ 南アメリカのウルグアイ沖の大西洋上。

(3) 北緯40度・東経130度の対蹠点の緯度と経度を求めなさい。（4点）
→ 南緯40度・西経50度。（緯度は北緯→南緯、経度は180度−130度＝50度で東経→西経）

(4) 高緯度地域で夏に太陽が一日中沈まない現象を何というか。（2点）
→ 白夜。

(5) 高緯度地域で冬に太陽が一日中昇らない現象を何というか。（2点）
→ 極夜。

(6) 白夜が起こる範囲を北半球について答えよ。（3点）
→ 北緯66.6度以北（北極圏）。

(7) 白夜と極夜が起こる原因を「地軸」「傾き」の語を使って説明しなさい。（3点）
→ 地球の地軸が約23.4度傾いた状態で太陽のまわりを公転しているため、高緯度地域では夏に太陽が沈まず、冬に太陽が昇らない現象が起こる。

## 大問3: 地球儀と世界地図（配点 22点）

(1) 地球をそのまま縮小した模型を何というか。（2点）
→ 地球儀。

(2) 地球儀が正しく表せる4つの要素を答えよ。（4点）
→ 面積・距離・方位・形。

(3) 地球儀の短所を2つ答えよ。（4点）
→ 持ち運びが不便であること、地球全体を一度に見ることが難しいこと。

(4) 世界地図で面積・距離・方位・形のすべてを同時に正しく表せない理由を答えよ。（4点）
→ 球体である地球を平面に表すため、必ず何かにゆがみが生じるから。

(5) 次の図法の特徴をそれぞれ答えよ。（6点）
ア メルカトル図法　イ モルワイデ図法　ウ 正距方位図法
→ ア：角度（航海の方位）が正しい。イ：面積が正しい。ウ：中心からの距離と方位が正しい。

(6) メルカトル図法の短所を具体的に答えよ。（2点）
→ 高緯度地域ほど面積が実際より大きく表される（例：グリーンランドが実際より巨大に見える）。

## 大問4: 方位の示し方（配点 18点）

(1) 基本的な4方位を答えよ。（2点）
→ 東・西・南・北。

(2) 8方位について、4方位の間にある方位を4つ答えよ。（4点）
→ 北東・北西・南東・南西。

(3) 16方位について、北と北東の間の方位を答えよ。（2点）
→ 北北東。

(4) 16方位について、南と南西の間の方位を答えよ。（2点）
→ 南南西。

(5) 4方位→8方位→16方位と方位を細かくしていく利点を説明しなさい。（4点）
→ 方位を細かくすることで、方向をより正確に表すことができ、航海や航空で目的地への進路を正確に決めることができる。

(6) 正距方位図法の地図で、中心から見た別の都市の方向を調べるとき、16方位を使うとどのような利点があるか答えよ。（4点）
→ 正距方位図法では中心からの方位が正しいため、16方位を使えば目的地の方角をより精密に読み取ることができ、最短経路の方向を正確に判断できる。

## 大問5: 総合問題（配点 20点）

次の問いに答えなさい。

(1) ある都市Xは北緯50度・東経10度に位置する。この都市Xの対蹠点の緯度と経度を求めよ。（4点）
→ 南緯50度・西経170度。

(2) 正距方位図法の中心を東京にした地図で、ロンドンの方向を調べたところ北北西の方角であった。実際の距離を地球儀で調べたところ約9,600kmであった。メルカトル図法でロンドンの方角を調べると正確な方角とは異なる。その理由を答えよ。（4点）
→ メルカトル図法は角度は正しいが、中心からの方位を正しく表す図法ではない。正距方位図法は中心からの方位と距離が正しいため、方角を調べるには正距方位図法が適している。

(3) モルワイデ図法で作られた世界地図を使って、各大陸の面積を比較するのが適している理由を答えよ。（4点）
→ モルワイデ図法は面積が正しく表される図法であるため、大陸や国の面積の大小を正確に比較することができる。

(4) メルカトル図法の地図でグリーンランドとアフリカ大陸の面積を比べると、グリーンランドが大きく見えるが、実際にはアフリカ大陸のほうがはるかに大きい。この理由を説明しなさい。（4点）
→ メルカトル図法は高緯度地域ほど面積が実際より大きく拡大されるため、高緯度にあるグリーンランドは実際以上に大きく見える。実際のアフリカ大陸の面積はグリーンランドの約14倍である。

(5) 航空機のパイロットが最短ルートを計画するとき、最も適した図法は何か。理由とあわせて答えよ。（4点）
→ 正距方位図法。中心からの距離と方位が正確に表されるため、最短ルート（大圏航路）の方向と距離を正確に読み取ることができる。
