# 地球の姿 発展問題

## 問1: 陸地と海洋の分布を読み解く

地球の表面は陸地が約3割、海洋が約7割を占めている。ある資料によると、北半球では陸地の割合が約39%、南半球では約19%である。

(1) 北半球と南半球の陸地の割合の差はおよそ何%か。また、この差が生じる理由を大陸の配置にふれて説明しなさい。
→ 差は約20%。北半球にはユーラシア大陸・北アメリカ大陸の大部分があり、アフリカ大陸の北部も含まれるため、陸地の割合が高い。南半球にはオーストラリア大陸・南極大陸はあるが、全体として海洋の面積が大きい。

(2) 「水半球」という考え方がある。地球を陸地が最も少なく見える方向から見たとき、中心はニュージーランド付近とされる。このとき、見える面の大部分がどのようになっているか説明しなさい。
→ 見える面のほとんどが海洋（太平洋）で占められ、わずかにオーストラリア大陸やニュージーランドなどの陸地が見える程度である。

(3) 「陸半球」の中心はフランス付近とされる。この見え方と「水半球」の見え方を比較して、地球上の陸地の分布の特徴を述べなさい。
→ 陸半球ではユーラシア大陸・アフリカ大陸を中心に多くの陸地が集まって見え、水半球ではほとんどが海洋で陸地がわずかしか見えない。このことから、地球上の陸地は北半球の一方に偏って分布していることがわかる。

## 問2: 大陸と大洋の位置関係

世界地図を参考にして、以下の問いに答えなさい。

(1) ユーラシア大陸に面している大洋をすべて答えなさい。
→ 太平洋・大西洋・インド洋（3つすべてに面している）。

(2) アフリカ大陸に面している大洋を答えなさい。
→ 大西洋とインド洋。

(3) 太平洋に面している大陸をすべて答えなさい。
→ ユーラシア大陸・オーストラリア大陸・南極大陸・南アメリカ大陸・北アメリカ大陸。

(4) 三つの大洋すべてに面している大陸が1つある。その大陸名を答え、なぜ3つすべてに面しているか、位置をふまえて説明しなさい。
→ ユーラシア大陸（および南極大陸）。ユーラシア大陸は東に太平洋、西に大西洋、南にインド洋と、地球上で広大な面積を占めるため3つすべてに接している。

## 問3: 六つの州と大陸の対応

(1) 六大陸と六つの州の対応について、下の表の空欄を埋めなさい。

| 大陸 | 対応する州 |
|------|-----------|
| ユーラシア大陸 | （ア）と（イ） |
| アフリカ大陸 | （ウ） |
| 北アメリカ大陸 | （エ） |
| 南アメリカ大陸 | （オ） |
| オーストラリア大陸＋太平洋の島々 | （カ） |
| 南極大陸 | （キ） |

→ ア：アジア州　イ：ヨーロッパ州　ウ：アフリカ州　エ：北アメリカ州　オ：南アメリカ州　カ：オセアニア州　キ：いずれの州にも属さない。

(2) ユーラシア大陸が「アジア州」と「ヨーロッパ州」に分けられる境界として使われている自然の地形を2つ答えなさい。
→ ウラル山脈とウラル川（カスピ海、コーカサス山脈なども使われる）。

(3) アジア州の5つの地域区分（東アジア・東南アジア・南アジア・西アジア・中央アジア）のうち、石油の産出で知られる地域はどこか。理由とあわせて答えなさい。
→ 西アジア。ペルシア湾周辺のサウジアラビアやイラン、イラクなどの国が世界有数の産油国であるため。

## 問4: 大陸の特徴を比較する

(1) ユーラシア大陸とアフリカ大陸の間にある海を答え、両大陸を結ぶ運河の名前も答えなさい。
→ 地中海（紅海もある）。スエズ運河。

(2) 北アメリカ大陸と南アメリカ大陸を結ぶ地峡にある有名な運河の名前を答えよ。
→ パナマ運河。

(3) オーストラリア大陸は六大陸のうち最も面積が小さいが、オセアニア州として独立した州に区分されている。その理由を「島々」「太平洋」の語を使って説明しなさい。
→ オーストラリア大陸は面積は小さいが、太平洋に広がる数多くの島々とあわせて独自の文化圏を形成しているため、オセアニア州として独立した州に区分されている。

## 問5: 統計資料を使った分析

次の資料は各州の面積と人口の割合（概数）である。

| 州 | 面積の割合 | 人口の割合 |
|----|----------|----------|
| アジア州 | 約30% | 約60% |
| アフリカ州 | 約20% | 約18% |
| ヨーロッパ州 | 約7% | 約9% |
| 北アメリカ州 | 約16% | 約8% |
| 南アメリカ州 | 約12% | 約5% |
| オセアニア州 | 約6% | 約1% |

(1) 面積の割合に対して人口の割合が特に高い州はどこか。その州の名前と、人口が集中する理由を答えなさい。
→ アジア州。面積の割合が約30%に対し人口は約60%を占める。温暖な気候の地域が多く、大河流域で古くから稲作などの農業が発達し、多くの人口を養えたため。

(2) 面積の割合に対して人口の割合が特に低い州はどこか。理由とあわせて答えなさい。
→ オセアニア州。面積の約6%に対し人口は約1%しかない。オーストラリア大陸の内陸部は乾燥した砂漠が広がり、太平洋の島々も面積が小さいため人口が少ない。

(3) この表に南極大陸が含まれていない理由を答えなさい。
→ 南極大陸は1年じゅう氷におおわれ、定住する住民がおらず、いずれの州にも区分されていないため。

(4) アジア州の人口密度が高い地域と低い地域の例をそれぞれ挙げ、その違いが生じる原因を自然環境に関連づけて説明しなさい。
→ 人口密度が高い地域：東アジアや南アジアの大河流域（長江流域、ガンジス川流域など）。温暖で水が豊富なため農業が盛んで多くの人が暮らしている。人口密度が低い地域：中央アジアの乾燥地帯やシベリア。厳しい気候のため人が住みにくい。
