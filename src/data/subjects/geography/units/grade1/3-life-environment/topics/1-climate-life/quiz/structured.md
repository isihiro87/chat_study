# 世界の気候と人々の暮らし 大問小問

## 大問1: 寒帯と冷帯の生活（配点 20点）

寒帯と冷帯（亜寒帯）の暮らしについて、次の問いに答えなさい。

(1) 寒帯の気候の特徴を「雪」「氷」の語を使って簡潔に説明しなさい。（3点）
→ 一年の大半が雪と氷におおわれ、太陽が高くのぼらないため気温がほとんど上がらない。

(2) カナダ北部の寒帯に住むイヌイットの人々がかつて冬に住んでいた住居の名称を答えよ。また、その住居はどのように作られたか説明しなさい。（4点）
→ イグルー。雪のブロックを積み上げてドーム型に作った住居。

(3) 寒帯の短い夏に見られる自然の変化を二つ答えなさい。（4点）
→ こけが生えること、カリブー（トナカイ）などの大型動物の移動が見られること。

(4) 亜寒帯のシベリアに広がる針葉樹の大森林を何というか。（3点）
→ タイガ。

(5) 永久凍土とは何か説明しなさい。また、その上に建物を建てるときにどのような工夫がされるか答えなさい。（6点）
→ 一年中凍ったままの土のこと。建物の熱で永久凍土が溶けて地面がしずむのを防ぐため、高床式の住居が建てられる。

## 大問2: 温帯の生活（配点 18点）

温帯の暮らしについて、次の問いに答えなさい。

(1) 温帯の気候の一般的な特徴を述べなさい。（3点）
→ 年間を通して温暖で適度な降水量がある。

(2) 日本と地中海沿岸は同じ温帯だが、夏の降水量が大きく異なる。それぞれの特徴を比較して説明しなさい。（4点）
→ 日本は梅雨や台風の影響で夏に雨が多いが、地中海沿岸は夏に乾燥して冬に雨が多い。

(3) 地中海沿岸の気候名を答えよ。（3点）
→ 地中海性気候。

(4) イタリアの伝統的な住居にはどのような工夫があるか。「壁」「窓」の語を使って説明しなさい。（4点）
→ 石の壁と小さな窓で日光をさえぎり、室内を涼しく保つ工夫がされている。

(5) 地中海沿岸で広く栽培されている農作物を三つ答え、代表的な加工品を一つ答えなさい。（4点）
→ ぶどう、オリーブ、トマト。代表的な加工品はワインやオリーブオイル。

## 大問3: 乾燥帯の生活（配点 22点）

乾燥帯の暮らしについて、次の問いに答えなさい。

(1) 砂漠の中で農業が可能な場所を何というか。また、その場所の特徴を説明しなさい。（4点）
→ オアシス。自然のわき水や井戸があり、水を利用して農業ができる場所。

(2) サヘルとはどのような地域か。「サハラ砂漠」の語を使って説明しなさい。（4点）
→ サハラ砂漠の南に広がる地域で、少量の降水量があり樹木や草が少し育つ。

(3) 遊牧とは何か説明しなさい。また、遊牧で飼育される代表的な家畜を二つ答えよ。（5点）
→ 草や水を求めて移動しながら家畜を飼育する牧畜のこと。代表的な家畜は羊やらくだ。

(4) 焼畑農業とはどのような農業か。その方法を説明しなさい。（4点）
→ わずかに育つ樹木を切りはらい、その灰を肥料として穀物を育てる農業。

(5) サヘル地域で砂漠化が進んでいる原因を「人口増加」の語を使って説明しなさい。（5点）
→ 人口増加にともなって耕作や放牧が過度に続けられ、土地がやせて植物が育たなくなり砂漠化が進んでいる。

## 大問4: 熱帯の生活（配点 18点）

熱帯の暮らしについて、次の問いに答えなさい。

(1) 熱帯はおもに地球上のどのあたりに分布しているか。（3点）
→ 赤道付近。

(2) 熱帯に広がる密林を何というか。そこにはどのような特徴があるか述べなさい。（4点）
→ 熱帯雨林。背の高い樹木が生い茂り、多種多様な動植物が生息している。

(3) サモアの伝統的な住居の特徴を説明し、そのような造りになっている理由を答えなさい。（5点）
→ 木の支柱とやしの葉の屋根で壁がなく開放的な造り。高温多湿の気候に対応するため風通しをよくしている。

(4) サモアで栽培されている主な作物を三つ答えなさい。（3点）
→ ココヤシ、バナナ、タロイモ。

(5) 熱帯の海岸沿いに見られるサンゴ礁とはどのようなものか簡潔に説明しなさい。（3点）
→ サンゴが長い年月をかけて積み重なってできた、浅い海に広がる地形。

## 大問5: 高山気候の生活（配点 22点）

高山地域の暮らしについて、次の問いに答えなさい。

(1) 標高と気温の関係を、具体的な数値を使って説明しなさい。（3点）
→ 標高が100m上がるごとに気温は約0.6℃下がる。

(2) アンデス山脈では赤道付近でも涼しい理由を説明しなさい。（4点）
→ 標高が非常に高いため、緯度に関係なく気温が低くなるから。

(3) アンデス山脈で放牧されている動物を二つ答え、その毛から何が作られるか述べなさい。（5点）
→ リャマとアルパカ。その毛からポンチョ（毛織物の衣服）が織られる。

(4) アンデス山脈の高地で栽培されている代表的な作物を二つ答えなさい。（4点）
→ じゃがいもととうもろこし。

(5) 高山地域では標高によって栽培できる作物が異なる理由を説明しなさい。（6点）
→ 標高が上がるにつれて気温が下がるため、それぞれの高さに適した作物しか育てることができないから。低い場所ではとうもろこしなどが育ち、高い場所ではじゃがいもなどの寒さに強い作物が栽培される。
