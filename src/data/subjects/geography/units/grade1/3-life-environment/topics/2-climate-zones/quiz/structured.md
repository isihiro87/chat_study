# 世界の気候区の分類 大問小問

## 大問1: 五つの気候帯（配点 18点）

世界の気候帯について、次の問いに答えなさい。

(1) 世界の気候帯を五つすべて答えよ。（5点）
→ 熱帯・乾燥帯・温帯・冷帯（亜寒帯）・寒帯。

(2) 五つの気候帯のほかに、標高の影響で気温が下がる気候を何というか。（3点）
→ 高山気候。

(3) 気候帯を分ける主な二つの要素は何か。（4点）
→ 気温と降水量。

(4) 赤道から極地に向かって、気温が低くなる理由を「太陽」の語を使って説明しなさい。（3点）
→ 赤道から離れるほど太陽の熱が地表に届きにくくなるため。

(5) 世界の気候区分を考案した学者の名前を答えよ。（3点）
→ ケッペン。

## 大問2: 熱帯と乾燥帯の気候区（配点 20点）

熱帯と乾燥帯の気候区について、次の問いに答えなさい。

(1) 熱帯に含まれる二つの気候区を答え、それぞれの降水量の特徴を説明しなさい。（6点）
→ 熱帯雨林気候は一年中雨が多い。サバナ気候は雨季と乾季がはっきり分かれている。

(2) 熱帯雨林気候の地域にはどのような植生が見られるか。（3点）
→ 背の高い樹木が密集した密林（ジャングル）が広がる。

(3) サバナ気候の地域に見られる植生の特徴を述べなさい。（3点）
→ 広い草原（サバナ）が広がり、まばらに樹木が点在する。

(4) 乾燥帯に含まれる二つの気候区を答え、その違いを説明しなさい。（5点）
→ 砂漠気候はほとんど雨が降らず植物がほとんど育たない。ステップ気候はわずかに雨が降り短い草が生える。

(5) ステップ気候の地域で見られる生活様式を一つ答えよ。（3点）
→ 遊牧（草を求めて移動しながら家畜を飼育する）。

## 大問3: 温帯の気候区（配点 22点）

温帯の気候区について、次の問いに答えなさい。

(1) 温帯に含まれる三つの気候区をすべて答えなさい。（6点）
→ 温暖湿潤気候・西岸海洋性気候・地中海性気候。

(2) 日本の大部分が属する気候区を答え、その気候の特徴を述べなさい。（4点）
→ 温暖湿潤気候。四季の変化がはっきりしており、夏に降水量が多い。

(3) 西岸海洋性気候はどのような特徴があるか。「気温差」の語を使って説明しなさい。（4点）
→ 一年を通じて気温差が小さく温暖で、偏西風と暖流の影響を受ける。

(4) 地中海性気候の降水量の特徴を「夏」「冬」の語を使って説明しなさい。（4点）
→ 夏に乾燥して雨が少なく、冬に雨が多い。

(5) 日本の温暖湿潤気候とイタリアの地中海性気候で、夏の気候の違いを比較して説明しなさい。（4点）
→ 日本は梅雨や台風の影響で夏に雨が多いが、イタリア（地中海性気候）は夏に乾燥して雨が少ない。

## 大問4: 冷帯・寒帯・高山気候（配点 22点）

冷帯・寒帯・高山気候について、次の問いに答えなさい。

(1) 冷帯（亜寒帯）の気候の特徴を述べなさい。（3点）
→ 夏と冬の気温差が非常に大きく、冬は厳しい寒さになる。

(2) 冷帯に広がる針葉樹の大森林を何というか。また、分布する代表的な地域を一つ答えよ。（4点）
→ タイガ。ロシア（シベリア）やカナダなど。

(3) 寒帯に含まれる二つの気候区を答え、それぞれの特徴を説明しなさい。（6点）
→ ツンドラ気候は短い夏にわずかにこけ類が生え、永久凍土が広がる。氷雪気候は一年中氷と雪に覆われ、植物はほとんど育たない。

(4) 高山気候について、標高100mあたりの気温の変化を答えなさい。（3点）
→ 標高が100m上がるごとに気温は約0.6℃下がる。

(5) 赤道付近のアンデス山脈やキリマンジャロ山の山頂付近に雪が見られる理由を説明しなさい。（6点）
→ 赤道付近で低緯度地域であっても、標高が非常に高いため気温が大幅に下がり、山頂付近では雪が積もるほど低温になるから。

## 大問5: 気候帯と気候区の総合（配点 18点）

次の特徴が当てはまる気候帯または気候区を答えなさい。

(1) 一年中高温で雨が多く、密林（ジャングル）が広がる。（3点）
→ 熱帯雨林気候。

(2) 夏に乾燥し、冬に雨が多く降る温帯の気候区である。（3点）
→ 地中海性気候。

(3) 短い夏にこけ類がわずかに生え、地下には永久凍土がある。（3点）
→ ツンドラ気候。

(4) 偏西風の影響で一年を通じて気温差が小さい温帯の気候区である。（3点）
→ 西岸海洋性気候。

(5) ほとんど雨が降らず、砂や岩の大地が広がる。（3点）
→ 砂漠気候。

(6) わずかに雨が降り、短い草が広がる乾燥帯の気候区で遊牧が行われる。（3点）
→ ステップ気候。
