# 世界の気候区の分類 発展問題

## 問1: 雨温図の読み取り

次のA〜Cの気候データを読んで、あとの問いに答えなさい。

【A地点】年平均気温27℃、年間降水量2500mm、月ごとの気温の変化はほとんどなく、毎月200mm前後の雨が降る。
【B地点】年平均気温16℃、年間降水量800mm、7〜9月の降水量は月20mm以下だが、12〜2月は月100mm以上。
【C地点】年平均気温−5℃、年間降水量400mm、夏の最高月平均気温は18℃、冬の最低月平均気温は−35℃。

(1) A〜Cの地点がそれぞれ属する気候区を答えなさい。
→ A：熱帯雨林気候、B：地中海性気候、C：冷帯（亜寒帯）。

(2) A地点で一年中雨が多い理由を「赤道」「上昇気流」の語を使って説明しなさい。
→ A地点は赤道付近に位置し、強い日射によって地面が暖められ上昇気流が発生しやすいため、一年を通じて雲が発達し多くの雨が降る。

(3) B地点では夏と冬で降水量が大きく異なる。この降水パターンが農業に与える影響を、「オリーブ」「ぶどう」の語を使って説明しなさい。
→ 夏に乾燥するため、乾燥に強いオリーブやぶどうなどの作物が栽培に適しており、地中海性気候の特徴を活かした農業が行われている。

(4) C地点では夏と冬の気温差が53℃にもなる。このような大きな気温差が生じる理由を説明しなさい。
→ 内陸部では海洋の温度調節効果が届きにくいため、夏は日照時間が長く気温が上がり、冬は極端に日照が短くなり放射冷却で気温が大幅に下がるから。

## 問2: 気候帯と農業の関係

次の表は、各気候帯で見られる代表的な農業や生活様式をまとめたものである。あとの問いに答えなさい。

| 気候帯 | 農業・生活様式 |
|:--|:--|
| 熱帯 | 稲作、プランテーション |
| 乾燥帯 | オアシス農業、遊牧 |
| 温帯 | 畑作、果樹栽培、稲作 |
| 冷帯 | 林業、わずかな畑作 |
| 寒帯 | 狩猟、漁業 |

(1) 熱帯でプランテーション（大規模農園）が発達した歴史的背景を簡潔に説明しなさい。
→ ヨーロッパの植民地時代に、温帯では育たないゴムやカカオ、さとうきびなどの熱帯作物を大規模に栽培するための農園が開かれたため。

(2) 乾燥帯のオアシス農業で必要不可欠なものは何か。また、オアシス周辺で栽培される代表的な作物を一つ答えなさい。
→ 水が必要不可欠。代表的な作物はなつめやし（デーツ）。

(3) 温帯では畑作と稲作の両方が行われるが、温帯のどのような条件が両方の農業を可能にしているか説明しなさい。
→ 温帯は適度な気温と降水量があり四季の変化があるため、小麦などの畑作物から米まで、さまざまな作物の栽培に対応できる条件がそろっている。

(4) 寒帯で農業がほとんど行われない理由を「気温」「土壌」の語を使って説明しなさい。
→ 寒帯は気温が極端に低いため植物の生育期間が短く、地下に永久凍土があるため土壌が凍って耕作に適さないから。

## 問3: 気候区と住居の関係

ある生徒が世界各地の住居について調べ、次のようにまとめた。あとの問いに答えなさい。

A：壁がなく柱と屋根だけの開放的な住居
B：厚い石壁と小さな窓の住居
C：地面から高く持ち上げた高床式の住居（冷帯地域）
D：日干しれんがで作られた平らな屋根の住居

(1) A〜Dの住居がそれぞれ見られる気候帯を答えなさい。
→ A：熱帯、B：温帯（地中海性気候）、C：冷帯（亜寒帯）、D：乾燥帯。

(2) Cの住居が地面から高く持ち上げられている理由を「永久凍土」の語を使って説明しなさい。
→ 建物の熱が地面に伝わると永久凍土が溶けて地面がしずみこみ建物がゆがむため、床を高くして建物の熱が地面に伝わりにくくする工夫がされている。

(3) Dの住居の屋根が平らなのはなぜか。乾燥帯の気候と関連づけて説明しなさい。
→ 乾燥帯ではほとんど雨が降らないため、雨水を流す傾斜のある屋根が不要である。平らな屋根は建築が容易で、夜に涼しい風にあたるために屋上で過ごすこともできる。

(4) これらの住居の多様性から、人間と自然環境の関わりについてどのようなことが言えるか、100字以内で述べなさい。
→ 人間はそれぞれの地域の気温・降水量・地形などの自然環境に適応するために、手に入る材料を活かして住居を工夫してきた。住居の多様性は人間の知恵と自然環境の密接な関係を示している。

## 問4: ケッペンの気候区分の理解

次の文章を読んで、あとの問いに答えなさい。

ケッペンは気温と降水量のデータをもとに世界の気候を体系的に分類した。この気候区分では、植生（植物の種類や分布）が気候の指標として重視されている。

(1) ケッペンの気候区分で、温帯を三つの気候区に分ける基準は主に何か。「降水量」「季節」の語を使って説明しなさい。
→ 降水量がどの季節に多いかによって区分される。夏に降水量が多いのが温暖湿潤気候、夏に乾燥して冬に降水量が多いのが地中海性気候、一年を通じて降水量が平均的なのが西岸海洋性気候。

(2) ケッペンが植生を重視した理由を考えて説明しなさい。
→ 植生はその地域の気温と降水量の長期的な特徴を総合的に反映しているため、植物の種類や分布を見ることで気候の特徴を客観的に判断できるから。

(3) 高山気候はケッペンの気候区分では独立した気候帯として扱われないことが多い。その理由を「標高」「緯度」の語を使って説明しなさい。
→ ケッペンの分類は主に緯度による気温と降水量の違いをもとにしているが、高山気候は緯度ではなく標高の影響で気温が下がるため、既存の五つの気候帯の枠組みには当てはまりにくいから。

## 問5: 日本の気候と世界の気候の比較

次の問いに答えなさい。

(1) 日本とイギリスはともに温帯に属するが、気候区が異なる。それぞれの気候区名を答え、降水量の季節変化の違いを説明しなさい。
→ 日本は温暖湿潤気候で夏に降水量が多い。イギリスは西岸海洋性気候で一年を通じて降水量が比較的均等に分布する。

(2) 日本の北海道は温帯ではなく冷帯（亜寒帯）に分類されることがある。その理由を説明しなさい。
→ 北海道は緯度が高く冬の寒さが厳しいため、最寒月の平均気温が温帯の基準を下回り、冷帯に区分される場合がある。

(3) 沖縄と北海道の気候の違いを、気候帯の観点から説明しなさい。同じ日本でもこのように気候が異なる理由を「緯度」の語を使って述べなさい。
→ 沖縄は亜熱帯（熱帯に近い温帯）に位置し年間を通じて温暖であるが、北海道は冷帯に近く冬は厳しい寒さになる。日本は南北に長く緯度の差が大きいため、同じ国内でも気候帯が異なる。
