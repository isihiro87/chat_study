# 多様な生活・文化と宗教 大問小問

## 大問1: 世界の宗教（配点 20点）

世界の主な宗教について、次の問いに答えなさい。

(1) 世界三大宗教をすべて答え、それぞれの信者数（概数）を書きなさい。（6点）
→ キリスト教（約24億人）、イスラム教（約19億人）、仏教（約5億人）。

(2) ヒンドゥー教の信者数は約12億人と多いにもかかわらず、世界三大宗教に含まれない理由を説明しなさい。（4点）
→ ヒンドゥー教は主にインドで信仰されており、発祥地から世界各地には広がっていないため。

(3) キリスト教が多く信仰されている地域を三つ答えなさい。（3点）
→ ヨーロッパ、南北アメリカ、オセアニア。

(4) イスラム教が多く信仰されている地域を三つ答えなさい。（3点）
→ 西アジア、北アフリカ、東南アジア。

(5) 仏教が主に信仰されている地域を二つ答えなさい。（4点）
→ 東南アジアと東アジア。

## 大問2: イスラム教の生活（配点 22点）

イスラム教（イスラーム）と生活の関わりについて、次の問いに答えなさい。

(1) イスラム教の聖典の名称を答えよ。（3点）
→ コーラン。

(2) イスラム教で一日に行う礼拝の回数を答え、礼拝の方角について述べなさい。（4点）
→ 一日5回。聖地メッカの方角に向かって礼拝する。

(3) イスラム教で「許されたもの」を意味する食品基準を何というか。また、食べることが禁じられている代表的なものを二つ答えなさい。（5点）
→ ハラール。禁じられているものは豚肉とアルコール。

(4) ラマダン（断食月）について、断食がどのように行われるか説明しなさい。（4点）
→ ラマダンの期間中、日の出から日没まで食事や飲み物を断つ。日没後には食事をとることができる。

(5) イスラム教の礼拝施設の名称を答えよ。また、イスラム教の聖地はどこか答えなさい。（3点）
→ 礼拝施設はモスク。聖地はメッカ。

(6) 日本でもハラール対応の食品や飲食店が増えている理由を考えて説明しなさい。（3点）
→ 観光やビジネスで訪日するイスラム教徒が増加しており、彼らが安心して食事できる環境を整える必要があるため。

## 大問3: ヒンドゥー教と仏教（配点 18点）

ヒンドゥー教と仏教について、次の問いに答えなさい。

(1) ヒンドゥー教で神聖とされている動物は何か。また、そのためにどのような食事の決まりがあるか。（4点）
→ 牛。牛肉を食べることが禁じられている。

(2) ヒンドゥー教の信者が沐浴を行う川の名前を答え、沐浴の意味を説明しなさい。（5点）
→ ガンジス川。ガンジス川の水で体を清めることで罪が洗い流されると信じられている。

(3) インドの街中で牛が自由に歩いている理由を説明しなさい。（4点）
→ ヒンドゥー教で牛が神聖な動物とされているため、人々は牛を大切にし自由に行動させているから。

(4) 仏教の礼拝施設の名称を答えよ。（2点）
→ 寺院。

(5) 仏教に由来する日本の行事や文化を一つ挙げなさい。（3点）
→ お盆（先祖の霊を迎えて供養する行事）。

## 大問4: 衣食住の工夫（配点 22点）

世界各地の気候に合わせた衣食住の工夫について、次の問いに答えなさい。

(1) 熱帯地域で高床式住居が建てられる理由を二つ答えなさい。（4点）
→ 風通しをよくして暑さをやわらげるため。湿気や害虫を防ぐため。

(2) 乾燥帯で日干しれんがが住居に使われる利点を説明しなさい。（4点）
→ 厚い壁が昼の猛暑を遮り夜の冷え込みも防ぐことができる。また現地で手に入る泥を材料にできるため建築しやすい。

(3) 寒帯のイヌイットがかつて利用していたイグルーはどのように作られ、なぜ暖かいのか説明しなさい。（5点）
→ 雪のブロックを積み上げてドーム型に作る。雪には空気が含まれていて断熱効果があるため、外の極寒をさえぎり体温で内部が暖まる。

(4) 世界の主食の違いについて、アジア・ヨーロッパ西アジア・草原地帯のそれぞれの主食を答えなさい。（3点）
→ アジア：米、ヨーロッパ・西アジア：パンやナン、草原地帯：乳製品や肉。

(5) 世界各地で主食が異なる理由を「気候」「栽培」の語を使って説明しなさい。（6点）
→ それぞれの地域の気候に適した作物が栽培されるため、栽培できる作物の種類によって主食が決まる。雨の多いアジアでは稲作に適した気候で米が主食となり、乾燥した地域では小麦が栽培されパンが主食となっている。

## 大問5: グローバル化と宗教の共存（配点 18点）

生活のグローバル化と宗教について、次の問いに答えなさい。

(1) 生活のグローバル化の具体例を三つ挙げなさい。（6点）
→ ファストフード店が世界中に広がったこと。ジーンズやTシャツなど共通の衣服が普及したこと。インターネットやスマートフォンが世界中で使われるようになったこと。

(2) グローバル化が進む中で懸念されていることは何か。「伝統文化」の語を使って説明しなさい。（4点）
→ それぞれの地域がもつ独自の伝統文化や生活様式が、共通の文化に置き換えられて失われてしまうこと。

(3) 異なる宗教をもつ人々が共に暮らすために必要なことは何か。（4点）
→ 互いの宗教や文化の違いを理解し、尊重し合うこと。食事の決まりや礼拝の時間など、宗教的な習慣への配慮も大切。

(4) 日本では近年、ハラール対応やベジタリアン対応の飲食店が増えている。このことが社会にとってもつ意味を説明しなさい。（4点）
→ さまざまな宗教や文化的背景をもつ人々を受け入れる多文化共生社会の実現につながり、観光や国際交流の促進にも貢献する。
