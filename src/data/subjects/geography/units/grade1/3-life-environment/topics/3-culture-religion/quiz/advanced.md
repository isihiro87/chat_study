# 多様な生活・文化と宗教 発展問題

## 問1: 宗教と食文化の関係

次の文章を読んで、あとの問いに答えなさい。

世界には宗教の教えによって食べてはいけないものが定められている例がある。イスラム教では（　A　）の肉を食べることが禁じられ、教えに従った食品は（　B　）とよばれる。ヒンドゥー教では（　C　）が神聖な動物とされ、その肉を食べない。また、仏教では殺生を避ける教えから菜食（ベジタリアン）の伝統がある地域もある。

(1) 空欄A〜Cに入る語句をそれぞれ答えなさい。
→ A：豚、B：ハラール、C：牛。

(2) ある国際会議で、イスラム教徒・ヒンドゥー教徒・仏教徒が参加する食事会を開くことになった。全員が食べられる料理のメニューを考える場合、避けるべき食材を三つ挙げなさい。
→ 豚肉（イスラム教で禁止）、牛肉（ヒンドゥー教で禁止）、アルコール（イスラム教で禁止）。

(3) 上記(2)を踏まえて、全員が安心して食べられるメニューの例を一つ提案し、その理由を説明しなさい。
→ 鶏肉や魚を使った料理（ハラール認証のもの）。豚肉と牛肉を避け、イスラム教の方法で処理されたハラール食品を使えば、三つの宗教の信者全員が食べることができるため。

(4) 宗教による食事の決まりが存在することは、国際社会でどのような配慮を必要とするか。100字以内で述べなさい。
→ 国際交流や観光の場面で、相手の宗教的な食事の決まりを事前に理解し、対応できる食事の選択肢を用意するなど、多様な文化・宗教への配慮が不可欠である。

## 問2: 宗教の分布と歴史的背景

次の地図情報を読んで、あとの問いに答えなさい。

【宗教の主な分布】
- キリスト教：ヨーロッパ、南北アメリカ、オセアニア
- イスラム教：西アジア、北アフリカ、中央アジア、東南アジアの一部
- ヒンドゥー教：インド、ネパール
- 仏教：東南アジア（タイ、ミャンマーなど）、東アジア（日本、中国、韓国）

(1) キリスト教がヨーロッパだけでなく南北アメリカやオセアニアにも広がっている理由を、歴史的な背景から説明しなさい。
→ 15世紀以降の大航海時代にヨーロッパの国々が南北アメリカやオセアニアを植民地化し、キリスト教の布教活動が行われたため、これらの地域にキリスト教が広まった。

(2) イスラム教が西アジアから北アフリカや東南アジアにまで広がった理由として考えられることを述べなさい。
→ イスラム帝国の拡大やアラブ商人の交易活動を通じて、西アジアから北アフリカ、さらに海上交易で東南アジアにまでイスラム教が伝わった。

(3) ヒンドゥー教がインド以外にあまり広がらなかった理由として考えられることを述べなさい。
→ ヒンドゥー教にはカースト制度と深く結びついた社会的な仕組みがあり、特定の民族や文化と密接に関係しているため、他の地域への布教が行われにくかったから。

(4) 東南アジアには仏教、イスラム教、キリスト教など複数の宗教が混在している。このことから東南アジアの歴史についてどのようなことが推測できるか。
→ 東南アジアは古くからインドや中国、アラブ世界、ヨーロッパとの交流が盛んであったため、さまざまな時代にさまざまな地域の文化や宗教の影響を受けてきたことが推測できる。

## 問3: グローバル化と伝統文化の保存

次の事例を読んで、あとの問いに答えなさい。

【事例A】アフリカのある村では、伝統的な祭りや儀式が若い世代に受け継がれにくくなっている。スマートフォンの普及により、若者が都市文化に憧れて村を離れるケースが増えている。

【事例B】日本の観光地では、外国人観光客向けに着物体験や茶道体験が人気を集めており、伝統文化への関心が高まっている。

(1) 事例Aで伝統文化が失われつつある原因を二つ挙げなさい。
→ スマートフォンやインターネットの普及により都市文化が流入し、若者が伝統文化より都市文化に魅力を感じていること。若者が村を離れて都市に移住するため、伝統を受け継ぐ人が減っていること。

(2) 事例Bのように、グローバル化が逆に伝統文化の再評価につながることがある。その理由を説明しなさい。
→ 外国の人々が日本の伝統文化に関心をもつことで、日本人自身も自国の文化の価値を再認識する機会となり、伝統文化を維持・発信しようとする動きが活発になるから。

(3) グローバル化のなかで伝統文化を守りながら発展させるために、どのような取り組みが考えられるか。具体的に二つ提案しなさい。
→ インターネットやSNSを活用して伝統文化の魅力を世界に発信し、関心をもつ人を増やす取り組み。学校教育のなかで地域の伝統行事や技術を学ぶ授業を設け、次の世代に知識や技能を継承する取り組み。

## 問4: 衣食住と環境問題

次の問いに答えなさい。

(1) 乾燥帯の日干しれんがの住居と、現代のコンクリートの住居を環境負荷の観点から比較しなさい。
→ 日干しれんがは現地の土を使い日光で乾燥させるため、製造にエネルギーをほとんど消費しない。一方、コンクリートはセメントの製造に大量のエネルギーとCO2の排出をともなう。環境負荷の面では日干しれんがのほうが小さい。

(2) 遊牧民が定住化を進めた場合、環境や文化にどのような影響があるか。プラス面とマイナス面をそれぞれ一つ挙げなさい。
→ プラス面：教育や医療へのアクセスが向上し生活の安定性が増す。マイナス面：定住地周辺の土地に放牧が集中し過放牧による砂漠化が進む恐れがある。

(3) 気候変動が世界各地の伝統的な暮らしに与える影響の例を二つ挙げなさい。
→ 永久凍土の融解により、冷帯の高床式住居の基盤が不安定になること。海面上昇により、サンゴ礁の島々に住む熱帯の人々の生活が脅かされること。

## 問5: 住居・宗教・文化の総合問題

次のア〜オは世界各地の生活の様子を述べたものである。あとの問いに答えなさい。

ア：金曜日になると人々はモスクに集まり、共同で礼拝を行う。
イ：毎朝、川で沐浴をして一日を始める人々がいる。
ウ：日曜日に家族そろって教会に行き、賛美歌を歌う。
エ：托鉢の僧侶に食事を施す習慣がある。
オ：ラマダンの期間中、日中は水も食事もとらない。

(1) ア〜オの生活がそれぞれどの宗教に関連するか答えなさい。
→ ア：イスラム教、イ：ヒンドゥー教、ウ：キリスト教、エ：仏教、オ：イスラム教。

(2) イの「川で沐浴」について、この川の名前を答え、沐浴が行われる宗教的な意味を説明しなさい。
→ ガンジス川。ヒンドゥー教では聖なる川の水で体を清めることで罪が洗い流され、魂が浄化されると信じられている。

(3) アとオはどちらもイスラム教に関するものである。イスラム教が日常生活に深く関わっていることがわかる理由を、この二つの事例をもとに説明しなさい。
→ 毎週金曜日の集団礼拝やラマダン中の断食は日常の行動を直接規定するものであり、信仰が食事や生活リズム、社会活動と密接に結びついていることを示している。

(4) 日本には仏教・神道・キリスト教など複数の宗教文化が共存している。日本における宗教文化の特徴を、お盆やクリスマスの例を使って説明しなさい。
→ 日本ではお盆（仏教）やクリスマス（キリスト教）、初詣（神道）など、異なる宗教に由来する行事が季節の文化として広く親しまれている。特定の宗教に強くこだわらず、さまざまな宗教の文化を柔軟に取り入れているのが日本の特徴である。
