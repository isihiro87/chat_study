# 日本の領域 発展問題

## 問1: 排他的経済水域の計算と比較

日本の国土面積は約38万km²で、排他的経済水域は約447万km²（国土面積の約10倍以上）である。

(1) 日本の排他的経済水域の面積を国土面積で割ると、およそ何倍になるか計算しなさい。
→ 447万km²÷38万km²≒11.8倍。約12倍になる。

(2) 日本と面積がほぼ同じドイツ（約36万km²）は内陸部が多く、EEZは日本よりはるかに小さい。日本のEEZがドイツよりも広い理由を「島」「海」の2語を用いて説明しなさい。
→ 日本は約6,800の島が広い海域に散らばっている島国であり、各島を基点に200海里のEEZが設定される。ドイツは大陸に位置し、面する海が北海とバルト海に限られるため、EEZが狭い。

(3) 沖ノ鳥島が水没した場合に失われる排他的経済水域（約40万km²）は、日本の国土面積（約38万km²）と比べてどのような大きさか。このことから沖ノ鳥島の護岸工事の重要性を説明しなさい。
→ 失われるEEZ（約40万km²）は日本の国土面積（約38万km²）とほぼ同じ広さである。たった1つの小さな島が水没するだけで、国土面積に匹敵する広大な海域の管理権を失うことになるため、護岸工事による島の保全は極めて重要である。

## 問2: 領土問題の多角的理解

(1) 北方領土・竹島・尖閣諸島の3つの領土問題について、「現在どの国が実効支配しているか」という観点で分類しなさい。
→ 日本が実効支配しているもの：尖閣諸島。他国が実効支配（不法占拠）しているもの：北方領土（ロシア）、竹島（韓国）。

(2) 尖閣諸島について、日本政府が「領土問題は存在しない」としている理由を、他の2つの領土問題との違いに着目して説明しなさい。
→ 北方領土や竹島は他国に占拠されているため、返還や解決に向けた「問題」が存在する。一方、尖閣諸島は日本が有効に支配しており、歴史的にも国際法上も日本固有の領土であることが明確であるため、解決すべき領土問題は存在しないというのが日本政府の立場である。

(3) 1970年代に国連の調査で尖閣諸島周辺に石油・天然ガスの存在が指摘されてから、中国が領有権を主張し始めた。このことから領土問題と資源の関係について考察しなさい。
→ 領土問題の背景には、その地域に存在する資源への関心が大きく関わっている場合がある。尖閣諸島の場合、地下資源の可能性が明らかになって以降に中国が領有権を主張し始めたことから、資源の存在が領土をめぐる対立を激化させる要因になりうることがわかる。

(4) 北方領土問題が解決していない歴史的経緯を「1945年」「ソ連」という語を用いて説明しなさい。
→ 1945年、第二次世界大戦の終戦直後にソ連が日ソ中立条約に違反して北方領土を占拠した。以後、ソ連（現ロシア）は返還に応じておらず、日本は一貫して返還を求めて交渉を続けている。

## 問3: 西之島と排他的経済水域の拡大

小笠原諸島の西之島では2013年から大規模な火山噴火が始まり、島の面積が大幅に拡大した。

(1) 火山の噴火によって島の面積が拡大すると、排他的経済水域がどのように変化するか説明しなさい。
→ 島の海岸線が広がることで、その海岸線を基点とする200海里の排他的経済水域の範囲も拡大する。新たに生まれた陸地の周囲にも主権がおよび、より広い海域を管理できるようになる。

(2) 逆に、火山活動や浸食により島が水没した場合、排他的経済水域にどのような影響があるか。
→ 島が完全に水没すると、その島を基点とする排他的経済水域が消滅する。島のない海域は公海となり、どの国も自由に利用できるようになるため、沿岸国は資源管理の権利を失う。

(3) 沖ノ鳥島と西之島は、日本のEEZ確保においてそれぞれどのような役割を果たしているか比較して説明しなさい。
→ 沖ノ鳥島は日本最南端に位置する既存の島で、護岸工事によって水没を防ぐことで約40万km²のEEZを維持する「守り」の役割を果たしている。西之島は火山噴火によって面積が拡大し、新たにEEZが広がるという「攻め」の役割を果たしている。

## 問4: 領海と排他的経済水域の法的比較

(1) 領海と排他的経済水域について、次の観点から比較する表を完成させなさい。

| 項目 | 領海 | 排他的経済水域 |
|------|------|-------------|
| 範囲 | （ア） | （イ） |
| 他国の船の通行 | （ウ） | （エ） |
| 沿岸国の権利 | （オ） | （カ） |

→ ア＝沿岸から12海里（約22km）、イ＝沿岸から200海里（約370km）。ウ＝無害通航のみ認められる、エ＝自由に航行できる。オ＝主権がおよび、すべての活動を管理できる、カ＝水産資源や鉱産資源の調査・開発を独占的に行える。

(2) 排他的経済水域内での他国の漁船による違法操業が問題になることがある。なぜEEZ内で他国の船が自由に航行できるのに、漁業は制限されるのか説明しなさい。
→ EEZでは航行の自由は認められているが、水産資源などの天然資源を利用する権利は沿岸国に独占的に属する。他国が漁業を行うには沿岸国の許可が必要であり、無許可の操業は違法操業となる。

(3) 日本が「海洋国」として排他的経済水域を重視している理由を、資源とエネルギーの観点から説明しなさい。
→ 日本は資源の多くを輸入に頼っている国であるが、広大なEEZ内には水産資源のほか、レアアース、メタンハイドレート、石油・天然ガスなどの鉱産・エネルギー資源が存在する可能性がある。EEZを確保し、海底資源の調査・開発を進めることで、将来的な資源の自給率向上につながる可能性がある。

## 問5: 総合記述

(1) 「日本の領域と排他的経済水域の特色」というテーマで、以下のキーワードをすべて用いて200字程度で説明しなさい。
キーワード：島国、約38万km²、12海里、200海里、沖ノ鳥島

→ 日本は約6,800の島からなる島国で、国土面積は約38万km²である。領海は沿岸から12海里、排他的経済水域は200海里の範囲に設定されている。島国であるため各島を基点にEEZが広がり、その総面積は国土の約10倍にもなる。特に日本最南端の沖ノ鳥島は、この島1つで約40万km²ものEEZを生み出しており、護岸工事により保全されている。日本は領土面積こそ大きくないが、海洋面積を含めれば世界有数の広さをもつ海洋国といえる。
