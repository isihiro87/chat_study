# 都道府県 大問小問

## 大問1: 廃藩置県と都道府県のなりたち（配点 20点）

(1) 江戸時代、日本はおよそいくつの藩に分かれていたか。（3点）
→ 約300の藩。

(2) 1871年に行われた、藩を廃止して府や県を置く改革を何というか。（3点）
→ 廃藩置県。

(3) 廃藩置県の直後、全国にはいくつの府と県が置かれたか。（3点）
→ 3府302県。

(4) その後、統合が進んで現在の都道府県の形が定まったのは何年か。（3点）
→ 1888年。

(5) 現在の47都道府県の内訳を「都・道・府・県」の数で答えなさい。（4点）
→ 1都（東京都）、1道（北海道）、2府（大阪府・京都府）、43県。

(6) 「都」「道」「府」の呼び方がそれぞれ異なる歴史的な理由を簡潔に答えなさい。（4点）
→ 「都」は日本の首都であること、「道」は広大な北海道の特殊な行政区分、「府」は京都・大阪が古くから政治の中心地であったことに由来する。

## 大問2: 7地方区分（配点 22点）

(1) 日本の7地方区分を北から順にすべて答えなさい。（7点）
→ 北海道地方、東北地方、関東地方、中部地方、近畿地方、中国・四国地方、九州地方。

(2) 東北地方に含まれる6県をすべて答えなさい。（3点）
→ 青森県、岩手県、宮城県、秋田県、山形県、福島県。

(3) 中部地方に含まれる9県をすべて答えなさい。（3点）
→ 新潟県、富山県、石川県、福井県、山梨県、長野県、岐阜県、静岡県、愛知県。

(4) 九州地方に含まれる8県をすべて答えなさい。（3点）
→ 福岡県、佐賀県、長崎県、熊本県、大分県、宮崎県、鹿児島県、沖縄県。

(5) 沖縄県が現在の47番目の都道府県として加わったのは何年か。どのような出来事がきっかけだったか。（6点）
→ 1972年。第二次世界大戦後にアメリカの統治下にあった沖縄が日本に返還されたことがきっかけ。

## 大問3: 県庁所在地（配点 24点）

(1) 県庁所在地とは何か。「政治」という語を用いて説明しなさい。（4点）
→ 都道府県庁が置かれている都市で、その都道府県の政治の中心的な機関（行政機関、議会、裁判所など）が集まる都市のこと。

(2) 次の県の県庁所在地をそれぞれ答えなさい。（各2点、計10点）
① 宮城県 → 仙台市
② 栃木県 → 宇都宮市
③ 愛知県 → 名古屋市
④ 兵庫県 → 神戸市
⑤ 沖縄県 → 那覇市

(3) 埼玉県の県庁所在地の正しい表記を答えなさい。また、なぜその表記になったか理由も答えなさい。（4点）
→ さいたま市（ひらがな表記）。2001年に浦和市・大宮市・与野市が合併して誕生した市で、特定の旧市名を使わず、ひらがなの「さいたま」が採用された。

(4) 県庁所在地が県名と異なる場合が多い理由を、「城下町」という語を用いて説明しなさい。（6点）
→ 明治時代に県庁を設置する際、その県で最も発展していた城下町や港町が選ばれた。藩の中心であった城下町は政治・経済の拠点であり、行政機関を置くのに適していたが、その都市名が必ずしも県名と一致しなかったため。

## 大問5: 県境と地理的特徴（配点 16点）

(1) 都道府県の境界が山脈を利用して引かれている例を1つ答えなさい。（3点）
→ 例：奥羽山脈が秋田県と岩手県の境界の一部になっている。

(2) 都道府県の境界が河川を利用して引かれている例を1つ答えなさい。（3点）
→ 例：利根川が埼玉県と群馬県の境界の一部になっている。

(3) 県境が自然地形で決められていることのメリットを1つ答えなさい。（4点）
→ 山や川は目に見える地形であるため、境界が明確でわかりやすい。また、山脈や河川は交通の障壁となるため、文化・経済圏の区切りとしても自然な境界となる。

(4) 7地方区分は、地理的な位置のほかにどのような基準で分けられているか答えなさい。（3点）
→ 歴史的なまとまりや文化的な共通点、経済的なつながりなどを基準にしている。

(5) 中部地方が北陸・中央高地・東海の3つにさらに細分されることがある理由を答えなさい。（3点）
→ 中部地方は9県と広大で、日本海側の北陸、内陸部の中央高地、太平洋側の東海では気候や産業、文化が大きく異なるため。
