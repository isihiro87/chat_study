# 時差 大問小問

## 大問1: 時差のしくみ（配点 18点）

(1) 地球が1回転するのにかかる時間と角度を答えなさい。（3点）
→ 24時間で360度回転する。

(2) 経度何度ごとに1時間の時差が生じるか。計算過程も示しなさい。（4点）
→ 360度÷24時間＝15度。経度15度ごとに1時間の時差が生じる。

(3) 時差が生まれる原因を、地球の運動と関連づけて説明しなさい。（4点）
→ 地球が自転しているため、場所によって太陽の当たり方が異なる。東側の方が先に太陽が昇るため、東ほど時刻が進んでいる。

(4) 経度0度の経線を何というか。また、どこを通っているか。（3点）
→ 本初子午線。イギリスのロンドン（旧グリニッジ天文台）を通る。

(5) 経度60度離れた2地点の時差は何時間か、計算しなさい。（4点）
→ 60度÷15度＝4時間。

## 大問2: 日本の標準時とイギリスとの時差（配点 22点）

(1) 日本の標準時子午線は東経何度の経線か。（3点）
→ 東経135度。

(2) この経線が通る代表的な都市名を答えなさい。（3点）
→ 兵庫県明石市。

(3) 日本とイギリス（ロンドン、経度0度）の時差を計算しなさい。計算過程も示すこと。（5点）
→ 日本の標準時子午線は東経135度、ロンドンは経度0度。経度の差＝135度。時差＝135度÷15度＝9時間。日本はイギリスより東にあるため、9時間進んでいる。

(4) ロンドンが1月15日の午前10時のとき、日本は何月何日の何時か。（5点）
→ ロンドンの午前10時に9時間を足す。10時＋9時間＝19時（午後7時）。日付は変わらず1月15日の午後7時。

(5) 日本が4月20日の午前2時のとき、ロンドンは何月何日の何時か。（6点）
→ 日本の午前2時から9時間を引く。2時−9時間＝−7時間。前日にさかのぼり、4月19日の17時（午後5時）。

## 大問3: さまざまな都市との時差計算（配点 25点）

(1) ニューヨーク（西経75度）と日本（東経135度）の経度の差を求めなさい。（4点）
→ 東経と西経にまたがるので足し算する。135度＋75度＝210度。

(2) (1)の経度の差から時差を求めなさい。（4点）
→ 210度÷15度＝14時間。日本はニューヨークより14時間進んでいる。

(3) 日本が8月10日の午後3時のとき、ニューヨークは何月何日の何時か。（5点）
→ 日本の午後3時（15時）から14時間を引く。15時−14時間＝1時。8月10日の午前1時。

(4) 北京（東経120度）と日本（東経135度）の時差を求めなさい。（4点）
→ ともに東経なので引き算する。135度−120度＝15度。15度÷15度＝1時間。日本は北京より1時間進んでいる。

(5) パリの標準時子午線を東経15度とする。パリが7月14日の午後8時のとき、日本は何月何日の何時か。計算過程を示して答えなさい。（8点）
→ 経度の差＝135度−15度＝120度。時差＝120度÷15度＝8時間。日本はパリより8時間進んでいる。パリの午後8時（20時）＋8時間＝28時＝翌日の午前4時。日本は7月15日の午前4時。

## 大問4: 日付変更線（配点 18点）

(1) 日付変更線はおよそ経度何度の線に沿って引かれているか。（3点）
→ 経度180度。

(2) 日付変更線を西から東に越えるとき、日付はどうなるか。（3点）
→ 日付を1日戻す。

(3) 日付変更線を東から西に越えるとき、日付はどうなるか。（3点）
→ 日付を1日進める。

(4) 日付変更線が直線ではなく曲がりくねって引かれている理由を説明しなさい。（4点）
→ 日付変更線上に国の領土がある場合、同じ国なのに日付が異なると生活に支障をきたす。そのため陸地を避けるように曲げて引かれている。

(5) 日本は日付変更線の西側にある。このことが日本の時刻にどのような影響を与えているか説明しなさい。（5点）
→ 日本は日付変更線の西側に位置するため、世界の中でも早い時刻に新しい1日を迎える国のひとつである。世界でいち早く元日を迎えることができるのもこの位置関係による。

## 大問5: 時差と生活（配点 17点）

(1) 日本のプロ野球の試合が夜7時に始まる場合、ロンドンでは何時に始まることになるか。（4点）
→ 日本の夜7時（19時）からロンドンとの時差9時間を引く。19時−9時間＝10時。ロンドンでは同日の午前10時に始まる。

(2) アメリカのロサンゼルス（西経120度）からの生中継を日本のテレビで午前10時に見ている場合、ロサンゼルスの現地時間は何時か。（5点）
→ 経度の差＝135度＋120度＝255度。ただし255度＞180度なので、360度−255度＝105度を使い、時差＝105度÷15度＝7時間（ただし逆方向）。正しくは255度÷15度＝17時間。日本の方が17時間進んでいる。日本の午前10時−17時間＝前日の午後5時。

(3) 日本で夜中の午前2時に海外のサッカーの試合を中継することがある。この試合がヨーロッパで行われていると考えると、現地では何時頃か。ヨーロッパとの時差を8〜9時間として推定しなさい。（4点）
→ 日本の午前2時から8〜9時間を引くと、前日の午後5〜6時頃。ヨーロッパでは夕方〜夜の時間帯に試合が行われていることになる。

(4) 時差がある国と電話する際に注意すべき点を、具体例をあげて説明しなさい。（4点）
→ 相手の国の時刻を計算して、夜中や早朝に電話しないように配慮する必要がある。例えば日本の午後3時にニューヨーク（時差14時間）に電話すると、現地は午前1時で真夜中になってしまう。
