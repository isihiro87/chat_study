# 並べ替え問題 作成マニュアル

## 概要

英語クイズの並べ替え問題（語順整序）を作成する際のガイドラインです。
全学年・全トピックで統一感のある問題を作成するために、このマニュアルに従ってください。

## データ構造

```typescript
{
  id: 'eng-[topic]-q[number]',
  type: 'reorder',
  question: '次の単語を正しい順番にならべて、\n「[日本語訳]」という英文を作ろう。',
  words: ['word1', 'word2', 'word3'],    // 並べ替える単語（シャッフル済み）
  correctOrder: [2, 0, 1],               // wordsの正しい並び順（インデックス）
  punctuation: '.' | '?',               // 文末記号
  options: [],                           // 空配列（型の互換性のため）
  correctIndex: -1,                      // -1固定（使用しない）
  explanation: '「[正解英文]」が正解。[文法解説]',
}
```

## 作成ルール

### 1. 問題数

- 各トピックに **3問** の並べ替え問題を追加
- 既存の選択問題（5問）の後に追加
- IDは既存問題の続き番号（例: q6, q7, q8）

### 2. 単語数の目安

| 学年 | 単語数 | 備考 |
|------|--------|------|
| 中1  | 3〜5語 | シンプルな文 |
| 中2  | 4〜6語 | やや複雑な文 |
| 中3  | 4〜7語 | 長すぎる場合は文の一部を並べ替え |

### 3. 長い文の対処法

単語数が7語を超える場合は、文の一部を固定して並べ替え範囲を限定する:

```typescript
// NG: 単語が多すぎる
words: ['I', 'have', 'lived', 'in', 'Tokyo', 'for', 'ten', 'years']

// OK: 文の一部を並べ替え
question: '次の単語を正しい順番にならべて、\n「私は10年間東京に住んでいます。」という英文を完成させよう。\nI (　　)(　　)(　　) in Tokyo for ten years.',
words: ['lived', 'have', 'in'],
// → "I have lived in Tokyo for ten years."
```

### 4. 単語の表記ルール

- **全て小文字で格納**: `words: ['she', 'is', 'kind']`
- **例外**: 固有名詞と `I` は大文字のまま: `words: ['I', 'am', 'a', 'student']`
- 文頭の大文字化はUI側で自動処理される
- 冠詞（a/an/the）、前置詞（in/on/at）も個別の単語として扱う

### 5. correctOrder の作成方法

`words` 配列のインデックスを正しい語順で並べる:

```
words:        ['a',  'I',  'student', 'am']
インデックス:   0     1      2         3

正しい文: I am a student
correctOrder: [1, 3, 0, 2]
              (I=1, am=3, a=0, student=2)
```

### 6. words のシャッフル

- 正解の語順とは異なる順番で配置する
- ランダムに見えるよう、正解の語順から十分にシャッフルする
- ただし、最初の単語が正解の最初の単語にならないようにする（ヒントになるため）

### 7. question の書き方

基本パターン:
```
'次の単語を正しい順番にならべて、\n「[日本語訳]」という英文を作ろう。'
```

文の一部を並べ替える場合:
```
'次の単語を正しい順番にならべて、\n「[日本語訳]」という英文を完成させよう。\n[固定部分] (　　)(　　)(　　) [固定部分]'
```

### 8. explanation の書き方

```
'「[正解英文]」が正解。[その文法のポイントを1〜2文で解説]'
```

## 学年別の問題設計指針

### 中1（Grade 1）

- 基本的な語順（SVO、SVC）を問う
- be動詞・一般動詞の否定文・疑問文の語順
- 簡単な文を完全に並べ替え

例:
- `I like music.`（SVO基本）
- `Do you play tennis?`（疑問文の語順）
- `She doesn't like math.`（否定文の語順）

### 中2（Grade 2）

- 接続詞を含む文の語順
- 不定詞・助動詞を含む語順
- 比較表現の語順
- やや長い文は一部並べ替えも可

例:
- `I want to be a teacher.`（不定詞）
- `You must not run here.`（助動詞の否定）
- `She is taller than her sister.`（比較級）

### 中3（Grade 3）

- 現在完了の語順
- 関係代名詞を含む文の一部並べ替え
- 仮定法の語順
- 複雑な文は一部並べ替えを積極的に使用

例:
- `I have lived here for five years.`（現在完了）
- 一部: `The boy (　　)(　　)(　　) is my brother.` → `who plays soccer`（関係代名詞）
- `If I were you, I would study harder.`（仮定法）

## チェックリスト

問題作成後、以下を確認:

- [ ] `type: 'reorder'` が設定されている
- [ ] `words` の単語数が適切（3〜7語）
- [ ] `correctOrder` が正しい語順になっている
- [ ] `punctuation` が適切（平叙文は `.`、疑問文は `?`）
- [ ] `options: []` と `correctIndex: -1` が設定されている
- [ ] `explanation` に正解文と解説がある
- [ ] IDが既存問題と重複していない
- [ ] 固有名詞と `I` 以外は小文字で格納されている
- [ ] words がシャッフルされている（正解順でない）
