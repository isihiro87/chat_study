#### index.ts（フラッシュカード：言葉答え→数式問題に変換）

フィードバック: 「言葉で答えるような問題は数学では原則なしに。それを使うような数式を出題すれば問題として成立する」

- **fc18**: 「1. 各かっこを展開 → 2. 同類項をまとめる」→ 「$2(3x+1)-(x-5)$ を計算すると？ → $5x+7$」
- **fc19**: 「式の中の各項の文字の次数のうち最も大きいもの」→ 「$3x^2+2x-1$ は何次式？ → $2$ 次式」
- **fc20**: 「同類項をまとめ、次数の高い順に書く」→ 「$2x-5+3x^2$ を整理すると？ → $3x^2+2x-5$」
- **fc21**: 「分母をそろえて（通分して）から計算する」→ 「$x/2+x/3$ を計算すると？ → $5x/6$」
- **fc23**: 「符号が全部変わる」→ 「$-2(3x-4)$ を展開すると？ → $-6x+8$」
- **fc24**: 「逆数をかける」→ 「$(4x+2)÷1/3$ を計算すると？ → $12x+6$」
- **fc26**: 「先に式を整理してから代入する」→ 「$3x+2y-1$ で $x=2,y=1$ のとき？ → $11$」

- **fc28**: 「交換法則と結合法則を使って…」→ 「$2x+3+3x+4$ を整理すると？ → $5x+7$」
