# つづもん タブ付きリッチメニュー画像生成プロンプト

PDF教材「つづもん」（問題集19冊＋参考書19冊）の購入者向けに、既存の公式LINE上で
**タブで切り替わる 2 枚組**のリッチメニューを出す。左タブ「毎日クイズ」＝既存の
一問一答、右タブ「つづもん」＝PDF教材の学習。

このメニュー画像は、**既存の `default.jpg` をベースに、テイストを一切変えずに**
「ヘッダーバーをタブバーに置き換える」＋「つづもん側の6タイルを差し替える」
だけを施して作る。

## 前提

- ベース画像: `data/line-richmenu/default.jpg`（現在全ユーザーに出ている本番メニュー）
- ベース画像のテイスト（**厳守**）:
  - 上部にネイビー(`#1a3a7a`〜`#0f2557` 系)の帯。斜めストライプの装飾、白抜きの太丸ゴシック、黄色のアクセント文字、白い吹き出し、右端に3Dの本＋鉛筆イラスト
  - 下に **6枚（3×2）の白い角丸カード**。カードごとに固有のアクセント色の細いボーダー、左上に斜めストライプ、右下にドットパターン
  - 各カードは **左に円形の3Dグロッシーなレンダー風アイコン**、右に濃紺の太字ラベル（1〜2行）、その下にアクセント色の短いアンダーライン、さらに下に小さなグレーのサブテキスト
  - 背景は白〜極薄グレー
- 構成・色味・装飾密度・アイコンの立体感は **`default.jpg` と同等**に保つ

## デザイン規約（`docs/design-guide.md`）

- **indigo / purple / violet 系は使用禁止**（ネイビー＝青系はベース画像通りOK）
- 「プレミアム」「¥680」「¥980」「お試し」「王冠 👑」等の**課金訴求・限定演出は一切入れない**
- つづもん側のアクセントは参考書PDFと揃えた**落ち着いたオレンジ茶**（`#b45309` / `#f59e0b` / `#7c2d12`）

## 画像仕様

| 項目 | 値 |
|------|------|
| サイズ | 2500 × 1686 px |
| タブバー | 上部 **250 px**（`setup-workbook-richmenu.ts` の `TAB_H` と一致） |
| ボディ | 1436 px = **718 px × 2段** |
| グリッド | **3列 × 2段 = 6タイル**、各 833 × 718 px |
| ファイル上限 | 1 MB（`prepare-richmenu-images.ts` が自動圧縮） |
| 整形コマンド | `npx tsx scripts/prepare-richmenu-images.ts` |

> ⚠️ ベース画像 `default.jpg` はヘッダー250px＋843px×2段の6タイルだが、
> 本メニューは**タブバー250pxを挟むためタイルの高さが 843→718 px に縮む**。
> カード内の要素（アイコン径・文字サイズ）を約 85% に詰めて収めること。

## タブバーの設計（2枚共通）

- 左タブ「📚 毎日クイズ」＝ネイビー系（既存の一問一答の色）
- 右タブ「📖 つづもん」＝アンバー系（参考書PDFの色）
- **アクティブ側**: 自分の色で塗り、白の太字、上角丸、下のボディと地続きに見せる
- **非アクティブ側**: 暗く沈ませる（彩度を落とす）＋ 小さく「タップで切替 ⇄」のヒント
- ボディの背景は、アクティブなタブの色をごく薄く敷いて「今どちらの世界にいるか」が一目で分かるようにする

### ③ `tz-tab-switch-mockup.png` — タブ切替のイメージ図（**LINEには登録しない参考資料**）

- サイズ自由（横長・2400px程度）。`data/line-richmenu/` 直下に置く
- スマホのLINEトーク画面のモックを**左右2つ**並べる
  - 左: 「毎日クイズ」タブがアクティブな状態（①の画像がリッチメニュー枠に入っている）
  - 右: 「つづもん」タブがアクティブな状態（②の画像）
- 左のモックの「つづもん」タブ部分に**指のタップアイコン**を重ね、左→右に**太い矢印**を引く
- 矢印の上に「タブをタップするだけ・トークは流れない」とキャプション
- 画面上部にトークルームのヘッダー（アカウント名「チャットでスタディ」）、リッチメニューの下にチャットバー「メニューをひらく」を描く
- 目的は**社内でレイアウトを合意するための説明図**。実機スクショではないことが分かる程度のシンプルなモックでよい

## 生成後の手順（画像承認後に実施）

```bash
# 1. 2500×1686 / 1MB以下に整形
npx tsx scripts/prepare-richmenu-images.ts

# 2. リッチメニュー作成・画像アップ・alias登録
npx tsx scripts/setup-workbook-richmenu.ts

# 3. 出力された richMenuId を functions/.env の LINE_RICHMENU_WORKBOOK_ID に設定
```
