# 公式LINE KPI ダッシュボード

公式LINE「チャットでスタディ」を**登録者数ではなくファネル（獲得→活性化→継続→学習→収益化）**で見るための定点ダッシュボード。
**毎週、実測列に数字を1列追記**してトレンドを追う。目標値は固定、実測は更新。

> 前提（2026-06）: 現在は**全機能無料・トライアル廃止モデル**（`memory/line-send-cost-strategy.md`）。
> 有料プラン未提供のため「有料転換率/premium有効数」は当面 N/A（導線の失敗ではない）。

## いま追うべき優先順位

1. **テスト範囲設定率（≈25%／新規初日11%）** — 唯一の構造的赤信号。登録直後の範囲設定動線を強化。
2. **無回答層のブロック（28.7%）＆ブロックの46%が48h以内** — 初日に「まず1問」へ確実に誘導すれば獲得・継続・ブロックを同時に改善。
3. 計測の穴を埋める：**配信開封率**（OA Manager）と **D30コホート継続率**（7月以降、登録30日超が出てから）。
