# default リッチメニュー画像生成プロンプト（全ユーザー共通・課金導線なし）

2026-06 のトライアル廃止・満足度優先への転換に伴い、公式LINE のリッチメニューを
**無料/有料の区別なく全ユーザー共通の 1 枚**に統一する。プレミアム案内ボタンは
持たない。

このメニュー画像は、**既存の `premium.jpg` をベースに、テイストを変えずに**
最小限の差分（ラベル変更 + 2 タイルの入れ替え + 課金訴求要素の除去）だけを
施して作る。

## 前提

- ベース画像: `/workspaces/marutto-study/data/line-richmenu/premium.jpg`
- ベース画像のテイスト: 上部ヘッダーバー + **6 ブロック（3×2）** の白カード + カラフルで
  やや立体感のあるイラストアイコン + 各ブロック大きめタイトル + 小さなサブテキスト
- 構成・色味・装飾密度は **`premium.jpg` と同等**に保つ
- デザイン規約（`docs/design-guide.md`）遵守:
  - アクセント色は **amber-500 `#F59E0B`** のみ
  - **indigo / purple / violet 系は使用禁止**
  - ボタンのグラデーション禁止（ソリッドカラー）
  - 「プレミアム」「¥680」「¥980」「お試し」「王冠 👑」「✨ 特別」等の**課金訴求・限定演出は一切入れない**

## premium.jpg からの差分（3 点だけ）

1. **左上ブロックのラベル変更**: 「追加で解く」→「**1問解く**」（ロケットアイコン・配置・トンマナはそのまま）
2. **右上と左下のタイルを入れ替え**: 「出題範囲設定」を左下へ、「じっくり学ぶ」を右上へ（各タイルのアイコン・サブテキストはそのまま一緒に移動）
3. **課金訴求要素の除去**: もし premium.jpg 内に王冠・¥価格・「プレミアム」「お試し」等の文言/装飾が残っていれば全て削除し、無料前提のニュートラルな見た目にする

## プロンプト本体（日本語版）

> 添付画像（`premium.jpg`）と一緒に画像生成 AI に投げてください。
> ChatGPT 画像生成 / Gemini 2.5 Flash Image / Nano Banana など、添付画像を参照できるモデル前提。

```
添付した画像（中学生向け学習サービス「チャットでスタディ」公式LINE のリッチメニュー、3×2 の 6 ブロック構成）をベースに、テイスト・配色・イラストの密度・全体の雰囲気はそのまま保ったまま、次の 3 点だけを変更してください。

1. 左上ブロックのタイトル「追加で解く」を「1問解く」に変更する（ロケットのアイコン・サブテキスト・配置はそのまま）。
2. 右上の「出題範囲設定」ブロックと、左下の「じっくり学ぶ」ブロックを、アイコンとサブテキストごと位置を入れ替える。つまり「じっくり学ぶ（暗記＋クイズ・本のアイコン）」を右上に、「出題範囲設定（クリップボードのアイコン）」を左下に配置する。
3. 画像内に「プレミアム」「王冠」「¥680」「¥980」「お試し」「特別」などの有料・限定をにおわせる文字や装飾があれば、すべて取り除き、無料で誰でも使えるニュートラルな見た目にする。

配色ルール:
- アクセント色はアンバー（#F59E0B 系）のみ使う。
- 紫・インディゴ・バイオレット系の色は使わない。
- ボタンはグラデーションを使わず、フラットな単色にする。
- 6 つのブロックはどれも同じ強さで、特定のボタンだけを目立たせない。

文字は日本語、サイズは横 2500 × 縦 1686 ピクセル、3×2 の 6 ブロックがそれぞれ均等（各 833×843 px 相当）に並ぶ構図にしてください。
```

## 反映手順

1. 生成画像を `data/line-richmenu/raw/default-source.png` に保存。
2. `npx tsx scripts/prepare-richmenu-images.ts` で 2500×1686 / ≤1MB に整形 → `data/line-richmenu/default.{png|jpg}` を出力。
3. `npx tsx scripts/manage-line-richmenu.ts create default` → `upload default` → `set-default default` で LINE に登録し、デフォルトリッチメニューにする。
4. 旧 `free` / `trial` / `premium` リッチメニューを LINE から削除すると、per-user link が残っている既存ユーザーもデフォルト（この新メニュー）にフォールバックする。
5. 実機で 6 ボタン表示・「1問解く」reply・「じっくり学ぶ」LIFF 起動を確認。

詳細な運用フローは `docs/operations/line-richmenu.md` を参照。
