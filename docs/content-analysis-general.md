# コンテンツ分析: 全教科共通

## 5. フィードバックログ

### [2026-03-22] カタカナへのルビ付与は不要
- **対象**: 全教科・全トピック
- **指摘**: カタカナ語にルビ（ふりがな）が付いている。カタカナは既に音を表す文字なのでルビは不要
- **対応**: 35ファイル・64件のカタカナルビを一括削除
- **汎用ルール**: カタカナ語にはルビを付けない。漢字混じりの語（例: テレビ放送）は漢字部分のみルビ可

### [2026-03-22] 用語補足(tooltip)と補足内容のずれに注意
- **対象**: 全教科・全トピック
- **指摘**: data-tooltip属性の補足テキストが、対象の用語と一致しないケースが報告されている
- **対応**: 確認した4トピックでは問題なし。継続的な注意が必要
- **汎用ルール**: tooltip追加時は、補足する用語と補足内容が一致しているか必ず確認する

### [2026-03-22] フラッシュカード・クイズから発展的すぎる用語を削除
- **対象**: 歴史（人類の出現と進化、古代文明の誕生、中国文明の発展）
- **指摘**: 中学生の学習範囲を超える発展的な用語がFC・クイズに含まれていた
- **対応**: 以下の用語を削除
  - 人類の出現: 二足歩行、移動生活、洞窟壁画、農業革命、出アフリカ、火の使用
  - 古代文明: 同害復讐法、ミイラ
  - 中国文明: ラクダ、神権政治、項羽、匈奴、度量衡の統一、半両銭、張騫、劉邦
- **汎用ルール**: FC・クイズの用語選定は教科書の太字・重要語に限定し、発展的な用語は解説セクションにのみ記載する

### [2026-03-22] クイズ選択肢の正誤判定に注意
- **対象**: 古代文明の誕生
- **指摘**: 「中国文明の代表的な川は？」の問題で、長江が誤答選択肢に含まれていたが、中国文明は黄河・長江両方の流域で発生しており不適切
- **対応**: 長江を別の川（メコン川、ガンジス川）に差し替え
- **汎用ルール**: クイズの誤答選択肢が「実は正解でもある」ケースがないか確認する

### [2026-03-22] チャット資料の会話の自然さ
- **対象**: 中国文明の発展
- **指摘**: 生徒のセリフが2つ連続し、その後に先生がまとめて回答するパターンが不自然
- **対応**: 生徒→先生→生徒→先生の交互のやり取りに整理
- **汎用ルール**: チャット資料では同じ話者が連続しないようにする（ナレーターは例外）
